# Customizing JDashboard Profile

You can configure the preferences and features of JDashboard by profile on the Server Console. This topic describes how you can customize JDashboard profile properties.

This topic contains the following sections:

* [Server Administrators](#Admin)
* [Any Users](#User)

## Server Administrators

You can navigate to the Administration > Server Profile > Customize Profile > JDashboard tab on the Server Console, create different profiles for end users to choose, set the default profile, and edit/remove the profiles you created.

![Server Profile - Customize Profile - JDashboard](https://devnet.logianalytics.com/hc/article_attachments/9905817770263/dshbrd_profile.gif)

Server provides the **Default** profile which enables all the JDashboard features. You can create different profiles with different features to serve for different purposes. To create a new profile, select **New Profile**. Then, in the [JDashboard Profile dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741405863703-JDashboard-Profile-Properties), define the features you want for the profile.

To set a default profile, select a profile from the **Default Profile** drop-down list. Server displays the **Default** profile and all other profiles that you have created in the drop-down list for your selection.

By selecting a profile name in the Name column, you can view the profile settings in the [JDashboard Profile dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741405863703-JDashboard-Profile-Properties).

For the profiles that you created, you can edit them by selecting **Edit** and then making changes in the [JDashboard Profile dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741405863703-JDashboard-Profile-Properties).

You can remove a profile if you no longer want it, select it in the profile table, then select **Delete**.

![Note icon](https://devnet.logianalytics.com/hc/article_attachments/9905612785687/noteicon.jpg)You cannot change or delete the **Default** profile.

## Any Users

You can navigate to the My Profile > Customize Profile > JDashboard tab on the Server Console, then set the default profile, or customize the profile for JDashboard.

![My Profile - Customize Profile - JDashboard](https://devnet.logianalytics.com/hc/article_attachments/9905817796119/dshbrd_profile1.gif)

To set a default profile, select a profile from the **Default Profile** drop-down list. Server displays the **Default** profile and all other profiles that server administrators have created in the drop-down list for your selection.

If you don't want to use any of the existing profiles, you can customize the profile for JDashboard.

1. Select **Customize Profile**. Server displays the [Customize Profile dialog box](https://devnet.logianalytics.com/hc/en-us/articles/7985371543191-Customize-Profile-Dialog-Box-Properties).
2. Select **Enable Customize Properties**.
3. Change the settings on the **Common** and **JDashboard** tabs as you want. Here you can also update the settings for Web Report Studio and Page Report Studio.
4. Select **OK**. Server applies your customized profile settings instead of the default profile you selected.
