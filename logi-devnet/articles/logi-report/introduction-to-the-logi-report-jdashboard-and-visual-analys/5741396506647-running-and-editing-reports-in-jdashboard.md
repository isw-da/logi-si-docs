# Running and Editing Reports in JDashboard

This topic describes how you can open a report in JDashboard and how JDashboard displays a report.

In addition to running reports from the server resource tree, JDashboard also provides an entry for displaying reports inside it, so that you do not have to switch to the resource tree on the Server Console to open a report and edit it.

Viewing a report in JDashboard launches its fully functional studio together with the report in a tab in JDashboard. The tab name is the report name. When it is a page report, JDashboard displays [Page Report Studio](https://devnet.logianalytics.com/hc/en-us/articles/5741462429975-Page-Report-Studio-Server-v19), and for a web report, [Web Report Studio](https://devnet.logianalytics.com/hc/en-us/articles/5741464180119-Web-Report-Studio-Server-v19). Using Web/Page Report Studio, you can edit reports directly in JDashboard.

To open a report in JDashboard, select the **Show Resources** icon ![Show Resources button](https://devnet.logianalytics.com/hc/article_attachments/9905736385175/btn_dshbrd_show.gif) on the toolbar to open the Resources panel. Then in the Reports node, browse to the target report and drag it to the editing area. JDashboard will load the report. If the report uses parameters, JDashboard asks you to specify the parameter values first before displaying the report.

To close an open report, select **x** on the dashboard tab for the report.
