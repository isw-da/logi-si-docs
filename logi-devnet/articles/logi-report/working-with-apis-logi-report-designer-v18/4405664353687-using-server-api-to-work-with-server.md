# Using Server API to Work with Server

The Server API is a set of Java programming interfaces that run reports, explore report resources, and provide access control for Logi Report Servers. You can use it to write servlets, JSPs, and Java applications.

For more information about using the Server API, see Using Server API to Work with Logi Report Server in the *Logi Report Server Guide*.
