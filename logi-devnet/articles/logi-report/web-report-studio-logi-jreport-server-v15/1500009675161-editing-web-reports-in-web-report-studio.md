# Editing Web Reports in Web Report Studio

Web Report Studio provides many powerful features for interfacing with reports, such as adding links, filtering report data, using web controls and so on.

To perform most of the operations in this section, you are required to use [Edit Mode](https://devnet.logianalytics.com/hc/en-us/articles/1500009675101-Web-Report-Studio#Mode) of Web Report Studio.

Pick a task from the following:

* [General Operations in Reports](https://devnet.logianalytics.com/hc/en-us/articles/1500009675201-General-Operations-in-Reports)
* [Inserting Components](https://devnet.logianalytics.com/hc/en-us/articles/1500009675221-Inserting-Components)
* [Making Simple Modifications to Components](https://devnet.logianalytics.com/hc/en-us/articles/1500009675241-Making-Simple-Modifications-to-Components)
* [Manipulating Data Components](https://devnet.logianalytics.com/hc/en-us/articles/1500009675261-Manipulating-Data-Components)
* [Binding Links to Objects](https://devnet.logianalytics.com/hc/en-us/articles/1500009650322-Binding-Links-to-Objects)
* [Using Dynamic Resources and Local Parameters](https://devnet.logianalytics.com/hc/en-us/articles/1500009650242-Using-Dynamic-Resources-and-Local-Parameters-)
* [Going Through the Report Data](https://devnet.logianalytics.com/hc/en-us/articles/1500009650302-Going-Through-the-Report-Data)
* [Applying Filters](https://devnet.logianalytics.com/hc/en-us/articles/1500009650282-Applying-Filters)
