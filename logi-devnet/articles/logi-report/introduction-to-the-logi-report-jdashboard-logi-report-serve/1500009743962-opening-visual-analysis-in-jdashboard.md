# Opening Visual Analysis in JDashboard

If your Logi Report Server enables [Visual Analysis](https://devnet.logianalytics.com/hc/en-us/articles/1500009742882-Using-Visual-Analysis-to-Analyze-Data-Intuitively-and-Instantly), you can access Visual Analysis directly from JDashboard without having to switch to the resource tree in the Server Console. JDashboard loads Visual Analysis as a tab with full functionality for you to perform data analysis with. This topic describes how you can open a new Visual Analysis or an analysis template in JDashboard.

This topic contains the following sections:

* [Opening a New Visual Analysis](#New)
* [Opening an Analysis Template](#Template)

## Opening a New Visual Analysis

You can use several ways to access Visual Analysis:

* On the dashboard title bar, leave the mouse pointer on the **+** button beside the rightmost tab for one second until JDashboard displays a drop-down menu, then select **Analysis** from the menu.
* On the toolbar, select the **New** button ![New button](https://devnet.logianalytics.com/hc/article_attachments/4404885307031/btn_newobj.gif) and then select **Analysis**.
* On the toolbar, select the **Options** button ![Options button](https://devnet.logianalytics.com/hc/article_attachments/4404880435735/btn_dshbrd_optn.gif) and then select **New** > **Analysis**.

## Opening an Analysis Template

You can access Visual Analysis for a specific analysis template, using [dialog box](#dialog), [the Resources panel](#Panel), or [the component title bar](#ComponentBar).

**Using dialog box**

1. In JDashboard, select the **Open** button ![Open button](https://devnet.logianalytics.com/hc/article_attachments/4404880136471/btn_open.gif) on the toolbar, or select the **Options** button ![Options button](https://devnet.logianalytics.com/hc/article_attachments/4404880435735/btn_dshbrd_optn.gif) and then select **Open** from the option list. JDashboard displays the Open Document dialog box.

   ![Open Document dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880442263/opendoc.gif)
2. Browse to the desired folder, select the analysis template you want to open and then select **OK**. Or simply double-click the analysis template to open it.

**Using the Resources panel**

1. Select the **Show Resources** button ![Show Resources button](https://devnet.logianalytics.com/hc/article_attachments/4404880425623/btn_dshbrd_show.gif) on the toolbar. JDashboard displays the Resources panel.
2. Expand the **Reports** node.
3. Browse to the target analysis template with **.va** as the suffix and drag it to the editing area.

   JDashboard loads Visual Analysis in a new dashboard tab and displays the selected analysis template.

**Using the component title bar**

After you [insert an analysis template in a dashboard](https://devnet.logianalytics.com/hc/en-us/articles/1500009771761-Creating-and-Saving-Dashboards#VA) as its component, select the **Options** button ![Options icon](https://devnet.logianalytics.com/hc/article_attachments/4404880135063/btn_dshbrd_more.gif) on the [component title bar](https://devnet.logianalytics.com/hc/en-us/articles/1500009771801-General-Operations-in-JDashboard#TitleBar), focus on **Analyze** on the drop-down menu, and then choose an item:

* **In New Tab**  
   Loads the specified analysis template in Visual Analysis in a new dashboard tab.
* **In New Window**  
   Loads the specified analysis template in Visual Analysis in a new web browser window.

![Note icon](https://devnet.logianalytics.com/hc/article_attachments/4404885305111/noteicon.jpg)JDashboard saves the changes that you made on the analysis template in Visual Analysis to the server resource tree. However, the changes does not affect the analysis template that you inserted as a component in a dashboard.
