# Connecting via JSON Connections

A JSON connection contains relational data which is transformed from a JSON data source. This topic introduces the workflow for Designer to get data from JSON data sources, and how you can set up JSON connections in a catalog and work with the connections to retrieve data. It also provides an example of connecting a catalog to Google Cloud BigQuery via a JSON connection.
