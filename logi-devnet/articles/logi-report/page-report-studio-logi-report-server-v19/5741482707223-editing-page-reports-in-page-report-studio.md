# Editing Page Reports in Page Report Studio

Page Report Studio provides you with a convenient and powerful tool to analyze your business information. This topic describes how you can edit page reports in Page Report Studio.

Page Report Studio provides many features for interfacing with reports, such as filtering and sorting the report data, converting between crosstabs and charts to view and analyze data from different aspects with different focuses, and adding conditional formats to highlight wanted values. Through a user-friendly web GUI, you can easily navigate, drill, and view report contents.

However, for a page report that you created in Logi Report Designer and that uses query as the data source, if you want to do analytic actions in Page Report Studio, such as adding a view element, converting between crosstabs and charts, drilling around the data, changing chart definition, or adding and converting columns and aggregating on detail columns in tables, Page Report Studio needs to convert its fields to view elements. If the conversion fails, you will not be able to perform these actions. For more information about the conversion, see Automatic Cube Initialization in the *Logi Report Designer Guide*.

To perform most of the operations in this topic, you need to use [Interactive View](https://devnet.logianalytics.com/hc/en-us/articles/5741462429975-Page-Report-Studio-Server-v19#View) of Page Report Studio.
