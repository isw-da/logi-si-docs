# View Sample Reports

You can view the sample reports Logi JReport provides in the Public Reports folder in the Logi JReport Server resource tree, provided that you have the necessary permissions on the reports.

To view the sample reports, you should have [logged onto the Logi JReport Server user console](https://devnet.logianalytics.com/hc/en-us/articles/1500009562162-Test-Server-Installation). Then:

1. On the Logi JReport Server Start page, select **Public Reports** in the Open category.

1. Open the **SampleReports** folder.
2. Browse to a web report, for example Coffee Sales.wls, then select the report name.

   ![Run Report](https://devnet.logianalytics.com/hc/article_attachments/4404889432983/viewrpt_run.gif)

   The report will be opened in Web Report Studio.

   ![View Report in Web Report Studio](https://devnet.logianalytics.com/hc/article_attachments/4404889433239/viewrpt_wls.gif)
