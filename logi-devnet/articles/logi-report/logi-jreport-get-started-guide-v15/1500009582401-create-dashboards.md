# Create Dashboards

JDashboard is a new way of information delivery. It displays rich visualizations of your data using the vast library of charting elements such as maps, crosstabs, tables and even live motion charts. These can all be combined with a robust set of interactions. With JDashboard, you can take data exploration to the next level by creating interactive views of business performance.

To create a dashboard, you should have [logged onto the Logi JReport Server user console](https://devnet.logianalytics.com/hc/en-us/articles/1500009562162-Test-Server-Installation). Then:

1. On the Logi JReport Server Start page, select **Dashboards** in the Create category.
2. A blank dashboard is then created.

   ![Blank Dashboard](https://devnet.logianalytics.com/hc/article_attachments/4404894017431/crtdshbrd_hdr.gif)
3. Select the first button ![Resources button](https://devnet.logianalytics.com/hc/article_attachments/4404889435671/btn_dsh_rsc.gif) on the toolbar to display the Resources panel.

   ![Resources Panel](https://devnet.logianalytics.com/hc/article_attachments/4404889435927/crtdshbrd_rscpnl.gif)
4. In the Resources panel, navigate to **Reports** > **Public Reports** > **SampleReports**, expand **Coffee Sales.wls**, drag **Donut Chart**:

   ![Drag Chart to Dashboard](https://devnet.logianalytics.com/hc/article_attachments/4404889436183/crtdshbrd_drag.gif)

   And then drop it to the dashboard:

   ![Drop Chart in Dashboard](https://devnet.logianalytics.com/hc/article_attachments/4404894017687/crtdshbrd_drop.gif)
5. Close the Resources panel to view the result:

   ![Chart in Dashboard](https://devnet.logianalytics.com/hc/article_attachments/4404894017943/crtdshbrd_rst.gif)
6. Save the dashboard.
