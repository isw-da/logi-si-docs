# Test Server Installation

After Logi JReport Server is installed, you can start it and log onto the Logi JReport Server user console to run reports and dashboards and create ad hoc reports.

**To start Logi JReport Server:**

On the machine where the server is installed, select **Start** > **Logi JReport 15** > **Start Server**.

**To log onto the server user console:**

1. Open a web browser and set the URL to `http://ip_or_hostname:port`. By default, the port for accessing the Logi JReport Server user console is 8888. If your browser is running on the same machine as the server, the URL is `http://localhost:8888`.
2. On the welcome page, type your user name and password.
3. Select **Login** and the Logi JReport Server user console home page is displayed, which by default is the Start Page. Choose a topic to start your work on the server.
