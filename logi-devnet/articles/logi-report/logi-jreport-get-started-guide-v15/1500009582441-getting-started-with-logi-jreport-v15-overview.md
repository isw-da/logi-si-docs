# Getting Started with Logi JReport v15 Overview

This Get Started guide provides overview material for v15 of the Logi JReport products. Select the links below to learn more.

[**Logi JReport Product Overview**](https://devnet.logianalytics.com/hc/en-us/articles/1500009582481-Logi-JReport-Product-Overview)

Logi JReport delivers operational business intelligence to enterprise applications through powerful embedded reporting.

**[System Requirements](https://devnet.logianalytics.com/hc/en-us/articles/1500009562142-System-Requirements)**

Both Logi JReport Designer and Logi JReport Server have some basic installation requirements. Before installing them, check your system to make sure all the requirements have been met.

**[Logi JReport Licenses](https://devnet.logianalytics.com/hc/en-us/articles/1500009562082-Logi-JReport-Licenses)**

Logi JReport has several special licenses which allow you to use some specific features in Logi JReport.

[**Supported Report Databases**](https://devnet.logianalytics.com/hc/en-us/articles/1500009582421-Supported-Report-Databases)

Logi JReport supports all of the current mainstream databases as well as most databases which support JDBC drivers.

**[Test Server Installation](https://devnet.logianalytics.com/hc/en-us/articles/1500009562162-Test-Server-Installation)**

After Logi JReport Server is installed, you can start it and log onto the user console to run reports and dashboards.

[**View Sample Reports**](https://devnet.logianalytics.com/hc/en-us/articles/1500009582521-View-Sample-Reports)

On the Logi JReport Console page, you can view the sample reports Logi JReport provides in different formats.

**[Create Dashboards](https://devnet.logianalytics.com/hc/en-us/articles/1500009582401-Create-Dashboards)**

You can create dashboards using JDashboard, which displays rich visualizations of your data using the vast library of charting elements such as maps, crosstabs, tables and even live motion charts.

**[Create Ad Hoc Reports](https://devnet.logianalytics.com/hc/en-us/articles/1500009582381-Create-Ad-Hoc-Reports)**

You can create ad hoc reports based on predefined business views that are published to Logi JReport Server.

[**Start a Visual Analysis Session**](https://devnet.logianalytics.com/hc/en-us/articles/1500009582501-Start-a-Visual-Analysis-Session)

You can visualize the result of every step of business data analysis by dragging and dropping business data fields to the layout module using Visual Analysis.

[**Open Sample Report Templates**](https://devnet.logianalytics.com/hc/en-us/articles/1500009562122-Open-Sample-Report-Templates)

There are many sample reports in Logi JReport Designer, you can open any one sample report and preview the report result.

**[Start Creating New Reports](https://devnet.logianalytics.com/hc/en-us/articles/1500009562102-Start-Creating-New-Reports)**

Logi JReport supports three types of reports, Page Report, Web Report and Library Component, to serve different reporting requirements. Before you create new reports using Logi JReport Designer, you have to first create a catalog and then create connections to your data sources.
