# Start a Visual Analysis Session

Visual Analysis is a WYSIWYG product to visualize the result of every step of your work. Simply by dragging and dropping data fields to the layout module, you are able to experience the detailed building up of crosstabs and charts step by step visually.

To start a new Visual Analysis session and perform data analysis, you should have [logged onto the Logi JReport Server user console](https://devnet.logianalytics.com/hc/en-us/articles/1500009562162-Test-Server-Installation). Then:

1. On the Logi JReport Server Start Page, select **Analysis** in the Create category.
2. A Visual Analysis window with the Select Data Source dialog is displayed. Select the required data source, for example, WorldWideSalesBV in `/SampleReports`, and then select **OK**.

   ![Select Data Source](https://devnet.logianalytics.com/hc/article_attachments/4404889433495/va_bv.gif)
3. The Visual Analysis window will be loaded. The display type shows Text ![Text button](https://devnet.logianalytics.com/hc/article_attachments/4404889433751/btn_va_txt.gif) by default.

   ![Visual Analysis Window](https://devnet.logianalytics.com/hc/article_attachments/4404894014103/va_window.gif)
4. To view total sales, drag **Total Sales** from the Resources panel and drop it to the Label button ![Label button](https://devnet.logianalytics.com/hc/article_attachments/4404894014359/btn_va_lbl.gif) in the legend section.

   ![Label Legend](https://devnet.logianalytics.com/hc/article_attachments/4404894014615/va_txt1.gif)

   There is only one value in the data presentation area which is the total sales in the whole business view.
5. Now let's view total sales in different years. Drag **Sales Year** to the column control box ![Column Control box](https://devnet.logianalytics.com/hc/article_attachments/4404894014871/btn_va_drpgrpclmn.gif).

   ![Add Column](https://devnet.logianalytics.com/hc/article_attachments/4404894015127/va_txt2.gif)
6. To add product category as the row header, drag **Category** to the row control box ![Row Control box](https://devnet.logianalytics.com/hc/article_attachments/4404894015383/btn_va_drpgrprw.gif).

   ![Add Row](https://devnet.logianalytics.com/hc/article_attachments/4404894015639/va_txt3.gif)
7. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404894015895/btn_save.gif) on the toolbar to save the result as a visual analysis template.
