# Logi Report Server System Requirements Logi Report Server v19

This topic describes the basic system requirements for using Logi Report Server. Check your system to meet all the requirements before installation.

|  |  |  |
| --- | --- | --- |
| Logi Report **Server System Requirements** | | |
|  | Recommended Requirements | Minimum Requirements |
| **OS** | Windows x64, UNIX x64, Linux x64, Linux on IBM Z | Windows, UNIX, Linux, Linux on IBM Z |
| **CPU** | Quad Core processor | Dual Core processor |
| **Free Memory** | 8 GB | 4 GB |
| **Free Disk** | 10 GB | 1 GB |
| **JDK** | Oracle JDK 8, 9, 11, 12, and 17 OpenJDK 8, 11, 12, 13, 14, 15, 16, and 17 | Oracle JDK 8, OpenJDK 8 |
| **Browser** | Latest releases of Chrome, Firefox, Edge, Safari | Chrome 23, Firefox 20, Edge, Safari 10 |

You can download the official JDK version from <https://www.oracle.com/technetwork/java/javase/downloads/index.html>.

![Note icon](https://devnet.logianalytics.com/hc/article_attachments/9905612785687/noteicon.jpg)

* If you want to use Server on the Linux on IBM Z system, you must download the JDK specially for IBM from <http://www-03.ibm.com/servers/eserver/zseries/software/java/>.
* Adobe no longer supports their Flash Player, and has blocked Flash content from running in a Flash player since January 12, 2021. Select this link for more information: <https://www.adobe.com/products/flashplayer/end-of-life.html>.
