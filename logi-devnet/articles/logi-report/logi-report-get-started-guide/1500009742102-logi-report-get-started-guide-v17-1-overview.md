# Logi Report Get Started Guide v17.1 Overview

This Get Started guide provides overview material for v17.1 of the Logi Report products. Select the links to learn more.

[**Logi Report Product Overview**](https://devnet.logianalytics.com/hc/en-us/articles/1500009742162-Logi-Report-Product-Overview)

Logi Report delivers operational business intelligence to enterprise applications through powerful embedded reporting.

**[System Requirements](https://devnet.logianalytics.com/hc/en-us/articles/1500009742182-System-Requirements)**

Both Designer and Server have some basic installation requirements. Before installing them, check your system to make sure all the requirements have been met.

**[Logi Report Licenses](https://devnet.logianalytics.com/hc/en-us/articles/1500009769781-Logi-Report-Licenses)**

Logi Report has several special licenses which enable you to use some specific features in Logi Report.

[**Supported Report Databases**](https://devnet.logianalytics.com/hc/en-us/articles/1500009742062-Supported-Report-Databases)

Logi Report supports all of the current mainstream databases as well as most databases which support JDBC drivers.

**[Test Server Installation](https://devnet.logianalytics.com/hc/en-us/articles/1500009769861-Test-Server-Installation)**

After Server is installed, you can start it and log onto the Server Console to run reports and dashboards.

[**View Sample Reports**](https://devnet.logianalytics.com/hc/en-us/articles/1500009769901-View-Sample-Reports-)

In the Resources page of the Server Console, you can view the sample reports Logi Report provides in different formats.

**[Create Dashboards](https://devnet.logianalytics.com/hc/en-us/articles/1500009742022-Create-Dashboards)**

You can create dashboards using JDashboard, which displays rich visualizations of your data using the vast library of charting elements such as maps, crosstabs, tables, and even live motion charts.

**[Create Ad Hoc Reports](https://devnet.logianalytics.com/hc/en-us/articles/1500009769681-Create-Ad-Hoc-Reports)**

You can create ad hoc reports based on predefined business views that are published to Server.

[**Start a Visual Analysis Session**](https://devnet.logianalytics.com/hc/en-us/articles/1500009769881-Start-a-Visual-Analysis-Session)

You can visualize the result of every step of business data analysis by dragging and dropping business data fields to the layout module using Visual Analysis.

[**Open Sample Report Templates**](https://devnet.logianalytics.com/hc/en-us/articles/1500009769821-Open-Sample-Report-Templates)

There are many sample reports in Designer, you can open any sample report and preview the report result.

**[Start Creating New Reports](https://devnet.logianalytics.com/hc/en-us/articles/1500009742142-Start-Creating-New-Reports)**

Logi Report supports three types of reports, Page Report, Web Report, and Library Component. Before you create new reports using Designer, you need to first create a catalog and then create connections to your data sources.

[**Online Help**](https://devnet.logianalytics.com/hc/en-us/articles/1500009769801-Online-Help)

Online documentation includes all information about the Logi Report products.
