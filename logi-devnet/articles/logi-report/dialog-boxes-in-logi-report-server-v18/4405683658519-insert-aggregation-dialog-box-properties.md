# Insert Aggregation Dialog Box Properties

You can use the Insert Aggregation dialog box to [add an aggregation in a crosstab](https://devnet.logianalytics.com/hc/en-us/articles/4405683959319-Manipulating-Data-Components-in-Page-Report#CTAgg) based on the selected field. This topic describes the properties in the dialog box.

![Insert Aggregation dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420453634839/insrtaggrgt.gif)

**Aggregate Function**

Select the function for the aggregation.

If the selected field has a predefined aggregate function, the property name is **Aggregation**. Server displays the name of the field in the text box, and you cannot edit it.

**Distinct On**

You should set this property when the aggregate function is **DistinctSum**. Specify the detail objects according to whose unique values to calculate DistinctSum. Select the ellipsis button ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4420461473943/btn_chsr.gif) to select the detail objects in the [Select Fields dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683680663-Select-Fields-Dialog-Box-Properties).

**Label**

Specify the label of the new aggregation in the crosstab. You can leave it blank.

**Auto Map Field Name**

Select if you want to automatically map the label text to the dynamic display name of the field used for the aggregation.

**OK**

Select to insert the aggregation in the crosstab.

**Cancel**

Select to close the dialog box without inserting an aggregation in the crosstab.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)

Select to close the dialog box without inserting an aggregation in the crosstab.
