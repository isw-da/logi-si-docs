# Insert Column Dialog Box Properties

This topic describes how you can use the Insert Column dialog box to [insert the selected group field as a column header](https://devnet.logianalytics.com/hc/en-us/articles/4405684043031-Manipulating-Data-Components-in-Web-Report#CTColumn) in a crosstab.

![Insert Column dialog](https://devnet.logianalytics.com/hc/article_attachments/4420447069463/insrtclmn.gif)

**Column**

Name of the field in the text box. It is read only.

**Label**

Type the label text of the new column header in the crosstab. You can leave it blank.

**Auto Map Field Name**

Specifies whether to automatically map the label text to the [dynamic display name](https://devnet.logianalytics.com/hc/en-us/articles/4405683929879-Managing-Dynamic-Display-Names-of-Business-View-Elements) of the field used for the new column header.

**OK**

Inserts the field and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)

Ignores the setting and closes this dialog box.
