# Select Another Catalog Dialog Box Properties

Use the Select Another Catalog dialog box to specify a catalog for a report, library component, or folder. This topic describes how you can select a catalog.

![Select Another Catalog dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420453595799/slctctlg.gif)

**Select a Folder**

Specify the folder in the server resource tree that contains the required catalog. You can select the ellipsis button ![](https://devnet.logianalytics.com/hc/article_attachments/4420453393175/btn_chsr2.gif) to open the [Select Folder dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405690565015-Select-Folder-Dialog-Box-Properties) to select the folder.

**Select a Catalog**

Server lists the catalogs in the specified folder. Select the one you want.

**OK**

Select to apply the catalog you selected and close the dialog box.

**Cancel**

Select to close the dialog box without selecting a catalog.

**Help**

Select to view information about the dialog box.
