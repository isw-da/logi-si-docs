# Edit Gradient Color Dialog Box Properties

This topic describes how you can use the Edit Gradient Color dialog box to specify gradient colors for the members of the aggregation field.

![Edit Gradient Color dialog](https://devnet.logianalytics.com/hc/article_attachments/4420447262231/edtgrdclr.gif)

**Pattern**

Select a pattern from the drop-down list. Server displays a preview of the gradient color of the pattern under the pattern name.

You can customize the gradient start color and end color. Select the leftmost or the rightmost triangle, then select a color in the color palette or select **More Colors** to access the [Color Picker dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683778071-Color-Picker-Dialog-Box-Properties) in which you can specify a color within a wider range. After you change a color, the pattern name will change to "Customize".

**Reversed**

Select to reverse the direction of the gradient color.

**OK**

Select to apply any changes you made here and exit the dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420453594007/btn_va_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420453394455/btn_delete.gif)**Close button**

Select to close the dialog box without saving any changes.
