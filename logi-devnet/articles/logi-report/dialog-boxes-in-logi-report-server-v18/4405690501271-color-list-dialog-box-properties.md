# Color List Dialog Box Properties

You can use the Color List dialog box to modify the color pattern of pointers in a gauge chart. This topic describes the properties in the dialog box.

![Color List dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420453661079/colorlist.gif)

**Colors**

Select an item, and then specify its fill color and transparency. Pointers in a group follow the color settings of the items.

* ![Create Condition](https://devnet.logianalytics.com/hc/article_attachments/4420461718807/btn_add3.gif)**Add button**  
   Select to add a new item.
* ![Delete button](https://devnet.logianalytics.com/hc/article_attachments/4420447344535/btn_delete1.gif)**Delete button**  
   Select to remove the selected item.
* ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4420453661335/btn_mvup2.gif)**Move Up button**  
  Select to move the selected item higher in the list.
* ![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4420447345175/btn_mvdown2.gif)**Move Down button**  
   Select to move the selected item lower in the list.
* **Fill**  
   Specify the color for the selected item.

  To change the color, select the color indicator, and then select a color from the color palette. You can select **More Colors** in the color palette to access the [Color Picker dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683616151-Color-Picker-Dialog-Box-Properties) in which you can select a color within a wider range, or select **More Fill Effects** to access the [Fill Effects dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683628823-Fill-Effects-Dialog-Box-Properties) to specify gradient fill effect. You can also type a hexadecimal RGB value directly in the text box to specify a color, for example, #9933ff.
* **Transparency**  
   Specify the transparency of the color.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
