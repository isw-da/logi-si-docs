# New Report Tab Dialog Box Properties

You can use the New Report Tab dialog box to [append a new report tab](https://devnet.logianalytics.com/hc/en-us/articles/4405683950743-Creating-Page-Reports) to the current report. This topic describes the properties in the dialog box.

![New Report Tab dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420453633047/newrpttab.gif)

**Report Title**

Specify the title of the report tab.

**Choose Report Layout**

Select the report layout on which you want to create your report tab.

* **Banded**  
   Select to create a report tab containing a banded object.
* **Table (Group Above)**  
   Select to create a report tab containing a table with group information above the detail row.
* **Table (Group Left)**  
   Select to create a report tab containing a table with group information left to the detail row.
* **Table (Group Left Above)**  
   Select to create a report tab containing a table with group information left above the detail row.
* **Summary Table**  
   Select to create a report tab containing a table with only group and summary information.
* **Chart**  
   Select to create a report tab containing a chart.
* **Crosstab**  
   Select to create a report tab containing a crosstab.
* **Blank**  
   Select to create a blank report tab.

**OK**

Select to open the corresponding report wizard or a new page report with a blank report tab if you select **Blank**.

* [Banded Wizard](https://devnet.logianalytics.com/hc/en-us/articles/4405683611159-Banded-Wizard-Properties)
* [Table Wizard](https://devnet.logianalytics.com/hc/en-us/articles/4405690529047-Table-Wizard-Properties)
* [Chart Wizard](https://devnet.logianalytics.com/hc/en-us/articles/4405690500375-Chart-Wizard-Properties)
* [Crosstab Wizard](https://devnet.logianalytics.com/hc/en-us/articles/4405683618199-Crosstab-Wizard-Properties)

**Cancel**

Select to close the dialog box without creating a page report tab.

**Help**

Select to view information about the dialog box.
