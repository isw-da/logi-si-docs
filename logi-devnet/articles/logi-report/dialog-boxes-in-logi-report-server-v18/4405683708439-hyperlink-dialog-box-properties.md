# Hyperlink Dialog Box Properties

Use the Hyperlink dialog box to [add a hyperlink](https://devnet.logianalytics.com/hc/en-us/articles/4405683594647-Creating-and-Saving-Dashboards#HTML) on the selected object. This topic describes how to add URL.

Server displays the dialog box when you select the Hyperlink button ![Hyperlink button](https://devnet.logianalytics.com/hc/article_attachments/4420461686039/btn_dshbrd_lnk.gif) in the [Edit HTML dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405690532759-Edit-HTML-Dialog-Box-Properties) or [Insert HTML dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683711511-Insert-HTML-Dialog-Box-Properties).

![Hyperlink dialog](https://devnet.logianalytics.com/hc/article_attachments/4420447286807/hyplnk.gif)

**URL**

Type the URL you want to direct the hyperlink to.

**OK**

Select to add the hyperlink.

**Cancel**

Select to close the dialog box without adding a hyperlink.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)**Help button**

Select to view information about the Hyperlink dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)**Close button**

Select to close the dialog box without adding a hyperlink.
