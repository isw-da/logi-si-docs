# Color Picker Dialog Box Properties

This topic describes how you can use the Color Picker dialog box to specify a color.

This topic contains the following sections:

* [Standard Tab Properties](#standard)
* [Custom Tab Properties](#Custom)

**OK**

Select to apply the color you specified here.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420453594007/btn_va_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420453394455/btn_delete.gif)**Close button**

Select to close the dialog box without saving any changes.

## Standard Tab Properties

Select a color from the color palette.

![Color Picker dialog - Standard button](https://devnet.logianalytics.com/hc/article_attachments/4420447262487/clrpkr_stdrd.gif)

**color palette**

Server lists the colors for you to select one.

**R**

Specify the amount of red in a color. The value is from 0 to 255.

**G**

Specify the amount of green in a color. The value is from 0 to 255.

**B**

Specify the amount of blue in a color. The value is from 0 to 255.

**#**

Specify the hexadecimal value of the color.

**New**

This part displays the color you select.

**Current**

This part always displays the original color.

## Custom Tab Properties

Customize the color within a wider range.

![Color Picker dialog - Custom button](https://devnet.logianalytics.com/hc/article_attachments/4420447262743/clrpkr_custm.gif)

**Color matrix**

Select a color you want in the color matrix. The hue of the matrix is based on the color you define on the color bar.

**Color bar**

Specify a color on the bar which will be the hue of the color matrix.

**R**

Specify the amount of red in a color. The value is from 0 to 255.

**G**

Specify the amount of green in a color. The value is from 0 to 255.

**B**

Specify the amount of blue in a color. The value is from 0 to 255.

**#**

Specify the hexadecimal value of the color.

**New**

This part displays the color you select.

**Current**

This part always displays the original color.
