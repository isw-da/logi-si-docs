# Fetch Data Dialog Box Properties

Use the Fetch Data dialog box to get data from the database for use in a [filter control](https://devnet.logianalytics.com/hc/en-us/articles/4405690494487-Filtering-the-Data-Components-in-a-Dashboard). This topic describes how to select data values.

Server displays the dialog box when you select **Fetch Data** in the [Insert Filter Control dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683710487-Insert-Filter-Control-Dialog-Box-Properties) or [Edit Filter Control dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683701143-Edit-Filter-Control-Dialog-Box-Properties).

![Fetch Data dialog](https://devnet.logianalytics.com/hc/article_attachments/4420447287703/ftchdt.gif)

**Search box**

You can search for values using the Search box. Type the keyword, and Server lists the values that contain the keyword.

**Values**

Select the values you want. Use the top checkbox to select or clear all the listed values at a time.

You can select the Values column header to sort the values in the ascending or descending order.

**OK**

Select to add the selected values to the value list in the Insert/Edit Filter Control dialog box.

**Cancel**

Select to close the dialog box without adding any values.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)**Help button**

Select to view information about the Fetch Data box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)**Close button**

Select to close the dialog box without adding any values.
