# Color Picker Dialog Box Properties

You can use the Color Picker dialog box to specify a color for an object. This topic describes the properties in the dialog box.

This topic contains the following sections:

* [Standard Tab Properties](#standard)
* [Custom Tab Properties](#Custom)

You see these elements on both tabs:

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.

## Standard Tab Properties

Select a color from the web safe color swatches.

![Color Picker dialog box - Standard tab](https://devnet.logianalytics.com/hc/article_attachments/4420461718295/colorpkr_stdrd.gif)

**Web Safe Colors**

Select a color from the web safe colors.

**R**

Specify the amount of red in a color. The value is from 0 to 255.

**G**

Specify the amount of green in a color. The value is from 0 to 255.

**B**

Specify the amount of blue in a color. The value is from 0 to 255.

**#**

Specify the hexadecimal value of the color.

**New**

Server displays the color you selected here.

**Current**

Server displays the original color.

## Custom Tab Properties

You can customize the color within a wider range.

![Color Picker dialog box - Custom tab](https://devnet.logianalytics.com/hc/article_attachments/4420461718551/colorpkr_custm.gif)

**Color matrix**

Select a color you want in the color matrix. The hue of the matrix is based on the color you define on the color bar.

**Color bar**

Select a color on the bar which is the hue of the color matrix.

**R**

Specify the amount of red in a color. The value is from 0 to 255.

**G**

Specify the amount of green in a color. The value is from 0 to 255.

**B**

Specify the amount of blue in a color. The value is from 0 to 255.

**#**

Specify the hexadecimal value of the color.

**New**

Server displays the color you selected in the color matrix.

**Current**

Server displays the original color.
