# Bottom N Dialog Box Properties

You can use the Bottom N dialog box to filter data to display the [Bottom N values](https://devnet.logianalytics.com/hc/en-us/articles/4405683956247-Applying-Filters-to-Page-Report#Component). This topic describes the properties in the dialog box.

![Bottom N dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420461721623/bottomn.gif)

**Value**

Type a positive integer number N if you want to display the bottom N values.

**OK**

Select to display the bottom N values.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
