# Unique Key Dialog Box Properties

Use the Unique Key dialog box to specify a unique key to ensure there are no duplicated records in the [real time chart](https://devnet.logianalytics.com/hc/en-us/articles/4405690496279-Manipulating-Data-Components#RTChart) or [library component](https://devnet.logianalytics.com/hc/en-us/articles/4405690495383-General-Operations-in-JDashboard#Refresh). This topic describes how to select the unique key.

Server displays the dialog box when you select **Incremental Fetch** in the [Real-time Settings dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405690539287-Real-time-Settings-Dialog-Box-Properties) or select the ellipsis button ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4420447035159/btn_chsr1.gif) beside the **Unique Key** text box in the [Auto Refresh dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683697687-Auto-Refresh-Dialog-Box-Properties#AutoRefreshBV).

![Unique Key dialog](https://devnet.logianalytics.com/hc/article_attachments/4420461675927/unqkey.gif)

**Resources**

Server lists the elements in the business view that the real time chart or library component uses.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4420453603991/btn_dshbrd_add2.gif)**Add button**

Select to add the selected fields to the Unique Key box.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4420447271959/btn_dshbrd_rmv2.gif)**Remove button**

Select to remove the selected fields from the Unique Key box if you do not want them.

**Unique Key**

Server lists the fields you have added as the unique key, which is to guarantee there are no two records with the same unique key value in the real time chart or library component.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
