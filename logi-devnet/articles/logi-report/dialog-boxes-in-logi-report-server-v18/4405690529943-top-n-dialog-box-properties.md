# Top N Dialog Box Properties

You can use the Top N dialog box to filter data to display the [Top N values](https://devnet.logianalytics.com/hc/en-us/articles/4405683956247-Applying-Filters-to-Page-Report#Component). This topic describes the properties in the dialog box.

![Top N dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420461692951/topn.gif)

**Value**

Type a positive integer N if you want to display the top N values.

**OK**

Select to display the top N values.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
