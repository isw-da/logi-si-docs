# New Dynamic Security Dialog Box Properties

This topic describes how you can use the New Dynamic Security dialog box to create [dynamic catalog security](https://devnet.logianalytics.com/hc/en-us/articles/4405684015511-Dynamic-Security).

Server displays the dialog box when an administrator selects **New Dynamic Security** in the Administration > Security > Dynamic Security page on the Server Console.

![New Dynamic Security dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420447265559/nwdynscrty.gif)

**Catalog**

Specify the catalog you want to create dynamic security for.

Type the catalog with the full resource path in the text box, for example, `/SampleReports/SampleReports.cat`, or select **Browse** to select a catalog in the [Select Catalog dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683769367-Select-Catalog-Dialog-Box-Properties).

**Security File**

Specify the security file that you generated in Logi Report Designer, to create dynamic security for the catalog. Select **Browse** to select a catalog security file, which defines business view security, record level security, and column level security in the catalog.

**OK**

Select to create the dynamic security and exit the dialog box.

**Cancel**

Select to close the dialog box without creating the dynamic security.

**Help**

Select to view information about the dialog box.
