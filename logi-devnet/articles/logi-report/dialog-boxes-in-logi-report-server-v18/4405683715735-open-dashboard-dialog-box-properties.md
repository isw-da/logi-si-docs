# Open Dashboard Dialog Box Properties

Use the Open Dashboard dialog box to open another dashboard in JDashboard. This topic describes how to select a dashboard.

Server displays the dialog box when you select the Open button ![Open button](https://devnet.logianalytics.com/hc/article_attachments/4420461454615/btn_open.gif) on the toolbar, or select the Options button ![Options button](https://devnet.logianalytics.com/hc/article_attachments/4420453604247/btn_dshbrd_optn.gif) and select **Open**.

![Open Dashboard dialog](https://devnet.logianalytics.com/hc/article_attachments/4420453607575/opendsh.gif)

**Open In**

Select the folder in the server resource tree where the dashboard you want to open is, and then select the dashboard. Use the arrow button ![Up Level button](https://devnet.logianalytics.com/hc/article_attachments/4420453485079/btn_dshbrd_up.gif) to go to the parent folder.

The resource table shows the resources in the current folder.

**File Name**

The file name of the selected dashboard.

**OK**

Select to open the selected dashboard.

**Cancel**

Select to close the dialog box without opening another dashboard.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)**Close button**

Select to close the dialog box without opening another dashboard.
