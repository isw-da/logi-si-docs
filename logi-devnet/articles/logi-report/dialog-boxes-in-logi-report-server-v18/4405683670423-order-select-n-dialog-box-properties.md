# Order/Select N Dialog Box Properties

You can use the Order/Select N dialog box to set the [sort order and Select N condition](https://devnet.logianalytics.com/hc/en-us/articles/4405683950743-Creating-Page-Reports#Chart) in a chart. This topic describes the properties in the dialog box.

![Order/Select N dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420447168535/ordrslctn.gif)

**Order**

Specify the order in which you want to display data on the category/series axis of the chart.

* **Ascend**  
   Select to display data in ascending order.
* **Descend**  
   Select to display data in descending order.
* **No Sort**  
   Select to keep data in the original order.

**Select N**

Define the number of category/series values you want to display.

* **Select N**
  + **All**  
     Select to display all category/series values in the chart.
  + **Top**  
     Select and then specify a number N in the text box on the right if you want to display the first N category/series values in the chart.
  + **Bottom**  
     Select and then specify a number N in the text box on the right if you want to display the last N category/series values in the chart.
* **Based On**  
   Select if you want to sort the category/series values by the values of a summary, and then select the summary and the order. Otherwise, Server sorts the category/series values by the order in the Order box.
* **Other**  
   Server enables this property when you select the Select N property of **Top** or **Bottom**. Select and then type a character string in the text box to group all the category/series values beyond the top/bottom N category/series values.

**OK**

Select to apply any changes you made here and exit the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
