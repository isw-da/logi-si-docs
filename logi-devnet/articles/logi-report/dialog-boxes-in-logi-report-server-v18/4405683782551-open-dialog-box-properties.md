# Open Dialog Box Properties

This topic describes how you can use the Open dialog box to open another analysis template in a new Visual Analysis window.

![Open dialog](https://devnet.logianalytics.com/hc/article_attachments/4420453594519/open.gif)

**Open In**

Browse to the directory in the server resource tree where the target analysis template is, and then select it. Use the button ![Up Level button](https://devnet.logianalytics.com/hc/article_attachments/4420453485079/btn_dshbrd_up.gif) to go to the parent folder.

The resource table shows the analysis templates in the current folder.

**File Name**

Server shows the selected analysis template file name.

**OK**

Select to open the selected analysis template.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420453594007/btn_va_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420453394455/btn_delete.gif) **Close button**

Select to close the dialog box without opening another analysis template.
