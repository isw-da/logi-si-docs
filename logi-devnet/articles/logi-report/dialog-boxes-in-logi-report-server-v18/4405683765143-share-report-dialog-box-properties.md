# Share Report Dialog Box Properties

This topic describes how you can use the Share Report dialog box to [share a web report](https://devnet.logianalytics.com/hc/en-us/articles/4405683936279-Sharing-Web-Reports#Share).

![Share Report dialog](https://devnet.logianalytics.com/hc/article_attachments/4420453520663/shrpt.gif)

**Select Folder**

Specify a folder in the server resource tree where you want to save the shared report.

**File Name**

Specify the file name of the shared report, with the suffix .wls.

**Description**

Provide a brief description about the shared report.

**Allow Edit**

Select to allow other users to edit the shared report.

**OK**

Select to save the shared report in the specified folder.

**Cancel**

Select to close the dialog box without sharing the report.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420461456151/btn_dltobj.gif)

Select to close the dialog box without sharing the report.
