# Select Catalog Dialog Box Properties

This topic describes how you can use the Select Catalog dialog box to select the catalog for [Quick Start With a Table Report](https://devnet.logianalytics.com/hc/en-us/articles/4405690690711-Quick-Start-with-a-Table-Report).

Server displays the dialog box when you do any of the following:

* In the Server Start Page, select **Web Report** in the **Create** category.
* Select New > Web Report on the Server Console > Resources page but the currently opened folder contains no catalog.
* Select the Change Catalog button in the [Select Data Source](https://devnet.logianalytics.com/hc/en-us/articles/4405683878679-Select-Data-Source-Dialog-Box-Properties#Report) dialog box.

![Select Catalog dialog](https://devnet.logianalytics.com/hc/article_attachments/4420447208087/slctctlg.gif)

In the dialog box, all the available catalogs that have been published to Logi Report Server are listed in a tree structure. You can select the catalog you want to use for the report from the tree.

**OK**

Specifies to use the selected catalog for creating the report and directs to the [Select Data Source](https://devnet.logianalytics.com/hc/en-us/articles/4405683878679-Select-Data-Source-Dialog-Box-Properties#Report) dialog box.

**Cancel**

Cancels the selection and closes this dialog box. Available only when the dialog box is opened by selecting Change Catalog in the Select Data Source dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)

Cancels the selection and closes this dialog box. Available only when the dialog box is opened by selecting Change Catalog in the Select Data Source dialog box.
