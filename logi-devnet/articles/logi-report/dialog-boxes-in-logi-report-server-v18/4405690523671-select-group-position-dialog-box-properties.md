# Select Group Position Dialog Box Properties

You can use the Select Group Position dialog box to specify the position of the group-by field when you [convert a table detail column to a group column](https://devnet.logianalytics.com/hc/en-us/articles/4405683959319-Manipulating-Data-Components-in-Page-Report#ConvertColumn). This topic describes the properties in the dialog box.

![Select Group Position dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420447150999/slctgrppstn.gif)

**Group Above**

Select if you want to add a new group header panel to hold the group-by field and remove the detail column.

**Group Left Above**

Select if you want to convert the detail column to a group column and add a new group header panel to hold the group-by field.

**Group Left**

Select if you want to convert the detail column to a group column and place the group-by field in the column.

**OK**

Select to convert the column.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
