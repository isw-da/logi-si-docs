# Web Action - Go To Dialog Box Properties

Use the Web Action - Go To dialog box to edit the go-to condition to change one field to another in a data component when it [receives a synchronization message](https://devnet.logianalytics.com/hc/en-us/articles/4405683601175-Synchronizing-the-Data-Components-in-a-Dashboard#Receive). This topic describes how to define the go-to condition.

Server displays the dialog box when you select the web action **\*Go To** and select the Edit button ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/4420447267735/btn_dshbrd_edt1.gif) in the **Actions** column of the [Receive Sync dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683719063-Receive-Sync-Dialog-Box-Properties).

![Web Action - Go To dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420461674775/wbactn_goto.gif)

**Go To On**

Select a field on which you want to perform the go-to action in the data component.

**To Column**

Select a field to change the Go To On field to.

**By Value**

Select **By Value** if you want to perform the go-to action by filtering a value of the Go To On field and then select the value. Server disables it if you select a crosstab aggregate field as the Go To On field.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
