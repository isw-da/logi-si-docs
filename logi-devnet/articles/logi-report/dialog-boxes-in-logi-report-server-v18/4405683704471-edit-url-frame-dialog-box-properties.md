# Edit URL Frame Dialog Box Properties

You can use the Edit URL Frame dialog box to [edit the URL](https://devnet.logianalytics.com/hc/en-us/articles/4405683594647-Creating-and-Saving-Dashboards#URL). This topic describes the properties in the dialog box.

Server displays the dialog box when you select the Edit button ![Options button](https://devnet.logianalytics.com/hc/article_attachments/4420453392151/btn_dshbrd_more.gif) on the title bar of a web page frame and then select **Edit Setting**.

![Edit URL Frame dialog](https://devnet.logianalytics.com/hc/article_attachments/4420461690775/edturl.gif)

**Title**

Type the title of the frame that holds the web page.

**URL**

Type the URL of the web page you would like to show in the dashboard.

Select **Add Parameter** if you want to insert a parameter into the URL to compose a dynamic URL. Server displays the [Select Data Source dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405690540183-Select-Data-Source-Dialog-Box-Properties) for you to select the data source that contains the required parameter when you haven't specified any data source, or the [Select Parameter dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683721495-Select-Parameter-Dialog-Box-Properties) when you have specified a data source.

**Auto refresh**

Select if you want to refresh the specified web page periodically. Then, type the time interval between two refreshes in the **Interval** text box.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
