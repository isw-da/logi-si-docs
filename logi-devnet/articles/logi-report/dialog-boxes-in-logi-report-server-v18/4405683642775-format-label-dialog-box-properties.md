# Format Label Dialog Box Properties

You can use the Format Label dialog box to format labels in a chart. This topic describes the properties in the dialog box.

This topic contains the following sections:

* [General Tab Properties](#General)
* [Border Tab Properties](#Border)
* [Font Tab Properties](#Font)

You see these elements on all the tabs:

**OK**

Select to apply any changes you made here and exit the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.

## General Tab Properties

Specify the general properties of the label.

![Format Label dialog box - General tab](https://devnet.logianalytics.com/hc/article_attachments/4420461707159/fmtlbl_gnrl.gif)

**Name**

Specify the display name of the label.

**Show NLS Value**

Select to show the translated name for the display name of the object in the **Name** text box if you have enabled the NLS feature and translated it, and when you have not modified the display name of the object.

**Text**

Specify the label text.

**X**

Specify the X coordinate of the label.

**Y**

Specify the Y coordinate of the label.

**Width**

Specify the width of the label.

**Height**

Specify the height of the label.

**Alignment**

Select the alignment mode of the label.

**Fill Type**

Select the type for filling the label.

**Color**

Specify the background color of the label.

To change the color, select the color indicator to access the [Select Color dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683679639-Select-Color-Dialog-Box-Properties), and then specify a new color. You can also type a hexadecimal RGB value to specify a color, for example, #9933ff. If you want to make the background transparent, type **Transparent** in the text box.

**Transparency**

Specify the transparency of the label background color.

## Border Tab Properties

Specify the border properties of the label.

![Format Label dialog box - Border tab](https://devnet.logianalytics.com/hc/article_attachments/4420461707543/fmtlbl_border.gif)

**Line Style**

Select the line style of the border.

**Border Type**

Select the type of the border.

**Color**

Specify the color of the border.

**Transparency**

Specify the transparency for the color of the border.

**Thickness**

Specify the thickness of the border, in inches.

## Font Tab Properties

Specify the font properties of the label.

![Format Label dialog box - Font tab](https://devnet.logianalytics.com/hc/article_attachments/4420447318423/fmtlbl_font.gif)

**Font**

Select the font face of the label text.

**Size**

Specify the font size of the label text.

**Fill Type**

Select the fill type of the label text.

**Color**

Specify the color of the label text. It takes effect when **Fill Type** on this tab is **color**.

**Transparency**

Specify the color transparency of the label text.

**Font Style**

Select the font style of the label text.

**Font Effect**

Select the font effect of the label text.

**Font Underline**

Select the style of the line under the label text.

**Font Strikeout**

Specify whether to attach a strikeout line to the label text.

**Font Rotation**

Specify the rotation angle of the label around its center, in degrees.

**Font Shearing**

Specify the shearing transformation of the label around its center, in degrees.

**Word Wrap**

Select **true** to enable word wrap of the label text.
