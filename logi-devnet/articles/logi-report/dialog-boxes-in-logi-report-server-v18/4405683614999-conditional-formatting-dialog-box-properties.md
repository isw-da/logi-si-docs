# Conditional Formatting Dialog Box Properties

You can use the Conditional Formatting dialog box to [add conditional formats to the values](https://devnet.logianalytics.com/hc/en-us/articles/4405683957271-Adding-Conditional-Formats-to-Fields-in-Page-Report) of a field. This topic describes the properties in the dialog box.

![Conditional Formatting dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420447157783/cndtnlfmt.gif)

**Condition**

Server displays all the conditions on the field.

* ![Create Condition](https://devnet.logianalytics.com/hc/article_attachments/4420461718807/btn_add3.gif)**Add button**  
  Select to open the [Edit Conditions dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683623447-Edit-Conditions-Dialog-Box-Properties) to create a new condition.
* ![Edit Condition](https://devnet.logianalytics.com/hc/article_attachments/4420461515287/btn_pgrpt_edit.gif)**Edit button**  
  Select to open the [Edit Conditions dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683623447-Edit-Conditions-Dialog-Box-Properties) to edit the selected condition.
* ![Delete button](https://devnet.logianalytics.com/hc/article_attachments/4420447344535/btn_delete1.gif)**Remove button**  
   Select to remove the selected condition.
* **Priority**  
   Specify the priority of the conditions. The higher a condition is in the list, the higher its priority is.
  + ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4420447094679/btn_mvup1.gif)**Move Up button**  
     Select to move a condition higher.
  + **![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4420453419415/btn_mvdown1.gif)****Move Down button**  
     Select to move a condition lower.

**Format**

Specify the format for each condition. Server applies the format to the field values when the specified condition is fulfilled.

* **Font**  
   Select the font face for the field values.
* **Border**  
   Select the border line style for the field values.
* **Size**  
   Specify the font size for the field values.
* **Bold**  
   Select to make the field values bold.
* **Italic**  
   Select to make the field values italic.
* **Underline**  
   Select to underline the field values.
* **Foreground Color**   
   Specify the foreground color for the field values.
* **Background Color**   
   Specify the background color for the field values.
* **Sample Text**  
   Server displays a preview sample of your settings.
* **Blinking Text**  
   Select to make the field values blink.
  Then, in the **Duration** text box, you can specify the amount of time it takes the field values to change from the foreground color to the transparent color, in seconds.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
