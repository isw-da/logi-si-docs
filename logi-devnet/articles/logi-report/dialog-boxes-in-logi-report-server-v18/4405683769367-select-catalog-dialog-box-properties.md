# Select Catalog Dialog Box Properties

This topic describes how you can use the Select Catalog dialog box to select a catalog in the server resource tree.

![Select Catalog dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420447263895/slctcat.gif)

**Resource tree box**

Server lists the folder and catalog resources in the server resource tree. Select the catalog you want.

**OK**

Select to apply the selected catalog and exit the dialog box.

**Cancel**

Select to close the dialog box without selecting a catalog.

**Help**

Select to view information about the dialog box.
