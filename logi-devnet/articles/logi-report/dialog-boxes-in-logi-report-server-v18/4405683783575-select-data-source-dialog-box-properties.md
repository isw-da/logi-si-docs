# Select Data Source Dialog Box Properties

This topic describes how you can use the Select Data Source dialog box to select a business view for performing visual analysis based on.

![Select Data Source dialog](https://devnet.logianalytics.com/hc/article_attachments/4420461665943/slctdtsrc.gif)

Server lists the available business views for your selection. They come from the current catalog in the current folder in the server resource tree. If the current catalog contains no business view or if it is the server root folder, Server will list all available business views.

![Business Views Tree Structure](https://devnet.logianalytics.com/hc/article_attachments/4420461666199/btn_va_slctdtsrc_trvw.gif)**Tree button**

Select to display the available business views in a tree structure.

![Business Views List](https://devnet.logianalytics.com/hc/article_attachments/4420461666455/btn_va_slctdtsrc_fltvw.gif)**List button**

Select to display the available business views as a list.

**OK**

Select to open Visual Analysis.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420453594007/btn_va_help.gif) **Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420453394455/btn_delete.gif) **Close button**

Select to close the dialog box without selecting a business view.
