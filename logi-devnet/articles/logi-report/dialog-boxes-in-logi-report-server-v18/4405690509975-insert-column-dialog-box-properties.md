# Insert Column Dialog Box Properties

You can use the Insert Column dialog box to [insert the selected group field as a column header](https://devnet.logianalytics.com/hc/en-us/articles/4405683959319-Manipulating-Data-Components-in-Page-Report#CTColumn) in a crosstab. This topic describes the properties in the dialog box.

![Insert Column dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420447149847/insrtclmn.gif)

**Column**

Server displays the name of the field. It is read only.

**Label**

Specify the display name of the field, which will label the field in the crosstab. You can leave it blank.

**OK**

Select to insert the field in the crosstab.

**Cancel**

Select to close the dialog box without inserting a field in the crosstab.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)

Select to close the dialog box without inserting a field in the crosstab.
