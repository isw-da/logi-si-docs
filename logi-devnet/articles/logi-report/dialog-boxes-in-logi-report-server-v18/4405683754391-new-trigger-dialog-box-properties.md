# New Trigger Dialog Box Properties

This topic describes how you can use the New Trigger dialog box to create a [trigger](https://devnet.logianalytics.com/hc/en-us/articles/4405690644375-Managing-Triggers).

Server displays the dialog box when an administrator selects **New Trigger** in the Administration > Other > Triggers page, or in the Conditions > Trigger tab when scheduling a task.

![New Trigger dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420461586327/nwtrigr.gif)

**Trigger Name**

Type the name for the new trigger.

**Conflict Check**

Select to check whether another trigger uses the name.

**Description**

Type the description about the trigger.

**OK**

Select to create the trigger.

**Cancel**

Select to close the dialog box without creating a trigger.
