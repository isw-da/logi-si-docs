# Color List Dialog Box Properties

This topic describes how you can use the Color List dialog box to modify the color patterns of data markers in the same data series respectively in a gauge chart or modify alternating color for a table.

Server displays the dialog box when you do any of the following:

* Select **Color List** in the Pointer tab of the [Format Dial Gauge dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683829783-Format-Dial-Gauge-Dialog-Box-Properties#Pointer), [Format Bar Gauge dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683824279-Format-Bar-Gauge-Dialog-Box-Properties#Pointer), or [Format Solid Gauge dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683841303-Format-Solid-Gauge-Dialog-Box-Properties#Pointer), or in the Circular Graph tab of the [Format Solid Gauge dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683841303-Format-Solid-Gauge-Dialog-Box-Properties#Circular) or [Format Activity Gauge dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683821207-Format-Activity-Gauge-Dialog-Box-Properties#Circular).
* Select the ellipsis button ![Chooser button](https://devnet.logianalytics.com/hc/article_attachments/4420447035159/btn_chsr1.gif) in the value cell of the Pattern List property for a table in the Inspector panel.

![Color List dialog](https://devnet.logianalytics.com/hc/article_attachments/4420447256599/colorlist.gif)

**Colors**

Specify the color and transparency for the chart data markers or for the alternating color of the table. Select the items in the box one by one to set the color pattern that each of the item stands for.

* ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4420447073687/btn_add1.gif)**Add button**  
   Select to add a new pattern item.
* ![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4420453394455/btn_delete.gif)**Remove button**  
   Select to add remove the selected pattern item.
* ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4420461479831/btn_mvup.gif)**Move Up button**  
   Select to add move the selected pattern item higher in the list.
* ![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4420453416727/btn_mvdown.gif)**Move Down button**  
   Select to add move the selected pattern item lower in the list.
* **Fill**  
   Specify the color schema for the pattern.

  To change the color, select the color indicator. Server displays the color palette. Select a color, or select **More Colors** to access the [Color Picker dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683802391-Color-Picker-Dialog-Box-Properties) in which you can specify a color within a wider range, or select **More Fill Effects** to access the [Fill Effects dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405683816983-Fill-Effects-Dialog-Box-Properties) to specify gradient fill effect. You can also type a hexadecimal RGB value to specify a color, for example, #9933ff.
* **Transparency**  
   Specify the transparency of the color.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
