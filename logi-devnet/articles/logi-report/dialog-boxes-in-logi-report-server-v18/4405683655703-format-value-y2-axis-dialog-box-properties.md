# Format Value (Y2) Axis Dialog Box Properties

You can use the Format Value (Y2) Axis dialog box to format the value (Y2) axis of a chart (unavailable to pie or indicator chart). For descriptions about the properties in the dialog box, refer to [Format Value (Y) Axis Dialog Box Properties](https://devnet.logianalytics.com/hc/en-us/articles/4405683654551-Format-Value-Y-Axis-Dialog-Box-Properties).
