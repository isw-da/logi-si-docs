# Edit Role Dialog Box Properties

This topic describes how you can use the Edit Role dialog box to edit a [role](https://devnet.logianalytics.com/hc/en-us/articles/4405684024983-Managing-Roles).

Server displays the dialog box when an administrator selects a role name in the Name column of the Administration > Security > Role page on the Server Console.

![Edit Role dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420453598231/edtrole.gif)

**Selected Realm**

Realm in which the role is.

**Role Name**

Name of the role.

**Privileges**

Select the privileges you want the role to have:

* **Publish**  
   The privilege of publishing resources to Logi Report Server.
* **Advanced Properties**  
   The privilege of viewing advanced version properties information, such as catalog connections and report related resources.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.
