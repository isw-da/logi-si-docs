# Suppress Column Subtotals Dialog Box Properties

This topic describes how you can use the Suppress Column Subtotals dialog box to customize the subtotals of the column fields in a crosstab that you want to suppress and display.

Server displays the dialog box when you select the ellipsis button ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4420453521175/btn_chsr4.gif) next to the **Suppress Column Subtotals** option in the **Layout** tab of the [Crosstab Wizard](https://devnet.logianalytics.com/hc/en-us/articles/4405690572823-Crosstab-Wizard-Properties#Layout) or [Insert Crosstab](https://devnet.logianalytics.com/hc/en-us/articles/4405683854103-Insert-Crosstab-Dialog-Box-Properties#Layout)  dialog box, or in the **Layout** tab for crosstab in the [Web Report Wizard](https://devnet.logianalytics.com/hc/en-us/articles/4405690615447-Web-Report-Wizard-Properties#Crosstab).

![Suppress Column Subtotal dialog](https://devnet.logianalytics.com/hc/article_attachments/4420453546775/sprsclmnsbtl.gif)

**Same for all**

If the option is selected, the setting for the subtotal of the first column field will apply to all the subtotals of the column fields in the crosstab.

**Name**

Lists the column fields excluding the one on the topmost column group level.

**Suppress**

Specifies whether the subtotal for a column field will be suppressed. To specify the value, select in the value cell and select true or false from the drop-down list.

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**![Help button](https://devnet.logianalytics.com/hc/article_attachments/4420447202071/btn_help.gif)**

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420447202455/btn_close.gif)

Does not retain any changes and closes the dialog box.
