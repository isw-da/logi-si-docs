# Printing and Exporting Reports

As you develop page reports and web reports in Designer, you can print the report result or export the result to a file so that you can review the result at any time or share with others. This topic introduces how you can print and export page and web reports (you can neither print nor export library components).

Select the following links to view the topics:

* [Printing Report Result](https://devnet.logianalytics.com/hc/en-us/articles/1500010099421-Printing-Report-Result)
* [Exporting Report Result](https://devnet.logianalytics.com/hc/en-us/articles/1500010060902-Exporting-Report-Result)
* [Exporting Report to Accessible HTML and PDF](https://devnet.logianalytics.com/hc/en-us/articles/1500010099241-Exporting-Reports-to-Accessible-HTML-and-PDF)

![Note icon](https://devnet.logianalytics.com/hc/article_attachments/4404856814359/noteicon.jpg)

* When you export or print a report which contains parameters, Designer prompts you to [specify values for the parameters](https://devnet.logianalytics.com/hc/en-us/articles/1500010061282-Specifying-Parameter-Values).
* When you export or print a report which [applies a style group](https://devnet.logianalytics.com/hc/en-us/articles/1500010062802-Applying-Styles-to-Reports), Designer prompts you to select the style group to use.
