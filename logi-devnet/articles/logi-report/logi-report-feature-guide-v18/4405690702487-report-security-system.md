# Report Security System

Report security is based on a type of filter mechanism that limits the data shown on a report according to the user's access permissions. No matter to whom you need to provide information, Designer enables you to control access to reports and different subsets of data by means of defining report security.

Generally, Logi Report classifies report security into these groups: Business View Security, Cached Report Bursting, Record Level Security, and Column Level Security.

After you publish a catalog to Server, you can apply new report security policy to the catalog at runtime without affecting the original catalog, which is called Dynamic Security.
