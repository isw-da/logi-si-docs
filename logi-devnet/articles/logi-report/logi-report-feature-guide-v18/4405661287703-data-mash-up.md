# Data Mash-Up

Data mash-up makes it possible for multiple separated application systems in your enterprise to provide data in decision making. For example, you can connect a MySQL database, a SOAP web service data source, or a MongoDB data source, and use the data from all of the database connections in a single query or business view, which is essential for creating reports in Logi Report. You can also create joins between the different data resources, which are known as distributed joins.

Logi Report provides the Add Table/View/Query dialog box, with which you can easily mash up multiple data resources from different connections. The data resources include tables, views, synonyms, queries, imported SQL files, stored procedures, and user defined data sources.

## Data Mash-up in Queries

You can mash up data resources from different connections in a query before or after the query is created.
