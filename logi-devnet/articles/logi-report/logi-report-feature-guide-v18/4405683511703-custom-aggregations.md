# Custom Aggregations

Custom aggregations are special aggregations used for sophisticated mathematical and logic calculation purposes. You can create custom aggregations via the Formula Editor using the business view elements and built-in functions including aggregation functions, mathematic, and logic calculation-related functions.

When using a custom aggregation in a data component, you need to make sure that you have added the group objects referenced in the custom aggregation as group by fields, column/row headers, or chart axis fields in the data component.
