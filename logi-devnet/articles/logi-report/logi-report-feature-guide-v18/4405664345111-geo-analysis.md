# Geo Analysis

Logi Report provides geographic visualization and analysis by overlaying customized areas and markers onto interactive maps in Google Maps, OpenStreetMap, and OpenCycleMap. Logi Report supports three types of maps: Roadmap, Satellite, and Terrain. You can adjust conditional properties such as Shape, Size, Color, and Label to provide an understanding of the map data at a quick glance. Geo Analysis extends traditional data analysis on static maps, where zooming and panning are common, to more dynamic methods of interacting with geographic data. You can analyze data using a familiar map interface and drill down hierarchies containing countries, states, counties, and cities. Logi Report supports Geo Analysis in Page Report Studio, Web Report Studio, and JDashboard.

In the following example, we create a geographic map in a library component and use it to create a dashboard.
