# Printing and Exporting Reports

As you develop reports in Logi JReport Designer, you can print the report results or export them to a file so that you can review it at any time or share with others.

The following topics describe report result formatting:

> * [Printing Report Results](https://devnet.logianalytics.com/hc/en-us/articles/1500009589461-Printing-Report-Results)
> * [Exporting Report Results](https://devnet.logianalytics.com/hc/en-us/articles/1500009589221-Exporting-Report-Results)

**Notes:**

* Library components can neither be printed nor exported.
* When exporting or printing a report which contains parameters, you will be prompted to specify values for the parameters in the [Enter Parameter Values](https://devnet.logianalytics.com/hc/en-us/articles/1500009586681-Enter-Parameter-Values-Dialog) dialog.
