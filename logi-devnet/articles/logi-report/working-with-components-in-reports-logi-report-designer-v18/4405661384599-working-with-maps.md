# Working with Maps

A map is a special kind of image in Logi Report. This topic introduces the usages of the two types of maps Logi Report supports: Geographic Map and Shape Map.

* [Geographic map](https://devnet.logianalytics.com/hc/en-us/articles/4405661386007-Using-Geographic-Maps) is a data component integrated with Google Maps, OpenStreetMap, or OpenCycleMap.
* [Shape map](https://devnet.logianalytics.com/hc/en-us/articles/4405664397975-Using-Shape-Maps) contains a picture and certain areas that are bound with groups in a banded object.
