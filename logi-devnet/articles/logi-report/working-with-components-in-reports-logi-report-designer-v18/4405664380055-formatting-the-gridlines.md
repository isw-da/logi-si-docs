# Formatting the Gridlines

This topic describes how you can customize the gridlines for the axes of a chart (pie, indicator, heat map, and organization charts do not have gridlines).

1. Right-click any chart element and select the desired item from the Format Gridlines submenu, or double-click any gridline of the desired axis. Designer displays one of the following dialog boxes according to your selection:
   * [Format Category (X) Gridline dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661568407-Format-Category-X-Gridline-Dialog-Box)
   * [Format Value (Y) Gridline dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661602455-Format-Value-Y-Gridline-Dialog-Box)
   * [Format Value (Y2) Gridline dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664509719-Format-Value-Y2-Gridline-Dialog-Box)
   * [Format Series (Z) Gridline dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661592471-Format-Series-Z-Gridline-Dialog-Box)

   The following shows a sample dialog box.

   ![Format Category (X) Gridline dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420556179735/fmtctgrygrdln.gif)
2. In the **Line Color** text box, specify the color of the gridlines. To edit the color, select the color indicator and select a color from the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/4405661953175-Menu-Tabs#Palette) or type the hexadecimal value of a color (for example, 0xff0000) in the text box.
3. From the **Line Style** drop-down list, select the line style to apply to the gridlines.
4. In the **Transparency** text box, specify the transparency of the color.
5. In the **Thickness** text box, specify the thickness of the gridlines, in pixels.
6. Select **OK** to accept the changes and close the dialog box.

![Note icon](https://devnet.logianalytics.com/hc/article_attachments/4420542072599/noteicon.jpg) To display the gridlines for an axis, you need to select the Show Gridlines option in the format dialog box of the axis.
