# Chart Types

This topic introduces the Logi Report chart types.

|  |  |  |
| --- | --- | --- |
| **Bar** | | |
| Clustered Bar 2-D button | Clustered Bar 2-D | Clustered Bar. Displays and compares data values across categories. |
| Stacked Bar 2-D button | Stacked Bar 2-D | Stacked Bar. Displays and compares the contribution of each data value to a total across categories. |
| 100% Stacked Bar 2-D button | 100% Stacked Bar 2-D | 100% Stacked Bar. Displays and compares the percentage that each data value contributes to a total across categories. |
| Clustered Bar 3-D button | Clustered Bar 3-D | Clustered bar with a 3-D visual effect. |
| Stacked Bar 3-D button | Stacked Bar 3-D | Stacked bar with a 3-D visual effect. |
| 100% Stacked Bar 3-D button | 100% Stacked Bar 3-D | 100% stacked bar with a 3-D visual effect. |
| Bar 3-D button | Bar 3-D | 3-D Bar. Displays and compares data values across categories and series. |
| **Bench** | | |
| Clustered Bench 2-D button | Clustered Bench 2-D | Clustered Bench. Displays and compares data values across categories. |
| Stacked Bench 2-D button | Stacked Bench 2-D | Stacked Bench. Displays and compares the contribution of each data value to a total across categories. |
| 100% Stacked Bench 2-D button | 100% Stacked Bench 2-D | 100% Stacked Bench. Displays and compares the percentage that each data value contributes to a total across categories. |
| Back to Back Bench 2-D button | Back to Back Bench 2-D | Back to Back Bench. Displays and compares the contribution of each data value in two groups across categories. |
| 100% Back to Back Bench 2-D button | 100% Back to Back Bench 2-D | 100% Back to Back Bench. Displays and compares the percentage that each data value contributes in two groups across categories. |
| Clustered Bench 3-D button | Clustered Bench 3-D | Clustered bench with a 3-D visual effect. |
| Stacked Bench 3-D button | Stacked Bench 3-D | Stacked bench with a 3-D visual effect. |
| 100% Stacked Bench 3-D button | 100% Stacked Bench 3-D | 100% stack bench with a 3-D visual effect. |
| Back to Back Bench 3-D button | Back to Back Bench 3-D | Back to Back Bench with a 3-D visual effect |
| 100% Back to Back Bench 3-D button | 100% Back to Back Bench 3-D | 100% Back to Back Bench with a 3-D visual effect |
| Bench 3-D button | Bench 3-D | 3-D Bench. Displays and compares data values across categories and series. |
| **Line** | | |
| Line 2-D button | Line 2-D | Line. Displays trend over categories. Usually used over date or time categories. |
| Stacked Line 2-D button | Stacked Line 2-D | Stacked Line. Displays the trend of the contribution of each data value over categories. |
| 100% Stacked Line 2-D button | 100% Stacked Line 2-D | 100% Stacked Line. Displays the trend of the percentage each data value contributes over categories. |
| Line 3-D button | Line 3-D | Line with a 3-D visual effect. |
| **Area** | | |
| Area 2-D button | Area 2-D | Displays the trend of the values over time or categories. |
| Stacked Area 2-D button | Stacked Area 2-D | Stacked Area. Displays the trend of the contribution of each data value over categories. |
| 100% Stacked Area 2-D button | 100% Stacked Area 2-D | 100% Stacked Area. Displays the trend of the percentage each data value contributes over categories. |
| Area 3-D button | Area 3-D | Area with a 3-D visual effect. |
| Stacked Area 3-D button | Stacked Area 3-D | Stacked area with a 3-D visual effect. |
| 100% Stacked Area 3-D button | 100% Stacked Area 3-D | 100% stacked area with a 3-D visual effect. |
| **Pie** | | |
| Clustered Pie button | Clustered Pie | Pie. Displays the contribution of each data value to a total over time or categories. |
| Clustered Donut button | Clustered Donut | Donut. Displays the contribution of each data value to the Category total. The size of each piece is proportional to the sum of the items. Each Series value creates a new instance of the chart. |
| **Radar** | | |
| Radar 2-D button | Radar 2-D | Displays and compares the data values relative to a center point. |
| **Organization (Org)** | | |
| Org 2-D button | Org 2-D | Displays the structure of an organization and the relationships and relative ranks of its parts, or the similar structures. |
| **Indicator** | | |
| Indicator 2-D button | Indicator 2-D | Displays each data value with a colored indicator. |
| **Gauge** | | |
| Gauge Dial 2-D button | Gauge Dial 2-D | Usually displays the performance of each member in a group, using three colors, green, yellow, and red (by default) to represent three levels: normal, alert, and error. This type, displays each data value by a dial. |
| Gauge Dial 2-D button | Gauge Solid 2-D | Displays each data value with an arc. |
| Gauge Dial 2-D button | Gauge Activity 2-D | Displays each data value with a circle. |
| Gauge Bar 2-D button | Gauge Bar 2-D | Displays each data value with a bar. |
| Gauge Bubble 2-D button | Gauge Bubble 2-D | Displays each data value with a colored bubble. |
| **Surface** | | |
| Surface 3-D  button | Surface 3-D | Indicates what level of different values reside in, and shows the cross-relationships between category and series. |
| **Scatter** | | |
| Scatter 2-D button | Scatter 2-D | Compares pairs of values. |
| Scatter Straight Line 2-D button | Scatter Straight Line 2-D | Scatter with data points connected by smoothed lines. |
| Scatter Curved Line 2-D button | Scatter Curved Line 2-D | Scatter with data points connected by lines. |
| **Bubble** | | |
| Bubble 2-D button | Bubble 2-D | Compares sets of 3 values. Similar to a scatter chart with the third value displayed as the size of the bubble marker. |
| **Stock** | | |
| High-Low 2-D button | High-Low 2-D | High-Low. Requires values of two fields. |
| High-Low-Close 2-D button | High-Low-Close 2-D | High-Low-Close. Requires values of three fields. |
| Open-High-Low-Close 2-D button | Open-High-Low-Close 2-D | Open-High-Low-Close. Requires values of four fields. |
| **Bullet** | | |
| Bullet 2-D button | Bullet 2-D | Bullet. Replaces the meters and gauges that are often used on dashboards. Its linear design not only gives it a small footprint, but also supports more efficient reading than radial meters. |
| **Heat Map** | | |
| Heat Map button | Heat Map | Displays values as rectangles marked by colors and sizes in a matrix. |
| **Combo** | | |
|  | Combo | Combination chart. You can use two or more types of data markers to represent different data values in a combo chart. |
