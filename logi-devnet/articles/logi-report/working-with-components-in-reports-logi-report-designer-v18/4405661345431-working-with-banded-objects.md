# Working with Banded Objects

A banded object is a type of component that can present grouped data and detail data. A banded object contains several banded panels with which you can easily organize data fields and other components, for example, you can embed additional data components inside a banded object and add data container links between the data components and the banded object. This topic introduces the features of banded objects and how you can use them to design the banded objects that best serve your reporting requirements.

![Note icon](https://devnet.logianalytics.com/hc/article_attachments/4420394642839/noteicon.jpg) You cannot use banded objects in library components.
