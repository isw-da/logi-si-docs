# Flash Properties

This topic describes how you can use the Flash Properties dialog box to update the properties of a flash object. Server displays the dialog box when you right-click a flash object and select **Properties** from the shortcut menu.

![Flash Properties dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885486615/flashprpty.gif)

**Name**

Specifies the display name of the flash.

**Show NLS Value**

Select this option to show the translated name for the display name of the object in the **Name** text box if you have enabled the NLS feature and translated it.

If you select this option, it takes effect only when you have not modified the display name of the object.

**Alternate Text**

Specifies the alternate text which will be shown if the flash cannot be displayed.

**Width**

Specifies the width of the flash.

**Height**

Specifies the height of the flash.

**Movie Quality**

Specifies the quality of the flash.

**Play**

Specifies whether to start the flash file automatically when the report is opened.

**Loop**

Specifies whether to play the flash file repeatedly.

**OK**

Applies the settings and closes this dialog box.

**Cancel**

Cancels the settings and closes this dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880314903/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885431447/btn_close.gif)

Ignores the setting and closes this dialog box.
