# Color List Dialog Box Properties

This topic describes how you can use the Color List dialog box to modify the color patterns of data markers in the same data series respectively in a gauge chart or modify alternating color for a table.

Server displays the dialog box when you do any of the following:

* Select the Color List button in the Pointer tab of the [Format Dial Gauge](https://devnet.logianalytics.com/hc/en-us/articles/1500009775481-Format-Dial-Gauge-Dialog-Box-Properties#Pointer) dialog box, [Format Bar Gauge](https://devnet.logianalytics.com/hc/en-us/articles/1500009747162-Format-Bar-Gauge-Dialog-Box-Properties#Pointer) dialog box, or [Format Solid Gauge](https://devnet.logianalytics.com/hc/en-us/articles/1500009747382-Format-Solid-Gauge-Dialog-Box-Properties#Pointer) dialog box, or in the Circular Graph tab of the [Format Solid Gauge](https://devnet.logianalytics.com/hc/en-us/articles/1500009747382-Format-Solid-Gauge-Dialog-Box-Properties#Circular) dialog box or [Format Activity Gauge](https://devnet.logianalytics.com/hc/en-us/articles/1500009747142-Format-Activity-Gauge-Dialog-Box-Properties#Circular) dialog box.
* Select ![Chooser button](https://devnet.logianalytics.com/hc/article_attachments/4404885376663/btn_chsr1.gif) in the value cell of the Pattern List property for a table in the Inspector panel.

![Color List dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885491095/colorlist.gif)

**Colors**

Specifies the color and transparency for the chart data markers or for the alternating color of the table. Select the items in the box one by one to set the color pattern that each of the item stands for.

* ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404880163351/btn_add1.gif)  
   Adds a new pattern item.
* ![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404885309335/btn_delete.gif)  
   Removes the selected pattern item.
* ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404885326231/btn_mvup.gif)**Move Up button**  
   Moves the selected pattern item higher in the list.
* ![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404880154263/btn_mvdown.gif)**Move Down button**  
   Moves the selected pattern item lower in the list.
* **Fill**  
   Specifies the color schema for the pattern.

  To change the color, select the color indicator to select a color from the color palette. You can select More Colors in the color palette to access the [Color Picker](https://devnet.logianalytics.com/hc/en-us/articles/1500009746842-Color-Picker-Dialog-Box-Properties) dialog box in which you can select a color within a wider range, or select More Fill Effects to access the [Fill Effects](https://devnet.logianalytics.com/hc/en-us/articles/1500009747082-Fill-Effects-Dialog-Box-Properties) dialog box to specify gradient fill effect. You can also type a color string in the format #RRGGBB directly in the text box.
* **Transparency**  
   Specifies the transparency of the color.

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880314903/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885431447/btn_close.gif)

Ignores the setting and closes this dialog box.
