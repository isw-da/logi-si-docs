# Font Dialog Box Properties

This topic describes how you can use the Font dialog box to specify the font format of a label or field. Server displays the dialog box when you select a label or field and select the Font button ![Font button](https://devnet.logianalytics.com/hc/article_attachments/4404885309591/btn_wbrpt_font1.gif) on the toolbar.

![Font dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880356119/font.gif)

**Font**

Specifies the font face.

**Size**

Specifies the font size.

**Color**

Specifies the font color.

To change the color, select the color indicator to select a color from the color palette. You can select More Colors in the color palette to access the [Color Picker](https://devnet.logianalytics.com/hc/en-us/articles/1500009746842-Color-Picker-Dialog-Box-Properties) dialog box in which you can select a color within a wider range. You can also type a color string in the format #RRGGBB directly in the text box.

![Bold](https://devnet.logianalytics.com/hc/article_attachments/4404880243351/btn_bold.gif)

Specifies whether to make the object bold.

![Italic](https://devnet.logianalytics.com/hc/article_attachments/4404880243863/btn_italic.gif)

Specifies whether to make the object italic.

![Underline](https://devnet.logianalytics.com/hc/article_attachments/4404880244119/btn_underline.gif)

Specifies whether to underline the object.

**Preview**

Shows the font effect.

**OK**

Applies the font format you specified and closes this dialog box.

**Cancel**

Cancels to change the font format and closes the dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880314903/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885431447/btn_close.gif)

Ignores the setting and closes this dialog box.
