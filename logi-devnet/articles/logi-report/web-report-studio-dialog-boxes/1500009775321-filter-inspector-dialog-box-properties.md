# Filter Inspector Dialog Box Properties

The Filter Inspector dialog box lists all the filters that the current web report is using for your information. Server displays the dialog box when you select the Inspector button in the [Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500009775301-Filter-Dialog-Box-Properties) dialog box.

![Filter Inspector dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880390551/fltrinspct.gif)

**Query Filter**

Filters that you created via the [Query Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500009778741-Applying-Filters-in-Web-Report#BV) dialog box.

**Dialog Filter**

Filters that you created via the [Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500009778741-Applying-Filters-in-Web-Report#Dialog) dialog box.

**On-screen Filter**

Filters that you created via the [Filter panel](https://devnet.logianalytics.com/hc/en-us/articles/1500009778741-Applying-Filters-in-Web-Report#Panel) and via [filter controls](https://devnet.logianalytics.com/hc/en-us/articles/1500009750222-Using-Web-Controls-in-Web-Report#Filter).

**Go To Filter**

Go-to filters that you created when performing the [go-to-by-value action](https://devnet.logianalytics.com/hc/en-us/articles/1500009778781-Going-Through-Web-Report-Data#Go-to-by-value) on the report.

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Cancels the changes and exits the dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880314903/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885431447/btn_close.gif)

Ignores the setting and closes this dialog box.
