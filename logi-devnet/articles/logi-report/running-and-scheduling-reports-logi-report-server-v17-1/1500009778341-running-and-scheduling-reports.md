# Running and Scheduling Reports Logi Report Server v17.1

In the Server Console, you can run or schedule the reports that you [published](https://devnet.logianalytics.com/hc/en-us/articles/1500009777241-Publishing-Resources) or saved into the resource tree, in a variety of formats. This topic describes how you can run and schedule to run reports and specify parameter values to reports.

When running a report, you can use either direct running or advanced running. Scheduling a report enables you to deliver a report to different destinations in various formats at one or more designated dates and times. Logi Report Server records the report running tasks in different task tables. You can view the status of these tasks, such as scheduled tasks that are waiting to run, the tasks that are currently running, and the tasks that have already finished running in different task tables.

Select the following links to view the topics:

* [Running Reports](https://devnet.logianalytics.com/hc/en-us/articles/1500009778381-Running-Reports)
* [Scheduling Reports](https://devnet.logianalytics.com/hc/en-us/articles/1500009749542-Scheduling-Reports)
* [Specifying Parameter Values](https://devnet.logianalytics.com/hc/en-us/articles/1500009778361-Specifying-Parameter-Values)

You can also use URL commands to directly run reports and schedule report tasks. For more information, see [Running Reports via URL](https://devnet.logianalytics.com/hc/en-us/articles/1500009750362-Running-Reports-via-URL) and [Scheduling Reports via URL](https://devnet.logianalytics.com/hc/en-us/articles/1500009778901-Scheduling-Reports-via-URL).

**Tip:** You can customize the Resources page in the Server Console to suit your requirements by setting your own preferences. To do this:

1. Select **Tools** > **Preferences** on the task bar of the Resources page.
2. In the **General** tab of the Preferences page, specify the default viewing format when directly running a report, and set which columns to show in the resource information table.
3. In the **Export Formats** tab, predefine the [properties for each export format](https://devnet.logianalytics.com/hc/en-us/articles/1500009743782-Configuring-Server-Profile#Export) which will be applied when you run or schedule a report.
4. Select **OK** to apply the settings. Server saves the specified default report viewing format and export properties into the [server profile](https://devnet.logianalytics.com/hc/en-us/articles/1500009743782-Configuring-Server-Profile).
