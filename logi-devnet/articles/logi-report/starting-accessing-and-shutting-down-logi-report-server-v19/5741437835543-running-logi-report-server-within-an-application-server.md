# Running Logi Report Server Within an Application Server

Logi Report Server can also run as a servlet inside a Java application server, in addition to running as a standalone server and as a service. This topic describes how to deploy Logi Report Server as a WAR or EAR.

Since Logi Report Server is implemented using servlets and JSPs, it can work with any servlet-enabled web application server by assembling and deploying Logi Report Server as a Web Application Archive (WAR) or Enterprise Application Archive (EAR).

For more information about creating the deployable archive and deploying the archive into popular application servers, see [Logi Report Server Integration](https://devnet.logianalytics.com/hc/en-us/articles/5741460910103--Server-Integration-Server-v19).
