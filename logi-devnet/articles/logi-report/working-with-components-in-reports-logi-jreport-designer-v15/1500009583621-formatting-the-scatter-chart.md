# Formatting the Scatter Chart

To format a scatter chart, follow the steps below:

1. Right-click any node in the scatter chart, and then select **Format Scatter** on the shortcut menu to bring up the [Format Scatter](https://devnet.logianalytics.com/hc/en-us/articles/1500009587421-Format-Scatter-Dialog) dialog.

   ![Format Scatter dialog - General](https://devnet.logianalytics.com/hc/article_attachments/4404893885847/fmtsctr_gnrl.gif)
2. In the General tab, set the general options for the scatter chart.

   In the Layout box, specify the layout of the chart. In the Line box, specify the thickness of the lines that connect the nodes if you have set the layout to Straight Line or Curved Line. In the Node box, set the node style, width and height as required.
3. In the Fill tab, specify the color and transparency of the color schema to fill the selected scatter markers in the same data series (to change the color, select the color image and select a color from the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/1500009593081-Menu-Tabs#Palette) or input the hexadecimal value of a color directly in the text box). You can also select the **Color List** button to specify the color pattern for scatter markers in the same data series respectively in the [Color List](https://devnet.logianalytics.com/hc/en-us/articles/1500009564822-Color-List-Dialog) dialog.

   ![Format Scatter dialog - Fill](https://devnet.logianalytics.com/hc/article_attachments/4404889342487/fmtsctr_fill.gif)
4. In the Data Label tab, set the font formats and effects of the data labels.
5. Select **OK** to apply the settings and close the dialog.
