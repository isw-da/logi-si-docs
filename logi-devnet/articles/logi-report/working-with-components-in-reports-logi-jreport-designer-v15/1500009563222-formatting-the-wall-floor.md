# Formatting the Wall/Floor

You can format the wall and floor of a chart at any time. Note that 2-D charts do not have floor, and pie and organization charts do not have floor or wall.

**To format the wall of a chart:**

1. Right-click any chart element and select **Format Walls** > **Format Wall** on the shortcut menu to bring up the [Format Wall](https://devnet.logianalytics.com/hc/en-us/articles/1500009566242-Format-Wall-Dialog) dialog.

   ![Format Wall dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893880983/fmtwall.gif)
2. Set the fill mode for the wall and bound wall, and the color, style, transparency and thickness of the wall border as required (to change a color, select the color image and select a color from the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/1500009593081-Menu-Tabs#Palette) or input the hexadecimal value of a color directly in the text box).
3. When done, select **OK** to apply the settings and close the dialog.

**To format the floor of a chart:**

1. Right-click any chart element and select **Format Walls** > **Format Floor** on the shortcut menu to bring up the [Format Floor](https://devnet.logianalytics.com/hc/en-us/articles/1500009587201-Format-Floor-Dialog) dialog.

   ![Format Floor dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889350167/fmtflr.gif)
2. Set the fill mode for the floor, and the color, style, transparency and thickness of the floor border as required (to change a color, select the color image and select a color from the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/1500009593081-Menu-Tabs#Palette) or input the hexadecimal value of a color directly in the text box).
3. When done, select **OK** to apply the settings and close the dialog.
