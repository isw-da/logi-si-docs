# KPI Components

A KPI (Key Performance Indicator) component is a data container based on a business view for displaying KPI value together with a chart to help illustrate the value. It is not supported in page reports.

In a KPI component the following objects can be added:

* Aggregations and formulas as the KPI values.
* Labels to describe the values and as the KPI title or description.
* Images to decorate the KPI component or used as logo.
* A KPI chart to assist the KPI values, for example, to show the trend of total sales over time. At most one KPI chart can be included in a KPI component. A KPI chart does not have the legend and axes compared to normal Logi JReport charts. It can only be inserted in a KPI component and inherit dataset from the KPI component. The following 2-D chart types are supported in KPI charts: Bar, Bench, Line, Area, Pie, and Bullet.

A KPI component is bound with a business view so the KPI value and KPI chart in it are both based on this business view. The objects in a KPI component can be freely positioned and resized by dragging inside the KPI component.

**To create a KPI component in a web report or library component:**

1. Position the mouse pointer at the destination where you want to insert the KPI component. A KPI component can be inserted in a web report or library component as listed in [Component placement](https://devnet.logianalytics.com/hc/en-us/articles/1500009583241-Working-with-Components-in-Reports#Relationship).
2. Do either of the following:
   * Drag ![KPI button](https://devnet.logianalytics.com/hc/article_attachments/4404893952791/btn_kpi.gif) from the Components panel to the destination.
   * Select **Insert** > **KPI**.
   * Select **Home** > **Insert** > **KPI**.
3. In the [Choose Data](https://devnet.logianalytics.com/hc/en-us/articles/1500009564662-Choose-Data-Dialog#BV) dialog, select a [business view](https://devnet.logianalytics.com/hc/en-us/articles/1500009562582-Business-Views) in the current catalog to bind with the KPI component and select **OK**.

   ![Choose Data dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889390231/chsdt-bv.gif)
4. A blank KPI component is inserted at the destination. You can then define the KPI component by adding objects into it.

   ![Blank KPI Component](https://devnet.logianalytics.com/hc/article_attachments/4404893986839/cmpnt_kpi_blank.gif)
5. Drag an aggregation or a formula from the Data panel as the KPI value. The name label of the object is added at the same time. Edit the label text if needed.
6. Insert a KPI chart to demonstrate data about the KPI value.
   1. Right-click in the blank area of the KPI component and then select **Insert KPI Chart** from the shortcut menu. The [Create Chart](https://devnet.logianalytics.com/hc/en-us/articles/1500009564882-Create-Chart-Dialog-Business-View-Based-#KPI) wizard is displayed.
   2. The Data screen shows the business view bound with the KPI component. It cannot be changed.
   3. In the Type, Display, Filter, Layout and Style screens, define the chart [as you would do for a normal chart](https://devnet.logianalytics.com/hc/en-us/articles/1500009563242-Inserting-a-Chart#Web).
   4. Select **Finish** to create the chart. A chart is now inserted into the KPI component.
7. Add more [aggregations](https://devnet.logianalytics.com/hc/en-us/articles/1500009563602-Summary-Fields), [formulas](https://devnet.logianalytics.com/hc/en-us/articles/1500009563362-Formula-Fields), [labels](https://devnet.logianalytics.com/hc/en-us/articles/1500009584041-Inserting-and-Formatting-a-Label) and [images](https://devnet.logianalytics.com/hc/en-us/articles/1500009583961-Images#Insert) into the KPI component if required.
8. Resize the KPI component and the objects in it and adjust the position of the objects by dragging to improve the layout.

When a KPI component is created in a report, you can further edit the objects in it according to your requirements. The KPI chart in a KPI component supports most of the operations that can be applied on a normal chart. For example, you can [modify its definition](https://devnet.logianalytics.com/hc/en-us/articles/1500009583701-Modifying-a-Chart) using the [Chart Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009585841-Chart-Wizard-Dialog-Business-View-Based-#KPI), [add link](https://devnet.logianalytics.com/hc/en-us/articles/1500009583681-Binding-a-Link-to-a-Chart) to its data markers, and format its [paper](https://devnet.logianalytics.com/hc/en-us/articles/1500009583541-Formatting-the-Paper), [platform](https://devnet.logianalytics.com/hc/en-us/articles/1500009583561-Formatting-the-Platform), [wall](https://devnet.logianalytics.com/hc/en-us/articles/1500009563222-Formatting-the-Wall-Floor), [label](https://devnet.logianalytics.com/hc/en-us/articles/1500009583521-Formatting-the-Labels) and data markers ([bars/benches](https://devnet.logianalytics.com/hc/en-us/articles/1500009583481-Formatting-the-Bars), [lines](https://devnet.logianalytics.com/hc/en-us/articles/1500009563142-Formatting-the-Lines), [areas](https://devnet.logianalytics.com/hc/en-us/articles/1500009562962-Formatting-the-Areas), [pies](https://devnet.logianalytics.com/hc/en-us/articles/1500009563162-Formatting-the-Pies-Donuts) or [bullets](https://devnet.logianalytics.com/hc/en-us/articles/1500009563022-Formatting-the-Bullets)). If you want to remove the KPI chart, select the KPI component, right-click and select **Remove KPI** chart on the shortcut menu.

The following example uses a KPI component to show total sales:

1. [Open the catalog file](https://devnet.logianalytics.com/hc/en-us/articles/1500009562822-Opening-a-Catalog) SampleReports.cat in `<install_root>\Demo\Reports\SampleReports`.
2. Select **Home** > **New** > **Web Report** to create a blank web report.
3. Drag ![KPI button](https://devnet.logianalytics.com/hc/article_attachments/4404893952791/btn_kpi.gif) from the Visual category of the Components panel to the destination.
4. In the Choose Data dialog, select **WorldWideSalesBV** in the current catalog to bind with the KPI component and select **OK**.
5. A blank KPI component is inserted at the destination.
6. In the Data panel, [create a dynamic formula](https://devnet.logianalytics.com/hc/en-us/articles/1500009592981-Web-Reports#Formula) named KPI\_TotalSales with the following expression. Use it as Aggregation.

   |  |
   | --- |
   | ``` if (@"Orders Detail.Total Sales">=1000000)     ToText(@"Orders Detail.Total Sales"/1000000, "$#,###.00")+"M" else if (@"Orders Detail.Total Sales">=1000)     ToText(@"Orders Detail.Total Sales"/1000, "$#,###.00")+"K" else      ToText(@"Orders Detail.Total Sales", "$#,###.00") ``` |
7. Drag the formula from the Data panel to the KPI component as the KPI value.
8. Right-click in the blank area of the KPI component and then select **Insert KPI Chart** from the shortcut menu.
9. In the Create Chart wizard, go to the Display screen, add the group object **Sales Quarter** to the category axis, the aggregation object **Total Sales** to the value axis, then and select **Finish** to insert the KPI chart.

   ![Create Chart dialog - Display screen](https://devnet.logianalytics.com/hc/article_attachments/4404889409943/cmpnt_kpi_cht.gif)
10. Adjust the position of the inserted objects. Widen the formula label and field so that the label and value can be fully displayed.

    ![KPI Component - Design](https://devnet.logianalytics.com/hc/article_attachments/4404889410199/cmpnt_kpi_rst.gif)
11. Select the **View** tab to view the real data.

    ![KPI Component - View](https://devnet.logianalytics.com/hc/article_attachments/4404889410455/cmpnt_kpi_vw.gif)
