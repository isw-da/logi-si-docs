# Formatting the Gridlines

The gridlines of a chart can be customized (pie, indicator, heat map and organization charts do not have gridlines). To do this:

1. Right-click any chart element and select the desired entry from the Format Gridlines submenu to bring up the corresponding Format Gridline dialog.
2. Specify the line color, style, transparency, and thickness of the gridline (to change the color, select the color image and select a color from the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/1500009593081-Menu-Tabs#Palette) or input the hexadecimal value of a color directly in the text box).
3. When done, select **OK** to accept the settings and close the dialog.

See also the following topics for detailed explanation about options in each dialog:

* [Format Category (X) Gridline](https://devnet.logianalytics.com/hc/en-us/articles/1500009565822-Format-Category-X-Gridline-Dialog) dialog
