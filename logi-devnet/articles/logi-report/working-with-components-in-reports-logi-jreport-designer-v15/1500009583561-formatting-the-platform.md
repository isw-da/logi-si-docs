# Formatting the Platform

The platform of a chart is the container of the other chart elements, such as the paper, legend, and labels.

**To format the platform of a chart:**

1. Right-click any chart element and select **Format Platform** from the shortcut menu to bring up the [Format Platform](https://devnet.logianalytics.com/hc/en-us/articles/1500009587321-Format-Platform-Dialog) dialog.
2. In the Fill tab, set the color and transparency of the color (to change the color, select the color image and select a color from the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/1500009593081-Menu-Tabs#Palette) or input the hexadecimal value of a color directly in the text box).

   ![Format Platform dialog - Fill](https://devnet.logianalytics.com/hc/article_attachments/4404889344663/fmtpltfm_fill.gif)
3. In the Border tab, set the border mode, including the border style, color, transparency, line style, thickness, end caps style, and line joint mode, and if you specify the line joint mode to be joint round, you can set the the radius for the border joint. You can also specify whether the border is to be outlined, and whether or not to automatically resize the border dashes if the border is drawn with dashes.

   ![Format Platform dialog - Border](https://devnet.logianalytics.com/hc/article_attachments/4404893888023/fmtpltfm_brdr.gif)
4. When done, select **OK** to apply the settings and close the dialog.
