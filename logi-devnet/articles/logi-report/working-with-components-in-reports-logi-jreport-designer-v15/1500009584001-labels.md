# Labels

A label is an object that contains a string. It is typically a brief description used to identify a field or other value nearby. In Logi JReport, the display type of a label can be changed, that is you can map the string to another value or image to be displayed instead.

The following topics describe tasks related to labels:

> * [Inserting and Formatting a Label](https://devnet.logianalytics.com/hc/en-us/articles/1500009584041-Inserting-and-Formatting-a-Label)
> * [Binding a Link to a Label](https://devnet.logianalytics.com/hc/en-us/articles/1500009563442-Binding-a-Link-to-a-Label)
> * [Changing the Display Type of a Label](https://devnet.logianalytics.com/hc/en-us/articles/1500009563422-Changing-the-Display-Type-of-a-Label)
> * [Applying Web Actions to a Label](https://devnet.logianalytics.com/hc/en-us/articles/1500009584021-Applying-Web-Actions-to-a-Label)

**See an example:** The SampleComponents catalog, included with Logi JReport Designer, contains reports that have examples of how each component type could be used in a report. For the label component example, open `<install_root>\Demo\Reports\SampleComponents\ForLabel.cls`.
