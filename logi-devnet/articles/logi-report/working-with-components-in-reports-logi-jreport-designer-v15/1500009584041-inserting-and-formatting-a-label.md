# Inserting and Formatting a Label

Labels can be inserted in the report areas listed in [Component placement](https://devnet.logianalytics.com/hc/en-us/articles/1500009583241-Working-with-Components-in-Reports#Relationship).

**To insert a label:**

* Drag the **Label** button ![Label button](https://devnet.logianalytics.com/hc/article_attachments/4404893986327/btn_instlabel.gif)from the Components panel to the desired report destination.
* Select **Insert** > **Label** or **Home** > **Insert** > **Label**, then select the mouse button in the location where you want the label to be displayed.

A label with the default text "Label" is inserted. You can then edit the text of a label as follow:

* To edit the full text of a label, double-click the label to select the text, then type the replacement text.
* To edit a portion of a label's text, double-click the label, then select in the label and drag to select the portion, after that, type the replacement text.

You can also modify a label's text by using the Text property value in the Report Inspector.

**To format a label:**

You can format the label text with the Format tab, for example, you can change the font size, color, and alignment of the text.

To format a label, first select the label, and then perform any of the following tasks:

* To change the alignment of a label, select the required alignment button: **Justify**![Justify button](https://devnet.logianalytics.com/hc/article_attachments/4404893865879/btn_justify.gif), **Left**![Left button](https://devnet.logianalytics.com/hc/article_attachments/4404893798551/btn_left.gif), **Center**![Center button](https://devnet.logianalytics.com/hc/article_attachments/4404889281815/btn_center.gif) or **Right**![Right button](https://devnet.logianalytics.com/hc/article_attachments/4404889282071/btn_right.gif). The text in the label will then be aligned according to your selection.
* To change the font face, select the **Font Face Box**![Font Face Box button](https://devnet.logianalytics.com/hc/article_attachments/4404893794583/btn_fntface.gif) and then select the required font from the drop-down list. The fonts introduced with \* are True Type Fonts. Refer to the chapter [True Type Fonts](https://devnet.logianalytics.com/hc/en-us/articles/1500009583001-True-Type-Fonts) for more information.
* To change the font size, select the **Font Size Box**![Font Size Box button](https://devnet.logianalytics.com/hc/article_attachments/4404893794839/btn_fntsize.gif) and then select the required size from the drop-down list (or enter the required value in the text box and then press **Enter**).
* To change the font style, select the corresponding font style buttons on the Format tab: **Bold**![Bold button](https://devnet.logianalytics.com/hc/article_attachments/4404889279511/btn_bold.gif), **Italic**![Italic button](https://devnet.logianalytics.com/hc/article_attachments/4404889279767/btn_italic.gif) and **Underline**![Underline button](https://devnet.logianalytics.com/hc/article_attachments/4404893795095/btn_underline.gif).
* To change the background color of the label, select the **Background Color** button ![Background Color button](https://devnet.logianalytics.com/hc/article_attachments/4404893795351/btn_bkgrdcolor.gif); to change the foreground color of the label text, select the **Foreground Color** button ![Foreground Color button](https://devnet.logianalytics.com/hc/article_attachments/4404893795607/btn_fntcolor.gif).

**Tip:** You cannot use drag and drop or the Insert menu to insert labels into charts. For details about chart labels, see the chart topic [Formatting the Labels](https://devnet.logianalytics.com/hc/en-us/articles/1500009583521-Formatting-the-Labels).
