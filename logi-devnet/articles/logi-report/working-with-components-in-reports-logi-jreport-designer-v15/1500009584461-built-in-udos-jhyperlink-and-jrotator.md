# Built-in UDOs: JHyperLink and JRotator

Logi JReport provides two built-in UDOs: JHyperLink and JRotator. JHyperLink is used to create hyperlinks in reports, and JRotator is used to rotate text or images

The following topics describe the usage of the two UDOs in detail:

> * [Creating Links with JHyperLink](https://devnet.logianalytics.com/hc/en-us/articles/1500009584481-Creating-links-with-JHyperLink)
> * [Rotating Text/Image with JRotator](https://devnet.logianalytics.com/hc/en-us/articles/1500009563802-Rotating-Text-Image-with-JRotator)

See also [Properties - UDO](https://devnet.logianalytics.com/hc/en-us/articles/1500009570002-Properties-of-UDO-in-Page-Report) for details about the properties of JHyperLink and JRotator in the Report Inspector.
