# Applying CSS Styles in Web Report

You can apply CSS styles to web reports to format their appearance and characteristics. This topic describes how you can apply a style to a data component and to the whole report.

You can create and set up your own CSS styles in Logi Report Designer or any other CSS editor. When you publish your catalogs to Logi Report Server, you can include these custom styles with the catalogs.

You can apply styles to a table, crosstab, chart (including a KPI chart), banded object, or KPI individually, or to the whole report at a time. The last style always takes effect when you applied more than one style to the same object.

* **Applying a style to a report**  
   When you apply a style to the whole report, all components in the report take a uniform appearance. You can apply a style to a report in either of the following ways:
  + Select a style in the [Style screen](https://devnet.logianalytics.com/hc/en-us/articles/1500009776601-Web-Report-Wizard-Properties#Style) of the Web Report Wizard.
  + In Web Report Studio, select **Menu** > **Edit** > **Style** then select a style from submenu.
* **Applying a style to a component**  
   To apply a style to a table, crosstab, chart, or KPI, hover mouse pointer on the component until Server displays the icon ![](https://devnet.logianalytics.com/hc/article_attachments/4404880145943/btn_pgrpt_drag.gif) at the top left, then right-click the icon and select a style from the Apply Style list on the shortcut menu.

![Note icon](https://devnet.logianalytics.com/hc/article_attachments/4404885305111/noteicon.jpg)When you apply a style, it replaces any custom formatting you may have applied to individual objects such as font and color if the style contains the formatting for the objects.
