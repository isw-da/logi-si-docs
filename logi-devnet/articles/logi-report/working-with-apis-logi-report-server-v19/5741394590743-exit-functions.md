# Exit Functions

Logi Report Designer provides three Exit functions: After Init Parameter, After Run, and Before Run.

With these functions, you can develop an action to be called before, during, or after the process of running a report. Designer will return a status. If it returns true, Logi Report Engine will go on running. If false, Engine will stop at this point.

For more information, see Using the Exit Functions in the *Logi Report Designer Guide*.
