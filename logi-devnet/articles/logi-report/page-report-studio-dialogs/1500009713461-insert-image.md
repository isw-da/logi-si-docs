# Insert Image

The Insert Image dialog helps you to insert an image into a report or specify an image as the pointer in a gauge chart. It varies according to different sources it is opened from.

---

When the Insert Image dialog is opened by selecting Menu > Insert > Image or dragging Image from the Toolbox panel into the current report, it helps you to [insert an image](https://devnet.logianalytics.com/hc/en-us/articles/1500009691522-Adding-Report-Objects#Image) into the report and contains the following options.

![Insert Image dialog](https://devnet.logianalytics.com/hc/article_attachments/4412131430807/insrtimg.gif)

**Image From**

Specifies the source of the image file.

* **Local File**  
   Specifies to use an image from the local file system. The types of the images that can be used are defined by the administrator in the Administration > Configuration > Upload page in the Logi JReport Server console.
  + **File Name**  
     Specifies the path and name of the image file. You can select the Browse button to locate the image file.
* **Web URL**  
   Specifies to use an image via URL.
  + **Image URL**  
     Specifies the URL of the image file. Logi JReport will record the latest 10 entered URLs in the drop-down list.
* **Library**  
   Specifies to use an existing image.
  + **My Images**  
     The My Images folder is a virtual location where Logi JReport Server stores the images that have once been inserted into reports. Select the one you want to use.
  + **Preview**  
     Displays a preview of the selected image.

**OK**

Inserts the image in the report and closes the dialog.

**Cancel**

Cancels the insertion and closes the dialog.

**Help**

Displays the help document about this feature.

---

When the Insert Image dialog is opened by selecting Customized in the Value Pointer or Target Pointer drop-down list in the [Format Bar Gauge](https://devnet.logianalytics.com/hc/en-us/articles/1500009687262-Format-Bar-Gauge), [Format Dial Gauge](https://devnet.logianalytics.com/hc/en-us/articles/1500009687302-Format-Dial-Gauge), or [Format Solid Gauge](https://devnet.logianalytics.com/hc/en-us/articles/1500009713341-Format-Solid-Gauge) dialog, or in the Value Pointer drop-down list in the [Style List](https://devnet.logianalytics.com/hc/en-us/articles/1500009713821-Style-List) dialog, it helps you to specify an image as the pointer in a gauge chart and contains the following options.

![Insert Image dialog](https://devnet.logianalytics.com/hc/article_attachments/4412112509207/insrtimg.gif)

**Image From**

Specifies the source of the image file.

* **Local File**  
   Specifies to use an image from the local file system.
  The types of the images that can be used are defined by the administrator in the Administration > Configuration > Upload page in the Logi JReport Server console.
  + **File Name**  
     Specifies the path and name of the image file. You can select the Browse button to locate the image file.
* **Web URL**  
   Specifies to use an image via URL.
  + **File URL**  
     Specifies the URL of the image file.

  **Note:** If your Logi JReport Server is in an intranet which requires a proxy, to successfully access the image via URL, you need to add the parameters `-Dhttp.proxyHost=XXX -Dhttp.proxyPort=XX` to the server's startup file JRServer.bat, which is located in `<install_root>\bin`.
* **Library**  
   Specifies to use an existing image.
  + **My Pictures**  
     The My Pictures folder is a virtual location where Logi JReport Server stores the images that have once been inserted into reports. Select the one you want to use.

**Preview**

Displays a preview of the selected image.

**OK**

Uses the image as the pointer in the gauge chart and closes the dialog.

**Cancel**

Cancels the operation and closes the dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
