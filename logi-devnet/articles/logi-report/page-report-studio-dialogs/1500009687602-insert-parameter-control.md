# Insert Parameter Control

The Insert Parameter Control dialog helps you to [insert a parameter control](https://devnet.logianalytics.com/hc/en-us/articles/1500009718201-Applying-Web-Controls#Parameter) into a report for delivering parameter values to the report.

![Insert Parameter Control dialog](https://devnet.logianalytics.com/hc/article_attachments/4412131418647/insrtprmctrl.gif)

**Select a Parameter**

Specifies a parameter from the list where all the parameters except cascading parameters used by the current report are displayed.

**OK**

Inserts a parameter control into the report and closes the dialog.

**Cancel**

Cancels the operation and closes the dialog.

**Help**

Displays the help document about this feature.
