# Save Report Template

The Save Report Template dialog requires your confirmation on whether or not to save the report template when [saving a report](https://devnet.logianalytics.com/hc/en-us/articles/1500009691682-Saving-the-Report).

![Save Report Template dialog](https://devnet.logianalytics.com/hc/article_attachments/4412139577495/svtmplt.gif)

**Save Sort Criteria**

Specifies whether or not to save the sorting criteria with the report.

**Save Filter Criteria**

Specifies whether or not to save the filtering criteria with the report.

**Yes**

Applies the settings and closes this report.

**Cancel**

Does not save the changes to the report and returns to the report.

**Help**

Displays the help document about this feature.
