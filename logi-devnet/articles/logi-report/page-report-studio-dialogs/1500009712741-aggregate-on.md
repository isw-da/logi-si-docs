# Aggregate On

The Aggregate On dialog helps you to [calculate data in the selected table detail column](https://devnet.logianalytics.com/hc/en-us/articles/1500009691642-Manipulating-Data-Components#AggregateOn).

![Aggregate On dialog](https://devnet.logianalytics.com/hc/article_attachments/4412112546071/agrgton.gif)

**Aggregate On**

Displays the field bound with the detail column. This field is also the one on which the summary is based.

**Function**

Specifies the function to compute the field in the detail column.

**Distinct On**

Available and should be set when DistinctSum is selected as the function. It specifies the fields according to whose unique values to calculate DistinctSum. Select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4412139488919/btn_chsr2.gif) to select the required fields in the [Select Fields](https://devnet.logianalytics.com/hc/en-us/articles/1500009713761-Select-Fields) dialog.

**OK**

Summarizes the detail column data with the specified function and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
