# Insert Column

The Insert Column dialog helps you to [insert the selected group field as a column header](https://devnet.logianalytics.com/hc/en-us/articles/1500009691642-Manipulating-Data-Components#CTColumn) in a crosstab.

![Insert Column dialog](https://devnet.logianalytics.com/hc/article_attachments/4412112544663/insrtclmn.gif)

**Row**

Displays the name of the field in the text box. It is read only.

**Label**

Specifies the display name of the field, which will label the field in the crosstab. You can leave it blank.

**OK**

Inserts the field and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
