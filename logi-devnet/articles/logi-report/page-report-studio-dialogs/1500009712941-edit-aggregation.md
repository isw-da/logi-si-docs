# Edit Aggregation

The Edit Aggregation dialog helps you to [edit an existing aggregation object](https://devnet.logianalytics.com/hc/en-us/articles/1500009691582-Using-Dynamic-Resources#Agg).

![Edit Aggregation dialog](https://devnet.logianalytics.com/hc/article_attachments/4412112627735/edtaggrgtn.gif)

**Aggregation Name**

Specifies the display name of the aggregation object.

**Mapping Name**

Specifies the mapping name of the field to which the aggregation object is mapped. Select ![Select Resource](https://devnet.logianalytics.com/hc/article_attachments/4412112493207/btn_chsr.gif) to select the field in the [Select Resource](https://devnet.logianalytics.com/hc/en-us/articles/1500009687862-Select-Resource) dialog.

This option and the button are disabled when you specify to create an aggregation object on a dynamic formula.

**Aggregate Function**

Specifies the aggregate function of the aggregation object. The function will be applied according to the group where the aggregation is placed or the entire report if placed in the report header or footer.

**Distinct On**

Available and should be set when DistinctSum is selected as the aggregate function. It specifies the fields according to whose unique values to calculate DistinctSum. Select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4412139488919/btn_chsr2.gif) to select the required fields in the [Select Fields](https://devnet.logianalytics.com/hc/en-us/articles/1500009713761-Select-Fields) dialog.

**OK**

Applies the changes to the aggregation object and closes the dialog.

**Cancel**

Cancels the operation and closes the dialog.

**Help**

Displays the help document about this feature.
