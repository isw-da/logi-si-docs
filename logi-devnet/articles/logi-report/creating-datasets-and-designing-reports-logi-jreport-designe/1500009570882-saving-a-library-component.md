# Saving a Library Component

To save a library component, select **Home** > **Save** or **File** > **Save**. This operation can also be accomplished by pressing the Ctrl + S keys. If the library component to be saved is new, the Save Library Component As dialog will appear for you to input a name.

**To save a library component with a different name in the current directory:**

1. Select **File** > **Save As**. The Save Library Component As dialog appears.

   ![Save Library Component As dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889290775/svlcas.gif)
2. Type a new file name in the Report Name field.
3. The library component file type (\*.lc) is the only default value in the Report of Type drop-down list.
4. Select **Save** to save the library component with the new name.
