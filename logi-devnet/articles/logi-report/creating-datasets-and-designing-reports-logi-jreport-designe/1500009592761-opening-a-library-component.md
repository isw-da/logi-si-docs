# Opening a Library Component

This topic demonstrates how to open a library component.

**To open a library component:**

1. Select **Home** > **Open** or **File** > **Open**. The Open Report dialog displays.

   ![Open Report dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889285783/opnrptst.gif)
2. Browse to find the catalog that contains the library component you want to open.
3. In the Reports tab, select **Library Component (\*.lc)** from the Report of Type drop-down list.
4. All the library components in the current catalog will be displayed. Select the desired library components.

   If you want to open library components that you have opened recently, switch to the **Recent** tab, and then select the library components you want to open.
5. Select **Open** to open the library components that you specified.

**Tip:** If you just want to access some recently open library components, select **File** > **Open Recent** and then select the library components one by one from the submenu.
