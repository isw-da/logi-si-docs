# Updating the Server License

Specific features of Logi JReport Server are restricted by [licenses](https://devnet.logianalytics.com/hc/en-us/articles/1500009648262-Logi-JReport-Licenses). While using Logi JReport Server, you may need to update the server license with a new one enabled with more features. This can be easily done on the Logi JReport Administration page.

**To update the server license:**

1. On the Logi JReport Administration page, select **Configuration** on the system toolbar and then select **License Key** from the drop-down menu.
2. In the prompted dialog, enter the valid user ID and license key received from your Jinfonet Software account manager.

   ![Update Server License](https://devnet.logianalytics.com/hc/article_attachments/4404920651799/config_license.gif)
3. Select the **Submit** button to apply the new license key.
4. After successful update you will be prompt with the new access information of your Logi JReport Server.

In the case when your server license expires, after the server is started, Logi JReport will also prompt you to update your license. After the license information is updated successfully, you will be able to continue using the server.
