# Appendices Logi JReport Server v16

Logi JReport Server v16 Help System has the following appendices:

* [Appendix 1: URL Properties](https://devnet.logianalytics.com/hc/en-us/articles/1500009686482-Appendix-1-URL-Properties)
* [Appendix 2: Properties in server.properties](https://devnet.logianalytics.com/hc/en-us/articles/1500009712261-Appendix-2-Properties-in-server-properties)

The purpose of these appendices is to make the use of Logi JReport Server much easier and more convenient by introducing you to the types of formula functions and operators, as well as the parameter values required for building user defined web actions.
