# Profile Control and Customization

Logi Report Server Profile enables you to customize the UI, formats, styles, specific features, and preferences of Server. In the Server Console > Administration > Server Profile page, administrators can define some initial settings of Server and customize some features by configuring the profile. Any user can also configure his own profile in the Server Console > My Profile page. The difference lies in that any settings specified in the My Profile page can only be applied to this user, while the settings specified by administrators in the Server Profile page will be applied to all users.

In the following example, we edit the profile to change the date and time format for the Server web pages, and customize the menus and panels for the Edit Mode of Web Report Studio.
