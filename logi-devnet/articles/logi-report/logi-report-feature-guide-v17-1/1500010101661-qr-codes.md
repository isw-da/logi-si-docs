# QR Codes

QR Code is a 2D barcode which takes up less space but provides richer and more complex information as compared to a barcode. You can use QR code to encrypt the values of a data field as well as label text in a page report that is created using query resources.

The following example shows regular barcode and QR Code and use QR Code to represent the home page of Logi Analytics.
