# One-Select to Drill Down Report Data

The Drill Down feature enables you to drill data to lower groups according to predefined hierarchies. For example, if you have a hierarchical relationship among the following data groups predefined in a business view: Region > Country > State > City, when you create a report using one of the groups (not City since it is the lowest level) in the business view, you can select a region to drill down to the countries in the region, then select a country name to further go down to the states in the country, and then select a state to view the cities in the state. The one-selecting drill only goes down to the nearest lower level by each selecting.
