# Track 5: Publishing, Running, and Administering Resources

In this track, it is assumed that you are a Server system administrator. You need to receive the reports the report developers designed in Designer and publish them to Server so that end users can access and view them.

In this track, you perform the following lessons:

* [Lesson 1: Starting Server](https://devnet.logianalytics.com/hc/en-us/articles/5741379895063-Lesson-1-Starting-Server)

* [Lesson 3: Running Reports](https://devnet.logianalytics.com/hc/en-us/articles/5741398599319-Lesson-3-Running-Reports)
* [Lesson 4: Scheduling Reports](https://devnet.logianalytics.com/hc/en-us/articles/5741384783639-Lesson-4-Scheduling-Reports)
* [Lesson 5: Administration of Server](https://devnet.logianalytics.com/hc/en-us/articles/5741393283991-Lesson-5-Administration-of-Logi-Report-Server)
* [Lesson 6: Security](https://devnet.logianalytics.com/hc/en-us/articles/5741361990295-Lesson-6-Security)
