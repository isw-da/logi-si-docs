# Using an LDAP Server's Security System

The Logi JReport Server security system can run two modes in which you can use an LDAP server's security system: importing mode and non-importing mode. Below diagram illustrates these two working modes:

![](https://devnet.logianalytics.com/hc/article_attachments/4404926659607/scrty_rsc_srv_ldap.gif)

In the importing mode, if you want to use the LDAP feature, you will have to import the security information from an LDAP server into the built-in security system (red line). With the non-importing mode, Logi JReport Server can access an LDAP server and obtain LDAP security information directly using the LDAP implementation of the Security API (blue line).
