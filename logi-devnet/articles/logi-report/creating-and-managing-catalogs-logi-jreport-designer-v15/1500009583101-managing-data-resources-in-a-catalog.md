# Managing Data Resources in a Catalog

The [Catalog Manager](https://devnet.logianalytics.com/hc/en-us/articles/1500009593041-Catalog-Manager) provides a hierarchical view of the data resources in the current catalog. You can use it to add, remove, or modify the data resources in a catalog.

![Catalog Manager](https://devnet.logianalytics.com/hc/article_attachments/4404893800599/rptenv_ctlgmngr.gif)

The following topics describe tasks related to managing the data resources in a catalog:

> * [Adding, Editing and Modifying Resources](https://devnet.logianalytics.com/hc/en-us/articles/1500009583121-Adding-Editing-and-Modifying-Resources)
> * [Setting Resource Properties](https://devnet.logianalytics.com/hc/en-us/articles/1500009583181-Setting-Resource-Properties)
> * [Searching for Resources](https://devnet.logianalytics.com/hc/en-us/articles/1500009583201-Searching-for-Resources)
> * [Filtering Resources](https://devnet.logianalytics.com/hc/en-us/articles/1500009562782-Filtering-Resources)
> * [Using a Reference Table to Clarify Resource Relationships](https://devnet.logianalytics.com/hc/en-us/articles/1500009583141-Using-a-Reference-Table-to-Clarify-Resource-Relationships)
> * [Defining the Join Relationships Between Resources](https://devnet.logianalytics.com/hc/en-us/articles/1500009583161-Defining-the-Join-Relationships-Between-Resources)
