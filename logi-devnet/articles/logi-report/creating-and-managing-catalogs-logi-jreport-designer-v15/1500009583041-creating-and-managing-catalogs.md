# Creating and Managing Catalogs

A catalog is a report repository. Every report must exist within the context of a catalog folder. A catalog folder contains two primary types of files: a physical file that stores data resources and object definitions used by the reports in the catalog, and the report files themselves. These two types of data, the report files and the catalog file, are stored in the same directory. In addition, any other resources such as images used by the reports are also stored in the catalog folder.

Logi JReport Designer comes with four catalog folders: JinfonetGourmetJava, SampleComponents, SampleReports and TutorialReports. The catalog files in the JinfonetGourmetJava and TutorialReports folders are used for the Logi JReport Tutorial manual. You can get samples from any of them to learn Logi JReport Designer.
