# Drilling Through Page Report Data

In a page report running in Page Report Studio, you can choose to show certain groups of records and switch among the groups to see the data you want. This topic describes the automatic drilling and going features in Page Report Studio.

* [Automatic Drilling Page Report Data](https://devnet.logianalytics.com/hc/en-us/articles/4405690649367-Automatic-Drilling-Page-Report-Data)  
   Automatic drilling enables you to switch from the current group to another group using system-defined commands on the shortcut menu, and is divided into four kinds: drill-to, drill-to-by-value, drill-down and drill-up. Automatic drilling is only available on data components that are based on a business view.
* [Going Through Page Report Data](https://devnet.logianalytics.com/hc/en-us/articles/4405683954071-Going-Through-Page-Report-Data)  
   Going enables you to switch among the groups in a banded object and contains the go-to, go-up, go-down, and go-to-detail sub types. Going actions are available only for banded objects that contain groups and have not been converted to use a business view.
