# Chart Zoom via Range Selection

For 2-D chart of Bar, Bench, Line, Area, or Stock type, Logi Report enables you to zoom in the values you are interested in to make them shown more clearly and filter out unnecessary data. You can easily return to the initial status of the chart with a single selecting.
