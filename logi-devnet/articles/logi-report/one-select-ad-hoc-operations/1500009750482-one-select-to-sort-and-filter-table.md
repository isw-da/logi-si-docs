# One-Select to Sort and Filter Table

You can easily sort or filter the table records by selecting the Sort or Filter icon on the table column headers. This is called One-Select Sort/Filter.

The One-Select Sort feature is enabled by default. To use One-Select Filter, you need to edit the Server profile first.
