# Logi JReport Server Guide v16 Overview

Logi JReport Server is a 100% Java report generation and management tool that can be deployed to any Java EE application server. By leveraging its high-performance report generation engine, Logi JReport scales to meet the most demanding requirements. Using the report scheduling, distributing, and alerting capabilities of Logi JReport Server, reporting can be integrated into the workflow of the application.

Logi JReport Server also supports on-demand, live report creation and modification, providing Logi JReport’s powerful ad hoc solution. End users are empowered with Logi JReport to create their own richly visual and interactive reports for powerful and secure data exploration in a completely self-service manner.

The Logi JReport Server Guide v16 Help System provides you with information you need to know to fully utilize the features of Logi JReport Server v16. You can access the information by looking at the Table of Contents, or by using Search.
