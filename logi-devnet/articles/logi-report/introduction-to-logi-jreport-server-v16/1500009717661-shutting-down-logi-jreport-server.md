# Shutting Down Logi JReport Server

This topic introduces how to shut down Logi JReport Server under different circumstances.

**To shut down Logi JReport Server normally:**

* In a standalone environment, select **Stop Server** in the JReport folder on the Start menu; or point to the button ![Shut Down the Server button](https://devnet.logianalytics.com/hc/article_attachments/4412112565399/btn_srv_shtdwn.gif) at the top right of the server console and select **Shut Down Server** from the drop-down menu.
* In an integrated environment, shut down the application server according to the vendor's instructions.

Also, Logi JReport provides a feature for handling an abnormal system exit that enables the program to close itself gracefully when the Java virtual machine (JVM) is terminated in response to a user interrupt, such as typing ^C, or a system-wide event such as user logoff or system shutdown.
