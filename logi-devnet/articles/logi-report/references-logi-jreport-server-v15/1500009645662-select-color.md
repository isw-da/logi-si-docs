# Select Color

The Select Color dialog helps you to specify a color. It contains the following tabs:

* [Swatches](#Swatches)
* [Color Picker](#ColorPicker)

**OK**

Closes this dialog and returns the color value specified in the current tab.

**Cancel**

Closes this dialog without changing the color value.

**Help**

Displays the help document about this feature.

[![](https://devnet.logianalytics.com/hc/article_attachments/4404920354583/totop.png)Back to top](#top "Click to go to the top")

## Swatches

This tab allows you to select one from the web safe color swatches. [See the tab](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920614039/slctcolor_swtch.gif)

**Web Safe Colors**

Lists the web safe colors for you to select one.

**Transparent**

Select this swatch to delete the color and specify the object to be transparent.

**Color indicator**

The upper part will be displayed in the color you selected from Web Safe Colors and labeled the hexadecimal value of the color (or "Transparent" if you select the Transparent swatch). The lower part is always displayed in the original color. Selecting the lower part will restore the color to the original.

[![](https://devnet.logianalytics.com/hc/article_attachments/4404920354583/totop.png)Back to top](#top "Click to go to the top")

## Color Picker

This tab allows you to customize the color in Page Report Studio. [See the tab](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920614423/slctcolor_pick.gif)

**Select Color**

Displays a color matrix. To define a color, select anywhere in the matrix. Change saturation by moving the pointer horizontally, and change brightness by moving the pointer vertically.

**Color slider**

Drag the slider to change the hue.

**Color indicator**

The upper part will be displayed in the color you have newly defined. The lower part is always displayed in the original color. Selecting the lower part will restore the color to the original.

**R**

Specifies the amount of red in a color.

**G**

Specifies the amount of green in a color.

**B**

Specifies the amount of blue in a color.

**Color value**

Specifies the hexadecimal value of the color.
