# To Crosstab

The To Crosstab dialog helps you to specify settings for [converting a chart into a crosstab](https://devnet.logianalytics.com/hc/en-us/articles/1500009649422-Manipulating-Data-Components#ToCrosstab). It contains the following tabs:

* [Display](#Display)
* [Style](#Style)

**Back**

Returns to the previous tab.

**Next**

Goes to the next tab.

**OK**

Applies the settings and closes the dialog.

**Cancel**

Cancels the settings and closes the dialog.

**Help**

Displays the help document about this feature.

[![](https://devnet.logianalytics.com/hc/article_attachments/4404920354583/totop.png)Back to top](#top "Click to go to the top")

## Display

This tab allows you to set the data fields that you want to display in the crosstab. [See the tab](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404926679447/tocrstb_dsply.gif)

**Resources**

Displays the view elements used in the chart.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920379543/btn_additem.gif)

Adds the selected view element to the crosstab.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920418583/btn_rmvitem.gif)

Removes the selected view element.

**Columns/Rows**

Lists the group objects ![](https://devnet.logianalytics.com/hc/article_attachments/4404920356887/btn_bvgrp.gif) that will be displayed on the columns/rows of the crosstab.

* **Field**  
   Lists the group objects that you selected to display in the crosstab.
* **Display Name**  
   Lists the display name for the selected group objects. You can edit the names if required.
* **Sort**  
   Specifies how the selected group objects will be sorted.

**Summaries**

Lists the aggregation objects ![](https://devnet.logianalytics.com/hc/article_attachments/4404926627351/btn_bvaggrgtn.gif) that will be the aggregate fields of the crosstab.

* **Field**  
   Lists the aggregation objects that you selected to display in the crosstab.
* **Display Name**  
   Lists the display name for the selected aggregation objects. You can edit the names if required.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926639255/btn_mvup.gif)

Moves the selected view element one step up.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920381207/btn_mvdown.gif)

Moves the selected view element one step down.

## Style

This tab allows you to select a style for the crosstab. It is hidden when there is only one style available. [See the tab](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920601495/tocrstb_style.gif)

**Style**

Lists all the styles for you to select one.

* **Custom**  
   There is no style information in this style and it is only used to support reports built with previous versions which did not bind any style or the bound style cannot be found in the style list.

**Preview**

Shows a preview of the selected style.

**Inherit Style**

Specifies to take the style of the parent component. The option is available when the chart is in a table or banded object.
