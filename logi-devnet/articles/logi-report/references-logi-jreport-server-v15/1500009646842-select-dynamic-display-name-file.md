# Select Dynamic Display Name File

The Select Dynamic Display Name File dialog appears when you select the Import link in the Dynamic Display Names column on the Logi JReport Administration > Configuration > Dynamic Display Names page. It helps you to import [dynamic display names](https://devnet.logianalytics.com/hc/en-us/articles/1500009648822-Managing-Dynamic-Display-Names-of-Business-View-Elements) from a local resource bundle file. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404926734615/slctdynfl.gif)

**Select File**

Specifies the file from which to import the dynamic display names.

Select the **Browse** button to select the local resource bundle file that defines the dynamic display names of the business view elements in a catalog.

**OK**

Imports the dynamic display names from the selected file and exits the dialog.

**Cancel**

Cancels the importation of dynamic display names and closes the dialog.

**Help**

Displays the help document about this feature.
