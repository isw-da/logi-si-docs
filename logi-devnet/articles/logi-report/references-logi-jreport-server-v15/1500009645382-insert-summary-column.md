# Insert Summary Column

The Insert Summary Column dialog helps you to [insert a new summary column](https://devnet.logianalytics.com/hc/en-us/articles/1500009649422-Manipulating-Data-Components#InsertColumn) into a table. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404926678167/insrtsumclmn.gif)

**Resources**

Lists all the available data resources for the new summary column. Select the resource you want from the list.

**OK**

Inserts the column with the specified resource and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
