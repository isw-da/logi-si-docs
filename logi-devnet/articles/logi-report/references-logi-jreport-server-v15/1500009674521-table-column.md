# Table Column

There are four types of columns in a table: common column, detail column, group column and summary column. These table column objects have the same properties as follows:

| Property Name | Description |
| --- | --- |
| General | | | | |
| Display Name | Specifies the display name of the object. Data type: String |
| Instance Name | Shows the instance name of the object. This property is read only. |
| Others | | | | |
| Auto Fit | Specifies whether to adjust the width of the object according to the contents. Data type: Boolean |
| Invisible | Specifies whether to show or hide the object. All formulas and calculations will still be performed if the property is set to true. Data type: Boolean |
