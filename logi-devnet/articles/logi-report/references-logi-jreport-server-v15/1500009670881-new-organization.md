# New Organization

The New Organization dialog appears after you select the New Organization link on the Logi JReport Administration > Configuration > Organizations page. It is used for creating an [organization](https://devnet.logianalytics.com/hc/en-us/articles/1500009675081-Multi-tenancy-Supported-via-Organizations). [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404926655639/nwognz.gif)

The options available in this dialog are as follows:

**Organization Name**

Specifies the name of the organization. The system wide organization name is *System*; therefore, it is reserved. The organization names should be distinct. A valid name can contain only '0' to '9', 'a' to 'z', 'A' to 'Z', '&', '-', '\_', '.' and blank and no more than 32 characters.

**Max Number of Users**

Specifies the maximum number of users allowed in the organization. The default value *unlimited* means that the number is not limited. You can select a value from the drop-down list or input an integer number in the combo box directly. The **x** button in the combo box is used to clear the input text.

**Description**

Provides information about the organization.

**OK**

Creates the organization and exits the dialog.

**Cancel**

Does not create an organization and closes the dialog.

**Help**

Displays the help document about this feature.
