# Select Folder

The Select Folder dialog helps you to select a folder in the server resource tree that contains the required resource. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920559511/slctfldr.gif)

**Resource tree box**

The box lists the folder resources in the server resource tree. Navigate to the required folder to select it.

**OK**

Selects the folder and closes the dialog.

**Cancel**

Cancels the selection of a folder and closes the dialog.

**Help**

Displays the help document about this feature.
