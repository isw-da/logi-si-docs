# Select Language Source

The Select Language Source dialog appears when you select the button ![](https://devnet.logianalytics.com/hc/article_attachments/4404920385047/btn_add1.gif) above the Language box in the [Global NLS](https://devnet.logianalytics.com/hc/en-us/articles/1500009670801-Global-NLS) dialog, [Resource Tree NLS](https://devnet.logianalytics.com/hc/en-us/articles/1500009671161-Resource-Tree-NLS) dialog or the [NLS Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009646522-NLS-Editor) when editing NLS for a catalog. It helps you to choose where to add new languages. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920559127/slctlngsc.gif)

**Languages Supported by Logi JReport**

Specifies to add new languages from the Available Languages box. After choosing the option and selecting OK, you will be shown the [Add Language](https://devnet.logianalytics.com/hc/en-us/articles/1500009646362-Add-Language) dialog.

**NLS Resource File**

Specifies to add new languages from NLS resource files in your local folder. After choosing the option and selecting OK, you can browse to the local folder where the NLS resource files are located.

NLS resource files should follow the naming rule: NLS\_[language]\_[region A2]\_[User-Defined].properties. The language argument is a valid ISO Language Code as defined by ISO-639. You can find a full list of these codes at a number of sites, for example: <http://www.loc.gov/standards/iso639-2/php/code_list.php>. The region argument is a valid ISO Country Code as defined by ISO-3166. You can find a full list of these codes at a number of sites, for example: <http://www.chemie.fu-berlin.de/diverse/doc/ISO_3166.html>.

**OK**

Goes to the next action.

**Cancel**

Cancels the action and exits the dialog.

**Help**

Displays the help document about this feature.
