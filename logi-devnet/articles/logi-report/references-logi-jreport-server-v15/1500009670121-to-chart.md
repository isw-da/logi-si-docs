# To Chart

The To Chart dialog helps you to specify settings for [converting a crosstab into a chart](https://devnet.logianalytics.com/hc/en-us/articles/1500009649422-Manipulating-Data-Components#ToChart). It contains the following tabs:

* [Chart Type](#ChartType)
* [Display](#Display)
* [Style](#Style)

**Back**

Returns to the previous tab.

**Next**

Goes to the next tab.

**OK**

Applies the settings and closes the dialog.

**Cancel**

Cancels the settings and closes the dialog.

**Help**

Displays the help document about this feature.

[![](https://devnet.logianalytics.com/hc/article_attachments/4404920354583/totop.png)Back to top](#top "Click to go to the top")

## Chart Type

This tab allows you to set the type of the chart. [See the tab](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920455191/tochart_type.gif)

**Chart Type**

Lists all chart types for you to select one.

**Subtype**

Shows subtypes of the selected chart type in thumbnail form.

**Chart Type Groups**

Lists the subtypes defined for the chart.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920361623/btn_delete.gif)

Removes the selected subtype.

[![](https://devnet.logianalytics.com/hc/article_attachments/4404920354583/totop.png)Back to top](#top "Click to go to the top")

## Display

This tab allows you to set the fields that will be displayed in the chart. [See the tab](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920455703/tochart_dsply.gif)

**Resources**

Displays all the view elements used in the crosstab.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920379543/btn_additem.gif)

Adds the selected group or aggregation object to be displayed in the chart.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920418583/btn_rmvitem.gif)

Removes the selected view element.

**Category**

Lists the group object ![](https://devnet.logianalytics.com/hc/article_attachments/4404920356887/btn_bvgrp.gif) that will be displayed on the category axis of the chart.

**Series**

Lists the group object ![](https://devnet.logianalytics.com/hc/article_attachments/4404920356887/btn_bvgrp.gif) that will be displayed on the series axis of the chart.

**Show Values**

Lists the aggregation objects ![](https://devnet.logianalytics.com/hc/article_attachments/4404926627351/btn_bvaggrgtn.gif) that will be displayed on the value axis of the chart.

**Order/Select N**

Opens the [Order/Select N](https://devnet.logianalytics.com/hc/en-us/articles/1500009645502-Order-Select-N) dialog to define the sort order and Select N condition in the chart.

[![](https://devnet.logianalytics.com/hc/article_attachments/4404920354583/totop.png)Back to top](#top "Click to go to the top")

## Style

This tab allows you to select a style for the chart. It is hidden when there is only one style available. [See the tab](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920601751/tochart_style.gif)

**Style**

Lists all the styles for you to select from.

* **Custom**  
   There is no style information in this style and it is only used to support reports built with previous versions which did not bind any style or the bound style cannot be found in the style list.

**Preview**

Shows a preview of the selected style.

**Inherit Style**

Specifies to take the style of the parent component. The option is available when the crosstab is in a banded object.
