# Insert Parameter Control

The Insert Parameter Control dialog helps you to [insert a parameter control](https://devnet.logianalytics.com/hc/en-us/articles/1500009674041-Applying-Web-Controls#Parameter) into a report for delivering parameter values to the report. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920447639/insrtprmctrl.gif)

**Select a Parameter**

Specifies a parameter from the list where all the parameters except cascading parameters used by the current report are displayed.

**OK**

Inserts a parameter control into the report and closes the dialog.

**Cancel**

Cancels the operation and closes the dialog.

**Help**

Displays the help document about this feature.
