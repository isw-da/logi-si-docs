# Select Target

The Select Target dialog is used for specifying the target resources you would like the implementation of the [Dashboard Listener API](https://devnet.logianalytics.com/hc/en-us/articles/1500009644002-Applying-the-Implementations-of-the-Dashboard-Listener-API) to apply to. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920558487/slcttgt.gif)

**All Dashboards**

If selected, the implementation will apply to all dashboards including the newly created ones that have not been saved yet.

**Customize**

If selected, you can specify the dashboards you want the implementation to apply to.

* **Newly Created Dashboards**  
   If selected, the implementation will apply to all newly created dashboards that have not been saved yet.
* **Existing Dashboards**  
   If selected, you can further specify one or more folders, and the implementation will apply to all of the dashboards in the selected folders.

**OK**

Applies the setting and exits the dialog.

**Cancel**

Cancels the setting and closes the dialog.

**Help**

Displays the help document about this feature.
