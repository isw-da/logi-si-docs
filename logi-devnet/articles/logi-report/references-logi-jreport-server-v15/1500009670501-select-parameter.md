# Select Parameter

The Select Parameter dialog appears when you select the Add Parameter button in the [Insert URL Frame](https://devnet.logianalytics.com/hc/en-us/articles/1500009646162-Insert-URL-Frame) dialog or [Edit URL Frame](https://devnet.logianalytics.com/hc/en-us/articles/1500009646002-Edit-URL-Frame) dialog if a data source has already been specified, or select a data source and select OK in the [Select Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009646262-Select-Data-Source) dialog. It helps you to [insert a parameter into the URL of the web page](https://devnet.logianalytics.com/hc/en-us/articles/1500009644702-Inserting-Components#URL) that you want to add to a dashboard. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404926749719/slctprmtr.gif)

**Parameters**

Lists all the available parameters in the currently selected data source for you to select one to insert to the URL.

**Change Data Source**

Opens the [Select Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009646262-Select-Data-Source) dialog to change the data source.

**OK**

Closes this dialog and inserts the selected parameter to the URL.

**Cancel**

Cancels the insertion and closes this dialog.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920501783/btn_help.gif)

Displays the help document about this feature.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926700311/btn_close.gif)

Ignores the setting and closes this dialog.
