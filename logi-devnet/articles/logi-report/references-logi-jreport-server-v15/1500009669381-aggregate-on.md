# Aggregate On

The Aggregate On dialog helps you to [summarize data in the selected table detail column](https://devnet.logianalytics.com/hc/en-us/articles/1500009649422-Manipulating-Data-Components#AggregateOn). [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920462615/agrgton.gif)

**Aggregate On**

Displays the field bound with the detail column. This field is also the one on which the summary is based.

**Function**

Specifies the function to summarize the field in the detail column. For details about each function, see [Math Functions](https://devnet.logianalytics.com/hc/en-us/articles/1500009648102-Math-).

**OK**

Summarizes the detail column data with the specified function and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
