# Edit Aggregation

The Edit Aggregation dialog helps you to [edit an existing aggregation object](https://devnet.logianalytics.com/hc/en-us/articles/1500009649362-Using-Dynamic-Resources#Agg). [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920636567/edtmsr.gif)

**Aggregation Name**

Specifies the display name of the aggregation object.

**Mapping Name**

Specifies the mapping name of the field to which the aggregation object is mapped. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404920385815/btn_chsr.gif) to select the field in the [Select Resource](https://devnet.logianalytics.com/hc/en-us/articles/1500009670001-Select-Resource) dialog.

This option and the button are disabled when you specify to create an aggregation object on a dynamic formula.

**Aggregate Function**

Specifies the aggregate function of the aggregation object. The function will be applied according to the group where the aggregation is placed or the entire report if placed in the report header or footer. For details about each function, see [Math Functions](https://devnet.logianalytics.com/hc/en-us/articles/1500009648102-Math-).

**OK**

Applies the changes to the aggregation object and closes the dialog.

**Cancel**

Cancels the operation and closes the dialog.

**Help**

Displays the help document about this feature.
