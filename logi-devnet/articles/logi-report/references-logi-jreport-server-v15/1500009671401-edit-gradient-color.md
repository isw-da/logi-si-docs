# Edit Gradient Color

The Edit Gradient Color dialog helps you to specify gradient colors for the members of the aggregation field. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920557335/edtgrdclr.gif)

**Pattern**

After selecting a pattern from the drop-down list, a preview of the gradient color of the pattern is provided below the pattern name.

The gradient start color and end color can be customized. Select the leftmost or the rightmost arrow to select a color in the color palette or select **More Colors** to access the [Color Picker](https://devnet.logianalytics.com/hc/en-us/articles/1500009646922-Color-Picker) dialog in which you can select a color within a wider range. After you change a color, the pattern name will be changed to Customize.

**Reversed**

Reverses the direction of the gradient color.

**OK**

Closes this dialog and returns the specified color.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926732567/btn_va_help.gif)

Displays the help document about this feature.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920361623/btn_delete.gif)

Ignores the setting and closes this dialog.
