# New Group

The New Group dialog appears when you select the New Group link on the Logi JReport Administration > Security > Group page. It helps you to create a [group](https://devnet.logianalytics.com/hc/en-us/articles/1500009674961-Managing-Groups). [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404926740375/newgroup.gif)

**Group Name**

Specifies the name for the new group.

**Parent Group Name**

Specifies the parent group for the new group.

**Description**

Specifies the description for the new group.

**Privileges**

Specifies whether the new group has the following privileges:

* **Publish**  
   The privilege of publishing resources to Logi JReport Server.
* **Advanced Properties**  
   The privilege of viewing advanced version properties information, such as catalog connections and report related resources.

**OK**

Creates a group with the information you have specified.

**Cancel**

Cancels operations and closes the dialog.
