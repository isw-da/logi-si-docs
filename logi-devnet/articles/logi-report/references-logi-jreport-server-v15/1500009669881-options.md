# Options

The Options dialog helps you to [set the user interface of Page Report Studio](https://devnet.logianalytics.com/hc/en-us/articles/1500009673961-General-Operations#Option). It contains the following tabs:

* [Option](#Option)
* [Customize](#Customize)

**Restore Defaults**

Restores the default settings.

**OK**

Applies the settings and closes the Options dialog.

**Cancel**

Cancels the settings and closes the Options dialog.

**Help**

Displays the help document about this feature.

[![](https://devnet.logianalytics.com/hc/article_attachments/4404920354583/totop.png)Back to top](#top "Click to go to the top")

## Option

This tab allows you to specify the skin of Page Report Studio user interface. [See the tab](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920463767/optn_optn.gif)

**Skin**

Specifies the skin of the Page Report Studio user interface. The skin can be the Standard, Classical, or Windows XP style.

[![](https://devnet.logianalytics.com/hc/article_attachments/4404920354583/totop.png)Back to top](#top "Click to go to the top")

## Customize

This tab allows you to customize the toolbars in Page Report Studio. [See the tab](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920464023/optn_cstmz.gif)

**Current Toolbar**

Lists all the available toolbars. If you want to customize a toolbar, select it to select.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920385047/btn_add1.gif)

Displays the New Toolbar Name dialog for you to input the name for the new toolbar. The new toolbar name will then be displayed in the Current Toolbar list box.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920361623/btn_delete.gif)

Removes the selected toolbar.

**Available Tools**

Displays the tools that you can add to the toolbars. The tools available in the box are determined by [which Page Report Studio feature profile is applied](https://devnet.logianalytics.com/hc/en-us/articles/1500009674101-Customizing-Page-Report-Studio-Profile).

**Selected Tools**

Displays the tools of the toolbar selected in the Current Toolbar list box in the order that they will appear (from left to the right) on the toolbar.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920379543/btn_additem.gif)

Adds the selected tool from the Available Tools list box to the Selected Tools list box.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920418583/btn_rmvitem.gif)

Removes the selected tool.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920620311/btn_pgrpt_rmvall.gif)

Empties the contents of the Selected Tools list box.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926765591/btn_pgrpt_addall.gif)

Adds all the available tools from the Available Tools list box to the Selected Tools list box.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926639255/btn_mvup.gif)

Adjusts the location of the selected tool by moving it to the left on the toolbar.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920381207/btn_mvdown.gif)

Adjusts the location of the selected tool by moving it to the right on the toolbar.
