# Insert Image

The Insert Image dialog appears when you drag Image from Toolbox in the Resources panel to the dashboard header, or select the Insert Image button ![](https://devnet.logianalytics.com/hc/article_attachments/4404920589591/btn_dshbrd_img.gif) in the [Insert HTML](https://devnet.logianalytics.com/hc/en-us/articles/1500009670321-Insert-HTML) dialog or [Edit HTML](https://devnet.logianalytics.com/hc/en-us/articles/1500009670261-Edit-HTML) dialog. It helps you to [insert an image](https://devnet.logianalytics.com/hc/en-us/articles/1500009644702-Inserting-Components#Image) into the dashboard header or into the HTML component. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920589847/insrtimg.gif)

**Image From**

Specifies the source of the image file.

* **Local File**  
   Specifies to use an image from the local file system.
  + **File Name**  
     Specifies the path and name of the image file. You can select the Browse button to locate the image file.
* **Web URL**  
   Specifies to use an image via URL.
  + **File URL**  
     Specifies the URL of the image file.

  **Note:** If your Logi JReport Server is in an intranet which requires a proxy, to successfully access the image via URL, you need to add the parameters `-Dhttp.proxyHost=XXX -Dhttp.proxyPort=XX` to the server's startup file JRServer.bat, which is located in `<install_root>\bin`.
* **Library**  
   Specifies to use an existing image.
  + **My Pictures**  
     The My Pictures folder is a virtual location where Logi JReport Server stores the images that have once been inserted into JDashboard. Select the one you want to use.

**Preview**

Displays a preview of the selected image.

**OK**

Closes this dialog and inserts the selected image in the dashboard header.

**Cancel**

Cancels the insertion and closes this dialog.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920501783/btn_help.gif)

Displays the help document about this feature.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926700311/btn_close.gif)

Ignores the setting and closes this dialog.
