# Edit Color

The Edit Color dialog helps you to specify colors for the members of group fields. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404926733591/edtclr.gif)

**Pattern**

After selecting a pattern from the drop-down list, a preview of the colors of the pattern is provided below the pattern name.

**Data Items**

Lists the combination of members of the bound fields with the colors in the selected pattern.

To customize the color of a data item, take either method of the following:

* Select the data item and then select a color in the pattern preview box.
* Double-select the data item, then in the displayed color palette select a color or select **More Colors** to access the [Color Picker](https://devnet.logianalytics.com/hc/en-us/articles/1500009646922-Color-Picker) dialog in which you can select a color within a wider range or input a color string in the format #RRGGBB.

**Opacity**

Specifies the opacity of the color for the selected member combination in the Data Items box.

**OK**

Closes this dialog and returns the specified color.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926732567/btn_va_help.gif)

![](https://devnet.logianalytics.com/hc/article_attachments/4404920361623/btn_delete.gif)
