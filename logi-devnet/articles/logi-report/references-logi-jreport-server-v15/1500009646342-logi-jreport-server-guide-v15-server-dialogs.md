# Logi JReport Server Dialogs

There are two classes of dialog in Logi JReport Server and they are distinguished by the types of users who can gain access to them. The Logi JReport Console page dialogs are available to general users, while the Logi JReport Administration page dialogs are only accessible to users who have been assigned the administrator's role.
