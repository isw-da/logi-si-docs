# Split Cell

The Split Cell dialog appears when you select a tabular cell and select Menu > Format > Split. It helps you to specify the number of rows and columns that the specified tabular cell will be split into. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404926703511/splitcel.gif)

**Split Cell Into**

Specifies the number of rows and columns to split the selected tabular cell into.

* **Number of Rows**  
   Specifies the number of rows the tabular cell will be split into. Input a positive integer in the text box to specify the number.
* **Number of Columns**  
   Specifies the number of columns the tabular cell will be split into. Input a positive integer in the text box to specify the number.

**OK**

Closes this dialog and splits the tabular cell.

**Cancel**

Cancels to split the cell and closes this dialog.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920501783/btn_help.gif)

Displays the help document about this feature.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926700311/btn_close.gif)
