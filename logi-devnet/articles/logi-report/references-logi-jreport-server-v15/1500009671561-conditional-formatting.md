# Conditional Formatting

The Conditional Formatting dialog appears when you right-click a field and select Conditional Formatting from the shortcut menu. It helps you to add some [conditional formats](https://devnet.logianalytics.com/hc/en-us/articles/1500009675181-Adding-Conditional-Formats-to-Fields) to values of the selected field. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920401687/cndtnlfmt.gif)

**Condition**

Displays all the conditions you have already added.

* ![](https://devnet.logianalytics.com/hc/article_attachments/4404920385047/btn_add1.gif)  
   Adds a new condition in the [Edit Conditions](https://devnet.logianalytics.com/hc/en-us/articles/1500009647222-Edit-Conditions) dialog.
* ![](https://devnet.logianalytics.com/hc/article_attachments/4404920401943/btn_pgrpt_edit.gif)  
   Edits the selected condition.
* ![](https://devnet.logianalytics.com/hc/article_attachments/4404920361623/btn_delete.gif)  
   Removes the selected condition.
* **Priority**  
   Specifies the priority of each condition.
  + **![](https://devnet.logianalytics.com/hc/article_attachments/4404926651671/btn_mvup1.gif)**  
     Moves a condition up for a higher priority.
  + **![](https://devnet.logianalytics.com/hc/article_attachments/4404920383255/btn_mvdown1.gif)**  
     Moves a condition down for a lower priority.

**Format**

Specifies the format which will be applied to the field values when the specified condition is fulfilled.

* **Font**  
   Specifies the font type for the field values.
* **Border**  
   Specifies the border line style for the field values.
* **Size**  
   Specifies the font size for the field values.
* **Bold**  
   Specifies whether or not to bold the field values.
* **Italic**  
  Specifies whether or not to make the field values italic.
* **Underline**  
   Specifies whether or not to underline the field values.
* **Foreground Color**   
   Specifies the foreground color for the field values.
* **Background Color**   
   Specifies the background color for the field values.
* **Sample Text**  
   Displays a preview sample of your settings.

**OK**

Applies the settings and exits the dialog.

**Cancel**

Cancels the settings and closes the dialog.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920501783/btn_help.gif)

Displays the help document about this feature.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926700311/btn_close.gif)

Ignores the setting and closes this dialog.
