# Open

The Open dialog helps you to open another analysis template in a new Visual Analysis window. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920556823/open.gif)

**Open In**

Browse to the directory in the server resource tree where the target analysis template is located and then select it. Use the button ![](https://devnet.logianalytics.com/hc/article_attachments/4404926673687/btn_dshbrd_up.gif) to go to the parent folder.

The resource table shows the analysis templates in the current directory.

**File Name**

Shows the selected analysis template file name.

**OK**

Closes this dialog and opens the selected analysis template.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926732567/btn_va_help.gif)

Displays the help document about this feature.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920361623/btn_delete.gif)

Ignores the setting and closes this dialog.
