# Zoom

The Zoom dialog helps you to set a magnification of the report page. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404926759447/zoom.gif)

**Zoom to**

Specifies the magnification in percentage.

**OK**

Closes this dialog and zooms the report page in or out to the magnification.

**Cancel**

Closes this dialog without changing the magnification of the report page.

**Help**

Displays the help document about this feature.
