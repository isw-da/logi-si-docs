# New Custom Field

The New Custom Field dialog appears when you select the New Custom Field link on the Logi JReport Administration > Configuration > Custom Fields page. It is used to create [user-defined fields](https://devnet.logianalytics.com/hc/en-us/articles/1500009648802-Working-with-Custom-Fields) which can be used as resource properties. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920490263/nwcstmfld.gif)

**Custom Field Name**

Specifies the name for the custom field.

**Field Description**

Specifies a description for the custom field.

**Enabled**

Specifies whether to enable the custom field. The enabled custom fields are available for showing in the resource properties.

**OK**

Creates the custom field and exits the dialog.

**Cancel**

Cancels operations and closes the dialog.

**Help**

Displays the help document about this feature.
