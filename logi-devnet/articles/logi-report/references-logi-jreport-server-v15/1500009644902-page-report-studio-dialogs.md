**Note:** The object properties dialogs may be different for an [advanced user](https://devnet.logianalytics.com/hc/en-us/articles/1500009674101-Customizing-Page-Report-Studio-Profile#AdvUser) and for a normal user. An advanced user can modify some properties that a common user cannot, and can use formulas to control properties. The documents for properties dialogs in this section are based on advanced user.

**To become an advanced user:**

1. Log onto the Logi JReport Administration page, select **Profile** on the system toolbar and select **Customize Profile** from the drop-down menu.
2. **Go to the Page Report Studio Properties**> > Advanced tab and check the **Advanced User** option.
3. Select **OK** to accept the change.
