# Select Resource

The Select Resource dialog helps you to specify the field to which the [dynamic aggregation object](https://devnet.logianalytics.com/hc/en-us/articles/1500009649362-Using-Dynamic-Resources#Agg) will be mapped. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920468887/slctrsc.gif)

In this dialog, all the group objects ![](https://devnet.logianalytics.com/hc/article_attachments/4404920356887/btn_bvgrp.gif) and detail objects ![](https://devnet.logianalytics.com/hc/article_attachments/4404920356631/btn_bvdtl.gif) in the current business view and the dynamic formulas ![](https://devnet.logianalytics.com/hc/article_attachments/4404920613271/btn_pgrpt_dynfml.gif) that have been created in the report will be listed. Select the required field and then select OK to make it as the mapping field of the aggregation object.

**OK**

Applies the settings and closes this dialog.

**Cancel**

Closes the dialog.

**Help**

Displays the help document about this feature.
