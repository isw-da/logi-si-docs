# Filter Inspector

The Filter Inspector dialog appears when you select the Inspector button in the [Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500009671821-Filter) dialog. It lists all the filters the current web report is using for your information. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404926725783/fltrinspct.gif)

**Query Filter**

Lists the filters created via the [Query Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500009650282-Applying-Filters#BV) dialog.

**Dialog Filter**

Lists the filters created via the [Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500009650282-Applying-Filters#Dialog) dialog.

**On-screen Filter**

Lists the filters created via the [Filter panel](https://devnet.logianalytics.com/hc/en-us/articles/1500009650282-Applying-Filters#Panel) and via [filter controls](https://devnet.logianalytics.com/hc/en-us/articles/1500009650282-Applying-Filters#Control).

**Go To Filter**

Lists all the go to filters that are created when performing the [go-to-by-value action](https://devnet.logianalytics.com/hc/en-us/articles/1500009650302-Going-Through-the-Report-Data#Go-to-by-value) on the report.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Cancels the changes and exits the dialog.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920501783/btn_help.gif)

Displays the help document about this feature.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926700311/btn_close.gif)

Ignores the setting and closes this dialog.
