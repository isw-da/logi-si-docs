# Select Business View Elements

The Select Business View Elements dialog appears when you select the Select Business View Elements link in the [New Dynamic Display Name](https://devnet.logianalytics.com/hc/en-us/articles/1500009646622-New-Dynamic-Display-Name) dialog. It helps you to select business view elements in the specified catalog for which to define the [dynamic display names](https://devnet.logianalytics.com/hc/en-us/articles/1500009648822-Managing-Dynamic-Display-Names-of-Business-View-Elements). [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920489751/slctbvelmnt.gif)

**Catalog**

Displays the current catalog with the resource path.

**Select Business View Elements**

The panel lists all the business views and their elements in the catalog. You can select multiple elements by selecting the corresponding checkboxes.

**All**

Selects/Deselects all the business view elements in the catalog.

**OK**

Applies the selection and exits the dialog.

**Cancel**

Cancels the selection and closes the dialog.

**Help**

Displays the help document about this feature.
