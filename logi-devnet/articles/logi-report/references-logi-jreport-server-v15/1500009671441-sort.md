# Sort

The Sort dialog helps you to sort the selected field. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920555927/sort.gif)

**Sort Order**

It works when Customize is not selected.

* **Ascending**  
   Sorts the field in the ascending order. If Sort Using Another Field is specified, it sorts the specified field in the ascending order.
* **Descending**  
   Sorts the field in the descending order. If Sort Using Another Field is specified, it sorts the specified field in the descending order.
* **Sort Using Another Field**  
   If checked, you can select an aggregation field from the drop-down list that follows to perfom the sort on.

**Customize**

Adjusts the order of the members of the current field manually. The field members are listed in the text box below. Select a member and then select the proper arrow button to move it up or down.

**OK**

Closes this dialog and applies the sorting order.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926732567/btn_va_help.gif)

Displays the help document about this feature.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920361623/btn_delete.gif)

Ignores the setting and closes this dialog.
