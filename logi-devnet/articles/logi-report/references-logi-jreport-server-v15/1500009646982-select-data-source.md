# Select Data Source

The Select Data Source dialog helps you to select a business view for performing visual analysis based on. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920556183/slctdtsrc.gif)

The available business views are listed for selection. The available business views come from the current catalog in the current directory in the server resource tree. If the current catalog contains no business view or if it is the server root directory, all available business views will be listed.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926732951/btn_va_slctdtsrc_trvw.gif)

Displays the available business views in a tree structure.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926733207/btn_va_slctdtsrc_fltvw.gif)

Displays the available business views as list.

**OK**

Closes this dialog and opens Visual Analysis.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926732567/btn_va_help.gif)

Displays the help document about this feature.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920361623/btn_delete.gif)

Ignores the setting and closes this dialog.
