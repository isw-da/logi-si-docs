# Select Data Source

The Select Data Source dialog appears when you select the Add Parameter button in the [Insert URL Frame](https://devnet.logianalytics.com/hc/en-us/articles/1500009646162-Insert-URL-Frame) dialog or [Edit URL Frame](https://devnet.logianalytics.com/hc/en-us/articles/1500009646002-Edit-URL-Frame) dialog if no data source has been specified, or select the Change Data Source button in the [Select Parameter](https://devnet.logianalytics.com/hc/en-us/articles/1500009670501-Select-Parameter) dialog. It helps you to select the data source that contains the parameter you want to [insert to a URL](https://devnet.logianalytics.com/hc/en-us/articles/1500009644702-Inserting-Components#URL). [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920582679/slctdtsrc.gif)

**Folder**

Specifies the folder containing the required data source.

**Catalog**

Displays the catalog containing
the required data source.

**Data Source**

Displays the selected data source.

**OK**

Selects the data source and closes this dialog.

**Cancel**

Cancels the selection and closes this dialog.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920501783/btn_help.gif)

Displays the help document about this feature.

![](https://devnet.logianalytics.com/hc/article_attachments/4404926700311/btn_close.gif)

Ignores the setting and closes this dialog.
