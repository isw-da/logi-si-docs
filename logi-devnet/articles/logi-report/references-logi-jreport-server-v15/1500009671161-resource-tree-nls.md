# Resource Tree NLS

The Resource Tree NLS dialog appears when you select Resources > Resource Tree NLS on the Logi JReport Administration page. It helps you to [edit NLS for resources](https://devnet.logianalytics.com/hc/en-us/articles/1500009649222-Editing-Resource-Tree-NLS) in the resource tree. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404920484247/rscnls.gif)

**Language**

Specifies the languages into which the resources will be translated.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920385047/btn_add1.gif)

Opens the [Select Language Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009646862-Select-Language-Source) dialog to choose where to add a language.

![](https://devnet.logianalytics.com/hc/article_attachments/4404920361623/btn_delete.gif)

Removes the selected language from the Language box.

**Path**

Shows current path with links to parent folders.

**Search**

Searches for the Name and Translation items that match the search condition in the Search text box. Here you are not allowed to search wildcards. The results will be listed in the table below.

**Name**

Lists the resource names in the original language.

**Translation**

Specifies to which the resource names will be translated in the target language.

**Export**

Specifies to export the resource tree NLS map to a resource tree NLS file in the specified location for further use.

Note that only the resource tree of one translated language can be exported at a time and a user can only use the resource tree NLS file exported by himself.

**OK**

Applies the settings and closes the dialog.

**Cancel**

Cancels the settings and exits the dialog.

**Help**

Displays the help document about this feature.
