# Working with Banded Objects

A banded object is a type of component that can present grouped data and detail data. A banded object contains several banded panels with which you can easily organize data fields and other components, for example, you can embed additional data components inside a banded object and add data container links between the data components and the banded object. This topic introduces the features of banded objects and how you can make use of them to design the banded objects that best serve your reporting requirements.

Designer does not support banded objects in library components.

The following topics discuss banded objects in further detail:

* [Inserting Banded Objects in a Report](https://devnet.logianalytics.com/hc/en-us/articles/1500010094301-Inserting-a-Banded-Object)
* [Modifying Banded Objects](https://devnet.logianalytics.com/hc/en-us/articles/1500010094321-Modifying-Banded-Objects)
* [Embedding Data Components in a Banded Object](https://devnet.logianalytics.com/hc/en-us/articles/1500010056742-Embedding-Data-Components-in-a-Banded-Object)
* [Obtaining Detailed Information from a Banded Object](https://devnet.logianalytics.com/hc/en-us/articles/1500010056722-Obtaining-Detailed-Information-from-a-Banded-Object)
* [Setting Up Data Container Links in Banded Objects](https://devnet.logianalytics.com/hc/en-us/articles/1500010056782-Setting-Up-Data-Container-Links-in-Banded-Objects)
