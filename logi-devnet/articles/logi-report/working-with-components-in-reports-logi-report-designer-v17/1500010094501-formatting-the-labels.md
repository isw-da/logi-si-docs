# Formatting the Labels

This topic describes how you can format the labels (including titles) of a chart to make them more meaningful.

In Designer, you can add as many labels as required into a chart. To add a label to a chart, right-click any chart element and select **Add Label** from the shortcut menu. Designer then adds a label with the text "Label" at the upper left corner of the chart platform. Drag and drop it to the desired position in the chart. To remove a label, right-click it and select **Remove**.

To format a label in a chart, take the following steps:

1. Double-click the label, or right-click the label and select **Format Label** from the shortcut menu. Designer displays the [Format Label dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010097121-Format-Label-Dialog-Box).

   ![Format Label dialog box - General](https://devnet.logianalytics.com/hc/article_attachments/4404848551447/fmtlabel_gnrl.gif)
2. In the **General** tab, specify the general properties of the label

   In the **Text** box, type the text to display in the label.

   In the **Icon** box, select an icon style from the **Style** drop-down list to attach an icon to the label, then specify the distance between the icon and the label text, the color and color transparency to fill the icon, and the width and height of the icon.

   In the **Alignment** box, specify the alignment of the text in the label, and the alignment of the icon relative to the text in the label.
3. In the **Fill** tab, specify the color and transparency of the color to fill the label (to change the color, select the color indicator and select a color from the [color palette,](https://devnet.logianalytics.com/hc/en-us/articles/1500010062922-Menu-Tabs#Palette) or type the hexadecimal value of a color in the text box).

   ![Format Label dialog box - Fill](https://devnet.logianalytics.com/hc/article_attachments/4404856949399/fmtlabel_fill.gif)
4. In the **Border** tab, specify the properties for the border of the label.

   ![Format Label dialog box - Border](https://devnet.logianalytics.com/hc/article_attachments/4404848551831/fmtlabel_brdr.gif)

   Specify the type, color, color transparency, line style, thickness, end caps style, and line joint mode of the border.

   In the **Path** box, specify the fill pattern of the border: Outline Path or Fill Path.

   In the **Dash** box, select to automatically adjust the dash size or use fixed dash size if you select a dash line style for the border.
5. In the **Font** tab, specify the font properties of the label, including the font face, size, color, color transparency, rotation angle, shearing angle, and the font effects such as style, strikethrough, and underline.

   ![Format Label dialog box - Font](https://devnet.logianalytics.com/hc/article_attachments/4404848552087/fmtlabel_font.gif)
6. In the **Orientation** tab, specify the rotation angle for the label text and the icon. You can either drag the **Text** hand in the **Orientation** box or type a value in the **Angle** text box to perform the operation.

   ![Format Label dialog box - Orientation](https://devnet.logianalytics.com/hc/article_attachments/4404856949911/fmtlabel_orntatn.gif)
7. Select **OK** to accept the changes and close the dialog box.
