# Images

An image is a digital representation of a picture. Logi Report supports the following image formats: .bmp, .gif, .jpg, and .png. This topic describes how to use the image component.

Images in a report can be used as the trigger object of [links](https://devnet.logianalytics.com/hc/en-us/articles/1500009613322-Adding-Links-in-Reports).

**To insert an image in a report:**

1. Position the mouse pointer at the destination where you want to insert the image. An image can be inserted in the report areas listed in [Component Placement](https://devnet.logianalytics.com/hc/en-us/articles/1500009605422-Working-with-Components-in-Reports#Relationship).
2. Do any of the following.
   * Drag ![Image button](https://devnet.logianalytics.com/hc/article_attachments/4404904436375/btn_instimg.gif) from the Components panel.
   * Select **Insert** > **Image**.
   * Select **Home** > **Insert** > **Image**.
3. In the Select an Image dialog, select the image you want to use.
   * If you want to use an image from your local file system, use the Look in drop-down list to browse for the image, then select **Open**. If the image file is not in the current catalog directory, you will be prompted to copy it to the directory.
   * If you want to use an image via URL, type the URL in the URL text box and then select **OK**.
4. Select **Open** to insert the image into the destination.

**Notes:**

* If want to use an image via URL and you are in an intranet, to successfully access the image, you need to add the parameters `-DproxyHost=XXX -DproxyPort=XX` to the batch file JReport.bat, which is located in `<install_root>\bin`.
* If you insert an image via URL into a report, when the report is saved, only the URL information will be saved into the report template.

**See an example:** The SampleComponents catalog, included with Logi Report Designer, contains reports that have examples of how each component type could be used in a report. For the image component example, open `<install_root>\Demo\Reports\SampleComponents\ForImage.cls`.
