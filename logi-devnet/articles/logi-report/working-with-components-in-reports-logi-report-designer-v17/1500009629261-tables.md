# Tables

Tables give you great control over how to present data, including placing fields, grouping them, and sorting them. A table is composed of rows and columns, and each contains several cells. With such a structure a table is a good way to show any two-dimensional dataset.

The following topics discuss tables in further detail:

* [Inserting Tables in a Report](https://devnet.logianalytics.com/hc/en-us/articles/1500009629281-Inserting-Tables-in-a-Report-)
* [Modifying Tables](https://devnet.logianalytics.com/hc/en-us/articles/1500009629301-Modifying-Tables)
* [Grouping the Data in Tables](https://devnet.logianalytics.com/hc/en-us/articles/1500009606502-Grouping-the-Data-in-Tables)
* [Using Customized Controls in Tables](https://devnet.logianalytics.com/hc/en-us/articles/1500009606462-Using-Customized-Controls-in-Tables)

**See an example:** The SampleComponents catalog, included with Logi Report Designer, contains reports that have examples of how each component type could be used in a report. For the table component example, open `<install_root>\Demo\Reports\SampleComponents\TableReport.cls`.
