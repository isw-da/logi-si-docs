# Formatting the Gridlines

This topic introduces how to customize the gridlines of a chart. Note that pie, indicator, heat map and organization charts do not have gridlines.

1. Right-click any chart element and select the desired entry from the Format Gridlines submenu to bring up the corresponding format gridline dialog.

   ![Format Category (X) Gridline dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904323863/fmtctgrygrdln.gif)
2. Specify the line color, style, transparency, and thickness of the gridline (to change the color, select the color indicator and select a color from the [color palette,](https://devnet.logianalytics.com/hc/en-us/articles/1500009613582-Menu-Tabs#Palette) or type the hexadecimal value of a color in the text box).
3. Select **OK** to accept the changes and close the dialog.

The following topics provide more information about options in each dialog:

* [Format Category (X) Gridline](https://devnet.logianalytics.com/hc/en-us/articles/1500009608402-Format-Category-X-Gridline-Dialog)
