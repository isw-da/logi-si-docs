# Formatting the Paper

This topic describes how you can to format the paper of a chart.

1. Right-click any chart element and select **Format Paper** from the shortcut menu, or double-click the paper of the chart. Designer displays the [Format Paper dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010059262-Format-Paper-Dialog-Box).
2. Set the scaling ratios or rotation angles for the chart axes in the paper.
   * If the chart is of 2-D type, in the **General** tab, drag the slider to scale up or down the display size of the chart along the X or Y axis, or input the scaling ratio in the text boxes. Select **Constrain Proportion** to apply the same scaling ratios for both the X and Y axes.

     ![Format Paper dialog box - General](https://devnet.logianalytics.com/hc/article_attachments/4404848545943/fmtpapr_gnrl.gif)
   * If the chart is of 3-D type, in the **Rotation** tab, set the rotation angle for the chart, and specify the scaling ratios for the X, Y, and Z axes accordingly. You can either select the icon ![No Rotation button](https://devnet.logianalytics.com/hc/article_attachments/4404856942487/btn_plot1.gif), ![Angle X button](https://devnet.logianalytics.com/hc/article_attachments/4404856942871/btn_plot2.gif), ![Angle Y button](https://devnet.logianalytics.com/hc/article_attachments/4404856943127/btn_plot3.gif), or ![Angle Z button](https://devnet.logianalytics.com/hc/article_attachments/4404856943639/btn_plot4.gif), and then drag the chart thumbnail in the preview box to rotate the chart around the origin point, X axis, Z axis, or Y axis, or adjust the degrees in the corresponding text boxes.
     Select **Constrain Proportion** to set the same scaling ratios for the X, Y, and Z axes.

     ![Format Paper dialog box - Rotate](https://devnet.logianalytics.com/hc/article_attachments/4404856942231/fmtpapr_rotate.gif)

     ![Note icon](https://devnet.logianalytics.com/hc/article_attachments/4404856814359/noteicon.jpg) You can rotate a 3-D chart in another intuitive way:

     1. Right-click anywhere in the chart and select **Rotation Mode** from the shortcut menu.
     2. Hold the mouse button and move the pointer to the desired position, then release the mouse button to confirm.

        To quit the rotation mode, right-click anywhere in the chart and select **Editing Mode** on the shortcut menu.
3. In the **Fill** tab, specify the color and transparency of the color to fill the paper (to change the color, select the color indicator and select a color from the [color palette,](https://devnet.logianalytics.com/hc/en-us/articles/1500010062922-Menu-Tabs#Palette) or type the hexadecimal value of a color in the text box).

   ![Format Paper dialog box - Fill](https://devnet.logianalytics.com/hc/article_attachments/4404848546455/fmtpapr_fill.gif)
4. In the **Border** tab, specify the properties for the border of the paper.

   ![Format Paper dialog box - Border](https://devnet.logianalytics.com/hc/article_attachments/4404856944023/fmtpapr_brdr.gif)

   Specify the type, color, color transparency, line style, thickness, end caps style, and line joint mode of the border.

   In the **Path** box, specify the fill pattern of the border: Outline Path or Fill Path.

   In the **Dash** box, select to automatically adjust the dash size or use fixed dash size if you select a dash line style for the border.
5. Select **OK** to apply the settings and close the dialog box.
