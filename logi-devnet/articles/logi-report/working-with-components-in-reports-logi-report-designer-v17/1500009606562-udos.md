# UDOs

A user defined object (UDO) is an object inherited from Logi Report system's object. You can add and modify new properties, as well as other existing methods and specify new methods for modifying your UDO. UDOs, which have been appended to the ADDCLASSPATH variable of setenv.bat in `<install_root>\bin`, can be inserted as components into a report and consequently become a part of the report.

UDOs are only supported in page reports that are created using query resources.

The following topics discuss UDOs in further detail:

* [Creating UDOs](https://devnet.logianalytics.com/hc/en-us/articles/1500009606582-Creating-UDOs)
* [Inserting UDOs into a Page Report](https://devnet.logianalytics.com/hc/en-us/articles/1500009629361-Inserting-UDOs-into-a-Page-Report)
