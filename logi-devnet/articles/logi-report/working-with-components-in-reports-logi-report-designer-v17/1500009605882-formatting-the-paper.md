# Formatting the Paper

The paper of a chart is the canvas on which the main chart is drawn. This topic introduces how to format the paper.

1. Right-click any chart element and select **Format Paper** from the shortcut menu, or double-click the paper of the chart. The [Format Paper](https://devnet.logianalytics.com/hc/en-us/articles/1500009608502-Format-Paper-Dialog) dialog appears.
2. Set the scaling ratios or rotation angles for axes in the paper.
   * If the chart is of 2-D type, in the General tab, drag the slider to scale up or down the display size of the chart along the X or Y axis. Select the **Constrain Proportion** option to set the same scaling ratios for both X and Y axes.

     ![Format Paper dialog - General](https://devnet.logianalytics.com/hc/article_attachments/4404904302231/fmtpapr_gnrl.gif)
   * If the chart is of 3-D type, in the Rotate tab, set the rotation angle for the chart, and specify the scaling ratios for the X, Y and Z axes accordingly. You can either select ![No Rotation button](https://devnet.logianalytics.com/hc/article_attachments/4404904302743/btn_plot1.gif), ![Angle X button](https://devnet.logianalytics.com/hc/article_attachments/4404904302999/btn_plot2.gif), ![Angle Y button](https://devnet.logianalytics.com/hc/article_attachments/4404904303255/btn_plot3.gif), or ![Angle Z button](https://devnet.logianalytics.com/hc/article_attachments/4404904303511/btn_plot4.gif), and then drag the chart thumbnail in the preview box to rotate the chart around the origin point, X axis, Z axis, or Y axis, or adjust the degrees directly in the spinners.

     ![Format Paper dialog - Rotate](https://devnet.logianalytics.com/hc/article_attachments/4404904302487/fmtpapr_rotate.gif)

     **Tip:** You can rotate a 3-D chart in another intuitive way as explained below:

     1. Right-click anywhere in the chart and select **Rotation Mode** from the shortcut menu.
     2. Hold the mouse button and move the pointer to the desired position, then release the mouse button to confirm.

        To quit the rotation mode, right-click anywhere in the chart and select **Editing Mode** on the shortcut menu.
3. In the Fill tab, set the color and transparency of the color for the paper (to change the color, select the color indicator and select a color from the [color palette,](https://devnet.logianalytics.com/hc/en-us/articles/1500009613582-Menu-Tabs#Palette) or type the hexadecimal value of a color in the text box).

   ![Format Paper dialog - Fill](https://devnet.logianalytics.com/hc/article_attachments/4404904303767/fmtpapr_fill.gif)
4. In the Border tab, set the border mode for the paper, including the border style, color, transparency, line style, thickness, end caps style, and line joint mode, and if you specify the line joint mode to be joint round, you can set the the radius for the border joint. You can also specify whether the border is to be outlined, and whether or not to automatically resize the border dashes if the border is drawn with dashes.

   ![Format Paper dialog - Border](https://devnet.logianalytics.com/hc/article_attachments/4404904304151/fmtpapr_brdr.gif)
5. Select **OK** to apply the settings and close the dialog.
