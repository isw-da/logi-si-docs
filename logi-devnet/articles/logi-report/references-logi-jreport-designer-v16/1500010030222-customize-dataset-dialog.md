# Customize Dataset Dialog

The Customize Dataset dialog helps you to [customize the datasets](https://devnet.logianalytics.com/hc/en-us/articles/1500010070881-Managing-Page-Reports#Import) used by components of the to be imported report tabs. It appears when you check the Customize Dataset for Components option and select the OK button in the [Select Report Tab](https://devnet.logianalytics.com/hc/en-us/articles/1500010032622-Select-Report-Tab-Dialog) dialog.

![Customize Dataset dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901131031/cstmzdtst.gif)

The following are details about options in this dialog:

**Report Tab Name**

Displays the name of the report tab in which the component is.

**Component Name**

Displays the name of the component.

**Component Type**

Displays the type of the component.

**Dataset Name**

Lists the datasets you can choose for the component.

**Description**

Displays the description of the component.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
