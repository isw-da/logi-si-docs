# Add Formula Fields Dialog

The Add Formula Fields dialog helps you to [add formulas into a query to be columns](https://devnet.logianalytics.com/hc/en-us/articles/1500010034642-Creating-Queries-in-a-Catalog#FormulaField) of the query. It appears when you select Menu > Column > Add Formula Fields in the [Query Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500010067501-Query-Editor-Dialog).

![Add Formula Fields dialog](https://devnet.logianalytics.com/hc/article_attachments/4404909132951/addfmlfld.gif)

The following are details about options in this dialog:

**Formulas**

Lists all the formulas available to be added to the query.

**Expression**

Shows expressions of the selected formula.

**Add**

Adds the selected formula to the query.

**Close**

Exits the dialog.

**Help**

Displays the help document about this feature.
