# Insert Parameter Control Dialog

The Insert Parameter Control dialog helps you to [insert a parameter control](https://devnet.logianalytics.com/hc/en-us/articles/1500010064061-Using-Advanced-Web-Controls#Param) into a report for delivering parameter values. It appears when you select Insert > Web Controls > Parameter Control, or drag the Parameter Control button ![Parameter Control](https://devnet.logianalytics.com/hc/article_attachments/4404901203351/btn_paramcntrl.gif) from the Components panel to a report.

![Insert Parameter Control dialog](https://devnet.logianalytics.com/hc/article_attachments/4404909181079/instprmtrctrl.gif)

The followings are details about options of this dialog:

**Select a Parameter**

Lists all the parameters except cascading parameters used in the current report. Specfiy the one you want to use in the parameter control.

**OK**

Inserts the parameter control into the report and closes the dialog.

**Cancel**

Cancels the operation and closes the dialog.

**Help**

Displays the help document about this feature.
