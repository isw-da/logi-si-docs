# Manage Customized Controls Dialog

The Managed Customized Controls dialog helps you to [manage customized controls](https://devnet.logianalytics.com/hc/en-us/articles/1500010029282-Using-Customized-Controls-in-Tables#Manage). It appears when you select Report > Manage Customized Control, or select Customized Control from Lib and select OK in the [Web Action List](https://devnet.logianalytics.com/hc/en-us/articles/1500010068321-Web-Action-List-Dialog) dialog.

![Manage Customized Controls dialog](https://devnet.logianalytics.com/hc/article_attachments/4404909170327/mngcstmzdctrl.gif)

The following are details about options in the dialog:

**Customized Control Files**

Lists all the customized control files in the customized control library.

right-click a customized control file and the following options are available on the shortcut menu:

* **Duplicate**  
   Creates a copy of the selected file.
* **Edit**  
   Edit the customized control in the [Customized Control - Web Action Builder](https://devnet.logianalytics.com/hc/en-us/articles/1500010030182-Customized-Control-Web-Action-Builder-Dialog) dialog.
* **Delete**  
   Removes the file from the library.
* **Rename**  
   Modify the name of the file and then select outside of the name area to accept the change.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404901125271/btn_add.gif)

Displays the [New Customized Control File](https://devnet.logianalytics.com/hc/en-us/articles/1500010067201-New-Customized-Control-File-Dialog) dialog for creating a new customized control file by naming the file and then defining the customized control.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404909120791/btn_rmv.gif)

Removes the selected customized control file from the library.

![Descending Sort button](https://devnet.logianalytics.com/hc/article_attachments/4404901197463/btn_sortcc2.gif)/![Ascending Sort button](https://devnet.logianalytics.com/hc/article_attachments/4404901197719/btn_sortcc1.gif)

Sorts the files according to their names ascending or descending. The descending order will not be remembered and whenever you access the dialog it is always the ascending order by default.

**OK**

* Loads the customized control definition from the selected file and exits the dialog if you are defining customized control. The button is enabled when a file is selected.
* Exits the dialog if you are managing customized controls.

**Cancel**

* Cancels selecting a customized control file and exits the dialog if you are defining customized control.
* Exits the dialog if you are managing customized controls.

**Help**

Displays the help document about this feature.
