# Flash Parameter Dialog

The Flash Parameter dialog helps you to [insert a flash object](https://devnet.logianalytics.com/hc/en-us/articles/1500010029162-Multimedia-Objects#Flash) (using the OLE mode) into a report. It appears when you select Insert > Multimedia Objects > Flash, or drag the Flash button ![Flash button](https://devnet.logianalytics.com/hc/article_attachments/4404901261079/btn_flash.gif) from the Components panel to a report.

![Flash Parameter dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901261335/flshprm.gif)

The following are details about options in the dialog:

**File Name/URL**

Specifies the URL of the plug-in page from which to download the player to play this flash file, if it isn't already installed on your local disk.

**Plug-in Page**

Specifies the URL of the plug-in page to play the flash file.

**Properties**

Specifies the properties for the flash file that is to be inserted into the report.

* **Play**  
   Specifies whether to play the flash file.
* **Loop**  
   Specifies whether to play the flash file repeatedly.
* **Movie Quality**  
   Specifies the quality for the flash file from the drop-down list.

**OK**

Accepts all changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
