# Go Down Dialog

The Go Down dialog helps you [define a Go Down web action](https://devnet.logianalytics.com/hc/en-us/articles/1500010034842-Delivering-Messages-Between-Library-Components-#UGoDown) for the specified data component in a library component as a response to the user defined message it receives at runtime. It appears when you select \*Go Down and select OK in the [Web Action List](https://devnet.logianalytics.com/hc/en-us/articles/1500010068321-Web-Action-List-Dialog) dialog.

![Go Down dialog](https://devnet.logianalytics.com/hc/article_attachments/4404909122711/godwn.gif)

The following are details about options in the dialog:

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404901125271/btn_add.gif)

Adds a new condition line. You can add one condition line at most.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404909120791/btn_rmv.gif)

Removes the condition.

**Go Down On**

Specifies the group on which the go-down action will be performed.

**Use Hierarchy**

Specifies the hierarchy that the go-down action will follow.

**By Value**

Specifies by which value of the specified group to perform the go-down action, that is to perform the go-down action with the filter condition *SpecifiedGroupField=TheValue*.

**OK**

Accepts the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
