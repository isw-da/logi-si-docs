# Edit Display Name Dialog

The Edit Display Name dialog helps you to [customize the display names of data fields](https://devnet.logianalytics.com/hc/en-us/articles/1500010070661-Creating-and-Managing-Datasets#Customize) in datasets created on query resources so as to make them easier to be identified at runtime. It appears when you select Report > Edit Display Name.

![Edit Display Name dialog](https://devnet.logianalytics.com/hc/article_attachments/4404909134743/edtdsplynm.gif)

The following are details about options in the dialog:

**Report Dataset**

Lists all the datasets in the current report.

**Auto Sort**

Specifies to sort the fields automatically.

**Resource Name**

Displays the names of fields in the selected dataset.

**Display Name**

Specifies the display names for fields in the selected dataset.

**Advanced**

Opens the [Edit Display Name for Component](https://devnet.logianalytics.com/hc/en-us/articles/1500010065581-Edit-Display-Name-for-Component-Dialog) dialog to configure advanced display name settings.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Cancels the changes and exits the dialog.

**Help**

Displays the help document about this feature.
