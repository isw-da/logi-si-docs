# Add Driver Dialog

The Add Driver dialog helps you to add the MySQL driver in order to [connect to the MySQL database via plugin](https://devnet.logianalytics.com/hc/en-us/articles/1500010029442-JDBC-Connection-Plugins). It appears when the MySQL driver cannot be found in the following cases:

* Select OK in the [Create Connection to MySQL](https://devnet.logianalytics.com/hc/en-us/articles/1500010030082-Create-Connection-Dialog) dialog.
* Select MySQL and select OK in the [New Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500010031882-New-Data-Source-Dialog) dialog.
* In the Catalog Manager, right-click a data source and select New JDBC Connection from the shortcut menu, then select MySQL in the Select Connection Type dialog and select OK.

![Add Driver dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901300503/adddrvr.gif)

The following are details about options in the dialog:

**Add**

Browses to select the MySQL driver jar file on your local disk.

**OK**

Loads the selected driver into the MySQL connection plugin and opens the [Connect to MySQL](https://devnet.logianalytics.com/hc/en-us/articles/1500010065021-Connect-to-MySQL-Dialog) dialog.

**Cancel**

Cancels the action and exits the dialog.

**Help**

Displays the help document about this feature.
