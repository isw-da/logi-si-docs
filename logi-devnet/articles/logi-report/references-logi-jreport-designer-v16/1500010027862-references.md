# References

Logi JReport Designer v16 Help System has two different types of references:

* [Dialog Reference](https://devnet.logianalytics.com/hc/en-us/articles/1500010064621-Dialog-Reference)
* [Object Property Reference](https://devnet.logianalytics.com/hc/en-us/articles/1500010033442-Object-Property-Reference)

The purpose of these references is to make the use of Logi JReport Designer much easier and more convenient by providing you with an overview of the functions and properties of objects.
