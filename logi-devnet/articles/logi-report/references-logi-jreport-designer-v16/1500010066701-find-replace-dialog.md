# Find/Replace Dialog

The Find/Replace dialog helps you to search for text in a formula or function, and replace the found text with different text. It appears when you select Menu > Edit > Search in the [Formula Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500010066181-Formula-Editor-Dialog) or [User Defined Function Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500010032902-User-Defined-Function-Editor-Dialog).

![Find/Replace dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901217687/fndrplc.gif)

The following are details about options in the dialog:

**Find**

Specifies what to find.

**Replace**

Specifies what to use to replace the found text.

**Options**

Specifies the finding limit.

* **Match case**  
   Limits the search to text that exactly matches the case of the text you want to find.
* **Match whole word**  
   Limits the search to text that matches one or more complete words.

**Find Next**

Jumps to and selects the next occurrence of the search words in the formula/function text.

**Find All**

Finds all occurrences of the search words in the formula/function text.

**Replace**

Replaces the found material one by one in the formula/function text.

**Replace All**

Replaces all of the found materials in the formula/function text.

**Close**

Closes the dialog.
