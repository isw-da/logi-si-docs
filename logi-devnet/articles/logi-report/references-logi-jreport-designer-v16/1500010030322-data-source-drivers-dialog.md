# Data Source Drivers Dialog

The Data Source Drivers dialog helps you to add or remove a data source driver [for cached query result](https://devnet.logianalytics.com/hc/en-us/articles/1500010034602-Cached-Query-Results#Manage) into or from a Logi JReport catalog. It appears when you select Data Source Drivers on the toolbar of Catalog Manager.

![Data Source Drivers dialog](https://devnet.logianalytics.com/hc/article_attachments/4404909133719/dtsrcdrvr.gif)

The following are details about options in this dialog:

**Data Source Drivers**

Lists all the available data source driver names.

**New**

Opens the [Data Source Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500010065381-Data-Source-Editor-Dialog) to add a new data source driver.

**Remove**

Removes the selected data source driver.

**Edit**

Edits the selected data source driver.

**OK**

Accepts changes and closes the dialog.

**Cancel**

Cancels the changes and closes the dialog.

**Help**

Displays the help document about this feature.
