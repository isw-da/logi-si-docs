# Time Zone and Locale Options Dialog

The Time Zone and Locale Options dialog helps you to specify the default time zone and locale for an XML instance. It appears when you select the Time Zone and Locale button in the [XML Connection Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500010033042-XML-Connection-Wizard-Dialog).

![Time Zone and Locale Options dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901172759/tmznlcl.gif)

The following are details about options in the dialog:

**Default Time Zone**

Specifies the default time zone, in order to apply it to the Time data type, which is not specified for them in the XML instance. If the time zone has been specified in the XML instance, the time zone you set here will be ignored.

**Locale**

Specifies the locale for the XML instance.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain changes and closes the dialog.

**Help**

Displays the help document about this feature.
