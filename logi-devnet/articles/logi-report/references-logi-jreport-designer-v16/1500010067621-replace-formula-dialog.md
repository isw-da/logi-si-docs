# Replace Formula Dialog

The Replace Formula dialog helps you to [Select another formula to replace the current one](https://devnet.logianalytics.com/hc/en-us/articles/1500010034642-Creating-Queries-in-a-Catalog#FormulaField). It appears when you double-click any formula added to be column of a query in the [Query Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500010067501-Query-Editor-Dialog) dialog.

![Replace Formula dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901184023/rplcfmla.gif)

The following are details about options in the dialog:

**Formulas**

Lists all the formulas available to be added as query columns.

**Expression**

Shows the expressions of the selected formula.

**Replace**

Replaces the current formula with the selected one.

**Close**

Closes the dialog.

**Help**

Displays the help document about this feature.
