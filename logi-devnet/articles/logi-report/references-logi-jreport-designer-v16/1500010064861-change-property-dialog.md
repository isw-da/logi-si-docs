# Change Property Dialog

The Change Property dialog helps you to define to change properties of the current data component as [a response to the message the data component receives](https://devnet.logianalytics.com/hc/en-us/articles/1500010034842-Delivering-Messages-Between-Library-Components-#UProperty) at runtime. It appears when you select \*Property and then select OK in the [Web Action List](https://devnet.logianalytics.com/hc/en-us/articles/1500010068321-Web-Action-List-Dialog) dialog.

![Change Property dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901127447/chgpty.gif)

The following are details about options in the dialog:

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404901125271/btn_add.gif)

Adds a new property line.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404909120791/btn_rmv.gif)

Removes the selected property line.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404909121047/btn_mvup.gif)

Moves the selected property line one step up.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404909121303/btn_mvdwn.gif)

Moves the selected property line one step down.

**Properties**

Specifies the property the value of which you want to change.

**Value**

Specifies the value of the property.

**OK**

Accepts the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
