# Choose Data Dialog

The Choose Data dialog helps you choose a dataset for a blank page report tab and differs for choosing a [query resource](#Query) or a [business view](#BV).

## For Choosing a Query Resource

When the dialog is opened via one of the following, it helps you to choose the dataset you want to use for  [the blank page report tab](https://devnet.logianalytics.com/hc/en-us/articles/1500010034782-Creating-Reports#Tab).

* In the [Select Component for Page Report](https://devnet.logianalytics.com/hc/en-us/articles/1500010032502-Select-Component-for-Page-Report-Dialog) dialog, select the Blank component and select OK.
* When creating a new report tab in a page report which uses query resource as the data source, in the [Select Component for Page Report Tab](https://devnet.logianalytics.com/hc/en-us/articles/1500010067781-Select-Component-for-Page-Report-Tab-Dialog) dialog, select the Blank component and select OK.

![Choose Data - query dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901137047/chsdt.gif)

The following are details about options in the dialog:

**Data resource box**

Lists the predefined data resources in the current catalog. Select one and a dataset based on it is created automatically for the page report tab.

**More Options/Less Options**

Shows or hides the dataset selection panel to choose a dataset for the report tab.

* **New Dataset**  
   Specifies to create a dataset from the current catalog data resources. When a query is selected, you can select the Edit button to edit the query in the [Query Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500010067501-Query-Editor-Dialog) if required.
* **Existing Dataset**  
   Specifies to use a dataset from the ones existing in the current page report. Select the Edit button to edit the dataset in the [Dataset Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500010030342-Dataset-Editor-Dialog) if required.

**OK**

Applies the specified dataset and closes this dialog.

**Cancel**

Does not retain changes and closes this dialog.

**Help**

Displays the help document about this feature.

## For Choosing a Business View

When the dialog is opened via one of the following, it helps you to choose a business view to bind to the [blank page report tab](https://devnet.logianalytics.com/hc/en-us/articles/1500010034782-Creating-Reports#Tab) or [KPI](https://devnet.logianalytics.com/hc/en-us/articles/1500010029082-KPIs).

* In the [Select Component for Page Report](https://devnet.logianalytics.com/hc/en-us/articles/1500010032502-Select-Component-for-Page-Report-Dialog) dialog, select Create Using Business View at bottom left, next select the Blank component, and then select OK.
* When creating a new report tab in a page report which uses business view as the data source, in the [Select Component for Page Report Tab](https://devnet.logianalytics.com/hc/en-us/articles/1500010067781-Select-Component-for-Page-Report-Tab-Dialog) dialog, select the Blank component and select OK.
* Drag ![KPI button](https://devnet.logianalytics.com/hc/article_attachments/4404901291415/btn_kpi.gif) from the Components box to the destination in an open web report or library component.

![Choose Data - business view dialog](https://devnet.logianalytics.com/hc/article_attachments/4404909244695/chsdt-bv.gif)

The following are details about options in the dialog:

**Business view box**

Lists all the predefined business views in the current catalog. Select the business view you want.

**OK**

Binds the selected business view to the target and closes this dialog.

**Cancel**

Cancels the binding of a business view and closes this dialog.

**Help**

Displays the help document about this feature.
