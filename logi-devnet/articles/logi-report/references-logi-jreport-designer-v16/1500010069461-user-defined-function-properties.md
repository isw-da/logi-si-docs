# User Defined Function Properties

This topic lists the properties of a User Defined Function object in a catalog.

| Property Name | Description |
| --- | --- |
| Expression | Displays the whole statement of the function. This property is read only. |
| Name | Specifies the name of the user defined function. Data type: String |
| [SQL Type](https://devnet.logianalytics.com/hc/en-us/articles/1500010069121-Formula-Properties#SQL) | Specifies the SQL type of the user defined function defined in Java. Choose an option from the drop-down list. Data type: Integer |
