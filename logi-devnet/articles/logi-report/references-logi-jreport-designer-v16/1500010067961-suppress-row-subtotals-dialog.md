# Suppress Row Subtotals Dialog

The Suppress Row Subtotals dialog helps you to customize which subtotals of the row fields in a crosstab will be suppressed, and which will be shown. It appears when you select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404909121559/btn_chsr1.gif) next to the Suppress Row Subtotals option in the Layout screen of the crosstab wizard.

![Suppress Row Subtotal dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901178519/sprsrowsbtl.gif)

The following are details about options in the dialog:

**Same for all**

If checked, the setting for the subtotal of the first row field in below box will be applied to all the subtotals of the row fields in the crosstab.

**Name**

Lists the row fields excluding the one on the topmost row group level; for a compound crosstab the row fields excluding the one on the topmost row group level in each row compound group.

**Suppress**

Specifies whether the subtotal for a row field will be suppressed. To specify the value, select in the value cell and select true or false from the drop-down list. The value can also be [controlled by a constant level formula](https://devnet.logianalytics.com/hc/en-us/articles/1500010033202-Using-Formulas-to-Control-Object-Properties).

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
