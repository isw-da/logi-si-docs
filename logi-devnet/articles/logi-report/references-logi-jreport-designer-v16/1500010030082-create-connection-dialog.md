# Create Connection Dialog

The Create Connection dialog helps you to create a connection to connect a catalog to one of these databases: [Oracle, SQL Server, MySQL, PostgreSQL, JDBC](https://devnet.logianalytics.com/hc/en-us/articles/1500010064301-JDBC-Connections), [JSON](https://devnet.logianalytics.com/hc/en-us/articles/1500010029482-JSON-Connections), [XML](https://devnet.logianalytics.com/hc/en-us/articles/1500010064601-Working-with-XML-Connections-in-a-Catalog), [SOAP Web Service](https://devnet.logianalytics.com/hc/en-us/articles/1500010064561-SOAP-Web-Service-Connections), [MongoDB](https://devnet.logianalytics.com/hc/en-us/articles/1500010029522-Working-with-MongoDB-Connections-in-a-Catalog) and [Hive](https://devnet.logianalytics.com/hc/en-us/articles/1500010064281-Hive-Connections). It appears when you select an item in the Connect category on the [Start Page](https://devnet.logianalytics.com/hc/en-us/articles/1500010071081-Start-Page-).

![Create Connection dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901281431/crtcnnctn.gif)

**Existing Catalog**

Specifies an existing catalog in the local disk.

* **Catalog**  
   Displays the current catalog. You can select another one by selecting ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404901122967/btn_chsr.gif).
* **Data Source**  
   Specifies the catalog data source in which to create the connection.

**New Catalog**

Creates a new catalog.

* **Directory**  
   Specifies the directory where to save the new catalog. You can input the directory manually in the text box or select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404901122967/btn_chsr.gif) to select it.
* **Catalog**  
   Specifies the name of the new catalog.

**OK**

Closes the dialog and opens one of the following dialogs:

* [Connect to Oracle](https://devnet.logianalytics.com/hc/en-us/articles/1500010030002-Connect-to-Oracle-Dialog)
* [Connect to MySQL](https://devnet.logianalytics.com/hc/en-us/articles/1500010065021-Connect-to-MySQL-Dialog)
* [Connect to SQL Server](https://devnet.logianalytics.com/hc/en-us/articles/1500010065061-Connect-to-SQL-Server-Dialog)
* [Connect to PostgreSQL](https://devnet.logianalytics.com/hc/en-us/articles/1500010065041-Connect-to-PostgreSQL-Dialog)
* [Get JDBC Connection Information](https://devnet.logianalytics.com/hc/en-us/articles/1500010031562-Get-JDBC-Connection-Information-Dialog)
* [JSON Connection Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500010031582-JSON-Connection-Wizard-Dialog)
* [XML Connection Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500010033042-XML-Connection-Wizard-Dialog)
* [SOAP Web Service Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500010068361-SOAP-Web-Service-Data-Source-Dialog)
* [MongoDB Connection Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500010031662-MongoDB-Connection-Wizard-Dialog)

**Cancel**

Cancels the connection creation and closes the dialog.

**Help**

Displays the help document about this feature.
