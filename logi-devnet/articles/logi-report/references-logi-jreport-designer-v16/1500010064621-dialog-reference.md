# Dialog Reference

This topic covers dialogs that you will come across when using Logi JReport Designer. It mainly provides description and explanation of the options available in these dialogs. The purpose is to make the use of Logi JReport Designer much easier and more convenient.

In Logi JReport Designer, the unit of the measurement properties like Width, Height, Thickness and so on is determined by the [Units](https://devnet.logianalytics.com/hc/en-us/articles/1500010031962-Options-Dialog#Display) setting in the Options dialog, which can either be inch or centimeter.
