# Enter Values Dialog

The Enter Values dialog helps you to [specify multiple values for a parameter](https://devnet.logianalytics.com/hc/en-us/articles/1500010033422-Specifying-Parameter-Values#Multiple). It appears when you select ![More button](https://devnet.logianalytics.com/hc/article_attachments/4404901122967/btn_chsr.gif) in the [Enter Parameter Values](https://devnet.logianalytics.com/hc/en-us/articles/1500010065841-Enter-Parameter-Values-Dialog) dialog.

![Enter Values dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901149335/entervlu.gif)

The following are details about options in the dialog:

**Available Values**

Lists all predefined parameter values for selection. When the parameter is bound with a column, but the display column is different from the bound column, values of the display column are listed here.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404909137431/btn_addarrow1.gif)

Adds the selected values from the Available Values box to the Selected Values box.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404909137943/btn_rmvarrow1.gif)

Removes the selected values from the Selected Values box.

![Add All button](https://devnet.logianalytics.com/hc/article_attachments/4404909137687/btn_addall1.gif)

Adds all the selected values from the Available Values box to the Selected Values box.

![Remove All button](https://devnet.logianalytics.com/hc/article_attachments/4404901149591/btn_rmvall1.gif)

Removes all the selected values from the Selected Values box.

**Selected Values**

Lists the values selected. The selected values are case sensitive.

**Search**

Available when the parameter's Allow Type-in of Value property is false. It is used to search for values among the available values.

**Enter Values**

Available when the parameter's Allow Type-in of Value property is true. It allows you to add values for the parameter manually.

* ![Calendar button](https://devnet.logianalytics.com/hc/article_attachments/4404909137175/btn_clndr.gif)  
   Opens the calendar to [specify a date and time value](https://devnet.logianalytics.com/hc/en-us/articles/1500010033422-Specifying-Parameter-Values#Date). Available only when the parameter is of the Date, DateTime, or Time type.

**All**

Available when the parameter's Enable the "All" Option property is true. It specifies whether to apply the "All" value to the parameter, which could be all values of the parameter determined by Logi JReport, all values in the database or all the available values according to the [Scope of All Values](https://devnet.logianalytics.com/hc/en-us/articles/1500010069001-Creating-Parameters-in-a-Catalog#AllScope) setting of the parameter.

**OK**

Closes the dialog and selects the specified values for the parameter.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
