# New Dataset Dialog

The New Dataset dialog helps you to [create a dataset](https://devnet.logianalytics.com/hc/en-us/articles/1500010070661-Creating-and-Managing-Datasets#Create) for a query based page report. It appears when you select the <New Dataset...> item in the Data tab of the report wizard, select the <Choose Data From...> item from the dataset drop-down list in the Data panel for a page report, or select the New button in the [Manage Datasets](https://devnet.logianalytics.com/hc/en-us/articles/1500010067121-Manage-Datasets-Dialog)  dialog.

![New Dataset dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901144471/nwdtst.gif)

The following are details about options in the dialog:

**Choose Data from**

Specifies the data resource on which to create the dataset. Data resources predefined in the current catalog are listed here. You can also select the first item in each data resource node to create your required data resource for the dataset.

**New Dataset Name**

Specifies the name of the dataset.

**OK**

Accepts changes and closes this dialog.

**Cancel**

Does not retain changes and closes this dialog.

**Help**

Displays the help document.
