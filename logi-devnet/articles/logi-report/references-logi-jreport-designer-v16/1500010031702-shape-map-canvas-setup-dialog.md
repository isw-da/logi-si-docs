# Shape Map Canvas Setup Dialog

The Shape Map Canvas Setup dialog helps you to change the width and height of a shape map object. It appears when you select Menu > File > Canvas Setup or Canvas Setup in the [Shape Map Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500010067141-Shape-Map-Editor-Dialog).

![Shape Map Canvas Setup dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901196695/mpcnvsstup.gif)

The following are details about options in the dialog:

**Width**

Specifies the width of the map object.

**Height**

Specifies the height of the map object.

**Automatic Filling Labels**

If checked, the label will not be displayed when the size of the area is changed to be smaller than the label.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
