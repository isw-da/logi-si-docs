# Add Role Dialog

The Add Role dialog helps you to add a role when editing a [security entry](https://devnet.logianalytics.com/hc/en-us/articles/1500010063241-Record-Level-and-Column-Level-Security#Data) or [business view security](https://devnet.logianalytics.com/hc/en-us/articles/1500010028582-Business-View-Security#Configure) so as to define its permission on different resources. It appears when you select ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404901125271/btn_add.gif) and then select Add Role from the drop-down menu in the [Security](https://devnet.logianalytics.com/hc/en-us/articles/1500010067681-Security-Dialog)dialog, [Edit Business View Security](https://devnet.logianalytics.com/hc/en-us/articles/1500010065501-Edit-Business-View-Security-Dialog) dialog or in the Security tab of the following dialogs: [Category Property](https://devnet.logianalytics.com/hc/en-us/articles/1500010065261-Category-Property-Dialog), [New View Element](https://devnet.logianalytics.com/hc/en-us/articles/1500010067281-New-View-Element-Dialog) or [Edit View Element](https://devnet.logianalytics.com/hc/en-us/articles/1500010065781-Edit-View-Element-Dialog).

The dialog contains two tabs: [General](#General) and [Relation](#Relation).

**OK**

Accepts all changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.

## General

Specifies the role information.

![Add Role dialog - General](https://devnet.logianalytics.com/hc/article_attachments/4404909254039/addrole_gen.gif)

**Role**

Specifies the name of the role.

**Source**

Displays the source of the role. It is read only.

**Parent Roles**

Displays the parent roles of the current role. It is read only.

**Description**

Specifies the description of the role.

## Relation

Specifies the relationship of the role and the existing users and groups.

![Add Role dialog - Relation](https://devnet.logianalytics.com/hc/article_attachments/4404901298455/addrole_rel.gif)

**Users**

All available users will be listed here. If a user is checked, the user will be added into the role.

**Groups**

All available groups will be listed here. If a group is checked, the group will be added into the role.
