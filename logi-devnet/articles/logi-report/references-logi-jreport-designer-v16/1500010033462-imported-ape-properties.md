# Imported APE Properties

This topic lists the properties of an Imported APE object in a catalog.

| Property Name | Description |
| --- | --- |
| Collection | Specifies the collection that you want to use to execute the aggregation pipeline expression. Data type: String |
| Connection Name | Specifies the connection to execute the aggregation pipeline expression. Data type: String |
| Database | Specifies the database that contains the selected collection and matches the aggregation pipeline expression. Data type: String |
| Description | Specifies the description of the object. Data type: String |
| Name | Specifies the name of the object. Data type: String |
