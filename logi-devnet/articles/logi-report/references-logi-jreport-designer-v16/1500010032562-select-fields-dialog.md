# Select Fields Dialog

The Select Fields dialog helps you to select the fields according to whose unique values to calculate DistinctSum. It appears when you do one of the following:

* Select DistinctSum from the Aggregate Function drop-down list and then select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404901122967/btn_chsr.gif) next to the Distinct On text box in the [New Summary](https://devnet.logianalytics.com/hc/en-us/articles/1500010031922-New-Summary-Dialog) dialog, [Edit Summary](https://devnet.logianalytics.com/hc/en-us/articles/1500010030602-Edit-Summary-Dialog) dialog, [Insert Summary](https://devnet.logianalytics.com/hc/en-us/articles/1500010031502-Insert-Summary-Dialog) dialog, [New View Element](https://devnet.logianalytics.com/hc/en-us/articles/1500010067281-New-View-Element-Dialog) dialog, [Edit View Element](https://devnet.logianalytics.com/hc/en-us/articles/1500010065781-Edit-View-Element-Dialog) dialog, [New Aggregation](https://devnet.logianalytics.com/hc/en-us/articles/1500010031782-New-Aggregation-Dialog) dialog, [Edit Aggregation](https://devnet.logianalytics.com/hc/en-us/articles/1500010065481-Edit-Aggregation-Dialog) dialog, [Insert Aggregation](https://devnet.logianalytics.com/hc/en-us/articles/1500010031422-Insert-Aggregation-Dialog) dialog or [Aggregate On](https://devnet.logianalytics.com/hc/en-us/articles/1500010064801-Aggregate-On-Dialog) dialog.
* In the Display screen of the [Create Crosstab](https://devnet.logianalytics.com/hc/en-us/articles/1500010030102-Create-Crosstab-Dialog) wizard or [Crosstab Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500010065081-Crosstab-Wizard-Dialog), select DistinctSum from the Aggregate text box and then select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404901122967/btn_chsr.gif) in the Distinct On text box.
* In the Summary screen of [Create Table](https://devnet.logianalytics.com/hc/en-us/articles/1500010065141-Create-Table-Dialog#Query) wizard, [Table Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500010068221-Table-Wizard-Dialog#Query), [Create Banded Object](https://devnet.logianalytics.com/hc/en-us/articles/1500010065101-Create-Banded-Object-Dialog#Query) wizard or [Banded Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500010029822-Banded-Wizard-Dialog#Query) based on query resources, [Horizontal Banded Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500010066821-Horizontal-Banded-Wizard-Dialog), [Mailing Label Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500010067101-Mailing-Label-Wizard-Dialog) or [Shape Map Data Binding Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500010031722-Shape-Map-Data-Binding-Wizard-Dialog), select DistinctSum from the Aggregate Function text box and then select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404901122967/btn_chsr.gif) in the Distinct On text box.
* Right-click one or more aggregations in a crosstab and on the shortcut menu point to Switch Aggregate Function, then select DistinctSum from the submenu.

![Select Fields dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901180823/slctflds.gif)

The following are details about options in the dialog:

**Resources**

Lists all the available data resources.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404901120791/btn_addarrow.gif)

Adds the selected fields from the Resources box to the Distinct On box.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404901121047/btn_rmvarrow.gif)

Removes the selected fields from the Distinct On box.

**Distinct On**

Lists the fields according to whose unique values to calculate DistinctSum.

**OK**

Applies the selected fields and closes the dialog.

**Cancel**

Cancels the selection and exits the dialog.

**Help**

Displays the help document about this feature.
