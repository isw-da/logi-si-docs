# MongoDB Database Properties

This topic lists the properties of a Database object in a MongoDB connection.

| Property Name | Description |
| --- | --- |
| Description | Specifies the description of the database in the MongoDB connection. Data type: String |
| Name | Specifies the name of the database in the MongoDB connection.  * If the database you specify exists in the MongoDB database, it will connect to the new database. * If the database you specify does not exist in the MongoDB database, no data will be got.   Data type: String |
