# Precision Settings Dialog

The Precision Settings dialog helps you to [predefine the precision level](https://devnet.logianalytics.com/hc/en-us/articles/1500010068401-Exporting-Report-Result#Precision) with which you want to export report result to different formats. It appears when you select the System Default Settings button in the [Advanced Export Settings](https://devnet.logianalytics.com/hc/en-us/articles/1500010029782-Advanced-Export-Settings-Dialog) dialog.

![Precision Settings dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901166871/prcsnsting.gif)

The precision level can be one of the following:

* **High**  
   If checked, the report result will be exported to the specified format with high precision.
* **Low**  
   If checked, the report result will be exported to the specified format with low precision.

**OK**

Applies the settings and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
