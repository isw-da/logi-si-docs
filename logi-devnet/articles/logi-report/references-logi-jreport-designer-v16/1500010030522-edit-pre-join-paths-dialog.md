# Edit Pre-join Paths Dialog

The Edit Pre-join Paths dialog helps you to [create and save paths for the joins](https://devnet.logianalytics.com/hc/en-us/articles/1500010028522-Editing-Pre-joins-in-a-Catalog#Define). It appears when you select Edit Paths in the [Pre-join Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500010032062-Pre-join-Editor-Dialog).

![Edit Pre-join Paths dialog](https://devnet.logianalytics.com/hc/article_attachments/4404901266839/edtprjnpth.gif)

The following are details about options in the dialog:

**Pre-join Paths**

Lists all the pre-join paths.

**New**

Opens the [Select Pre-join](https://devnet.logianalytics.com/hc/en-us/articles/1500010032602-Select-Pre-join-Dialog) dialog to create a new pre-join path.

**Edit**

Edits the highlighted pre-join path.

**Delete**

Deletes the highlighted pre-join path.

**Detail**

Shows the joins information of the highlighted pre-join path.

**Rename**

Specifies the new name for the highlighted pre-join.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
