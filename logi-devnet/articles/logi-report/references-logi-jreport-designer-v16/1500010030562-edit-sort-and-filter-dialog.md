# Edit Sort and Filter Dialog

The Edit Sort and Filter dialog helps you to [predefine the sort and filter conditions](https://devnet.logianalytics.com/hc/en-us/articles/1500010070821-Editing-Reports#Predefine) for the specified data component in a page report. It appears when you select a table, chart, crosstab or banded object in a query based page report and select Report > Edit Sort & Filter.

![Edit Sort and Filter dialog](https://devnet.logianalytics.com/hc/article_attachments/4404909126807/edtsrtfltr_srt.gif)

The following are details about options in the dialog:

**Sort tab**

Defines the sorting conditions.

* **Fields**  
   Lists all fields that are available to be sorted on.
* ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404901120791/btn_addarrow.gif)  
   Adds the selected fields in the Fields box to the Sort By box.
* ![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404901121047/btn_rmvarrow.gif)  
   Removes the selected fields from the Sort By box.
* **Sort By**  
   Lists the fields that are selected as the sort-by fields. Report data will be sorted according to the fields.
  + **Direction**  
     Specifies the sort order of the specified field.
  + **Name**  
     Displays the field name.
* ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404909121047/btn_mvup.gif)  
   Moves the selected field up one step. The order of the sort-by fields determines the sort priority of the fields.
* ![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404909121303/btn_mvdwn.gif)  
   Moves the selected field down one step.

**Filter tab**

Defines the filtering conditions. The options in the tab as the same as those in the [Edit Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500010065601-Edit-Filter-Dialog) dialog.

**Edit Display Name**

Opens the [Edit Display Name](https://devnet.logianalytics.com/hc/en-us/articles/1500010065541-Edit-Display-Name-Dialog) dialog to add or remove display name fields.

**OK**

Accepts the changes you've made and closes this dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
