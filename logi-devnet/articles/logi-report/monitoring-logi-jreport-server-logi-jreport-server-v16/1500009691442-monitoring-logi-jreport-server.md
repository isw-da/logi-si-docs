# Monitoring Logi JReport Server Logi JReport Server v16

Logi JReport Server Monitor, which is a standalone web-based application used to monitor the overall performance of Logi JReport Server, allows you to inspect the status of Logi JReport Server, including the status of the servers in the cluster, the status of different reports, the status of on-line users and so on. Logi JReport Server Monitor is able to generate and display the performance chart of Logi JReport Server according to its statistics. Thus you can view the performance of Logi JReport Server in the form of Line Chart Graph and Text. You can also use Logi JReport Server Monitor to maintain Logi JReport Server, such as shutting down servers, stopping waiting/running tasks, logging out a valid user session and so on. By creating profiling reports using Logi JReport Server Monitor, you can inspect server performance in a certain period of time.

**To access Logi JReport Server Monitor from server console:**

1. Make sure that the web.monitor.link.enable property in the server.properties file in `<server_install_root>\bin` is set to true.
2. Launch MonitorServer.bat in `<monitor_install_root>\bin`.
3. In the Logi JReport Server console, point to **Administration** on the system toolbar, and then select **Other** > **Monitor** from the drop-down menu.
