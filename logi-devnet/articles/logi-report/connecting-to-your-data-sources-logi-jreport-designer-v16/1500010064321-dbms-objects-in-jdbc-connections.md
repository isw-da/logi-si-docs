# DBMS Objects in JDBC Connections

Tables, views, synonyms, stored procedures and SQL files are all from the raw database. If you want to use them in Logi JReport Designer, you first have to add them to a catalog via [the JDBC connection you set up](https://devnet.logianalytics.com/hc/en-us/articles/1500010064401-Setting-Up-JDBC-Connections-in-a-Catalog) which connects Logi JReport Designer to the database driver.

The following topics detail how to work with these objects separately:

* [Tables/Views/Synonyms](https://devnet.logianalytics.com/hc/en-us/articles/1500010064361-Tables-Views-Synonyms)
* [Stored Procedures](https://devnet.logianalytics.com/hc/en-us/articles/1500010029422-Stored-Procedures)
* [Imported SQL Files](https://devnet.logianalytics.com/hc/en-us/articles/1500010064341-Imported-SQL-Files)
