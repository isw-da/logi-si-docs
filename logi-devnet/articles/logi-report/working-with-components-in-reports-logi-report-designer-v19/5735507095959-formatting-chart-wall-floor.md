# Formatting Chart Wall/Floor

This topic introduces how you can format the wall and floor of a chart.

This topic contains the following sections:

* [Formatting the Wall of a Chart](#Wall)
* [Formatting the Floor of a Chart](#Floor)

![Note icon](https://devnet.logianalytics.com/hc/article_attachments/9898403122327/noteicon.jpg) 2-D charts do not have floor, and pie and organization charts do not have either floor or wall.

## Formatting the Wall of a Chart

1. Right-click any chart element and navigate to **Format Walls** > **Format Wall** on the shortcut menu, or double-click the wall of the chart. Designer displays the [Format Wall dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735551562647-Format-Wall-Dialog-Box).

   ![Format Wall dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898462513175/fmtwall.gif)
2. In the **Wall** box, specify the fill pattern and transparency of the fill pattern to serve as the basic fill of the wall. To edit the fill pattern, select the color indicator and specify a color or effect using the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/5735570054167-Menu-Tabs#Palette). You can also type the hexadecimal value of a color (for example, 0xff0000) in the text box.
3. In the **Bound** box, specify the fill pattern and transparency of the fill pattern to compound with the basic fill of the wall.
4. In the **Border** box, specify the color, transparency, line style, and thickness for the border of the wall.
5. Select **OK** to accept the changes and close the dialog box.

## Formatting the Floor of a Chart

1. Right-click any chart element and navigate to **Format Walls** > **Format Floor** on the shortcut menu, or double-click the floor of the chart. Designer displays the [Format Floor dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735500933015-Format-Floor-Dialog-Box).

   ![Format Floor dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898480704791/fmtflr.gif)
2. In the **Floor** box, specify the fill pattern and transparency of the fill pattern for the floor. To edit the fill pattern, select the color indicator and specify a color or effect using the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/5735570054167-Menu-Tabs#Palette). You can also type the hexadecimal value of a color (for example, 0xff0000) in the text box.
3. In the **Border** box, specify the color, transparency, line style, and thickness for the border of the floor.
4. Select **OK** to accept the changes and close the dialog box.
