# Working with UDOs

A user-defined object (UDO) is an object inherited from Logi Report system's object. You can add and modify new properties and other existing methods, and specify new methods for modifying your UDO. Once you have appended UDOs to the ADDCLASSPATH variable of setenv.bat in `<install_root>\bin`, you can insert them as components into a report and the UDOs consequently become a part of the report. This topic describes how you can create and insert UDOs in reports. It also introduces the two built-in UDOs of Logi Report: JHyperLink and JRotator.

![Note icon](https://devnet.logianalytics.com/hc/article_attachments/9898403122327/noteicon.jpg) You can use UDOs in query-based page reports only.
