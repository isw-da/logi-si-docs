# Formatting Chart Elements

You can format the elements of a chart, such as the platform, wall, legend, and data markers, to give them the look you want and enable different functionalities on them. This topic introduces how you can format each element of a chart.

For the chart elements including the category (X) axis, series (Z) axis, the legend, and data markers, you can use them as the trigger object of [links](https://devnet.logianalytics.com/hc/en-us/articles/5735569846807-Adding-Links-in-a-Report).
