# Editing Page Reports in Page Report Studio

Page Report Studio provides you with a convenient and powerful tool to analyze your business information. It provides many features for interfacing with reports, such as filter and sorting the report data, converting between crosstabs and charts to view and analyze data from different aspects with different focuses, adding conditional formats to highlight wanted values, and so on. Through a user-friendly web GUI, report contents can be easily navigated, drilled, and viewed in detail.

However, for a page report created in Logi Report Designer that uses query as the data source, if you want to do analytic actions in Page Report Studio, such as adding a view element, converting between crosstabs and charts, drilling around the data, changing chart definition, or adding and converting columns and aggregating on detail columns in tables, Page Report Studio will need to convert its fields to view elements. If the conversion fails, you will not be able to perform these actions. For more information about the conversion, refer to Automatic Cube Initialization in the *Logi Report Designer Guide*.

To perform most of the operations in this topic, you are required to use [Interactive View](https://devnet.logianalytics.com/hc/en-us/articles/1500009723302-Page-Report-Studio#View) of Page Report Studio.
