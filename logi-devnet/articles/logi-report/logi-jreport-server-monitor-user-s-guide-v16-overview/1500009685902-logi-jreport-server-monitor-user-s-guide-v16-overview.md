# Logi JReport Server Monitor User's Guide v16 Overview

This User's Guide describes Logi JReport Server Monitor. Logi JReport Server Monitor, which is a standalone web-based application used to monitor the overall performance of Logi JReport Server, allows you to inspect the status of Logi JReport Server, including the status of the servers in the cluster, the status of different reports, the status of online users and so on. Logi JReport Server Monitor is able to generate and display Logi JReport Server performance statistics via line charts and text. You can also use Logi JReport Server Monitor to maintain different functions of Logi JReport Server, such as shutting down servers, stopping, pausing, running tasks, logging out a valid UserSession, and so on. By creating profiling reports using Logi JReport Server Monitor, you can inspect server performance during different periods of time.

The Logi JReport Server Guide v16 Help System provides you with everything you need to know to fully utilize all of the features of Logi JReport Server Monitor v16. You can access the information by looking at the Table of Contents, or by using Search.
