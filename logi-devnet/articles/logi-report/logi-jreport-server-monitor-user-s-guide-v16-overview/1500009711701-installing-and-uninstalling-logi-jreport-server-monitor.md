# Installing and Uninstalling Logi JReport Server Monitor

Logi JReport Server Monitor is an application for the administrator which does not need to go on the same Logi JReport Server as the report system. So it is recommended that Monitor be installed on a separate system or on systems for system administrators.

**To install Logi JReport Server Monitor on Windows:**

1. Download the Logi JReport Server Monitor installation file from the [Jinfonet download center](https://clm.logianalytics.com/rdPage.aspx?rdReport=ProductDownloads).
2. Execute the installation file.
3. Once the Installation Wizard has been successfully loaded, the Welcome screen appears. Select **Next** to continue.
4. In the License Agreement screen, read the license agreement carefully. If you agree with all the terms, select **I accept the terms of the License Agreement**, and then select **Next**. The installation process will not continue unless you accept the terms stated.
5. In the Choose Installation Folder screen, provide the absolute path for where you want to install Logi JReport Server Monitor. Select **Browse** to navigate to the destination folder. Select **Next** to continue.
6. In the Configuration screen, specify the administration port and the active host address. Select **Next** to continue.
7. In the Select JDK screen, select the JDK version with which Logi JReport Server Monitor will be installed. The selected JDK will be used to execute Logi JReport Server Monitor. The Installation Wizard collects and lists all the available JDK versions installed on your machine, you can then select one of the JDK versions from the list. Select **Next** to continue.

   **Note:** The Installation Wizard doesn't list JRE versions found on your machine. The lowest JDK version Logi JReport supports is 1.8.0. JDK 9 is currently not supported.
8. In the Add Class Path screen, a class path is necessary when using a JDBC driver or user-defined object. A class path is composed of a file path plus a zip file, jar file, or directory path. For example, `C:\jdk1.8.0_51\lib\tools.jar`.

   Select the **Add** button to add the selected class path to the class path list. Select **Delete** to delete the selected class path from the class path list. Then select **Next** to continue.

   Or, you can skip this screen by directly selecting **Install**. Later you can manually edit the batch file or the command line starting Logi JReport Server Monitor to add class paths.
9. In the Install Summary screen, the product name, location, and disk space information are shown. Click **Install**, and the Installation Wizard will start to install Logi JReport Server Monitor.
10. In the Installing screen, the installing process and status are shown.
11. After installation, the Read Me screen displays. Read the information and select **Done** to close the Installation Wizard.

**To remove Logi JReport Server Monitor on Windows:**

* Open **Control Panel** > **Programs and Features** to remove it.
* Run uninstaller.exe (uninstaller on Unix) in `<install_root>\_uninst`.

**Note:** The uninstaller will remove all the files generated by the installer, while the files that are created later by the program will be retained. You can remove them manually.
