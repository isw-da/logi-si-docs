# Introduction to Logi JReport Server Monitor

Logi JReport Server Monitor is a standalone web-based application used for monitoring the overall performance of Logi JReport Server. This topic provides an overall introduction to Logi JReport Server Monitor, including the characteristics and functions of the product.

The following are the main features of Logi JReport Server Monitor:

* **Inspects the status of Logi JReport Server**  
  Logi JReport Server Monitor allows you to view the status of servers in a cluster. It also allows you to view the status of different reports and the status of online users and so on.
* **Shows server performance statistics in Graph/Text mode**  
  Logi JReport Server Monitor is able to generate and display a performance chart of Logi JReport Server according to its statistics. In this way, you can view the performance of Logi JReport Server in the form of Line chart Graph and Text.
* **Maintains Logi JReport Server**   
   You can perform maintenance tasks from Logi JReport Server Monitor, such as shutting down servers, stopping waiting/running tasks, and logging out a valid UserSession and so on.
* **Creates profiling reports: performance reports and statistic reports**   
   By creating profiling reports using Logi JReport Server Monitor, you can inspect server performance in a certain period of time.
* **Provides JMX remote/local monitoring feature**   
  Logi JReport has constructed JMX MBeans which correspond to all the monitored objects (including Cluster, Logi JReport Server, Report Task, UserSession and the Database Connection Pool). You can use MBeanServer to find these registered MBeans and perform any operation on them. These JMX MBeans provide the same monitoring functions as with our normal Server Monitor's jsp pages.
