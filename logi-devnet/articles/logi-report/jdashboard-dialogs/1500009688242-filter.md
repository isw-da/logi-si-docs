# Filter

The Filter dialog helps you to choose whether to export/print all the data of the library component or just the current page displayed in the dashboard. This dialog appears when you right-click a table or crosstab in the Design tab of the [Export](https://devnet.logianalytics.com/hc/en-us/articles/1500009714101-Export#Dashboard) dialog or [Print](https://devnet.logianalytics.com/hc/en-us/articles/1500009714261-Print) dialog and choose Filter from the shortcut menu.

![Filter dialog](https://devnet.logianalytics.com/hc/article_attachments/4412112612503/fltr.gif)

**Current View**

Specifies to export/print the table or crosstab's current page displayed in the dashboard.

**All**

Specifies to export/print all the data of the table or crosstab.

**OK**

Closes this dialog and applies the setting.

**Cancel**

Cancels the setting and closes this dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
