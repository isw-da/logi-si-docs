# Edit HTML

The Edit HTML dialog helps you to edit the title and contents of the [HTML component](https://devnet.logianalytics.com/hc/en-us/articles/1500009712541-Creating-and-Saving-Dashboards#HTML). This dialog appears after you select ![Options button](https://devnet.logianalytics.com/hc/article_attachments/4412139482135/btn_dshbrd_more.gif) on the title bar of an HTML component and then select Edit Setting from the drop-down menu.

![Edit HTML dialog](https://devnet.logianalytics.com/hc/article_attachments/4412131485719/edthtml.gif)

**Title**

Specifies a title for the HTML component.

**HTML text box**

Type text in the box directly. You can make use of the buttons above the text box to format the text, insert images, and create hyperlinks.

Here are the buttons:

* ![Font Face](https://devnet.logianalytics.com/hc/article_attachments/4412139572119/btn_dshbrd_fontfc.gif)**Font Face**  
   Specifies the font of the text from the drop-down list.
* ![Font Size](https://devnet.logianalytics.com/hc/article_attachments/4412131482391/btn_dshbrd_fontsz.gif)**Font Size**  
   Specifies the font size of the text from the drop-down list.
* ![Bold](https://devnet.logianalytics.com/hc/article_attachments/4412131417367/btn_bold.gif)  
   Specifies whether to make the text bold.
* ![Italic](https://devnet.logianalytics.com/hc/article_attachments/4412112534039/btn_italic.gif)  
   Specifies whether to make the text italic.
* **![Underline](https://devnet.logianalytics.com/hc/article_attachments/4412131417623/btn_underline.gif)**  
   Specifies whether to underline the text.
* ![Font Color](https://devnet.logianalytics.com/hc/article_attachments/4412131482647/btn_dshbrd_fontclr.gif)**Font Color**  
   Specifies the color of the text.

  To change the color, select the color indicator to select a color, or select **More Colors** in the color indicator to access the [Color Picker](https://devnet.logianalytics.com/hc/en-us/articles/1500009688122-Color-Picker) dialog in which you can select a color within a wider range.
* ![Background Color](https://devnet.logianalytics.com/hc/article_attachments/4412139572375/btn_dshbrd_bkgrndclr.gif)**Background Color**  
   Specifies the background color of the text.

  To change the color, select the color indicator to select a color, or select **More Colors** in the color indicator to access the [Color Picker](https://devnet.logianalytics.com/hc/en-us/articles/1500009688122-Color-Picker) dialog in which you can select a color within a wider range.
* ![Align Left](https://devnet.logianalytics.com/hc/article_attachments/4412131417879/btn_alignleft.gif)**Align Left**  
   Specifies to align the text to the left.
* ![Align Center](https://devnet.logianalytics.com/hc/article_attachments/4412139523735/btn_aligncntr.gif)**Align Center**  
   Specifies to align the text in the center.
* ![Align Right](https://devnet.logianalytics.com/hc/article_attachments/4412131418135/btn_alignright.gif)**Align Right**  
   Specifies to align the text to the right.
* ![Insert Image](https://devnet.logianalytics.com/hc/article_attachments/4412131482007/btn_dshbrd_img.gif)**Insert Image**  
   Opens the [Insert Image](https://devnet.logianalytics.com/hc/en-us/articles/1500009714181-Insert-Image) dialog for inserting an image into the HTML component.
* ![Hyperlink](https://devnet.logianalytics.com/hc/article_attachments/4412139572631/btn_dshbrd_lnk.gif)**Hyperlink**  
   Opens the [Hyperlink](https://devnet.logianalytics.com/hc/en-us/articles/1500009688282-Hyperlink) dialog to create a hyperlink on the selected object.

**OK**

Closes this dialog and applies the changes to the HTML component.

**Cancel**

Cancels the changes and closes this dialog.

**Apply**

Applies the changes to the HTML component without exiting the dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
