# Real-time Settings

The Real-time Settings dialog helps you to [configure the settings for the real time chart](https://devnet.logianalytics.com/hc/en-us/articles/1500009712621-Manipulating-Data-Components#RTChart). This dialog appears when you right-click a real time chart and select Real-time Settings from the shortcut menu.

![Real-time Settings dialog](https://devnet.logianalytics.com/hc/article_attachments/4412139571351/rltmset.gif)

**Refresh Interval**

Specifies the time interval at which the chart will get data and refresh itself automatically.

**Show Most Recent N Points**

Specifies the number of records that will be kept for the real time data on the chart.

**Incremental Fetch**

Opens the [Unique Key](https://devnet.logianalytics.com/hc/en-us/articles/1500009688402-Unique-Key) dialog to configure a unique key for the real time chart.

**Use Scrollable Bar**

If checked, a scrollbar will be added to the chart, with which you can control the visible value range on the X axis of the chart.

* **Scrollable Visible Values**  
   Specifies how many data items will be selected on the scrollbar and displayed on the axis by default.
* **Scrolling Area Percentage**  
   Specifies the percentage the scrollbar occupies the whole size of the chart.
* **Show a Chart in Scrollable Bar**  
   Specifies whether to show the thumbnail chart on the scrollbar.

**OK**

Closes this dialog and applies the settings to the real time chart.

**Cancel**

Cancels configuring the settings and closes this dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
