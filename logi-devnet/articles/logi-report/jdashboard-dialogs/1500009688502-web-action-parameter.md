# Web Action - Parameter

The Web Action-Parameter dialog helps you to edit the parameter values used to re-run a data component when it [receives a synchronization message.](https://devnet.logianalytics.com/hc/en-us/articles/1500009712681-Synchronizing-the-Data-Components-in-a-Dashboard#Receive) This dialog appears when the web action \*Parameter is selected and you select ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/4412139569559/btn_dshbrd_edt1.gif) in the Actions column of the [Receive Sync](https://devnet.logianalytics.com/hc/en-us/articles/1500009714301-Receive-Sync) dialog.

![Web Action - Parameter dialog](https://devnet.logianalytics.com/hc/article_attachments/4412131476503/wbactn_param.gif)

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4412112607767/btn_dshbrd_add1.gif)

Adds a line to specify parameter value.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4412139569815/btn_dshbrd_rmv1.gif)

Removes the selected parameter.

**Parameter Name**

Lists all the parameters used by the data component. Select one from the drop-down list or input the parameter name by yourself.

**Operator**

Shows the operator to compose the expression. It can only be =.

**Parameter Value**

Specifies the value for the parameter.

**OK**

Accepts the changes and closes this dialog.

**Cancel**

Does not retain any changes and closes the dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
