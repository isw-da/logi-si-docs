# Web Action - Go Down

The Web Action-Go Down dialog helps you to edit the go-down condition for [receiving synchronization](https://devnet.logianalytics.com/hc/en-us/articles/1500009712681-Synchronizing-the-Data-Components-in-a-Dashboard#Receive) in a dashboard. This dialog appears when the web action \*Go Down is selected and you select ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/4412139569559/btn_dshbrd_edt1.gif) in the Actions column of the [Receive Sync](https://devnet.logianalytics.com/hc/en-us/articles/1500009714301-Receive-Sync) dialog.

![Web Action - Go Down dialog](https://devnet.logianalytics.com/hc/article_attachments/4412139570327/wbactn_godwn.gif)

**Go Down On**

Specifies the group on which the go-down action will be performed in the data component.

**Use Hierarchy**

Specifies the hierarchy that the go-down action will follow.

**By Value**

Specifies by which value of the specified group to perform the go-down action, that is to perform the go-down action with the filter condition *SpecifiedGroupField=TheValue*.

**OK**

Accepts the changes and closes this dialog.

**Cancel**

Does not retain any changes and closes the dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
