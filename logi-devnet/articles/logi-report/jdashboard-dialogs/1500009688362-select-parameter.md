# Select Parameter

The Select Parameter dialog helps you to [insert a parameter into the URL of the web page](https://devnet.logianalytics.com/hc/en-us/articles/1500009712541-Creating-and-Saving-Dashboards#URL) that you want to add to a dashboard. This dialog appears when you select the Add Parameter button in the [Insert URL Frame](https://devnet.logianalytics.com/hc/en-us/articles/1500009714201-Insert-URL-Frame) dialog or [Edit URL Frame](https://devnet.logianalytics.com/hc/en-us/articles/1500009714041-Edit-URL-Frame) dialog if a data source has already been specified, or select a data source and select OK in the [Select Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009688342-Select-Data-Source) dialog.

![Select Parameter dialog](https://devnet.logianalytics.com/hc/article_attachments/4412112609687/slctprmtr.gif)

**Parameters**

Lists all the available parameters in the currently selected data source for you to select one to insert to the URL.

**Change Data Source**

Opens the [Select Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009688342-Select-Data-Source) dialog to change the data source.

**OK**

Closes this dialog and inserts the selected parameter to the URL.

**Cancel**

Cancels the insertion and closes this dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
