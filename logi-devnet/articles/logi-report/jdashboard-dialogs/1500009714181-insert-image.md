# Insert Image

The Insert Image dialog helps you to [insert an image](https://devnet.logianalytics.com/hc/en-us/articles/1500009712541-Creating-and-Saving-Dashboards#Image) into the dashboard header or into the HTML component. This dialog appears when you drag Image from Toolbox in the Resources panel to the dashboard header, or select the Insert Image button ![Insert Image button](https://devnet.logianalytics.com/hc/article_attachments/4412131482007/btn_dshbrd_img.gif) in the [Insert HTML](https://devnet.logianalytics.com/hc/en-us/articles/1500009714161-Insert-HTML) dialog or [Edit HTML](https://devnet.logianalytics.com/hc/en-us/articles/1500009688162-Edit-HTML) dialog.

![Insert Image dialog](https://devnet.logianalytics.com/hc/article_attachments/4412139571863/insrtimg.gif)

**Image From**

Specifies the source of the image file.

* **Local File**  
   Specifies to use an image from the local file system.
  + **File Name**  
     Specifies the path and name of the image file. You can select the Browse button to locate the image file.
* **Web URL**  
   Specifies to use an image via URL.
  + **File URL**  
     Specifies the URL of the image file.

  **Note:** If your Logi JReport Server is in an intranet which requires a proxy, to successfully access the image via URL, you need to add the parameters `-Dhttp.proxyHost=XXX -Dhttp.proxyPort=XX` to the server's startup file JRServer.bat, which is located in `<install_root>\bin`.
* **Library**  
   Specifies to use an existing image.
  + **My Pictures**  
     The My Pictures folder is a virtual location where Logi JReport Server stores the images that have once been inserted into JDashboard. Select the one you want to use.

**Preview**

Displays a preview of the selected image.

**OK**

Closes this dialog and inserts the selected image in the dashboard header.

**Cancel**

Cancels the insertion and closes this dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
