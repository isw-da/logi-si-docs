# Web Action - Go Up

The Web Action-Go Up dialog helps you to edit the go-up condition for [Receiving synchronization](https://devnet.logianalytics.com/hc/en-us/articles/1500009712681-Synchronizing-the-Data-Components-in-a-Dashboard#Receive) in a dashboard. This dialog appears when the web action \*Go Up is selected and you select ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/4412139569559/btn_dshbrd_edt1.gif) in the Actions column of the [Receive Sync](https://devnet.logianalytics.com/hc/en-us/articles/1500009714301-Receive-Sync) dialog.

![Web Action - Go Up dialog](https://devnet.logianalytics.com/hc/article_attachments/4412131476759/wbactn_goup.gif)

**Go Up On**

Specifies the group on which the go-up action will be performed in the data component.

**Use Hierarchy**

Specifies the hierarchy that the go-up action will follow.

**OK**

Accepts the changes and closes this dialog.

**Cancel**

Does not retain any changes and closes the dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
