# Fetch Data

The Fetch Data dialog helps you to get data from the database for using in a [filter control](https://devnet.logianalytics.com/hc/en-us/articles/1500009712581-Filtering-the-Data-Components-in-a-Dashboard). This dialog appears when you select Fetch Data in the [Insert Filter Control](https://devnet.logianalytics.com/hc/en-us/articles/1500009714141-Insert-Filter-Control) dialog or [Edit Filter Control](https://devnet.logianalytics.com/hc/en-us/articles/1500009688142-Edit-Filter-Control) dialog.

![Fetch Data dialog](https://devnet.logianalytics.com/hc/article_attachments/4412139573655/ftchdt.gif)

**Search bar**

It allows you to search for values. Type in the keyword and the values containing the keyword will be listed.

**Checkboxes**

The checkbox at the top allows you select or unselect all the listed values. The following checkboxes are used for selecting the corresponding values on the same rows.

**Values**

The listed values can be sorted in the ascending or descending order by selecting the Values column header.

**OK**

Adds the selected values to the value list right after the cursor in the Insert Filter control dialog.

**Cancel**

Cancels the value selection and exits the dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
