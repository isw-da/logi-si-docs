# Unique Key

The Unique Key dialog helps you to specify a unique key to ensure there are no duplicated records in the [real time chart](https://devnet.logianalytics.com/hc/en-us/articles/1500009712621-Manipulating-Data-Components#RTChart) or [library component](https://devnet.logianalytics.com/hc/en-us/articles/1500009712601-General-Operations-in-JDashboard#Refresh). This dialog appears when you select the Incremental Fetch button in the [Real-time Settings](https://devnet.logianalytics.com/hc/en-us/articles/1500009714321-Real-time-Settings) dialog or select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4412112488471/btn_chsr1.gif) beside the Unique Key text box in the [Auto Refresh](https://devnet.logianalytics.com/hc/en-us/articles/1500009713981-Auto-Refresh#AutoRefreshBV) dialog.

![Unique Key dialog](https://devnet.logianalytics.com/hc/article_attachments/4412131477271/unqkey.gif)

**Resources**

Lists all the elements in the business view used by the real time chart or library component.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4412131477527/btn_dshbrd_add2.gif)

Adds the selected fields to the Unique Key box.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4412131477783/btn_dshbrd_rmv2.gif)

Removes the selected fields from the Unique Key box if they are not required.

**Unique Key**

Lists the fields you have added as the unique key, which is used to guarantee there are no two records with the same unique key value in the real time chart or library component.

**OK**

Closes this dialog and applies the unique key to the real time chart or library component.

**Cancel**

Cancels specifying the unique key and closes this dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
