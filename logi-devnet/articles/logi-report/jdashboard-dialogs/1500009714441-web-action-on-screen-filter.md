# Web Action - On-screen Filter

The Web Action-On-screen Filter dialog helps you to edit the filter condition to filter a data component when it [receives a synchronization message.](https://devnet.logianalytics.com/hc/en-us/articles/1500009712681-Synchronizing-the-Data-Components-in-a-Dashboard#Receive) This dialog appears when the web action \*On-screen Filter is selected and you select ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/4412139569559/btn_dshbrd_edt1.gif) in the Actions column of the [Receive Sync](https://devnet.logianalytics.com/hc/en-us/articles/1500009714301-Receive-Sync) dialog.

![Web Action - On-screen Filter dialog](https://devnet.logianalytics.com/hc/article_attachments/4412139570071/wbactn_onscrnfltr.gif)

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4412112607767/btn_dshbrd_add1.gif)

Adds a new filter condition.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4412139569815/btn_dshbrd_rmv1.gif)

Removes the selected filter condition.

**Filter On**

Specifies the field on which the filter is based.

**Value**

Specifies the value of how to filter the field.

**OK**

Accepts the changes and closes this dialog.

**Cancel**

Does not retain any changes and closes the dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
