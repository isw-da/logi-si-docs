# Web Action - Change Property

The Web Action-Change Property dialog helps you to specify the properties to be changed for a data component when it [receives a synchronization message.](https://devnet.logianalytics.com/hc/en-us/articles/1500009712681-Synchronizing-the-Data-Components-in-a-Dashboard#Receive) This dialog appears when the web action \*Property is selected and you select ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/4412139569559/btn_dshbrd_edt1.gif) in the Actions column of the [Receive Sync](https://devnet.logianalytics.com/hc/en-us/articles/1500009714301-Receive-Sync) dialog.

![Web Action - Change Property](https://devnet.logianalytics.com/hc/article_attachments/4412139570583/wbactn_prpty.gif)

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4412112607767/btn_dshbrd_add1.gif)

Adds a new property line.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4412139569815/btn_dshbrd_rmv1.gif)

Removes the selected property line.

**Properties**

Specifies the property the value of which you want to change. Select the desired property from the drop-down list or input the property name by yourself. Only properties of the data component itself can be changed.

**Value**

Specifies the value of the property.

**OK**

Accepts the changes and closes this dialog.

**Cancel**

Does not retain any changes and closes the dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
