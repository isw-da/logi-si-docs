# Setting a Dashboard as the Server Home Page

Setting a dashboard as the Logi JReport Server user console home page allows for a faster and easier access to the frequently visited dashboard with its latest data. When you log onto the Logi JReport Server user console, you will be directed to the Home tab right away which displays the dashboard. Later you can select the Home tab on the system toolbar of the Logi JReport Console page to switch to the dashboard directly. The Home tab is available after you have set a dashboard as the server home page.

Before you can set a dashboard as the server home page, you need to first enable the feature in the [server profile](https://devnet.logianalytics.com/hc/en-us/articles/1500009644582-Configuring-Server-Profile). To do this, go to the Profile > Customize Server Preferences > General tab, check the **Use a Dashboard** checkbox for the option Home Page. Then:

1. Open the dashboard that you are going to view a lot and make sure it has been saved.
2. Select the **Options** button ![](https://devnet.logianalytics.com/hc/article_attachments/4404926748951/btn_dshbrd_optn.gif) on the toolbar and you can see Set as Server Home is enabled on the option list. Select the option to set the dashboard as the home page.
