# Running and Editing Reports in JDashboard

In addition to running reports from the server resource tree, JDashboard also provides an entry for displaying reports inside it, so that users do not have to switch to the console page in order to open a report and edit it.

Viewing a report in JDashboard launches its fully functional report studio together with the report in a tab in JDashboard. The tab name is the report name. When it is a page report, [Page Report Studio](https://devnet.logianalytics.com/hc/en-us/articles/1500009649242-Page-Report-Studio) is available, and for a web report, [Web Report Studio](https://devnet.logianalytics.com/hc/en-us/articles/1500009675101-Web-Report-Studio). With the report studio, reports can be edited directly in JDashboard.

To open a report in JDashboard, select **Show Resources**![](https://devnet.logianalytics.com/hc/article_attachments/4404920571287/btn_dshbrd_show.gif) on the toolbar to open the Resources panel. Then in the Reports node, browse to the target report and drag it to the editing area, the report will be loaded. If the report uses parameters, you will be asked to specify the parameter values first before the report is displayed.

To close an open report, select **x** on the dashboard tab for the report.
