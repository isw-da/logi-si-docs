# Configuring and Setting Up Logi Report Server on Amazon EC2

**Step 1:**Search **Report** from [AWS Marketplace](https://aws.amazon.com/marketplace) and launch an eligible EC2 instance.

**Step 2:**Open your web browser and type in your EC2 instance’s public domain name or the host IP address.

**Step 3:**You will see the screen below:

![config_license1.gif](https://devnet.logianalytics.com/hc/article_attachments/12549092322071)

**Step 4:**Enter your license information into the required fields. If you do not have the information, [contact support](https://devnet.logianalytics.com/hc/en-us/signin)to request the necessary license information.

**Step 5:**Make sure the license information is correct and click **Submit**. You will see the screen below:

![config_license2.gif](https://devnet.logianalytics.com/hc/article_attachments/12549249179031)

**Step 6:**Refresh the page to see the login screen. You can use the default User Name and Password or create your own.

**Step 7:**For more Report documentation and guides, please click **Quick Start**, **Tutorial** **Demos**, or other links.

**Step 8:**Click **Login** to get started with Logi Report!
