# XML Connections

By setting up XML connections in Logi Report catalogs, you can connect to and transform XML hierarchical data sources to relational data sources.

Logi Report Designer supports parsing schemas from XML instances or importing schemas from an XSD file, and then transforming them to relational schemas. In the transformation process, elements in the XML schemas will be transformed to either tables or columns in tables. The transformed tables can then be accessed in the same way as JDBC-supplied tables.
