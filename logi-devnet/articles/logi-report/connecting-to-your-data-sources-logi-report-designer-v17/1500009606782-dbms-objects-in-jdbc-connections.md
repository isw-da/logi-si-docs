# DBMS Objects in JDBC Connections

Tables, views, synonyms, stored procedures and imported SQLs are all from the raw database. If you want to use them in Logi Report Designer, you first have to add them to a catalog via [the JDBC connection you set up](https://devnet.logianalytics.com/hc/en-us/articles/1500009629581-Setting-Up-JDBC-Connections-in-a-Catalog) which connects Logi Report Designer to the database driver.

The following topics detail how to work with these objects separately:

* [Tables/Views/Synonyms](https://devnet.logianalytics.com/hc/en-us/articles/1500009629561-Tables-Views-Synonyms)
* [Stored Procedures](https://devnet.logianalytics.com/hc/en-us/articles/1500009606802-Stored-Procedures)
* [Creating and Importing SQL Statements](https://devnet.logianalytics.com/hc/en-us/articles/1500009606822-Creating-and-Importing-SQL-Statements)
