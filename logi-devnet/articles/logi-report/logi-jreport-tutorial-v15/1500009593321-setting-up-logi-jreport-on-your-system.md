# Setting Up Logi JReport on Your System

The *Logi JReport Tutorial* assumes that you have downloaded and installed both Logi JReport Designer and Logi JReport Server on a Windows 10 machine.

The *Logi JReport Tutorial* also assumes that you are performing the lessons on a Windows 10 machine and therefore operating system specific commands and screen shots reflect this. However Logi JReport can run on multiple platforms.

Logi JReport Designer has the following system requirements:

| Logi JReport Designer System Requirements | | |
| --- | --- | --- |
|  | Recommended Requirements | Minimum Requirements |
| **OS:** | Windows x64, Unix x64, Linux x64, z/Linux64, OS X EI Capitan | Windows, Unix, Linux, z/Linux, MacOS Sierra |
| **CPU:** | Intel Core 2 | Intel P4 Xeon 3.0GHz |
| **Free Memory:** | 4 GB | 2 GB |
| **Free Disk:** | 2 GB | 1 GB |
| **JDK:** | 7 or above | 7 or above |
| **Browser:** | Latest releases of Chrome, Firefox, IE, Edge, Safari | Chrome 23, Firefox 20, IE 9, Edge, Safari 10 |

Logi JReport Server has the following system requirements:

| Logi JReport Server System Requirements | | |
| --- | --- | --- |
|  | Recommended Requirements | Minimum Requirements |
| **OS:** | Windows x64, Unix x64, Linux x64, z/Linux64, OS X EI Capitan | Windows, Unix, Linux, z/Linux, MacOS Sierra |
| **CPU:** | Intel Core 2 | Intel P4 Xeon 3.0GHz |
| **Free Memory:** | 8 GB | 4 GB |
| **Free Disk:** | 10 GB | 1 GB |
| **JDK:** | 7 or above | 7 or above |
| **Browser:** | Latest releases of Chrome, Firefox, IE, Edge, Safari | Chrome 23, Firefox 20, IE 9, Edge, Safari 10 |

You can download a JDK version at <http://www.oracle.com/technetwork/java/javase/downloads/index.html>.

**Notes:**

* Logi JReport supports Java VMs from Sun and IBM. You can try using other Java VMs, but their compatibility cannot be guaranteed. Jinfonet technical support welcomes any feedback on the use of other Java VMs.
* If you want to use Logi JReport Server on a z/Linux system, you must download the JDK specially used for IBM from<http://www-03.ibm.com/servers/eserver/zseries/software/java/> and the version should be at least V7.
* You are not recommended to run Logi JReport in the Internet Explorer Compatibility View mode.
