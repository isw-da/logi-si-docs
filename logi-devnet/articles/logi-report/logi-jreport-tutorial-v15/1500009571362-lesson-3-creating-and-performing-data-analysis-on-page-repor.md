# Lesson 3: Creating and Performing Data Analysis on Page Reports

Using Page Report Wizard the user can create a single component page report which is viewed in Page Report Studio. Page Report Studio provides the user a page mode view of the report very similar to a printed report.

There are several types of reports in the Page Report Wizard which collect information for you: blank (no component), banded, crosstab, table, and chart. You can add components to or remove them from the report after it is generated by the wizard. In this lesson, we will create a page report that contains three report tabs, and then perform data analysis on them to look at the data in other ways.

Data analysis is a process of evaluating data using analytical/logical reasoning. In JReport, business view elements can be added to existing data components. Business views greatly enhance your ability to discover useful information, draw conclusions and support the decision making process.

## Task 1: Create a Banded Report

1. On the [Logi JReport Server Start Page](https://devnet.logianalytics.com/hc/en-us/articles/1500009582241-Lesson-1-Starting-Logi-JReport-Server), select **Page Reports** in the Create category.
2. In the displayed page, select the **SampleReports** folder and **SampleReports.cat** catalog file, then select **OK**.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889257879/adhc_lsn3_tsk1_cat.gif)
3. In the New Page Report dialog, type **Customer Orders Report** in the Report Title text field, select **Banded** in the layout box, then select **OK**.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889258135/adhc_lsn3_tsk1_nwrptst.gif)

   The Banded Wizard appears.
4. In the Data screen, select **WorldWideSalesBV** in Data Source 1 from the Available Data Resources list.
   Select **Next**.

   A page layout ad hoc report is supported by only one business view. The definition of a business view determines what can be done on a report. Typically this information is communicated to the business analyst who builds the report.
5. In the Display screen, expand the **Orders Detail** category in the Resources box, add **Order Date** and **Total** in it as detail fields one by one by selecting ![](https://devnet.logianalytics.com/hc/article_attachments/4404889258391/btn_additem.gif). Select **Next**.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889258647/adhc_lsn3_tsk1_bddsply.gif)
6. In the Group screen, select **Country** in the Customers category and select ![](https://devnet.logianalytics.com/hc/article_attachments/4404889258391/btn_additem.gif) to add it as a group by field, keep Ascend as the sort manner, and then select **Next**.
7. Skip the Summary screen and select **Next**, the same to the Query Filter screen.
8. In the Style screen, set the style to **ClassicBlue**, and then select **Finish** to create the report.

   The report appears as follows in Page Report Studio:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889258903/adhc_lsn3_tsk1_bdrpt.gif)
9. Select **Menu** > **File** > **Rename Report Tab**.
10. In the Rename Report Tab dialog, enter **CustomerOrders** as the report tab name, then select **OK**.
11. Select the **Save** button ![](https://devnet.logianalytics.com/hc/article_attachments/4404893772055/btn_save.gif) on the toolbar.
12. In the Save As dialog, type **CustomerOrders.cls** in the File Name text field, then select **OK**. In the Confirm dialog, select **OK**.

## Task 2: Create a Table Report

1. In the current open report window, select the **New Report Tab**  button ![](https://devnet.logianalytics.com/hc/article_attachments/4404889259159/btn_nwrpt.gif) on the toolbar.
2. In the New Report Tab dialog, enter **Current Customers** in the Report Title text field, select **Table (Group Above)**  in the layout box, then select **OK**. The Table Wizard appears.
3. In the Data screen of the wizard, select **WorldWideSalesBV** in Data Source 1, then select **Next**.
4. In the Display screen, add **Customer Name**, **Phone** and **CustomerCityStateZip**  (change its display name to **Zip**) from the Customers category as the detail fields. Select **Next**.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889259415/adhc_lsn3_tsk2_tbldsply.gif)
5. In the Group screen, add **Country** from the Customers category as the group by field, keep Ascend as the sort manner, then select **Next**.
6. Skip the Summary screen and select **Next** to access the Query Filter screen.

As we would like to focus on customers not in USA, we will apply a filter to the business view. However, we find that there is no predefined filter in the business view, so we will define one.

1. In the Query Filter screen, define the filter as **Country != 'USA'**, then select **Next**.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893772567/adhc_lsn3_tsk2_tblfltr.gif)
2. In the Style screen, set the style to **ClassicBlue**, and then select **Finish** to create the table report.

   The table report appears as follows:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889259671/adhc_lsn3_tsk2_tblrpt.gif)
3. Select **Menu** > **File** > **Rename Report Tab**. In the Rename Report Tab dialog, enter **CurrentCustomers** as the report tab name, then select **OK**.
4. Select the **Save** button ![](https://devnet.logianalytics.com/hc/article_attachments/4404893772055/btn_save.gif) on the toolbar. In the Save Report Template dialog, select **Yes** to save the report, then select **OK** in the prompt message window.

## Task 3: Create a Crosstab Report

1. In the current open report window, select the **New Report Tab** button ![](https://devnet.logianalytics.com/hc/article_attachments/4404889259159/btn_nwrpt.gif) on the toolbar.
2. In the New Report Tab dialog, type **Sales Report** in the Report Title text field, select **Crosstab** in the layout box, then select **OK**. The Crosstab Wizard appears.
3. In the Data screen, select **WorldWideSalesBV** in Data Source 1 and then select **Next**.
4. In the Display screen, add **Country** from the Customers category in the Resources box as the row field and **Category** from the Products category as the column field (keep Ascend as the sort manner for both columns and rows), then add **Total Sales** from the Orders Detail category as the summary field. Select **Next**.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893772823/adhc_lsn3_tsk3_crstbdsply.gif)
5. Skip the Query Filter screen. Select **Next**.
6. In the Style screen, set the crosstab style as **ClassicBlue**, then select **Finish** to create the crosstab report.

   The crosstab report appears as follows:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893773079/adhc_lsn3_tsk3_crstbrpt.gif)
7. Select **Menu** > **File** > **Rename Report Tab**. In the Rename Report Tab dialog, enter **Sales** as the report tab name, then select **OK**.
8. Select the **Save** button ![](https://devnet.logianalytics.com/hc/article_attachments/4404893772055/btn_save.gif) on the toolbar. In the Save Report Template dialog, select **Yes** to save the report, then select **OK** in the prompt message window.

## Task 4: Insert a Chart

In this task, we insert a chart into the Sales report created in Task 3.

1. In the current report window, select **Menu** > **View** > **Toolbox** to display the Toolbox panel.
2. Drag **Chart** to place it under the last row of the crosstab.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889259927/adhc_lsn3_tsk4_drg.gif)

   The Chart Wizard appears.
3. In the Data screen, select **WorldWideSalesBV** in Data Source 1, then select **Next**.
4. In the Type screen, keep the default chart type Clustered Bar 2-D and select **Next**.
5. In the Display screen, add **Sales Quarter**  from the Orders Detail category to the Category box and **Product Type** in the Products category to the Series box, then add **Total Sales** in the Orders Detail category to the Show Values box.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893773463/adhc_lsn3_tsk4_chtdsply.gif)
6. Select **Finish** to create the chart. The chart appears as follows:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893773719/adhc_lsn3_tsk4_chtrpt.gif)
7. Select the **Save** button ![](https://devnet.logianalytics.com/hc/article_attachments/4404893772055/btn_save.gif) on the toolbar. In the Save Report Template dialog, select **Yes** to save the report, then select **OK** in the prompt message window.

## Task 5: Analyze Data of a Banded Object

1. In the current report window, select **CustomerOrders** from the Go To drop-down list on the toolbar to switch to this report tab.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893774103/adhc_lsn3_tsk5_rptstbr.gif)

Now, we want to make the records in each Country group further grouped in product category, and to know the discount.

2. On the left side, close the Toolbox panel by selecting **X** on the Toolbox title bar, then select **Menu** > **View** > **Resource View** to display the Resource View panel.
3. In the panel, expand the **Products** category, drag **Category** to the banded object and drop it as the mouse pointer moves below the Country group header and a tip Group Header appears.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893774359/adhc_lsn3_tsk5_addgrp.gif)

   The report tab appears as follows:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889260183/adhc_lsn3_tsk5_ctgry.gif)
4. Drag the **Discount** field in the Orders Detail category to the detail panel of the banded object:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889260439/adhc_lsn3_tsk5_adddtl.gif)

   Now the report tab looks as follows:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893774743/adhc_lsn3_tsk5_discnt.gif)

Next, we will do some adjustments to improve the appearance of the report tab.

1. Right-click the label **Customer Orders Report** and select **Properties** from the shortcut menu. Then in the Label Properties dialog, go to the **Font** tab, check the **Bold** option and then select **OK**.
2. Right-click any value of the Order Date field and select **Properties** from the shortcut menu.
3. In the General tab of the Data Field Properties dialog, set the Format property to **MM/dd/yyyy** and select **OK**.

There are two group levels in the banded object, so we will change the background color of the Category group in order to distinguish the two levels.

1. Right-click the group header panel of the Category group and select **Properties** on the shortcut menu.
2. In the General tab of the Banded Panel Properties dialog, set the Background property to **#668cb2**, and then select **OK** to confirm.

Select the **Next** button ![](https://devnet.logianalytics.com/hc/article_attachments/4404889260695/btn_next.gif) on the toolbar (or just simply scroll down the mouse wheel) to navigate to the next page (page 2). Repeat this to navigate to the following pages. There is a large space or gap before each new Category begins. This is caused by a blank Group Footer panel at the end of each category. We will remove the Group Footer panel. Return to the first page.

1. Right-click the footer panel of the Category group and select **Hide** from the shortcut menu.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893774999/adhc_lsn3_tsk5_grpftr.gif)
2. Now the report tab looks as follows:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889260951/adhc_lsn3_tsk5_frmt.gif)

Next, we will sort and filter records. The totals will be sorted in ascending order for each group. The records will be filtered by discount. The report will only display orders in which the discount is 30.

1. Select the **Sort** button ![](https://devnet.logianalytics.com/hc/article_attachments/4404893775255/btn_sort.gif) on the toolbar to open the Sort dialog.
2. In the Sort dialog, choose **Total** from the field drop-down list, keep the sorting manner as **Ascend**, and then select **OK** to apply the settings.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893775511/adhc_lsn3_tsk5_srt.gif)
3. Right-click any value of the Discount field, and on the shortcut menu, select **Filter** > **30.00**.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889261207/adhc_lsn3_tsk5_fltr.gif)

   After doing the sorting and filtering, the report tab looks as follows:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893775767/adhc_lsn3_tsk5_srtfltr.gif)

In the above two steps, we used two ways to sort and filter the report results: one by dialog and the other by right-click menu command. In fact, both sorting and filtering can be achieved by either way.

We have defined two grouping levels in the banded report: one by Country and the other by Category. Next, we will change one of the grouping criteria: grouping the banded object by Product Type instead of Category.

1. Right-click any of the Category value ("Bold" for example) and select **Drill To** > **Product Type** on the shortcut menu.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893776023/adhc_lsn3_tsk5_drl.gif)

   The report result finally changes to:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893776279/adhc_lsn3_tsk5_final.gif)
2. Select the  **Save** button ![](https://devnet.logianalytics.com/hc/article_attachments/4404893772055/btn_save.gif) on the toolbar. In the Save Report Template dialog, select **Yes** to save the report, then select **OK** in the prompt message window.

## Task 6: Analyze Data of a Table

1. Select **CurrentCustomers** from the Go To drop-down list on the toolbar to switch to this report tab.

First, we want to remove the zip information from the table as we think it is not required.

1. Right-click the **Zip** label and select **Remove Column** from the shortcut menu to delete the Zip label and its corresponding DBField.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889261463/adhc_lsn3_tsk6_rmv.gif)

Next, we will add a group Region, and make its level higher than the existing Country group.

1. From the Resource View panel, drag **Region** from the Customers category to above the Country group header and drop it when a horizontal blue line appears above the Country group header.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893776535/adhc_lsn3_tsk6_addgrp.gif)

Next we want to add some detail fields to the table. We can do this either by dragging or using the dialog.

1. Drag **State** from the Resource View panel to the top right boundary of the Phone column. Hover over the right boundary of the Phone table header and drop it when a vertical blue line appears to the right of Phone. This requires careful mouse movements and a steady hand. Select **OK** in the prompt message.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893776791/adhc_lsn3_tsk6_addst.gif)

   The report tab appears as follows:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889261719/adhc_lsn3_tsk6_state.gif)
2. Select blank area in the table, when the icon ![](https://devnet.logianalytics.com/hc/article_attachments/4404893777303/btn_crsarw.gif) appears at its upper left corner, right-click the icon and select **Insert** > **Detail Column** from the shortcut menu.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893777559/adhc_lsn3_tsk6_instclmn.gif)
3. In the Insert Detail Column dialog, select **Address 1** from the Customers category and select **OK**. A detail column, Address 1, will be added to the right of the State column.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893777815/adhc_lsn3_tsk6_adrs.gif)

However, we now consider the state information is not very useful because we will care more about the city information. So we can overwrite State with City.

1. Drag **City** in the Customers category from the Resource View panel to the State column label and drop it:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889261975/adhc_lsn3_tsk6_ovwrt.gif)

   The report result becomes:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893778071/adhc_lsn3_tsk6_city.gif)
2. Resize the Customer Name column by dragging the boundary to the right of the column to make all the customer names in the column displayed entirely.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889262231/adhc_lsn3_tsk6_resz.gif)
3. Repeat the resizing step for the Phone and Address 1 columns.

   The report result finally changes to:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889262487/adhc_lsn3_tsk6_final.gif)
4. Select the  **Save** button ![](https://devnet.logianalytics.com/hc/article_attachments/4404893772055/btn_save.gif) on the toolbar. In the Save Report Template dialog, select **Yes** to save the report, then select **OK** in the prompt message window.

## Task 7: Analyze Data of a Crosstab

1. Select **Sales** from the Go To drop-down list on the toolbar to switch to the Sales report tab, which contains a crosstab and a chart.

Since Country is a middle level predefined in the hierarchy Geography of WorldWideSalesBV, we can drill it up or down to the adjacent level.

1. Right-click any country in the row header, and select **Drill Up**  > **Region**.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893778327/adhc_lsn3_tsk7_trtry.gif)

Next we want to go down from Region to Country, however it is not a reversed process of going up from Country to Region, but applying an additional filter.

1. Right-click **Asia-Pacific** in the row header, and select **Drill Down** > **Country** from the shortcut menu.
   Now, only data of countries in Asia-Pacific is displayed:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889262743/adhc_lsn3_tsk7_drldw.gif)

We can further drill Country down to State and then to City in this way if necessary. Here, let's leave the crosstab as it is.

1. Right-click **China** in the row header, and select **Drill to by Value** > **Customer Name**.
   The crosstab is regenerated to show data of customers in China:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893778583/adhc_lsn3_tsk7_drlto.gif)
2. Next, we want to pivot the crosstab, namely interchange the rows and columns. Select the **Rotate** button ![](https://devnet.logianalytics.com/hc/article_attachments/4404889263511/btn_rotate.gif) on the toolbar and the crosstab finally changes to:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893778839/adhc_lsn3_tsk7_pvt.gif)
3. Select the  **Save** button ![](https://devnet.logianalytics.com/hc/article_attachments/4404893772055/btn_save.gif) on the toolbar. In the Save Report Template dialog, select **Yes** to save the report, then select **OK** in the prompt message window.

## Task 8: Analyze Data of a Chart

Now let's focus on the chart at the lower part of the Sales report tab. In JReport, a chart can be converted to a crosstab, and vice versa. So we will convert the chart to a crosstab first.

1. Right-click anywhere on the chart and select **To Crosstab** from the shortcut menu.
2. In the Display tab of the To Crosstab dialog, add **Product Type**  to the Columns box, **Sales Quarter**  to the Rows box, and **Total Sales** to the Summaries box, then select **Next**.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893779095/adhc_lsn3_tsk8_dsply.gif)
3. In the Style tab, set the crosstab style as **ClassicBlue**, and then select **OK**. The chart is successfully converted to a crosstab.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893779607/adhc_lsn3_tsk8_crstb.gif)

Now, we want to return the crosstab to a chart, and change the chart type to line chart.

1. Select the **Undo** button ![](https://devnet.logianalytics.com/hc/article_attachments/4404889263767/btn_undo.gif) on the toolbar to cancel the operation of converting chart to crosstab.
2. Right-click anywhere on the chart, and on the shortcut menu, select **Chart Type** > **Line** > **Line 2-D**. The chart appears as follows:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893779863/adhc_lsn3_tsk8_line.gif)

Next, we will format the chart.

1. Right-click the platform of the chart and select **Format Chart** from the shortcut menu.
2. In the Chart Type tab of Chart Definition dialog, select **Bench** from the Chart Type box, and select the **Clustered Bench 2-D** thumbnail.
3. Select the **Display** tab, select **Sales Quarter**  in the Category box and select ![](https://devnet.logianalytics.com/hc/article_attachments/4404893780375/btn_rmvitem.gif) to remove it, then expand the **Products** category in the Resources box and add **Product Type** to the Category box. Remove **Product Type** from the Series box and add **Sales Quarter**  in the Orders Detail category to the box. Add **Total Sales** in the Orders Detail category to the Show Values box.
   Then select **OK**.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889264023/adhc_lsn3_tsk8_dfntn.gif)
4. The chart now appears as follows:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889264279/adhc_lsn3_tsk8_bench.gif)
5. Right-click the chart and select **Format Platform** on the shortcut menu.
6. In the Format Platform dialog, select the **Border** tab, set Border Type to **solid**  and Color to **#0000FF**, then select the **Data** tab, set Sort Category to **descend** and Sort Series to **descend**. Select **OK** to confirm.

   Finally the chart looks as follows:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893780631/adhc_lsn3_tsk8_final.gif)

Next, we want to apply a filter to the business view the chart uses to show the data of North America.

1. Right-click anywhere on the chart and select **Query Filter** on the shortcut menu.
2. In the Query Filter dialog, specify the filter condition as follows. Select **OK** to apply the changes.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404889264535/adhc_lsn3_tsk8_cdtn.gif)

   We can find that the filter is then saved as a user defined filter in the business view.

   The chart now becomes:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404893780887/adhc_lsn3_tsk8_fltr.gif)
3. Select the  **Save** button ![](https://devnet.logianalytics.com/hc/article_attachments/4404893772055/btn_save.gif) on the toolbar. In the Save Report Template dialog, select **Yes** to save the report, then select **OK** in the prompt message window.
