# Logi JReport Tutorial v15 Overview

Thank you for your interest in Logi JReport, the leading Embedded Reporting Solution for Java Applications. With Logi JReport, intuitive and sophisticated reporting can be customized and integrated as a natural extension of any application.

This guide is intended to provide an overview of the Logi JReport solution, embedded reporting concepts, and detailed step-by-step lessons on using Logi JReport. To allow for lessons to be completed in a reasonable time frame, not all of the features of Logi JReport are covered.

This guide contains three parts:

* [Part I: Logi JReport Overview and Setup](https://devnet.logianalytics.com/hc/en-us/articles/1500009571342-Part-I-Logi-JReport-Overview-and-Setup)  
  This part briefly introduces reports and Logi JReport.
* [Part II: Quick Start](https://devnet.logianalytics.com/hc/en-us/articles/1500009593401-Part-II-Quick-Start)  
  This part introduces the most powerful yet easy-to-use web tools of Logi JReport: JDashboard, Visual Analysis, Web Report Studio and Page Report Studio.
* [Part III: Advanced Reporting](https://devnet.logianalytics.com/hc/en-us/articles/1500009561782-Part-III-Advanced-Reporting)  
  This part talks about some advanced reporting features of Logi JReport: how to use Logi JReport Designer to create data resources and use the resources to build reports; how the resources are published to Logi JReport Server and managed on the server, and so on.

This version of the Logi JReport Tutorial is developed and written for Windows. File names and paths, sample JSP code, and screen shots in this guide are based on the Windows operating system. However, you can get a Unix version of the Logi JReport Tutorial by changing the paths to the Unix format.
