# Unique Key Dialog Box Properties

Use the Unique Key dialog box to specify a unique key to ensure there are no duplicated records in the [real time chart](https://devnet.logianalytics.com/hc/en-us/articles/1500009771821-Manipulating-Data-Components#RTChart) or [library component](https://devnet.logianalytics.com/hc/en-us/articles/1500009771801-General-Operations-in-JDashboard#Refresh). This topic describes how to select the unique key.

JDashboard displays the dialog box when you select the Incremental Fetch button in the [Real-time Settings](https://devnet.logianalytics.com/hc/en-us/articles/1500009745622-Real-time-Settings-Dialog-Box-Properties) dialog box or select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404885376663/btn_chsr1.gif) beside the Unique Key text box in the [Auto Refresh](https://devnet.logianalytics.com/hc/en-us/articles/1500009745342-Auto-Refresh-Dialog-Box-Properties#AutoRefreshBV) dialog box.

![Unique Key dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880434583/unqkey.gif)

**Resources**

JDashboard list the elements in the business view that the real time chart or library component uses.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404880434967/btn_dshbrd_add2.gif)

Select to add the selected fields to the Unique Key box.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404885518487/btn_dshbrd_rmv2.gif)

Select to remove the selected fields from the Unique Key box if you do not want them.

**Unique Key**

JDashboard list the fields you have added as the unique key, which is to guarantee there are no two records with the same unique key value in the real time chart or library component.

**OK**

Select **OK** to apply the unique key to the real time chart or library component.

**Cancel**

Select **Cancel** to close the dialog box without specifying the unique key.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880314903/btn_help.gif)

Select to view information about the Unique Key dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885431447/btn_close.gif)

Select to close the dialog box without specifying the unique key.
