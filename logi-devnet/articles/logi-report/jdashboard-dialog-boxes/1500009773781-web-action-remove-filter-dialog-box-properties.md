# Web Action - Remove Filter Dialog Box Properties

Use the Web Action - Remove Filter dialog box to specify the filters you want to remove from a data component when it [receives a synchronization message](https://devnet.logianalytics.com/hc/en-us/articles/1500009771881-Synchronizing-the-Data-Components-in-a-Dashboard#Receive). This topic describes how to select the filters to remove.

JDashboard displays the dialog box when you select the web action \*Remove Filter and select ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/4404880429719/btn_dshbrd_edt1.gif) in the Actions column of the [Receive Sync](https://devnet.logianalytics.com/hc/en-us/articles/1500009745602-Receive-Sync-Dialog-Box-Properties) dialog box.

![Web Action - Remove Filter dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880430999/wbactn_rmvfltr.gif)

**Remove Filters**

Select **Remove Filters** if you want to remove the filter conditions that are created via the Filter option on a field’s shortcut menu, the Filter web action, and the Filter dialog box.

**Remove \*On-screen Filters**

Select **Remove \*On-screen Filters** if you want to remove the filter conditions that are created by the On-screen Filter web action.

* **All Fields**  
   By default, JDashboard removes all the filter conditions generated by the On-screen Filter web action. You can clear this option and then specify to remove filter conditions related to specific fields only.
* ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404885517079/btn_dshbrd_add1.gif)  
   Select to add a condition line and then select a field to remove the filter condition related to the field.
* ![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404880430743/btn_dshbrd_rmv1.gif)  
   Select to remove the selected condition line.

**OK**

Select **OK** to apply any changes you made here.

**Cancel**

Select **Cancel** to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880314903/btn_help.gif)

Select to view information about the Web Action - Remove Filter dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885431447/btn_close.gif)

Select to close the dialog box without saving any changes.
