# Open Document Dialog Box Properties

Use the Open Document dialog box to open a dashboard or an analysis template in JDashboard. This topic describes how to select a dashboard or an analysis template.

JDashboard displays the dialog box when you select the Open button ![Open button](https://devnet.logianalytics.com/hc/article_attachments/4404880136471/btn_open.gif) on the toolbar, or select the Options button ![Options button](https://devnet.logianalytics.com/hc/article_attachments/4404880435735/btn_dshbrd_optn.gif) and select Open from the option list when your Logi Report Server enables Visual Analysis.

![Open Document dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880442263/opendoc.gif)

**Open In**

Select the directory in the server resource tree where the target dashboard or analysis template is located and then select the resource. Use the button ![Up Level button](https://devnet.logianalytics.com/hc/article_attachments/4404885385367/btn_dshbrd_up.gif) to go to the parent folder.

The resource table shows the resources in the current directory.

**File Name**

File name of the selected resource.

**OK**

Select **OK** to open the selected resource.

**Cancel**

Select **Cancel** to close the dialog box without opening another resource.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880314903/btn_help.gif)

Select to view information about the Open Document dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885431447/btn_close.gif)

Select to close the dialog box without opening another resource.
