# Web Action - Go To Dialog Box Properties

Use the Web Action - Go To dialog box to edit the go-to condition to change one field to another in a data component when it [receives a synchronization message](https://devnet.logianalytics.com/hc/en-us/articles/1500009771881-Synchronizing-the-Data-Components-in-a-Dashboard#Receive). This topic describes how to define the go-to condition.

JDashboard displays the dialog box when you select the web action \*Go To and select ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/4404880429719/btn_dshbrd_edt1.gif) in the Actions column of the [Receive Sync](https://devnet.logianalytics.com/hc/en-us/articles/1500009745602-Receive-Sync-Dialog-Box-Properties) dialog box.

![Web Action - Go To dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880432919/wbactn_goto.gif)

**Go To On**

Select a field on which to perform the go-to action in the data component.

**To Column**

Select a field to change the Go To On field to.

**By Value**

Select **By Value** if you want to perform the go-to action by filtering a value of the Go To On field and then select the value. JDashboard disables this option if you select a crosstab aggregate field as the Go To On field.

**OK**

Select **OK** to apply any changes you made here.

**Cancel**

Select **Cancel** to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880314903/btn_help.gif)

Select to view information about the Web Action - Go To dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885431447/btn_close.gif)

Select to close the dialog box without saving any changes.
