# Cube Status Dialog Box Properties

You can use the Cube Status dialog box to view the information about whether the data components in the current dashboard [apply in-memory cubes and the reasons if not](https://devnet.logianalytics.com/hc/en-us/articles/1500009771801-General-Operations-in-JDashboard#CubeStatus). This topic describes the options in the dialog box.

JDashboard displays the dialog box after you select the Cube Status button ![Cube Status button](https://devnet.logianalytics.com/hc/article_attachments/4404880188823/btn_cbstts.gif) on the toolbar.

![Cube Status dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880459031/cbstts.gif)

**Component**

Lists the data components in the current dashboard.

**Business View**

Shows the business view each data component uses.

**Data Source**

Shows the catalog data source that contains the business view.

**Catalog**

Shows the path and name of the catalog in which the business view is saved.

**Cube Status**

Shows whether a data component applies an in-memory cube. This column is sortable.

**Reason**

Shows the reason why a data component does not apply an in-memory cube. JDashboard may not display the reasons fully and you can put the mouse pointer on a reason to get its full content.

**OK**

Select **OK** to close the dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880314903/btn_help.gif)

Select to view information about the Cube Status dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885431447/btn_close.gif)

Select to close the dialog box.
