# Select Parameter Dialog Box Properties

Use the Select Parameter dialog box to [insert a parameter into the URL of the web page](https://devnet.logianalytics.com/hc/en-us/articles/1500009771761-Creating-and-Saving-Dashboards#URL) that you want to add to a dashboard. This topic describes how to select a parameter.

JDashboard displays the dialog box when you select Add Parameter in the [Insert URL Frame](https://devnet.logianalytics.com/hc/en-us/articles/1500009745522-Insert-URL-Frame-Dialog-Box-Properties) dialog box or [Edit URL Frame](https://devnet.logianalytics.com/hc/en-us/articles/1500009773321-Edit-URL-Frame-Dialog-Box-Properties) dialog box if a data source has already been specified, or select a data source and select OK in the [Select Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009773621-Select-Data-Source-Dialog-Box-Properties) dialog box.

![Select Parameter dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880437527/slctprmtr.gif)

**Parameters**

JDashboard lists the available parameters in the data source for you to select one to insert to the URL.

**Change Data Source**

Select **Change Data Source** to open the [Select Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009773621-Select-Data-Source-Dialog-Box-Properties) dialog box to change the data source.

**OK**

Select **OK** to insert the selected parameter to the URL.

**Cancel**

Select **Cancel** to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880314903/btn_help.gif)

Select to view information about the Select Parameter dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885431447/btn_close.gif)

Select to close the dialog box without saving any changes.
