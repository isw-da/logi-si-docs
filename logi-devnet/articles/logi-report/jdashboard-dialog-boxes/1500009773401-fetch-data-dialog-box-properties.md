# Fetch Data Dialog Box Properties

Use the Fetch Data dialog box to get data from the database for use in a [filter control](https://devnet.logianalytics.com/hc/en-us/articles/1500009743862-Filtering-the-Data-Components-in-a-Dashboard). This topic describes how to select data values.

JDashboard displays the dialog box when you select Fetch Data in the [Insert Filter Control](https://devnet.logianalytics.com/hc/en-us/articles/1500009773461-Insert-Filter-Control-Dialog-Box-Properties) dialog box or [Edit Filter Control](https://devnet.logianalytics.com/hc/en-us/articles/1500009773281-Edit-Filter-Control-Dialog-Box-Properties) dialog box.

![Fetch Data dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880450967/ftchdt.gif)

**Search box**

You can search for values using the Search box. Type the keyword and JDashboard lists the values that contain the keyword.

**Check box**

Select the check box if you want to select a value. You can use the top check box to select or clear all the listed values.

**Values**

You can select the Values column header to sort the values in the ascending or descending order.

**OK**

Select **OK** to add the selected values to the value list right after the cursor in the Insert Filter control dialog box.

**Cancel**

Select **Cancel** to close the dialog box without selecting any values.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880314903/btn_help.gif)

Select this button to view information about the Fetch Data box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885431447/btn_close.gif)

Select this button to close the dialog box without selecting any values.
