# Hyperlink Dialog Box Properties

Use the Hyperlink dialog box to [add a hyperlink](https://devnet.logianalytics.com/hc/en-us/articles/1500009771761-Creating-and-Saving-Dashboards#HTML) on the selected object. This topic describes how to add a URL.

JDashboard displays the dialog box when you select the Hyperlink button ![Hyperlink button](https://devnet.logianalytics.com/hc/article_attachments/4404880447127/btn_dshbrd_lnk.gif) in the [Edit HTML](https://devnet.logianalytics.com/hc/en-us/articles/1500009745382-Edit-HTML-Dialog-Box-Properties) dialog box or [Insert HTML](https://devnet.logianalytics.com/hc/en-us/articles/1500009745502-Insert-HTML-Dialog-Box-Properties) dialog box.

![Hyperlink dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885532439/hyplnk.gif)

**URL**

Type a URL to direct the hyperlink to.

**OK**

Select **OK** to add the hyperlink.

**Cancel**

Select **Cancel** to close the dialog box without adding the hyperlink.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880314903/btn_help.gif)

Select this button to view information about the Hyperlink dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885431447/btn_close.gif)

Select this button to close the dialog box without adding the hyperlink.
