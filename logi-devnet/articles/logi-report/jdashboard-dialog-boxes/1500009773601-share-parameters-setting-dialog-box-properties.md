# Share Parameters Setting Dialog Box Properties

Use the Share Parameters Setting dialog box to [share parameters](https://devnet.logianalytics.com/hc/en-us/articles/1500009743882-Specifying-Parameter-Values-for-a-Dashboard#Share) among library components. This topic describes how to share parameters.

JDashboard displays the dialog box when you select the Options button ![Options button](https://devnet.logianalytics.com/hc/article_attachments/4404880435735/btn_dshbrd_optn.gif) on the toolbar and then select Share Parameter from the option list.

![Next Page dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885521687/shareprm.gif)

**All Parameters**

JDashboard lists the parameters used in the library components in the current dashboard. Cascading parameters in the same group are placed within one row.

* To share parameters, hold the **Ctrl** key to select the target parameters and then select the **Share** button. JDashboard adds the parameters into one sharing group.
* To add another parameter into a sharing group, select any parameter in the group while holding Ctrl and then select the parameter, then select **Share**.
* To remove parameters from a sharing group, select the parameters and then select **Cancel Share**.

**OK**

Select **OK** to apply any changes you made here.

**Cancel**

Select **Cancel** to close the dialog box without saving any changes.

**Apply**

Select **Apply** to apply the changes without exiting the dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880314903/btn_help.gif)

Select to view information about the Share Parameters Setting dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885431447/btn_close.gif)

Select to close the dialog box without saving any changes.
