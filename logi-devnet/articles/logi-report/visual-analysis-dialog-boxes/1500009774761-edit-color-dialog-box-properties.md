# Edit Color Dialog Box Properties

This topic describes how you can use the Edit Color dialog box to specify colors for the members of group fields.

![Edit Color dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880410391/edtclr.gif)

**Pattern**

After selecting a pattern from the drop-down list, a preview of the colors of the pattern is provided below the pattern name.

**Data Items**

Lists the combination of members of the bound fields with the colors in the selected pattern.

To customize the color of a data item, take either method of the following:

* Select the data item and then select a color in the pattern preview box.
* Double-click the data item, then in the displayed color palette select a color or select **More Colors** to access the [Color Picker](https://devnet.logianalytics.com/hc/en-us/articles/1500009746542-Color-Picker-Dialog-Box-Properties) dialog box in which you can select a color within a wider range or type a color string in the format #RRGGBB.

**Opacity**

Specifies the opacity of the color for the selected member combination in the Data Items box.

**OK**

Closes this dialog box and returns the specified color.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880408343/btn_va_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885309335/btn_delete.gif)

Ignores the setting and closes this dialog box.
