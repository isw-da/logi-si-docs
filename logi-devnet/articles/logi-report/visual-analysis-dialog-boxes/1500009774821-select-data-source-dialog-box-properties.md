# Select Data Source Dialog Box Properties

This topic describes how you can use the Select Data Source dialog box to select a business view for performing visual analysis based on.

![Select Data Source dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885498519/slctdtsrc.gif)

The available business views are listed for selection. The available business views come from the current catalog in the current directory in the server resource tree. If the current catalog contains no business view or if it is the server root directory, all available business views will be listed.

![Business Views Tree Structure](https://devnet.logianalytics.com/hc/article_attachments/4404885498775/btn_va_slctdtsrc_trvw.gif)

Displays the available business views in a tree structure.

![Business Views List](https://devnet.logianalytics.com/hc/article_attachments/4404885499159/btn_va_slctdtsrc_fltvw.gif)

Displays the available business views as list.

**OK**

Closes this dialog box and opens Visual Analysis.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880408343/btn_va_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885309335/btn_delete.gif)

Ignores the setting and closes this dialog box.
