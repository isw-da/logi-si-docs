# Edit Gradient Color Dialog Box Properties

This topic describes how you can use the Edit Gradient Color dialog box to specify gradient colors for the members of the aggregation field.

![Edit Gradient Color dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880409879/edtgrdclr.gif)

**Pattern**

After selecting a pattern from the drop-down list, a preview of the gradient color of the pattern is provided below the pattern name.

The gradient start color and end color can be customized. Select the leftmost or the rightmost arrow to select a color in the color palette or select **More Colors** to access the [Color Picker](https://devnet.logianalytics.com/hc/en-us/articles/1500009746542-Color-Picker-Dialog-Box-Properties) dialog box in which you can select a color within a wider range. After you change a color, the pattern name will be changed to Customize.

**Reversed**

Reverses the direction of the gradient color.

**OK**

Closes this dialog box and returns the specified color.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404880408343/btn_va_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404885309335/btn_delete.gif)

Ignores the setting and closes this dialog box.
