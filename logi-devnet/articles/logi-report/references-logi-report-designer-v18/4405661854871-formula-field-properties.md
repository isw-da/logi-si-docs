# Formula Field Properties

This topic describes the properties of a [Formula Field object](https://devnet.logianalytics.com/hc/en-us/articles/4405661380119-Working-with-Formula-Fields).

Designer provides some properties only when you use the object in certain [report types](https://devnet.logianalytics.com/hc/en-us/articles/4405661831319-Report-Object-Properties#ReportType). You can get details from the Available For column in the property table.

| General | | |
| Data Type | Query Page Report | Shows the data type of the formula. This property is read only. |
| Expression | Query Page Report | Shows the expression of the formula. This property is read only. |
| Field Type | Query Page Report | Shows what kind of field it is. This property is read only. |
| Formula Editor | Query Page Report | Enables you to edit the formula. Select the ellipsis Ellipsis button in the value cell to open the [Formula Editor dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664483735-Formula-Editor-Dialog-Box) to edit the formula. |
