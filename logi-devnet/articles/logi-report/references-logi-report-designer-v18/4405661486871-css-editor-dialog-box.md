# CSS Editor Dialog Box

You can use the CSS Editor dialog box to [edit the styles](https://devnet.logianalytics.com/hc/en-us/articles/4405661944471-Working-with-CSS-Styles#Manage) in a specified CSS file. This topic describes the options in the dialog box.

Designer displays the CSS Editor dialog box when you navigate to Home > CSS Editor or View > CSS Editor.

![CSS Editor](https://devnet.logianalytics.com/hc/article_attachments/4420542169111/cssedtr.gif)

You see the following options in the dialog box:

**CSS File**

This box lists all the available CSS files. Select a CSS file to edit its styles.

**Styles**

This box lists all the available styles in the selected CSS file.

**Import**

Select to import a CSS file from a specified directory.

**Export**

Select to export the specified CSS file to a designated directory.

**Duplicate**

Select to duplicate the specified CSS file.

**Remove**

Select to remove the specified CSS file.

**New**

Select to open the [New CSS Style dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664533655-New-CSS-Style-Dialog-Box) to create a CSS style.

**Edit**

Select to open the [CSS Style Definition dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664442775-CSS-Style-Definition-Dialog-Box) to edit the specified style.

**Remove**

Select to remove the specified style.

**Done**

Select to apply the changes and close the dialog box.

**Help**

Select to view information about the dialog box.
