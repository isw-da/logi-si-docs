# Dialog Box Reference

This topic introduces the dialog boxes in Designer. It mainly provides descriptions and explanations of the options available in these dialog boxes. The purpose is to make the use of Designer much easier and more convenient.

![Note icon](https://devnet.logianalytics.com/hc/article_attachments/4420542072599/noteicon.jpg) You can use either inch or centimeter as the unit of the measurement options such as Width, Height, and Thickness in Designer by customizing the [Units](https://devnet.logianalytics.com/hc/en-us/articles/4405661667735-Options-Dialog-Box#Display) option in the Options dialog box.
