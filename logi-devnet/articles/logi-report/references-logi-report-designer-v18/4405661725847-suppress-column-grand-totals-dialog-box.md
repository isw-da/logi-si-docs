# Suppress Column Grand Totals Dialog Box

You can use the Suppress Column Grand Totals dialog box to specify whether to show the grand total column for a common crosstab, or to customize the visibility of the grand totals of the column compound groups for a compound crosstab. This topic describes the options in the dialog box.

Designer displays the Suppress Column Grand Totals dialog box when you select the ellipsis ![Ellipsis button](https://devnet.logianalytics.com/hc/article_attachments/4420556128279/btn_ellipsis3.gif) next to Suppress Column Grand Totals in the Layout screen of the [Create Crosstab dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664439575-Create-Crosstab-Dialog-Box) or [Crosstab Wizard dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661479063-Crosstab-Wizard-Dialog-Box).

![Suppress Column Grand Totals dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420556128919/sprsclmngrdtl.gif)

You see the following options in the dialog box:

**Same for all**

It is meaningful to use this option for a compound crosstab. Select it to apply the setting for the grand total of the first column compound group in the box below to all the grand totals of the column compound groups in the crosstab.

**Name**

This column shows all the column compound groups in the crosstab. A common crosstab has only one column compound group.

**Suppress**

This column shows whether you specify to suppress the grand totals of the column compound groups. To specify the value, select in the value cell and select "true" or "false" from the drop-down list.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
