# Precision Settings Dialog Box

You can use the Precision Settings dialog box to [predefine the precision level](https://devnet.logianalytics.com/hc/en-us/articles/4405664595479-Exporting-Reports#Precision) at which you want to export reports to different formats. This topic describes the options in the dialog box.

Designer displays the Precision Settings dialog box when you select System Default Settings in the [Advanced Export Settings dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664427671-Advanced-Export-Settings-Dialog-Box).

![Precision Settings dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420542106519/prcsnsting.gif)

You see the following options in the dialog box:

* **High**  
  Select to export the report to the specified format with high precision.
* **Low**  
  Select to export the report to the specified format with low precision.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
