# Select Pre-join Dialog Box

You can use the Select Pre-join dialog box to [define new pre-join paths](https://devnet.logianalytics.com/hc/en-us/articles/4405661331735-Editing-Pre-joins-in-a-Catalog#Define) for the specified data source in the current catalog. This topic describes the options in the dialog box.

Designer displays the Select Pre-join dialog box when you select New in the [Edit Pre-join Paths dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664462487-Edit-Pre-join-Paths-Dialog-Box), type a path name and then select OK.

![Select Pre-join dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420550862487/slctprjn.gif)

You see the following options in the dialog box:

**All Joins**

This box lists all the pre-joins that you have created for the catalog data source. Select the joins you want to add to the new path.

**Joins in <path name>**

This box lists the joins that you add to the new path.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4420550823447/btn_addarrow.gif)**Add button**

Select to add the specified joins in the All Joins box to the new path.

![Add All button](https://devnet.logianalytics.com/hc/article_attachments/4420550823959/btn_addall.gif)**Add All button**

Select to add all the joins in the All Joins box to the new path.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4420550824215/btn_rmvarrow.gif)**Remove button**

Select to remove the specified joins from the path.

![Remove All button](https://devnet.logianalytics.com/hc/article_attachments/4420556095639/btn_rmvall.gif)**Remove All button**

Select to remove all joins from the path.

**Hide Joins Added in Other Paths**

Select to hide the joins that you have added to other paths from the All Joins box.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
