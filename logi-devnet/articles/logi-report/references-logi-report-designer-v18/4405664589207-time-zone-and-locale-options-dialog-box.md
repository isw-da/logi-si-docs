# Time Zone and Locale Options Dialog Box

You can use the Time Zone and Locale Options dialog box to specify the default time zone and locale for an XML instance. This topic describes the options in the dialog box.

Designer displays the Time Zone and Locale Options dialog box when you select Time Zone and Locale in the [XML Connection Wizard dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661750167-XML-Connection-Wizard-Dialog-Box).

![Time Zone and Locale Options dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420394718999/tmznlcl.gif)

You see the following options in the dialog box:

**Default Time Zone**

Specify the default time zone to apply to the Time type data for the XML instance. If the XML instance already contains time zone definition, Designer ignores the time zone you set here.

**Locale**

Specify the locale for the XML instance.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
