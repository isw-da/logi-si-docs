# Reset All Dialog Box

You can use the Reset All dialog box to reset properties of all areas and labels [in a shape map](https://devnet.logianalytics.com/hc/en-us/articles/4405664397975-Using-Shape-Maps#Create) to the global settings. This topic describes the options in the dialog box.

Designer displays the Reset All dialog box when you navigate to Edit > Reset All in the [Shape Map Editor dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661645975-Shape-Map-Editor-Dialog-Box).

![Reset All dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420402407703/resetall.gif)

To reset properties of all areas to the global area settings, select **Global Area** in the left box, then in the right box, select the checkboxes ahead of the required properties. Then, after you select OK in the dialog box, Designer resets the selected properties of all areas in the map to the corresponding global area settings, which you specify on the Global Area object in the Map Area Inspector tree. If you want to reset all properties of the areas to the global settings, select the **All Properties** checkbox. After resetting all properties to the global settings, Designer changes the Use Global Setting property of all areas to "true".

You can reset properties of all labels to the global label settings in the same way.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
