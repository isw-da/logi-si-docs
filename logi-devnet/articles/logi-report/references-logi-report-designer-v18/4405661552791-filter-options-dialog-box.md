# Filter Options Dialog Box

You can use the Filter Options dialog box to define the [filter-related shortcut menu commands](https://devnet.logianalytics.com/hc/en-us/articles/4405661869207-Label-Properties#FilterOptions) you want to display for a field when users right-click the field in Page Report Studio. This topic describes the options in the dialog box.

Designer displays Filter Options dialog box when you select the ellipsis ![Ellipsis button](https://devnet.logianalytics.com/hc/article_attachments/4420550841879/btn_ellipsis1.gif) in the value cell to specify the Filter Options property for a field in the Report Inspector.

![Filter Options dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420550842647/fltroptn.gif)

You see the following options in the dialog box:

**Remove Filter**

Select to display the Remove Filter item on the shortcut menu when users right-click the field in Page Report Studio.

**Top N**

Select to display the Top N item on the shortcut menu when users right-click the field in Page Report Studio.

**Bottom N**

Select to display the Bottom N item on the shortcut menu when users right-click the field in Page Report Studio.

**More**

Select to display the More item on the shortcut menu when users right-click the field in Page Report Studio.

**Default**

Select to apply the default settings.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
