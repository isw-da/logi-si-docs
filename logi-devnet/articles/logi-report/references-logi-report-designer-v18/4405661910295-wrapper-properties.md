# Wrapper Properties

This topic describes the properties of the Wrapper object in a library component.

| Property Name | Description |
| --- | --- |
| General | |
| Author | Specifies the author of the library component. Data type: String |
| Author Email | Specifies the author's email address. Data type: String |
| Description | Specifies the description of the library component. Data type: String |
| Title | Specifies the title of the library component. Data type: String |
| CSS | |
| Class | Indicates the CSS class that will be applied to the title bar of the library component. This property is read only. The style of the title bar is controlled by the JDashboard theme, which can only be edited in the CSS file. |
| Refresh | |
| Enable Auto Refresh | Specifies whether to automatically refresh the objects in the library component which are not bound with data at runtime based on a defined interval. Data type: Boolean |
| Interval | Specifies the time interval between two auto refreshes. It ranges from 0 to 86400 seconds. Type a numeric value to change the interval. It is meaningful to specify this property after setting Enable Auto Refresh to "true". Data type: Integer |
