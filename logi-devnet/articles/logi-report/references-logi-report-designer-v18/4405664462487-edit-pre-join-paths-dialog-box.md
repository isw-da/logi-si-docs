# Edit Pre-join Paths Dialog Box

You can use the Edit Pre-join Paths dialog box to [create and save paths for the joins](https://devnet.logianalytics.com/hc/en-us/articles/4405661331735-Editing-Pre-joins-in-a-Catalog#Define). This topic describes the options in the dialog box.

Designer displays the Edit Pre-join Paths dialog box when you select Edit Paths in the [Pre-join Editor dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664547991-Pre-join-Editor-Dialog-Box).

![Edit Pre-join Paths dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420402485527/edtprjnpth.gif)

You see the following options in the dialog box:

**Pre-join Paths**

This box lists all the pre-join paths you define.

**New**

Select to open the [Select Pre-join dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661713175-Select-Pre-join-Dialog-Box) to create a pre-join path.

**Edit**

Select to edit the specified pre-join path.

**Delete**

Select to delete the specified pre-join path.

**Detail**

This box shows the join information of the selected pre-join path.

**Rename**

Specify the new name for the selected pre-join.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
