# Security Entry Properties

This topic describes the properties of a [Security Entry object](https://devnet.logianalytics.com/hc/en-us/articles/4405661342359-Working-with-RLS-and-CLS-at-Data-Source-Scope) in a catalog.

| Property Name | Description |
| --- | --- |
| Description | Specifies the description of the object. Data type: String |
| Name | Specifies the mapped name of the object in the catalog. Data type: String |
