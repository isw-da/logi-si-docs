# Edit User Dialog Box

You can use the Edit User dialog box to edit the specified user when editing a [security entry](https://devnet.logianalytics.com/hc/en-us/articles/4405661342359-Working-with-RLS-and-CLS-at-Data-Source-Scope#Create) or [business view security](https://devnet.logianalytics.com/hc/en-us/articles/4405664363415-Configuring-Business-View-Security-in-a-Catalog). This topic describes the options in the dialog box.

Designer displays Edit User dialog box when you select a user and then select Edit ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/4420556150039/btn_edit.gif) in the [Security dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661696919-Security-Dialog-Box), [Edit Business View Security dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661506455-Edit-Business-View-Security-Dialog-Box), or in the Security tab of the following dialog boxes: [Category Property dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664450071-Category-Property-Dialog-Box), [New View Element dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661660823-New-View-Element-Dialog-Box), or [Edit View Element dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661533079-Edit-View-Element-Dialog-Box).

The dialog box contains the following tabs:

* [General Tab](#General)
* [Relation Tab](#Relation)

You see these buttons in the tabs:

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.

## General Tab

Use this tab to specify the general information of the user.

![Edit User dialog box - General](https://devnet.logianalytics.com/hc/article_attachments/4420556200599/edtuser_gen.gif)

**User**

Specify the name of the user.

**Source**

This option shows the source of the user, which is read only.

**E-mail**

Specify the email address of the user.

**Description**

Specify the description of the user.

## Relation Tab

Use this tab to specify the relationship between the user and the existing roles and groups.

![Edit User dialog box - Relation](https://devnet.logianalytics.com/hc/article_attachments/4420550929047/edtuser_rel.gif)

**Roles**

This box lists all the available roles. Select a role to grant it to the user.

**Groups**

This box lists all the available groups. Select a group to add the user into it.
