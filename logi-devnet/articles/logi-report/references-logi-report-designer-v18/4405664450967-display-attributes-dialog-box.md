# Display Attributes Dialog Box

You can use the Display Attributes dialog box to define certain attributes for the display values, which you can also specify via the [Options dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661667735-Options-Dialog-Box#Display). This topic describes the options in the dialog box.

Designer displays the Display Attributes dialog box when you navigate to View > Display Attributes.

![Display Attributes dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420394827287/dsplyatrbt.gif)

You see the following options in the dialog box:

**Decimal**

Specify the scale of the fractional data in reports. For instance, if you set Decimal to 3, Designer rounds all the property values of the objects in the Report Inspector to the thousandth.

**Unit**

Select the unit of the measurement to use in the report properties: centimeter or inch.

**Resolution**

Select the resolution in dots per inch. If you select "default", Designer applies the resolution of your monitor.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
