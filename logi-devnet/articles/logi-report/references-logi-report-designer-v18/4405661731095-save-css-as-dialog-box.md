# Save CSS As Dialog Box

You can use the Save CSS As dialog box to [save the format of an object as a CSS style](https://devnet.logianalytics.com/hc/en-us/articles/4405661944471-Working-with-CSS-Styles#Create) in order to use the same format in other reports, without having to set all the properties again. This topic describes the options in the dialog box.

Designer displays the Save CSS As dialog box when you select New Style Sheet File from the Add Selector to Style drop-down list and select OK in the [New CSS Style dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664533655-New-CSS-Style-Dialog-Box).

![Save CSS As dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420410585879/svcss.gif)

You see the following options in the dialog box:

**Save the CSS File to**

This option shows the directory to which the CSS file will be saved.

**Existing CSS Files**

This box lists all the existing CSS files in the specified directory.

**File Name**

Specify the name for the new CSS file.

**Files of Type**

This option shows the type of the file, which can only be CSS File(\*.css).

**Save**

Select to open the [CSS Style Definition dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664442775-CSS-Style-Definition-Dialog-Box) to edit the style.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
