# User Defined Function Properties

This topic describes the properties of a [User Defined Function (UDF) object](https://devnet.logianalytics.com/hc/en-us/articles/4405661765527-Using-User-Defined-Formula-Functions) in a catalog.

| Property Name | Description |
| --- | --- |
| Expression | Shows the whole statement of the function. This property is read only. |
| Name | Specifies the mapped name of the object in the catalog. Data type: String |
| [SQL Type](https://devnet.logianalytics.com/hc/en-us/articles/4405664627479-Formula-Properties#SQL) | Specifies the SQL type of the object defined in Java. Choose an option from the drop-down list. Data type: Integer |
