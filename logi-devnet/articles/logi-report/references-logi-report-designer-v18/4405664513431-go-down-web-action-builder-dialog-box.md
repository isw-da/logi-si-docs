# Go Down - Web Action Builder Dialog Box

You can use the Go Down - Web Action Builder dialog box to [build a Go Down web action](https://devnet.logianalytics.com/hc/en-us/articles/4405661415703-Using-Basic-Web-Controls-in-a-Report#GoDown) on an object, so that users can drill data of the specified data components in a report to lower-level groups according to the predefined hierarchies, when they trigger the event bound with the action on the object at runtime. This topic describes the options in the dialog box.

Designer displays the Go Down - Web Action Builder dialog box when you select \*Go Down and select OK in the [Web Action List dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661744791-Web-Action-List-Dialog-Box).

![Go Down - Web Action Builder dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420402430231/godwnwbactnbld.gif)

You see the following options in the dialog box:

**Apply Action To**

This box lists the data components in the report. Select a data component to define the go-down condition for it.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4420394650391/btn_add.gif)**Add button**

Select to add a new condition line. You can add one condition at most.

![Trash Can button](https://devnet.logianalytics.com/hc/article_attachments/4420402325015/btn_trashcan.gif)**Remove button**

Select to delete the condition.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4420410588439/btn_mvup.gif)**Move Up button**

Designer does not enable this button for building the Go Down web action.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4420410588695/btn_mvdwn.gif)**Move Down button**

Designer does not enable this button for building the Go Down web action.

**Go Down On**

Specify the group on which to perform the go-down action for the selected data component.

**Use Hierarchy**

Specify the hierarchy for the go-down action to follow for the selected data component.

**By Value**

Specify by which value of the specified group to perform the go-down action for the selected data component, that is to perform the go-down action with the filter condition *SpecifiedGroupField=TheValue*.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
