# Create Query's Username and Password Dialog Box

You can use the Create Query's Username and Password dialog box to [set a password to restrict users](https://devnet.logianalytics.com/hc/en-us/articles/4405661921687-Locking-Queries) from editing the query at random. This topic describes the options in the dialog box.

Designer displays the Create Query's Username and Password dialog box when you right-click a query and then select Create User from the shortcut menu.

![Create Query's Username and Password dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420556107159/crtqrynmpw.gif)

You see the following options in the dialog box:

**User Name**

Specify the user name that can edit the query.

**Password**

Specify the password to restrict users from editing the query.

**Confirm password**

Confirm the password that you specify in the Password text box.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
