# Aggregation Properties

This topic describes the properties of an Aggregation object in a business view.

| Property Name | Description |
| --- | --- |
| Aggregate Function | Specifies the [aggregate function](https://devnet.logianalytics.com/hc/en-us/articles/4405661767063-Working-with-Summaries#Function) of the object. Data type: String |
| Customized Contents | Shows the contents of the [custom aggregation](https://devnet.logianalytics.com/hc/en-us/articles/4405661913239-Business-View-Elements#CA). This property is read only. |
| Data Mapping File | Specifies the data mapping file you want to apply to the object [for NLS use](https://devnet.logianalytics.com/hc/en-us/articles/4405661798039-Mapping-Data-Values-to-Different-Languages). For example, if the data mapping file name is Category\_de\_DE.properties, set the value as "Category" (without the locale part).  Data type: String |
| Description | Specifies the description of the object. Data type: String |
| Display Name | Specifies the display name of the object. Data type: String |
| Is Customized | Specifies whether the aggregation object is a [custom aggregation](https://devnet.logianalytics.com/hc/en-us/articles/4405661913239-Business-View-Elements#CA). Data type: Boolean |
| Mapping Name | Specifies the name of the column to which the object is mapped. Data type: String |
| Tip | Specifies the tool tip of the object, which displays when users hover over the object in the business view resource tree at runtime. Data type: String |
