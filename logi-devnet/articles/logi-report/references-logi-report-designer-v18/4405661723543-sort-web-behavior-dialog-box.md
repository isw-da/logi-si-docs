# Sort - Web Behavior Dialog Box

You can use the Sort - Web Behavior dialog box to [apply the Sort web action to a web control](https://devnet.logianalytics.com/hc/en-us/articles/4405661414679-Using-Basic-Web-Controls-in-the-Configuration-Panel-of-a-Library-Component#Sort) in a library component so as to sort data of the library component at runtime. This topic describes the options in the dialog box.

Designer displays the Sort - Web Behavior dialog box when you right-click a web control in the configuration panel of a library component and navigate to Web Behaviors > Sort on the shortcut menu.

![Sort - Web Behavior dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420556129175/sortwbbhvr.gif)

You see the following options in the dialog box:

**Apply Action To**

Select to which data component in the library component, or table group to apply the sort action.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4420550832791/btn_add.gif)**Add button**

Select to add a new sort condition for the action.

![Trash Can button](https://devnet.logianalytics.com/hc/article_attachments/4420542080407/btn_trashcan.gif)**Remove button**

Select to remove the specified sort condition.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4420542078103/btn_mvup.gif)**Move Up button**

Select to move the specified sort condition higher in the list. The order determines the sort priority of the conditions at runtime.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4420542078359/btn_mvdwn.gif)**Move Down button**

Select to move the specified sort criterion lower in the list.

**Sort On**

This column shows the fields based on which you select to sort the specified data component, or table group.

**Sort Value**

This column shows the order you select to sort the fields: Ascending, Descending, or obtained from a web control at runtime.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
