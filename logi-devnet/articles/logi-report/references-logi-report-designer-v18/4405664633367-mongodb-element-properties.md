# MongoDB Element Properties

This topic describes the properties of an Element object in a MongoDB collection.

| Property Name | Description |
| --- | --- |
| BSON Data Type | Specifies the [data type](https://devnet.logianalytics.com/hc/en-us/articles/4405664419479-How-Designer-Gets-Data-from-MongoDB-Databases#ConvertRule) of the element. Choose an option from the drop-down list. Data type: Enumeration |
| Data Type Name | Shows the data type name of the element. This property is read only and Designer changes its value automatically according to the BSON Data Type property. |
| Name | Specifies the name of the element. Data type: String |
| Sub Data Type | Specifies the [sub data type](https://devnet.logianalytics.com/hc/en-us/articles/4405664419479-How-Designer-Gets-Data-from-MongoDB-Databases#Sub) of the element. This property takes effect when you set BSON Data Type to 5. Choose an option from the drop-down list. Data type: Enumeration |
| Sub Type Name | Shows the sub data type name of the element. This property is read only and Designer changes its value automatically according to the Sub Data Type property. |
