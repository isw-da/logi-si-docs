# Query Modifier Properties

This topic describes the properties of a Query Modifier object in a catalog.

| Property Name | Description |
| --- | --- |
| Data Source Name | Specifies the data source that includes the query modifier. Data type: String |
| Description | Specifies the description of the query modifier. Data type: String |
| Name | Specifies the mapped name of the query modifier in the catalog. Data type: String |
| Query | Specifies the query which the query modifier refers to. Data type: String |
| Query Modifier | Specifies the WHERE string of the query modifier. Data type: String |
| Structured | Specifies the type for defining the WHERE string. When you select "true", it is Structured; when "false", it is String. Data type: Boolean |
