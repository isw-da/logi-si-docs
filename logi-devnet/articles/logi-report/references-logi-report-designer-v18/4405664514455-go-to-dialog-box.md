# Go To Dialog Box

You can use the Go To dialog box to [define a Go To web action](https://devnet.logianalytics.com/hc/en-us/articles/4405661935255-Delivering-Messages-Between-Library-Components#UGoTo) for the specified data component in a library component as a response to the user-defined message it receives at runtime. This topic describes the options in the dialog box.

Designer displays the Go To dialog box when you select \*Go To and select OK in the [Web Action List dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661744791-Web-Action-List-Dialog-Box).

![Go To dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420542120727/goto.gif)

You see the following options in the dialog box:

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4420550832791/btn_add.gif)**Add button**

Select to add a new condition line. You can add one condition at most.

![Trash Can button](https://devnet.logianalytics.com/hc/article_attachments/4420542080407/btn_trashcan.gif)**Remove button**

Select to delete the condition.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4420542078103/btn_mvup.gif)**Move Up button**

Designer does not enable this button for building the Go To web action.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4420542078359/btn_mvdwn.gif)**Move Down button**

Designer does not enable this button for building the Go To web action.

**Go To On**

Specify the field on which to perform the go-to action.

**To Column**

Specify to which field the Go To On field will be changed.

**By Value**

Specify to perform the go-to action by filtering the specified value of the Go To On field. Designer disables this option if the action is to be performed on a crosstab aggregate field.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
