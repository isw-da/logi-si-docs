# E-mail Settings Dialog Box

You can use the E-mail Settings dialog box to edit additional settings of the email when you specify to [distribute a report to email](https://devnet.logianalytics.com/hc/en-us/articles/4405661925143-Creating-Bursting-Reports#Create). This topic describes the options in the dialog box.

Designer displays the E-mail Settings dialog box when you select E-mail or Logi Report Server User/Group/Role > User E-mail in the Recipient section and then select E-mail Settings in the [Bursting dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664430615-Bursting-Dialog-Box).

![E-mail Settings dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420410606615/emlst.gif)

You see the following options in the dialog box:

**Cc**

Select the field from the recipient query which retrieves email addresses that you want to send additional copies of the email to.

**Bcc**

Select the field from the recipient query which retrieves email addresses that you want to secretly send additional copies of the email to.

**Subject**

Specify the subject of the email.

**Comments**

Specify the contents of the email or comments to the contents.

**Add Dynamic Field**

Designer enables this button when you select in the Subject or Comments text box. Select it to open the [Select Field dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664571671-Select-Field-Dialog-Box) to select a field and insert it in the text box so as to compose dynamic information.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
