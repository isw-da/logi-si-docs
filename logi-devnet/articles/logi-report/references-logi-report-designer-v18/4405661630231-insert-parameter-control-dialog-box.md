# Insert Parameter Control Dialog Box

You can use the Insert Parameter Control dialog box to [insert a parameter control](https://devnet.logianalytics.com/hc/en-us/articles/4405661411351-Using-Advanced-Web-Controls#Param) into a report for delivering parameter values. This topic describes the options in the dialog box.

Designer displays the Insert Parameter Control dialog box when you navigate to Insert > Web Controls > Parameter Control, or drag the Parameter Control icon ![Parameter Control](https://devnet.logianalytics.com/hc/article_attachments/4420556152343/btn_paramcntrl.gif) from the Components panel to a report.

![Insert Parameter Control dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420542116247/instprmtrctrl.gif)

You see the following options in the dialog box:

**Select a Parameter**

This box lists all the parameters except cascading parameters that the current report uses. Select the one you want to use in the parameter control.

**OK**

Select to insert the parameter control into the report and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
