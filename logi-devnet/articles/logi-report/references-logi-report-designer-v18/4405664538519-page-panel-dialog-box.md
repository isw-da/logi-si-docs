# Page Panel Dialog Box

You can use the Page Panel dialog box to create a [page panel](https://devnet.logianalytics.com/hc/en-us/articles/4405664695447-Designing-the-Report-Pages#Panel). This topic describes the options in the dialog box.

Designer displays the Page Panel dialog box when you navigate to Insert > Page Panel.

![Page Panel dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420402323863/pgpnl.gif)

You see the following options in the dialog box:

**Keep Current Page Setting**

Select to create a page panel with the current page settings.

**Use New Page Setting**

Select to create a page panel with new page settings.

**OK**

Select to create a page panel with the current page settings, or open the [Page Setup dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664543511-Page-Setup-Dialog-Box) to specify the page settings for the new page panel.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
