# Save Style Dialog Box

You can use the Save Style dialog box to [edit and save a style into a style group (XSD style)](https://devnet.logianalytics.com/hc/en-us/articles/4405661945495-Working-with-XSD-Styles#Create) in a catalog. This topic describes the options in the dialog box.

Designer displays the Save Style dialog box when you select New Style to create a style in a style group in the Style Editor or select New Style in the [Edit Style Group dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664466199-Edit-Style-Group-Dialog-Box), then in the New Style dialog box, specify the name, type, and description of the new style and select OK.

![Save Style dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420542073367/svstyl.gif)

You see the following options in the dialog box:

**Style Group Name**

This option shows the name of the style group to which the style will be saved.

**Style Name**

This option shows the name of the style.

**All Properties**

This box lists all the available properties that you can add to the style.

**Selected Properties**

This box lists the properties that you select to include in the style. You can edit the property values.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4420550823447/btn_addarrow.gif)**Add button**

Select to add the specified property in the All Properties box to the style.

![Add All button](https://devnet.logianalytics.com/hc/article_attachments/4420550823959/btn_addall.gif)**Add All button**

Select to add all the properties in the All Properties box to the style.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4420550824215/btn_rmvarrow.gif)**Remove button**

Select to remove the specified property in the Selected Properties box from the style.

![Remove All button](https://devnet.logianalytics.com/hc/article_attachments/4420556095639/btn_rmvall.gif)**Remove All button**

Select to remove all the properties in the Selected Properties box from the style.

**Save**

Select to save the style and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
