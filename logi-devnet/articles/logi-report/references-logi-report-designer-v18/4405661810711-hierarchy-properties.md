# Hierarchy Properties

This topic describes the properties of a Hierarchy object in a business view.

| Property Name | Description |
| --- | --- |
| Description | Specifies the description of the object. Data type: String |
| Display Name | Specifies the display name of the object. Data type: String |
| Tip | Specifies the tool tip of the object, which displays when users hover over the object in the business view resource tree at runtime. Data type: String |
