# Set Grids Dialog Box

You can use the Set Grids dialog box to specify the grid settings in the design area. This topic describes the options in the dialog box.

Designer displays the Set Grids dialog box when you navigate to View > Grids > Grid Size. You can also specify the grid options in the Editor category of the [Options dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661667735-Options-Dialog-Box#Editor).

![Set Grids dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420550864791/setgrid.gif)

You see the following options in the dialog box:

**Size**

Specify the size of the grids in the design area.

**Color**

- Select the color of the grids. Select a color from the drop-down list or select Custom to customize a color in the [Pick a Color dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405664544407-Pick-a-Color-Dialog-Box#Color).

**Show Grids**

Select to show grids in the report.

**Snap to Grids**

Select to snap and lock objects with the grids when you move and place objects around.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Apply**

Select to apply all changes and leave the dialog box open.

**Help**

Select to view information about the dialog box.
