# Choose Page Report Dialog Box

You can use the Choose Page Report dialog box to manage the page reports in the current catalog by [adding them into folders](https://devnet.logianalytics.com/hc/en-us/articles/4405664693399-Managing-Page-Reports#AddRpt). This topic describes the options in the dialog box.

Designer displays the Choose Page Report dialog box when you select a folder and select Add Page Reports on the toolbar in the Page Report Manager.

![Choose Page Report dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420410593815/chspgrpt.gif)

You see the following options in the dialog box:

**Catalog**

This option shows the path and name of the current catalog.

**Reports box**

This box lists the page reports of the selected type in the current catalog. Select the reports you want to add.

**Report Name**

This option shows the name of the last selected report.

**Report of Type**

You can save a page report as four types of files: cls (report binary file), rpt (report source file), cls.xml (XML format report), and clx (self contained report). This drop-down list contains these four types. Select the type of the page reports that you want to add.

**Add**

Select to add the specified page reports to the folder.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
