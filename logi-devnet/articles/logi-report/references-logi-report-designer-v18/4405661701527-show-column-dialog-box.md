# Show Column Dialog Box

You can use the Show Column dialog box to [show or hide columns from a table](https://devnet.logianalytics.com/hc/en-us/articles/4405664405527-Modifying-Tables#Show). This topic describes the options in the dialog box.

Designer displays the Show Column dialog box when you right-click a table and select Show Column from the shortcut menu.

![Show Column dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420550864407/shwcol.gif)

You see the following options in the dialog box:

**Auto Sort**

Select to sort the columns alphabetically; otherwise, Designer lists the columns according to their order in the table.

**Column box**

This box lists all the columns in the table, with the text in the table header panel representing the corresponding columns. You can show or hide some of them by selecting or clearing the checkboxes ahead of the column names.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
