# Web Report Option Dialog Box

You can use the Web Report Option dialog box to predefine the options for directly running a report in the Web Report format on Server. This topic describes the options in the dialog box.

Designer displays the Web Report Option dialog box when you set the Default Format for Viewing Report property of a web report to "Web Report" and then select the ellipsis ![Ellipsis button](https://devnet.logianalytics.com/hc/article_attachments/4420550841879/btn_ellipsis1.gif) in the value cell in the Report Inspector.

![Web Report Option dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420556122647/wbrptoptn.gif)

You see the following options in the dialog box:

**Resolution**

Specify the resolution to zoom in/out the report, in DPI. Designer obtains the default value from the operation system, which is the resolution of your monitor, for example, 72 DPI on UNIX or 96 on Windows. You can set higher/lower value to zoom in/out.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
