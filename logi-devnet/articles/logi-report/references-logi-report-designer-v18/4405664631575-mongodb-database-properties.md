# MongoDB Database Properties

This topic describes the properties of a Database object in a MongoDB connection.

| Property Name | Description |
| --- | --- |
| Description | Specifies the description of the database. Data type: String |
| Name | Specifies the name of the database. If the database you specify exists in the MongoDB database, Designer connects to the database; otherwise, Designer gets no data. Data type: String |
