# Select Filter Resources Dialog Box

You can use the Select Filter Resources dialog box to [customize the view elements you want to use as the filter resources](https://devnet.logianalytics.com/hc/en-us/articles/4405664681111-Creating-Business-Views-in-a-Catalog#FilterResource) of a business view. This topic describes the options in the dialog box.

Designer displays the Select Filter Resources dialog box when you right-click a business view and select Select Filter Resources on the shortcut menu in the Catalog Manager, or select Select Filter Resources on the toolbar in the [Business View Editor dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661463063-Business-View-Editor-Dialog-Box).

![Select Filter Resources dialog box](https://devnet.logianalytics.com/hc/article_attachments/4420542087319/slctfltrrsc.gif)

You see the following options in the dialog box:

**Resource box**

This box lists the view elements contained in the business view. Select the checkboxes ahead of the elements that you want to use as filter resources.

![Sort icon](https://devnet.logianalytics.com/hc/article_attachments/4420542068375/btn_sort2.gif)**Sort icon**

Select to display the Sort drop-down menu to specify how to sort the view elements.

* **Ascending**  
  Select to sort the view elements in the ascending order.
* **Descending**  
  Select to sort the view elements in the descending order.
* **No Sort**  
  Select to list the view elements based on the original order of their mapping fields in the database.

Designer determines the default sort order according to the [Sort](https://devnet.logianalytics.com/hc/en-us/articles/4405661667735-Options-Dialog-Box#Sort) option in the Catalog category of the Options dialog box. If you do not select Sort in the Options dialog box, Designer selects No Sort by default in the drop-down menu. In addition, the change of sort order is a one-off action which Designer does not remember after you exit the dialog box, meaning, each time when you open the dialog box, Designer always applies the default sort order.

![Search icon](https://devnet.logianalytics.com/hc/article_attachments/4420542068631/btn_srch.gif)**Search icon**

Select to open the search box to search for view elements in the business view.

To start searching, type the text you want to search for in the search box and Designer lists the elements containing the matched text.

You can use the following options in the search box:

* ![More Search Option button](https://devnet.logianalytics.com/hc/article_attachments/4420550825111/btn_srchbox_adv.gif)**Drop-down icon**  
  Select to list more search options.
  + **Highlight All**  
    Select to highlight all the matched text.
  + **Match Case**  
    Select to search for text that meets the case of the text you type.
  + **Match Whole Word**  
    Select to search for text that looks the same as the text you type.
* ![Close button](https://devnet.logianalytics.com/hc/article_attachments/4420550826007/btn_close1.gif)**Delete icon**  
  Select to close the search box and cancel the search.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
