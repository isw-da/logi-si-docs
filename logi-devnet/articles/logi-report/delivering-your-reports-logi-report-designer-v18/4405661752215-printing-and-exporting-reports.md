# Printing and Exporting Reports

As you develop page reports and web reports in Designer, you can print or export the reports so you can review the outputs at any time or share with others. This topic introduces how you can print and export page and web reports (you can neither print nor export library components).

Select the following links to view the topics:

* [Printing Reports](https://devnet.logianalytics.com/hc/en-us/articles/4405664599447-Printing-Reports)
* [Exporting Reports](https://devnet.logianalytics.com/hc/en-us/articles/4405664595479-Exporting-Reports)
* [Exporting Reports to Accessible HTML and PDF](https://devnet.logianalytics.com/hc/en-us/articles/4405664594583-Exporting-Reports-to-Accessible-HTML-and-PDF)

![Note icon](https://devnet.logianalytics.com/hc/article_attachments/4420542072599/noteicon.jpg)

* When you export or print a report which contains parameters, Designer prompts you to [specify values for the parameters](https://devnet.logianalytics.com/hc/en-us/articles/4405661802903-Specifying-Parameter-Values).
* When you export or print a report which [applies a style group](https://devnet.logianalytics.com/hc/en-us/articles/4405661943319-Applying-Styles-to-Reports), Designer prompts you to select the style group to use.
