# Insert Parameter Control

The Insert Parameter Control dialog box is used to [insert a parameter control](https://devnet.logianalytics.com/hc/en-us/articles/1500009751581-Using-Web-Controls#Parameter) to a report for delivering parameter values to the report. It appears when you drag Parameter Control from the Components panel to a report.

![Insert Parameter Control dialog](https://devnet.logianalytics.com/hc/article_attachments/4404942450199/insrtprmctrl.gif)

**Select a Parameter**

Specifies a parameter from the list where all the parameters except cascading parameters used by the current report are displayed.

**OK**

Inserts a parameter control into the report and closes the dialog box.

**Cancel**

Cancels the operation and closes the dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404936816535/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404942519575/btn_close.gif)

Ignores the setting and closes this dialog box.
