# Add Format

The Add Format dialog box is used to add available format keys in the specified resource for [editing NLS](https://devnet.logianalytics.com/hc/en-us/articles/1500009750321-NLS-at-Report-Level#Local). It appears when an admin user selects ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404936734999/btn_add1.gif) in the Format tab of the [NLS Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009720602-NLS-Editor#Format).

![Add Format dialog](https://devnet.logianalytics.com/hc/article_attachments/4404942566423/addfmt.gif)

**Available Format**

Lists all available format keys in the specified catalog/report/library component.

**Search box**

Searches for the keys that contain the specified text. After typing in the text, select ![Search button](https://devnet.logianalytics.com/hc/article_attachments/4404936713623/btn_srch.gif) in the box or press Enter on the keyboard to start searching.

**check box**

Specifies whether to add the format keys to edit NLS. Select the check box on the column header to select all keys.

**Key**

Lists keys to indicate the objects in the original language. You can select the underlined column header to sort the items by key in an ascending or descending order.

**Format**

Specifies the formats for the keys in the target language. You can select the underlined column header to sort the items by format in an ascending or descending order.

**OK**

Adds the selected keys to the Display tab of the NLS Editor.

**Cancel**

Cancels the action and exits the dialog box.

**Reset**

Resets the dialog box to its initial status.

**Help**

Displays the help document about this feature.
