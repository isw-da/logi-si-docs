# Select Business View Elements

The Select Business View Elements is used to select business view elements in the specified catalog for which to define the [dynamic display names](https://devnet.logianalytics.com/hc/en-us/articles/1500009723082-Managing-Dynamic-Display-Names-of-Business-View-Elements). It appears when an admin user selects the Select Business View Elements link in the [New Dynamic Display Name](https://devnet.logianalytics.com/hc/en-us/articles/1500009747581-New-Dynamic-Display-Name) dialog box.

![Select Business View Elements dialog](https://devnet.logianalytics.com/hc/article_attachments/4404942503191/slctbvelmnt.gif)

**Catalog**

Displays the current catalog with the resource path.

**Select Business View Elements**

The panel lists all the business views and their elements in the catalog. You can select multiple elements by selecting the corresponding check boxes.

**All**

Selects/Deselects all the business view elements in the catalog.

**OK**

Applies the selection and exits the dialog box.

**Cancel**

Cancels the selection and closes the dialog box.

**Help**

Displays the help document about this feature.
