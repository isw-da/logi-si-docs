# Report Body Properties

The Report Body Properties dialog box is used to set the properties of the report body.

There are the following tabs in this dialog box: [General](#General), [Border](#Border) and [Others](#Others).

**OK**

Applies the settings and closes this dialog box.

**Cancel**

Cancels the settings and closes this dialog box.

**Help**

Displays the help document about this feature.

## General

This tab shows some general information of the report body.

![Report Body Properties dialog - General tab](https://devnet.logianalytics.com/hc/article_attachments/4404936910871/rptbdyprpty_gnrl.gif)

**Name**

Specifies the display name of the report body, which will be shown on its shortcut menu.

**Show NLS Value**

Select this option to show the translated name for the display name of the object in the Name text box if you have enabled the NLS feature and translated it.

If selected, this option takes effect only when the display name of the object is not modified.

**Background**

Specifies the background color and fill effect of the report body.

To change the color, select the color indicator to access the [Select Color](https://devnet.logianalytics.com/hc/en-us/articles/1500009746261-Select-Color) dialog box and then specify a new color, or type a color string in the format #RRGGBB. If you want to make the background transparent, type Transparent in the text box.

## Border

This tab shows information about borders of the report body.

![Report Body Properties dialog - Border tab](https://devnet.logianalytics.com/hc/article_attachments/4404936912023/rptbdyprpty_border.gif)

**Color**

Specifies the border color.

**Width**

Specifies the border width.

**Top Line**

Specifies the style of the top border line.

**Bottom Line**

Specifies the style of the bottom border line.

**Left Line**

Specifies the style of the left border line.

**Right Line**

Specifies the style of the right border line.

## Others

You can use this tab to view and configure some miscellaneous settings.

![Report Body Properties dialog - Others tab](https://devnet.logianalytics.com/hc/article_attachments/4404936912663/rptbdyprpty_other.gif)

**TOC Anchor**

Specifies whether or not to add the node that represents the report body to the TOC tree in the TOC Browser.

**Suppress When No Records**

Specifies whether to display the report body in the report result when no record is returned to its parent data component.

**Export to XLS**

If true, the report body will be exported when you save the report result as an XLS file (make sure to check Data Format in the Export dialog box).

**Export to CSV**

If true, the report body will be exported when you save the report result as a TXT file with Delimited Format selected.
