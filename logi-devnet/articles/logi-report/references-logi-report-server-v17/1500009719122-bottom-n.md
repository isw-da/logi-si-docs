# Bottom N

The Bottom N dialog box is used to filter data to display records that meet the [Bottom N condition](https://devnet.logianalytics.com/hc/en-us/articles/1500009750441-Applying-Filters#Component).

![Bottom N dialog](https://devnet.logianalytics.com/hc/article_attachments/4404942604439/bottomn.gif)

**Value**

Type a positive integer number N here so that records with the field value equal to one of the bottom N field values will be displayed.

**OK**

Applies the settings and closes the dialog box.

**Cancel**

Closes the dialog box and discards any changes.

**Help**

Displays the help document about this feature.
