# Preference

The Preference dialog box is used to [set preferences](https://devnet.logianalytics.com/hc/en-us/articles/1500009724582-Managing-User-Accounts#Preference) for the specified user. It appears when an admin user selects the Preference link of a user in the Control column of the Administration > Security > User page in the server console.

![Preference dialog](https://devnet.logianalytics.com/hc/article_attachments/4404942475543/prfrnc.gif)

It contains two subjects: Customize Server Preferences and Customize Profile. You can specify the [server preferences](https://devnet.logianalytics.com/hc/en-us/articles/1500009744941-Configuring-Server-Profile) and [Page Report Studio preferences](https://devnet.logianalytics.com/hc/en-us/articles/1500009750581-Customizing-Page-Report-Studio-Profile) for the user accordingly.
