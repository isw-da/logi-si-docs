# References Logi Report Server v17

Logi Report Server Help System has two different types of references:

* [Dialog Reference](https://devnet.logianalytics.com/hc/en-us/articles/1500009719042-Dialog-Reference)
* [Web Report Object Property Reference](https://devnet.logianalytics.com/hc/en-us/articles/1500009750621-Web-Report-Object-Property-Reference)

The purpose of these references is to make the use of Logi Report Server much easier and more convenient by providing you with an overview of the functions and properties of objects.
