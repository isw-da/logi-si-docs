# Select Data Source

The Select Data Source dialog box is used to select the data source that contains the parameter you want to [insert to a URL](https://devnet.logianalytics.com/hc/en-us/articles/1500009745021-Creating-and-Saving-Dashboards#URL). It appears when you select the Add Parameter button in the [Insert URL Frame](https://devnet.logianalytics.com/hc/en-us/articles/1500009746941-Insert-URL-Frame) dialog box or [Edit URL Frame](https://devnet.logianalytics.com/hc/en-us/articles/1500009720062-Edit-URL-Frame) dialog box if no data source has been specified, or select the Change Data Source button in the [Select Parameter](https://devnet.logianalytics.com/hc/en-us/articles/1500009720282-Select-Parameter) dialog box.

![Select Data Source dialog](https://devnet.logianalytics.com/hc/article_attachments/4404936892183/slctdtsrc.gif)

**Folder**

Specifies the folder containing the required data source.

**Catalog**

Displays the catalog containing
the required data source.

**Data Source**

Displays the selected data source.

**OK**

Selects the data source and closes this dialog box.

**Cancel**

Cancels the selection and closes this dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404936816535/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404942519575/btn_close.gif)

Ignores the setting and closes this dialog box.
