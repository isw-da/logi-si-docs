# Insert Summary Column

The Insert Summary Column dialog box is used to [insert a new summary column](https://devnet.logianalytics.com/hc/en-us/articles/1500009723442-Manipulating-Data-Components#InsertColumn) into a table.

![Insert Summary Column dialog](https://devnet.logianalytics.com/hc/article_attachments/4404942491799/insrtsumclmn.gif)

**Resources**

Lists all the available data resources for the new summary column. Select the resource you want from the list.

**OK**

Inserts the column with the specified resource and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
