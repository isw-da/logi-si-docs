# Open Document

The Open Document dialog box is used to open a dashboard or an analysis template in JDashboard. It appears when you select the Open button ![Open button](https://devnet.logianalytics.com/hc/article_attachments/4404942445591/btn_open.gif) on the toolbar, or select the Options button ![Options button](https://devnet.logianalytics.com/hc/article_attachments/4404936890647/btn_dshbrd_optn.gif) and select Open from the option list when your Logi Report Server is enabled with Visual Analysis.

![Open Document dialog](https://devnet.logianalytics.com/hc/article_attachments/4404936895255/opendoc.gif)

**Open In**

Browse to the directory in the server resource tree where the target dashboard or analysis template is located and then select the resource. Use the button ![Up Level button](https://devnet.logianalytics.com/hc/article_attachments/4404936778135/btn_dshbrd_up.gif) to go to the parent folder.

The resource table shows the resources in the current directory.

**File Name**

Shows the selected resource's file name.

**OK**

Closes this dialog box and opens the selected resource.

**Cancel**

Cancels opening the resource and closes this dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404936816535/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404942519575/btn_close.gif)

Ignores the setting and closes this dialog box.
