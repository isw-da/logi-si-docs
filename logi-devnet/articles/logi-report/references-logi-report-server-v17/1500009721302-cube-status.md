# Cube Status

The Cube Status dialog box is used to view the information about whether the data components in the current report apply in-memory cubes and the reasons if not. It appears after you select the Cube Status button ![Cube Status button](https://devnet.logianalytics.com/hc/article_attachments/4404936748055/btn_cbstts.gif) on the toolbar.

![Cube Status dialog](https://devnet.logianalytics.com/hc/article_attachments/4404936868503/cbstts.gif)

**Component**

Lists the data components in the current report.

**Business View**

Shows the business view each data component uses.

**Data Source**

Shows the catalog data source that contains the business view.

**Catalog**

Shows the path and name of the catalog in which the business view is saved.

**Cube Status**

Shows whether a data component applies an in-memory cube. This column is sortable.

**Reason**

Shows the reason why a data component does not apply an in-memory cube. The reasons may not be displayed fully and you can put the mouse pointer on a reason to get its full content.

**OK**

Closes this dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404936816535/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404942519575/btn_close.gif)

Closes this dialog box.
