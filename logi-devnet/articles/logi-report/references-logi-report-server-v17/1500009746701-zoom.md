# Zoom

The Zoom dialog box is used to set a magnification of the report page.

![Zoom dialog](https://devnet.logianalytics.com/hc/article_attachments/4404942575255/zoom.gif)

**Zoom to**

Specifies the magnification in percentage.

**OK**

Closes this dialog box and zooms the report page in or out to the magnification.

**Cancel**

Closes this dialog box without changing the magnification of the report page.

**Help**

Displays the help document about this feature.
