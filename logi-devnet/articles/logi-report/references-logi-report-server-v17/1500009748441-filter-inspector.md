# Filter Inspector

The Filter Inspector dialog box lists all the filters the current web report is using for your information. It appears when you select the Inspector button in the [Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500009748421-Filter) dialog box.

![Filter Inspector dialog](https://devnet.logianalytics.com/hc/article_attachments/4404936858903/fltrinspct.gif)

**Query Filter**

Lists the filters created via the [Query Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500009751481-Applying-Filters#BV) dialog box.

**Dialog Filter**

Lists the filters created via the [Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500009751481-Applying-Filters#Dialog) dialog box.

**On-screen Filter**

Lists the filters created via the [Filter panel](https://devnet.logianalytics.com/hc/en-us/articles/1500009751481-Applying-Filters#Panel) and via [filter controls](https://devnet.logianalytics.com/hc/en-us/articles/1500009751581-Using-Web-Controls#Filter).

**Go To Filter**

Lists all the go to filters that are created when performing the [go-to-by-value action](https://devnet.logianalytics.com/hc/en-us/articles/1500009724802-Going-Through-the-Report-Data#Go-to-by-value) on the report.

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Cancels the changes and exits the dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404936816535/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404942519575/btn_close.gif)

Ignores the setting and closes this dialog box.
