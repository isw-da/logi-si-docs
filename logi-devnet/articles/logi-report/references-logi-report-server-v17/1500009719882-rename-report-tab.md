# Rename Report Tab

The Rename Report Tab dialog box is used to rename the current report tab.

![Rename Report Tab dialog](https://devnet.logianalytics.com/hc/article_attachments/4404936913687/renmrpt.gif)

**Input New Name**

Specifies the new name for the report tab.

**OK**

Applies the new name to the report tab and closes this dialog box.

**Cancel**

Discards the setting and closes this dialog box.

**Help**

Displays the help document about this feature.
