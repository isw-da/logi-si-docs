# Add Language

The Add Language dialog box is used to add one or more languages, in which to run a resource. It appears when an admin user selects ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404936734999/btn_add1.gif)above the Language box in the [NLS Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009720602-NLS-Editor) when editing NLS for a resource, or select OK in the [Select Language Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009747981-Select-Language-Source) dialog box with Languages Supported by Logi Report selected.

![Add Language dialog](https://devnet.logianalytics.com/hc/article_attachments/4404942566167/addlng.gif)

**Available Languages**

Lists all the languages that are available for you to choose. Select the required languages from the box.

**OK**

Adds the selected languages and closes the dialog box.

**Cancel**

Cancels the action and exits the dialog box.

**Help**

Displays the help document about this feature.
