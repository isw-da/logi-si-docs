# Select Catalog

The Select Catalog helps you to select the catalog for [creating a web report in the quick start way](https://devnet.logianalytics.com/hc/en-us/articles/1500009751621-Quick-Start-with-a-Table-Report).

Report Server displays the dialog box when you do any of the following:

* In the Logi Report Server Start Page, select Web Report in the Create category.
* Select New > Web Report in the server console > Resources page but the currently opened folder contains no catalog.
* Select the Change Catalog button in the [Select Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009722222-Select-Data-Source#Report) dialog box.

![Select Catalog dialog](https://devnet.logianalytics.com/hc/article_attachments/4404942525079/slctctlg.gif)

In the dialog box, all the available catalogs that have been published to Logi Report Server are listed in a tree structure. You can select the catalog you want to use for the report from the tree.

**OK**

Specifies to use the selected catalog for creating the report and directs to the [Select Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009722222-Select-Data-Source#Report) dialog box.

**Cancel**

Cancels the selection and closes this dialog box. Available only when the dialog box is opened by selecting Change Catalog in the Select Data Source dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404936816535/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404942519575/btn_close.gif)

Cancels the selection and closes this dialog box. Available only when the dialog box is opened by selecting Change Catalog in the Select Data Source dialog box.
