# New Trigger

The New Trigger dialog box is used to create a [trigger](https://devnet.logianalytics.com/hc/en-us/articles/1500009723162-Managing-Triggers). It appears when an admin user clicks New Trigger in the Administration > Other > Triggers page in the server console.

![New Trigger dialog](https://devnet.logianalytics.com/hc/article_attachments/4404942501271/nwtrigr.gif)

**Trigger Name**

Specifies the name for the new trigger.

**Conflict Check**

Checks if the name has been used for another trigger.

**Description**

Specifies the description of the trigger.

**OK**

Creates the trigger with the information you have specified.

**Cancel**

Cancels the trigger creating process.
