# Select Role, Group and User

The Select Role, Group and User dialog box is used to select the scheduling recipients to use their e-mail addresses to send the e-mail. It appears when you select ![Select User button](https://devnet.logianalytics.com/hc/article_attachments/4404942482711/btn_chsr5.gif) next to the To/Cc/Bcc text box in the Publish > [To E-mail tab](https://devnet.logianalytics.com/hc/en-us/articles/1500009747821-Schedule#Mail) of the Schedule dialog box when creating or editing an e-mail.

![Select Role, Group and User dialog](https://devnet.logianalytics.com/hc/article_attachments/4404936876823/slctusr.gif)

**Search box**

Searches for scheduling recipients in the list below. Type the keyword and the recipients containing the keyword will be listed.

**Scheduling recipient list**

Specifies the scheduling recipients, meaning the users, groups, and roles in the Logi Report Server security system, whose e-mail addresses you want to use to send the e-mails. Select the top check box to select or deselect all the recipients.

![This image notes any new content for version 17 of the product. For more information please see the Release Notes or What's New topics.](https://devnet.logianalytics.com/hc/article_attachments/4404942452887/__newv17.jpg "New for version 17.")Logi Report Server adopts permission control on the scheduling recipients each user can access. For details, refer to [Managing User Accounts](https://devnet.logianalytics.com/hc/en-us/articles/1500009724582-Managing-User-Accounts#Recipient).

**Role/Group/User**

Specifies whether to display roles/groups/users in the list above.

**OK**

Applies the selected principals and exits the dialog box.

**Cancel**

Cancels the selection and closes the dialog box.

**Help**

Displays the help document about this feature.
