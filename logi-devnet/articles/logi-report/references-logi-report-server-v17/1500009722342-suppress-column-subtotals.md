# Suppress Column Subtotals

The Suppress Column Subtotals dialog box is used to customize which subtotals of the column fields in a crosstab will be suppressed and which will be shown. It appears when you select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404936798615/btn_chsr4.gif) next to the Suppress Column Subtotals option in the Layout tab of the [Crosstab Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009721442-Crosstab-Wizard#Layout) or [Insert Crosstab](https://devnet.logianalytics.com/hc/en-us/articles/1500009748981-Insert-Crosstab#Layout)  dialog box, or in the Layout tab for crosstab in the [Web Report Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009722522-Web-Report-Wizard#Crosstab).

![Suppress Column Subtotal dialog](https://devnet.logianalytics.com/hc/article_attachments/4404936825623/sprsclmnsbtl.gif)

**Same for all**

If the option is selected, the setting for the subtotal of the first column field in below box will be applied to all the subtotals of the column fields in the crosstab.

**Name**

Lists the column fields excluding the one on the topmost column group level.

**Suppress**

Specifies whether the subtotal for a column field will be suppressed. To specify the value, select in the value cell and select true or false from the drop-down list.

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404936816535/btn_help.gif)**

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404942519575/btn_close.gif)

Does not retain any changes and closes the dialog box.
