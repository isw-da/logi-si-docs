# Select Target

The Select Target dialog box is used to specify the target resources the implementation of the [Dashboard Listener API](https://devnet.logianalytics.com/hc/en-us/articles/1500009744301-Applying-the-Implementations-of-the-Dashboard-Listener-API) will be applied to. It appears when an admin user selects ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404936734999/btn_add1.gif) in the [New Dashboard Listener](https://devnet.logianalytics.com/hc/en-us/articles/1500009747541-New-Dashboard-Listener#Resource) dialog box.

![Select Target dialog](https://devnet.logianalytics.com/hc/article_attachments/4404936877207/slcttgt.gif)

**All Dashboards**

If selected, the implementation will apply to all dashboards including the newly created ones that have not been saved yet.

**Customize**

If selected, you can specify the dashboards you want the implementation to apply to.

* **Newly Created Dashboards**  
   If selected, the implementation will apply to all newly created dashboards that have not been saved yet.
* **Existing Dashboards**  
   If selected, you can further specify one or more folders, and the implementation will apply to all of the dashboards in the selected folders.

**OK**

Applies the setting and exits the dialog box.

**Cancel**

Cancels the setting and closes the dialog box.

**Help**

Displays the help document about this feature.
