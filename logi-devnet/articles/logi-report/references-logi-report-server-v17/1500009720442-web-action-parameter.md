# Web Action - Parameter

The Web Action - Parameter dialog box is used to edit the parameter values used to re-run a data component when it [receives a synchronization message.](https://devnet.logianalytics.com/hc/en-us/articles/1500009718982-Synchronizing-the-Data-Components-in-a-Dashboard#Receive) It appears when the web action \*Parameter is selected and you select ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/4404936888087/btn_dshbrd_edt1.gif) in the Actions column of the [Receive Sync](https://devnet.logianalytics.com/hc/en-us/articles/1500009747021-Receive-Sync) dialog box.

![Web Action - Parameter dialog](https://devnet.logianalytics.com/hc/article_attachments/4404936889111/wbactn_param.gif)

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404936888599/btn_dshbrd_add1.gif)

Adds a line to specify parameter value.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404942567703/btn_dshbrd_rmv1.gif)

Removes the selected parameter.

**Parameter Name**

Lists all the parameters used by the data component. Select one from the drop-down list or type the parameter name by yourself.

**Operator**

Shows the operator to compose the expression. It can only be =.

**Parameter Value**

Specifies the value for the parameter.

**OK**

Accepts the changes and closes this dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404936816535/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404942519575/btn_close.gif)

Ignores the setting and closes this dialog box.
