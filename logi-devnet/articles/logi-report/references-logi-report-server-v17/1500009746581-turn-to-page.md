# Turn to Page

The Turn to Page dialog box is used to input the page number so as to turn to the specified page.

![Turn to Page dialog](https://devnet.logianalytics.com/hc/article_attachments/4404936905239/tn2pg.gif)

**Input Page Number**

Specifies the page number you would like to turn to.

**Go**

Goes to the specified page.

**Cancel**

Closes this dialog box without turning the report page.

**Help**

Displays the help document about this feature.
