# New Group

The New Group dialog box is used to create a [group](https://devnet.logianalytics.com/hc/en-us/articles/1500009724542-Managing-Groups). It appears when an admin user selects the New Group link in the Administration > Security > Group page in the server console.

![New Group dialog](https://devnet.logianalytics.com/hc/article_attachments/4404942477591/nwgrp.gif)

**Group Name**

Specifies the name for the new group.

**Parent Group Name**

Specifies the parent group for the new group.

**Description**

Specifies the description for the new group.

**Privileges**

Specifies whether the new group has the following privileges:

* **Publish**  
   The privilege of publishing resources to Logi Report Server.
* **Advanced Properties**  
   The privilege of viewing advanced version properties information, such as catalog connections and report related resources.

**OK**

Creates a group with the information you have specified.

**Cancel**

Cancels operations and closes the dialog box.
