# Conditional Formatting

The Conditional Formatting dialog box is used to [add conditional formats to values](https://devnet.logianalytics.com/hc/en-us/articles/1500009750461-Adding-Conditional-Formats-to-Fields) of the selected field.

![Conditional Formatting dialog](https://devnet.logianalytics.com/hc/article_attachments/4404942494359/cndtnlfmt.gif)

**Condition**

Displays all the conditions created on the field.

* ![Create Condition](https://devnet.logianalytics.com/hc/article_attachments/4404936734999/btn_add1.gif)  
   Creates a new condition using the [Edit Conditions](https://devnet.logianalytics.com/hc/en-us/articles/1500009719242-Edit-Conditions) dialog box.
* ![Edit Condition](https://devnet.logianalytics.com/hc/article_attachments/4404936751127/btn_pgrpt_edit.gif)  
   Edits the selected condition using the [Edit Conditions](https://devnet.logianalytics.com/hc/en-us/articles/1500009719242-Edit-Conditions) dialog box.
* ![Delete button](https://devnet.logianalytics.com/hc/article_attachments/4404936717719/btn_delete.gif)  
   Removes the selected condition.
* **Priority**  
   Specifies the priority of each condition.
  + **![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404936751383/btn_mvup1.gif)**  
     Moves a condition up for a higher priority.
  + **![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404936733463/btn_mvdown1.gif)**  
     Moves a condition down for a lower priority.

**Format**

You can specify a format for each condition. The format will be applied to the field values when the specified condition is fulfilled.

* **Font**  
   Specifies the font type for the field values.
* **Border**  
   Specifies the border line style for the field values.
* **Size**  
   Specifies the font size for the field values.
* **Bold**  
   Specifies whether or not to bold the field values.
* **Italic**  
   Specifies whether or not to make the field values italic.
* **Underline**  
   Specifies whether or not to underline the field values.
* **Foreground Color**   
   Specifies the foreground color for the field values.
* **Background Color**   
   Specifies the background color for the field values.
* **Sample Text**  
   Displays a preview sample of your settings.
* **Blinking Text**  
   Specifies whether or not to make the field values blink.
  + **Duration**  
     Specifies how long it takes the field values to complete the transition from the foreground color to the transparent color, in seconds.

**OK**

Applies the settings and exits the dialog box.

**Cancel**

Cancels the settings and closes the dialog box.

**Help**

Displays the help document about this feature.
