# Select Values

The Select Values dialog box is used to specify one or more values to [apply a filter criterion](https://devnet.logianalytics.com/hc/en-us/articles/1500009750441-Applying-Filters#Component).

![Select Value dialog](https://devnet.logianalytics.com/hc/article_attachments/4404936908951/slctvalue.gif)

**Available**

Specifies the values from this list to filter report data with.

![Add Value](https://devnet.logianalytics.com/hc/article_attachments/4404936738839/btn_additem.gif)

Adds the selected value to the Chosen list.

**Chosen**

Lists all the values that have been chosen.

**Add**

Adds a new line to the Chosen list for you to type a value.

**Edit**

Edits the selected item in the Chosen list. You can also double-click the item to edit it.

**Remove**

Removes the selected item from the Chosen list.

**Go to**

Goes to the specified value list. Type a number N here and select the Get button to fetch a value list starting from the Nth value.

**Prev**

Fetches the previous value list.

**Next**

Fetches the next value list.

**Number of List Items**

Specifies the number of list items here and select the Get button to fetch a value list.

**OK**

Applies all changes and closes this dialog box.

**Cancel**

Closes the dialog box and deletes any unsaved changes.

**Help**

Displays the help document about this feature.
