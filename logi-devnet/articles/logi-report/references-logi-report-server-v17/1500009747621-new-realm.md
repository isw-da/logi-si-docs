# New Realm

The New Realm dialog box is used to create a [realm](https://devnet.logianalytics.com/hc/en-us/articles/1500009751261-Managing-Realms). It appears when an admin user selects the New Realm link in the Administration > Security > Realm page in the server console.

![New Realm dialog](https://devnet.logianalytics.com/hc/article_attachments/4404936762775/nwrealm.gif)

**Realm Name**

Specifies the name of the new realm.

**Scheme**

Specifies the scheme of the new realm. It can be one of the following:

* **Basic Authentication**  
   Specifies to use the Base64 method (re-conversion is easy).
* **Digest Authentication**  
   Specifies to use the MD5 digest method (re-conversion is impossible).

**OK**

Creates a realm with the information you have specified.

**Cancel**

Cancels all operations and closes the dialog box.
