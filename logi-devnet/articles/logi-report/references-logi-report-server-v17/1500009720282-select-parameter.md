# Select Parameter

The Select Parameter dialog box is used to [insert a parameter into the URL of the web page](https://devnet.logianalytics.com/hc/en-us/articles/1500009745021-Creating-and-Saving-Dashboards#URL) that you want to add to a dashboard. It appears when you select the Add Parameter button in the [Insert URL Frame](https://devnet.logianalytics.com/hc/en-us/articles/1500009746941-Insert-URL-Frame) dialog box or [Edit URL Frame](https://devnet.logianalytics.com/hc/en-us/articles/1500009720062-Edit-URL-Frame) dialog box if a data source has already been specified, or select a data source and select OK in the [Select Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009747081-Select-Data-Source) dialog box.

![Select Parameter dialog](https://devnet.logianalytics.com/hc/article_attachments/4404942569751/slctprmtr.gif)

**Parameters**

Lists all the available parameters in the currently selected data source for you to select one to insert to the URL.

**Change Data Source**

Opens the [Select Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009747081-Select-Data-Source) dialog box to change the data source.

**OK**

Closes this dialog box and inserts the selected parameter to the URL.

**Cancel**

Cancels the insertion and closes this dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4404936816535/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4404942519575/btn_close.gif)

Ignores the setting and closes this dialog box.
