# Sort

The Sort dialog box is used to specify the settings in order to [sort the records or groups](https://devnet.logianalytics.com/hc/en-us/articles/1500009723482-Sorting-Report-Data) in a banded object.

![Sort dialog](https://devnet.logianalytics.com/hc/article_attachments/4404942489623/sort.gif)

**Sort in Scope**

Specifies on which level the sorting is based.

![Delete button](https://devnet.logianalytics.com/hc/article_attachments/4404936748823/btn_pgrpt_no.gif)

Deletes the specified sort expression line.

**Field**

Specifies the field by which to sort the records or groups.

**Ascend**

Specifies to sort the data ascending.

**Descend**

Specifies to sort the data descending.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404942457495/btn_mvup.gif)

Moves up a sort expression line.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404936729239/btn_mvdown.gif)

Moves down a sort expression line.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404936734999/btn_add1.gif)

Adds a new sort expression line after the current one. This button appears after the first sort condition has been composed.

**OK**

Accepts the changes and re-loads the report result using the sort settings with this dialog box closed.

**Cancel**

Cancels the operation and closes this dialog box.

**Apply**

Accepts the changes and re-loads the report result using the sort settings with this dialog box displayed.

**Reset**

Restores this dialog box to its opening status.

**Help**

Displays the help document about this feature.
