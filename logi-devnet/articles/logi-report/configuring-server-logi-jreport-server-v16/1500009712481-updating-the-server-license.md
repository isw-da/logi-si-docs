# Updating the Server License

Specific features of Logi JReport Server are restricted by [licenses](https://devnet.logianalytics.com/hc/en-us/articles/1500009691022-Logi-JReport-licenses). While using Logi JReport Server, you may need to update the server license with a new one enabled with more features.

**To update the server license:**

1. In the Logi JReport Server console, point to **Administration** on the system toolbar, and then select **Configuration** >  **License Key** from the drop-down menu.
2. In the displayed page, select the **here** link to update the key.

   ![Update Server License](https://devnet.logianalytics.com/hc/article_attachments/4412131509655/config_license1.gif)
3. Enter the valid user ID and license key received from your Logi Analytics account manager.

   ![Update Server License](https://devnet.logianalytics.com/hc/article_attachments/4412131509911/config_license2.gif)
4. Select the **Submit** button to apply the new license key.

   After successful update, you will be prompt with the new access information of your Logi JReport Server.

In the case when your server license expires, after the server is started, Logi JReport will also prompt you to update your license. After the license information is updated successfully, you will be able to continue using the server.
