# Searching for Resources

On the Logi JReport Console/Administration> Resources page you can use the quick search toolbar to easily locate any resources including folders in the server resource tree.

1. On the root of the server resource tree or browse to the desired folder, select the **Search**![](https://devnet.logianalytics.com/hc/article_attachments/4404926626967/btn_srch.gif) button on the path bar to display the quick search toolbar.

   ![Search Resource](https://devnet.logianalytics.com/hc/article_attachments/4404920488471/mng_rsrc_srch.gif)
2. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404920383255/btn_mvdown1.gif) on the toolbar to specify the search options.
   * **Highlight All**   
      Specifies whether to highlight all matched text.
   * **Match Case**   
      Specifies whether to search for text that meets the case of the typed text.
   * **Match Whole Word**   
      Specifies whether to search for text that looks the same as the typed text.
3. The quick search toolbar treats the display names and descriptions of the resources as strings and searches by consecutive text. Type in the text you want to search for and the resources containing the matched text will be listed.
4. To cancel the search operation, select ![](https://devnet.logianalytics.com/hc/article_attachments/4404920361623/btn_delete.gif) beside the quick search toolbar.
