# Managing Tasks

Usually, a task is a set of operations you perform on Logi JReport Server to achieve a goal, such as publishing a folder, deleting a resource, and viewing resource properties. However, the tasks that Logi JReport Server manages are those associated with report-running issues, the report-running tasks.

Logi JReport Server allows you to quickly view a report result (Run), view a report result using selected options and parameters (Advanced Run), and schedule a report to run unattended at a specific time or periodically (Schedule). These are the fundamental modes that Logi JReport Server uses to perform its report-running tasks.

You can view the status of these report-running tasks, such as scheduled tasks that are waiting to be performed by Logi JReport Server, the tasks that are currently being performed, and the tasks that have already been performed.
