# Managing Logi JReport Server Logi JReport Server Guide v15

The administration tasks on Logi JReport Server are mainly about the management of resources, versions, tasks, server data, cached report data, in-memory cubes and triggers. The topics listed below discuss these task types in detail:

* [Managing Resources](https://devnet.logianalytics.com/hc/en-us/articles/1500009648782-Managing-Resources)
* [Managing Versions](https://devnet.logianalytics.com/hc/en-us/articles/1500009673701-Managing-Versions)
* [Managing Tasks](https://devnet.logianalytics.com/hc/en-us/articles/1500009673681-Managing-Tasks)
* [Managing Server Data](https://devnet.logianalytics.com/hc/en-us/articles/1500009648722-Managing-Server-Data)
* [Managing Cached Report Data](https://devnet.logianalytics.com/hc/en-us/articles/1500009673381-Managing-Cached-Report-Data)
* [Managing In-Memory Cubes](https://devnet.logianalytics.com/hc/en-us/articles/1500009648682-Managing-In-memory-Cubes)
* [Managing Triggers](https://devnet.logianalytics.com/hc/en-us/articles/1500009648982-Managing-Triggers)

After going through these topics, you will be able to manage your Logi JReport Server more effectively.

**Note:** A [Logi JReport Live license for Logi JReport Server](https://devnet.logianalytics.com/hc/en-us/articles/1500009648262-Logi-JReport-Licenses#ServerLive) is required in order to use web report related features, and a [JDashboard license](https://devnet.logianalytics.com/hc/en-us/articles/1500009648262-Logi-JReport-Licenses#JDashboard) is required in order to use dashboard and library component related features. If you do not have the licenses, contact your Jinfonet Software account manager to obtain them.
