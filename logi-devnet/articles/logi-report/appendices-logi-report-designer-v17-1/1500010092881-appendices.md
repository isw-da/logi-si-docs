# Appendices

The Logi Report Designer Help System has the following appendices:

* [Appendix 1: Formula Functions](https://devnet.logianalytics.com/hc/en-us/articles/1500010093961-Appendix-1-Formula-Functions)
* [Appendix 2: Formula Operators](https://devnet.logianalytics.com/hc/en-us/articles/1500010093981-Appendix-2-Formula-Operators)
* [Appendix 3: Parameters of User-Defined Web Actions](https://devnet.logianalytics.com/hc/en-us/articles/1500010094001-Appendix-3-Parameters-of-User-Defined-Web-Actions)

The purpose of these appendices is to make the use of Logi Report Designer much easier and more convenient by introducing you to the types of formula functions and operators, as well as the parameter values required for building user-defined web actions.
