# Logi Report Tutorial v17.1 Overview

Thank you for your interest in Logi Report, the leading Embedded Reporting Solution for Java Applications. With Logi Report, intuitive and sophisticated reporting can be customized and integrated as a natural extension of any application.

This Tutorial is intended to provide an overview of the Logi Report solution, embedded reporting concepts, and detailed step-by-step lessons on using Logi Report. To enable lessons to be completed in a reasonable time frame, not all of the features of Logi Report are covered.

This Tutorial requires that you have already installed both Logi Report Server and Logi Report Designer on a Windows machine and therefore file names and paths, and screen shots in the Tutorial are based on the Windows operating system. However Logi Report can run on multiple platforms.

The Tutorial contains the following parts. You can access the information by looking at the Table of Contents, or by using Search.

* [Part I: Logi Report Overview](https://devnet.logianalytics.com/hc/en-us/articles/1500010063062-Part-I-Logi-Report-Overview)  
  This part briefly introduces reports and Logi Report.
* [Part II: Logi Report Basics for Developers](https://devnet.logianalytics.com/hc/en-us/articles/1500010056362-Part-II-Logi-Report-Basics-for-Developers)  
  This part introduces the basic knowledge of Designer for report developers, including its development environment, catalogs, connections, queries, business views, and reports.
* [Part III: Advanced Reporting](https://devnet.logianalytics.com/hc/en-us/articles/1500010056042-Part-III-Advanced-Reporting)  
  This part talks about some advanced reporting features of Logi Report based on the story of a fictional retail coffee chain: how to use Designer to create different types of reports for the company; how the reports and related resources are published to Server and managed on the server, and so on.
* [Part IV: End User Experience](https://devnet.logianalytics.com/hc/en-us/articles/1500010093601-Part-IV-End-User-Experience)  
  This part focuses on the most powerful yet easy-to-use web tools of Logi Report: JDashboard, Visual Analysis, Web Report Studio, and Page Report Studio. End users can use them to create and edit their own reports, dashboards, and dynamic analysis without expert assistance.
