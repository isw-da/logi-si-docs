# Modifying the Joins Between Data Resources

When you create a business view, you need to create joins between the data resources you have selected to use for the business view to define the relationships between them. Logi JReport allows you to join data resources that come from different connections and this type of joins are called distributed joins.

**To modify the joins between data resources in a business view:**

1. In the Business View Editor, select **Join** on the toolbar or select **Menu** > **Tools** > **Join**. The Query Editor appears.
2. [Add more data resources](https://devnet.logianalytics.com/hc/en-us/articles/1500009570762-Adding-and-Deleting-Tables-in-a-Query) if needed and then select the columns
   in the added resources to use in the business view.
3. [Set up the joins](https://devnet.logianalytics.com/hc/en-us/articles/1500009592501-Joining-Tables-in-a-Query) between the data resources, then select **OK** to return to Business View Editor.

The newly added data resources will be displayed in the Resource Objects panel in the Business View Editor. You can then [add them as view elements to the business view](https://devnet.logianalytics.com/hc/en-us/articles/1500009562642-Adding-and-Modifying-Elements) or [use them in hierarchies](https://devnet.logianalytics.com/hc/en-us/articles/1500009583021-Defining-Hierarchies).
