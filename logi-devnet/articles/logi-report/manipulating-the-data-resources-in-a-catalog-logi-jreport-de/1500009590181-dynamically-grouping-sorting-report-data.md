# Dynamically Grouping/Sorting Report Data

An important usage of parameters is to use them to dynamically group and sort data in a report, which enables the end users to specify the grouping and sorting conditions at runtime according to their requirements.

For details, refer to the topics [Grouping Data Dynamically](https://devnet.logianalytics.com/hc/en-us/articles/1500009584341-Grouping-Data-Dynamically) and [Sorting Data Dynamically](https://devnet.logianalytics.com/hc/en-us/articles/1500009563722-Sorting-the-Data#Dynamic).
