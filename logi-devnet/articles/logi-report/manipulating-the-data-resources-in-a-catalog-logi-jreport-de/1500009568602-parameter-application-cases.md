In Logi JReport, parameters can be used in many situations. The usage of parameters in Logi JReport Designer can greatly facilitate your reporting work.

The following topics detail the use of parameters:

> * [Binding a Parameter to a DBField](https://devnet.logianalytics.com/hc/en-us/articles/1500009590081-Binding-a-Parameter-to-a-DBField)
> * [Referencing Parameters in a Join Condition](https://devnet.logianalytics.com/hc/en-us/articles/1500009568682-Referencing-Parameters-in-a-Join-Condition)
> * [Applying RLS to a Parameter](https://devnet.logianalytics.com/hc/en-us/articles/1500009590061-Applying-RLS-to-a-Parameter)
> * [Filtering a Parameter with Another Parameter](https://devnet.logianalytics.com/hc/en-us/articles/1500009568642-Filtering-a-Parameter-with-Another-Parameter) (cascading parameters)
