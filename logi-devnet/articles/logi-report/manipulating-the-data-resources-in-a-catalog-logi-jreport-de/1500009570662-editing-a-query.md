# Editing a Query

You can edit existing queries in a catalog at any time. To edit a query, in the Catalog Manager right-click the query and select **Edit Query** from the shortcut menu, then in the [Query Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009567142-Query-Editor-Dialog) edit the query as required. For example, you can add additional data resources to use in the query and define join relationships among the resources.
