# Using Design API to Design Reports

You can design and modify reports and library components programmatically via Logi Report Design API by creating a Designer instance in your Java program, besides using the Design panel in Logi Report Designer. You can edit the layout of a report or library component, insert objects such as tables, crosstabs, charts, DBFields, formulas, labels, and images, and change property values.

For more information, refer to Using Design API to Design Reports in the *Logi Report Designer Guide*.
