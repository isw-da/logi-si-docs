# Profile Control and Customization

Logi Report Server Profile enables you to customize the UI, formats, styles, specific features, and preferences of Server.

On the Server Console > Administration > Server Profile page, administrators can define initial settings of Server and customize features by configuring the profile. Any user can also configure their own profile on the Server Console > My Profile page. The difference lies in that any settings on the My Profile page can only apply to this user, while the settings on the Server Profile page apply to all users.
