# Quick Start a Web Report

The Quick Start feature enables quick creation of components and reports. You only need to select data fields from a business view and then Web Report Studio generates a table report based on the data fields. You can then change the component type as need be. This saves a lot of time compared to building a report using the Web Report Wizard, in which you have to choose the module and data component types and define the data in different components.
