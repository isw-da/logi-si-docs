# One-Select Ad Hoc Operations

Logi Report supports several one-select ad hoc operations, with which you can easily change the data presentation of a report and sort and filter the report data.

This guide explores the following one-select actions:

* [One-Select to Pivot Crosstab](https://devnet.logianalytics.com/hc/en-us/articles/5741471653527-One-Select-to-Pivot-Crosstab)
* [One-Select to Swap Group/Value in Chart](https://devnet.logianalytics.com/hc/en-us/articles/5741471688215-One-Select-to-Swap-Group-Value-in-Chart)
* [One-Select to Sort and Filter Table](https://devnet.logianalytics.com/hc/en-us/articles/5741491500695-One-Select-to-Sort-and-Filter-Table)
* [One-Select to Drill Down Report Data](https://devnet.logianalytics.com/hc/en-us/articles/5741465226903-One-Select-to-Drill-Down-Report-Data)
* [One-Select to Convert Table/Crosstab/Chart](https://devnet.logianalytics.com/hc/en-us/articles/5741456639127-One-Select-to-Convert-Table-Crosstab-Chart)
* [Chart Zoom via Range Selection](https://devnet.logianalytics.com/hc/en-us/articles/5741465297815-Chart-Zoom-via-Range-Selection)
