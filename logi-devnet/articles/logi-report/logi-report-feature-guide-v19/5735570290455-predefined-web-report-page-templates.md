# Predefined Web Report Page Templates

Rather than always starting from scratch to build reports, Logi Report Server administrators can also build predefined page templates with information such as customer logos, page numbers, dates, styles, and anything else required as a corporate standard for end users to readily create distribution-ready documents. This ensures that even when building ad hoc reports, end users are creating reports with the same requirements as those created by IT without any extra effort.

Administrators can define, edit, and save web report page templates using the Web Report Wizard or in Web Report Studio. Then end users can access them in the Web Report Wizard and use them to create web reports, or select the default page template in the Web Report Studio profile in advance for web reports you create using the quick start method.
