# One-Select to Swap Group/Value in Chart

You can swap the category and series axis values in a chart by one-selecting. If the chart contains one field on the category axis and two or more fields on the value axis, the fields on the value axis will be classified as one group and positioned on the series axis, in this case, the category and value axes can also be switched.
