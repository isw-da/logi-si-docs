# National Language Support

Logi Report includes a National Language Support feature, with which you can run Logi Report in different language environments. Logi Report NLS is fully customizable, enabling you to customize the application UI, and the metadata and actual data. You can edit NLS in both Designer and Server, and you can create global NLS resources that can be shared by all reports to reduce the translation cost.
