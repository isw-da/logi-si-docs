# Server Monitor

Server Monitor, a standalone web-based application used to monitor the overall performance of Server, enables you to inspect the status of Server, such as the status of the Servers in a cluster, the status of different reports, and the status of online users. Server Monitor is able to generate and display a performance chart of Server, thus you can view the performance of Server in the form of Line Chart Graph or Text. You can also use Server Monitor to maintain other administrative features of Server, such as shutting down Server, stopping waiting/running tasks, and logging out a valid UserSession.
