# Using XML File as a Data Source via OOJDBC

Logi JReport Designer supports XML as a data source which can be accessed via OOJDBC. Logi JReport also supports using XML directly as a data source which is recommended over using OOJDBC. See [XML Connections](https://devnet.logianalytics.com/hc/en-us/articles/1500009585421-XML-Connections).

The following examples explain how to use an XML file as a data source via OOJDBC:

> * [Example 1: Importing the XML File Indirectly](https://devnet.logianalytics.com/hc/en-us/articles/1500009584921-Example-1-Importing-the-XML-File-Indirectly)
> * [Example 2: Importing the XML File Directly](https://devnet.logianalytics.com/hc/en-us/articles/1500009564062-Example-2-Importing-the-XML-File-Directly)
> * [Example 3: XML File with Dynamic Parameters](https://devnet.logianalytics.com/hc/en-us/articles/1500009564082-Example-3-XML-File-with-Dynamic-Parameters)
