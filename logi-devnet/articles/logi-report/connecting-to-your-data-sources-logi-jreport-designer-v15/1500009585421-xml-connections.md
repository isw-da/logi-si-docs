# XML Connections

An XML connection contains relational data which is transformed from an XML data source.

Logi JReport Designer supports parsing of the schema from XML instances, or importing them from an XSD file and then transforming them to relational tables. The tables can then be added to the current connection from the transformed relational tables and accessed in the same way as JDBC-supplied tables.

The following topics describe tasks associated with XML connections:
