# DBMS Objects in a JDBC Connection

Tables, views, synonyms, stored procedures and SQL files are all from the raw database. If you want to use them in Logi JReport Designer, you first have to add them to a catalog via [the JDBC connection you set up](https://devnet.logianalytics.com/hc/en-us/articles/1500009584941-Setting-Up-a-JDBC-Connection) which connects Logi JReport Designer to the database driver.

The following topics discuss how to work with these objects separately:

> * [Tables/Views/Synonyms](https://devnet.logianalytics.com/hc/en-us/articles/1500009584821-Tables-Views-Synonyms)
> * [Stored Procedures](https://devnet.logianalytics.com/hc/en-us/articles/1500009564022-Stored-Procedures)
> * [Predefined SQL Files](https://devnet.logianalytics.com/hc/en-us/articles/1500009564042-Predefined-SQL-Files)
