# Editing Web Reports in Web Report Studio

Web Report Studio provides many powerful features for interfacing with reports. This topic describes how you can insert and manipulate components, create links, filter report data, use web controls, and more.

To perform most of the operations in this topic, you need to use [Edit Mode](https://devnet.logianalytics.com/hc/en-us/articles/4405684031639-Web-Report-Studio-Server-v18#Mode) of Web Report Studio.
