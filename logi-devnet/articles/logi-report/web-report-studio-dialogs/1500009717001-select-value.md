# Select Value

The Select Value dialog helps you to select a filter object like filter control to pass values between the filter objects in the primary report and the linked report. This dialog appears when you select the ![Select Value](https://devnet.logianalytics.com/hc/article_attachments/4412112493207/btn_chsr.gif) button in the Advanced tab for the Report link type in the [Insert Link](https://devnet.logianalytics.com/hc/en-us/articles/1500009716601-Insert-Link) dialog or [Edit Link](https://devnet.logianalytics.com/hc/en-us/articles/1500009689802-Edit-Link) dialog.

![Select Value dialog](https://devnet.logianalytics.com/hc/article_attachments/4412131444759/slctvlu.gif)

The resources box lists all the objects in the report. Only filter objects are available for selection. Select one to use.

**OK**

Selects a filter object and closes the dialog.

**Cancel**

Cancels the selection and closes the dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
