# Color List

The Color List dialog is used to modify the color patterns of data markers in the same data series respectively in a gauge chart or modify alternating color for a table.

This dialog appears when you do either of the following:

* Select the Color List button in the Pointer tab of the [Format Dial Gauge](https://devnet.logianalytics.com/hc/en-us/articles/1500009690022-Format-Dial-Gauge#Pointer) dialog, [Format Bar Gauge](https://devnet.logianalytics.com/hc/en-us/articles/1500009716201-Format-Bar-Gauge#Pointer) dialog or [Format Solid Gauge](https://devnet.logianalytics.com/hc/en-us/articles/1500009716381-Format-Solid-Gauge#Pointer) dialog, or in the Circular Graph tab of the [Format Solid Gauge](https://devnet.logianalytics.com/hc/en-us/articles/1500009716381-Format-Solid-Gauge#Circular) dialog or [Format Activity Gauge](https://devnet.logianalytics.com/hc/en-us/articles/1500009689922-Format-Activity-Gauge#Circular) dialog.
* Select ![Chooser button](https://devnet.logianalytics.com/hc/article_attachments/4412112488471/btn_chsr1.gif) in the value cell of the Pattern List property for a table in the Inspector panel.

![Color List dialog](https://devnet.logianalytics.com/hc/article_attachments/4412112602903/colorlist.gif)

**Colors**

Specifies the color and transparency for the chart data markers or for the alternating color of the table. Select the items in the box one by one to set the color pattern that each of the item stands for.

* ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4412131380631/btn_add1.gif)  
   Adds a new pattern item.
* ![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4412112484759/btn_delete.gif)  
   Removes the selected pattern item.
* ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4412112497431/btn_mvup.gif)  
   Moves the selected pattern item one step up.
* ![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4412112497687/btn_mvdown.gif)  
   Moves the selected pattern item one step down.
* **Fill**  
   Specifies the color schema for the pattern.

  To change the color, select the color indicator to select a color from the color palette. You can select More Colors in the color palette to access the [Color Picker](https://devnet.logianalytics.com/hc/en-us/articles/1500009715801-Color-Picker-) dialog in which you can select a color within a wider range, or select More Fill Effects to access the [Fill Effects](https://devnet.logianalytics.com/hc/en-us/articles/1500009716141-Fill-Effects) dialog to specify gradient fill effect. You can also input a color string in the format #RRGGBB directly in the text box.
* **Transparency**  
   Specifies the transparency of the color.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
