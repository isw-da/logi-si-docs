# Filter Inspector

The Filter Inspector dialog lists all the filters the current web report is using for your information. This dialog appears when you select the Inspector button in the [Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500009689882-Filter) dialog.

![Filter Inspector dialog](https://devnet.logianalytics.com/hc/article_attachments/4412131451927/fltrinspct.gif)

**Query Filter**

Lists the filters created via the [Query Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500009719141-Applying-Filters#BV) dialog.

**Dialog Filter**

Lists the filters created via the [Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500009719141-Applying-Filters#Dialog) dialog.

**On-screen Filter**

Lists the filters created via the [Filter panel](https://devnet.logianalytics.com/hc/en-us/articles/1500009719141-Applying-Filters#Panel) and via [filter controls](https://devnet.logianalytics.com/hc/en-us/articles/1500009719281-Using-Web-Controls#Filter).

**Go To Filter**

Lists all the go to filters that are created when performing the [go-to-by-value action](https://devnet.logianalytics.com/hc/en-us/articles/1500009692802-Going-Through-the-Report-Data#Go-to-by-value) on the report.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Cancels the changes and exits the dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
