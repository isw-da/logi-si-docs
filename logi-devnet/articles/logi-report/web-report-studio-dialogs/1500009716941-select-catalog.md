# Select Catalog

The Select Catalog helps you to select the catalog for [creating a web report in the quick start way](https://devnet.logianalytics.com/hc/en-us/articles/1500009692882-Quick-Start-with-a-Table-Report).

This dialog appears when you do any of the following:

* In the Logi JReport Server Start Page, select Web Report in the Create category.
* Select New > Web Report in the server console > Resources page but the currently opened folder contains no catalog.
* Select the Change Catalog button in the [Select Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009690662-Select-Data-Source#Report) dialog.

![Select Catalog dialog](https://devnet.logianalytics.com/hc/article_attachments/4412139543447/slctctlg.gif)

In the dialog, all the available catalogs that have been published to Logi JReport Server are listed in a tree structure. You can select the catalog you want to use for the report from the tree.

**OK**

Specifies to use the selected catalog for creating the report and directs to the [Select Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009690662-Select-Data-Source#Report) dialog.

**Cancel**

Cancels the selection and closes this dialog. Available only when the dialog is opened by selecting Change Catalog in the Select Data Source dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Cancels the selection and closes this dialog. Available only when the dialog is opened by selecting Change Catalog in the Select Data Source dialog.
