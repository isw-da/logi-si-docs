# Suppress Column Subtotals

The Suppress Column Subtotals dialog helps you to customize which subtotals of the column fields in the crosstab will be suppressed and which will be shown. This dialog appears when you select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4412139533719/btn_chsr4.gif) next to the Suppress Column Subtotals option in the Layout tab of the [Crosstab Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009689682-Crosstab-Wizard#Layout) or [Insert Crosstab](https://devnet.logianalytics.com/hc/en-us/articles/1500009690382-Insert-Crosstab#Layout)  dialog, or in the Layout tab for crosstab in the [Web Report Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009717201-Web-Report-Wizard#Crosstab).

![Suppress Column Subtotal dialog](https://devnet.logianalytics.com/hc/article_attachments/4412112579607/sprsclmnsbtl.gif)

**Same for all**

If checked, the setting for the subtotal of the first column field in below box will be applied to all the subtotals of the column fields in the crosstab.

**Name**

Lists the column fields excluding the one on the topmost column group level.

**Suppress**

Specifies whether the subtotal for a column field will be suppressed. To specify the value, select in the value cell and select true or false from the drop-down list.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)**

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Does not retain any changes and closes the dialog.
