# Comparison Function

The Comparison Function dialog helps you to add a [comparison function](https://devnet.logianalytics.com/hc/en-us/articles/1500009719181-Inserting-Components#Comparison) as an aggregate field for a crosstab. This dialog appears when you select the Comparison Function button in the the [Crosstab Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009689682-Crosstab-Wizard), [Insert Crosstab](https://devnet.logianalytics.com/hc/en-us/articles/1500009690382-Insert-Crosstab) dialog, [Convert to Crosstab](https://devnet.logianalytics.com/hc/en-us/articles/1500009689582-Convert-to-Crosstab) dialog, or [To Crosstab](https://devnet.logianalytics.com/hc/en-us/articles/1500009690902-To-Crosstab) dialog, or in the Bind Data screen for crosstab in the [Web Report Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009717201-Web-Report-Wizard#Crosstab).

![Comparison Function dialog](https://devnet.logianalytics.com/hc/article_attachments/4412112504599/cmprsnfctn.gif)

**Function**

Specifies the function of the calculation: Percentage, Permillage, or Difference.

**Comparison Function Spans on Row Direction**

If checked, the comparison function will be placed into the column total cell of the crosstab.

**Comparison Function Spans on Column Direction**

If checked, the comparison function will be placed into the row total cell of the crosstab.

**Break By**

Specifies the first parameter of the comparison function.

**Refer To**

Specifies the other parameter of the comparison function.

**OK**

Accepts all changes and closes the dialog.

**Cancel**

Does not retain any changes and closes this dialog.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412139537431/btn_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112572951/btn_close.gif)

Ignores the setting and closes this dialog.
