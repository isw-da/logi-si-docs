# Instant Dashboards

Dashboards combine tables, crosstabs, charts, and maps from Logi JReport’s managed library of shared widgets. Intuitive drag-and-drop makes it easy to mashup and layout informative dashboards. Report components and visual analysis results are reusable in dashboards. One single select to rearrange the layout is supported. Components are able to be responsive to the browser window. Parameter and filter values are changeable at runtime. A dashboard can be set as the home page of Logi JReport Server console for easier access.

See JDashboard in the *Logi JReport Server User's Guide* for more information. A [JDashboard license](https://devnet.logianalytics.com/hc/en-us/articles/1500010068641-Logi-JReport-Licenses "Select to see JDashboard License") is required in order to use JDashboard.
