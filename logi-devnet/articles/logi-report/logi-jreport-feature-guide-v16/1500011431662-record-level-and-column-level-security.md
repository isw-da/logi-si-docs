# Record Level and Column Level Security

Record Level Security controls user accessibility to data records by composing an expression, while Column Level Security restricts user accessibility to data columns. Both can be defined in a data source security entry and then applied to different reports and library components.

![Define Securities in Catalog](https://devnet.logianalytics.com/hc/article_attachments/4404909299991/rptscrty_rls.gif)

Record Level Security can also be defined directly on a page report.

![Define Report-level Security on Page Report](https://devnet.logianalytics.com/hc/article_attachments/4404901355799/rptscrty_rls_pg.gif)
