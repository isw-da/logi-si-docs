# XML Connection

An XML connection contains relational data which is transformed from an XML data source.

Logi JReport Designer supports parsing of the XML schema from XML instances, or importing them from an XSD file and then transforming them to relational tables. The transformed relational tables work in the same way as JDBC-supplied tables.
