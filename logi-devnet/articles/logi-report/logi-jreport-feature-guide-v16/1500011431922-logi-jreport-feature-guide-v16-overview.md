# Logi JReport Feature Guide v16 Overview

This Feature Guide demonstrates some of the significant features of Logi JReport using GIF animations, which can help you learn the features in a visual way. The topics are listed following the alphabetical order for easier navigation.

[**Banded Objects**](https://devnet.logianalytics.com/hc/en-us/articles/1500011432142-Banded-Objects)

A banded object is a type of component that can present grouped data and detail data. A banded object is composed of several banded panels in which you can easily organize data fields and other components with absolute layout for precise design.

[**Charts**](https://devnet.logianalytics.com/hc/en-us/articles/1500011432122-Charts)

Visualizing data through charts is essential for business intelligence. Charts organize and graphically present data in a way that makes it easy for end users to see comparisons, trends, and patterns in data. It represents report data in a visually straightforward form. You can build interactive charts of various types and styles in seconds with Logi JReport.

[**Component Synchronization**](https://devnet.logianalytics.com/hc/en-us/articles/1500011463641-Component-Synchronization)

It is important for different parts of a visualization to be fully consistent. With component synchronization, you can make the filter, sort or other operations made on some data reflected on all or a certain set of data components in the same dashboard.

[**Compound Crosstabs**](https://devnet.logianalytics.com/hc/en-us/articles/1500011464001-Compound-Crosstabs)

Compound crosstab allows you to put multiple crosstabs together in a very flexible structure, which makes far more sophisticated crosstab analysis possible.

[**Conditional Formatting**](https://devnet.logianalytics.com/hc/en-us/articles/1500011463981-Conditional-Formatting)

Conditional formatting is very useful when you want to highlight significant data values. You can adjust the look and feel of visualization properties based on data value conditions and computations.

[**CSS Styles**](https://devnet.logianalytics.com/hc/en-us/articles/1500011463961-CSS-Styles)

Logi JReport UI is fully customizable using CSS to control the look, format, style and the availability of certain features.

[**Custom Aggregations**](https://devnet.logianalytics.com/hc/en-us/articles/1500011463941-Custom-Aggregations)

You can customize aggregations for sophisticated mathematic and logic calculation purposes using Formula Editor.

[**Customizing UI by Modifying CSS**](https://devnet.logianalytics.com/hc/en-us/articles/1500011463921-Customizing-UI-by-Modifying-CSS)

You can customize the width, height, color, font, borders, padding and so on of the window elements in Web Report Studio and JDashboard by modifying CSS files directly.

**[Data Mashup](https://devnet.logianalytics.com/hc/en-us/articles/1500011431942-Data-Mashup)**

Multiple data resources from different database connections can be used in one single query or business view.

**[Data Source Connections](https://devnet.logianalytics.com/hc/en-us/articles/1500011432102-Data-Source-Connections)**

Logi JReport is extremely efficient when working with data sources that are in a relational database (RDBMS).

[**Dynamic Resources**](https://devnet.logianalytics.com/hc/en-us/articles/1500011463901-Dynamic-Resources)

You can create dynamic resources at anytime based on existing view elements in a business view and use them in a report or library component.

[**Geo Analysis**](https://devnet.logianalytics.com/hc/en-us/articles/1500011463861-Geo-Analysis)

Logi JReport provides geographic visualization and analysis by overlaying customized areas and markers onto interactive maps in Google Maps, OpenStreetMap and OpenCycleMap.

**[Instant Dashboards](https://devnet.logianalytics.com/hc/en-us/articles/1500011431962-Instant-Dashboards)**

Dashboards display multiple components for browsing different data aspects. Data components can be freely selected from the component library or reports and dragged onto dashboards.

[**Logi JReport Server Monitor**](https://devnet.logianalytics.com/hc/en-us/articles/1500011431882-Logi-JReport-Server-Monitor)

Logi JReport Server Monitor is a standalone web-based application used to monitor the overall performance of Logi JReport Server.

**[Logi JReport Server Security System](https://devnet.logianalytics.com/hc/en-us/articles/1500011463661-Logi-JReport-Server-Security-System)**

Only registered users can log onto Logi JReport Server. Then they can interact with reports based on their permissions on the reports. Logi JReport Server can also use user information from an LDAP system.

[**Links in Report**](https://devnet.logianalytics.com/hc/en-us/articles/1500011463841-Links-in-Report)

A report can be linked with other targets such as reports and locations specified by URLs to present dynamic link visualizations.

**[Multi-tenancy](https://devnet.logianalytics.com/hc/en-us/articles/1500011431862-Multi-tenancy)**

Different users can be grouped into different organizations. Each organization has its own administrator, server resource allocation, and server profile settings.

[**National Language Support**](https://devnet.logianalytics.com/hc/en-us/articles/1500011431842-National-Language-Support)

Logi JReport can work in different national languages simultaneously to support international applications. Extensive customization can be applied to the Logi JReport product UI, report UI, as well as metadata and actual data.

**[One-Select Ad Hoc Operations](https://devnet.logianalytics.com/hc/en-us/articles/1500011463821-One-select-Ad-Hoc-Operations)**

Logi JReport supports several one-select ad hoc operations, with which you can easily change the data presentation of a report, sort and filter the report data and so on.

**[Partial and Full Data View](https://devnet.logianalytics.com/hc/en-us/articles/1500011463781-Partial-and-Full-Data-View-)**

When a report contains a large amount of data, it can be helpful to retrieve only a specific number of records for better performance. Logi JReport provides you with the option to switch between the full data and partial data views with a single select.

**[Predefined Web Report Templates](https://devnet.logianalytics.com/hc/en-us/articles/1500011431622-Predefined-Web-Report-Templates)**

Using predefined templates, end users can create web reports in corporate standard documents.

**[Profile Control and Customization](https://devnet.logianalytics.com/hc/en-us/articles/1500011463761-Profile-Control-and-Customization)**

Logi JReport profile provides both system level and user level customization on UI, formats, styles, specific features, and preferences.

[**QR Codes**](https://devnet.logianalytics.com/hc/en-us/articles/1500011463741-QR-Codes)

You can use QR Codes to encrypt values of data fields and label text that is long and complex in a page report using Logi JReport Designer.

[**Quick Start a Web Report**](https://devnet.logianalytics.com/hc/en-us/articles/1500011463721-Quick-Start-a-Web-Report)

Using Logi JReport’s intuitive web-based Quick Start ad hoc feature, you can quickly create a web report by adding required data fields from a business view into a table.

**[Report Bursting](https://devnet.logianalytics.com/hc/en-us/articles/1500011431722-Report-Bursting)**

You can run one report once and distribute the report results to multiple recipients according to a distinguishing data field. You can also schedule a bursting task at a time when runtime load is not a problem.

**[Report Security System](https://devnet.logianalytics.com/hc/en-us/articles/1500011431702-Report-Security-System)**

By applying security policies on reports you can limit the data shown on reports according to users' access permissions.

**[Select N Filter Condition](https://devnet.logianalytics.com/hc/en-us/articles/1500011431642-Select-N-Filter-Condition)**

The Select N feature is very useful when you want to view just partial data in a report.

**[Web Controls](https://devnet.logianalytics.com/hc/en-us/articles/1500011463621-Web-Controls)**

You can use web controls in reports for changing parameter values, filtering report data and so on at runtime. They are powerful and easy to use.
