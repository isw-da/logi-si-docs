# One-Select Ad Hoc Operations

Logi JReport supports several one-select ad hoc operations, with which you can easily change the data presentation of a report, sort and filter the report data and so on.

This topic explores the following one-select actions:

* [One-Select to Pivot Crosstab](https://devnet.logianalytics.com/hc/en-us/articles/1500011431782-One-Select-to-Pivot-Crosstab)
* [One-Select to Swap Group/Value in Chart](https://devnet.logianalytics.com/hc/en-us/articles/1500011463801-One-Select-to-Swap-Group-Value-in-Chart)
* [One-Select to Sort and Filter Table](https://devnet.logianalytics.com/hc/en-us/articles/1500011431762-One-Select-to-Sort-and-Filter-Table)
* [One-Select to Drill Down Report Data](https://devnet.logianalytics.com/hc/en-us/articles/1500011431802-One-Select-to-Drill-Down-Report-Data)
* [One-Select to Convert Table/Crosstab/Chart](https://devnet.logianalytics.com/hc/en-us/articles/1500011431822-One-Select-to-Convert-Table-Crosstab-Chart)
* [Chart Zoom via Range Selection](https://devnet.logianalytics.com/hc/en-us/articles/1500011431742-Chart-Zoom-via-Range-Selection)
