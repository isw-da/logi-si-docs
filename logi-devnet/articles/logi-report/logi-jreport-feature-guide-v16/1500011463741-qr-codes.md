# QR Codes

QR Code is a 2D barcode which takes up less space but provides richer and more complex information as compared to a barcode. It can be used to encrypt the values of a data field as well as label text in a page report based on query resources.

The following example shows regular barcode and QR Code and use QR Code to represent the home page of Logi Analytics. For detailed information about how to insert a barcode in a page report, [select here](https://devnet.logianalytics.com/hc/en-us/articles/1500010028682-Barcodes).
