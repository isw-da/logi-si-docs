# National Language Support

Logi JReport includes a National Language Support feature, with which you can run Logi JReport in different language environments. Logi JReport NLS is fully customizable, allowing you to customize the application UI, reporting UI, as well as the metadata and data. You can edit NLS in both Logi JReport Designer and Logi JReport Server, and you can create global NLS resources that can be shared by all reports and library components to reduce translation cost.

For detailed information about NLS in Logi JReport, [select here](https://devnet.logianalytics.com/hc/en-us/articles/1500010033362-Applying-National-Language-Support), or refer to National Language Support in the *Logi JReport Server User's Guide*.
