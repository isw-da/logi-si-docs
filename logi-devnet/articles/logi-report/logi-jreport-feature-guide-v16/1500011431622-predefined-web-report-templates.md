# Predefined Web Report Templates

Rather than always starting from scratch to build reports, Logi JReport also allows developers to build predefined templates with information such as customer logos, page numbers, dates, styles and anything else required as a corporate standard for end users to readily create distribution-ready documents. This ensures that even when building ad hoc reports, end users are creating reports with the same requirements as those created by IT without any extra effort.

Web report templates can be defined, edited, and saved using Web Report Wizard or Web Report Studio. Then end users can access them in the Web Report Wizard and use them to create web reports.
