# Logi JReport Server Monitor

Logi JReport Server Monitor, a standalone web-based application used to monitor the overall performance of Logi JReport Server, allows you to inspect the status of Logi JReport Server, including the status of the servers in the cluster, the status of different reports, the status of on-line users and so on. Logi JReport Server Monitor is able to generate and display a performance chart of Logi JReport Server, thus you can view the performance of Logi JReport Server in the form of Line Chart Graph and Text. You can also use Logi JReport Server Monitor to maintain other administrative features of Logi JReport Server, such as shutting down servers, stopping waiting/running tasks, logging out a valid UserSession and so on.

For more detailed information about using the monitor, refer to the Logi JReport[Server Monitor User's Guide](http://www.jinfonet.com/kbase/monitor15.5/userguide/index.htm).
