# Custom Aggregations

Custom aggregations are special aggregations used for sophisticated mathematic and logic calculation purposes. They are created using the Formula Editor and are defined based on business view elements and built-in functions including aggregation functions, mathematic and logic calculation-related functions. For details about how to compose the expression of a custom aggregation, [select here](https://devnet.logianalytics.com/hc/en-us/articles/1500010034582-Business-View-Elements#CA).

When using a custom aggregation in a data component, you need to make sure that the group objects referenced in the custom aggregation are also added as group by fields, column/row headers or chart axis fields in the data component.
