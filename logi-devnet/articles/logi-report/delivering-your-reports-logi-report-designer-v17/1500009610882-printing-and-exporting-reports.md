# Printing and Exporting Reports

As you develop reports in Logi Report Designer, you can print the report result or export the result to a file so that you can review the result at any time or share with others.

Library components can neither be printed nor exported.

The following topics provide more information on report results:

* [Printing Report Result](https://devnet.logianalytics.com/hc/en-us/articles/1500009611102-Printing-Report-Result)
* [Exporting Report Result](https://devnet.logianalytics.com/hc/en-us/articles/1500009634061-Exporting-Report-Result)

**Tips:**

* When exporting or printing a report which contains parameters, you will be prompted to [specify values for the parameters](https://devnet.logianalytics.com/hc/en-us/articles/1500009634541-Specifying-Parameter-Values).
* When exporting or printing a report which [applies a style group](https://devnet.logianalytics.com/hc/en-us/articles/1500009613462-Applying-Styles), you will be prompted to select the style group to use.
