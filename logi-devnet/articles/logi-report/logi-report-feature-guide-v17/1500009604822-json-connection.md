# JSON Connection

A JSON connection contains relational data which is transformed from a JSON data source.

Logi Report Designer can parse JSON data to extract the schema including JSON metadata (JSON objects and the relation between the objects), then transform the schema to logic RDBMS tables, namely, map JSON object classes to tables, and the relation to primary/foreign key columns in tables. The tables can then be added to the current connection from the transformed relational tables. In the following example, the data is connected via REST web service with JSON format.
