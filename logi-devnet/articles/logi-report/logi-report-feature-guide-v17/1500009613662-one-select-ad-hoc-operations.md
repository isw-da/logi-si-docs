# One-Select Ad Hoc Operations

Logi Report supports several one-select ad hoc operations, with which you can easily change the data presentation of a report, sort and filter the report data, and so on.

This guide explores the following one-select actions:

* [One-Select to Pivot Crosstab](https://devnet.logianalytics.com/hc/en-us/articles/1500009636361-One-Select-to-Pivot-Crosstab)
* [One-Select to Swap Group/Value in Chart](https://devnet.logianalytics.com/hc/en-us/articles/1500009636381-One-Select-to-Swap-Group-Value-in-Chart)
* [One-Select to Sort and Filter Table](https://devnet.logianalytics.com/hc/en-us/articles/1500009613702-One-Select-to-Sort-and-Filter-Table)
* [One-Select to Drill Down Report Data](https://devnet.logianalytics.com/hc/en-us/articles/1500009636341-One-Select-to-Drill-Down-Report-Data)
* [One-Select to Convert Table/Crosstab/Chart](https://devnet.logianalytics.com/hc/en-us/articles/1500009613682-One-Select-to-Convert-Table-Crosstab-Chart)
* [Chart Zoom via Range Selection](https://devnet.logianalytics.com/hc/en-us/articles/1500009613722-Chart-Zoom-via-Range-Selection)
