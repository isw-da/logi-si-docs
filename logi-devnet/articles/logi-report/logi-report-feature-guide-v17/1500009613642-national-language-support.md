# National Language Support

Logi Report includes a National Language Support feature, with which you can run Logi Report in different language environments. Logi Report NLS is fully customizable, allowing you to customize the application UI, reporting UI, as well as the metadata and data. You can edit NLS in both Logi Report Designer and Logi Report Server, and you can create global NLS resources that can be shared by all reports to reduce the translation cost.
