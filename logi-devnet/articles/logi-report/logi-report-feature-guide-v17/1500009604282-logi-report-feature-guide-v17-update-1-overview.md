# Logi Report Feature Guide v17 Update 1 Overview

This Feature Guide demonstrates some of the significant features of Logi Report using GIF animations, which can help you learn the features in a visual way. The topics are listed following the alphabetical order for easier navigation.

[**Banded Objects**](https://devnet.logianalytics.com/hc/en-us/articles/1500009627761-Banded-Objects)

A banded object is a type of component that can present grouped data and detail data. A banded object is composed of several banded panels in which you can easily organize data fields and other components with absolute layout for precise design.

[**Charts**](https://devnet.logianalytics.com/hc/en-us/articles/1500009627781-Charts)

Visualizing data through charts is essential for business intelligence. Charts organize and graphically present data in a way that makes it easy for end users to see comparisons, trends, and patterns in data. It represents report data in a visually straightforward form. You can build interactive charts of various types and styles in seconds with Logi Report.

[**Component Synchronization**](https://devnet.logianalytics.com/hc/en-us/articles/1500009613822-Component-Synchronization)

It is important for different parts of a visualization to be fully consistent. With component synchronization, you can make the filter, sort or other operations made on some data reflected on all or a certain set of data components in the same dashboard.

[**Compound Crosstabs**](https://devnet.logianalytics.com/hc/en-us/articles/1500009627801-Compound-Crosstabs)

Compound crosstab allows you to put multiple crosstabs together in a very flexible structure, which makes far more sophisticated crosstab analysis possible.

[**Conditional Formatting**](https://devnet.logianalytics.com/hc/en-us/articles/1500009604782-Conditional-Formatting)

Conditional formatting is very useful when you want to highlight significant data values. You can adjust the look and feel of visualization properties based on data value conditions and computations.

[**CSS Styles**](https://devnet.logianalytics.com/hc/en-us/articles/1500009604882-CSS-Styles)

Logi Report UI is fully customizable using CSS to control the look, format, style, and the availability of certain features.

[**Custom Aggregations**](https://devnet.logianalytics.com/hc/en-us/articles/1500009627901-Custom-Aggregations)

You can customize aggregations for sophisticated mathematic and logic calculation purposes using Formula Editor.

[**Customizing UI by Modifying CSS**](https://devnet.logianalytics.com/hc/en-us/articles/1500009627921-Customizing-UI-by-Modifying-CSS)

You can customize the width, height, color, font, borders, padding, and so on of the window elements in Web Report Studio and JDashboard by modifying CSS files directly.

[**Data Mash-up**](https://devnet.logianalytics.com/hc/en-us/articles/1500009627941-Data-Mash-up)

Multiple data resources from different database connections can be used in one single query or business view.

[**Data Source Connections**](https://devnet.logianalytics.com/hc/en-us/articles/1500009627821-Data-Source-Connections)

Logi Report is extremely efficient when working with data sources that are in a relational database (RDBMS).

[**Dynamic Resources**](https://devnet.logianalytics.com/hc/en-us/articles/1500009604982-Dynamic-Resources)

You can create dynamic resources at anytime based on existing view elements in a business view and use them in a report.

[**Geo Analysis**](https://devnet.logianalytics.com/hc/en-us/articles/1500009628141-Geo-Analysis)

Logi Report provides geographic visualization and analysis by overlaying customized areas and markers onto interactive maps in Google Maps, OpenStreetMap and OpenCycleMap.

**[Instant Dashboards](https://devnet.logianalytics.com/hc/en-us/articles/1500009604902-Instant-Dashboards)**

Dashboards display multiple components for browsing different data aspects. Data components can be freely selected from the component library or reports and dragged onto dashboards.

[**Logi Report Server Monitor**](https://devnet.logianalytics.com/hc/en-us/articles/1500009636321-Logi-Report-Server-Monitor)

Logi Report Server Monitor is a standalone web-based application used to monitor the overall performance of Logi Report Server.

**[Logi Report Server Security System](https://devnet.logianalytics.com/hc/en-us/articles/1500009636581-Logi-Report-Server-Security-System)**

Only registered users can log onto Logi Report Server. Then they can interact with reports based on their permissions on the reports. Logi Report Server can also use user information from an LDAP system.

[**Links in Report**](https://devnet.logianalytics.com/hc/en-us/articles/1500009636301-Links-in-Report)

A report can be linked with other targets such as reports and locations specified by URLs to present dynamic link visualizations.

**[Multitenancy](https://devnet.logianalytics.com/hc/en-us/articles/1500009613622-Multitenancy)**

Different users can be grouped into different organizations. Each organization has its own administrator, server resource allocation, and server profile settings.

[**National Language Support**](https://devnet.logianalytics.com/hc/en-us/articles/1500009613642-National-Language-Support)

Logi Report can work in different national languages simultaneously to support international applications. Extensive customization can be applied to the Logi Report product UI, report UI, as well as metadata and actual data.

**[One-Select Ad Hoc Operations](https://devnet.logianalytics.com/hc/en-us/articles/1500009613662-One-Select-Ad-Hoc-Operations)**

Logi Report supports several one-select ad hoc operations, with which you can easily change the data presentation of a report, sort and filter the report data, and so on.

**[Partial and Full Data View](https://devnet.logianalytics.com/hc/en-us/articles/1500009636401-Partial-and-Full-Data-View-)**

When a report contains a large amount of data, it can be helpful to retrieve only a specific number of records for better performance. Logi Report provides you with the option to switch between the full data and partial data views with a single select.

**[Predefined Web Report Templates](https://devnet.logianalytics.com/hc/en-us/articles/1500009636621-Predefined-Web-Report-Templates)**

Using predefined templates, end users can create web reports in corporate standard documents.

**[Profile Control and Customization](https://devnet.logianalytics.com/hc/en-us/articles/1500011552862-Profile-Control-and-Customization)**

Logi Report profile provides both system level and user level customization on UI, formats, styles, specific features, and preferences.

[**QR Codes**](https://devnet.logianalytics.com/hc/en-us/articles/1500009636441-QR-Codes)

You can use QR Codes to encrypt values of data fields and label text that is long and complex in a page report using Logi Report Designer.

[**Quick Start a Web Report**](https://devnet.logianalytics.com/hc/en-us/articles/1500009636461-Quick-Start-a-Web-Report)

Using Logi Report’s intuitive web-based Quick Start ad hoc feature, you can quickly create a web report by adding required data fields from a business view into a table.

**[Report Bursting](https://devnet.logianalytics.com/hc/en-us/articles/1500009636481-Report-Bursting)**

You can run one report once and distribute the report results to multiple recipients according to a distinguishing data field. You can also schedule a bursting task at a time when run time load is not a problem.

**[Report Security System](https://devnet.logianalytics.com/hc/en-us/articles/1500009636501-Report-Security-System)**

By applying security policies on reports you can limit the data shown on reports according to users' access permissions.

**[Select N Filter Condition](https://devnet.logianalytics.com/hc/en-us/articles/1500009636561-Select-N-Filter-Condition)**

The Select N feature is very useful when you want to view just partial data in a report.

**[Web Controls](https://devnet.logianalytics.com/hc/en-us/articles/1500009636601-Web-Controls)**

You can use web controls in reports for changing parameter values, filtering report data, and so on at run time. They are powerful and easy to use.
