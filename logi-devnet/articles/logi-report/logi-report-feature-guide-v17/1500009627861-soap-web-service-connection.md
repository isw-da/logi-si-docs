# SOAP Web Service Connection

Web Services provide a standard means of interoperating between different software applications, running on a variety of platforms and/or frameworks. It is a way to connect to a remote application and request data directly from the application without going directly to a DBMS. Logi Report Designer supports SOAP Web Services defined by WSDL 1.1 or WSDL 2.0 as data sources. You can add a SOAP Web Service data source to a catalog by importing a WSDL file.
