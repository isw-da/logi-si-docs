# Org Chart Line

The properties of the line object in an org chart are:

| Property Name | Description |
| --- | --- |
| Line Property | |
| Line Color | Specifies the color of the line. Choose a color from the drop-down list or select Custom to define a color in the [Color Picker](https://devnet.logianalytics.com/hc/en-us/articles/1500009715801-Color-Picker-) dialog. You can also enter a hexadecimal RGB value (for example, 0xff0000) to specify a color. Data type: String |
| Line Style | Specifies the style of the line. Choose an option from the drop-down list. Data type: Enumeration |
| Line Thickness | Specifies the thickness of the line. Enter a numeric value to change the thickness. Data type: Float |
