# Using Hierarchical Data Sources

Designer supports XML data sources by wrapping the provided Hierarchical Data Source (HDS) API. Designer's built-in classes can implement the XML hierarchical data source interface. This topic introduces the HDS API, and describes how you can work with hierarchical data sources in a catalog and create reports using HDS.

Designer supports both general hierarchical data sources and hierarchical data sources with a parallel structure (parallel HDS). The way of using a parallel HDS is the same as with general HDS. There are, however, some key points that you should be aware of when you use a parallel HDS to develop reports. For more information, see [Example 2: Developing a Report from a Parallel HDS](https://devnet.logianalytics.com/hc/en-us/articles/5735527967639-Developing-Reports-from-Hierarchical-Data-Sources#Example2).
