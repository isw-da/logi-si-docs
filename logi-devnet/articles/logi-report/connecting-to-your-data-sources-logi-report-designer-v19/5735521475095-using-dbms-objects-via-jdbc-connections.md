# Using DBMS Objects via JDBC Connections

Tables, views, synonyms, stored procedures, and imported SQLs are all from the raw database. If you want to use them in Designer, you first need to add them to a catalog via [the JDBC connection you set up](https://devnet.logianalytics.com/hc/en-us/articles/5735521536663-Setting-Up-JDBC-Connections-in-a-Catalog) which connects Designer to the database driver. This topic introduces how you can add these DBMS objects into a catalog via a JDBC connection and work with them in the catalog.
