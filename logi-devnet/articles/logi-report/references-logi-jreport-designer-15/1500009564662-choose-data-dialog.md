# Choose Data Dialog

The Choose Data dialog differs for choosing a [query resource](#Query) or a [business view](#BV).

## For Choosing Query Resource

This dialog appears when you do one of the following:

* In the [Select Component for Page Report](https://devnet.logianalytics.com/hc/en-us/articles/1500009567422-Select-Component-for-Page-Report-Dialog) dialog, select the Blank component and select OK.
* When creating a new report tab in a page report which uses query resource as the data source, in the [Select Component for Page Report Tab](https://devnet.logianalytics.com/hc/en-us/articles/1500009588661-Select-Component-for-Page-Report-Tab-Dialog) dialog, select the Blank component and select OK.

It helps you to choose the dataset you want to use for  [the blank page report tab](https://devnet.logianalytics.com/hc/en-us/articles/1500009571062-Creating-Report-Tabs-in-a-Page-Report). [See the dialog](javascript:%20void(null)).

![Choose Data - query dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889286807/chsdt.gif)

The following are details about options in the dialog:

**Resources**

Specifies the dataset from which the report will be created.

**More Options/Less Options**

Specifies to show or hide the dataset selection panel.

* **New Dataset**  
   If checked, select a data source from the catalog resources to create a dataset that will be used to build the report. When you choose to create the dataset from a query, you can select the Edit button to edit the query in the [Query Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009567142-Query-Editor-Dialog).
* **Existing Dataset**  
   If checked, select a dataset from the ones existing in the current page report to create the report tab. Select the Edit button to edit the selected dataset in the [Dataset Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009586301-Dataset-Editor-Dialog) if needed.

**OK**

Applies the specified dataset and closes this dialog.

**Cancel**

Does not retain changes and closes this dialog.

**Help**

Displays the help document about this feature.

## For Choosing Business View

This dialog appears when you do one of the following:

* In the [Select Component for Page Report](https://devnet.logianalytics.com/hc/en-us/articles/1500009567422-Select-Component-for-Page-Report-Dialog) dialog, select Create Using Business View at bottom left, next select the Blank component, and then select OK.
* When creating a new report tab in a page report which uses business view as the data source, in the [Select Component for Page Report Tab](https://devnet.logianalytics.com/hc/en-us/articles/1500009588661-Select-Component-for-Page-Report-Tab-Dialog) dialog, select the Blank component and select OK.
* Drag ![KPI button](https://devnet.logianalytics.com/hc/article_attachments/4404893952791/btn_kpi.gif) from the Components box to the destination in an open web report or library component.

It helps you to choose a business view to bind to the [blank page report tab](https://devnet.logianalytics.com/hc/en-us/articles/1500009571062-Creating-Report-Tabs-in-a-Page-Report) or [KPI component](https://devnet.logianalytics.com/hc/en-us/articles/1500009583981-KPI-Components). [See the dialog](javascript:%20void(null)).

![Choose Data - business view dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889390231/chsdt-bv.gif)

The following are details about options in the dialog:

**Resources**

The resources box lists all the business views in the catalog. Select a business view you want.

**OK**

Binds the selected business view to the target and closes this dialog.

**Cancel**

Cancels the binding of a business view and closes this dialog.

**Help**

Displays the help document about this feature.
