# Report Object Properties

All the objects contained in a report are listed in a tree structure in the Report Inspector. You can edit the object properties here to change the appearance of your report.

The properties for each specific object may vary with different report types:

* [Page Report](https://devnet.logianalytics.com/hc/en-us/articles/1500009590981-Properties-for-Objects-in-Page-Report)
* [Web Report](https://devnet.logianalytics.com/hc/en-us/articles/1500009570022-Properties-for-Objects-in-Web-Report)
* [Library Component](https://devnet.logianalytics.com/hc/en-us/articles/1500009590681-Properties-for-Objects-in-Library-Component)
