# Size List Dialog

This dialog appears when you select the Size List button in the Size tab of the Format Bar dialog for [page report](https://devnet.logianalytics.com/hc/en-us/articles/1500009565722-Format-Bar-Dialog-for-Page-Report-#Size), [web report](https://devnet.logianalytics.com/hc/en-us/articles/1500009587021-Format-Bar-Dialog-for-Web-Report-#Size) or [library component](https://devnet.logianalytics.com/hc/en-us/articles/1500009565702-Format-Bar-Dialog-for-Library-Component-#Size). It is used to set the [size for bars in the same data series respectively](https://devnet.logianalytics.com/hc/en-us/articles/1500009583481-Formatting-the-Bars). [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404893855127/sizelst.gif)

The following are details about options in the dialog:

**Sizes**

Specifies the bar size. Select the items in the box one by one to set the size for the bars each of the item stands for.

* ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893791255/btn_add.gif)  
  Adds a new size item.
* ![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404893855383/btn_rmv1.gif)  
  Removes the selected size item.
* ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404893808791/btn_mvup.gif)  
  Moves the selected size item one step up.
* ![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404889289751/btn_mvdwn.gif)  
  Moves the selected size item one step down.
* ![Wider button](https://devnet.logianalytics.com/hc/article_attachments/4404889321879/btn_wider.gif)**/**![Narrower button](https://devnet.logianalytics.com/hc/article_attachments/4404889322135/btn_narrower.gif)  
  Specifies how much the bars will be wider/narrower than a normal bar in percentage. Available when Fixed Bar Width in the Size tab of the Format Bar dialog is unchecked.
* **Width**  
  Specifies the width of the bars. You can also [use a formula to control the width.](https://devnet.logianalytics.com/hc/en-us/articles/1500009568362-Using-Formulas-to-Control-Object-Properties) Available when Fixed Bar Width in the Size tab of the Format Bar dialog is checked.
* **Gap**  
  Specifies the gap between the specified bars and the bars besides them. You can also [use a formula to control the gap.](https://devnet.logianalytics.com/hc/en-us/articles/1500009568362-Using-Formulas-to-Control-Object-Properties) Available when Fixed Bar Width in the Size tab of the Format Bar dialog is checked.

**Reset**

Resets the settings in this dialog to their default settings.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
