# Insert Aggregation Dialog

The Insert Aggregation dialog appears when you drag a field that can be used as aggregate field from the Data panel to the cell above/under any subtotal or grand total of a crosstab. It helps you to [insert the selected field as an aggregate field in the crosstab](https://devnet.logianalytics.com/hc/en-us/articles/1500009583801-Managing-the-Aggregate-Fields-in-a-Crosstab).  [See the dialog](javascript:%20void(null)).

![Insert Aggregation dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893874839/instaggrgt.gif)

**Aggregate Function**

Specifies the function to summarize the selected field. For details about each function, see [Math Functions](https://devnet.logianalytics.com/hc/en-us/articles/1500009589541-Math-Functions).

If the selected field has been predefined with an aggregate function, the option is Aggregation. The name of the field is displayed in the text box and cannot be edited.

**Label**

Specifies the display name of the aggregate field, which will label the newly created summaries in the crosstab. You can leave it blank.

**OK**

Inserts the aggregate field and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
