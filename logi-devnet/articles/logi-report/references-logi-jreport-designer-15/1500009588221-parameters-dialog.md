# Parameters Dialog

The Parameters dialog appears when you select a web action which requires parameters in the [Web Action List](https://devnet.logianalytics.com/hc/en-us/articles/1500009567862-Web-Action-List-Dialog) dialog and then select OK. It helps you to [specify values of the parameters of the web action](https://devnet.logianalytics.com/hc/en-us/articles/1500009584021-Applying-Web-Actions-to-a-Label). [See the dialog](javascript:%20void(null)).

![Parameters dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893860375/prmtr.gif)

The following are details about options in the dialog:

**Parameter Name**

Displays the parameters the specified web action requires.

**Value**

Specifies the values for the parameters. For details about the parameter values, refer to [Appendix 2: Parameters of User Defined Web Actions](https://devnet.logianalytics.com/hc/en-us/articles/1500009582961-Appendix-2-Parameters-of-User-Defined-Web-Actions).

**OK**

Accepts the changes and closes the dialog.

**Cancel**

Discards the changes and closes the dialog.

**Help**

Displays the help document about this feature.
