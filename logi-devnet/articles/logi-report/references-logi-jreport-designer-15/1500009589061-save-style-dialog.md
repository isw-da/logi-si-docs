# Save Style Dialog

The Save Style dialog helps you edit and save the style in an [XSD style](https://devnet.logianalytics.com/hc/en-us/articles/1500009593261-XSD-Styles) you create into a catalog. [See the dialog](javascript:%20void(null)).

![Save Style dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893789207/svstyl.gif)

The following are details about options in the dialog:

**Style Group Name**

Specifies the name of the style group.

**Style Name**

Specifies the name of the saved style.

**All Properties**

Lists all the properties available. Select the properties you want to save in the style from the box.

**Selected Properties**

Lists the selected properties.

**![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893789463/btn_addarrow.gif)**

Adds a specified property to the style.

![Add All button](https://devnet.logianalytics.com/hc/article_attachments/4404893789719/btn_addall.gif)

Adds all the properties listed in the All Properties box to the style.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404893789975/btn_rmvarrow.gif)

Removes the specified property from the Selected Properties box, so that it will not be applied to the style.

![Remove All button](https://devnet.logianalytics.com/hc/article_attachments/4404889274263/btn_rmvall.gif)

Removes all the properties from the Selected Properties box, so that they will not be applied to the style.

**Save**

Saves the style and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
