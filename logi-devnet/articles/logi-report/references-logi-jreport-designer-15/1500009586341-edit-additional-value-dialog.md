# Edit Additional Value Dialog

The Edit Additional Value dialog appears when you select Constant Value/Average Value in the Resources box in the Display screen of the chart wizard and select ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893789463/btn_addarrow.gif) to add it to the value box, or select an additional value in the value box and then select ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/4404893928215/btn_edit1.gif) when you create or modify a chart. It helps you to add [additional values](https://devnet.logianalytics.com/hc/en-us/articles/1500009563242-Inserting-a-Chart#Additional) to be shown in the chart or modify the existing additional values. It varies according to different value types.

If it is a constant value, options in the dialog are as follows. [See the dialog](javascript:%20void(null)).

![Edit Additional Value - Constant Value](https://devnet.logianalytics.com/hc/article_attachments/4404893928471/edtaddtnvlu_cnst.gif)

**Name**

Specifies the display name of the constant value.

**Value**

Specifies the constant value. Its data type must be numerical.

**OK**

Creates the constant value and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.

If it is an average value, options in the dialog are as follows. [See the dialog](javascript:%20void(null)).

![Edit Additional Value - Average Value](https://devnet.logianalytics.com/hc/article_attachments/4404893928727/edtaddtnvlu_avg.gif)

**Name**

Specifies the display name of the average value.

**Based On**

Specifies a summary or formula field based on which the average value will be calculated.

**OK**

Creates average value and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
