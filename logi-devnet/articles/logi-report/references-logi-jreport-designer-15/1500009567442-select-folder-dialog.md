# Select Folder Dialog

The Select Folder dialog appears when you select the Browse button next to the Publish Resource To text field in the [Publish to Logi JReport Server](https://devnet.logianalytics.com/hc/en-us/articles/1500009588321-Publish-to-Logi-JReport-Server-Dialog) dialog or the Browse button next to the Download Resource From text field in the [Download from Logi JReport Server](https://devnet.logianalytics.com/hc/en-us/articles/1500009586321-Download-from-JReport-Server-Dialog) dialog. It helps you to specify the folder on the server where you want the [resources to be published](https://devnet.logianalytics.com/hc/en-us/articles/1500009592641-Publishing-Resources#Remotely) or from which [resources will be downloaded](https://devnet.logianalytics.com/hc/en-us/articles/1500009592621-Downloading-Resources-from-Logi-JReport-Server). [See the dialog](javascript:%20void(null)).

![Select Folder dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889321239/slctfldr.gif)

The following are details about options in the dialog:

**Folder**

Lists folders on Logi JReport Server in a tree structure.

**Detail Information**

Shows the detailed information of sub folders in the selected folder.

* **Name**  
   Displays the names of sub folders.
* **Type**  
   Displays the file type.
* **Last Modified**   
   Displays the last modified time of the sub folders.
* **Description**  
   Other necessary descriptions of the sub folders.

**New Folder**

Creates a new folder in the selected folder. You can rename the folder by double-clicking the folder in the folder tree.

This button is available only when the dialog is opened by selecting the Browse button next to the Publish Resource To text field in the Publish to Logi JReport Server dialog.

**OK**

Saves the information and closes the dialog.

**Cancel**

Closes the dialog, leaving any changes unsaved.

**Help**

Displays the help document about this feature.
