# Table Type Dialog

The Table Type dialog appears when you select Insert > Table. It helps you to choose a type for the table you want to create. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404889318935/tbltype.gif)

**Table (Group Above)**

Creates a table with group information above the detail row.

**Table (Group Left)**

Creates a table with group information left to the detail row.

**Table (Group Left Above)**

Creates a table with group information left above the detail row.

**Summary Table**

Creates a table with only group and summary information.

**OK**

Applies the changes and opens the [Create Table (for Page Report)](https://devnet.logianalytics.com/hc/en-us/articles/1500009564962-Create-Table-Dialog-for-Page-Report-), [Create Table (for Web Report and Library Component)](https://devnet.logianalytics.com/hc/en-us/articles/1500009586101-Create-Table-Dialog-for-Web-Report-and-Library-Component-).

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
