# Open As Dialog

The Open As dialog appears when you open a self-contained page report from the Open Report dialog (a self-contained page report is a binary report file, which has a .clx suffix and contains not only the report's layout but also the catalog with its own resources). It helps you to specify the way in which the self-contained page report will be opened. [See the dialog](javascript:%20void(null)).

![Open As dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893863703/opnas.gif)

The following are details about options in the dialog:

**Open a self-contained page report at current catalog**

Specifies to open the selected self-contained page report in the current catalog and make it work under the current catalog.

Logi JReport Designer will compare the resources in the catalog of the self-contained page report with the current catalog according to the checking level (whether the Check all data match is selected). If conflicts exist between the two catalogs, the page report cannot be opened.

**Open a self-contained page report directly**

Specifies to open the selected self-contained page report directly to make it work under a new catalog.

**Open a self-contained page report at other catalog**

Specifies to open the selected self-contained page report at another catalog and make it work under that catalog.

If this option is checked, you will be prompted to open an existing catalog for the self-contained page report. Logi JReport Designer will then compare the resources in the catalog of the self-contained page report with the newly selected catalog according to the checking level (whether the Check all data match is selected). If conflicts exist between the two catalogs, the page report cannot be opened.

**Check all data match**

This checkbox is enabled when Open a self-contained page report at current catalog or Open a self-contained page report at other catalog is checked. It is used to specify the checking level between two catalogs.

* If checked, all the differences between the two catalogs will be checked by Logi JReport Designer.
* If unchecked, only the differences that can cause the report engine to fail when running the page report will be checked by Logi JReport Designer.

**OK**

Opens the page report in the specified way and closes the dialog.

**Cancel**

Closes the dialog, leaving any changes unsaved.

**Help**

Displays the help document about this feature.
