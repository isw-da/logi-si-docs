# Save CSS As Dialog

The Save CSS As dialog appears when you select OK in the [New CSS Style](https://devnet.logianalytics.com/hc/en-us/articles/1500009587901-New-CSS-Style-Dialog) dialog if you select New Style Sheet File from the Add Selector to Style drop-down list. After building a report, if you are pleased with the format of certain object, you can [save it as a CSS style](https://devnet.logianalytics.com/hc/en-us/articles/1500009571302-Creating-a-CSS-Style) in order to use the same format again in other reports, without having to set all the properties again. [See the dialog](javascript:%20void(null)).

![Save CSS As dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893851415/svcss.gif)

The following are details about options in the dialog:

**Save the CSS File to**

Shows the directory that the CSS File will be saved to.

**Existing CSS Files**

Lists all the existing CSS files.

**File Name**

Specifies the name for the new CSS file. By default, it is New CSS File.css.

**Files of Type**

Specifies the type for the new file. Here it should be CSS File(\*.css).

**Save**

Saves the CSS style and goes into the [CSS Style Definition](https://devnet.logianalytics.com/hc/en-us/articles/1500009586121-CSS-Style-Definition-Dialog) dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
