# New Catalog Dialog

The New Catalog dialog appears when you select File > New Catalog. It helps you to [create a catalog](https://devnet.logianalytics.com/hc/en-us/articles/1500009562722-Creating-a-Catalog) in Logi JReport Designer. [See the dialog](javascript:%20void(null)).

![New Catalog dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889327895/nwctlg.gif)

The following are details about options in the dialog:

**Name**

Specifies the name of the catalog file. Input the name directly in the text box, and do not forget to add the extension .cat or .cat.xml.

**Data Source Name**

Specifies the name of the data source in the catalog. Input the name directly in the text box. The name cannot contain spaces and special characters other than underscore.

**Directory**

Specifies the location to save the new catalog. Type the directory of the catalog in the text box or select the button ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404889276567/btn_chsr.gif) to find a directory path. You do not want to select a directory that already contains a catalog.

**OK**

Accepts all changes and closes the dialog.

**Cancel**

Closes the dialog, leaving any changes unsaved.

**Help**

Displays the help document about this feature.
