# Node Pattern List Dialog

The Node Pattern dialog appears when you select the Node Pattern List button in the Format Line dialog for [page report](https://devnet.logianalytics.com/hc/en-us/articles/1500009587281-Format-Line-Dialog-for-Page-Report-), [web report](https://devnet.logianalytics.com/hc/en-us/articles/1500009565962-Format-Line-Dialog-for-Web-Report-) or [library component](https://devnet.logianalytics.com/hc/en-us/articles/1500009565942-Format-Line-Dialog-for-Library-Component-). It helps you to [modify the node pattern for each chart line](https://devnet.logianalytics.com/hc/en-us/articles/1500009563142-Formatting-the-Lines). [See the dialog](javascript:%20void(null)).

![Node Pattern List dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889328151/ndptrnlist.gif)

The followings are details about options in the dialog:

**Patterns**

Specifies the pattern for the nodes in each chart line. Select the item in the box one by one to set the node pattern for each line.

* ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893791255/btn_add.gif)  
   Adds a new pattern item.
* ![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404893855383/btn_rmv1.gif)  
   Removes the selected pattern item.
* ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404893808791/btn_mvup.gif)  
   Moves the selected pattern item one step up.
* ![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404889289751/btn_mvdwn.gif)  
   Moves the selected pattern item one step down.

**Shape**

Specifies the shape properties of the nodes in a chart line.

* **Style**  
   Specifies the style of the nodes.
* **Transparency**  
   Specifies the transparency of the nodes.
* **Width**  
   Specifies the width of the nodes.
* **Height**  
   Specifies the height of the nodes.
* **Show Node**  
   Specifies whether to show nodes in the chart line. Only when it is set to true, the node related properties for the line can take effect. However, it does not control the property Show High/Low Point and the Color property for each one.
  + **Color**  
     Specifies the color of the nodes. To edit the color, select the color image and select a color from the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/1500009593081-Menu-Tabs#Palette) or input the hexadecimal value (for example, 0xff0000) of a color directly in the text box.
* **Show High Point**  
   Specifies whether to show the line node in the highest point in the chart line.
  + **Color**  
     Specifies the color of the node in the highest point.
* **Show Low Point**  
   Specifies whether to show the line node in the lowest point in the chart line.
  + **Color**  
     Specifies the color of the node in the lowest point.
* **Show Highlight Point**  
   Specifies whether to show the highlight point when you hover the mouse over a node in the selected chart line.
  + **Color**  
     Specifies the color of the highlight point in the selected chart line.

**Border**

Specifies the border properties of the nodes in a chart line.
Available when the node is not of the plus, multiplication, star1 or star 2 style.

* **Border Color**  
   Specifies the color for the border of the nodes.
* **Transparency**  
   Specifies the transparency for the border of the nodes.
* **Line Style**  
   Specifies the line style for the border of the nodes.
* **Thickness**  
   Specifies the thickness for the border of the nodes.

**Line**

Specifies the line properties of the nodes in a chart line. Available when the node is of plus, multiplication, star1 or star2 style.

* **Line Style**  
   Specifies the line style of the nodes.
* **Thickness**  
   Specifies the line thickness of the nodes.

![Formula button](https://devnet.logianalytics.com/hc/article_attachments/4404893837335/btn_fmla.gif)

Indicates the value of an option can be [controlled by a formula](https://devnet.logianalytics.com/hc/en-us/articles/1500009568362-Using-Formulas-to-Control-Object-Properties). Available only for line charts in page reports.

**Sample**

Displays a preview sample of your selection.

**OK**

Applies the node patterns and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
