# Precision Settings Dialog

The Precision Settings dialog appears when you select the System Default Settings button in the [Advanced Export Settings](https://devnet.logianalytics.com/hc/en-us/articles/1500009585661-Advanced-Export-Settings-Dialog) dialog. It helps you to [predefine the precision level](https://devnet.logianalytics.com/hc/en-us/articles/1500009589221-Exporting-Report-Results#Precision) with which you want to export report results to different formats. [See the dialog](javascript:%20void(null)).

![Precision Settings dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889310871/prcsnsting.gif)

The precision level can be one of the following:

* **High**  
   If checked, the report results will be exported to the specified format with high precision.
* **Low**  
   If checked, the report results will be exported to the specified format with low precision.

**OK**

Applies the settings and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
