# Built-in Functions

The Functions panel in the Formula Editor lists all the built-in functions Logi JReport provides. They are powerful and commonly used. If you are are not able to find your own required function, you can write [user defined formula functions](https://devnet.logianalytics.com/hc/en-us/articles/1500009589721-User-Defined-Formula-Functions).

The following are types of the built-in functions:

> * [Array](https://devnet.logianalytics.com/hc/en-us/articles/1500009589501-Array-Functions)
> * [Date Range](https://devnet.logianalytics.com/hc/en-us/articles/1500009568202-Date-Range-Functions)
> * [Date/Time](https://devnet.logianalytics.com/hc/en-us/articles/1500009589521-Date-Time-Functions)
> * [Financial](https://devnet.logianalytics.com/hc/en-us/articles/1500009568222-Financial-Functions)
> * [Math](https://devnet.logianalytics.com/hc/en-us/articles/1500009589541-Math-Functions)
> * [String](https://devnet.logianalytics.com/hc/en-us/articles/1500009568262-String-Functions)
> * [Others](https://devnet.logianalytics.com/hc/en-us/articles/1500009589561-Other-Functions)
