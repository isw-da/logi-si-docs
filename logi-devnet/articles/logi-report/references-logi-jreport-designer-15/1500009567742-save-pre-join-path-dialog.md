# Save Pre-join Path Dialog

The Save Pre-join Path dialog appears when you select the Paths button in the [Pre-join Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009567082-Pre-join-Editor-Dialog). It helps you to [create and save paths for the joins](https://devnet.logianalytics.com/hc/en-us/articles/1500009583161-Defining-the-Join-Relationships-Between-Resources). [See the dialog](javascript:%20void(null)).

![Save Pre-join Path dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889319191/svprjnpth.gif)

The following are details about options in the dialog:

**Pre-join Paths**

Lists all the pre-join paths.

**New**

Opens the [Select Pre-join](https://devnet.logianalytics.com/hc/en-us/articles/1500009567502-Select-Pre-join-Dialog) dialog to create a new pre-join path.

**Edit**

Edits the highlighted pre-join path.

**Delete**

Deletes the highlighted pre-join path.

**Detail**

Shows the joins information of the highlighted pre-join path.

**Rename**

Specifies the new name for the highlighted pre-join.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
