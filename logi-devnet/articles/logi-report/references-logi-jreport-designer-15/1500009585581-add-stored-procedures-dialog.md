# Add Stored Procedures Dialog

The Add Stored Procedures dialog appears when you right-click a JDBC connection node and then select Add Stored Procedure from the shortcut menu in the Catalog Manager. It displays all the stored procedures in the database, and allows you to [add the required stored procedures](https://devnet.logianalytics.com/hc/en-us/articles/1500009564022-Stored-Procedures#Add) to a catalog. [See the dialog](javascript:%20void(null)).

![Add Stored Procedures dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893959191/addprcdr.gif)

The following are details about options in the dialog:

**Database Catalog**

Lists all the catalogs in the database.

**Stored Procedures**

Lists all the stored procedures in a three level tree. The top level is SQL-catalog, second is SQL-schema and the third are stored procedures. You can select one or more stored procedures to add to the catalog.

**Add**

Adds the selected stored procedures to the catalog.

**Done**

Applies all changes and closes the dialog.

**Help**

Displays the help document about this feature.
