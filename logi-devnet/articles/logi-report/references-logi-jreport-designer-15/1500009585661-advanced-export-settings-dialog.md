# Advanced Export Settings Dialog

The Advanced Export Settings dialog appears when you select the Layout Precision button in the [Export to](https://devnet.logianalytics.com/hc/en-us/articles/1500009588081-Options-Dialog#Export) category of the Options dialog. It helps you to [customize the precision used for exporting](https://devnet.logianalytics.com/hc/en-us/articles/1500009589221-Exporting-Report-Results#Precision) in Logi JReport Designer. For a page report, the settings in this dialog take effect only in report tabs whose [Precision Sensitive](https://devnet.logianalytics.com/hc/en-us/articles/1500009591741-Properties-of-Page-Report-Tab#PrecisionSensitive) property is set to true in the Report Inspector. [See the dialog](javascript:%20void(null)).

![Advanced Export Settings dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893843991/advexptsting.gif)

The following are details about options in the dialog:

**Customize for each format**

If checked, you can customize the precision level for each format. Check the formats to apply the precision level defined for them in the Precision Settings dialog in the report results. For formats that are unchecked, the precision settings defined for them in the Precision Settings dialog are ignored, and Logi JReport will decide their precision.

* **System Default Settings**  
   Opens the [Precision Settings](https://devnet.logianalytics.com/hc/en-us/articles/1500009588161-Precision-Settings-Dialog) dialog to define the precision level for each format.

**Optimize for speed over visual effect**

If checked, Logi JReport will decide the precision level which is oriented towards speed more than visualization.

**OK**

Applies the settings and closes the dialog.

**Cancel**

Do not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
