# Replace Formula Dialog

The Replace Formula dialog appears when you double-click any formula added to be column of a query in the [Query Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009567142-Query-Editor-Dialog) dialog. It helps you to [select another formula to replace the current one](https://devnet.logianalytics.com/hc/en-us/articles/1500009570702-Adding-Formula-Fields-to-a-Query). [See the dialog](javascript:%20void(null)).

![Replace Formula dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893858711/rplcfmla.gif)

The following are details about options in the dialog:

**Formulas**

Lists all the formulas available to be added as query columns.

**Expression**

Shows the expressions of the selected formula.

**Replace**

Replaces the current formula with the selected one.

**Close**

Closes the dialog.

**Help**

Displays the help document about this feature.
