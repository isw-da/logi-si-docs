# MongoDB Collection Properties

The properties of a MongoDB collection are as follows:

| Property Name | Description |
| --- | --- |
| Description | Specifies the description of the collection in the MongoDB database. Data type: String |
| Name | Specifies the name of the collection in the MongoDB database. Data type: String |
