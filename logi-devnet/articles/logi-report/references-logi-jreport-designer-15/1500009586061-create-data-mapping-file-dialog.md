# Create Data Mapping File Dialog

The Create Data Mapping File dialog appears when you select the New button in the [Data Mapping Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009565102-Data-Mapping-Editor-Dialog) dialog. It helps you to [create a data mapping file](https://devnet.logianalytics.com/hc/en-us/articles/1500009590001-Mapping-Data-Values-to-Different-Languages) that saves information of a certain language. [See the dialog](javascript:%20void(null)).

![Create Data Mapping File dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893831191/crtdtmpfile.gif)

The following are details about options in the dialog:

**File Name**

Specifies the name of the data mapping file, which will be saved as filename\_language\_locale.properties.

**Language**

Specifies the language of the data mapping file. One language corresponds to one data mapping file.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Cancels the changes and exits the dialog.

**Help**

Displays the help document about this feature.
