# Select a Report Dialog (for Page Report)

The Select a Report dialog for page report appears when you select the Browse button next to the Report text field in the [Insert Link](https://devnet.logianalytics.com/hc/en-us/articles/1500009566502-Insert-Link-Dialog-for-Page-Report-)  or [Edit Link](https://devnet.logianalytics.com/hc/en-us/articles/1500009586481-Edit-Link-Dialog-for-Page-Report-) dialog for page report. It helps you to specify the report with which to link the [primary report](https://devnet.logianalytics.com/hc/en-us/articles/1500009563442-Binding-a-Link-to-a-Label#Report)  or [master report](https://devnet.logianalytics.com/hc/en-us/articles/1500009563442-Binding-a-Link-to-a-Label#Detail). [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404889322647/selectrpt.gif)

The following are details about options in the dialog:

**Catalog**

Shows the directory of the catalog that is being used now.

**Reports**

Specifies the report which contains the linked/detail report tab.

* ![](https://devnet.logianalytics.com/hc/article_attachments/4404889322903/btn_list.gif)**List**  
   Selects to go to the List mode where only the names of the files are displayed.
* ![](https://devnet.logianalytics.com/hc/article_attachments/4404893856791/btn_dtl.gif)**Details**  
   Selects to go to the Details mode where the files with detailed information such as name, size, type and last modified time are displayed.
* **Name**  
   Shows the names of all the reports included in this catalog, based on the type you selected from the Report of Type drop-down list.
* **Size**  
   Shows the size of the reports.
* **Type**  
   Shows the type of the reports.
* **Modified**  
   Shows the time when the reports were modified at the last time.

**Report Name**

Shows the names of the selected reports.

**Report of Type**

Specifies the type of the reports.

**OK**

Accepts the selecting report result and closes the dialog.

**Cancel**

Cancels the selecting report process and exits the dialog.

**Help**

Displays the help document about this feature.
