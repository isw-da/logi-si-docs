# New Customized Control File Dialog

The New Customized Control File dialog appears when you select Save As in the [Customized Control - Web Action Builder](https://devnet.logianalytics.com/hc/en-us/articles/1500009586181-Customized-Control-Web-Action-Builder-Dialog "Customized Control - Web Action Builder dialog") dialog, or select ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893791255/btn_add.gif) in the [Customized Control Manager](https://devnet.logianalytics.com/hc/en-us/articles/1500009587761-Manage-Customized-Controls-Dialog "Customized Control Manager dialog") dialog. It helps you to create a file in the customized control library to save the customized control definition into.  [See the dialog](javascript:%20void(null)).

![New Customized Control](https://devnet.logianalytics.com/hc/article_attachments/4404893864855/nwcstmzdctrlfl.gif)

The following are details about options in the dialog:

**Input File Name**

Input the name of the file, with or without the appendix .wctrl, to save the customized control definition into.

**OK**

* Saves the specified file to the customized control library and exits the current dialog if the dialog is opened after you select Save As in the Customized Control - Web Action Builder dialog.
* Exits the current dialog and displays the Customized Control - Web Action Builder dialog for defining a customized control in the specified file if the dialog is opened after you select ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893791255/btn_add.gif) in the Customized Control Manager dialog.

**Cancel**

Cancels the settings and exits this dialog.

**Help**

Displays the help document about this feature.
