# Suppress Column Grand Totals Dialog

The dialog appears when you select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404893790999/btn_chsr1.gif) next to the Suppress Column Grand Totals option in the Layout screen of the crosstab wizard. It helps you to control which grand totals of the column groups in the crosstab will be suppressed, and which will be shown. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404893853591/sprsclmngrdtl.gif)

**Same for all**

If checked, the setting for the first grand total will be applied to all the grand totals.

**Name**

Lists the names of all grand totals of column compound groups.

**Suppress**

Specifies whether to suppress the selected grand total by selecting true or false from the drop-down list. It could also be controlled by using a [constant level formula](https://devnet.logianalytics.com/hc/en-us/articles/1500009589701-Formula-Levels#CLF).

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
