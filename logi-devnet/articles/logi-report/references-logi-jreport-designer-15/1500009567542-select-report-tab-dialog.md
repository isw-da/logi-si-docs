# Select Report Tab Dialog

The Select Report Tab dialog appears when you select File > Import> Logi JReport Page Report, specify a report in the Open Report dialog and then select Open. It helps you to select the [report tabs you want to import](https://devnet.logianalytics.com/hc/en-us/articles/1500009592941-Managing-Report-Tabs-in-a-Page-Report#Import) from the specified report.  [See the dialog](javascript:%20void(null)).

![Select Report Tab dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889288087/slctrpt.gif)

The following are details about options in this dialog:

**Name**

Displays the names of the report tabs.

**Description**

Displays the description of the report tabs.

**Customize Dataset for Components**

Specifies to customize datasets used by components within the selected report tabs. If this option is checked, when you select OK in the dialog, the [Customize Dataset](https://devnet.logianalytics.com/hc/en-us/articles/1500009586221-Customize-Dataset-Dialog) dialog will be displayed.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
