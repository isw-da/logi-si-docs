# Show Fields Dialog

The dialog appears when you select Customized from the drop-down list of the Show Fields property, which is in the Hint category of the chart paper of a heat map in the Report Inspector. It allows you to select fields to show as the heat map hint, which displays when the mouse pointer points at the data marker in Designer view mode, in HTML result, or at runtime on Logi JReport Server. [See the dialog](javascript:%20void(null)).

![Show Fields dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889322391/shwfld.gif)

From the Resources box select a field and select ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893789463/btn_addarrow.gif) to add it into the right box to make its value shown as the hint. Repeat this to add more fields. The names of the selected fields will be displayed in the value cell of the Show Fields property, separated by ",".

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
