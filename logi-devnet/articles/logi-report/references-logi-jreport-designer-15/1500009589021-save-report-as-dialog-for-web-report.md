# Save Report As Dialog (for Web Report)

The Save Report As dialog for web report appears when you select File > Save As. It helps you to [save the current web report](https://devnet.logianalytics.com/hc/en-us/articles/1500009592981-Web-Reports#Save) with another name in the current location. [See the dialog](javascript:%20void(null)).

![Save Report As dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893804567/svrptas_wb.gif)

The following are details about options in the dialog:

**Report Name**

Specifies the name of the report.

**Report of Type**

Displays the report type .wls by default.

**Save**

Saves the report with the name you specified.

**Cancel**

Closes the dialog, leaving any changes unsaved.

**Help**

Displays the help document about this feature.
