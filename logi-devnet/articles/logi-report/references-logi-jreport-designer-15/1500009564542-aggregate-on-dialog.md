# Aggregate On Dialog

The Aggregate On dialog appears when you right-click a detail column in a table and select Aggregate On from the shortcut menu. It helps you to [summarize data of the detail column](https://devnet.logianalytics.com/hc/en-us/articles/1500009584421-Modifying-a-Table#Aggregate). [See the dialog](javascript:%20void(null)).

![Aggregate On dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893956119/agrgton.gif)

**Aggregate On**

Displays the field bound with the detail column. The field is also the one on which the summary is based.

**Aggregate Function**

Specifies the function to compute the field in the detail column. For details about each function, see [Math Functions](https://devnet.logianalytics.com/hc/en-us/articles/1500009589541-Math-Functions).

**Group By**

* If checked, you can select a group by field from the drop-down list below, to which the summary will be applied. If no field is selected, the summary will be applied to the whole dataset.
* If unchecked, a dynamic summary will be created.

**OK**

Summarizes the detail column data with the specified function and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
