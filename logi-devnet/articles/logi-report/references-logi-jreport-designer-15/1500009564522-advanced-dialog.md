# Advanced Dialog

The Advanced dialog varies according to different source it is opened from.

If it is opened by selecting the Advanced button in the [Color List](https://devnet.logianalytics.com/hc/en-us/articles/1500009564822-Color-List-Dialog) dialog, it helps you to define certain pattern to some specific values in a chart to make the same value the same effect, and the options in the dialog are as follows. Note that this feature does not apply to surface 3-D, gauge bubble 2-D and all scatter chart types, and the settings in the dialog have higher priority than the ones in the Color List dialog. [See the dialog](javascript:%20void(null)).

![Advanced dialog - Color List](https://devnet.logianalytics.com/hc/article_attachments/4404893956375/advncd.gif)

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893791255/btn_add.gif)

Adds a new value line.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404889276311/btn_rmv.gif)

Removes the selected value.

**Values**

* If the chart has series field, specify a series value to which the fill pattern will be applied.
* If the chart has no series field, specify a category value to which the fill pattern will be applied.

**Fill**

Specifies the color schema for the pattern. To edit the color, select the color image and select a color from the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/1500009593081-Menu-Tabs#Palette) or input the hexadecimal value (for example, 0xff0000) of a color directly in the text box.

**Transparency**

Specifies the transparency of the color.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.

If it is opened by selecting the Advanced button in the [Line Pattern List](https://devnet.logianalytics.com/hc/en-us/articles/1500009587741-Line-Pattern-List-Dialog) dialog, it helps you to define certain pattern to some specific values in a chart to make the same value the same effect, and the options in the dialog are as follows. Note that this feature does not apply to surface 3-D, gauge bubble 2-D and all scatter chart types, and the settings in the dialog have higher priority than the ones in the Line Pattern List dialog. [See the dialog](javascript:%20void(null)).

![Advanced dialog - Line Pattern List](https://devnet.logianalytics.com/hc/article_attachments/4404893956631/advncd1.gif)

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893791255/btn_add.gif)

Adds a new value line.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404889276311/btn_rmv.gif)

Removes the selected value.

**Values**

Specifies a series value to which the fill pattern will be applied.

**Line Style**

Specifies the line style for the pattern.

**Thickness**

Specifies the thickness for the line.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.

If it is opened by selecting the button ![Shape button](https://devnet.logianalytics.com/hc/article_attachments/4404893878935/btn_shp.gif) next to the Shape By text box when no field is added to the box and then the Advanced option in the Display screen of the map wizard, it helps you to specify an image as the shape of the markers for the selected group in a geographic map, and the options in the dialog are as follows. [See the dialog](javascript:%20void(null)).

![Advanced dialog - map](https://devnet.logianalytics.com/hc/article_attachments/4404893956887/advncd2.gif)

**Default Icon**

Specifies to use the default icon as the shape of the markers. If you want to use your own icon, uncheck the option.

**Image**

Specifies the image of the markers. Input the URL for a web image or select the Browse button to specify a local image.

* **Width**  
   Specifies the width of the image.
* **Height**  
   Specifies the height of the image.

**Shadow**

Specifies the shadow image of the markers. Input the URL for a web image or select Browse button to specify a local image. Available only to Google Maps types.

* **Width**  
   Specifies the width of the shadow image.
* **Height**  
   Specifies the height of the shadow image.

**Constrain Proportions**

Specifies whether to change the width and height of the images at the same time in a certain proportion. If checked, when the width/height is changed, the height/width will also be changed in a certain proportion.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
