# Add Formula Fields Dialog

The Add Formula Fields dialog appears when you select Menu > Column > Add Formula Fields in the [Query Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009567142-Query-Editor-Dialog). It helps you to [add formulas into a query to be columns](https://devnet.logianalytics.com/hc/en-us/articles/1500009570702-Adding-Formula-Fields-to-a-Query) of the query. [See the dialog](javascript:%20void(null)).

![Add Formula Fields dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893823767/addfmlfld.gif)

The following are details about options in this dialog:

**Formulas**

Lists all the formulas available to be added to the query.

**Expression**

Shows expressions of the selected formula.

**Add**

Adds the selected formula to the query.

**Close**

Exits the dialog.

**Help**

Displays the help document about this feature.
