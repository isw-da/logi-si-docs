# Properties of Wrapper in Library Component

The properties of the wrapper object of a library component are as follows:

| Property Name | Description |
| --- | --- |
| General | |
| Author | Specifies the author of the library component. Data type: String |
| Author Email | Specifies the author's e-mail address. Data type: String |
| Description | Specifies the description of the library component. Data type: String |
| Title | Specifies the title of the library component. Data type: String |
| CSS | |
| [Class](https://devnet.logianalytics.com/hc/en-us/articles/1500009569642-Properties-of-Label-in-Page-Report#CSS) | Indicates the CSS class that will be applied to the title bar of the library component. This property is read only. |
| Refresh | |
| Enable Auto Refresh | Specifies whether to automatically refresh the objects in the library component which are not bound with data at runtime based on a defined interval. Data type: Boolean |
| Interval | Specifies the time interval between two auto refreshes. It ranges from 0 to 86400 seconds. Enter a numeric value to change the interval. It is meaningful to specify the property when Enable Auto Refresh is set to true. Data type: Integer |
