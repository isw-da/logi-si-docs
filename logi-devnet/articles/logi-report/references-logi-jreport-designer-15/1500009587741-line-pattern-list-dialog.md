# Line Pattern List Dialog

The Line Pattern List dialog appears when you select the Line Pattern List button in the Format Line dialog for [page report](https://devnet.logianalytics.com/hc/en-us/articles/1500009587281-Format-Line-Dialog-for-Page-Report-), [web report](https://devnet.logianalytics.com/hc/en-us/articles/1500009565962-Format-Line-Dialog-for-Web-Report-), or [library component](https://devnet.logianalytics.com/hc/en-us/articles/1500009565942-Format-Line-Dialog-for-Library-Component-). It helps you to [modify the line pattern in a line chart](https://devnet.logianalytics.com/hc/en-us/articles/1500009563142-Formatting-the-Lines). [See the dialog](javascript:%20void(null)).

![Line Pattern List dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889331863/lineptrnlist.gif)

The following are details about options in the dialog:

**Patterns**

Specifies the pattern for the lines. Select the items in the box one by one to set the pattern for each line.

* ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893791255/btn_add.gif)  
   Adds a new pattern item.
* ![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404893855383/btn_rmv1.gif)  
   Removes the selected pattern item.
* ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404893808791/btn_mvup.gif)  
   Moves the selected pattern item one step up.
* ![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404889289751/btn_mvdwn.gif)  
   Moves the selected pattern item one step down.

**Line Style**

Specifies the style of the line.

**Thickness**

Specifies the thickness of the line, in pixels.

**Advanced**

Opens the [Advanced](https://devnet.logianalytics.com/hc/en-us/articles/1500009564522-Advanced-Dialog) dialog to specify the line pattern for certain values.

**OK**

Applies the line patterns and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
