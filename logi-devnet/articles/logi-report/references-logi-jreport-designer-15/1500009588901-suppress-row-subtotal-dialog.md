# Suppress Row Subtotal Dialog

The Suppress Row Subtotal dialog appears when you select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404893790999/btn_chsr1.gif) next to the Suppress Row Subtotal option in the Layout screen of the crosstab wizard. It helps you to control which subtotals of the rows in the crosstab will be suppressed, and which will be shown. [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404889319703/sprsrowsbtl.gif)

**Same for all**

If checked, the setting for the first subtotal will be applied to all the subtotals.

**Name**

Lists the names of all subtotals of rows in each compound group.

**Suppress**

Specifies whether to suppress the selected subtotal by selecting true or false from the drop-down list. It could also be controlled by using a [constant level formula](https://devnet.logianalytics.com/hc/en-us/articles/1500009589701-Formula-Levels#CLF).

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
