# Edit Role Dialog

The Edit Role dialog appears when you select a role and then select ![Edit](https://devnet.logianalytics.com/hc/article_attachments/4404893791511/btn_edit.gif) in the [Security](https://devnet.logianalytics.com/hc/en-us/articles/1500009588521-Security-Dialog) dialog, [Edit Business View Security](https://devnet.logianalytics.com/hc/en-us/articles/1500009565182-Edit-Business-View-Security-Dialog) dialog or in the Security tab of the following dialogs: [Category Property](https://devnet.logianalytics.com/hc/en-us/articles/1500009564982-Category-Property-Dialog), [New View Element](https://devnet.logianalytics.com/hc/en-us/articles/1500009588021-New-View-Element-Dialog)  or [Edit View Element](https://devnet.logianalytics.com/hc/en-us/articles/1500009586641-Edit-View-Element-Dialog).

It helps you to edit the role, and consists of the following tabs:

* [General](#General)
* [Relation](#Relation)

**OK**

Accepts all changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.

## General

Specifies the role information. [See the dialog](javascript:%20void(null)).

![Edit Role dialog - General](https://devnet.logianalytics.com/hc/article_attachments/4404893922583/edtrole_gen.gif)

**Role**

Specifies the name of the role.

**Source**

Displays the source of the role. It is read only.

**Parent Roles**

Displays the parent roles of the current role. It is read only.

**Description**

Specifies the description of the role.

## Relation

Specifies the relationship of the role and the existing users and groups. [See the dialog](javascript:%20void(null)).

![Edit Role dialog - Relation](https://devnet.logianalytics.com/hc/article_attachments/4404889366935/edtrole_rel.gif)

**Users**

All available users will be listed here. If a user is checked, the user will be added into the role.

**Groups**

All available groups will be listed here. If a group is checked, the group will be added into the role.
