# Select Pre-join Dialog

The Select Pre-join dialog appears when you select the New button in the [Save Pre-join Path](https://devnet.logianalytics.com/hc/en-us/articles/1500009567742-Save-Pre-join-Path-Dialog) dialog, input a path name and then select OK. It helps you to [define new pre-join paths](https://devnet.logianalytics.com/hc/en-us/articles/1500009583161-Defining-the-Join-Relationships-Between-Resources) in the current catalog.  [See the dialog](javascript:%20void(null)).

![Select Pre-join dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889320471/slctprjn.gif)

The following are details about options in this dialog:

**All Joins**

Lists all the pre-joins in the connection.

**Joins in <path name>**

Lists all joins that are added to the new path.

**Hide Joins Added in Other Paths**

Specifies whether to hide those joins that have been added in other paths.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893789463/btn_addarrow.gif)

Adds a join to the new path.

![Add All button](https://devnet.logianalytics.com/hc/article_attachments/4404893789719/btn_addall.gif)

Adds all listed join from the new path.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404893789975/btn_rmvarrow.gif)

Removes a join from the path.

![Remove All button](https://devnet.logianalytics.com/hc/article_attachments/4404889274263/btn_rmvall.gif)

Removes all joins from the path.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
