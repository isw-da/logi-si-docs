# Add Group Dialog

The Add Group dialog appears when you select ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893791255/btn_add.gif) and then select Add Group from the drop-down menu in the [Security](https://devnet.logianalytics.com/hc/en-us/articles/1500009588521-Security-Dialog) dialog, [Edit Business View Security](https://devnet.logianalytics.com/hc/en-us/articles/1500009565182-Edit-Business-View-Security-Dialog) dialog or in the Security tab of the following dialogs: [Category Property](https://devnet.logianalytics.com/hc/en-us/articles/1500009564982-Category-Property-Dialog), [New View Element](https://devnet.logianalytics.com/hc/en-us/articles/1500009588021-New-View-Element-Dialog) or [Edit View Element](https://devnet.logianalytics.com/hc/en-us/articles/1500009586641-Edit-View-Element-Dialog).

It helps you to add a group to the security policy so as to define its permission on the related resource, and consists of the following tabs:

* [General](#General)
* [Relation](#Relation)

**OK**

Accepts all changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.

## General

Specifies the group information. [See the dialog](javascript:%20void(null)).

![Add Group dialog - General](https://devnet.logianalytics.com/hc/article_attachments/4404893959447/addgroup_gen.gif)

**Group**

Specifies the name of the group.

**Source**

Displays the source of the group. It is read only.

**Parent Groups**

Displays the parent groups of the current group. It is read only.

**Description**

Specifies the description of the group.

## Relation

Specifies the relationship of the group and the existing users and roles. [See the dialog](javascript:%20void(null)).

![Add Group dialog - Relation](https://devnet.logianalytics.com/hc/article_attachments/4404893959703/addgroup_rel.gif)

**Users**

All available users will be listed here. If a user is checked, the user will be added into the group.

**Roles**

All available roles will be listed here. If a role is checked, the group will be added into the role.
