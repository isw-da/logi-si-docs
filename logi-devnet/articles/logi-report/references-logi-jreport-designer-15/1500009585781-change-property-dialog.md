# Change Property Dialog

The Change Property dialog appears when you select \*Property and then select OK in the [Web Action List](https://devnet.logianalytics.com/hc/en-us/articles/1500009567862-Web-Action-List-Dialog) dialog. It helps you to define to change properties of the current data component as a response to the [message](https://devnet.logianalytics.com/hc/en-us/articles/1500009570842-Delivering-Messages-Between-Library-Components) the data component receives at runtime. [See the dialog](javascript:%20void(null)).

![Change Property dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893813143/cpty.gif)

The following are details about options in the dialog:

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893791255/btn_add.gif)

Adds a new property line.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404889276311/btn_rmv.gif)

Removes the selected property line.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404893808791/btn_mvup.gif)

Moves the selected property line one step up.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404889289751/btn_mvdwn.gif)

Moves the selected property line one step down.

**Properties**

Specifies the property the value of which you want to change. Select the desired property from the drop-down list or input the property name by yourself. Only properties of the data component itself can be changed at runtime.

**Value**

Specifies the value of the property.

**OK**

Accepts the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
