# Open Report Dialog

The Open Report dialog appears when you select File > Open or File > Import > Logi JReport Page Report. It helps you to open a report (page report or web report) or a library component, or import report tabs from existing page reports into current page report. [See the dialog](javascript:%20void(null)).

![Open Report dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889285783/opnrptst.gif)

The following are details about options in the dialog:

**Catalog**

Shows the directory of the default catalog that is being used now. You can also select the Browse button to specify a new one.

**Note:** If you are going to import report tabs into the current page report, you can only choose from page reports of the same catalog. In this case, the Browse button is disabled.

**Reports tab**

Specifies one or more reports to open.

* ![](https://devnet.logianalytics.com/hc/article_attachments/4404889322903/btn_list.gif)**List**  
   Selects to go to the List mode where only the names of the files are displayed.
* ![](https://devnet.logianalytics.com/hc/article_attachments/4404893856791/btn_dtl.gif)**Details**  
   Selects to go to the Details mode where the files with detailed information such as name, size, type and last modified time are displayed.
* **Name**  
   Shows the name of all the reports included in this catalog, based on the type you selected from the Report of Type drop-down list.
* **Size**  
   Shows the size of the reports.
* **Type**  
   Shows the type of the reports.
* **Modified**  
   Shows the time when the reports were modified at the last time.
* **Report Name**  
   Shows the names of the selected reports.
* **Report of Type**  
   Specifies the type of the reports.

**Recent tab**

Lists all the reports that have been opened recently.

* **Report Name**  
   Shows the names of all the recently opened reports.
* **Catalog Name**  
   Shows the name of the catalog that the reports are located in.
* **Report Location**  
   Shows the location of all the reports.

**Open**

Opens the selected reports and closes the dialog. If a self-contained page report is selected, the [Open As](https://devnet.logianalytics.com/hc/en-us/articles/1500009588061-Open-As-Dialog) dialog will appear, asking you to specify the way in which the report will be opened.

* To open consecutive reports, select the first report, press and hold down **SHIFT**, and then select the last report. Then select the **Open** button. The last report will be the active report.
* To open reports that are not consecutive, press and hold down **CTRL**, select each report, and then select the **Open** button. The last selected report will be the active report.

**Cancel**

Closes the dialog, leaving any changes unsaved.

**Help**

Displays the help document about this feature.
