# Edit Sort and Filter Dialog

The Edit Sort and Filercdialog appears when you select a data component, which refer to banded object, table, chart and crosstab, in a page report, and select Report > Edit Sort & Filter. It helps you to [predefine the sort and filter conditions](https://devnet.logianalytics.com/hc/en-us/articles/1500009570962-Predefining-Sort-and-Filter-Conditions) for the data component. [See the dialog](javascript:%20void(null)).

![Edit Sort and Filter dialog - Sort](https://devnet.logianalytics.com/hc/article_attachments/4404893808023/edtsrtfltr_srt.gif)

The following are details about options in the dialog:

**Sort tab**

Defines the sorting conditions.

* **Fields**  
   Lists all fields that are available to be sorted on.
* ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893789463/btn_addarrow.gif)  
   Adds the selected fields in the Fields box to the Sort By box.
* ![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404893789975/btn_rmvarrow.gif)  
   Removes the selected fields from the Sort By box.
* **Sort By**  
   Lists the fields that are selected. Report data will be sorted according to the sort by fields you selected.
  + **Direction**  
     Specifies the sort order of the specified field.
  + **Name**  
     Displays the field name.
* ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404893808791/btn_mvup.gif)  
   Moves the selected field up one step. The order of the sort by fields determines the order in which the report data will be sorted.
* ![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404889289751/btn_mvdwn.gif)  
   Moves the selected field down one step.

**Filter tab**

Defines the filtering conditions. The options in the tab as the same as those in the [Edit Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500009565242-Edit-Filter-Dialog) dialog.

**Edit Display Name**

Opens the [Edit Display Name](https://devnet.logianalytics.com/hc/en-us/articles/1500009586401-Edit-Display-Name-Dialog) dialog to add or remove display name fields.

**OK**

Accepts the changes you've made and closes this dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
