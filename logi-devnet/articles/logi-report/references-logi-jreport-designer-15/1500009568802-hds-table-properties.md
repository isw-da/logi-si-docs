# HDS Table Properties

The properties of an HDS table are as follows:

| Property Name | Description |
| --- | --- |
| Attributes | Specifies whether or not the HDS columns of this HDS table can be used to group by. If true, they cannot. Data type: Boolean |
| Description | Specifies the description of the HDS table. Data type: String |
| Full Name | Displays the full path name of the HDS table automatically generated by Logi JReport. This property is read only. |
| Maximum Duration | Specifies the maximum elapsed time allowed to fetch data from the data source when the HDS table runs, measured in seconds. If the value is set to zero or is blank, it means the time will be unlimited. For usage about the property, see [Limiting the Query Run Time and Number of Records](https://devnet.logianalytics.com/hc/en-us/articles/1500009592481-Data-Manager#Limit).  Data type: Integer |
| Maximum Rows | Specifies the maximum number of rows that will be fetched from the data source when the HDS table runs. If the value is set to zero or is blank, it means the number will be unlimited. For usage about the property, see [Limiting the Query Run Time and Number of Records](https://devnet.logianalytics.com/hc/en-us/articles/1500009592481-Data-Manager#Limit).  Data type: Integer |
| Name | Specifies the name of the HDS table. Data type: String |
