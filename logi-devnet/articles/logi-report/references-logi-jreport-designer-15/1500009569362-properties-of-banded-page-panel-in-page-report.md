# Properties of Banded Page Panel in Page Report

A banded page panel can have the following child objects: [Banded Page Header Panel](https://devnet.logianalytics.com/hc/en-us/articles/1500009569422-Properties-of-Banded-Page-Header-Panel-in-Page-Report), [Group Panel](https://devnet.logianalytics.com/hc/en-us/articles/1500009591121-Properties-of-Group-Panel-in-Page-Report) and [Banded Page Footer Panel](https://devnet.logianalytics.com/hc/en-us/articles/1500009591101-Properties-of-Banded-Page-Footer-Panel-in-Page-Report).

The properties of a banded page panel object in a page report are as follows:

| Property Name | Description |
| --- | --- |
| General | |
| Class Type | Indicates the class type of the object. This property is read only. |
| Instance Name | Shows the instance name of the object. This property is read only. |
| Paper | |
| Height | Specifies the height of the page. The Width and Height values change when Orientation changes. The Page Type setting determines the Width and Height values but you can change the latter as desired. Enter a numeric value to change the height. Data type: Float |
| Orientation | Specifies how to position the report page. The Orientation setting affects the Width and Height values. Choose an option from the drop-down list.  * **portrait** - The page is positioned vertically. * **landscape** - The page is positioned horizontally.   Data type: Enumeration |
| Page Type | Specifies the report page dimensions. Choose a size from the drop-down list. Data type: Enumeration |
| Width | Specifies the width of the page. The Width and Height values change when Orientation changes. The Page Type setting determines the Width and Height values but you can change the latter as desired. Enter a numeric value to change the width. Data type: Float |
| Margin | |
| Bottom Margin | Specifies the distance between the bottom page edge and the report results. Enter a numeric value to change the margin. Data type: Float |
| Left Margin | Specifies the distance between the left page edge and the report results. Enter a numeric value to change the margin. Data type: Float |
| Right Margin | Specifies the distance between the right page edge and the report results. Enter a numeric value to change the margin. Data type: Float |
| Top Margin | Specifies the distance between the upper page edge and the report results. Enter a numeric value to change the margin. Data type: Float |
