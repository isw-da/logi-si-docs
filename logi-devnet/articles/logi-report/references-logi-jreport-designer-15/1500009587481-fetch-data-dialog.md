# Fetch Data Dialog

The Fetch Data dialog appears when you select Fetch Data in the following dialogs: [Insert Filter Control (for Report)](https://devnet.logianalytics.com/hc/en-us/articles/1500009566462-Insert-Filter-Control-Dialog-for-Report-), [Insert Filter Control (for Library Component)](https://devnet.logianalytics.com/hc/en-us/articles/1500009587601-Insert-Filter-Control-Dialog-for-Library-Component-), [Edit Filter Control (for Report)](https://devnet.logianalytics.com/hc/en-us/articles/1500009565262-Edit-Filter-Control-Dialog-for-Report-) or [Edit Filter Control (for Library Component)](https://devnet.logianalytics.com/hc/en-us/articles/1500009586441-Edit-Filter-Control-Dialog-for-Library-Component-). It helps you to get data from the database for using in a [filter control](https://devnet.logianalytics.com/hc/en-us/articles/1500009584601-Using-Filter-Control-to-Filter-Data). [See the dialog](javascript:%20void(null)).

![Fetch Data dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893880727/ftchdt.gif)

The following are details about options in the dialog:

**Search toolbar**

It allows you to search for values. Type keyword in the text box and then the values containing the keyword will be listed.

**Checkboxes**

The checkbox at the top allows you select or unselect all the listed values. The following checkboxes are used for selecting the corresponding values on the same rows.

**Values**

The listed values can be sorted in the ascending or descending order by selecting the Values column header.

**OK**

Adds the selected values to the value list right after the cursor in the Insert Filter control dialog.

**Cancel**

Cancels the value selection and exits the dialog.

**Help**

Displays the help document about this feature.
