# New Dataset Dialog

The New Dataset dialog appears when you select the <New Dataset...> item in the Data tab of the report wizard, select the <Choose Data From...> item from the dataset drop-down list in the Data panel for a page report, or select the Add button in the [Manage Datasets](https://devnet.logianalytics.com/hc/en-us/articles/1500009587781-Manage-Datasets-Dialog)  dialog. It helps you to [create a dataset](https://devnet.logianalytics.com/hc/en-us/articles/1500009592421-Creating-and-Managing-Datasets) for a report. [See the dialog](javascript:%20void(null)).

![New Dataset dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889298327/nwdtst.gif)

The following are details about options in the dialog:

**Choose Data from**

Specifies the data source for the dataset. Data sources that exist in the current catalog will be listed here.

**New Dataset Name**

Specifies the name of the dataset.

**OK**

Accepts changes and closes this dialog.

**Cancel**

Does not retain changes and closes this dialog.

**Help**

Displays the help document.
