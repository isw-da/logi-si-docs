# Web Action List Dialog

The Web Action List dialog appears when you select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404893790999/btn_chsr1.gif) in the Actions column in the Web Behaviors box of the [Display Type](https://devnet.logianalytics.com/hc/en-us/articles/1500009565002-Display-Type-Dialog) dialog, in the [Web Behaviors](https://devnet.logianalytics.com/hc/en-us/articles/1500009567882-Web-Behaviors-Dialog) dialog, in the Behaviors tab of some chart formatting dialogs, or in the [Receive Message](https://devnet.logianalytics.com/hc/en-us/articles/1500009567242-Receive-Message-Dialog) dialog when receiving a user defined message. It helps you to bind one or more web actions to an object in a page report or library component. [See the dialog](javascript:%20void(null)).

![Web Action List dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889313687/wbactnlst.gif)

The following are details about options in the dialog:

**Action Name**

Lists the names of actions.

* **Input an Action**  
   If selected, select OK to open the Input Action dialog to input directly an action and its parameter and bind it to the object. Not supported on library components.
* **\*Parameter**  
   If selected, select OK to open the [Parameter - Web Action Builder](https://devnet.logianalytics.com/hc/en-us/articles/1500009588241-Parameter-Web-Action-Builder-Dialog) dialog or [Parameter](https://devnet.logianalytics.com/hc/en-us/articles/1500009567102-Parameter-Dialog) dialog to build the Parameter web action to the object or to do the Parameter action to respond a received message.
* **\*Filter**  
   If selected, select OK to open the [Filter - Web Action Builder](https://devnet.logianalytics.com/hc/en-us/articles/1500009586901-Filter-Web-Action-Builder-Dialog) dialog or [Filter](https://devnet.logianalytics.com/hc/en-us/articles/1500009565642-Filter-dialog-Dialog) dialog to build the Filter web action to the object or to do the Filter action to respond a received message.
* **\*Sort**  
   If selected, select OK to open the [Sort - Web Action Builder](https://devnet.logianalytics.com/hc/en-us/articles/1500009588801-Sort-Web-Action-Builder-Dialog) dialog or [Sort](https://devnet.logianalytics.com/hc/en-us/articles/1500009588781-Sort-Dialog) dialog to build the Sort web action to the object or to do the Sort action to respond a received message.
* **\*Customized Control**  
   If selected, select OK to open the [Customized Control – Web Action Builder](https://devnet.logianalytics.com/hc/en-us/articles/1500009586181-Customized-Control-Web-Action-Builder-Dialog) dialog to bind a customized control web action. Available to page report tables only.
* **\*Customized Control from Lib**   
   If selected, select OK to open the [Customized Control Manager](https://devnet.logianalytics.com/hc/en-us/articles/1500009587761-Manage-Customized-Controls-Dialog) dialog to reference a customized control file from the library.
* **\*Property**  
   If selected, select OK to open the [Change Property - Web Action Builder](https://devnet.logianalytics.com/hc/en-us/articles/1500009585801-Change-Property-Web-Action-Builder-Dialog) dialog or [Change Property](https://devnet.logianalytics.com/hc/en-us/articles/1500009585781-Change-Property-Dialog) dialog to build the Change Property web action to the object or to do the Change Property action to respond a received message. Available to library components only.
* **\*SendMessage**  
   If selected, select OK to open the [Send Message - Web Action Builder](https://devnet.logianalytics.com/hc/en-us/articles/1500009588761-Send-Message-Web-Action-Builder-Dialog) dialog to bind the Send Message web action to the object. Available to library components only.
* **User defined web action - with parameter**  
   If selected, select OK to open the [Parameters](https://devnet.logianalytics.com/hc/en-us/articles/1500009588221-Parameters-Dialog) dialog to specify values of the parameters of a web action. Not supported on library components.
* **User defined web action - without parameter**  
   If selected, select OK to bind the selected action to the web control. Not supported on library components.

**Notes**

Displays some tips about the specified action.

**OK**

Accepts the changes and closes the dialog.

**Cancel**

Discards the changes and closes the dialog.

**Help**

Displays the help document about this feature.
