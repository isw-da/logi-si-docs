# Data Source Drivers Dialog

The Data Source Drivers dialog appears when you select Data Source Drivers on Catalog Manager to [manage cached query results](https://devnet.logianalytics.com/hc/en-us/articles/1500009570622-Cached-Query-Results). It helps you to add or remove a user data source driver into or from a Logi JReport catalog.

The following are details about options in this dialog. [See the dialog](javascript:%20void(null)).

![Data Source Drivers dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893824791/dtsrcdrvr.gif)

**Data Source Drivers**

Lists all the available data source driver names.

**New**

Opens the [Data Source Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009586281-Data-Source-Editor-Dialog) to add a new data source driver.

**Remove**

Removes the selected data source driver.

**Edit**

Edits the selected data source driver.

**OK**

Accepts changes and closes the dialog.

**Cancel**

Cancels the changes and closes the dialog.

**Help**

Displays the help document about this feature.
