# Select Area Name Dialog

The Select Area Name dialog appears when you specify to import areas from a .shp file. It helps you to [define the map area name](https://devnet.logianalytics.com/hc/en-us/articles/1500009584141-Inserting-a-Shape-Map). [See the dialog](javascript:%20void(null)).

![](https://devnet.logianalytics.com/hc/article_attachments/4404893854871/slctareanm.gif)

**Select area name from the ESRI shape file**

Lists all the available fields in the selected ESRI shape file. Select one field in the box to define the area name of the map object.

**Tips**

Lists the fields from which to get the area tooltips, which will be displayed when you hover the mouse over the areas when the report runs in HTML format or in Page Report Studio. The order of the fields in this box determines the display sequence of the tooltips and the tooltips can be displayed only when [Alternate Content Type](https://devnet.logianalytics.com/hc/en-us/articles/1500009569662-Properties-of-Shape-Map-in-Page-Report#Type) on the map object is set to customized.

* To add a field to use its value as tooltip, select it from the box above, then select ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893791255/btn_add.gif).
* To remove a field from the tooltip, select it in the Tips box, then select ![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404889276311/btn_rmv.gif).
* To adjust the order of the fields in the Tips box, select a field, then select ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404893808791/btn_mvup.gif) to move the field one step up, or ![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404889289751/btn_mvdwn.gif) to move it one step down.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
