# CSS Style Definition Dialog

The CSS Style Definition dialog appears when you select the Edit button in the [CSS Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009586141-CSS-Editor-Dialog) or select OK when you select an existing CSS file in the [New CSS Style](https://devnet.logianalytics.com/hc/en-us/articles/1500009587901-New-CSS-Style-Dialog) dialog. It helps you to [edit the specified style](https://devnet.logianalytics.com/hc/en-us/articles/1500009571322-Managing-CSS-Styles) as required. [See the dialog](javascript:%20void(null)).

![CSS Style Definition dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893790743/cssdfntn.gif)

The following are details about options in the dialog:

**Style**

Shows the style that is to be edited.

**All Properties**

Lists all the properties of the selected style.

**Selected Properties**

Lists all the selected properties that you want to specify values to apply to this selected style.

* **Name**  
   Lists the names of the properties.
* **Value**  
   Specifies values for the selected properties.
* **Important**  
   Specifies whether this property is important or not.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893789463/btn_addarrow.gif)

Adds a property to apply to the style.

![Add All button](https://devnet.logianalytics.com/hc/article_attachments/4404893789719/btn_addall.gif)

Adds all the properties listed in the All Properties box to apply to the style.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404893789975/btn_rmvarrow.gif)

Removes a property that is not required from the Selected Properties box.

![Remove All button](https://devnet.logianalytics.com/hc/article_attachments/4404889274263/btn_rmvall.gif)

Removes all the properties listed in the Selected Properties box.

**Save**

Applies the changes and closes this dialog.

**Cancel**

Does not retain the changes and closes this dialog.

**Help**

Displays the help document about this feature.
