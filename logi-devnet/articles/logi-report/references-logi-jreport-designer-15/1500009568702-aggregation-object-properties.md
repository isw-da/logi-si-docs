# Aggregation Object Properties

The properties of an aggregation object are as follows:

| Property Name | Description |
| --- | --- |
| Aggregate Function | Specifies the aggregate function of the object. Data type: String |
| Customized Contents | Displays the contents of the[custom aggregation](https://devnet.logianalytics.com/hc/en-us/articles/1500009562702-Business-View-Elements#CA) . This property is read only. |
| Data Mapping File | Specifies the data mapping file to the object [for NLS use](https://devnet.logianalytics.com/hc/en-us/articles/1500009589921-Applying-National-Language-Support). For example, if the data mapping file name is Category\_de\_DE.properties, set the value as Category (without the language and locale part).  Data type: String |
| Description | Specifies the description of the object, which will be shown when you hover the mouse pointer over the object in the Resources panel of Web Report Studio or the Resource View panel of Page Report Studio. Data type: String |
| Display Name | Specifies the display name of the object. Data type: String |
| Is Customized | Specifies whether the aggregation object is a [custom aggregation](https://devnet.logianalytics.com/hc/en-us/articles/1500009562702-Business-View-Elements#CA). Data type: Boolean |
| Mapping Name | Specifies the mapping name of the column to which the object is mapped. Data type: String |
| Tip | Specifies the tooltip of the object, which will be shown when you hover the mouse pointer over the object in the Resources panel of Web Report Studio or the Resource View panel of Page Report Studio. Data type: String |
| Type | Indicates the type of the view element. This property is read only. |
