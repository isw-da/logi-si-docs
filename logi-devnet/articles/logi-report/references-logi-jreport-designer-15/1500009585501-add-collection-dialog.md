# Add Collection Dialog

The Add Collection dialog appears when you right-click the Schemas node or a database node in the Schemas node of a MongoDB connection and select Add Collection on the shortcut menu in the Catalog Manager. It helps you to [add collections](https://devnet.logianalytics.com/hc/en-us/articles/1500009585181-Managing-Schemas-in-a-MongoDB-Connection) into a Logi JReport catalog via the MongoDB connection you set up.  [See the dialog](javascript:%20void(null)).

![Add Collection dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889392791/addclctn.gif)

**Schemas**

* If you open the dialog by right-clicking the Schemas node, the collections in all databases that you can access will be listed here.
* If you open the dialog by right-clicking a database node, the collections in this database will be listed here.

**Added Schemas**

Lists the schemas with the collections that you have added.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893789463/btn_addarrow.gif)

Adds the selected collections or all the collections in the selected schema to the MongoDB connection.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404893789975/btn_rmvarrow.gif)

Removes the selected collections or all the collections in the selected schema from the Added Schema box.

**OK**

Adds the selected collections to the MongoDB connection and closes the dialog.

**Cancel**

Does not retain changes and closes the dialog.

**Help**

Displays the help document about this feature.
