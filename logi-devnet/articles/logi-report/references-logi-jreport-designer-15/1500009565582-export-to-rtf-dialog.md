# Export to RTF Dialog

The Export to RTF dialog appears when you select File > Export > To RTF. It helps you to [export a report to RTF format](https://devnet.logianalytics.com/hc/en-us/articles/1500009589421-Exporting-to-RTF). [See the dialog](javascript:%20void(null)).

![Export to RTF dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889309591/expt2rtf.gif)

The following are details about options in the dialog:

**Directory**

Specifies the destination directory where the exported RTF file will be placed.

**File Name**

Specifies the name of the exported RTF file. If you do not type the name, Logi JReport will use the report name as the default RTF file name.

**Select Report Tabs**

When exporting a page report, you can specify the report tabs in the page report you want to export to the RTF file. The selected report tabs will be exported in the list order. If the report has only one report tab, it is selected by default.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404893808791/btn_mvup.gif)

Moves the specified report tab one step up.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404889289751/btn_mvdwn.gif)

Moves the specified report tab one step down.

**Best Editing**

Specifies whether to apply flow layout when exporting the report to RTF.

**No Margin**

Specifies whether to remove the margins originally set when the report is designed.

**Run Linked Report**

If the report is linked with other reports, you can check whether or not to generate the linked reports (not including the detail reports) in the exported RTF file.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Cancels the changes and exits the dialog.

**Help**

Displays the help document about this feature.
