# Create Query's Username and Password Dialog

The Create Query's Username and Password dialog appears when you right-click a query and then select Create User from the shortcut menu. It helps you to [set a password to restrict users](https://devnet.logianalytics.com/hc/en-us/articles/1500009570782-Locking-Queries) from editing the query at random. [See the dialog](javascript:%20void(null)).

![Create Query's Username and Password dialog](https://devnet.logianalytics.com/hc/article_attachments/4404889295383/crtqrynmpw.gif)

The following are details about options in the dialog:

**User Name**

Specifies the user name that can edit the query.

**Password**

Specifies the password to restrict users from editing the query.

**Confirm password**

Confirms the password you have specified in the Password text field.

**OK**

Applies all changes and closes the dialog.

**Cancel**

Cancels the changes and exits the dialog.

**Help**

Displays the help document about this feature.
