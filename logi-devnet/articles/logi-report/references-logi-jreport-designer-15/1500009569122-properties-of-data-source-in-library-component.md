# Properties of Data Source in Library Component

The properties of the data source object in a library component are as follows:

| Property Name | Description |
| --- | --- |
| Type | Indicates the type of the data source used by the library component. This property is read only. |

The data source object has two child objects: [Dataset](https://devnet.logianalytics.com/hc/en-us/articles/1500009569142-Properties-of-Dataset-in-Library-Component) and [Refresh Object](https://devnet.logianalytics.com/hc/en-us/articles/1500009569162-Properties-of-Refresh-Object-in-Library-Component).
