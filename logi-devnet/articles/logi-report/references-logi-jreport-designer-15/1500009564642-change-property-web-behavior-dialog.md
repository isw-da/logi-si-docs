# Change Property - Web Behavior Dialog

The Change Property-Web Behavior dialog appears when you right-click a web control in the configuration panel of a library component and select Web Behaviors > Change Property from the shortcut menu. It helps you to build a [Change Property web behavior](https://devnet.logianalytics.com/hc/en-us/articles/1500009570822-Using-the-Configuration-Panel#Property) to the web control so as to change properties of a specified object in the library component at runtime. [See the dialog](javascript:%20void(null)).

![Change Property - Web Behavior dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893817751/cptywbbhvr.gif)

The following are details about options in the dialog:

**Apply Action To**

Specifies to which object to apply the change property action. Select the desired object from the drop-down list.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893791255/btn_add.gif)

Adds a new property line.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404889276311/btn_rmv.gif)

Removes the selected property line.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404893808791/btn_mvup.gif)

Moves the selected property line one step up.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404889289751/btn_mvdwn.gif)

Moves the selected property line one step down.

**Properties**

Specifies which property you want to change. Which properties are available here depends on the object you select from the Apply Action To drop-down list. Select the property you want to change from the drop-down list.

**Value**

Specifies the value of the property. The value may be input by yourself, or be specified by the value of a web control.

**OK**

Accepts the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
