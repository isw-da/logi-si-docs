# Custom Sort Dialog

The Custom Sort dialog appears when you select Custom Sort from the Sort column in the Group screen of the report wizard. It helps you to [set the sorting manner](https://devnet.logianalytics.com/hc/en-us/articles/1500009584401-Setting-the-Sort-Manner) of groups at the specified group level. [See the dialog](javascript:%20void(null)).

![Custom Sort dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893933719/cstmsrt.gif)

The following are details of the dialog:

**Resources**

Lists all the available data resources.

**Sort Group By**

Displays the fields on which the groups will be sorted.

**Sort**

Specifies the sorting manner.

* **Ascend**  
   If selected, groups will be sorted by the values of the first record in each group on the specified field in ascending order.
* **Descend**  
   If selected, groups will be sorted by the values of the first record in each group on the specified field in descending order.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893789463/btn_addarrow.gif)

Adds the selected field as the sort by field.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404893789975/btn_rmvarrow.gif)

Removes the specified sort by field if it is not required.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404893808791/btn_mvup.gif)

Moves the selected sort by field one step up.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404889289751/btn_mvdwn.gif)

Moves the selected sort by field one step down.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Cancels the changes and exits the dialog.

**Help**

Displays the help document about this feature.
