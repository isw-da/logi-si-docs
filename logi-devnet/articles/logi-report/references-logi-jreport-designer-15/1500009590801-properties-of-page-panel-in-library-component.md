# Properties of Page Panel in Library Component

The properties of a page panel object in a library component are the same as [those in a web report](https://devnet.logianalytics.com/hc/en-us/articles/1500009570322-Properties-of-Page-Panel-in-Web-Report).

A page panel object in a library component has a child object, Export Page Setting Object, the properties of which are also the same as [those in a web report](https://devnet.logianalytics.com/hc/en-us/articles/1500009570362-Properties-of-Export-Page-Setting-Object-in-Web-Report).
