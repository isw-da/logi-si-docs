# Format Pattern Dialog

The Format Pattern dialog appears when you set the Format Pattern Type property to be Customize or select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404893790999/btn_chsr1.gif) to modify the Format Pattern property in the Modify Schema Properties screen of the [XML Connection Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009567962-XML-Connection-Wizard-Dialog). It helps you to specify [patterns](https://devnet.logianalytics.com/hc/en-us/articles/1500009564382-Importing-an-XML-Schema#Format) to format the Date, Time or DateTime data in the XML schema. [See the dialog](javascript:%20void(null)).

![Format Pattern dialog](https://devnet.logianalytics.com/hc/article_attachments/4404893887767/fmtptrn.gif)

The following are details about options in the dialog:

**Pattern Fragments**

Lists all the pattern fragments that are supported by Logi JReport Designer.

**Description**

Displays the detail descriptions of the selected pattern fragment.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893789463/btn_addarrow.gif)

Adds a selected pattern fragment to the Pattern box.

**Pattern**

Specifies the pattern to format the Date, Time or DateTime data in the XML schema.

To specify a pattern, select a pattern fragment and select ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404893789463/btn_addarrow.gif) to add it to the Pattern box, or double-click a pattern fragment to add it to the Pattern box. You can add several fragments as you want and any character can be input manually between two pattern fragments as a delimiter.

**Result**

Displays a sample of the current date and time using the patterns you defined in the Pattern box.

**OK**

Accepts the changes and closes the dialog.

**Cancel**

Does not retain the changes and closes the dialog.

**Help**

Displays the help document about this feature.
