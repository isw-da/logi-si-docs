# Using an LDAP Server's Security System

Lightweight Directory Access Protocol (LDAP) is a lightweight client-server protocol for accessing directory services. This topic describes how you can use an LDAP Server's security system and the directory servers that Logi Report Server supports.

With LDAP support, Logi Report Server enables you to import users and groups from the directory servers. Logi Report Server can also access an LDAP server directly using the LDAP Security API implementation.

Currently Logi Report supports the following directory servers: Novell Directory Server, Microsoft Site Server, iPlanet Directory Server, Active Directory, Lotus Domino on NT, and OpenLDAP Directory Server.
