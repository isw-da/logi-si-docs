# Logi Report Designer Guide v19Overview

Logi Report Designer (formerly Logi JReport Designer) is a Swing-based Integrated Development Environment (IDE) that enables sophisticated report design and presentation of critical business data. It provides an intuitive interface, reusable report components, flexible layout, and a tool set for designing and testing reports. With Logi Report Designer, you can build reports using simple drag-and-drop techniques or by using wizard; you can access data from any data source to design and preview reports in order to deliver information to end users in the most relevant and intuitive manner; you can accomplish rapid creation and modification of reports by toggling between design mode and view mode where the report displays with the actual dataset. Once report design is complete, you can publish the report to Logi Report Server for generation, delivery, and management.

The Logi Report Designer Guide v19 Help System provides you with everything you need to know to fully use all of the features of the Designer application. You can access the information by looking at the Table of Contents, or by using the Search feature.
