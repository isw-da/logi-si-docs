# Logi Report Get Started Guide v17 Update 1 Overview

This Get Started Guide provides overview material for version 17 of the Logi Report products. Select the links below to learn more.

[**Logi Report Product Overview**](https://devnet.logianalytics.com/hc/en-us/articles/1500009604362-Logi-Report-Product-Overview)

Logi Report delivers operational business intelligence to enterprise applications through powerful embedded reporting.

**[System Requirements](https://devnet.logianalytics.com/hc/en-us/articles/1500009604422-System-Requirements)**

Both Logi Report Designer and Logi Report Server have some basic installation requirements. Before installing them, check your system to make sure all the requirements have been met.

**[Logi Report Licenses](https://devnet.logianalytics.com/hc/en-us/articles/1500009604302-Logi-Report-Licenses)**

Logi Report has several special licenses which allow you to use some specific features in Logi Report.

[**Supported Report Databases**](https://devnet.logianalytics.com/hc/en-us/articles/1500009604262-Supported-Report-Databases)

Logi Report supports all of the current mainstream databases as well as most databases which support JDBC drivers.

**[Test Server Installation](https://devnet.logianalytics.com/hc/en-us/articles/1500009627601-Test-Server-Installation)**

After Logi Report Server is installed, you can start it and log onto the server console to run reports and dashboards.

[**View Sample Reports**](https://devnet.logianalytics.com/hc/en-us/articles/1500009604482-View-Sample-Reports-)

On the Logi Report Server console page, you can view the sample reports Logi Report provides in different formats.

**[Create Dashboards](https://devnet.logianalytics.com/hc/en-us/articles/1500009627501-Create-Dashboards)**

You can create dashboards using JDashboard, which displays rich visualizations of your data using the vast library of charting elements such as maps, crosstabs, tables and even live motion charts.

**[Create Ad Hoc Reports](https://devnet.logianalytics.com/hc/en-us/articles/1500009604222-Create-Ad-Hoc-Reports)**

You can create ad hoc reports based on predefined business views that are published to Logi Report Server.

[**Start a Visual Analysis Session**](https://devnet.logianalytics.com/hc/en-us/articles/1500009604462-Start-a-Visual-Analysis-Session)

You can visualize the result of every step of business data analysis by dragging and dropping business data fields to the layout module using Visual Analysis.

[**Open Sample Report Templates**](https://devnet.logianalytics.com/hc/en-us/articles/1500009627581-Open-Sample-Report-Templates)

There are many sample reports in Logi Report Designer, you can open any sample report and preview the report result.

**[Start Creating New Reports](https://devnet.logianalytics.com/hc/en-us/articles/1500009604322-Start-Creating-New-Reports)**

Logi Report supports three types of reports, Page Report, Web Report and Library Component. Before you create new reports using Logi Report Designer, you need to first create a catalog and then create connections to your data sources.

[**Online Help**](https://devnet.logianalytics.com/hc/en-us/articles/1500009604342-Online-Help)

Online documentation includes all information about the Logi Report products.
