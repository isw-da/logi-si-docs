# Create Dashboards

JDashboard is a new way of information delivery that displays rich visualizations of your data using the vast library of charting elements such as maps, crosstabs, tables and even live motion charts. These can all be combined with a robust set of interactions. With JDashboard, you can take data exploration to the next level by creating interactive views of business performance.

To create a dashboard, you should have [logged onto the Logi Report Server console](https://devnet.logianalytics.com/hc/en-us/articles/1500009627601-Test-Server-Installation). Then:

1. In the Logi Report Server Start Page, select **Dashboard** in the Create category.

   A blank dashboard is then created.

   ![Blank Dashboard](https://devnet.logianalytics.com/hc/article_attachments/4404904602647/crtdshbrd_hdr.gif)
2. In the Resources panel, navigate to **Component Library** > **Public Components** > **SampleReports**, select **Product Daily Sales.lc**:

   ![Select Library Component](https://devnet.logianalytics.com/hc/article_attachments/4404904602903/crtdshbrd_slctlc.gif)

   And then drag and drop it to the dashboard body:

   ![Drag and Drop Library Component into Dashboard](https://devnet.logianalytics.com/hc/article_attachments/4404904603159/crtdshbrd_drop.gif)
3. Close the Resources panel to view the result:

   ![View the Dashboard](https://devnet.logianalytics.com/hc/article_attachments/4404904603415/crtdshbrd_rst.gif)
4. Save the dashboard.
