# Start a Visual Analysis Session

Visual Analysis is an in-context analysis tool to visualize the result of every step of your work. Simply by dragging and dropping data fields to the layout module, you are able to experience the detailed building up of crosstabs and charts step by step visually.

To start a new Visual Analysis session and perform data analysis, you should have [logged onto the Logi Report Server console](https://devnet.logianalytics.com/hc/en-us/articles/1500009627601-Test-Server-Installation). Then:

1. In the Logi Report Server Start Page, select **Public Folder** in the Open category.
2. Select **New** > **Analysis** on the task bar of the Resources page in the server console.
3. A Visual Analysis window with the Select Data Source dialog is displayed. Select **WorldWideSalesBV** in Data Source 1, and then select **OK**.

   ![Select Data Source](https://devnet.logianalytics.com/hc/article_attachments/4404916886679/va_bv.gif)
4. The Visual Analysis window is loaded. The display type shows Text ![Text button](https://devnet.logianalytics.com/hc/article_attachments/4404904597655/btn_va_txt.gif) by default.

   ![Visual Analysis Window](https://devnet.logianalytics.com/hc/article_attachments/4404916887191/va_window.gif)
5. To view total sales, drag **Total Sales** from the Data Source panel and drop it to the Label button ![Label button](https://devnet.logianalytics.com/hc/article_attachments/4404904597911/btn_va_lbl.gif) in the legend section.

   ![Label Legend](https://devnet.logianalytics.com/hc/article_attachments/4404904598167/va_txt1.gif)

   There is only one value in the data presentation area which is the total sales in the whole business view.
6. Now let's view total sales in different years. Drag **Sales Year** to the column control box ![Column Control box](https://devnet.logianalytics.com/hc/article_attachments/4404904598807/btn_va_drpgrpclmn.gif).

   ![Add Column](https://devnet.logianalytics.com/hc/article_attachments/4404904599063/va_txt2.gif)
7. To add product category as the row header, drag **Category** to the row control box ![Row Control box](https://devnet.logianalytics.com/hc/article_attachments/4404916887447/btn_va_drpgrprw.gif).

   ![Add Row](https://devnet.logianalytics.com/hc/article_attachments/4404916887703/va_txt3.gif)
8. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404904599575/btn_save.gif) on the toolbar to save the result as a visual analysis template.
