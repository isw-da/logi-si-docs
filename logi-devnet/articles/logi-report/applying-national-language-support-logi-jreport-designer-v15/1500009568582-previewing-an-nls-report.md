# Previewing an NLS Report

After you develop a report with National Language Support, you can preview it by selecting the language in which you want the report to display from the Language drop-down menu on the Home/View tab.

You can also select **View** > **Preview As** to [preview the report in other format](https://devnet.logianalytics.com/hc/en-us/articles/1500009592961-Previewing-a-Page-Report#other).
