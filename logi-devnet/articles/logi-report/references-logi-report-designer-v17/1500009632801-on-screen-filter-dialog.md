# On-screen Filter Dialog

The On-screen Filter dialog helps you define to filter data of the current library component as a response to the message the library component receives at runtime. This dialog appears in two scenarios:

* When it appears after you select 0004 - On-screen Filter from the drop-down list of the Message ID column and then select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404904183063/btn_chsr1.gif) in the Actions column of the [Receive Message](https://devnet.logianalytics.com/hc/en-us/articles/1500009610002-Receive-Message-Dialog) dialog, the dialog is used for defining to [receive the built-in On-screen Filter message](https://devnet.logianalytics.com/hc/en-us/articles/1500009613362-Delivering-Messages-Between-Library-Components-#BOnScreenFilter) at runtime.
* When it appears after you select a user defined message from the drop-down list of the Message ID column, select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404904183063/btn_chsr1.gif) in the Actions column of the [Receive Message](https://devnet.logianalytics.com/hc/en-us/articles/1500009610002-Receive-Message-Dialog) dialog, and then select \*On-screen Filter and select OK in the [Web Action List](https://devnet.logianalytics.com/hc/en-us/articles/1500009633961-Web-Action-List-Dialog) dialog, the dialog is used for defining to [receive the user defined On-screen Filter message](https://devnet.logianalytics.com/hc/en-us/articles/1500009613362-Delivering-Messages-Between-Library-Components-#UOnScreenFilter) at runtime.

![On-screen Filter dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904184727/onscrnfltr.gif)

The following are details about options in the dialog:

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404904181783/btn_add.gif)

Adds a new filter condition.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404904182167/btn_rmv.gif)

Removes the selected filter condition.

**Filter On**

Specifies the field on which the filter is based.

**Value**

Specifies the value of how to filter the field.

**OK**

Accepts the changes and closes this dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
