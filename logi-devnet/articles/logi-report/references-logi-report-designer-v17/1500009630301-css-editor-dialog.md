# CSS Editor Dialog

The CSS editor helps you [edit the styles](https://devnet.logianalytics.com/hc/en-us/articles/1500009636241-CSS-Styles#Manage) in a specified CSS file. It appears when you select Home > CSS Editor or View > CSS Editor.

![CSS Editor](https://devnet.logianalytics.com/hc/article_attachments/4404911606679/cssedtr.gif)

The following are details about options in the editor:

**CSS File**

Lists all the available CSS files. Select the specified CSS file to edit its styles.

**Styles**

Displays all the available styles in the selected CSS file.

**Import**

Imports a CSS file from a specified directory.

**Export**

Exports the selected CSS file to a specified directory.

**Duplicate**

Duplicates the selected CSS file.

**Remove**

Removes the selected CSS file.

**New**

Opens the [New CSS Style](https://devnet.logianalytics.com/hc/en-us/articles/1500009609182-New-CSS-Style-Dialog) dialog to create a CSS style.

**Edit**

Opens the [CSS Style Definition](https://devnet.logianalytics.com/hc/en-us/articles/1500009607582-CSS-Style-Definition-Dialog) dialog to edit the selected style.

**Remove**

Removes the selected style.

**Done**

Applies the changes and closes the dialog.

**Help**

Displays the help document about this feature.
