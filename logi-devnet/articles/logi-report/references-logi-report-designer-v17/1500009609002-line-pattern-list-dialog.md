# Line Pattern List Dialog

The Line Pattern List dialog helps you to modify the line pattern for each chart line. It appears when you select the Line Pattern List button in the [Format Line](https://devnet.logianalytics.com/hc/en-us/articles/1500009631481-Format-Line-Dialog#Fill) dialog.

![Line Pattern List dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904259991/lineptrnlist.gif)

The following are details about options in the dialog:

**Patterns**

Specifies the pattern for the lines. Select the items in the box one by one to set the pattern for each line.

* ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404904181783/btn_add.gif)  
   Adds a new pattern item.
* ![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404904239767/btn_rmv1.gif)  
   Removes the selected pattern item.
* ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404904182551/btn_mvup.gif)  
   Moves the selected pattern item one step up.
* ![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404904182807/btn_mvdwn.gif)  
   Moves the selected pattern item one step down.

**Line Style**

Specifies the style of the line.

**Thickness**

Specifies the thickness of the line, in pixels.

**Advanced**

Opens the [Advanced](https://devnet.logianalytics.com/hc/en-us/articles/1500009607222-Advanced-Dialog) dialog to specify the line pattern for certain values.

**OK**

Applies the line patterns and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
