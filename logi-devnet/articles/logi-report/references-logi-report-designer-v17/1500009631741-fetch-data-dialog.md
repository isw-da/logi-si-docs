# Fetch Data Dialog

The Fetch Data dialog helps you to [get data from the database](https://devnet.logianalytics.com/hc/en-us/articles/1500009629421-Using-Advanced-Web-Controls#CusValue) for using in a filter control. It appears when you select Fetch Data in the [Insert Filter Control](https://devnet.logianalytics.com/hc/en-us/articles/1500009632161-Insert-Filter-Control-Dialog) dialog or [Edit Filter Control](https://devnet.logianalytics.com/hc/en-us/articles/1500009630681-Edit-Filter-Control-Dialog) dialog.

![Fetch Data dialog](https://devnet.logianalytics.com/hc/article_attachments/4404916788119/ftchdt.gif)

The following are details about options in the dialog:

**Search box**

It allows you to search for values. Type in the text and the values containing the matched text will be listed.

**Checkbox**

The checkbox at the top allows you select or clear all the listed values. The following checkboxes are used for selecting the corresponding values on the same rows.

**Values**

The listed values can be sorted in the ascending or descending order by selecting the Values column header.

**OK**

Adds the selected values to the value list right after the cursor in the Insert Filter control dialog.

**Cancel**

Cancels the value selection and exits the dialog.

**Help**

Displays the help document about this feature.
