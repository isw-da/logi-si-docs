# Advanced Export Settings Dialog

The Advanced Export Settings dialog helps you to [customize the precision used for exporting](https://devnet.logianalytics.com/hc/en-us/articles/1500009634061-Exporting-Report-Result#Precision) in Logi Report Designer (for a page report, the settings take effect only in report tabs whose [Precision Sensitive](https://devnet.logianalytics.com/hc/en-us/articles/1500009635621-Page-Report-Tab-Properties#PrecisionSensitive) property is set to true in the Report Inspector). It appears when you select the Layout Precision button in the [Export to](https://devnet.logianalytics.com/hc/en-us/articles/1500009609482-Options-Dialog#Export) category of the Options dialog.

![Advanced Export Settings dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904228119/advexptsting.gif)

The following are details about options in the dialog:

**Customize for each format**

If the option is selected, you can customize the precision level for each format. Select the formats to apply the precision level defined for them in the Precision Settings dialog in the report result. For formats that are unselected, the precision settings defined for them in the Precision Settings dialog are ignored, and Logi Report will decide their precision.

* **System Default Settings**  
   Opens the [Precision Settings](https://devnet.logianalytics.com/hc/en-us/articles/1500009609662-Precision-Settings-Dialog) dialog to define the precision level for each format.

**Optimize for speed over visual effect**

If the option is selected, Logi Report will decide the precision level which is oriented towards speed more than visualization.

**OK**

Applies the settings and closes the dialog.

**Cancel**

Do not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
