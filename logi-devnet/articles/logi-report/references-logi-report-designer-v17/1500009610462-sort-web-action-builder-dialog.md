# Sort - Web Action Builder Dialog

The Sort-Web Action Builder dialog helps you to [build a Sort web action](https://devnet.logianalytics.com/hc/en-us/articles/1500009629441-Using-Basic-Web-Controls-in-a-Report#Sort) on an object so as to sort the data components in a report when the event bound with the action is triggered at runtime. It appears when you select \*Sort and select OK in the [Web Action List](https://devnet.logianalytics.com/hc/en-us/articles/1500009633961-Web-Action-List-Dialog) dialog.

![Sort - Web Action Builder dialog](https://devnet.logianalytics.com/hc/article_attachments/4404911682583/sortwbactnbld.gif)

The following are details about options in the dialog:

**Apply Action To**

Lists all the data components in the report. Select a data component to specify sort conditions on it.

**Get Input From**

Specifies where to get the object. You can select Form if you add some objects to a form. Otherwise, only the item Other in Report you can choose.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404904181783/btn_add.gif)

Adds a new sort criterion for the action.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404904182167/btn_rmv.gif)

Removes the selected sort criterion.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404904182551/btn_mvup.gif)

Moves the selected sort criterion up a step.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404904182807/btn_mvdwn.gif)

Moves the selected sort criterion down a step.

**Sort On**

Specifies on which field of the selected data component to perform the sort.

**Sort Value**

Specifies the value of how to sort on the field. You can select one from the drop-down list. It could be Ascending or Descending.

**OK**

Accepts the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
