# Bind Data Dialog

The Bind Data dialog helps you to [bind a business view](https://devnet.logianalytics.com/hc/en-us/articles/1500009634221-Using-Formulas-to-Control-Object-Properties#Hide)  to a web report or library component and then create formulas based on this business view in order to control the Invisible property of objects in the web report or library component. It appears when you select Report > Bind Data.

![Bind Data dialog](https://devnet.logianalytics.com/hc/article_attachments/4404911667991/bddt.gif)

The following are details about options in the dialog:

**Available data resources**

Lists all the predefined business views in the current catalog that can be bound to the web report or library component. Select the one you want to use.

**OK**

Binds the business view to the web report or library component.

**Cancel**

Does not retain the changes and closes the dialog.

**Help**

Displays the help document about this feature.
