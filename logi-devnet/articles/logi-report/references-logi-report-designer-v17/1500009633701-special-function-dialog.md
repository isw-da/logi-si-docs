# Special Function Dialog

The Special Function dialog helps you to [specify a special function](https://devnet.logianalytics.com/hc/en-us/articles/1500009628961-Modifying-Charts#Group) to the field displayed on the category/series axis of a chart, which is of Numeric/String/Date/Time data type. It appears when you select the Special Function button in the Display screen of the chart wizard.

![](https://devnet.logianalytics.com/hc/article_attachments/4404911682327/spclfctn.gif)

The following are details about options in the dialog:

**Field**

Displays on which field the special function will be applied.

**Function**

Specifies the special function for the field.

**OK**

Accepts the changes and closes the dialog.

**Cancel**

Discards the changes and closes the dialog.

**Help**

Displays the help document about this feature.
