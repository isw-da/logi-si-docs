# Merge Dialog

The Merge dialog helps you to [merge two catalogs](https://devnet.logianalytics.com/hc/en-us/articles/1500009628321-Merging-Catalogs). It appears when you select Save in the Save To dialog to save your report in another directory in which a catalog of the same name exists.

![Merge dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904258839/merge.gif)

The following are details about options in the dialog:

**Resources to be merged box**

Displays all the resources referenced by the report with conflicting resources marked.

**Rename**

Renames the selected conflicting resource and copy it to the target catalog with the new name.

**Replace**

Replaces the resource in the target catalog with the selected conflicting resource from the source catalog.

**Skip**

Keeps the values in the target catalog for the selected conflicting resource.

**Differences**

Displays the Properties Differences dialog to show the property values of the selected conflicting resource in the two catalogs. The different properties are highlighted.

**Target Relation**

Displays the Target Relation dialog to show the name of the resource and its parent node in the target catalog.

**Merge**

Closes the dialog and merges the two catalogs according to the changes that you have made.

**Cancel**

Closes the dialog, leaving any changes unsaved.

**Help**

Displays the help document.
