# Suppress Column Subtotals Dialog

The Suppress Column Subtotals dialog helps you to customize which subtotals of the column fields in a crosstab will be suppressed and which will be shown. It appears when you select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404904183063/btn_chsr1.gif) next to the Suppress Column Subtotals option in the Layout screen of the crosstab wizard.

![Suppress Column Subtotal dialog](https://devnet.logianalytics.com/hc/article_attachments/4404911682071/sprsclmnsbtl.gif)

The following are details about options in the dialog:

**Same for all**

If the option is selected, the setting for the subtotal of the first column field in below box will be applied to all the subtotals of the column fields in the crosstab.

**Name**

Lists the column fields excluding the one on the topmost column group level; for a compound crosstab the column fields excluding the one on the topmost column group level in each column compound group.

**Suppress**

Specifies whether the subtotal for a column field will be suppressed. To specify the value, select in the value cell and select true or false from the drop-down list. The value can also be [controlled by a constant level formula](https://devnet.logianalytics.com/hc/en-us/articles/1500009634221-Using-Formulas-to-Control-Object-Properties).

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
