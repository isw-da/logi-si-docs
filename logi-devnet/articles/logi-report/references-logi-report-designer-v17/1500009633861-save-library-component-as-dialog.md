# Save Library Component As Dialog

The Save Library Component As dialog helps you to [save the current library component](https://devnet.logianalytics.com/hc/en-us/articles/1500009636221-Saving-Reports) with another name in the current location. It appears when you select File > Save As when working with library components.

![Save Library Component As dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904238743/svlcas.gif)

The following are details about options in the dialog:

**Report Name**

Specifies the name of the library component.

**Report of Type**

Displays the type of the library component which is .lc.

**Save**

Saves the library component with the name you specified.

**Cancel**

Closes the dialog, leaving any changes unsaved.

**Help**

Displays the help document about this feature.
