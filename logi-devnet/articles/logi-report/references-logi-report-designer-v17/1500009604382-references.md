# References

The Logi Report Designer Help System has two different types of references:

* [Dialog Reference](https://devnet.logianalytics.com/hc/en-us/articles/1500009607082-Dialog-Reference)
* [Object Property Reference](https://devnet.logianalytics.com/hc/en-us/articles/1500009611522-Object-Property-Reference)

The purpose of these references is to make the use of Logi Report Designer much easier and more convenient by providing you with an overview of the functions and properties of objects.
