# Select Field Dialog

The Select Field dialog helps you to [insert a field](https://devnet.logianalytics.com/hc/en-us/articles/1500009613322-Adding-Links-in-Reports#URL) into the URL or e-mail address that an object is linked to, or [insert a field](https://devnet.logianalytics.com/hc/en-us/articles/1500009636121-Creating-Bursting-Reports#MailSettings) in the e-mail subject or comments when you specify to distribute a bursting report to e-mail. It appears when you select the Add Dynamic Field button in the [Insert Link](https://devnet.logianalytics.com/hc/en-us/articles/1500009632221-Insert-Link-Dialog) dialog or [Edit Link](https://devnet.logianalytics.com/hc/en-us/articles/1500009607902-Edit-Link-Dialog) dialog when the link type is URL or E-mail, or in the [E-mail Settings](https://devnet.logianalytics.com/hc/en-us/articles/1500009630981-E-mail-Settings-Dialog) dialog.

![Select Field dialog](https://devnet.logianalytics.com/hc/article_attachments/4404911623191/slctfld.gif)

The following are details about options in the dialog:

**Resources box**

Lists all the available data resources in the currently used dataset. Select a field from the resource tree to insert it into the URL, e-mail address, or e-mail subject/comments to compose dynamic information. You should make sure the field you select contains the appropriate data to compose the information.

* **Current Field**  
  Available when you inserting a field into the URL or e-mail address that an object is linked to. Select the item to use the data field that is related to the trigger object of the link to compose the URL or e-mail address.

**OK**

Applies the selected field and closes the dialog.

**Cancel**

Cancels the selection and exits the dialog.

**Help**

Displays the help document about this feature.
