# Format Pattern Dialog

The Format Pattern dialog helps you to specify [patterns](https://devnet.logianalytics.com/hc/en-us/articles/1500009607062-How-Logi-Report-Gets-Data-from-XML-Data-Sources#Format) to format the Date, Time or DateTime data in the XML schema. It appears when you set the Format Pattern Type property to Customize or select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404904183063/btn_chsr1.gif) to modify the Format Pattern property in the Modify Schema Properties screen of the [XML Connection Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009634021-XML-Connection-Wizard-Dialog).

![Format Pattern dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904299415/fmtptrn.gif)

The following are details about options in the dialog:

**Pattern Fragments**

Lists all the pattern fragments that are supported by Logi Report Designer.

**Description**

Displays the detail descriptions of the selected pattern fragment.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404904175639/btn_addarrow.gif)

Adds a selected pattern fragment to the Pattern box.

**Pattern**

Specifies the pattern to format the Date, Time or DateTime data in the XML schema.

To specify a pattern, select a pattern fragment and select ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404904175639/btn_addarrow.gif) to add it to the Pattern box, or double-click a pattern fragment to add it to the Pattern box. You can add several fragments as you want and use any character between two pattern fragments as a delimiter.

**Result**

Displays a sample of the current date and time using the patterns you defined in the Pattern box.

**OK**

Accepts the changes and closes the dialog.

**Cancel**

Does not retain the changes and closes the dialog.

**Help**

Displays the help document about this feature.
