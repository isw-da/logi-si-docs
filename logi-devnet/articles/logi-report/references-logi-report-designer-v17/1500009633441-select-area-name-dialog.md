# Select Area Name Dialog

The Select Area Name dialog helps you to [define the map area name and tooltip](https://devnet.logianalytics.com/hc/en-us/articles/1500009629181-Shape-Maps#Create). It appears when you specify to import areas from a .shp file.

![Select Area Name dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904243223/slctareanm.gif)

The following are details about options in the dialog:

**Select area name from the ESRI shape file**

Lists all the available fields in the selected ESRI shape file. Select one field in the box to define the area name of the map object.

**Tips**

Lists the fields from which to get the area tooltip.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404904181783/btn_add.gif)

Adds the selected field to use its value in the tooltip.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404904182167/btn_rmv.gif)

Removes the selected field from the tooltip.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404904182551/btn_mvup.gif)

Moves the selected field one step up.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404904182807/btn_mvdwn.gif)

Moves the selected field one step down.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
