# Select Folder Dialog

The Select Folder dialog helps you to specify the folder on the server where you want the [resources to be published](https://devnet.logianalytics.com/hc/en-us/articles/1500009636101-Publishing-and-Downloading-Resources#Remotely) or from which [resources will be downloaded](https://devnet.logianalytics.com/hc/en-us/articles/1500009636101-Publishing-and-Downloading-Resources#DownReport). It appears when you select the Browse button next to the Publish Resource To text box in the [Publish to Logi Report Server](https://devnet.logianalytics.com/hc/en-us/articles/1500009633081-Publish-to-Logi-Report-Server-Dialog) dialog or the Browse button next to the Download Resource From text box in the [Download from Logi Report Server](https://devnet.logianalytics.com/hc/en-us/articles/1500009630581-Download-from-Logi-Report-Server-Dialog) dialog.

![Select Folder dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904242967/slctfldr.gif)

The following are details about options in the dialog:

**Folder**

Lists folders on Logi Report Server in a tree structure.

**Detail Information**

Shows the detailed information of subfolders in the selected folder.

* **Name**  
   Displays the names of subfolders.
* **Type**  
   Displays the file type.
* **Last Modified**   
   Displays the last modified time of the subfolders.
* **Description**  
   Other necessary descriptions of the subfolders.

**New Folder**

Creates a new folder in the selected folder. You can rename the folder by double-clicking the folder in the folder tree.

This button is available only when the dialog is opened by selecting the Browse button next to the Publish Resource To text box in the Publish to Logi Report Server dialog.

**OK**

Saves the information and closes the dialog.

**Cancel**

Closes the dialog, leaving any changes unsaved.

**Help**

Displays the help document about this feature.
