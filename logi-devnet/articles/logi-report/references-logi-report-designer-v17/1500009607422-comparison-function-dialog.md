# Comparison Function Dialog

The Comparison Function dialog helps you to [add a comparison function](https://devnet.logianalytics.com/hc/en-us/articles/1500009629061-Modifying-Crosstabs#Comparison) as an aggregation for a crosstab. It appears when you select the Comparison Function button in the Display screen of the crosstab wizard.

![Comparison Function dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904386583/cmprsnfctn.gif)

The following are details about options in the dialog:

**Function**

Specifies the function of the calculation: Percentage, Permillage, or Difference.

**Comparison Function Spans on Row Direction**

If the option is selected, the comparison function will be placed into the column total cell of the crosstab.

**Comparison Function Spans on Column Direction**

If the option is selected, the comparison function will be placed into the row total cell of the crosstab.

**Break By**

Specifies the first parameter of the comparison function.

**Refer To**

Specifies the other parameter of the comparison function.

**OK**

Accepts all changes and closes the dialog.

**Cancel**

Does not retain any changes and closes this dialog.

**Help**

Displays the help document about this feature.
