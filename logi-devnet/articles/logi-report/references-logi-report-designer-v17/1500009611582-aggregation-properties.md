# Aggregation Properties

This topic lists the properties of an Aggregation object in a business view.

| Property Name | Description |
| --- | --- |
| Aggregate Function | Specifies the [aggregate function](https://devnet.logianalytics.com/hc/en-us/articles/1500009611182-Summaries#Function) of the object. Data type: String |
| Customized Contents | Displays the contents of the[custom aggregation](https://devnet.logianalytics.com/hc/en-us/articles/1500009613122-Business-View-Elements#CA) . This property is read only. |
| Data Mapping File | Specifies the data mapping file to the object [for NLS use](https://devnet.logianalytics.com/hc/en-us/articles/1500009611362-Applying-National-Language-Support). For example, if the data mapping file name is Category\_de\_DE.properties, set the value as Category (without the language and locale part).  Data type: String |
| Description | Specifies the description of the object. Data type: String |
| Display Name | Specifies the display name of the object. Data type: String |
| Is Customized | Specifies whether the aggregation object is a [custom aggregation](https://devnet.logianalytics.com/hc/en-us/articles/1500009613122-Business-View-Elements#CA). Data type: Boolean |
| Mapping Name | Specifies the mapping name of the column to which the object is mapped. Data type: String |
| Tip | Specifies the tooltip of the object, which will be shown when end users hover the mouse pointer over it in the business view resource tree at server runtime. Data type: String |
| Type | Indicates the type of the view element. This property is read only. |
