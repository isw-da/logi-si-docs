# Insert Detail Column Dialog

The Insert Detail Column dialog helps you to [insert new detail columns](https://devnet.logianalytics.com/hc/en-us/articles/1500009629301-Modifying-Tables#DetailColumn) into a table. It appears when you right-click a table or a table column and select Insert > Detail Column from the shortcut menu.

![Insert Detail Column dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904267543/instdtlclmn.gif)

The following are details about options in the dialog:

**Resources**

Lists the data fields in and related to the data resource the table uses that can be used for a detail column. Select the field you want from the list.

**OK**

Inserts the column with the specified resource and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
