# Category Properties

This topic lists the properties of a Category object in a business view.

| Property Name | Description |
| --- | --- |
| Description | Specifies the description of the object. Data type: String |
| Display Name | Specifies the display name of the object. Data type: String |
| Type | Indicates the type of the view element. This property is read only. |
