# Add Collection Dialog

The Add Collection dialog helps you to [add more schemas](https://devnet.logianalytics.com/hc/en-us/articles/1500009629621-Working-with-MongoDB-Connections-in-a-Catalog#AddSchema)  into a Logi Report catalog via the MongoDB connection you set up. It appears when you right-click the Schemas node or a database node in the Schemas node of a MongoDB connection and select Add Collection on the shortcut menu in the Catalog Manager.

![Add Collection dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904401047/addclctn.gif)

**Collections**

* If you open the dialog by right-clicking the Schemas node, the collections in all databases that you can access will be listed here.
* If you open the dialog by right-clicking a database node, the collections in this database will be listed here.

**Added Schemas**

Lists schemas that are transformed from the selected collections.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404904175639/btn_addarrow.gif)

Adds the selected collections you want to transform to relational schemas.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404911602327/btn_rmvarrow.gif)

Removes the selected schemas from the Added Schema box.

**OK**

Adds the selected schemas to the Logi Report catalog and closes the dialog.

**Cancel**

Does not retain changes and closes the dialog.

**Help**

Displays the help document about this feature.
