# Create Connection Dialog

The Create Connection dialog helps you to create a connection to connect a catalog to one of these databases: [Oracle, SQL Server, MySQL, InterSystems IRIS, PostgreSQL](https://devnet.logianalytics.com/hc/en-us/articles/1500009606862-JDBC-Connection-Plug-ins), [JDBC](https://devnet.logianalytics.com/hc/en-us/articles/1500009629541-JDBC-Connections), [JSON](https://devnet.logianalytics.com/hc/en-us/articles/1500009606902-JSON-Connections), [XML](https://devnet.logianalytics.com/hc/en-us/articles/1500009629721-Working-with-XML-Connections-in-a-Catalog), [SOAP Web Service](https://devnet.logianalytics.com/hc/en-us/articles/1500009629701-SOAP-Web-Service-Connections), [MongoDB](https://devnet.logianalytics.com/hc/en-us/articles/1500009629621-Working-with-MongoDB-Connections-in-a-Catalog), and [Hive](https://devnet.logianalytics.com/hc/en-us/articles/1500009606762-Hive-Connections). It appears when you select an item in the Connect category on the [Start Page](https://devnet.logianalytics.com/hc/en-us/articles/1500009613602-Start-Page-).

![Create Connection dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904376087/crtcnnctn.gif)

**Existing Catalog**

Specifies an existing catalog in the local disk.

* **Catalog**  
  Displays the current catalog. You can select another one by selecting ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404904178967/btn_chsr.gif).
* **Data Source**  
   Specifies the catalog data source in which to create the connection.

**New Catalog**

Creates a new catalog.

* **Directory**  
  Specifies the directory where to save the new catalog. You can type the directory manually in the text box or select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404904178967/btn_chsr.gif) to select it.
* **Catalog**  
  Specifies the name of the new catalog.

**OK**

Closes the dialog and opens one of the following dialogs:

* [Connect to Oracle](https://devnet.logianalytics.com/hc/en-us/articles/1500009630121-Connect-to-Oracle-Dialog)
* [Connect to MySQL](https://devnet.logianalytics.com/hc/en-us/articles/1500009630101-Connect-to-MySQL-Dialog)
* [Connect to SQL Server](https://devnet.logianalytics.com/hc/en-us/articles/1500009630141-Connect-to-SQL-Server-Dialog)
* [Connect to InterSystems IRIS](https://devnet.logianalytics.com/hc/en-us/articles/1500009630081-Connect-to-InterSystems-IRIS-Dialog)
* [Connect to PostgreSQL](https://devnet.logianalytics.com/hc/en-us/articles/1500009607462-Connect-to-PostgreSQL-Dialog)
* [Get JDBC Connection Information](https://devnet.logianalytics.com/hc/en-us/articles/1500009608882-Get-JDBC-Connection-Information-Dialog)
* [JSON Connection Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009608942-JSON-Connection-Wizard-Dialog)
* [XML Connection Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009634021-XML-Connection-Wizard-Dialog)
* [SOAP Web Service Data Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009610842-SOAP-Web-Service-Data-Source-Dialog)
* [MongoDB Connection Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009609042-MongoDB-Connection-Wizard-Dialog)

**Cancel**

Cancels the connection creation and closes the dialog.

**Help**

Displays the help document about this feature.
