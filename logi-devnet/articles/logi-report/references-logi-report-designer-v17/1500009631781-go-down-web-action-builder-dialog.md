# Go Down - Web Action Builder Dialog

The Go Down-Web Action Builder dialog helps you to [build a Go Down web action](https://devnet.logianalytics.com/hc/en-us/articles/1500009629441-Using-Basic-Web-Controls-in-a-Report#GoDown) on an object so as to drill data of the data components in a report to lower-level groups according to the predefined hierarchies when the event bound with the action is triggered at runtime. It appears when you select \*Go Down and select OK in the [Web Action List](https://devnet.logianalytics.com/hc/en-us/articles/1500009633961-Web-Action-List-Dialog) dialog.

![Go Down - Web Action Builder dialog](https://devnet.logianalytics.com/hc/article_attachments/4404916787479/godwnwbactnbld.gif)

The following are details about options in the dialog:

**Apply Action To**

Lists the data components in the report. Select a data component to define the go-down condition for it.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404904181783/btn_add.gif)

Adds a new condition line. You can add one condition at most.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404904182167/btn_rmv.gif)

Removes the condition.

**Go Down On**

Specifies the group on which the go-down action will be performed.

**Use Hierarchy**

Specifies the hierarchy that the go-down action will follow.

**By Value**

Specifies by which value of the specified group to perform the go-down action, that is to perform the go-down action with the filter condition *SpecifiedGroupField=TheValue*.

**OK**

Accepts the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
