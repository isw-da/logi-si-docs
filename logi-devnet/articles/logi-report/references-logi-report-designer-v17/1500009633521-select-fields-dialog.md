# Select Fields Dialog

The Select Fields dialog helps you to select the fields according to whose unique values to calculate DistinctSum. It appears when you do one of the following:

* Select DistinctSum from the Aggregate Function drop-down list and then select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404904178967/btn_chsr.gif) next to the Distinct On text box in the [New Summary](https://devnet.logianalytics.com/hc/en-us/articles/1500009609362-New-Summary-Dialog) dialog, [Edit Summary](https://devnet.logianalytics.com/hc/en-us/articles/1500009630881-Edit-Summary-Dialog) dialog, [Insert Summary](https://devnet.logianalytics.com/hc/en-us/articles/1500009632261-Insert-Summary-Dialog) dialog, [New View Element](https://devnet.logianalytics.com/hc/en-us/articles/1500009632741-New-View-Element-Dialog) dialog, [Edit View Element](https://devnet.logianalytics.com/hc/en-us/articles/1500009630941-Edit-View-Element-Dialog) dialog, [New Aggregation](https://devnet.logianalytics.com/hc/en-us/articles/1500009632561-New-Aggregation-Dialog) dialog, [Edit Aggregation](https://devnet.logianalytics.com/hc/en-us/articles/1500009607822-Edit-Aggregation-Dialog) dialog, [Insert Aggregation](https://devnet.logianalytics.com/hc/en-us/articles/1500009632081-Insert-Aggregation-Dialog) dialog or [Aggregate On](https://devnet.logianalytics.com/hc/en-us/articles/1500009607262-Aggregate-On-Dialog) dialog.
* In the Display screen of the [Create Crosstab](https://devnet.logianalytics.com/hc/en-us/articles/1500009607522-Create-Crosstab-Dialog) wizard or [Crosstab Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009630181-Crosstab-Wizard-Dialog), select DistinctSum from the Aggregate text box and then select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404904178967/btn_chsr.gif) in the Distinct On text box.
* In the Summary screen of [Create Table](https://devnet.logianalytics.com/hc/en-us/articles/1500009630281-Create-Table-Dialog#Query) wizard, [Table Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009610662-Table-Wizard-Dialog#Query), [Create Banded Object](https://devnet.logianalytics.com/hc/en-us/articles/1500009630201-Create-Banded-Object-Dialog#Query) wizard or [Banded Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009629921-Banded-Wizard-Dialog#Query) based on query resources, [Horizontal Banded Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009631901-Horizontal-Banded-Wizard-Dialog), [Mailing Label Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009632361-Mailing-Label-Wizard-Dialog) or [Shape Map Data Binding Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009609062-Shape-Map-Data-Binding-Wizard-Dialog), select DistinctSum from the Aggregate Function text box and then select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404904178967/btn_chsr.gif) in the Distinct On text box.
* Right-click one or more aggregations in a crosstab and on the shortcut menu point to Switch Aggregate Function, then select DistinctSum from the submenu.

![Select Fields dialog](https://devnet.logianalytics.com/hc/article_attachments/4404911683607/slctflds.gif)

The following are details about options in the dialog:

**Resources**

Lists all the available data resources.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404904175639/btn_addarrow.gif)

Adds the selected fields from the Resources box to the Distinct On box.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404911602327/btn_rmvarrow.gif)

Removes the selected fields from the Distinct On box.

**Distinct On**

Lists the fields according to whose unique values to calculate DistinctSum.

**OK**

Applies the selected fields and closes the dialog.

**Cancel**

Cancels the selection and exits the dialog.

**Help**

Displays the help document about this feature.
