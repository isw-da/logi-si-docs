# Unique Key Dialog

The Unique Key dialog helps you to specify a unique key to ensure there are no duplicated records in a [real time chart](https://devnet.logianalytics.com/hc/en-us/articles/1500009606162-Creating-Real-Time-Charts), or [in the data components of a library component](https://devnet.logianalytics.com/hc/en-us/articles/1500011552882-Refresh-Object-Properties#AutoRefresh). It appears when you select the Incremental Fetch button in the Display screen of the chart wizard, select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404904183063/btn_chsr1.gif) in the value cell of the Unique Key property of a refresh object for a business view used by a library component in the Report Inspector.

![Unique Key dialog](https://devnet.logianalytics.com/hc/article_attachments/4404911644567/unqkey.gif)

The following are details about options in the dialog:

**Resources**

Lists all the available data resources in the business view used by the real time chart or the library component.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404904175639/btn_addarrow.gif)

Adds the selected field to the Unique Key box.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404911602327/btn_rmvarrow.gif)

Removes the selected field from the Unique Key box.

**Unique Key**

Lists the fields you have added as the unique key, which is used to guarantee there are no two records with the same unique key value in the real time chart, or in the data components created on the business view in the library component.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
