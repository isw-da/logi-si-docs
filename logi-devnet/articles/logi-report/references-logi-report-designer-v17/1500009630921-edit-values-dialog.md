# Edit Values Dialog

The Edit Values dialog helps you to [edit the field values](https://devnet.logianalytics.com/hc/en-us/articles/1500009605382-Business-View-Security#Configure) for the allowed set or denied set when configuring business view security. It appears after you select users, roles, and groups, select a group field in the Resources panel, unselect Use Default if it is selected, and then select ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/4404911588375/btn_edit.gif) in the Member Level Security box, in one of the following dialogs or tabs: [Edit Business View Security](https://devnet.logianalytics.com/hc/en-us/articles/1500009607842-Edit-Business-View-Security-Dialog) dialog, Category Property dialog > [Security](https://devnet.logianalytics.com/hc/en-us/articles/1500009630441-Category-Property-Dialog#Security) tab, New View Element dialog > [Security](https://devnet.logianalytics.com/hc/en-us/articles/1500009632741-New-View-Element-Dialog#Security) tab, and Edit View Element dialog > [Security](https://devnet.logianalytics.com/hc/en-us/articles/1500009630941-Edit-View-Element-Dialog#Security) tab.

![Edit Values dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904344727/edtvlu.gif)

There are two ways of specifying the field values: select values from the available ones or compose an expression to retrieve the values. Only one can be used.

* **Selected Values**  
   In this way, you can select values from the available ones.
  + **Available Values**  
     Lists all the available values of the field.
  + **<All>**  
     If the option is selected, it means all possible values, not restricted to all the current displayed values. If unselected, you can select the specific values.
  + **Search box**   
     It allows you to search for values. Type in the text and the values containing the matched text will be listed.
  + ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404904175639/btn_addarrow.gif)  
     Adds the selected value from the Available Values box to the Selected Values box.
  + ![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404911602327/btn_rmvarrow.gif)  
     Removes the selected value from the Selected Values box.
  + ![Add All button](https://devnet.logianalytics.com/hc/article_attachments/4404911602071/btn_addall.gif)  
     Adds all the values from the Available Values box to the Selected Values box.
  + ![Remove All button](https://devnet.logianalytics.com/hc/article_attachments/4404911605783/btn_rmvall.gif)  
     Removes all the values from the Selected Values box.
  + **Selected Values**  
     Lists the values that are selected.
* **Expression**  
   In this way, you can compose an expression to retrieve the values.
  + ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404904178967/btn_chsr.gif)  
     Opens the [Edit Conditions](https://devnet.logianalytics.com/hc/en-us/articles/1500009607862-Edit-Conditions-Dialog#FieldValue) dialog for composing an expression to retrieve the values.

**OK**

Accepts the changes and closes the dialog.

**Cancel**

Cancels the changes and closes the dialog.

**Help**

Displays the help document about this feature.
