# Page Panel Dialog

The Page Panel dialog helps you to create a [page panel](https://devnet.logianalytics.com/hc/en-us/articles/1500009613402-Designing-the-Report-Pages#Panel). It appears when you select Insert > Page Panel.

![Page Panel dialog](https://devnet.logianalytics.com/hc/article_attachments/4404911609239/pgpnl.gif)

The following are details about the options in the dialog:

**Keep Current Page Setting**

Specifies to create a new page panel with the current page settings.

**Use New Page Setting**

Opens the [Page Setup](https://devnet.logianalytics.com/hc/en-us/articles/1500009609582-Page-Setup-Dialog) dialog to create a new page panel with new page settings.

**OK**

Accepts all changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
