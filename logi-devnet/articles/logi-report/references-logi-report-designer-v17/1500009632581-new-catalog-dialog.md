# New Catalog Dialog

The New Catalog dialog helps you to [create a catalog](https://devnet.logianalytics.com/hc/en-us/articles/1500009628301-Creating-Opening-and-Saving-Catalogs#Create) in Logi Report Designer. It appears when you select File > New Catalog.

![New Catalog dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904251159/nwctlg.gif)

The following are details about options in the dialog:

**Name**

Specifies the name of the catalog file. The name should contain the extension .cat or .cat.xml.

**Data Source Name**

Specifies the name of the data source in the catalog. The name cannot contain spaces and special characters other than underscore.

**Directory**

Specifies the location to save the new catalog. Type the directory of the catalog in the text box or select the button ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404904178967/btn_chsr.gif) to find a directory path. You do not want to select a directory that already contains a catalog.

**OK**

Accepts all changes and closes the dialog.

**Cancel**

Closes the dialog, leaving any changes unsaved.

**Help**

Displays the help document about this feature.
