# Add Group Dialog

The Add Group dialog helps you to add a group when editing a [security entry](https://devnet.logianalytics.com/hc/en-us/articles/1500009605402-Record-Level-and-Column-Level-Security#Data) or [business view security](https://devnet.logianalytics.com/hc/en-us/articles/1500009605382-Business-View-Security#Configure) so as to define its permission on different resources. It appears when you select ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404904181783/btn_add.gif) and then select Add Group from the drop-down menu in the [Security](https://devnet.logianalytics.com/hc/en-us/articles/1500009633321-Security-Dialog) dialog, [Edit Business View Security](https://devnet.logianalytics.com/hc/en-us/articles/1500009607842-Edit-Business-View-Security-Dialog) dialog or in the Security tab of the following dialogs: [Category Property](https://devnet.logianalytics.com/hc/en-us/articles/1500009630441-Category-Property-Dialog), [New View Element](https://devnet.logianalytics.com/hc/en-us/articles/1500009632741-New-View-Element-Dialog) or [Edit View Element](https://devnet.logianalytics.com/hc/en-us/articles/1500009630941-Edit-View-Element-Dialog).

The dialog contains two tabs: [General](#General) and [Relation](#Relation).

**OK**

Accepts all changes and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.

## General

Specifies the group information.

![Add Group dialog - General](https://devnet.logianalytics.com/hc/article_attachments/4404904399767/addgroup_gen.gif)

**Group**

Specifies the name of the group.

**Source**

Displays the source of the group. It is read only.

**Parent Groups**

Displays the parent groups of the current group. It is read only.

**Description**

Specifies the description of the group.

## Relation

Specifies the relationship of the group and the existing users and roles.

![Add Group dialog - Relation](https://devnet.logianalytics.com/hc/article_attachments/4404904400279/addgroup_rel.gif)

**Users**

All available users will be listed here. If a user is selected, the user will be added into the group.

**Roles**

All available roles will be listed here. If a role is selected, the group will be added into the role.
