# Edit Style Group Dialog

The Edit Style Group dialog helps you to [edit a style group](https://devnet.logianalytics.com/hc/en-us/articles/1500009613482-XSD-Styles#Modify). It appears when you right-click a style group and select Edit Group from the shortcut menu after selecting Style on Catalog Manager.

![Edit Style Group dialog](https://devnet.logianalytics.com/hc/article_attachments/4404904176407/edtstylgrp.gif)

The following are details about options in the dialog:

**Style Group Name**

Shows the name of the style group.

**New Style**

Opens the New Style dialog to create a new style.

* **Style Name**  
   Specifies the name of the new style.
* **Style Type**  
   Specifies the type of the new style.
* **Description**  
   Specifies the description of the new style.
* **OK**  
   Creates a new style in this style group.
* **Cancel**  
   Cancels creating a new style.

**All Styles List**

Lists all the styles that are not selected to apply to this group.

**Selected Styles List**

Lists all the selected styles that are selected to apply to this group.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404904175639/btn_addarrow.gif)

Adds the selected style into the group.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404911602327/btn_rmvarrow.gif)

Removes the selected style from the group.

![Remove All button](https://devnet.logianalytics.com/hc/article_attachments/4404911605783/btn_rmvall.gif)

Removes all the styles from the group.

**OK**

Applies the changes and closes the dialog.

**Cancel**

Closes the dialog, leaving any changes unsaved.
