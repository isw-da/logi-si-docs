# National Language Support

Logi Report products implement a National Language Support (NLS) feature, with which you can run Logi Report Server (including Page Report Studio, Web Report Studio, JDashboard and Visual Analysis), and reports in different language environments.

NLS can also be edited on Logi Report Designer. For more information, refer to National Language Support in the *Logi Report Designer Guide*.

The following topics provide more information about NLS:

* [NLS at Application Level](https://devnet.logianalytics.com/hc/en-us/articles/1500009750301-NLS-at-Application-Level)
* [NLS at Report Level](https://devnet.logianalytics.com/hc/en-us/articles/1500009750321-NLS-at-Report-Level)

**Note**: NLS feature is not supported on shape map.
