# Using Logi JReport Server Monitor

This topic lists the main features of Logi JReport Server Monitor. Logi JReport Server Monitor is a standalone web-based application used for monitoring the overall performance of Logi JReport Server. It should be used together with Logi JReport Server.

The following are the main features of Logi JReport Server Monitor:

* Inspects the status of Logi JReport Server.
* Shows server performance statistics in Graph/Text mode.
* Maintains Logi JReport Server.
* Creates profiling reports, performance reports and statistic reports.
