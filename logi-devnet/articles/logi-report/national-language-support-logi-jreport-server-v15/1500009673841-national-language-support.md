# National Language Support Logi JReport Server Guide v15

Logi JReport products implement a National Language Support (NLS) feature, with which you can run Logi JReport Server (including Page Report Studio, Web Report Studio, JDashboard and Visual Analysis), and reports in different language environments.

NLS can also be edited on Logi JReport Designer. For detailed information, see "National Language Support" in the *Logi JReport Designer User's Guide*.

National Language Support information is split into two topics:

* [NLS at Application Level](https://devnet.logianalytics.com/hc/en-us/articles/1500009673861-NLS-at-Application-Level)
* [NLS at Report Level](https://devnet.logianalytics.com/hc/en-us/articles/1500009649142-NLS-at-Report-Level)

**Note**: NLS feature is not supported on shape map.
