# Design/View Area

This topic introduces the Design/View area of Designer.

![Design/View Area](https://devnet.logianalytics.com/hc/article_attachments/4420550821271/rptenv_area.gif)

The Design/View area, or report designing window, comprises of the following sections:

**Title bar**

The title bar shows the report file name, and three icons for you to minimize, maximize, and close the active report.

**Design tab**

The Design tab, which is also called the report editor, is for laying out your report. You can use rulers, grids, and guidelines to position objects. You can customize the appearance of the tab via the [Options dialog box](https://devnet.logianalytics.com/hc/en-us/articles/4405661667735-Options-Dialog-Box#Editor).

**View tab**

The View tab is for previewing the report layout and result. The report may not display exactly as users would see it on Server. Interactive objects such as parameter and filter objects are not active and report links are not active. In this tab, there is a toolbar which consists of the following icons:

* ![First Page button](https://devnet.logianalytics.com/hc/article_attachments/4420550821655/btn_fstpg.gif)**First Page**  
  Select to display the first page of the report.
* ![Previous Topic button](https://devnet.logianalytics.com/hc/article_attachments/4420542069783/btn_prvspg.gif)**Previous Page**  
  Select to display the previous page of the report.
* ![Next Topic button](https://devnet.logianalytics.com/hc/article_attachments/4420542070423/btn_nxtpg.gif)**Next Page**  
  Select to display the next page of the report.
* ![Last Page button](https://devnet.logianalytics.com/hc/article_attachments/4420556088855/btn_lastpg.gif)**Last Page**  
  Select to display the last page of the report.
* ![TOC button](https://devnet.logianalytics.com/hc/article_attachments/4420556089111/btn_toc.gif)**TOC**  
  Select to show the Table of Contents.
* ![Refresh Data button](https://devnet.logianalytics.com/hc/article_attachments/4420556089623/btn_refresh.gif)**Refresh Data**  
  Select to refresh the data with the same parameters.
* ![Refetch Data button](https://devnet.logianalytics.com/hc/article_attachments/4420542071447/btn_refetchdt.gif)**Refetch Data**  
  Select to refetch data to rerun the report with different parameters.
* ![Stop button](https://devnet.logianalytics.com/hc/article_attachments/4420550822295/btn_stop.gif)**Stop**  
  Select to stop Designer from generating the report.
* ![Back button](https://devnet.logianalytics.com/hc/article_attachments/4420542071703/btn_bck.gif)**Back**  
  Select to navigate to the last view.
* ![Forward button](https://devnet.logianalytics.com/hc/article_attachments/4420542071959/btn_fwd.gif)**Forward**  
  Select to navigate forward.
* ![Drag Tool button](https://devnet.logianalytics.com/hc/article_attachments/4420556090775/btn_drgtl.gif)**Drag Tool**  
  Select to drag the page up and down. Select the icon, then place your mouse pointer on any blank area of the report. When the mouse pointer becomes a small hand, you can drag the page.
* ![Zoom In Tool button](https://devnet.logianalytics.com/hc/article_attachments/4420556091031/btn_zmin.gif)**Zoom In Tool**  
  Select to zoom in to see more detail. Select the icon, then select in the border of the report repeatedly to zoom in.
* ![Zoom Out Tool button](https://devnet.logianalytics.com/hc/article_attachments/4420542072343/btn_zmout.gif)**Zoom Out Tool**  
  Select to zoom out to see the overall view of the page. Select the icon, then select in the border of the report repeatedly to zoom out.

**Report tab bar**

The report tab bar is for switching between the report tabs in a page report. It also provides a shortcut menu that enables you to rename, duplicate, and remove the report tabs. For more information, see [Managing Report Tabs with the Report Tab Bar](https://devnet.logianalytics.com/hc/en-us/articles/4405664693399-Managing-Page-Reports#TabBar).
