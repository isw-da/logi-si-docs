# Design/View Area

This topic introduces the Design/View area of Designer.

![Design/View Area](https://devnet.logianalytics.com/hc/article_attachments/4404848382231/rptenv_area.gif)

The Design/View area, or report designing window, comprises of the following parts:

**Title bar**

The title bar shows the report file name, and three icons for you to minimize, maximize, and close the active report.

**Design tab**

The Design tab, which is also called the report editor, is for laying out your report. You can make use of rulers, grids, and guidelines to position objects. You can customize the appearance of the tab via the [Options dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010098161-Options-Dialog-Box#Editor).

**View tab**

The View tab is for previewing the report layout and result. The report may not display exactly as the report users would see it in Server. Interactive objects such as parameter and filter objects are not active and report links are not active. In this tab, there is a toolbar which consists of the following icons:

* ![First Page button](https://devnet.logianalytics.com/hc/article_attachments/4404856808855/btn_fstpg.gif)**First Page**  
  Select to display the first page of the report.
* ![Previous Topic button](https://devnet.logianalytics.com/hc/article_attachments/4404848382871/btn_prvspg.gif)**Previous Topic**  
  Select to display the Previous Topic of the report.
* ![Next Topic button](https://devnet.logianalytics.com/hc/article_attachments/4404848383127/btn_nxtpg.gif)**Next Topic**  
  Select to display the Next Topic of the report.
* ![Last Page button](https://devnet.logianalytics.com/hc/article_attachments/4404856810135/btn_lastpg.gif)**Last Page**  
  Select to display the last page of the report.
* ![TOC button](https://devnet.logianalytics.com/hc/article_attachments/4404856811671/btn_toc.gif)**TOC**  
  Select to show the Table of Contents.
* ![Refresh Data button](https://devnet.logianalytics.com/hc/article_attachments/4404848383767/btn_refresh.gif)**Refresh Data**  
  Select to refresh the data with the same parameters.
* ![Refetch Data button](https://devnet.logianalytics.com/hc/article_attachments/4404848385303/btn_refetchdt.gif)**Refetch Data**  
  Select to re-fetch data to re-run the report with different parameters.
* ![Stop button](https://devnet.logianalytics.com/hc/article_attachments/4404848385559/btn_stop.gif)**Stop**  
  Select to stop Designer from generating the report.
* ![Back button](https://devnet.logianalytics.com/hc/article_attachments/4404856813079/btn_bck.gif)**Back**  
  Select to navigate to the last view.
* ![Forward button](https://devnet.logianalytics.com/hc/article_attachments/4404856813335/btn_fwd.gif)**Forward**  
  Select to navigate forward.
* ![Drag Tool button](https://devnet.logianalytics.com/hc/article_attachments/4404856813591/btn_drgtl.gif)**Drag Tool**  
  Select to drag the page up and down. Select the icon, then place your mouse pointer on any blank area of the report. When the mouse pointer becomes a small hand, you can drag the page.
* ![Zoom In Tool button](https://devnet.logianalytics.com/hc/article_attachments/4404848386199/btn_zmin.gif)**Zoom In Tool**  
  Select to zoom in to see more detail. Select the icon, then select in the border of the report repeatedly to zoom in.
* ![Zoom Out Tool button](https://devnet.logianalytics.com/hc/article_attachments/4404856814103/btn_zmout.gif)**Zoom Out Tool**  
  Select to zoom out to see the overall view of the page. Select the icon, then select in the border of the report repeatedly to zoom out.

**Report tab bar**

The report tab bar is for switching between the report tabs in a page report. It also provides a shortcut menu that enables you to rename, duplicate, and remove the report tabs. For more information, see [Managing Report Tabs with the Report Tab Bar](https://devnet.logianalytics.com/hc/en-us/articles/1500010101281-Managing-Page-Reports#TabBar).
