# Report Development Environment

This topic introduces the layout of the Logi Report Designer reporting environment, and the functions of each part of Logi Report Designer. For simple operations, you have been given a brief how-to introduction. Other complex commands associated with this topic have been covered in detail in the specific feature's documents.

![Logi Report Designer](https://devnet.logianalytics.com/hc/article_attachments/4404904172823/rptenv.gif)

The Logi Report development environment consists of the following parts:

* [Start Page](https://devnet.logianalytics.com/hc/en-us/articles/1500009613602-Start-Page-)
* [Menu Tabs](https://devnet.logianalytics.com/hc/en-us/articles/1500009613582-Menu-Tabs)
* [Catalog Manager](https://devnet.logianalytics.com/hc/en-us/articles/1500009613542-Catalog-Manager)
* [Data Panel](https://devnet.logianalytics.com/hc/en-us/articles/1500009636281-Data-Panel)
* [Inspector Panel](https://devnet.logianalytics.com/hc/en-us/articles/1500009613562-Inspector-Panel)
* [Components Panel](https://devnet.logianalytics.com/hc/en-us/articles/1500009613522-Components-Panel)
* [Design/View Area](https://devnet.logianalytics.com/hc/en-us/articles/1500009613502-Design-View-Area)

**Tips:**

* At any time you can select **Quick Start** in the menu tab section to open the Quick Start page which provides entries to some key functions of Logi Report Designer.
* The three panels can be docked into Logi Report Designer or displayed as floating windows. To make a panel a floating one, move the pointer to the panel bar till it becomes a four-way arrow, drag it out of the docking area. This is especially useful when you have multiple monitors available. To return a panel to it's docked position, simply drag the bar containing the tip “Drag here to dock.” of the panel to the left or right side of Logi Report Designer and when you see a box attached to your mouse pointer, release the mouse button and the panel will be docked there.
