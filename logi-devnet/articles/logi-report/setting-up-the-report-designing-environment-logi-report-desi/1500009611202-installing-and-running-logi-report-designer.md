# Installing and Running Logi Report Designer

Logi Report Designer can run on Windows, macOS, Unix, and Linux platforms. This topic describes installing and running Designer on different platforms using different methods.

The Logi Report Designer installer requires that you choose a Java Development Kit (JDK) to complete the installation. These JDKs are supported: Oracle JDK 8, 9, 11, and 12 and OpenJDK 8, 11, 12, and 13. Contact Logi Analytics Support at [support@logianalytics.com](mailto:support@logianalytics.com) for help if you have problems when using other JDK.
