# Introducing the Designer Reporting Environment

This topic introduces the layout of the Designer reporting environment, and the functions of each part of Designer. For simple operations, it only provides a brief how-to introduction. You can get more information about the features in the linked topics.

![Logi Report Designer](https://devnet.logianalytics.com/hc/article_attachments/4404848387351/rptenv.gif)

The Designer development environment consists of the following parts:

* [Start Page](https://devnet.logianalytics.com/hc/en-us/articles/1500010062942-Start-Page)
* [Menu Tabs](https://devnet.logianalytics.com/hc/en-us/articles/1500010062922-Menu-Tabs)
* [Catalog Manager](https://devnet.logianalytics.com/hc/en-us/articles/1500010101481-Catalog-Manager)
* [Data Panel](https://devnet.logianalytics.com/hc/en-us/articles/1500010101501-Data-Panel)
* [Inspector Panel](https://devnet.logianalytics.com/hc/en-us/articles/1500010062902-Inspector-Panel)
* [Components Panel](https://devnet.logianalytics.com/hc/en-us/articles/1500010062882-Components-Panel)
* [Design/View Area](https://devnet.logianalytics.com/hc/en-us/articles/1500010062862-Design-View-Area)

![Note icon](https://devnet.logianalytics.com/hc/article_attachments/4404856814359/noteicon.jpg) You can dock the three panels into Designer or display them as floating windows. To make a panel a floating one, move the pointer to the panel bar till it becomes a four-way arrow, drag it out of the docking area. This is especially useful when you have multiple monitors available. To return a panel to it's docked position, simply drag the bar containing the tip "Drag here to dock." of the panel to the left or right side of Designer and when you see a box attached to your mouse pointer, release the mouse button and Designer docks the panel there.
