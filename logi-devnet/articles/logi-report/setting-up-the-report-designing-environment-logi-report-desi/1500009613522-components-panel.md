# Components Panel

The Components panel lists all the components that can be inserted into a report (to display the panel, select **Components** on the Home or View menu tab).

![Components panel](https://devnet.logianalytics.com/hc/article_attachments/4404904169111/rptenv_cmpnt.gif)

Components in the panel are classified as follows and they vary according to different report types.

* **Visual**: chart, map, KPI, rank
* **Grid**: table, crosstab and tabular
* **Basic**: label, text box, image, subreport, banded object
* **Web Controls**
* **Others**: UDO, barcode, and multimedia objects

You can view the components in either icon view or list view. Select ![List View button](https://devnet.logianalytics.com/hc/article_attachments/4404911594007/btn_listvw.gif) to switch to list view, or ![Icon View button](https://devnet.logianalytics.com/hc/article_attachments/4404904169367/btn_iconvw.gif) to icon view.

The method to create a component using the Components panel is to drag an icon representing that component from the panel to the desired location in the report design area. For details about the usage of each component, refer to [Components](https://devnet.logianalytics.com/hc/en-us/articles/1500009605422-Working-with-Components-in-Reports).
