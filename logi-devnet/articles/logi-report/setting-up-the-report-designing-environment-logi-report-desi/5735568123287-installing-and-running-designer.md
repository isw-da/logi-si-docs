# Installing and Running Designer

Designer can run on Windows, macOS, UNIX, and Linux platforms. This topic introduces installing and running Designer on different platforms using different methods. It also describes how you can update your Designer license key without reinstalling it, configure the Logi Report logging system, and predefine proxy settings when you need to connect to a REST Web Service in a restricted network.

![Note icon](https://devnet.logianalytics.com/hc/article_attachments/9898403122327/noteicon.jpg) You need to choose a Java Development Kit (JDK) to install Designer. Logi Report supports the Oracle JDK and OpenJDK versions as stated in [System Requirements](https://devnet.logianalytics.com/hc/en-us/articles/5735516407191-System-Requirements#JDK). If you need more information, contact [Customer Service](mailto:CustomerService@LogiAnalytics.com?subject=I%20have%20a%20product%20related%20question.).
