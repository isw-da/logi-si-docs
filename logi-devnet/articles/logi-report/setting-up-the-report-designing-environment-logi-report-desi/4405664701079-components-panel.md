# Components Panel

The Components panel lists all the components that you can insert into a report (to display the panel, navigate to **Home**/**View** > **Components**). This topic introduces the usage of the panel generally.

![Components panel](https://devnet.logianalytics.com/hc/article_attachments/4420542068887/rptenv_cmpnt.gif)

Designer classifies the components into the following categories. You can use different components in different report types.

* **Visual**: Chart, Map, KPI, and Rank
* **Grid**: Table, Crosstab, and Tabular
* **Basic**: Label, Text Box, Image, Subreport, and Banded Object
* **Web Controls**
* **Others**: UDO, Barcode, QR Code, and Multimedia Object

You can view the components in either icon view or list view. Select ![List View button](https://devnet.logianalytics.com/hc/article_attachments/4420550820375/btn_listvw.gif) to switch to list view, or ![Icon View button](https://devnet.logianalytics.com/hc/article_attachments/4420550820631/btn_iconvw.gif) to icon view.

The method to create a component using the Components panel is to drag an icon representing that component from the panel to the desired location in the report design area. For more information about the usage of each component, see [Components](https://devnet.logianalytics.com/hc/en-us/articles/4405664365463-Working-with-Components-in-Reports).
