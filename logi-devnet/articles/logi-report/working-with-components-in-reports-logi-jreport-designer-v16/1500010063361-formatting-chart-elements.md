# Formatting Chart Elements

The elements of a chart, such as the platform, wall, legend, data markers and so on can be formatted to take on special appearances.

Chart elements including the category (X) axis, series (Z) axis, the legend and data markers can also be used as the trigger object of [links](https://devnet.logianalytics.com/hc/en-us/articles/1500010034822-Adding-Links-in-Reports).
