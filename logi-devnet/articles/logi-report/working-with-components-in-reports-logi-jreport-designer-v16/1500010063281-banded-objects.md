# Banded Objects

A banded object is a type of component that can present grouped data and detail data. A banded object is composed of several banded panels with which you can easily organize data fields and other elements.

You can embed additional data components inside a banded object and add data container links between the data components and the banded object.

Banded objects are not supported in library components.

The following topics discuss banded objects in further detail:

* [Inserting Banded Objects in a Report](https://devnet.logianalytics.com/hc/en-us/articles/1500010063301-Inserting-a-Banded-Object)
* [Modifying Banded Objects](https://devnet.logianalytics.com/hc/en-us/articles/1500010063321-Modifying-Banded-Objects)
* [Embedding Data Components in a Banded Object](https://devnet.logianalytics.com/hc/en-us/articles/1500010028642-Embedding-Data-Components-in-a-Banded-Object)
* [Obtaining Detailed Information from a Banded Object](https://devnet.logianalytics.com/hc/en-us/articles/1500010028622-Obtaining-Detailed-Information-from-a-Banded-Object)
* [Setting Up Data Container Links in Banded Objects](https://devnet.logianalytics.com/hc/en-us/articles/1500010028662-Setting-Up-Data-Container-Links-in-Banded-Objects)
