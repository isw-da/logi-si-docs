# Maps

A map is a special kind of image in Logi JReport. Logi JReport now supports two kinds of maps: geographic map and shape map.

* [Geographic map](https://devnet.logianalytics.com/hc/en-us/articles/1500010029122-Geographic-Maps) is a data component integrated with Google Maps, OpenStreetMap or OpenCycleMap. Geographic map features only work in Page Report Studio, Web Report Studio and JDashboard.
* [Shape map](https://devnet.logianalytics.com/hc/en-us/articles/1500010063881-Shape-Maps) contains a picture and certain areas that are bound with groups in a banded object. Shape map features only work when reports run in HTML format or in Page Report Studio.
