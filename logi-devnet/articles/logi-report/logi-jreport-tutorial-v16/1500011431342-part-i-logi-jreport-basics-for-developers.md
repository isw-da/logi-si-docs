# Part I: Logi JReport Basics for Developers

This topic introduces some basic knowledge of Logi JReport Designer for report developers, including the following:

* [Report Development Environment](https://devnet.logianalytics.com/hc/en-us/articles/1500011431302-Report-Development-Environment)  
  A general introduction to the different elements of Logi JReport Designer.
* [Catalogs](https://devnet.logianalytics.com/hc/en-us/articles/1500011463381-Catalogs)  
  This section covers what a catalog is in Logi JReport and why you use catalogs.
* [Connections](https://devnet.logianalytics.com/hc/en-us/articles/1500011463401-Connections)  
  This section covers the data sources that are supported by Logi JReport, and how to connect to different data sources.
* [Queries](https://devnet.logianalytics.com/hc/en-us/articles/1500011463361-Queries)  
  This section shows the usage of queries and how to create them.
* [Business Views](https://devnet.logianalytics.com/hc/en-us/articles/1500011431322-Business-Views)  
  This section shows the usage of business views and how to create them.
* [Page Reports vs. Web Reports vs. Library Components](https://devnet.logianalytics.com/hc/en-us/articles/1500011463341-Page-Reports-vs-Web-Reports-vs-Library-Components)  
  The section covers the differences between different report types, and how to choose the right report type for your data.
