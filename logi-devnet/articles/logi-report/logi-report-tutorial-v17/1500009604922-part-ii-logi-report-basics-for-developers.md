# Part II: Logi Report Basics for Developers

This part introduces some basic knowledge of Logi Report Designer for report developers, including the following:

* [Report Development Environment](https://devnet.logianalytics.com/hc/en-us/articles/1500009628001-Report-Development-Environment)  
  A general introduction to the different elements of Logi Report Designer.
* [Catalogs](https://devnet.logianalytics.com/hc/en-us/articles/1500009604942-Catalogs)  
  This section covers what a catalog is in Logi Report and why you use catalogs.
* [Connections](https://devnet.logianalytics.com/hc/en-us/articles/1500009627981-Connections)  
  This section covers the data sources that are supported by Logi Report, and how to connect to different data sources.
* [Queries](https://devnet.logianalytics.com/hc/en-us/articles/1500009628021-Queries)  
  This section shows the usage of queries and how to create them.
* [Business Views](https://devnet.logianalytics.com/hc/en-us/articles/1500009627961-Business-Views)  
  This section shows the usage of business views and how to create them.
* [Page Reports vs. Web Reports vs. Library Components](https://devnet.logianalytics.com/hc/en-us/articles/1500009604962-Page-Reports-vs-Web-Reports-vs-Library-Components)  
  The section covers the differences between different report types, and how to choose the right report type for your data.
