# Track 5: Publishing, Running, and Administering Resources

In this track it is assumed that you are a server system administrator. You need to receive the reports which are designed in Logi Report Designer from the report developers and publish them to Logi Report Server so the end users can access them and view them.

In this track you perform the following lessons:

* [Lesson 1: Starting Logi Report Server](https://devnet.logianalytics.com/hc/en-us/articles/1500009627641-Lesson-1-Starting-Logi-Report-Server)
* [Lesson 2: Publishing Resources](https://devnet.logianalytics.com/hc/en-us/articles/1500009604522-Lesson-2-Publishing-Resources)
* [Lesson 3: Running Reports](https://devnet.logianalytics.com/hc/en-us/articles/1500009604542-Lesson-3-Running-Reports)
* [Lesson 4: Scheduling Reports](https://devnet.logianalytics.com/hc/en-us/articles/1500009604562-Lesson-4-Scheduling-Reports)
* [Lesson 5: Administration of Logi Report Server](https://devnet.logianalytics.com/hc/en-us/articles/1500009627661-Lesson-5-Administration-of-Logi-Report-Server)
* [Lesson 6: Security](https://devnet.logianalytics.com/hc/en-us/articles/1500009604582-Lesson-6-Security)
