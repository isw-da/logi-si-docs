# Track 5: Publishing, Running, and Administering Resources

In this track, it is assumed that you are a Server system administrator. You need to receive the reports which the report developers designed in Designer and publish them to Server so that the end users can access and view them.

In this track, you perform the following lessons:

* [Lesson 1: Starting Server](https://devnet.logianalytics.com/hc/en-us/articles/4405683487511-Lesson-1-Starting-Server)
* [Lesson 2: Publishing Resources](https://devnet.logianalytics.com/hc/en-us/articles/4405690445079-Lesson-2-Publishing-Resources)
* [Lesson 3: Running Reports](https://devnet.logianalytics.com/hc/en-us/articles/4405683488663-Lesson-3-Running-Reports)
* [Lesson 4: Scheduling Reports](https://devnet.logianalytics.com/hc/en-us/articles/4405683489687-Lesson-4-Scheduling-Reports)
* [Lesson 5: Administration of Server](https://devnet.logianalytics.com/hc/en-us/articles/4405683490711-Lesson-5-Administration-of-Logi-Report-Server)
* [Lesson 6: Security](https://devnet.logianalytics.com/hc/en-us/articles/4405683491735-Lesson-6-Security)
