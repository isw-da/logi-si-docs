# Part II: Logi ReportBasics for Developers

This part introduces some basic knowledge of Logi Report Designer for report developers, including the following:

* [Report Development Environment](https://devnet.logianalytics.com/hc/en-us/articles/4405664340887-Report-Development-Environment)  
  This topic provides a general introduction to the different elements of Designer.
* [Catalogs](https://devnet.logianalytics.com/hc/en-us/articles/4405664339991-Catalogs)  
  This topic describes what a catalog is in Logi Report and why you use catalogs.
* [Connections](https://devnet.logianalytics.com/hc/en-us/articles/4405661290007-Connections)  
  This topic introduces the data sources Logi Report supports, and how you can connect to different data sources in Designer.
* [Queries](https://devnet.logianalytics.com/hc/en-us/articles/4405661291543-Queries)  
  This topic shows the usage of queries and how to create them.
* [Business Views](https://devnet.logianalytics.com/hc/en-us/articles/4405664338967-Business-Views)  
  This topic shows the usage of business views and how to create them.
* [Report Types](https://devnet.logianalytics.com/hc/en-us/articles/4405661292567-Report-Types)  
  This topic introduces the differences between different report types, and how to choose the right report type for your data.
