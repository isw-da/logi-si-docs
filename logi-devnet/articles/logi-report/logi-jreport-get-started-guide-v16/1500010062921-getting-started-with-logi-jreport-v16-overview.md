# Getting Started with Logi JReport v16 Overview

This Get Started guide provides overview material for v16 of the Logi JReport products. Select the links below to learn more.

**[Test Server Installation](https://devnet.logianalytics.com/hc/en-us/articles/1500010028382-Test-Server-Installation "Select to view Test Server Installation")**

After Logi JReport Server is installed, you can start it and log onto the server console to run reports and dashboards.

[**View Sample Reports**](https://devnet.logianalytics.com/hc/en-us/articles/1500010062981-View-Sample-Reports- "Select to view Sample Reports")

On the Logi JReport Server console page, you can view the sample reports Logi JReport provides in different formats.

**[Create Dashboards](https://devnet.logianalytics.com/hc/en-us/articles/1500010062901-Create-Dashboards "Select to view Create Dashboards")**

You can create dashboards using JDashboard, which displays rich visualizations of your data using the vast library of charting elements such as maps, crosstabs, tables and even live motion charts.

**[Create Ad Hoc Reports](https://devnet.logianalytics.com/hc/en-us/articles/1500010028342-Create-Ad-Hoc-Reports "Select to view Create Ad Hoc Reports")**

You can create ad hoc reports based on predefined business views that are published to Logi JReport Server.

[**Start a Visual Analysis Session**](https://devnet.logianalytics.com/hc/en-us/articles/1500010062961-Start-a-Visual-Analysis-Session "Select to view Visual Analysis Session")

You can visualize the result of every step of business data analysis by dragging and dropping business data fields to the layout module using Visual Analysis.

[**Open Sample Report Templates**](https://devnet.logianalytics.com/hc/en-us/articles/1500010062941-Open-Sample-Report-Templates "Select to view Open Sample Report Templates")

There are many sample reports in Logi JReport Designer, you can open any sample report and preview the report result.

**[Start Creating New Reports](https://devnet.logianalytics.com/hc/en-us/articles/1500010028362-Start-Creating-New-Reports "Select to view Start Creating New Reports")**

Logi JReport supports three types of reports, Page Report, Web Report and Library Component. Before you create new reports using Logi JReport Designer, you need to first create a catalog and then create connections to your data sources.
