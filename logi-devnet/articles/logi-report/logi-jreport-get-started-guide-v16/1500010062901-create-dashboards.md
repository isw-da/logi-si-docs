# Create Dashboards

JDashboard is a new way of information delivery that displays rich visualizations of your data using the vast library of charting elements such as maps, crosstabs, tables and even live motion charts. These can all be combined with a robust set of interactions. With JDashboard, you can take data exploration to the next level by creating interactive views of business performance.

To create a dashboard, you should have [logged onto the Logi JReport Server console](https://devnet.logianalytics.com/hc/en-us/articles/1500010028382-Test-Server-Installation). Then:

1. In the Logi JReport Server Start Page, select **Dashboard** in the Create category.

   A blank dashboard is then created.

   ![Blank Dashboard](https://devnet.logianalytics.com/hc/article_attachments/4404901352983/crtdshbrd_hdr.gif)
2. In the Resources panel, navigate to **Component Library** > **Public Components** > **SampleReports**, drag **Product Daily Sales.lc**:

   ![Drag Chart to Dashboard](https://devnet.logianalytics.com/hc/article_attachments/4404901353239/crtdshbrd_drag.gif)

   And then drop it to the dashboard:

   ![Drop Chart in Dashboard](https://devnet.logianalytics.com/hc/article_attachments/4404901353495/crtdshbrd_drop.gif)
3. Close the Resources panel to view the result:

   ![Chart in Dashboard](https://devnet.logianalytics.com/hc/article_attachments/4404901353751/crtdshbrd_rst.gif)
4. Save the dashboard.
