# Test Server Installation

After Logi JReport Server is installed, you can start it and log onto the Logi JReport Server console to run reports and dashboards and create ad hoc reports.

**To start Logi JReport Server:**

On the machine where the server is installed, select **Start Server** in the JReport folder on the Start menu.

**To log onto the server console:**

1. If you are accessing the Logi JReport Server console from the same machine on which Logi JReport Server is installed, after the server is started, a welcome page is automatically displayed in your default web browser, prompting you to provide the login information.

   If you are accessing the server console from a remote computer, open a web browser and set the URL to `http://ip_or_hostname:port`. By default, the port for accessing the Logi JReport Server console is 8888.
2. Type your user name and password as assigned by your administrator.
3. Select **LOGIN** and the Logi JReport Server console home page is displayed, which by default is the Start Page. Choose a topic to start your work on the server.
