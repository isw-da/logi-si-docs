# References Logi JReport Server v16

Logi JReport Server v16 Help System has two different types of references:

* [Dialog Reference](https://devnet.logianalytics.com/hc/en-us/articles/1500009686982-Dialog-Reference)
* [Web Report Object Property Reference](https://devnet.logianalytics.com/hc/en-us/articles/1500009691822-Web-Report-Object-Property-Reference)

The purpose of these references is to make the use of Logi JReport Server much easier and more convenient by providing you with an overview of the functions and properties of objects.
