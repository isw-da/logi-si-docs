# Add Language

The Add Language dialog helps you to add one or more languages, in which reports can be run. This dialog appears when you select the Add button ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4412131380631/btn_add1.gif) in the [NLS Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009714681-NLS-Editor) when editing NLS for a catalog, report or library component, or select OK in the [Select Language Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009689102-Select-Language-Source) dialog with Languages Supported by Logi JReport selected.

![Add Language dialog](https://devnet.logianalytics.com/hc/article_attachments/4412139568663/addlng.gif)

**Available Languages**

Lists all the languages that are available for you to choose. Select the required languages from the box.

**OK**

Adds the selected languages and closes the dialog.

**Cancel**

Cancels the action and exits the dialog.

**Help**

Displays the help document about this feature.
