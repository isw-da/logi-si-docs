# Add Format

The Add Format dialog helps you to add available format keys in the specified catalog/report/library component for [editing NLS](https://devnet.logianalytics.com/hc/en-us/articles/1500009718081-NLS-at-Report-Level#Local). This dialog appears when you select the button ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4412131380631/btn_add1.gif) in the Format tab of the [NLS Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009714681-NLS-Editor#Format).

![Add Format dialog](https://devnet.logianalytics.com/hc/article_attachments/4412112607511/addfmt.gif)

**Available Format**

Lists all available format keys in the specified catalog/report/library component.

**Search box**

Searches for the keys that contain the specified text. After typing in the text, select ![Search button](https://devnet.logianalytics.com/hc/article_attachments/4412131375383/btn_srch.gif) in the box or press Enter on the keyboard to start searching.

**Checkbox**

Specifies whether to add the format keys to edit NLS. Select the checkbox on the column header to select all keys.

**Key**

Lists keys to indicate the objects in the original language. You can select the underlined column header to sort the items by key in an ascending or descending order.

**Format**

Specifies the formats for the keys in the target language. You can select the underlined column header to sort the items by format in an ascending or descending order.

**OK**

Adds the selected keys to the Display tab of the NLS Editor.

**Cancel**

Cancels the action and exits the dialog.

**Reset**

Resets the dialog to its initial status.

**Help**

Displays the help document about this feature.
