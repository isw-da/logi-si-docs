# Select Role, Group and User

The Select Role, Group, and User dialog helps you to select users, groups, and roles in the Logi JReport Server security system to use their e-mail addresses to send the e-mail. It appears when you select ![Select User button](https://devnet.logianalytics.com/hc/article_attachments/4412139564183/btn_chsr5.gif) next to the To/Cc/Bcc text box in the Publish > [To E-mail tab](https://devnet.logianalytics.com/hc/en-us/articles/1500009715061-Schedule#Mail) of the Schedule dialog when creating or editing an e-mail.

![Select Role, Group and User dialog](https://devnet.logianalytics.com/hc/article_attachments/4412131459095/slctusr.gif)

**Search box**

Searches for users, groups and roles in the list below. Type in the keyword and the principals containing the keyword will be listed.

**Principal list**

Specifies the users, groups and roles in the [Logi JReport Server security system](https://devnet.logianalytics.com/hc/en-us/articles/1500009718821-Security-System-Data-Model) whose mail addresses you want to use to send the mail. You can select the top checkbox to select or deselect all the principals.

**Role/Group/User**

Specifies whether to display roles/groups/users in the list above.

**OK**

Applies the selected principals and exits the dialog.

**Cancel**

Cancels the selection and closes the dialog.

**Help**

Displays the help document about this feature.
