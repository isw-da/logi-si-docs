# Enter Values

The Enter Values dialog helps you to [specify multiple values for the parameter](https://devnet.logianalytics.com/hc/en-us/articles/1500009692442-Specifying-Parameter-Values#MultiValue). It appears when you select the button ![Add Values button](https://devnet.logianalytics.com/hc/article_attachments/4412112530711/btn_chsr3.gif) while specifying values for a parameter.

![Enter Values dialog](https://devnet.logianalytics.com/hc/article_attachments/4412112530967/entervlu.gif)

**Available Values**

Lists all predefined parameter values for selection. When the parameter is bound with a column, but the display column is different from the bound column, values of the display column are listed here.

**Selected Values**

Lists the values selected. The selected values are case sensitive.

![Add Values](https://devnet.logianalytics.com/hc/article_attachments/4412112531223/btn_add.gif)

Adds the selected values from the Available Values box to the Selected Values box.

![Remove Values](https://devnet.logianalytics.com/hc/article_attachments/4412112531735/btn_rmv.gif)

Removes the selected values from the Selected Values box.

![Add All Values](https://devnet.logianalytics.com/hc/article_attachments/4412112531479/btn_addall.gif)

Adds all the selected values from the Available Values box to the Selected Values box.

![Remove All Values](https://devnet.logianalytics.com/hc/article_attachments/4412112531991/btn_rmvall.gif)

Removes all the selected values from the Selected Values box.

**Enter Values**

Available when the parameter's Allow Type-in of Value property is true. It allows you to add values for the parameter manually.

To add a value, enter a value manually in the text box and then select the button next to add it to the Selected Values box. When the parameter is bound with a column, but the display column is different from the bound column, make sure the value you enter is that of the bound column.

* ![Calendar button](https://devnet.logianalytics.com/hc/article_attachments/4412112486935/btn_clndr.gif)  
   Opens the calendar to specify a date and time value for the parameter. Available only for Date, DateTime, or Time parameter which allows for type-in values and is not bound with any column.

**Search**

Available when the parameter's Allow Type-in of Value property is false. It is used to search for values among the available values. After you type text in the Search text box, the matched text in the available values will be highlighted.

**All**

Available when the parameter's Enable the "All" Option property is true. It specifies whether to apply all values of the parameter. When the parameter is inserted as a field into a report and All is selected as the value, the field will show the string "All".

**OK**

Applies the selected values to the parameter and closes the dialog.

**Cancel**

Does not retain any changes and closes the dialog.

**Help**

Displays the help document about this feature.
