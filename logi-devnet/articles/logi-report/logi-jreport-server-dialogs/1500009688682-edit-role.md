# Edit Role

The Edit Role dialog helps you to edit a [role](https://devnet.logianalytics.com/hc/en-us/articles/1500009692682-Managing-Roles). It appears when you select a role name link in the Name column of the Administration > Security > Role page in the server console.

![Edit Role dialog](https://devnet.logianalytics.com/hc/article_attachments/4412131468695/edtrole.gif)

**Selected Realm**

Displays the realm in which the selected role is.

**Role Name**

Displays the name of the selected role.

**Privileges**

Specifies whether the selected role has the following privileges:

* **Publish**  
   The privilege of publishing resources to Logi JReport Server.
* **Advanced Properties**  
   The privilege of viewing advanced version properties information, such as catalog connections and report related resources.

**OK**

Creates a role with the information you have specified.

**Cancel**

Cancels operations and closes the dialog.
