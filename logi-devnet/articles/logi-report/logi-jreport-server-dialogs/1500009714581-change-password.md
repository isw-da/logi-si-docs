# Change Password

The Change Password dialog helps you to [change the password](https://devnet.logianalytics.com/hc/en-us/articles/1500009718921-Managing-User-Accounts#Password) of a user. It appears when you select the Change Password link of a user in the Control column of the Administration > Security > User page in the server console.

![Change Password dialog](https://devnet.logianalytics.com/hc/article_attachments/4412131405079/chgpswd.gif)

**Selected Realm**

Displays the realm in which the selected user is.

**User Name**

Displays the name of the selected user.

**System Administrator Password**

Specifies the password of the current logged in user.

**New Password**

Specifies the new password for the user.

**Confirm New Password**

Confirms the password by entering it a second time.

**OK**

Saves the new password.

**Cancel**

Cancels operations and closes the dialog.
