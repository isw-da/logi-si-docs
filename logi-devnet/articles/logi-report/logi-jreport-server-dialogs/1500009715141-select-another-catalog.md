# Select Another Catalog

The Select Another Catalog dialog helps you to specify a catalog for the current report/library component or folder.

![Select Another Catalog dialog](https://devnet.logianalytics.com/hc/article_attachments/4412131460119/slctctlg.gif)

**Select a Folder**

Specifies the folder in the server resource tree that contains the required catalog. You can select the button ![](https://devnet.logianalytics.com/hc/article_attachments/4412139488919/btn_chsr2.gif) to open the [Select Folder](https://devnet.logianalytics.com/hc/en-us/articles/1500009715161-Select-Folder) dialog for selecting the folder.

**Select a Catalog**

Lists the catalogs in the specified folder. Select the required one from the drop-down list.

**OK**

Accepts the settings and closes the dialog.

**Cancel**

Cancels the settings and closes the dialog.

**Help**

Displays the help document about this feature.
