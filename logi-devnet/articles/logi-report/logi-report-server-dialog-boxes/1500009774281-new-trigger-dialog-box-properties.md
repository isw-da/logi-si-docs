# New Trigger Dialog Box Properties

This topic describes how you can use the New Trigger dialog box to create a [trigger](https://devnet.logianalytics.com/hc/en-us/articles/1500009777321-Managing-Triggers). Server displays the dialog box when an admin user clicks New Trigger in the Administration > Other > Triggers page in the Server Console.

![New Trigger dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885409303/nwtrigr.gif)

**Trigger Name**

Specifies the name for the new trigger.

**Conflict Check**

Checks if the name has been used for another trigger.

**Description**

Specifies the description of the trigger.

**OK**

Creates the trigger with the information you have specified.

**Cancel**

Cancels the trigger creating process.
