# Change Password Dialog Box Properties

Use the Change Password dialog box to [change the password](https://devnet.logianalytics.com/hc/en-us/articles/1500009749862-Managing-User-Accounts#Password) of a user. This topic describes how to change the password.

Server displays the dialog box when an admin user selects the Change Password link of a user in the Control column of the Administration > Security > User page in the Server Console.

![Change Password dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885358231/chgpswd.gif)

**Selected Realm**

Realm in which the selected user is.

**User Name**

Name of the selected user.

**System Administrator Password**

Type the password of the current logged in user.

**New Password**

Type the new password for the user.

**Confirm New Password**

Type the password again to confirm it.

**OK**

Select **OK** to save the new password.

**Cancel**

Select **Cancel** to close the dialog box without saving any changes.
