# Add Language Dialog Box Properties

This topic describes how to use the Add Language dialog box to select a language.

Server displays the dialog box when an admin user selects ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404880163351/btn_add1.gif)above the Language box in the [NLS Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009746082-NLS-Editor-Dialog-Box-Properties) when editing NLS for a resource, or select OK in the [Select Language Source](https://devnet.logianalytics.com/hc/en-us/articles/1500009746442-Select-Language-Source-Dialog-Box-Properties) dialog box with **Languages Supported by** Logi Report selected.

![Add Language dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885515031/addlng.gif)

**Available Languages**

Server lists all the available languages for you to choose. Select a language from the box.

**OK**

Select **OK** to add the selected language.

**Cancel**

Select **Cancel** to close the dialog box without saving any changes.

**Help**

Select **Help** to view information about the Add Language dialog box.
