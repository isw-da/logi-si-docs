# Select Another Catalog Dialog Box Properties

This topic describes how you can use the Select Another Catalog dialog box to specify a catalog for the current report/library component or folder.

![Select Another Catalog dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885502999/slctctlg.gif)

**Select a Folder**

Specifies the folder in the server resource tree that contains the required catalog. You can select the button ![](https://devnet.logianalytics.com/hc/article_attachments/4404880145687/btn_chsr2.gif) to open the [Select Folder](https://devnet.logianalytics.com/hc/en-us/articles/1500009746422-Select-Folder-Dialog-Box-Properties) dialog box for selecting the folder.

**Select a Catalog**

Lists the catalogs in the specified folder. Select the required one from the drop-down list.

**OK**

Accepts the settings and closes the dialog box.

**Cancel**

Cancels the settings and closes the dialog box.

**Help**

Displays the help document about this feature.
