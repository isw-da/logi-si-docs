# Add Format Dialog Box Properties

Use the Add Format dialog box to add available format keys in the specified resource for [editing NLS](https://devnet.logianalytics.com/hc/en-us/articles/1500009777481-NLS-at-Report-Level#Local). This topic describes how to translate format keys.

Server displays the dialog box when an admin user selects ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404880163351/btn_add1.gif) in the Format tab of the [NLS Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009746082-NLS-Editor-Dialog-Box-Properties#Format).

![Add Format dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880428439/addfmt.gif)

**Available Format**

Server lists all available format keys in the specified catalog/report/library component.

**Search box**

Search for the keys that contain the specified text. After typing in the text, select ![Search button](https://devnet.logianalytics.com/hc/article_attachments/4404880135831/btn_srch.gif) in the box or press Enter to start searching.

**check box**

Select to add the format keys to edit NLS. Select the check box on the column header to select all keys.

**Key**

Keys to indicate the objects in the original language. You can select the underlined column header to sort the items by key in the ascending or descending order.

**Format**

Type the formats for the keys in the target language. You can select the underlined column header to sort the items by format in the ascending or descending order.

**OK**

Select **OK** to add the selected keys to the Display tab of the NLS Editor.

**Cancel**

Select **Cancel** to close the dialog box without saving any changes.

**Reset**

Select **Reset** to reset the dialog box to its initial status.

**Help**

Select **Help** to view information about the Add Format dialog box.
