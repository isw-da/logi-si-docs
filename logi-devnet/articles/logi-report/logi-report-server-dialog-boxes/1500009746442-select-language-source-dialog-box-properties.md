# Select Language Source Dialog Box Properties

This topic describes how you can use the Select Language Source dialog box to choose where to add new languages.

Server displays the dialog box when an admin user selects the button ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404880163351/btn_add1.gif) above the Language box in the [Global NLS page](https://devnet.logianalytics.com/hc/en-us/articles/1500009777481-NLS-at-Report-Level#Global), [Resource Tree NLS page](https://devnet.logianalytics.com/hc/en-us/articles/1500009777481-NLS-at-Report-Level#Resource), or the [NLS Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009746082-NLS-Editor-Dialog-Box-Properties) when editing NLS for a catalog.

![Select Langugae Source dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880413463/slctlngsc.gif)

**Languages Supported by Logi Report**

Specifies to add new languages from the Available Languages box. After choosing the option and selecting OK, you will be shown the [Add Language](https://devnet.logianalytics.com/hc/en-us/articles/1500009745862-Add-Language-Dialog-Box-Properties) dialog box.

**NLS Resource File**

Specifies to add new languages from NLS resource files in your local folder. After choosing the option and selecting OK, you can browse to the local folder where the NLS resource files are located.

NLS resource files should follow the naming rule: NLS\_[language]\_[region A2]\_[UserDefined].properties. The language argument is a valid ISO Language Code as defined by ISO-639. You can find a full list of these codes at a number of sites, for example: <http://www.loc.gov/standards/iso639-2/php/code_list.php>. The region argument is a valid ISO Country Code as defined by ISO-3166. You can find a full list of these codes at a number of sites, for example: <http://www.chemie.fu-berlin.de/diverse/doc/ISO_3166.html>.

**OK**

Goes to the next action.

**Cancel**

Cancels the action and exits the dialog box.

**Help**

Displays the help document about this feature.
