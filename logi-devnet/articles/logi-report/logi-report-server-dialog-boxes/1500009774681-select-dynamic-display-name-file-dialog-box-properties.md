# Select Dynamic Display Name File Dialog Box Properties

This topic describes how you can use the Select Dynamic Display Name File dialog box to import [dynamic display names](https://devnet.logianalytics.com/hc/en-us/articles/1500009777181-Managing-Dynamic-Display-Names-of-Business-View-Elements) from a local resource bundle file.

Server displays the dialog box when an admin user selects the Import link in the Dynamic Display Names column of the Administration > Other > Dynamic Display Names page in the Server Console.

![Select Dynamic Display Name File dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885502743/slctdynfl.gif)

**Select File**

Specifies the file from which to import the dynamic display names.

Select the **Browse** button to select the local resource bundle file that defines the dynamic display names of the business view elements in a catalog.

**OK**

Imports the dynamic display names from the selected file and exits the dialog box.

**Cancel**

Cancels the importation of dynamic display names and closes the dialog box.

**Help**

Displays the help document about this feature.
