# New Organization Dialog Box Properties

This topic describes how you can use the New Organization dialog box to create an [organization](https://devnet.logianalytics.com/hc/en-us/articles/1500009749922-Multitenancy-Supported-via-Organizations). Server displays the dialog box when an admin user selects the New Organization link in the Administration > Security > Organization page in the Server Console.

![New Organization dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885354391/nwognz.gif)

The options available in this dialog box are as follows:

**Organization Name**

Specifies the name of the organization. The system wide organization name is *System*; therefore, it is reserved. The organization names should be distinct. A valid name can contain only '0' to '9', 'a' to 'z', 'A' to 'Z', '&', '-', '\_', '.' and blank and no more than 32 characters.

**Max Number of Users**

Specifies the maximum number of users allowed in the organization. The default value *unlimited* means that the number is not limited. You can select a value from the drop-down list or type an integer number in the combo box directly.

**Description**

Provides information about the organization.

**OK**

Creates the organization and exits the dialog box.

**Cancel**

Does not create an organization and closes the dialog box.

**Help**

Displays the help document about this feature.
