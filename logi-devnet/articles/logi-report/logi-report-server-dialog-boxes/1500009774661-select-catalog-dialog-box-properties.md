# Select Catalog Dialog Box Properties

This topic describes how you can use the Select Catalog dialog box to select a catalog in the server resource tree.

![Select Catalog dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885503895/slctcat.gif)

**Resource tree box**

The box lists the folder and catalog resources in the server resource tree. Navigate to select the required catalog.

**OK**

Selects the catalog and exits the dialog box.

**Cancel**

Cancels the selection of a catalog and closes the dialog box.

**Help**

Displays the help document about this feature.
