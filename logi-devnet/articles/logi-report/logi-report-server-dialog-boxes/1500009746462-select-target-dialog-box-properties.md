# Select Target Dialog Box Properties

This topic describes how you can use the Select Target dialog box to specify the target resources that the implementation of the [Dashboard Listener API](https://devnet.logianalytics.com/hc/en-us/articles/1500009771021-Applying-the-Implementations-of-the-Dashboard-Listener-API) will apply to.

Server displays the dialog box when an admin user selects ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404880163351/btn_add1.gif) in the [New Dashboard Listener](https://devnet.logianalytics.com/hc/en-us/articles/1500009774081-New-Dashboard-Listener-Dialog-Box-Properties#Resource) dialog box.

![Select Target dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880412823/slcttgt.gif)

**All Dashboards**

If selected, the implementation will apply to all dashboards including the newly created ones that have not been saved yet.

**Customize**

If selected, you can specify the dashboards you want the implementation to apply to.

* **Newly Created Dashboards**  
   If selected, the implementation will apply to all newly created dashboards that have not been saved yet.
* **Existing Dashboards**  
   If selected, you can further specify one or more folders, and the implementation will apply to the dashboards in the selected folders.

**OK**

Applies the setting and exits the dialog box.

**Cancel**

Cancels the setting and closes the dialog box.

**Help**

Displays the help document about this feature.
