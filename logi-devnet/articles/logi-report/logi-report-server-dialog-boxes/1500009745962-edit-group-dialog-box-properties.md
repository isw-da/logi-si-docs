# Edit Group Dialog Box Properties

This topic describes how you can use the Edit Group dialog box to edit a [group](https://devnet.logianalytics.com/hc/en-us/articles/1500009778621-Managing-Groups). Server displays the dialog box when an admin user selects a group name link in the Name column of the Administration > Security > Group page in the Server Console.

![Edit Group dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880426647/edtgrp.gif)

**Selected Realm**

Displays the realm in which the selected group is.

**Group Name**

Displays the name of the selected group.

**Privileges**

Specifies whether the selected group has the following privileges:

* **Publish**  
   The privilege of publishing resources to Logi Report Server.
* **Advanced Properties**  
   The privilege of viewing advanced version properties information, such as catalog connections and report related resources.

**OK**

Saves the group with the information you have specified.

**Cancel**

Cancels operations and closes the dialog box.
