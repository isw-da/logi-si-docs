# Sharing Properties

This topic describes how you can use the Sharing Properties dialog box to [set the properties for sharing a report](https://devnet.logianalytics.com/hc/en-us/articles/1500009777281-Sharing-Web-Reports#Manage).

![Sharing Properties dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885411095/shrprpty.gif)

**Original Resource**

Displays the resource name and path of the original report.

**Original Resource Type**

Displays the resource type of the original report.

**Original Resource Description**

Displays the description about the original report.

**[Custom Field Name]**

Specifies value of the custom field for the report. A [custom field](https://devnet.logianalytics.com/hc/en-us/articles/1500009748662-Working-with-Custom-Fields) can be regarded as a resource property and is available when it is enabled.

**Shared Resource Name**

Displays the resource name of the shared report.

**Shared To**

Displays the resource name and path of the shared report. You can select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404885411479/btn_chsr4.gif) and then select a different folder in the server resource tree to change the location of the shared report in the [Select Folder](https://devnet.logianalytics.com/hc/en-us/articles/1500009746422-Select-Folder-Dialog-Box-Properties) dialog box.

**Shared Resource Description**

Provides a brief description about the shared report.

**Shared Time**

Displays the time when the shared report was created.

**Allow Edit**

Specifies whether to allow other users to edit the shared report.

**OK**

Applies the settings and exits the dialog box.

**Cancel**

Cancels any settings and closes the dialog box.

**Reset**

Discards your modifications and restores the dialog box to its default status.

**Help**

Displays the help document about this feature.
