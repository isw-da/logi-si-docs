# Add Display Dialog Box Properties

Use the Add Display dialog box to add available display text keys in the specified resource for [editing NLS](https://devnet.logianalytics.com/hc/en-us/articles/1500009777481-NLS-at-Report-Level#Local). This topic describes how to translate display text keys.

Server displays the dialog box when an admin user selects ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404880163351/btn_add1.gif) in the Display tab of the [NLS Editor](https://devnet.logianalytics.com/hc/en-us/articles/1500009746082-NLS-Editor-Dialog-Box-Properties#Display).

![Add Display dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885515671/adddsply.gif)

**Available Display**

Server lists all the available display text keys in the specified catalog/report/library component.

**Search box**

Search for the keys that contain the specified text. After typing the text, select ![Search button](https://devnet.logianalytics.com/hc/article_attachments/4404880135831/btn_srch.gif) in the box or press Enter to start searching.

**Check box**

Select to add the display text keys to edit NLS. Select the check box on the column header to select all keys.

**Type**

Types of display text for different objects. You can select the underlined column header to sort the items by type in an ascending or descending order.

* **Column**  
   This type is only for reports running in Page Report Studio. It is the type of display text of columns.
* **DisplayName**  
   Type of display text of object display name.
* **Label**  
   Type of display text of label, some web controls, and UDOs.
* **Metadata**  
   Type of display text of metadata. Metadata mainly refers to catalog resources, such as table/view columns, business views, formulas, summaries, and parameters.
* **Prompt**  
   Type of display text of parameter prompt value.
* **Title**  
   Type of display text of filter control and library component.
* **TOC**  
   Type of display text in the TOC tree.

**Key**

Keys to indicate the objects in the original language. You can select the underlined column header to sort the items by key in the ascending or descending order.

**Translate**

Translate the keys in the target language. You can select the underlined column header to sort the items by translation in the ascending or descending order.

**OK**

Select **OK** to add the selected keys to the Display tab of the NLS Editor.

**Cancel**

Select **Cancel** to close the dialog box without saving any changes.

**Reset**

Select **Reset** to reset the dialog box to its initial status.

**Help**

Select **Help** to view information about the Add Display dialog box.
