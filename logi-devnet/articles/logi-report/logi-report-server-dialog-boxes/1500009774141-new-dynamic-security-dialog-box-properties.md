# New Dynamic Security Dialog Box Properties

This topic describes how you can use the New Dynamic Security dialog box to create [dynamic catalog security](https://devnet.logianalytics.com/hc/en-us/articles/1500009778541-Dynamic-Security). Server displays the dialog box when an admin user selects the New Dynamic Security link in the Administration > Security > Dynamic Security page in the Server Console.

![New Dynamic Security dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880423703/nwdynscrty.gif)

**Catalog**

Specifies the catalog you want to create dynamic security for.

Type the catalog with the full resource path in the text box, for example, `/SampleReports/SampleReports.cat`, or select the Browse button to select a catalog in the [Select Catalog](https://devnet.logianalytics.com/hc/en-us/articles/1500009774661-Select-Catalog-Dialog-Box-Properties) dialog box.

**Security File**

Specifies the security file that is generated in Logi Report Designer and used to create dynamic security for the catalog. Select the Browse button to select a catalog security file that defines business view security, record level security and column level security in the catalog.

**OK**

Creates the dynamic security and exits the dialog box.

**Cancel**

Cancels the creation of the dynamic security and closes the dialog box.

**Help**

Displays the help document about this feature.
