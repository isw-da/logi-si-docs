# Preference Dialog Box Properties

This topic describes how you can use the Preference dialog box to [set preferences](https://devnet.logianalytics.com/hc/en-us/articles/1500009749862-Managing-User-Accounts#Preference) for the specified user. Server displays the dialog box when an admin user selects the Preference link of a user in the Control column of the Administration > Security > User page in the Server Console.

![Preference dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880216471/prfrnc.gif)

It contains two subjects: Customize Server Preferences and Customize Profile. You can specify the [server preferences](https://devnet.logianalytics.com/hc/en-us/articles/1500009743782-Configuring-Server-Profile) and [Page Report Studio preferences](https://devnet.logianalytics.com/hc/en-us/articles/1500009777681-Customizing-Page-Report-Studio-Profile) for the user accordingly.
