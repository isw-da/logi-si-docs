# Select Folder Dialog Box Properties

This topic describes how you can use the Select Folder dialog box to select a folder in the server resource tree that contains the required resources or that you want to save the resources.

![Select Folder dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880413847/slctfldr.gif)

**Resource tree box**

The box lists the folder resources in the server resource tree that are available to you in the current scenario. Navigate to the required folder to select it.

**OK**

Selects the folder and closes the dialog box.

**Cancel**

Cancels the selection of a folder and closes the dialog box.

**Help**

Displays the help document about this feature.
