# Using Visual Analysis

Logi Report Visual Analysis is an in-context analysis tool to visualize the result of every step of your work. You can easily transform data into rich visualizations simply by dragging and dropping data fields onto a layout module. You can also view and analyze data from different perspectives and discover more actionable intelligence. This topic describes the main features, the data sources which are business views, and the license of Visual Analysis.

Visual Analysis has the following main features:

* Using groups & aggregations to quickly highlight colors, sizes & shapes
* Instantly change data visualizations with over 50 options
* Powerful [in-memory cube](https://devnet.logianalytics.com/hc/en-us/articles/1500009750021-Managing-In-memory-Cubes) technology

[Business views](https://devnet.logianalytics.com/hc/en-us/articles/1500009751441-An-Introduction-to-Business-Views) are the data sources used in Visual Analysis. Each time you can perform data analysis based on one business view, and later you can save the result as an analysis template into the server resource tree. However, Visual Analysis does not support the customization of parameter values, so if a business view contains parameters, the default parameter values will be applied when you perform visual analysis on the business view.

Visual Analysis is a separately licensed feature of Logi Report Server. It is installed together with Logi Report Server so only the license key needs to be updated to enable it to run. To find out how to license Visual Analysis, contact Logi Analytics sales at [salesteam@logianalytics.com](mailto:salesteam@logianalytics.com) or contact your Enterprise Account Manager.
