# Split Cell Dialog Box Properties

This topic describes how you can use the Split Cell dialog box to specify the number of rows and columns that you want to split a tabular cell into. Server displays the dialog box when you select a tabular cell and select Menu > Format > Split.

![Split Cell dialog](https://devnet.logianalytics.com/hc/article_attachments/9905678984471/splitcel.gif)

**Split Cell Into**

Specify the number of rows and columns you want to split the selected tabular cell into.

* **Number of Rows**  
   Specify the number of rows you want to split the tabular cell into. Type a positive integer in the text box to specify the number.
* **Number of Columns**  
   Specify the number of columns you want to split the tabular cell into. Type a positive integer in the text box to specify the number.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
