# Aggregate On Dialog Box Properties

You can use the Aggregate On dialog box to [calculate data in the selected table detail column](https://devnet.logianalytics.com/hc/en-us/articles/5741475063447-Manipulating-Data-Components-in-Page-Report#AggregateOn). This topic describes the properties in the dialog box.

![Aggregate On dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905661271063/agrgton.gif)

**Aggregate On**

Server displays the field that is bound with the detail column.

**Function**

Select the function you use to compute the field in the detail column.

**Distinct On**

Server enables this property when you have selected **DistinctSum** as the aggregate function. Select the ellipsis button ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/9905577763991/btn_chsr2.gif) to open the [Select Fields dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741440574103-Select-Fields-Dialog-Box-Properties). Then, select the required fields according to whose unique values you want to calculate the DistinctSum function.

**OK**

Select to summarize the detail column data using the specified function and exit the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
