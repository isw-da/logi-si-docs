# Fetch Data Dialog Box Properties

You can use the Fetch Data dialog box to select field values from the database for using in a [filter control](https://devnet.logianalytics.com/hc/en-us/articles/5741483099159-Applying-Web-Controls-in-Page-Report#Filter). This topic describes the properties in the dialog box.

![Fetch Data dialog](https://devnet.logianalytics.com/hc/article_attachments/9905763154071/ftchdt.gif)

**Search box**

Type the keyword and Server lists the values containing the keyword.

**Values**

Select the values you want. Or select the top checkbox to select all the listed values. You can sort the listed values in the ascending or descending order by selecting the Values column header.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
