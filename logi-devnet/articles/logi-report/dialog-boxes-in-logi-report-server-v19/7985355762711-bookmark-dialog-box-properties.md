# Bookmark Dialog Box Properties

This topic describes how you can use the Bookmark dialog box to bookmark the current web report or update a bookmark.

Server displays the dialog box when you select the **Save Bookmark** button ![Save Bookmark button](https://devnet.logianalytics.com/hc/article_attachments/9905574599191/btn_wbrpt_svbkmk.gif) on the toolbar in Web Report Studio or **Details** in the [Bookmark Setting dialog box](https://devnet.logianalytics.com/hc/en-us/articles/7985324116503-Bookmark-Setting-Dialog-Box-Properties).

![Bookmark dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905612658583/bkmk.gif)

**Parameter**

Parameters that the report uses.

**Dataset Filter**

Filters that you created via the [Edit Dataset Filter dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741470839191-Applying-Filters-in-Web-Report#BV).

**Browser Filter**

Filters that you created via the [Filter dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741470839191-Applying-Filters-in-Web-Report#Dialog).

**Drill Filter**

Go-to filters that you created when performing the [go-to-by-value action](https://devnet.logianalytics.com/hc/en-us/articles/5741476511127-Going-Through-Web-Report-Data#Go-to-by-value) on the report.

**On-screen Filter**

Filters that you created via the [Filter panel](https://devnet.logianalytics.com/hc/en-us/articles/5741470839191-Applying-Filters-in-Web-Report#Panel) and [filter controls](https://devnet.logianalytics.com/hc/en-us/articles/5741484852887-Using-Web-Controls-in-Web-Report#Filter).

**Name**

Specify the name for the bookmark.

**Description**

Provide a description for the bookmark.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
