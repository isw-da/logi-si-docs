# Edit Shapes Dialog Box Properties

This topic describes how you can use the Edit Shapes dialog box to specify shapes for the members of the shape legend field.

![Edit Shapes dialog](https://devnet.logianalytics.com/hc/article_attachments/9905735211671/edtshp.gif)

**Pattern**

Select a pattern from the drop-down list. Server displays a preview of the shapes of the pattern under the pattern name. Changing a pattern will apply the shapes in the new pattern to all the listed field members.

**Data Items**

Server lists the members of the field with the shapes in the selected pattern.

To customize the shape of a data item, select the data item and then select a shape in the pattern preview box.

**OK**

Select to apply any changes you made here and exit the dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905756962455/btn_va_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905578411927/btn_delete.gif)**Close button**

Select to close the dialog box without saving any changes.
