# Insert Column Dialog Box Properties

This topic describes how you can use the Insert Column dialog box to [insert the selected group field as a column header](https://devnet.logianalytics.com/hc/en-us/articles/5741456063639-Manipulating-Data-Components-in-Web-Report#CTColumn) in a crosstab.

![Insert Column dialog](https://devnet.logianalytics.com/hc/article_attachments/9905630430103/insrtclmn.gif)

**Column**

Name of the group field that you want to use as a column header in the crosstab. It is read only.

**Label**

Type the label text of the new column header in the crosstab. You can leave it blank.

**Auto Map Field Name**

Select to automatically map the label text of the new column header to the [dynamic display name](https://devnet.logianalytics.com/hc/en-us/articles/5741468400151-Managing-Dynamic-Display-Names-of-Business-View-Elements) of the group field.

**OK**

Select to insert the group field as a column header into the crosstab and close the dialog box.

**Cancel**

Select to close the dialog box without inserting a column header.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without inserting a column header.
