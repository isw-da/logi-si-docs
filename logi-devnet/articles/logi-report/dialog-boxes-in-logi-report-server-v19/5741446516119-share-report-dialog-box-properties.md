# Share Report Dialog Box Properties

This topic describes how you can use the Share Report dialog box to [share the current web report](https://devnet.logianalytics.com/hc/en-us/articles/5741476487063-General-Operations-in-Web-Report#Share). Server displays the dialog box when you select Menu > File > Share or select the Share button ![Share button](https://devnet.logianalytics.com/hc/article_attachments/9905616075543/btn_share.gif) on the toolbar.

![Share Report dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905679221015/shrpt.gif)

**Select Folder**

Select a folder in the server resource tree to share the report to.

**File Name**

Specify the file name of the shared report, with the suffix .wls.

**Description**

Provide a brief description about the shared report.

**Allow Edit**

Select if you want to allow other users to edit the shared report. If you don't select this option, other users cannot [run the shared report using the Edit option](https://devnet.logianalytics.com/hc/en-us/articles/5741439314455-Running-Reports#Edit).

**OK**

Select to save the shared report in the specified folder.

**Cancel**

Select to close the dialog box without sharing the report.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without sharing the report.
