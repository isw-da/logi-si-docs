# Format Platform Dialog Box Properties

This topic describes how you can use the Format Platform dialog box to format the platform of a chart. Server displays the dialog box when you right-click a chart and select **Format Platform** from the shortcut menu.

This topic contains the following sections:

* [General Tab Properties](#General)
* [Border Tab Properties](#Border)
* [Data Tab Properties](#Data)

You see these elements on all the tabs:

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.

## General Tab Properties

Specify the color schema to fill the platform.

![Format Platform dialog box - General tab](https://devnet.logianalytics.com/hc/article_attachments/9905731674519/fmtpltfrm_gnrl.gif)

**Color**

Specify the color with which you want to fill the platform.

To change the color, select the color indicator. Server displays the color palette. Select a color, or select **More Colors** to access the [Color Picker dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741428592023-Color-Picker-Dialog-Box-Properties) in which you can specify a color within a wider range, or select **More Fill Effects** to access the [Fill Effects dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741444679959-Fill-Effects-Dialog-Box-Properties) to specify a gradient or an image as the fill effect.

**Transparency**

Specify the transparency of the color.

## Border Tab Properties

Specify the border properties of the chart platform.

![Format Platform dialog box - Border tab](https://devnet.logianalytics.com/hc/article_attachments/9905731695767/fmtpltfrm_border.gif)

**Border Type**

Specify the type for the border of the platform.

* **None**  
   Select if you don't want border lines.
* **Raised**  
   Select if you want the border to show a 3D effect as if the border raises off the page.
* **Recess**  
   Select if you want the border to show a 3D effect as if the border presses into the page.
* **Shadow**  
   Select if you want the border to show a shadow effect beneath and to the right of the platform.
* **Solid**  
   Select if you want the border to use single lines.

**Color**

Specify the color for the border of the platform.

**Transparency**

Specify the transparency for the color of the border.

**Line Style**

Specify the line style for the border of the platform.

**Thickness**

Specify the thickness of the border.

**End Caps**

Specify the ending style of the border line.

* **Butt**  
   Select to end unclosed subpaths and dash segments with no added decoration.
* **Round**  
   Select to end unclosed subpaths and dash segments with a round decoration that has a radius equal to half of the width of the pen.
* **Square**  
   Select to end unclosed subpaths and dash segments with a square projection that extends beyond the end of the segment to a distance equal to half of the line width.

**Line Joint**

Specify the line joint style for the border line.

* **Miter**  
   Select if you want to join path segments by extending their outside edges until they meet.
* **Round**  
   Select if you want to join path segments by rounding off the corner at a radius of half the line width.
* **Bevel**  
   Select if you want to join path segments by connecting the outer corners of their wide outlines with a straight segment.
* **Joint Round**  
   Select if you want to join path segments by rounding off the corner at a specified radius.

**Radius**

Specify the radius for the border joint of the platform border line. Available only when Line Joint is Joint Round.

**Dash**

Specify the dash size of the border line.

* **Auto Adjust Dash**  
   Select if you want to adjust the dash size automatically.
* **Fixed Dash Size**  
   Select if you want to use fixed dash size.

## Data Tab Properties

Available to heat maps only. In this tab, you can specify whether to calculate the size-by/color-by summary based on logarithm functions.

![Format Platform dialog - Data tab](https://devnet.logianalytics.com/hc/article_attachments/9905719938199/fmtpltfrm_dt.gif)

**Size by Logarithm**

Select if you want to calculate the size-by summary based on logarithm function.

* **None**  
   Select if you want to return the real value of the field instead of any logarithmic value.
* **log x**  
   Select if you want to get the logarithm of the field with base 10.
* **ln x**  
   Select if you want to get the logarithm of the field with base e.

**Color by Logarithm**

Select if you want to calculate the color-by summary based on logarithm function.

* **None**  
   Select if you want to return the real value of the field instead of any logarithmic value.
* **log x**  
   Select if you want to get the logarithm of the field with base 10.
* **ln x**  
   Select if you want to get the logarithm of the field with base e.
