# Insert Row Dialog Box Properties

You can use the Insert Row dialog box to [insert the selected group field as a row header](https://devnet.logianalytics.com/hc/en-us/articles/5741475063447-Manipulating-Data-Components-in-Page-Report#CTRow) in a crosstab. This topic describes the properties in the dialog box.

![Insert Row dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905661091351/insrtrw.gif)

**Row**

Server displays the name of the field. It is read only.

**Label**

Specify the display name of the field, which will label the field in the crosstab. You can leave it blank.

**OK**

Select to insert the field in the crosstab.

**Cancel**

Select to close the dialog box without the insertion.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)

Select to close the dialog box without the insertion.
