# Web Action - On-screen Filter Dialog Box Properties

Use the Web Action - On-screen Filter dialog box to edit the filter condition to filter a data component when it [receives a synchronization message.](https://devnet.logianalytics.com/hc/en-us/articles/5741401855639-Synchronizing-the-Data-Components-in-a-Dashboard#Receive) This topic describes how to define the filter condition.

Server displays the dialog box when you select the web action **\*On-screen Filter** and select the Edit button ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/9905758829591/btn_dshbrd_edt1.gif) in the **Actions** column of the [Receive Sync dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741426371607-Receive-Sync-Dialog-Box-Properties).

![Web Action - On-screen Filter dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905758979351/wbactn_onscrnfltr.gif)

![Add button](https://devnet.logianalytics.com/hc/article_attachments/9905736979863/btn_dshbrd_add1.gif)**Add button**

Select to add a new filter condition.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/9905758900887/btn_dshbrd_rmv1.gif)**Remove button**

Select to remove the selected filter condition.

**Filter On**

Select the field you want to filter.

**Value**

Select the value of the field.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
