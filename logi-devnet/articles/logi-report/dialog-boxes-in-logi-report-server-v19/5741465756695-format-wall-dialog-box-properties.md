# Format Wall Dialog Box Properties

This topic describes how you can use the Format Wall dialog box to format the wall of a chart. Server displays the dialog box when you right-click a chart and select **Format Walls** > **Format Wall** from the shortcut menu.

This topic contains the following sections:

* [General Tab Properties](#General)
* [Border Tab Properties](#Border)

You see these elements on both tabs:

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.

## General Tab Properties

Specify the general properties of the wall.

![Format Wall dialog box - General tab](https://devnet.logianalytics.com/hc/article_attachments/9905718894615/fmtwall_gnrl.gif)

**Wall**

Specify the color or fill effect of the wall of the chart.

* **Fill Type**  
   Specify the fill type of the wall. It can be one of the following: None, Color, Texture, Gradient, and Image. If you select Gradient or Image, you can specify the gradient or image by the property [Fill Type](https://devnet.logianalytics.com/hc/en-us/articles/5741484398615-Chart-Paper-Properties#FillType2) in the Wall category of the chart paper in the Inspector.
* **Color**  
   Specify the color of the wall. It takes effect only when Fill Type in this category is Color.
* **Transparency**  
   Specify the transparency of the color schema that you want to apply to the wall of the chart.

**Bound**

Specify the fill color or effect to compound with the basic fill that you set in the Wall box.

* **Fill Type**  
  Specify the fill type of the wall. It can be one of the following: None, Color, Texture, and Gradient. If you select Gradient, you can specify the gradient by the property [Bound Fill Type](https://devnet.logianalytics.com/hc/en-us/articles/5741484398615-Chart-Paper-Properties#BoundFillType) in the Wall category of the chart paper in the Inspector.
* **Color**  
   Specify the fill color schema or effect to compound with the basic fill that you set for the wall of a chart in the Wall box. It takes effect only when Fill Type in this category is Color.
* **Transparency**  
   Specify the transparency of the color schema.

## Border Tab Properties

Specify the border properties of the wall.

![Format Wall dialog box - Border tab](https://devnet.logianalytics.com/hc/article_attachments/9905730612247/fmtwall_border.gif)

**Color**

Specify the color schema for the border of the wall.

**Transparency**

Specify the transparency of the color schema that you want to apply to the border of the wall.

**Thickness**

Specify the thickness for the border of the wall.

**Line Style**

Specify the line style for the border of the wall.
