# Select Role, Group and User Dialog Box Properties

This topic describes how you can use the Select Role, Group and User dialog box to select the scheduling recipients to use their email addresses to send the email.

Server displays the dialog box when you select the ellipsis button ![Select User button](https://devnet.logianalytics.com/hc/article_attachments/9905659094679/btn_chsr5.gif) next to the To/Cc/Bcc text box in the Publish > [To E-mail tab](https://devnet.logianalytics.com/hc/en-us/articles/5741433620631-Schedule-Dialog-Box-Properties#Mail) of the Schedule dialog box when creating or editing an email.

![Select Role, Group and User dialog](https://devnet.logianalytics.com/hc/article_attachments/9905757415447/slctusr.gif)

**Search box**

Search for scheduling recipients in the list. Type the keyword, and Server lists the recipients containing the keyword.

**Scheduling recipient list**

Specify the scheduling recipients, meaning the users, groups, and roles in the Logi Report Server security system, whose email addresses you want to use to send the email. Select the top checkbox to select or clear all the recipients.

Server adopts permission control on the scheduling recipients each user can access. For more information, see [Managing User Accounts](https://devnet.logianalytics.com/hc/en-us/articles/5741463728407-Managing-User-Accounts#Recipient).

**Role/Group/User**

Select to display roles/groups/users in the list.

**OK**

Select to apply the selected principals and exit the dialog box.

**Cancel**

Select to close the dialog box without selecting principals.

**Help**

Select to view information about the dialog box.
