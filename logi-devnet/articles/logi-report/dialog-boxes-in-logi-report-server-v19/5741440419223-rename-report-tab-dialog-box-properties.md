# Rename Report Tab Dialog Box Properties

You can use the Rename Report Tab dialog box to rename the current report tab. This topic describes the properties in the dialog box.

![Rename Report Tab dialog](https://devnet.logianalytics.com/hc/article_attachments/9905774012695/renmrpt.gif)

**Input New Name**

Type the new name for the report tab.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
