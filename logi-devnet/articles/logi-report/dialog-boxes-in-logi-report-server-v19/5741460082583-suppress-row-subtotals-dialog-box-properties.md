# Suppress Row Subtotals Dialog Box Properties

This topic describes how you can use the Suppress Row Subtotals dialog box to customize the subtotals of the row fields in a crosstab that you want to suppress and display.

Server displays the dialog box when you select the ellipsis button ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/9905675890327/btn_chsr4.gif) next to **Suppress Row Subtotals** in the **Layout** tab of the [Crosstab Wizard](https://devnet.logianalytics.com/hc/en-us/articles/5741428630807-Crosstab-Wizard-Properties#Layout) or [Insert Crosstab dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741445839383-Insert-Crosstab-Dialog-Box-Properties#Layout), or in the **Layout** tab for crosstab in the [Web Report Wizard](https://devnet.logianalytics.com/hc/en-us/articles/5741437169815-Web-Report-Wizard-Properties#Crosstab).

![Suppress Row Subtotal dialog](https://devnet.logianalytics.com/hc/article_attachments/9905717506071/sprsrowsbtl.gif)

**Same for all**

Select if you want to apply the setting for the subtotal of the first row field to all the subtotals of the row fields in the crosstab.

**Name**

Server lists the row fields excluding the one on the topmost row group level.

**Suppress**

Select **true** to suppress the subtotal for a row field.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
