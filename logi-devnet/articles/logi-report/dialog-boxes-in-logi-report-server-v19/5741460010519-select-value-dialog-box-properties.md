# Select Value Dialog Box Properties

This topic describes how you can use the Select Value dialog box to select a filter object like filter control to pass values between the filter objects in the primary report and the linked report.

Server displays the dialog box when you select the ellipsis button ![Select Value](https://devnet.logianalytics.com/hc/article_attachments/9905630122007/btn_chsr.gif) in the **Advanced** tab for the **Report** link type in the [Insert Link dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741459269271-Insert-Link-Dialog-Box-Properties) or [Edit Link dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741434764823-Edit-Link-Dialog-Box-Properties).

![Select Value dialog](https://devnet.logianalytics.com/hc/article_attachments/9905679018263/slctvlu.gif)

The resources box lists all the objects in the report. Only filter objects are available for selection. Select a filter object.

**OK**

Select to apply the filter object you selected and close the dialog box.

**Cancel**

Select to close the dialog box without selecting a filter object.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without selecting a filter object.
