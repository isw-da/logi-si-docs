# Insert HTML Dialog Box Properties

Use the Insert HTML dialog box to [insert an HTML component](https://devnet.logianalytics.com/hc/en-us/articles/5741409582359-Creating-and-Saving-Dashboards#HTML) into the dashboard body. This topic describes how to create an HTML component.

Server displays the dialog box when you drag **HTML** from Toolbox in the Resources panel to the dashboard body.

![Insert HTML dialog](https://devnet.logianalytics.com/hc/article_attachments/9905760076567/insrthtml.gif)

**Title**

Specify the title for the HTML component.

**HTML text box**

Type text in the box directly. You can make use of the buttons above the text box to format the text, insert images, and create hyperlinks.

You see these buttons:

* ![Font Face](https://devnet.logianalytics.com/hc/article_attachments/9905771775255/btn_dshbrd_fontfc.gif)**Font Face**  
   Select the font of the text from the drop-down list.
* ![Font Size](https://devnet.logianalytics.com/hc/article_attachments/9905771800855/btn_dshbrd_fontsz.gif)**Font Size**  
   Select the font size of the text from the drop-down list.
* ![Bold](https://devnet.logianalytics.com/hc/article_attachments/9905672154647/btn_bold.gif)**Bold button**  
   Select if you want to make the text bold.
* ![Italic](https://devnet.logianalytics.com/hc/article_attachments/9905672184599/btn_italic.gif)**Italic button**  
   Select if you want to make the text italic.
* **![Underline](https://devnet.logianalytics.com/hc/article_attachments/9905672206743/btn_underline.gif)****Underline button**  
   Select if you want to underline the text.
* ![Font Color](https://devnet.logianalytics.com/hc/article_attachments/9905760161431/btn_dshbrd_fontclr.gif)**Font Color button**  
   Select the color of the text.

  To change the color, select the color indicator. Then, select a color, or select **More Colors** to access the [Color Picker dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741404530839-Color-Picker-Dialog-Box-Properties) in which you can select a color within a wider range.
* ![Background Color](https://devnet.logianalytics.com/hc/article_attachments/9905760177815/btn_dshbrd_bkgrndclr.gif)**Background Color button**  
   Select the background color of the text.
* ![Align Left](https://devnet.logianalytics.com/hc/article_attachments/9905672228247/btn_alignleft.gif)**Align Left button**  
   Select if you want to align the text to the left.
* ![Align Center](https://devnet.logianalytics.com/hc/article_attachments/9905660223895/btn_aligncntr.gif)**Align Center button**  
   Select if you want to align the text in the center.
* ![Align Right](https://devnet.logianalytics.com/hc/article_attachments/9905660238999/btn_alignright.gif)**Align Right button**  
   Select if you want to align the text to the right.
* ![Insert Image](https://devnet.logianalytics.com/hc/article_attachments/9905760016663/btn_dshbrd_img.gif)**Insert Image button**  
   Select to open the [Insert Image dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741432195351-Insert-Image-Dialog-Box-Properties) for inserting an image into the HTML component.
* ![Hyperlink](https://devnet.logianalytics.com/hc/article_attachments/9905760207127/btn_dshbrd_lnk.gif)**Hyperlink button**  
   Select to open the [Hyperlink dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741412887831-Hyperlink-Dialog-Box-Properties) to create a hyperlink on the selected object.

**OK**

Select to insert the HTML component in the dashboard body.

**Cancel**

Select to close the dialog box without the insertion.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without the insertion.
