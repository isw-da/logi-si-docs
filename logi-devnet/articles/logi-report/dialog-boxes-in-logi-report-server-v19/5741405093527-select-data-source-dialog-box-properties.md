# Select Data Source Dialog Box Properties

Use the Select Data Source dialog box to select the data source that contains the parameter you want to [insert into a URL](https://devnet.logianalytics.com/hc/en-us/articles/5741409582359-Creating-and-Saving-Dashboards#URL). This topic describes how to select a data source.

Server displays the dialog box when you select **Add Parameter** in the [Insert URL Frame dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741419705239-Insert-URL-Frame-Dialog-Box-Properties) or [Edit URL Frame dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741426072471-Edit-URL-Frame-Dialog-Box-Properties) if no data source has been specified, or select **Change Data Source** in the [Select Parameter dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741432374807-Select-Parameter-Dialog-Box-Properties).

![Select Data Source dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905759470231/slctdtsrc.gif)

**Folder**

Select the folder that contains the data source you want.

**Catalog**

Select the catalog that contains
the data source you want.

**Data Source**

Select the data source.

**OK**

Select to apply the data source and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif) **Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif) **Close button**

Select to close the dialog box without saving any changes.
