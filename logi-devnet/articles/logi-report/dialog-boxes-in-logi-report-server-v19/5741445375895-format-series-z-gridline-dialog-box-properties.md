# Format Series (Z) Gridline Dialog Box Properties

This topic describes how you can use the Format Series (Z) Gridline dialog box to format the gridlines on the series (Z) axis of a chart.

Server displays the dialog box when you right-click a chart and select **Format Gridlines** > **Format Series (Z) Gridline** (available to gauge bubble charts only) from the shortcut menu.

![Format Series (Z) Gridline dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905731485335/fmtseriesgrdln.gif)

**Color**

Specify the color schema of the gridlines.

**Transparency**

Specify the transparency of the color you want to apply to the gridlines.

**Line Style**

Specify the line style of value gridlines.

**Thickness**

Specify the thickness of value gridlines.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
