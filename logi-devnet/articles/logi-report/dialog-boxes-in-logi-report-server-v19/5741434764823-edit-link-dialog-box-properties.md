# Edit Link Dialog Box Properties

This topic describes how you can use the Edit Link dialog box to [edit the link](https://devnet.logianalytics.com/hc/en-us/articles/5741456004887-Adding-Links-in-Web-Report) on the selected object. Server displays the dialog box when you select an object which has a link, right-click it, and select **Edit Link** on the shortcut menu.

![Edit Link dialog - Report](https://devnet.logianalytics.com/hc/article_attachments/9905733872919/edtlnk_rpt.gif)

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.

For information about the other properties in the dialog box, see [Insert Link Dialog Box Properties](https://devnet.logianalytics.com/hc/en-us/articles/5741459269271-Insert-Link-Dialog-Box-Properties).
