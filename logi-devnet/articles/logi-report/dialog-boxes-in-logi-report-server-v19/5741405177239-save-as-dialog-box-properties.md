# Save As Dialog Box Properties

Use the Save As dialog box to save different resources when you access it from different places. This topic describes how to save a new dashboard or layout or save a copy of it.

---

When you access the Save As dialog box via the Save or Save As command on the JDashboard toolbar, you can use it to save the current dashboard.

![Save As dialog box - Dashboard](https://devnet.logianalytics.com/hc/article_attachments/9905737407127/svas_dshlc.gif)

**Save In**

Select the folder in the server resource tree where you want to save the dashboard. Use the arrow button ![Up Level button](https://devnet.logianalytics.com/hc/article_attachments/9905672802327/btn_dshbrd_up.gif) to go to the parent folder. The root folder cannot store resources.

The resource table shows the resources in the current folder. You can select the column names to change the order of the resources in the table.

* **Name**  
   File name.
* **Size**  
   File size.
* **Type**  
   File type.
* **Last Modified**  
   Last modified time of a file.

**File Name**

Type the file name of the dashboard you are going to save, without suffix.

**Save Sort Criteria**

Clear if you do not want to save the changes made by sorting.

**Save Filter Criteria**

Clear if you do not want to save the changes made by filtering.

**OK**

Select to save the dashboard.

**Cancel**

Select to close the dialog box without saving the dashboard.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif) **Help button**

Select to view information about the Save As dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif) **Close button**

Select to close the dialog box without saving the dashboard.

---

When you access the Save As dialog box by selecting the Save button ![Save button](https://devnet.logianalytics.com/hc/article_attachments/9905577991703/btn_save.gif) or Save As button ![Save As button](https://devnet.logianalytics.com/hc/article_attachments/9905615943319/btn_saveas.gif) in the [Export dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741426133271-Export-Dialog-Box-Properties#Dashboard) or [Print dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741441698711-Print-Dialog-Box-Properties), you can save a newly created export or print layout or an existing layout with a different name.

![Save As dialog box - Layout](https://devnet.logianalytics.com/hc/article_attachments/9905737431575/svas_layout.gif)

**Existing Layout**

Server lists the names of the layouts you already saved and the time when you modified them last time.

**Name**

Type the name of the layout you are going to save. You can use the file name without adding the file suffix.

**OK**

Select to save the current layout.

**Cancel**

Select to close the dialog box without saving the layout.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif) **Help button**

Select to view information about the Save As dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif) **Close button**

Select to close the dialog box without saving the layout.
