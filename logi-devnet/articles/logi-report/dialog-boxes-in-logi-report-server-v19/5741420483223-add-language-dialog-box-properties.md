# Add Language Dialog Box Properties

This topic describes how you can use the Add Language dialog box to select a language.

Server displays the dialog box when an administrator selects the Add button ![Add button](https://devnet.logianalytics.com/hc/article_attachments/9905619223831/btn_add1.gif)above the Language box in the [NLS Editor](https://devnet.logianalytics.com/hc/en-us/articles/5741420821783-NLS-Editor-Dialog-Box-Properties) when editing NLS for a resource, or selects **Languages Supported by** Logi Report and selects **OK** in the [Select Language Source dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741414956951-Select-Language-Source-Dialog-Box-Properties).

![Add Language dialog](https://devnet.logianalytics.com/hc/article_attachments/9905736783895/addlng.gif)

**Available Languages**

Server lists all the available languages for you to choose. Select the language you want.

**OK**

Select to add the selected language.

**Cancel**

Select to close the dialog box without adding a language.

**Help**

Select to view information about the Add Language dialog box.
