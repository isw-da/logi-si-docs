# Export Dialog Box Properties

This topic describes how you can use the Export dialog box to set settings for [exporting the report](https://devnet.logianalytics.com/hc/en-us/articles/5741470801175-Exporting-Printing-a-Web-Report) to different formats. Server displays the dialog box when you select Menu > File > Export.

![Export dialog](https://devnet.logianalytics.com/hc/article_attachments/9905633836695/expt.gif)

**File Name**

Specify the name for the exported output file.

**View Report Result**

Select if you want to directly open the report in the web browser if the format is supported by a plug-in of the web browser (otherwise, Server prompts you to save the output file).

**Save to File System**

Select if you want to save the report to a specified folder.

**Save to Version System**

Select if you want to save the report as a result version in Logi Report Server's [versioning system](https://devnet.logianalytics.com/hc/en-us/articles/5741462048023-Managing-Resource-Versions).

**Select Format**

Select the format in which you want to export the report.

**More/Less Options**

Select to show/hide additional settings for exporting the report to the specified format.

* **Style Group**  
   Specify the style group you want to apply to the exported output.
* **Properties**  
   Specify the properties for the selected format:
  + [PDF](https://devnet.logianalytics.com/hc/en-us/articles/5741420507799-Advanced-Run-Dialog-Box-Properties#PDF)
  + [HTML](https://devnet.logianalytics.com/hc/en-us/articles/5741420507799-Advanced-Run-Dialog-Box-Properties#HTML)
  + [Excel](https://devnet.logianalytics.com/hc/en-us/articles/5741420507799-Advanced-Run-Dialog-Box-Properties#Excel)
  + [Text](https://devnet.logianalytics.com/hc/en-us/articles/5741420507799-Advanced-Run-Dialog-Box-Properties#Text)
  + [RTF](https://devnet.logianalytics.com/hc/en-us/articles/5741420507799-Advanced-Run-Dialog-Box-Properties#RTF)
  + [XML](https://devnet.logianalytics.com/hc/en-us/articles/5741420507799-Advanced-Run-Dialog-Box-Properties#XML)
  + [PostScript](https://devnet.logianalytics.com/hc/en-us/articles/5741420507799-Advanced-Run-Dialog-Box-Properties#PS)

**OK**

Select to export the report with the settings you specified here.

**Cancel**

Select to close the dialog box without exporting the report.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without exporting the report.
