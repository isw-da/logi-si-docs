# PDF Sign Option Dialog Box Properties

This topic describes how you can use the PDF Sign Option dialog box to set signing options for the PDF output. Server displays the dialog box when you select **Sign** in the [Export dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741458025111-Export-Dialog-Box-Properties).

![PDF Sign Option dialog](https://devnet.logianalytics.com/hc/article_attachments/9905718442391/pdfsign.gif)

**Method**

Select the digital signing method.

**Digital ID File**

Select **Browse** to select a digital ID file from a local folder.

**User Password**

Specify a password for this digital ID file.

**Confirm Password**

Type the password again to confirm it.

**Reason for Signing Document**

Select the reason for signing the document.

**Edit by Yourself**

Type the reason for signing the document, when you select **I agree to 'specified' portions of this document** from the **Reason for Signing Document** drop-down list.

**Location**

Specify your location if you want.

**Your Contact Information**

Specify your contact information if you want.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
