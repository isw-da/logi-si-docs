# Insert Parameter Control Dialog Box Properties

You can use the Insert Parameter Control dialog box to [insert a parameter control](https://devnet.logianalytics.com/hc/en-us/articles/5741483099159-Applying-Web-Controls-in-Page-Report#Parameter) into a report for delivering parameter values to the report. This topic describes the properties in the dialog box.

![Insert Parameter Control dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905672545175/insrtprmctrl.gif)

**Select a Parameter**

Server displays all the parameters the current report uses. Select a parameter from the list.

**OK**

Select to insert a parameter control into the report.

**Cancel**

Select to close the dialog box without the insertion.

**Help**

Select to view information about the dialog box.
