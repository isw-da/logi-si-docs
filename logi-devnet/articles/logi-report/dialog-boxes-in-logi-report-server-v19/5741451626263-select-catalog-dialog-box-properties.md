# Select Catalog Dialog Box Properties

This topic describes how you can use the Select Catalog dialog box to select the catalog for [Quick Start With a Table Report](https://devnet.logianalytics.com/hc/en-us/articles/5741476791831-Quick-Start-with-a-Table-Report).

Server displays the dialog box when you do any of the following:

* In the Server Start Page, select **Web Report** in the **Create** category.
* Select New > Web Report on the Server Console > Resources page but the currently opened folder contains no catalog.
* Select **Change Catalog** in the [Select Data Source dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741446563095-Select-Data-Source-Dialog-Box-Properties#Report).

![Select Catalog dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905717770903/slctctlg.gif)

In the dialog box, Server lists all the available catalogs in a tree structure that you have published to Server. Select the catalog you want to use for the report.

**OK**

Select to use the selected catalog for creating a report. Server opens the [Select Data Source dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741446563095-Select-Data-Source-Dialog-Box-Properties#Report).

**Cancel**

Select to close the dialog box without selecting a catalog. This button is available only when you have opened the dialog box by selecting Change Catalog in the Select Data Source dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without selecting a catalog. This button is available only when you have opened the dialog box by selecting Change Catalog in the Select Data Source dialog box.
