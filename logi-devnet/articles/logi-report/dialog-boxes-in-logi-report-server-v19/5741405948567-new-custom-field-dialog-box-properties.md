# New Custom Field Dialog Box Properties

This topic describes how you can use the New Custom Field dialog box to create [user defined fields](https://devnet.logianalytics.com/hc/en-us/articles/5741461739671-Working-with-Custom-Fields), for use as resource properties.

Server displays the dialog box when an administrator selects **New Custom Field** in the Administration > Other > Custom Fields page on the Server Console.

![New Custom Field dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905714720791/nwcstmfld.gif)

**Custom Field Name**

Type the name for the custom field.

**Field Description**

Type the description about the custom field.

**Enabled**

Select to enable the custom field and show it in the resource properties.

**OK**

Select to create the custom field and close the dialog box.

**Cancel**

Select to close the dialog box without creating a custom field.

**Help**

Select to view information about the dialog box.
