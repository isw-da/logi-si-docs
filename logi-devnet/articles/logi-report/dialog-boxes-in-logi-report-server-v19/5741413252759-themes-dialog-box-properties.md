# Themes Dialog Box Properties

Use the Themes dialog box to change the theme of your JDashboard. This topic describes how to change the JDashboard theme.

Server displays the dialog box after you select the Options button ![Options button](https://devnet.logianalytics.com/hc/article_attachments/9905759210647/btn_dshbrd_optn.gif) on the toolbar and then select **Themes** from the option list.

![Themes dialog](https://devnet.logianalytics.com/hc/article_attachments/9905759242519/theme.gif)

**Themes**

Server lists the themes in the left panel. Each theme box contains the theme name and three colors. Hover over a color to view its hexadecimal value. Hover over a theme box, and server displays an arrow. Select the arrow, and Server shows the window element each color in the theme applies to.

![Theme List](https://devnet.logianalytics.com/hc/article_attachments/9905737370647/theme_lst.gif)

**Preview**

Preview of the JDashboard window with the selected theme, in the right panel.

**Apply**

Select to apply the selected theme to JDashboard without closing the dialog box.

**OK**

Select to apply the selected theme to JDashboard.

**Cancel**

Select to close the dialog box without changing the theme.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the Themes dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without changing the theme.
