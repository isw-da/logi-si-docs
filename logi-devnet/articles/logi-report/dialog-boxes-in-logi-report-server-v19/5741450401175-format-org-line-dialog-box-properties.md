# Format Org Line Dialog Box Properties

This topic describes how you can use the Format Org Line dialog box to format the lines of an org chart. Server displays the dialog box when you right-click an org chart and select **Format Line** from the shortcut menu.

![Format Org Line dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905720123159/fmtorgln.gif)

**Line Style**

Select the style of the lines.

**Color**

Specify the color of the line border.

**Thickness**

Specify the thickness of the line border.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
