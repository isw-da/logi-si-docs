# Change Password Dialog Box Properties

Use the Change Password dialog box to [change the password](https://devnet.logianalytics.com/hc/en-us/articles/5741463728407-Managing-User-Accounts#Password) of a user. This topic describes how to change the password.

Server displays the dialog box when an administrator selects **Change Password** for a user in the Control column of the Administration > Security > User page on the Server Console.

![Change Password dialog box](https://devnet.logianalytics.com/hc/article_attachments/9305329808663/chgpswd.gif)

**Selected Realm**

The current realm.

**User Name**

Name of the user whose password you want to change.

**System Administrator Password**

Type your password as an administrator.

**New Password**

Type the new password for the user.

**Confirm New Password**

Type the password again to confirm it.

**OK**

Select to apply the new password and exit the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.
