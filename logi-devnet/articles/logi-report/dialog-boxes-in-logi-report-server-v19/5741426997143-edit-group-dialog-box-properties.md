# Edit Group Dialog Box Properties

This topic describes how you can use the Edit Group dialog box to edit a [group](https://devnet.logianalytics.com/hc/en-us/articles/5741475829015-Managing-Groups).

Server displays the dialog box when an administrator selects a group name in the Name column of the Administration > Security > Group page on the Server Console.

![Edit Group dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905736629783/edtgrp.gif)

**Selected Realm**

Realm in which the group is.

**Group Name**

Name of the group.

**Privileges**

Select the privileges you want the group to have:

* **Publish**  
   The privilege of publishing resources to Logi Report Server.
* **Advanced Properties**  
   The privilege of viewing advanced version properties information, such as catalog connections and report related resources.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.
