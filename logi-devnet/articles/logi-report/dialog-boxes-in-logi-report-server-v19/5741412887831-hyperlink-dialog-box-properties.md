# Hyperlink Dialog Box Properties

Use the Hyperlink dialog box to [add a hyperlink](https://devnet.logianalytics.com/hc/en-us/articles/5741409582359-Creating-and-Saving-Dashboards#HTML) on the selected object. This topic describes how to add URL.

Server displays the dialog box when you select the Hyperlink button ![Hyperlink button](https://devnet.logianalytics.com/hc/article_attachments/9905760207127/btn_dshbrd_lnk.gif) in the [Edit HTML dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741441310743-Edit-HTML-Dialog-Box-Properties) or [Insert HTML dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741432184471-Insert-HTML-Dialog-Box-Properties).

![Hyperlink dialog](https://devnet.logianalytics.com/hc/article_attachments/9905760429335/hyplnk.gif)

**URL**

Type the URL you want to direct the hyperlink to.

**OK**

Select to add the hyperlink.

**Cancel**

Select to close the dialog box without adding a hyperlink.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the Hyperlink dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without adding a hyperlink.
