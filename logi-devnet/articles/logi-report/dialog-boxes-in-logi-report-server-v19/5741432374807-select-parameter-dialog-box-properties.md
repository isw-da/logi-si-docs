# Select Parameter Dialog Box Properties

Use the Select Parameter dialog box to [insert a parameter into the URL of the web page](https://devnet.logianalytics.com/hc/en-us/articles/5741409582359-Creating-and-Saving-Dashboards#URL) that you want to add to a dashboard. This topic describes how to select a parameter.

Server displays the dialog box when you select **Add Parameter** in the [Insert URL Frame dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741419705239-Insert-URL-Frame-Dialog-Box-Properties) or [Edit URL Frame dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741426072471-Edit-URL-Frame-Dialog-Box-Properties) if you have specified a data source, or select a data source and select **OK** in the [Select Data Source dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741405093527-Select-Data-Source-Dialog-Box-Properties).

![Select Parameter dialog](https://devnet.logianalytics.com/hc/article_attachments/9905759456023/slctprmtr.gif)

**Parameters**

Server lists the available parameters in the data source. Select one to insert into the URL.

**Change Data Source**

Select to open the [Select Data Source dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741405093527-Select-Data-Source-Dialog-Box-Properties) to use another data source.

**OK**

Select to insert the selected parameter into the URL.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif) **Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif) **Close button**

Select to close the dialog box without saving any changes.
