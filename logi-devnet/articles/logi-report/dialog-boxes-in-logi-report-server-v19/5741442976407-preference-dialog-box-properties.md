# Preference Dialog Box Properties

This topic describes how you can use the Preference dialog box to [set preferences](https://devnet.logianalytics.com/hc/en-us/articles/5741463728407-Managing-User-Accounts#Preference) for the specified user.

Server displays the dialog box when an administrator selects **Preference** of a user in the Control column of the Administration > Security > User page on the Server Console.

![Preference dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905636053143/prfrnc.gif)

It contains two subjects: Customize Server Preferences and Customize Profile. You can specify the [server preferences](https://devnet.logianalytics.com/hc/en-us/articles/5741409420567-Configuring-Server-Profile) for the user as well as the [Default](https://devnet.logianalytics.com/hc/en-us/articles/5741427459479-Page-Report-Studio-Profile-Properties#Default) and [Advanced](https://devnet.logianalytics.com/hc/en-us/articles/5741427459479-Page-Report-Studio-Profile-Properties#PropertiesAdvanced) properties you want the user to use in Page Report Studio.
