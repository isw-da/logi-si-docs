# Printer Properties

This topic describes how you can use the Printer Properties dialog box to update the printing properties. Server displays the dialog box when you select **Properties** in the [Print dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741459652119-Print-Dialog-Box-Properties).

![Printer Properties dialog](https://devnet.logianalytics.com/hc/article_attachments/9905729704855/prntprpty.gif)

**Print Method**

Specify the printing method to print the report.

**Media Tray**

Select the media tray for the printer.

**Copies**

Specify the information of the copies.

* **Number of Copies**   
   Specify the number of copies you want to print. The number of copies will apply to all specified pages.
* **Collate**  
   Specify the printing order for printing multiple pages. If you select **Collate**, when you print two copies of a three-page report, the page order is 1, 2, 3, 1, 2, 3. Otherwise, the order will be 1, 1, 2, 2, 3, 3.

**Color**

Specify the color of the output.

* **Monochrome**  
   Select to print the report in Monochrome.
* **Color**  
   Select to print the report in Color.

**Sides**

Specify the print sides for the printed output.

* **One Side**  
   Select to impose each consecutive print-stream page upon the same side of consecutive media sheets.
* **Duplex**  
   Select to impose each consecutive pair of print-stream pages upon the front and back sides of consecutive media sheets, so that the orientation of each pair of print-stream pages on the medium is correct for the reader as if for binding on the long edge.
* **Tumble**  
   Select to impose each consecutive pair of print-stream pages upon the front and back sides of consecutive media sheets, so that the orientation of each pair of print-stream pages on the medium is correct for the reader as if for binding on the short edge.

**Quality**

Specify the printing quality.

* **Draft**  
   Select to print the report in draft quality.
* **Normal**  
   Select to print the report in normal quality.
* **High**  
   Select to print the report in high quality.

**Job Attributes**

Specify the properties of the print task.

* **Priority**  
   Specify the print priority for this task.
* **Job Name**   
   Specify the name of the print task.
* **User Name**   
   Specify the name of the user.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.
