# Insert URL Frame Dialog Box Properties

Use the Insert URL Frame dialog box to [insert a web page](https://devnet.logianalytics.com/hc/en-us/articles/5741409582359-Creating-and-Saving-Dashboards#URL) into the dashboard. This topic describes how to specify the URL of a web page.

Server displays the dialog box when you drag **URL Frame** from Toolbox in the Resources panel to the dashboard body.

![Insert URL Frame dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905738073367/insrturl.gif)

**Title**

Type the title of the frame that holds the web page.

**URL**

Type the URL of the web page you would like to show in the dashboard.

Select **Add Parameter** if you want to insert a parameter into the URL to compose a dynamic URL. Server displays the [Select Data Source dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741405093527-Select-Data-Source-Dialog-Box-Properties) for you to select the data source that contains the required parameter when you haven't specified any data source, or the [Select Parameter dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741432374807-Select-Parameter-Dialog-Box-Properties) when you have specified a data source.

**Auto refresh**

Select if you want to refresh the specified web page periodically. Then, type the time interval between two refreshes in the **Interval** text box.

**OK**

Select to insert the web page in the dashboard body.

**Cancel**

Select to close the dialog box without the insertion.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the Insert URL Frame dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without the insertion.
