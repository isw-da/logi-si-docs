# Turn to Page Dialog Box Properties

You can use the Turn to Page dialog box to jump to the specified page. This topic describes the properties in the dialog box.

![Turn to Page dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905761412631/tn2pg.gif)

**Input Page Number**

Type the page number you would like to turn to.

**OK**

Select to display the specified page.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
