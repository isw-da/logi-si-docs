# Comparison Function Dialog Box Properties

You can use the Comparison Function dialog box to add a [comparison function](https://devnet.logianalytics.com/hc/en-us/articles/5741476545943-Inserting-Components-in-Web-Report#Comparison) as an aggregate field for a crosstab. This topic describes the properties in the dialog box.

Server displays the dialog box when you select **Comparison Function** in the [Crosstab Wizard](https://devnet.logianalytics.com/hc/en-us/articles/5741428630807-Crosstab-Wizard-Properties), [Insert Crosstab dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741445839383-Insert-Crosstab-Dialog-Box-Properties), [Convert to Crosstab dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741449434903-Convert-to-Crosstab-Dialog-Box-Properties), or [To Crosstab dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741460447511-To-Crosstab-Dialog-Box-Properties), or in the Bind Data screen for crosstab in the [Web Report Wizard](https://devnet.logianalytics.com/hc/en-us/articles/5741437169815-Web-Report-Wizard-Properties#Crosstab).

![Comparison Function dialog](https://devnet.logianalytics.com/hc/article_attachments/9905632517911/cmprsnfctn.gif)

**Function**

Select the function of the calculation: Percentage, Permillage, or Difference.

**Comparison Function Spans on Row Direction**

Select if you want to place the comparison function into the column total cell of the crosstab.

**Comparison Function Spans on Column Direction**

Select if you want to place the comparison function into the row total cell of the crosstab.

**Break By**

Select the first parameter of the comparison function.

**Refer To**

Select the other parameter of the comparison function.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
