# Share Parameters Setting Dialog Box Properties

Use the Share Parameters Setting dialog box to [share parameters](https://devnet.logianalytics.com/hc/en-us/articles/5741423625879-Specifying-Parameter-Values-for-a-Dashboard#Share) among library components. This topic describes how to share parameters.

Server displays the dialog box when you select the Options button ![Options button](https://devnet.logianalytics.com/hc/article_attachments/9905759210647/btn_dshbrd_optn.gif) on the toolbar and then select **Share Parameter** from the option list.

![Next Page dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905759488663/shareprm.gif)

**All Parameters**

Server lists the parameters that the library components in the current dashboard use. Server places cascading parameters in the same group within one row.

* To share parameters, hold the **Ctrl** key to select the parameters you want, and then select **Share**. Server adds the parameters into one sharing group.
* To add another parameter into a sharing group, select any parameter in the group while holding Ctrl, select the parameter, and then select **Share**.
* To remove parameters from a sharing group, select the parameters, and then select **Cancel Share**.

**Apply**

Select to apply any changes you made here without exiting the dialog box.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
