# Edit Label Dialog Box Properties

You can use the Edit Label dialog box to [edit the properties of the label](https://devnet.logianalytics.com/hc/en-us/articles/5741409582359-Creating-and-Saving-Dashboards#Label) such as font, size, color, border, and the way of aligning text. This topic describes the properties in the dialog box.

Server displays the dialog box when you hover over a label in the dashboard header and then select the Edit button ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/9905772452759/btn_dshbrd_edt.gif) that displays around the label.

![Edit Label dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905760898967/edtlbl.gif)

![Font Face](https://devnet.logianalytics.com/hc/article_attachments/9905672060311/btn_fontface.gif) **Font Face**

Select the font of the text from the drop-down list.

![Font Size](https://devnet.logianalytics.com/hc/article_attachments/9905672109719/btn_fontsize.gif) **Font Size**

Select the font size of the text from the drop-down list.

![Font Color](https://devnet.logianalytics.com/hc/article_attachments/9905760161431/btn_dshbrd_fontclr.gif) **Font Color button**

Specify the color of the text.

To change the color, select the color indicator. Then, select a color, or select **More Colors** to access the [Color Picker dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741404530839-Color-Picker-Dialog-Box-Properties) in which you can select a color within a wider range.

![Background Color](https://devnet.logianalytics.com/hc/article_attachments/9905760177815/btn_dshbrd_bkgrndclr.gif)**Background Color button**

Specify the background color of the label.

![Bold](https://devnet.logianalytics.com/hc/article_attachments/9905672154647/btn_bold.gif)**Bold button**

Select to make the text bold.

![Italic](https://devnet.logianalytics.com/hc/article_attachments/9905672184599/btn_italic.gif)**Italic button**

Select to make the text italic.

![Underline](https://devnet.logianalytics.com/hc/article_attachments/9905672206743/btn_underline.gif)**Underline button**

Select to underline the text.

![Align Left](https://devnet.logianalytics.com/hc/article_attachments/9905672228247/btn_alignleft.gif) **Align Left button**

Select to align the text to the left.

![Align Center](https://devnet.logianalytics.com/hc/article_attachments/9905660223895/btn_aligncntr.gif)**Align Center button**

Select to align the text in the center.

![Align Right](https://devnet.logianalytics.com/hc/article_attachments/9905660238999/btn_alignright.gif)**Align Right button**

Select to align the text to the right.

**Preview box**

Server displays a preview of the label with the specified properties.

**Width**

Specify the width of the label.

**Height**

Specify the height of the label.

**Border**

Specify the border properties of the label.

* **![Border Width](https://devnet.logianalytics.com/hc/article_attachments/9905772501655/btn_dshbrd_brdrwdth.gif) Border Width**Specify the width of the border.
* **![Border Line](https://devnet.logianalytics.com/hc/article_attachments/9905772534551/btn_dshbrd_brdrstyl.gif) Border Line**Select the line style of the border from the drop-down list.
* **![Border Color](https://devnet.logianalytics.com/hc/article_attachments/9905760873111/btn_dshbrd_brdrclr.gif) Border Color button**Specify the color of the border.

**Padding**

Specify the space between the text and the borders of the label.

**Auto Fit**

Select to adjust the width and height of the label according to the contents.

**Apply**

Select to apply any changes you made here without closing the dialog box.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
