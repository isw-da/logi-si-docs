# Real-time Settings Dialog Box Properties

Use the Real-time Settings dialog box to [configure the settings for the real time chart](https://devnet.logianalytics.com/hc/en-us/articles/5741401738263-Manipulating-Data-Components#RTChart). This topic describes how to update real-time settings.

Server displays the dialog box when you right-click a real time chart and select **Real-time Settings** from the shortcut menu.

![Real-time Settings dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905737627543/rltmset.gif)

**Refresh Interval**

Specify the time interval at which the chart will get data and refresh itself automatically.

**Show Most Recent N Points**

Type the number of records you want to keep for the real time data in the chart.

**Incremental Fetch**

Select to open the [Unique Key dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741432454935-Unique-Key-Dialog-Box-Properties) to configure a unique key for the real time chart.

**Use Scrollable Bar**

Select if you want to add a scroll bar to the chart, to control the visible value range on the X axis of the chart.

* **Scrollable Visible Values**  
   Type the number of data items you want to select on the scroll bar and to display on the axis by default.
* **Scrolling Area Percentage**  
   Type the percentage the scroll bar occupies the whole chart.
* **Show a Chart in Scrollable Bar**  
   Select to display the thumbnail chart on the scroll bar.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
