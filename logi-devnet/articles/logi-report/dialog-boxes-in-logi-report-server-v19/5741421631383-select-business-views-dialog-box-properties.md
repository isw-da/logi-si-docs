# Select Business Views Dialog Box Properties

This topic describes how you can use the Select Business Views dialog box to [add or update the business views in the business view security in a catalog](https://devnet.logianalytics.com/hc/en-us/articles/5741438178199-Changing-Resource-Properties#BVSecurity).

Server displays the dialog box when an administrator selects **Select** on the **Business View Security** tab of the [Properties dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741414587159-Resource-Properties) for a catalog.

![Select Business Views dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905714417815/slctbv.gif)

**Select Business Views**

Server lists all the business views in the catalog. You can select multiple business views by selecting the corresponding checkboxes.

**All**

Select to select all the business views in the catalog.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
