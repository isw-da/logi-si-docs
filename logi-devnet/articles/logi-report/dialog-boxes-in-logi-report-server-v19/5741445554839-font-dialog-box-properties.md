# Font Dialog Box Properties

This topic describes how you can use the Font dialog box to specify the font format of a label or field. Server displays the dialog box when you select a label or field and select the Font button ![Font button](https://devnet.logianalytics.com/hc/article_attachments/9905578431511/btn_wbrpt_font1.gif) on the toolbar.

![Font dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905718869143/font.gif)

**Font**

Specify the font face.

**Size**

Specify the font size.

**Color**

Specify the font color. To change the color, select the color indicator. Server displays the color palette. Select a color, or select **More Colors** to access the [Color Picker dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741428592023-Color-Picker-Dialog-Box-Properties) in which you can specify a color within a wider range.

![Bold](https://devnet.logianalytics.com/hc/article_attachments/9905672154647/btn_bold.gif)

Select if you want to make the object bold.

![Italic](https://devnet.logianalytics.com/hc/article_attachments/9905672184599/btn_italic.gif)

Select if you want to make the object italic.

![Underline](https://devnet.logianalytics.com/hc/article_attachments/9905672206743/btn_underline.gif)

Select if you want to underline the object.

**Preview**

Server shows an example of the font effect.

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.
