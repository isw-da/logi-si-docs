# This image notes any new content for version 19.2 of the product. For more information please see the Release Notes or What's New topics.Color Picker Dialog Box Properties

You can use the Color Picker dialog box to update a color for an object. This topic describes the properties in the dialog box.

This topic contains the following sections:

* [Standard Tab Properties](#standard)
* [Custom Tab Properties](#Custom)

You see these elements on both tabs:

**Cancel**

Select to close the dialog box without saving any changes.

**OK**

Select to apply any changes you made here and close the dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905902377623/btn_lrstd_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905880426903/btn_lrstd_delete.gif)**Close button**

Select to close the dialog box without saving any changes.

## Standard Tab Properties

Select a color from the web safe color swatches.

![Color Picker dialog box - Standard tab](https://devnet.logianalytics.com/hc/article_attachments/9905902419095/colorpkr_stdrd.gif)

**Web Safe Colors**

Select a web safe color.

**R**

Specify the amount of red in a color, from 0 to 255.

**G**

Specify the amount of green in a color, from 0 to 255.

**B**

Specify the amount of blue in a color, from 0 to 255.

**#**

Specify the hexadecimal value of the color.

**New**

Server displays the color you selected.

**Current**

Server always displays the original color.

## Custom Tab Properties

Customize the color within a wider range.

![Color Picker dialog box - Custom tab](https://devnet.logianalytics.com/hc/article_attachments/9905902446103/colorpkr_custm.gif)

**Color matrix**

Select anywhere in the color matrix. Change saturation horizontally or change brightness vertically.

**Color bar**

Select a color on the bar to change the hue of the color matrix.

**R**

Specify the amount of red in a color, from 0 to 255.

**G**

Specify the amount of green in a color, from 0 to 255.

**B**

Specify the amount of blue in a color, from 0 to 255.

**#**

Specify the hexadecimal value of the color.

**New**

Server displays the color you selected.

**Current**

Server always displays the original color.
