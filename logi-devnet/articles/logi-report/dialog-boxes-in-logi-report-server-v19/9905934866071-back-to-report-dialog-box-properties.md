# Back to Report Dialog Box Properties

# This image notes any new content for version 19.2 of the product. For more information please see the Release Notes or What's New topics.Back to Report Dialog Box Properties

You can use the Back to Report dialog box to [specify whether to display full report data and return to Page Report Studio](https://devnet.logianalytics.com/hc/en-us/articles/9905923717783-Working-with-Logi-Report-Studio#Return). This topic describes the properties in the dialog box.

Server displays the dialog box when you select the **Return** button ![Return button](https://devnet.logianalytics.com/hc/article_attachments/9905902325911/btn_lrstd_return.gif).

![Back to Report dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905902351127/bck2rpt.gif)

**Full Data**

Select if you want to display full data in the report.

**Partial Data**

Select if you don't want to display full data in the report. Then, specify the number of data records you want to display in the report.

**Cancel**

Select to close the dialog box and stays in Logi Report Studio.

**OK**

Select to return to Page Report Studio and display the report with full data or partial data.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905902377623/btn_lrstd_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905880426903/btn_lrstd_delete.gif)**Close button**

Select to close the dialog box and stays in Logi Report Studio.
