# Filter Inspector Dialog Box Properties

This topic describes how you can use the Filter Inspector dialog box to view all the filters that the current web report is using. Server displays the dialog box when you select **Inspector** in the [Filter dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741449994391-Filter-Dialog-Box-Properties).

![Filter Inspector dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905733448471/fltrinspct.gif)

**Dataset Filter**

Filters that you create via the [Edit Dataset Filter dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741470839191-Applying-Filters-in-Web-Report#BV).

**Filter**

Filters that you create via the [Filter dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741470839191-Applying-Filters-in-Web-Report#Dialog).

**On-screen Filter**

Filters that you create via the [Filter panel](https://devnet.logianalytics.com/hc/en-us/articles/5741470839191-Applying-Filters-in-Web-Report#Panel) and [filter controls](https://devnet.logianalytics.com/hc/en-us/articles/5741484852887-Using-Web-Controls-in-Web-Report#Filter).

**Go To Filter**

Go-to filters that you create when performing the [go-to-by-value action](https://devnet.logianalytics.com/hc/en-us/articles/5741476511127-Going-Through-Web-Report-Data#Go-to-by-value) on the report.

**OK**

Select to close the dialog box.

**Cancel**

Select to close the dialog box.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box.
