# Group Footer Properties

This topic describes how you can use the Group Footer Properties dialog box to specify the properties of a table group footer in a web report.

Server displays the dialog box when you select anywhere in a table group footer row, then select the triangle icon ![Select Row button](https://devnet.logianalytics.com/hc/article_attachments/9905617583767/btn_wbrpt_slctrow.gif) that Server displays on the group footer to select the row, right-click on the row, and select **Properties** from the shortcut menu.

This topic contains the following sections:

* [General Tab Properties](#General)
* [Border Tab Properties](#Border)

You see these elements on both tabs:

**OK**

Select to apply any changes you made here and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/9905613168791/btn_help.gif)**Help button**

Select to view information about the dialog box.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/9905575212823/btn_close.gif)**Close button**

Select to close the dialog box without saving any changes.

## General Tab Properties

Specify the general properties of the group footer.

![Group Footer Properties dialog box - General tab](https://devnet.logianalytics.com/hc/article_attachments/9905730488855/grpfotrprpty_gnrl.gif)

**Name**

Specify the display name of the group footer.

**Show NLS Value**

Select to show the translated name for the display name of the object in the **Name** text box if you have enabled the NLS feature and translated it, and when you have not modified the display name of the object.

**Height**

Specify the height of the group footer.

**Background**

Specify the background color of the group footer.

To change the color, select the color indicator. Server displays the color palette. Select a color, or select **More Colors** to access the [Color Picker dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741428592023-Color-Picker-Dialog-Box-Properties) in which you can specify a color within a wider range, or select **More Fill Effects** to access the [Fill Effects dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5741444679959-Fill-Effects-Dialog-Box-Properties) to specify a gradient or an image as the fill effect.

## Border Tab Properties

Specify the border properties of the group footer.

![Group Footer Properties dialog box - Border tab](https://devnet.logianalytics.com/hc/article_attachments/9905730502679/grpfotrprpty_border.gif)

**Color**

Specify the border color.

**Width**

Specify the border width in inches.

**Top Line**

Select the style of the top border line.

**Bottom Line**

Select the style of the bottom border line.

**Left Line**

Select the style of the left border line.

**Right Line**

Select the style of the right border line.
