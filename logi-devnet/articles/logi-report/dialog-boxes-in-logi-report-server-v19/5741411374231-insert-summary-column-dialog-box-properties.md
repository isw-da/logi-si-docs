# Insert Summary Column Dialog Box Properties

You can use the Insert Summary Column dialog box to [insert a new summary column](https://devnet.logianalytics.com/hc/en-us/articles/5741475063447-Manipulating-Data-Components-in-Page-Report#InsertColumn) into a table. This topic describes the properties in the dialog box.

![Insert Summary Column dialog box](https://devnet.logianalytics.com/hc/article_attachments/9905673152407/insrtsumclmn.gif)

**Resources**

Server lists all the available data resources for the new summary column. Select the one you want.

**OK**

Select to insert a summary column in the table.

**Cancel**

Select to close the dialog box without the insertion.

**Help**

Select to view information about the dialog box.
