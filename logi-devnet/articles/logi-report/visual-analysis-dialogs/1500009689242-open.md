# Open

The Open dialog helps you to open another analysis template in a new Visual Analysis window.

![Open dialog](https://devnet.logianalytics.com/hc/article_attachments/4412131457815/open.gif)

**Open In**

Browse to the directory in the server resource tree where the target analysis template is located and then select it. Use the button ![Up Level button](https://devnet.logianalytics.com/hc/article_attachments/4412112541463/btn_dshbrd_up.gif) to go to the parent folder.

The resource table shows the analysis templates in the current directory.

**File Name**

Shows the selected analysis template file name.

**OK**

Closes this dialog and opens the selected analysis template.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412112604951/btn_va_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112484759/btn_delete.gif)

Ignores the setting and closes this dialog.
