# Edit Shapes

The Edit Shapes dialog helps you to specify shapes for the members of the shape legend field.

![Edit Shapes dialog](https://devnet.logianalytics.com/hc/article_attachments/4412139563415/edtshp.gif)

**Pattern**

After selecting a pattern from the drop-down list, a preview of the shapes of the pattern is provided below the pattern name. Changing a pattern will apply the shapes in the new pattern to all of the listed field members.

**Data Items**

Lists the members of the field with the shapes in the selected pattern.

To customize the shape of a data item, select the data item and then select a shape in the pattern preview box.

**OK**

Closes this dialog and applies the specified shapes.

![Help button](https://devnet.logianalytics.com/hc/article_attachments/4412112604951/btn_va_help.gif)

Displays the help document about this feature.

![Close button](https://devnet.logianalytics.com/hc/article_attachments/4412112484759/btn_delete.gif)

Ignores the setting and closes this dialog.
