# Appendices

Logi Report Server Help System has the following appendices:

* [Appendix 1: URL Properties for Running, Scheduling, and Viewing Reports via URL](https://devnet.logianalytics.com/hc/en-us/articles/1500009743382-Appendix-1-URL-Properties-for-Running-Scheduling-and-Viewing-Reports-via-URL)
* [Appendix 2: Configuring Logi Report Server Using Properties in server.properties](https://devnet.logianalytics.com/hc/en-us/articles/1500009771281-Appendix-2-Configuring-Logi-Report-Server-Using-Properties-in-server-properties)

The purpose of these appendices is to make the use of Logi Report Server much easier and more convenient by introducing you to the types of formula functions and operators, as well as the parameter values required for building user defined web actions.
