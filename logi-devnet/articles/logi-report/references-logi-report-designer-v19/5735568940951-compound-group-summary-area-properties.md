# Compound Group/Summary Area Properties

This topic describes the properties of a Compound Group, including Column Compound Group and Row Compound Group, and Summary Area object in a crosstab.

You can use these objects in crosstabs in all report types. However, only in query-based page reports, the objects contain the following properties.

| Property Name | Description |
| --- | --- |
| Class Type | Shows the class type of the object. Read only. |
| Instance Name | Shows the instance name of the object. Read only. |
