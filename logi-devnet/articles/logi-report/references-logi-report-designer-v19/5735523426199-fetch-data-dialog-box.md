# Fetch Data Dialog Box

You can use the Fetch Data dialog box to [get data from the database](https://devnet.logianalytics.com/hc/en-us/articles/5735527778327-Using-Advanced-Web-Controls#CusValue) for using in a filter control. This topic describes the options in the dialog box.

Designer displays the Fetch Data dialog box when you select Fetch Data in the [Insert Filter Control dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735551751575-Insert-Filter-Control-Dialog-Box) or [Edit Filter Control dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735565634071-Edit-Filter-Control-Dialog-Box).

![Fetch Data dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898479592343/ftchdt.gif)

Designer displays these options:

**Search box**

Type the keyword in the box and Designer lists the values containing the matched text.

**Value box**

This box lists the values you can select to use in the filter control. Select the checkboxes for corresponding values you need. To select or clear all the listed values, select or clear the checkbox on the top. You can sort the listed values in the ascending or descending order by selecting the Values column header.

**OK**

Select to add the specified values to the value list right after the cursor in the Insert Filter Control dialog box or Edit Filter Control dialog box.

**Cancel**

Select to quite selecting the values and close the dialog box.

**Help**

Select to view information about the dialog box.
