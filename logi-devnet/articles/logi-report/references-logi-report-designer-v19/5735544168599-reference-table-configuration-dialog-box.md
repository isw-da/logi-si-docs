# Reference Table Configuration Dialog Box

You can use the Reference Table Configuration dialog box to select the catalog resource types that you want to add into the [reference table](https://devnet.logianalytics.com/hc/en-us/articles/5735526938391-Clarifying-the-Reference-Relationships-Between-Resources-in-a-Catalog#Configure). This topic describes the options in the dialog box.

Designer displays the Reference Table Configuration dialog box when you select Configure Reference on the Catalog Manager toolbar.

![Reference Table Configuration dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898461089687/reftblcnfg.gif)

Designer displays these options:

**Data Sources**

This box lists all the data sources available in the catalog. Select a data source to specify its resource types to add into the reference table.

**Catalog Resource Types**

This box lists all the resource types that you can add into the reference table. Select the resource types you want to add into the reference table.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
