# Edit Line Style Dialog Box

You can use the Edit Line Style dialog box to [edit the line style for the selected condition](https://devnet.logianalytics.com/hc/en-us/articles/5735511924119-Applying-Conditional-Color-Fills-to-Charts#More). This topic describes the options in the dialog box.

Designer displays the Edit Line Style dialog box when you select Edit ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/7933688838807/btn_edit.gif) in the Fill tab of the [Format Line dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735509343767-Format-Line-Dialog-Box#Fill) while selecting Use Single Color with Condition.

![Edit Line Style dialog box](https://devnet.logianalytics.com/hc/article_attachments/7933755998487/edtlnstly.gif)

Designer displays these options:

**Fill**

You can specify the fill pattern of the chart values that meet the condition in this box. Designer disables this box when you select Apply Color to Line in the Node tab of the Format Line dialog box.

* **Color**  
  Specify the color of the chart values. To edit the color, select the color indicator and select a color from the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/5735570054167-Menu-Tabs#Palette), or type the hexadecimal value of a color (for example, 0xff0000) in the text box.
* **Transparency**  
  Specify the transparency of the color.

**Area**

You can specify the fill pattern of the areas that meet the condition in this box (the areas are formed by the chart axes and the chart line). Designer applies these properties to 2-D lines only.

* **Color**  
  Specify the color of the areas. To edit the color, select the color indicator and select a color from the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/5735570054167-Menu-Tabs#Palette), or type the hexadecimal value of a color (for example, 0xff0000) in the text box.
* **Transparency**  
   Specify the transparency of the color.

**Line**

You can specify the line pattern of the chart values that meet the condition in this box.

* **Line Style**  
  Specify the style of the chart line.
* **Thickness**  
  Specify the line thickness, in pixels.

**Sample**

This box displays a preview sample of your settings.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
