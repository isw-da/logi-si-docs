# Replace Formula Dialog Box

You can use the Replace Formula dialog box to [select another formula to replace the current one](https://devnet.logianalytics.com/hc/en-us/articles/5735547138327-Creating-Queries-in-a-Catalog#FormulaField) in a query. This topic describes the options in the dialog box.

Designer displays the Replace Formula dialog box when you double-click any formula that you add to be the column of a query in the [Query Editor dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735552449687-Query-Editor-Dialog-Box).

![Replace Formula dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898461058839/rplcfmla.gif)

Designer displays these options:

**Formulas**

This box lists all the available formulas that you can use as query columns. Select the formula you want to use to replace the current one.

**Expression**

This box shows the expression of the selected formula.

**Replace**

Select to replace the current formula with the specified one in the Formulas box.

**Close**

Select to close the dialog box.

**Help**

Select to view information about the dialog box.
