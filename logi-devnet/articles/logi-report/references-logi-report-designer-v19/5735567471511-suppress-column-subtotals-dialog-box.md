# Suppress Column Subtotals Dialog Box

You can use the Suppress Column Subtotals dialog box to customize the visibility of the subtotals of the column fields in a crosstab. This topic describes the options in the dialog box.

Designer displays the Suppress Column Subtotals dialog box when you select the ellipsis ![Ellipsis button](https://devnet.logianalytics.com/hc/article_attachments/7933651735063/btn_ellipsis3.gif) next to Suppress Column Subtotals in the Layout screen of the [Create Crosstab dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735522178583-Create-Crosstab-Dialog-Box) or [Crosstab Wizard dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735528674583-Crosstab-Wizard-Dialog-Box).

![Suppress Column Subtotal dialog box](https://devnet.logianalytics.com/hc/article_attachments/7933693061655/sprsclmnsbtl.gif)

Designer displays these options:

**Same for all**

Select to apply the setting for the subtotal of the first column field in the box below to all the subtotals of the column fields in the crosstab.

**Name**

For a common crosstab, the column shows the column fields in the crosstab, excluding the one on the topmost column group level; for a compound crosstab, the column shows the column fields excluding the one on the topmost column group level in each column compound group.

**Suppress**

This column shows whether you specify to suppress the subtotals of the column fields. To specify the value, select in the value cell and select "true" or "false" from the drop-down list. You can also [use a constant level formula to control the value](https://devnet.logianalytics.com/hc/en-us/articles/5735545031703-Using-Formulas-to-Control-Object-Properties).

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
