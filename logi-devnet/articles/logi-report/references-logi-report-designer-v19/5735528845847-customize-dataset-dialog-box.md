# Customize Dataset Dialog Box

You can use the Customize Dataset dialog box to [customize the datasets](https://devnet.logianalytics.com/hc/en-us/articles/5735534254999-Managing-Page-Reports#Import) used by components of the imported report tabs. This topic describes the options in the dialog box.

Designer displays the Customize Dataset dialog box when you select Customize Dataset for Components and select OK in the [Select Report Tab dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735531292311-Select-Report-Tab-Dialog-Box).

![Customize Dataset dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898473919511/cstmzdtst.gif)

Designer displays these options:

**Report Tab Name**

This column shows the names of the report tabs containing the components.

**Component Name**

This column shows the names of the components.

**Component Type**

This column shows the types of the components.

**Dataset Name**

This column shows the datasets that you select for the components.

**Description**

This column shows the descriptions of the components.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
