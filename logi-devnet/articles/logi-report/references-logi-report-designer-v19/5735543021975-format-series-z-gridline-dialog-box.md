# Format Series (Z) Gridline Dialog Box

You can use the Format Series (Z) Gridline dialog box to [format the gridlines on the series axis (the Z axis)](https://devnet.logianalytics.com/hc/en-us/articles/5735563944855-Formatting-Chart-Axis-Gridlines) of a chart. This topic describes the options in the dialog box.

Designer displays the Format Series (Z) Gridline dialog box when you right-click a chart and navigate to Format Gridlines > Series (Z) Gridline on the shortcut menu, or double-click any gridline on the series axis of a chart.

![Format Series (Z) Gridline dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898480044823/fmtseriesgrdln.gif)

Designer displays these options:

**Line Color**

Specify the color of the gridlines. To edit the color, select the color indicator and select a color from the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/5735570054167-Menu-Tabs#Palette), or type the hexadecimal value of a color (for example, 0xff0000) in the text box.

**Line Style**

Specify the line style of the gridlines.

**Transparency**

Specify the transparency of the gridlines.

**Thickness**

Specify the thickness of the gridlines, in pixels.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Apply**

Select to apply all changes and leave the dialog box open.

**Help**

Select to view information about the dialog box.
