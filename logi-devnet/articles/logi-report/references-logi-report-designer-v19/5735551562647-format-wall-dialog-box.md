# Format Wall Dialog Box

You can use the Format Wall dialog box to [format the wall of a chart](https://devnet.logianalytics.com/hc/en-us/articles/5735507095959-Formatting-Chart-Wall-Floor). This topic describes the options in the dialog box.

Designer displays the Format Wall dialog box when you right-click any chart element and navigate to Format Walls > Format Wall on the shortcut menu, or double-click the wall of a chart.

![Format Wall dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898462513175/fmtwall.gif)

Designer displays these options:

**Wall**

You can specify the fill pattern of the chart wall in this box.

* **Fill**  
  Specify the fill pattern of the wall. To edit the fill pattern, select the color indicator and specify a color or effect using the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/5735570054167-Menu-Tabs#Palette). You can also type the hexadecimal value of a color (for example, 0xff0000) in the text box.
* **Transparency**  
  Specify the transparency of the fill pattern.

**Bound**

You can specify the fill pattern to compound with the basic fill that you define in the Wall box in this box.

* **Color**  
  Specify the fill pattern to compound with the basic fill. To edit the fill pattern, select the color indicator and specify a color or effect using the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/5735570054167-Menu-Tabs#Palette). You can also type the hexadecimal value of a color (for example, 0xff0000) in the text box.
* **Transparency**  
  Specify the transparency of the fill pattern.

**Border**

You can specify properties for the border of the chart wall in this box.

* **Color**  
  Specify the color of the border. To edit the color, select the color indicator and select a color from the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/5735570054167-Menu-Tabs#Palette), or type the hexadecimal value of a color (for example, 0xff0000) in the text box.
* **Line Style**  
  Select the line style of the border.
* **Transparency**  
  Specify the transparency of the border.
* **Thickness**  
  Specify the width of the border, in pixels.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Apply**

Select to apply all changes and leave the dialog box open.

**Help**

Select to view information about the dialog box.
