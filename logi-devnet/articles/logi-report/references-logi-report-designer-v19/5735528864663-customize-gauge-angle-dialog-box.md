# Customize Gauge Angle Dialog Box

You can use the Customize Gauge Angle dialog box to customize the angle of the gauge. This topic describes the options in the dialog box.

Designer displays the Customize Gauge Angle dialog box when you select Customized from the Angle drop-down list in the Circular Graph tab of the [Format Dial Gauge dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735523061015-Format-Dial-Gauge-Dialog-Box#Circular) or [Format Solid Gauge dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735551380887-Format-Solid-Gauge-Dialog-Box#Circular).

![Customize Gauge Angle dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898465018263/cstmzgugeagl.gif)

Designer displays these options:

**Angle**

You can specify the angle of the arcs in this box.

* **Start Angle**   
  Specify the start angle of the arcs.
* **Stop Angle**   
  Specify the stop angle of the arcs.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
