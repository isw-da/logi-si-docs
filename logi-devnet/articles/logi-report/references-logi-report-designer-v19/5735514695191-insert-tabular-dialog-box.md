# Insert Tabular Dialog Box

You can use the Insert Tabular dialog box to [insert a tabular](https://devnet.logianalytics.com/hc/en-us/articles/5735564433175-Working-with-Tabulars#Insert) into a report. This topic describes the options in the dialog box.

Designer displays the Insert Tabular dialog box when you navigate to Insert > Tabular, or drag the Tabular icon ![Tabular icon](https://devnet.logianalytics.com/hc/article_attachments/9898462046231/icon_tabular.gif) from the Components panel to the destination.

![Insert Tabular dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898462066071/insttblr.gif)

Designer displays these options:

**Rows**

Specify the number of rows to create in the tabular.

**Columns**

Specify the number of columns to create in the tabular.

**Width**

Specify the width of the tabular.

**Border**

Specify the length for the border of the tabular.

**OK**

Select to insert the tabular into the report and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
