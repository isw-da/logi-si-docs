# Add Group Dialog Box

You can use the Add Group dialog box to add a group when editing a [security entry](https://devnet.logianalytics.com/hc/en-us/articles/5735527007767-Working-with-RLS-and-CLS-at-Data-Source-Scope#Create) or [business view security](https://devnet.logianalytics.com/hc/en-us/articles/5735526952343-Configuring-Business-View-Security-in-a-Catalog) so as to define its permission on different resources. This topic describes the options in the dialog box.

Designer displays the Add Group dialog box when you select Add ![Add button](https://devnet.logianalytics.com/hc/article_attachments/9898403108631/btn_add.gif) and then select Add Group from the drop-down menu in the [Security dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735524608407-Security-Dialog-Box), [Edit Business View Security dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735565601559-Edit-Business-View-Security-Dialog-Box), or in the Security tab of the following dialog boxes: [Category Property dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735508509719-Category-Property-Dialog-Box), [New View Element dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735530525975-New-View-Element-Dialog-Box), or [Edit View Element dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735529270039-Edit-View-Element-Dialog-Box).

This dialog box contains the following tabs:

* [General Tab](#General)
* [Relation Tab](#Relation)

Designer displays these buttons in all the tabs:

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.

## General Tab

Use this tab to specify the general information of the group.

![Add Group dialog box - General](https://devnet.logianalytics.com/hc/article_attachments/9898533469079/addgroup_gen.gif)

**Group**

Specify the name of the group.

**Source**

This option shows the source of the group, which is read only.

**Parent Groups**

This option shows the parent groups of the current group, which is read only.

**Description**

Specify the description of the group.

## Relation Tab

Use this tab to specify the relationship between the group and the existing users and roles.

![Add Group dialog box - Relation](https://devnet.logianalytics.com/hc/article_attachments/9898533473303/addgroup_rel.gif)

**Users**

This box lists all the available users. Select a user to add it into the group.

**Roles**

This box lists all the available roles. Select a role to grant it to the group.
