# Select Component for Page Report Tab Dialog Box

You can use the Select Component for Page Report Tab dialog box to [create a page report tab](https://devnet.logianalytics.com/hc/en-us/articles/5735569787671-Creating-Reports#Tab) in an existing page report. This topic describes the options in the dialog box.

Designer displays the Select Component for Page Report Tab dialog box when you navigate to Home/File > New > Page Report Tab, or right-click an existing page report tab on the report tab bar and select Insert from the shortcut menu.

![Select Component for Page Report Tab dialog box](https://devnet.logianalytics.com/hc/article_attachments/7933647932567/slctcmpntpgrpttb.gif)

Designer displays these options:

**Report Title**

Specify the title of the new page report tab.

**Component box**

This box lists the components that you can select to create in the page report tab.

* **Banded**  
  Select to create a page report tab which contains a vertical banded object.
* **Table (Group Above)**  
  Select to create a page report tab which contains a table with group information above the detail row.
* **Table (Group Left)**  
  Select to create a page report tab which contains a table with group information left to the detail row.
* **Table (Group Left Above)**  
  Select to create a page report tab which contains a table with group information left above the detail row.
* **Summary Table**  
  Select to create a page report tab which contains a table with only group and summary information.
* **Chart**  
  Select to create a page report tab which contains a chart.
* **Crosstab**  
  Select to create a page report tab which contains a crosstab.
* **Horizontal Banded**  
  Select to create a page report tab which contains a horizontal banded object. Designer does not display this option if you are creating a report tab in a business view-based page report.
* **Mailing Label**  
  Select to create a page report tab which contains a banded object in the form of a mailing label layout. Designer does not display this option if you are creating a report tab in a business view-based page report.
* **Tabular**  
  Select to create a page report tab which contains a tabular component. Designer does not display this option if you are creating a report tab in a business view-based page report.
* **Blank**  
  Select to create a page report tab with no component in it.

**OK**

If you select a component other than "Blank", after selecting OK, Designer displays the corresponding component wizard for you to create the component in the page report tab; otherwise, Designer creates a blank page report tab.

**Cancel**

Select to quit creating the page report tab and close the dialog box.

**Help**

Select to view information about the dialog box.
