# Format Floor Dialog Box

You can use the Format Floor dialog box to [format the floor of a chart](https://devnet.logianalytics.com/hc/en-us/articles/5735507095959-Formatting-Chart-Wall-Floor). This topic describes the options in the dialog box.

Designer displays the Format Floor dialog box when you right-click any element of a 3-D chart and navigate to Format Walls > Format Floor on the shortcut menu, or double-click the floor of a 3-D chart.

![Format Floor dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898480704791/fmtflr.gif)

Designer displays these options:

**Floor**

You can specify the fill pattern of the chart floor in this box.

* **Fill**  
  Specify the fill pattern of the floor. To edit the fill pattern, select the color indicator and specify a color or effect using the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/5735570054167-Menu-Tabs#Palette). You can also type the hexadecimal value of a color (for example, 0xff0000) in the text box.
* **Transparency**  
  Specify the transparency of the fill pattern.

**Border**

You can specify properties for the border of the chart floor in this box.

* **Color**  
  Specify the color of the border. To edit the color, select the color indicator and select a color from the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/5735570054167-Menu-Tabs#Palette), or type the hexadecimal value of a color (for example, 0xff0000) in the text box.
* **Line Style**  
  Select the line style of the border.
* **Transparency**  
  Specify the transparency of the border.
* **Thickness**  
  Specify the width of the border, in pixels.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Apply**

Select to apply all changes and leave the dialog box open.

**Help**

Select to view information about the dialog box.
