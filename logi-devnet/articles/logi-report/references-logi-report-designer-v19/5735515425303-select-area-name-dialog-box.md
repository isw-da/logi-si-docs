# Select Area Name Dialog Box

You can use the Select Area Name dialog box to [define the map area name and tool tip](https://devnet.logianalytics.com/hc/en-us/articles/5735521118231-Using-Shape-Maps#Create). This topic describes the options in the dialog box.

Designer displays the Select Area Name dialog box when you specify to import areas from a .shp file.

![Select Area Name dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898477981591/slctareanm.gif)

Designer displays these options:

**Select area name from the ESRI shape file**

This box lists all the available fields in the selected ESRI shape file, which you can use to define the area name of the map object.

**Tips**

This box lists the fields that you add to get the area tool tip.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/9898403108631/btn_add.gif)**Add button**

Select to add the specified field to use its value in the tool tip.

![Trash Can button](https://devnet.logianalytics.com/hc/article_attachments/9898403114647/btn_trashcan.gif)**Remove button**

Select to delete the specified field from the tool tip.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/9898406165911/btn_mvup.gif)**Move Up button**

Select to move the specified field higher in the list to display its value in a forward position in the tool tip.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/9898423100695/btn_mvdwn.gif)**Move Down button**

Select to move the specified field lower in the list to display its value in a latter position in the tool tip.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
