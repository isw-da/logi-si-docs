# Edit Display Name Dialog Box

You can use the Edit Display Name dialog box to [customize the display names of data fields](https://devnet.logianalytics.com/hc/en-us/articles/5735569625367-Customizing-Field-Display-Names-for-Datasets-in-a-Report) in datasets created on query resources in order to help users identify the data fields easier at runtime. This topic describes the options in the dialog box.

Designer displays the Edit Display Name dialog box when you navigate to Report > Edit Display Name.

![Edit Display Name dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898458554775/edtdsplynm.gif)

Designer displays these options:

**Report Dataset**

This drop-down list contains all the datasets in the current report. Select the dataset the data fields of which you want to customize.

**Auto Sort**

Select to sort the data fields automatically.

**Resource Name**

This column shows the names of the data fields in the selected dataset.

**Display Name**

This column shows the display names you specify for the data fields in the selected dataset.

**Advanced**

Select to open the [Edit Display Name for Component dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735500381463-Edit-Display-Name-for-Component-Dialog-Box) to configure advanced display name settings.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
