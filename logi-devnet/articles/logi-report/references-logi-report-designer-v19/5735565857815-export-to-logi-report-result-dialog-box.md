# Export to Logi Report Result Dialog Box

You can use the Export to Logi Report Result dialog box to [export a report to the Logi Report Result format](https://devnet.logianalytics.com/hc/en-us/articles/5735525374615-Exporting-a-Report-to-Logi-Report-Result). This topic describes the options in the dialog box.

Designer displays the Export to Logi Report Result dialog box when you navigate to File > Export > To Logi Report Result.

![Export to Logi Report Result dialog box](https://devnet.logianalytics.com/hc/article_attachments/7933691756183/expt2rst.gif)

Designer displays these options:

**Save in**

Specify the destination directory where you want to save the result file.

**File name**

Specify the name of the result file.

**Files of type**

Select the type of the result file. You can export a page report to a .rst file and a web report to a .wst file.

**Result**

You can specify settings of the result file in this panel.

* **Zip**  
  Select to export the report to a zip file.
* **Precision Level**  
  Specify the precision level with which to export the selected report tabs.
* **Run Linked Report**  
  Select to generate the reports that you link with the report (not including the detail reports) in the result file.
* ![This image notes any new content for version 19 of the product. For more information please see the Release Notes or What's New topics.](https://devnet.logianalytics.com/hc/article_attachments/7933646501655/___newv19.png "New for version 19.")**Use Custom File Extension**  
  Select to use the extension you want for the result file. By default, Designer uses .rst/.wst. Select this option and type your file name with any extension, or no extension.
* **Report tab box**  
  This box lists all the report tabs in the report. Select the checkboxes to export the corresponding report tabs.

**Save**

Select to save the report into a Logi Report result file to the specific directory.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
