# Imported APE Properties

This topic describes the properties of an [Imported APE object](https://devnet.logianalytics.com/hc/en-us/articles/5735512889367-Using-Imported-APEs) in a catalog.

| Property Name | Description |
| --- | --- |
| Collection | Specifies the collection to execute the aggregation pipeline expression. You can reference a type-in [parameter](https://devnet.logianalytics.com/hc/en-us/articles/5735532413719-Working-with-Parameters) of String data type or [constant level formulas](https://devnet.logianalytics.com/hc/en-us/articles/5735525461399-Formula-Levels#CLF) in the current catalog data source in the format *@FieldName*, the default value or return value of which is the same as the name of an available collection to dynamically specify the collection. Data type: String |
| Connection Name | Specifies the connection to execute the aggregation pipeline expression. Data type: String |
| Database | Specifies the database that contains the selected collection and matches the aggregation pipeline expression. You can reference a type-in parameter of String data type or constant level formulas in the current catalog data source in the format *@FieldName*, the default value or return value of which is the same as the name of an available database to dynamically specify the database. Data type: String |
| Description | Specifies the description of the object. Data type: String |
| Name | Specifies the mapped name of the object in the catalog. Data type: String |
