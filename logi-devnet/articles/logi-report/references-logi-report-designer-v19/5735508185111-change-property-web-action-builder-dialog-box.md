# Change Property - Web Action Builder Dialog Box

You can use the Change Property - Web Action Builder dialog box to [build a Property web action](https://devnet.logianalytics.com/hc/en-us/articles/9898540480023-Defining-Web-Behaviors-of-Objects-in-a-Report#Property) on an object in a report, so as to change properties of the objects in the report when users trigger the event bound with the action at runtime. This topic describes the options in the dialog box.

Designer displays the Change Property - Web Action Builder dialog box when you select \*Property and then select OK in the [Web Action List dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735531755159-Web-Action-List-Dialog-Box).

![Change Property - Web Action Builder dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898533153431/chgptywbactnbld.gif)

Designer displays these options:

**Apply Action To**

This box lists all the objects in the report. Select an object to edit its property values.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/9898403108631/btn_add.gif)**Add button**

Select to add a new property line.

![Trash Can button](https://devnet.logianalytics.com/hc/article_attachments/9898403114647/btn_trashcan.gif)**Remove button**

Select to delete the specified property line.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/9898406165911/btn_mvup.gif)**Move Up button**

Select to move the specified property line higher in the list.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/9898423100695/btn_mvdwn.gif)**Move Down button**

Select to move the specified property line lower in the list.

**Properties**

This column shows the properties of the object you select to change.

**Value**

This column shows the values you specify for the properties.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
