# Special Function Dialog Box

You can use the Special Function dialog box to [specify a special function](https://devnet.logianalytics.com/hc/en-us/articles/5735498861207-Modifying-Charts#Group) to the category/series field of a chart, which is Numeric, String, Date, or Time data type. This topic describes the options in the dialog box.

Designer displays the Special Function dialog box when you select Special Function in the Display screen of the chart wizard.

![Special Function dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898477846935/spclfctn.gif)

Designer displays these options:

**Field**

This option shows to which field the special function is to be applied.

**Function**

Select the [special function](https://devnet.logianalytics.com/hc/en-us/articles/5735527697943-Grouping-Data-in-Tables#Function) for the field.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
