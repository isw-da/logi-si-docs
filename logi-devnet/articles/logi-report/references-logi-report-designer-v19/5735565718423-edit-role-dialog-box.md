# Edit Role Dialog Box

You can use the Edit Role dialog box to edit the specified role when configuring a [security entry](https://devnet.logianalytics.com/hc/en-us/articles/5735527007767-Working-with-RLS-and-CLS-at-Data-Source-Scope#Create) or [business view security](https://devnet.logianalytics.com/hc/en-us/articles/5735526952343-Configuring-Business-View-Security-in-a-Catalog). This topic describes the options in the dialog box.

Designer displays the Edit Role dialog box when you select a role and then select Edit ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/7933688838807/btn_edit.gif) in the [Security dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735524608407-Security-Dialog-Box), [Edit Business View Security dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735565601559-Edit-Business-View-Security-Dialog-Box), or in the Security tab of the following dialog boxes: [Category Property dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735508509719-Category-Property-Dialog-Box), [New View Element dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735530525975-New-View-Element-Dialog-Box) or [Edit View Element dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735529270039-Edit-View-Element-Dialog-Box).

This dialog box contains the following tabs:

* [General Tab](#General)
* [Relation Tab](#Relation)

Designer displays these buttons in all the tabs:

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.

## General Tab

Use this tab to specify the general information of the role.

![Edit Role dialog box - General tab](https://devnet.logianalytics.com/hc/article_attachments/7933728881559/edtrole_gen.gif)

**Role**

Specify the name of the role.

**Source**

This option shows the source of the role, which is read only.

**Parent Roles**

This option shows the parent roles of the current role, which is read only.

**Description**

Specify the description of the role.

## Relation Tab

Use this tab to specify the relationship between the role and the existing users and groups.

![Edit Role dialog box - Relation tab](https://devnet.logianalytics.com/hc/article_attachments/7933728908183/edtrole_rel.gif)

**Users**

This box lists all the available users. Select a user to add it into the role.

**Groups**

This box lists all the available groups. Select a group to add it into the role.
