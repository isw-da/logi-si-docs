# Go Down Dialog Box

You can use the Go Down dialog box to [define a Go Down web action](https://devnet.logianalytics.com/hc/en-us/articles/5735576827671-Delivering-Messages-Between-Library-Components#UGoDown) for the specified data component in a library component as a response to the user-defined message it receives at runtime. This topic describes the options in the dialog box.

Designer displays the Go Down dialog box when you select \*Go Down and select OK in the [Web Action List dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735531755159-Web-Action-List-Dialog-Box).

![Go Down dialog box](https://devnet.logianalytics.com/hc/article_attachments/7933633132183/godwn.gif)

Designer displays these options:

![Add button](https://devnet.logianalytics.com/hc/article_attachments/7933632750231/btn_add.gif)**Add button**

Select to add a new condition line. You can add one condition line at most.

![Trash Can button](https://devnet.logianalytics.com/hc/article_attachments/7933647176855/btn_trashcan.gif)**Remove button**

Select to delete the condition.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/7933647186583/btn_mvup.gif)**Move Up button**

Designer does not enable this button for building the Go Down web action.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/7933647199511/btn_mvdwn.gif)**Move Down button**

Designer does not enable this button for building the Go Down web action.

**Go Down On**

Specify the group on which to perform the go-down action.

**Use Hierarchy**

Specify the hierarchy for the go-down action to follow.

**By Value**

Specify by which value of the specified group to perform the go-down action, that is to perform the go-down action with the filter condition *SpecifiedGroupField=TheValue*.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
