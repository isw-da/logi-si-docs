# Line Pattern List Dialog Box

You can use the Line Pattern List dialog box to modify the line pattern for each chart line in a line chart. This topic describes the options in the dialog box.

Designer displays the Line Pattern List dialog box when you select Line Pattern List in the [Format Line dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735509343767-Format-Line-Dialog-Box#Fill).

![Line Pattern List dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898478906263/lineptrnlist.gif)

Designer displays these options:

**Patterns**

This box lists items to stand for the chart lines. Select the items one by one to set the pattern of each line.

* ![Add button](https://devnet.logianalytics.com/hc/article_attachments/9898403108631/btn_add.gif)**Add button**  
  Select to add a new item.
* ![Trash Can button](https://devnet.logianalytics.com/hc/article_attachments/9898403114647/btn_trashcan.gif)**Remove button**  
  Select to delete the specified item.
* ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/9898406165911/btn_mvup.gif)**Move Up button**  
  Select to move the specified item higher in the box.
* ![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/9898423100695/btn_mvdwn.gif)**Move Down button**  
  Select to move the specified item lower in the box.

**Line Style**

Select the style of the chart line which the specified item in the Patterns box represents.

**Thickness**

Specify the thickness of the chart line in pixels, which the specified item in the Patterns box represents.

**Advanced**

Select to open the [Advanced dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735528384919-Advanced-Dialog-Box) to specify the line pattern for certain values.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
