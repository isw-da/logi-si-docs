# Edit Style Group Dialog Box

You can use the Edit Style Group dialog box to [edit a style group](https://devnet.logianalytics.com/hc/en-us/articles/5735576944407-Working-with-XSD-Styles#EditGroup). This topic describes the options in the dialog box.

Designer displays the Edit Style Group dialog box when you right-click a style group and select Edit Group from the shortcut menu after selecting Style in the Catalog Manager.

![Edit Style Group dialog box](https://devnet.logianalytics.com/hc/article_attachments/7933659067031/edtstylgrp.gif)

Designer displays these options:

**Style Group Name**

This option shows the name of the style group.

**New Style**

Select to open the New Style dialog box to create a style.

* **Style Name**  
  Specify the name of the new style.
* **Style Type**  
  Specify the type of the new style.
* **Description**  
  Specify the description of the new style.
* **OK**  
  Select to create the style in this style group.
* **Cancel**  
  Select to quit creating the style.

**All Styles**

This box lists all the styles that you have not selected to apply to this group.

**Selected Styles**

This box lists the styles that you select to apply to this group.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/7933646833559/btn_addarrow.gif)**Add button**

Select to add the specified style in the All Styles box to use in the group.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/7933659051287/btn_rmvarrow.gif)**Remove button**

Select to remove the specified style from the group.

![Remove All button](https://devnet.logianalytics.com/hc/article_attachments/7933632592279/btn_rmvall.gif)**Remove All button**

Select to remove all the styles from the group.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
