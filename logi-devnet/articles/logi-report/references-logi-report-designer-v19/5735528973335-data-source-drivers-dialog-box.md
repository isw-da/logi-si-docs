# Data Source Drivers Dialog Box

You can use the Data Source Drivers dialog box to add or remove a data source driver [for cached query result](https://devnet.logianalytics.com/hc/en-us/articles/5735586187287-Working-with-Cached-Query-Results#Manage) into or from a catalog. This topic describes the options in the dialog box.

Designer displays the Data Source Drivers dialog box when you select Data Source Drivers on the Catalog Manager toolbar.

![Data Source Drivers dialog box](https://devnet.logianalytics.com/hc/article_attachments/9898475484695/dtsrcdrvr.gif)

Designer displays these options:

**Data Source Drivers**

This box lists all the data source drivers you have added.

**New**

Select to open the [Data Source Editor dialog box](https://devnet.logianalytics.com/hc/en-us/articles/5735508584471-Data-Source-Editor-Dialog-Box) to add a new data source driver.

**Remove**

Select to remove the specified data source driver.

**Edit**

Select to edit the specified data source driver.

**OK**

Select to apply your settings and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
