# Shape Map Canvas Setup Dialog Box

You can use the Shape Map Canvas Setup dialog box to change the width and height of a shape map object. This topic describes the options in the dialog box.

Designer displays the Shape Map Canvas Setup dialog box when you select Menu > File > Canvas Setup, or Canvas Setup on the toolbar in the [Shape Map Editor dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010059842-Shape-Map-Editor-Dialog-Box).

![Shape Map Canvas Setup dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856909079/mpcnvsstup.gif)

You see the following options in the dialog box:

**Width**

Specify the width of the map object.

**Height**

Specify the height of the map object.

**Automatic Filling Labels**

Select if you do not want to display the label for an area when you change the size of the area to be smaller than the label.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
