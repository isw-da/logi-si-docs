# Image Button Dialog Box

You can use the Image Button dialog box to [specify the source of an image button](https://devnet.logianalytics.com/hc/en-us/articles/1500010095041-Using-Advanced-Web-Controls#Navigate). This topic describes the options in the dialog box.

Designer displays the Image Button dialog box when you right-click a button in a navigation control and select Button Type > Image Button from the shortcut menu.

![Image Button dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856921239/imgbtn.gif)

You see the following options in the dialog box:

**URL**

Specify the source of the image button. You can select the Browse button to specify the path or type the URL in the text box.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
