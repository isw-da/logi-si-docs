# Filter View Dialog Box

You can use the Filter View dialog box to select some elements to [exclude from the view in the Catalog Manager](https://devnet.logianalytics.com/hc/en-us/articles/1500010056642-Managing-the-Data-Resources-in-a-Catalog#Filter), if you think they are not necessary to display. This topic describes the options in the dialog box.

Designer displays the Filter View dialog box when you select Filter View on the Catalog Manager.

![Filter View dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856971031/fltrvw.gif)

You see the following options in the dialog box:

**Filter Expression**

Specify the filter condition to determine the resource names that you want to display.

**Case Sensitive**

Select to distinguish between uppercase and lowercase characters.

**Select the elements to include in the view**

The box lists all the elements in the Catalog Manager. Select the elements you want to include in the view.

**Select All**

Select to select all the elements in the list box to include them in the view.

**Deselect All**

Select to clear all the elements in the list box to exclude them from the view.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
