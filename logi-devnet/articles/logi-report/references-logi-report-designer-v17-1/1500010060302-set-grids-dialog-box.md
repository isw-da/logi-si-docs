# Set Grids Dialog Box

The Set Grids dialog box helps you to specify the grid settings in the design area. It appears when you select View > Grids > Grid Size.

![Set Grids dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848498583/setgrid.gif)

The following are details about options in the dialog box:

**Size**

Specifies the size of the grids in the design area.

**Color**

Specifies the color of the grids. Select a color from the drop-down list or select Custom to customize a color in the [Pick a Color dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010060102-Pick-a-Color-Dialog-Box#Color).

**Show Grids**

Specifies whether to show grids in the report.

**Snap to Grids**

Specifies whether to snap and lock objects with the grids when moving and placing objects around.

**OK**

Accepts changes and closes the dialog box.

**Apply**

Applies the grid settings and closes the dialog box.

**Cancel**

Does not retain the changes and closes the dialog box.

**Help**

Displays the help document about this feature.
