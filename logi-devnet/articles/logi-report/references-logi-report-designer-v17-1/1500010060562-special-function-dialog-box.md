# Special Function Dialog Box

The Special Function dialog box helps you to [specify a special function](https://devnet.logianalytics.com/hc/en-us/articles/1500010094661-Modifying-Charts#Group) to the field displayed on the category/series axis of a chart, which is of Numeric/String/Date/Time data type. It appears when you select the Special Function button in the Display screen of the chart wizard.

![Special Function dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848493463/spclfctn.gif)

The following are details about options in the dialog box:

**Field**

Displays on which field the special function will be applied.

**Function**

Specifies the special function for the field.

**OK**

Accepts the changes and closes the dialog box.

**Cancel**

Discards the changes and closes the dialog box.

**Help**

Displays the help document about this feature.
