# Create Connection Dialog Box

You can use the Create Connection dialog box to create a connection to connect a catalog to one of these databases: [Oracle, SQL Server, MySQL, InterSystems IRIS, PostgreSQL](https://devnet.logianalytics.com/hc/en-us/articles/1500010095221-Creating-JDBC-Connections-via-Plug-ins), [JDBC](https://devnet.logianalytics.com/hc/en-us/articles/1500010057642-Connecting-via-JDBC-Connections), [JSON](https://devnet.logianalytics.com/hc/en-us/articles/1500010095241-Connecting-via-JSON-Connections), [XML](https://devnet.logianalytics.com/hc/en-us/articles/1500010095381-Working-with-XML-Connections-in-a-Catalog), [SOAP Web Service](https://devnet.logianalytics.com/hc/en-us/articles/1500010057862-Connecting-via-SOAP-Web-Service-Connections), [MongoDB](https://devnet.logianalytics.com/hc/en-us/articles/1500010057802-Working-with-MongoDB-Connections-in-a-Catalog), and [Hive](https://devnet.logianalytics.com/hc/en-us/articles/1500010057622-Connecting-via-Hive-Connections). This topic describes the options in the dialog box.

Designer displays the Create Connection dialog box when you select an item in the Connect category on the [Start Page](https://devnet.logianalytics.com/hc/en-us/articles/1500010062942-Start-Page).

![Create Connection dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848611863/crtcnnctn.gif)

You see the following options in the dialog box:

**Existing Catalog**

Select to create the connection in an existing catalog.

* **Catalog**  
  The option shows the current catalog. You can select another one by selecting the ellipsis button ![Ellipsis button](https://devnet.logianalytics.com/hc/article_attachments/4404848397591/btn_ellipsis.gif).
* **Data Source**  
  Select the catalog data source in which to create the connection.

**New Catalog**

Select to create the connection in a new catalog.

* **Directory**  
  Specify the directory where to save the new catalog. You can type the directory manually in the text box or select ![Ellipsis button](https://devnet.logianalytics.com/hc/article_attachments/4404848397591/btn_ellipsis.gif) to select it.
* **Catalog**  
  Specify the name of the new catalog.

**OK**

Select to close the dialog box and open one of the following dialog boxes:

* [Connect to Oracle dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010095781-Connect-to-Oracle-Dialog-Box)
* [Connect to MySQL dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010095761-Connect-to-MySQL-Dialog-Box)
* [Connect to SQL Server dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010095801-Connect-to-SQL-Server-Dialog-Box)
* [Connect to InterSystems IRIS dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010058262-Connect-to-InterSystems-IRIS-Dialog-Box)
* [Connect to PostgreSQL dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010058282-Connect-to-PostgreSQL-Dialog-Box)
* [Get JDBC Connection Information dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010097761-Get-JDBC-Connection-Information-Dialog-Box)
* [JSON Connection Wizard dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010097801-JSON-Connection-Wizard-Dialog-Box)
* [XML Connection Wizard dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010099221-XML-Connection-Wizard-Dialog-Box)
* [SOAP Web Service Data Source dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010099201-SOAP-Web-Service-Data-Source-Dialog-Box)
* [MongoDB Connection Wizard dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010097901-MongoDB-Connection-Wizard-Dialog-Box)

**Cancel**

Select to cancel creating the connection and close the dialog box.

**Help**

Select to view information about the dialog box.
