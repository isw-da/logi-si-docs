# New Dataset Dialog Box

You can use the New Dataset dialog box to [create a dataset](https://devnet.logianalytics.com/hc/en-us/articles/1500010062542-Creating-and-Managing-Datasets#Create) for a query-based page report. This topic describes the options in the dialog box.

Designer displays the New Dataset dialog box when you select the <New Dataset...> item in the Data tab of the report wizard, select the <Choose Data From...> item from the dataset drop-down list in the Data panel for a page report in the main window, or select the New button in the [Manage Datasets dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010059762-Manage-Datasets-Dialog-Box).

![New Dataset dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848437527/nwdtst.gif)

You see the following options in the dialog box:

**Choose Data from**

Select a data source in the current catalog from the drop-down list and then a data resource in the list box on which to create the dataset. After you select the catalog data source, Designer displays all the data resources that you have predefined in the data source. You can also select the first item in each data resource node to create your required data resource for the dataset.

**New Dataset Name**

Specify the name of the dataset.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
