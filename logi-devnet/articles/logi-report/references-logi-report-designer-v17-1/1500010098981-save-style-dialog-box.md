# Save Style Dialog Box

The Save Style dialog box helps you to edit and [save the style in an XSD style you create](https://devnet.logianalytics.com/hc/en-us/articles/1500010101461-Working-with-XSD-Styles#Create) into a catalog.

![Save Style dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848390807/svstyl.gif)

The following are details about options in the dialog box:

**Style Group Name**

Specifies the name of the style group.

**Style Name**

Specifies the name of the saved style.

**All Properties**

Lists all the properties available. Select the properties you want to save in the style from the box.

**Selected Properties**

Lists the selected properties.

**![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404848391191/btn_addarrow.gif)**

Adds a specified property to the style.

![Add All button](https://devnet.logianalytics.com/hc/article_attachments/4404848391575/btn_addall.gif)

Adds all the properties listed in the All Properties box to the style.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404848391831/btn_rmvarrow.gif)

Removes the specified property from the Selected Properties box, so that it will not be applied to the style.

![Remove All button](https://devnet.logianalytics.com/hc/article_attachments/4404848392215/btn_rmvall.gif)

Removes all the properties from the Selected Properties box, so that they will not be applied to the style.

**Save**

Saves the style and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
