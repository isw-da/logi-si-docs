# Insert UDO Dialog Box

You can use the Insert UDO dialog box to [insert a user-defined object](https://devnet.logianalytics.com/hc/en-us/articles/1500010095021-Inserting-UDOs-into-a-Page-Report) into a banded object in a query-based page report. This topic describes the options in the dialog box.

Designer displays the Insert UDO dialog box when you select Insert > UDO, or drag the UDO icon ![UDO icon](https://devnet.logianalytics.com/hc/article_attachments/4404856917655/icon_udo.gif) from the Components panel to the destination.

![Insert UDO dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856917911/instudo.gif)

You see the following options in the dialog box:

**UDO**

The drop-down list contains the UDOs that you have predefined. Select the UDO you want to insert into the report.

**OK**

Select to insert the specified UDO into the report and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.
