# Page Report Option Dialog Box

You can use the Page Report Option dialog box to predefine the options for directly running a report in the Page Report format on Server. This topic describes the options in the dialog box.

Designer displays the Page Report Option dialog box when you set the Default Format for Viewing Report property of a report tab to "Page Report" and then select the ellipsis button ![Ellipsis button](https://devnet.logianalytics.com/hc/article_attachments/4404856823191/btn_ellipsis1.gif) in the value cell in the Report Inspector.

![Page Report Option dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856903575/dhtmloptn.gif)

You see the following options in the dialog box:

**Resolution**

Specify the resolution of the report to zoom in/out, in DPI. Designer obtains the default value from your operation system, which is the resolution of your monitor, for example, 72 DPI on Unix or 96 DPI on Windows. You can set higher/lower value to zoom in/out.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
