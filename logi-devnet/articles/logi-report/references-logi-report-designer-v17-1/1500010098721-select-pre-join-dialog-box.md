# Select Pre-join Dialog Box

The Select Pre-join dialog box helps you to [define new pre-join paths](https://devnet.logianalytics.com/hc/en-us/articles/1500010094041-Editing-Pre-joins-in-a-Catalog#Define) in the current catalog. It appears when you select the New button in the [Edit Pre-join Paths dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010058822-Edit-Pre-join-Paths-Dialog-Box), type a path name and then select OK.

![Select Pre-join dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848494231/slctprjn.gif)

The following are details about options in this dialog box:

**All Joins**

Lists all the pre-joins in the connection.

**Joins in <path name>**

Lists all joins that are added to the new path.

**Hide Joins Added in Other Paths**

Specifies whether to hide those joins that have been added in other paths.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404848391191/btn_addarrow.gif)

Adds a join to the new path.

![Add All button](https://devnet.logianalytics.com/hc/article_attachments/4404848391575/btn_addall.gif)

Adds all listed join from the new path.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404848391831/btn_rmvarrow.gif)

Removes a join from the path.

![Remove All button](https://devnet.logianalytics.com/hc/article_attachments/4404848392215/btn_rmvall.gif)

Removes all joins from the path.

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
