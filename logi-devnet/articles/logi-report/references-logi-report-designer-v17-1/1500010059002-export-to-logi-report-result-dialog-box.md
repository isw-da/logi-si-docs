# Export to Logi Report Result Dialog Box

You can use the Export to Logi Report Result dialog box to [export a report to the Logi Report Result format](https://devnet.logianalytics.com/hc/en-us/articles/1500010060942-Exporting-to-Logi-Report-Result). This topic describes the options in the dialog box.

Designer displays the Export to Logi Report Result dialog box when you select File > Export > To Logi Report Result.

![Export to Logi Report Result dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856876439/expt2rst.gif)

You see the following options in the dialog box:

**Save in**

Specify the destination directory where you want to save the result file.

**File Name**

Specify the name of the result file.

**Files of type**

Select the type of the result file. You can export a page report to a .rst file and a web report to a .wst file.

**Report tab box**

The box lists all the report tabs in the report. Select the checkboxes to export the corresponding report tabs.

**Zip**

Select to export the report to a zip file.

**Precision Level**

Specify the precision level with which to export the selected report tabs.

**Run Linked Report**

Select to generate the reports that you link with the report (not including the detail reports) in the result file.

**Save**

Select to save the report into a Logi Report result file to the specific directory.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
