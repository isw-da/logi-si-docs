# Select Field Dialog Box

You can use the Select Field dialog box to [insert a field](https://devnet.logianalytics.com/hc/en-us/articles/1500010101261-Adding-Links-in-Reports#URL) into the URL or e-mail address that you link an object to, or [insert a field](https://devnet.logianalytics.com/hc/en-us/articles/1500010101161-Creating-Bursting-Reports#MailSettings) in the e-mail subject or comments when you specify to distribute a bursting report to e-mail. This topic describes the options in the dialog box.

Designer displays the Select Field dialog box when you select the URL or E-mail link type and select the Add Dynamic Field button in the [Insert Link dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010097661-Insert-Link-Dialog-Box) or [Edit Link dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010096301-Edit-Link-Dialog-Box), or select the Add Dynamic Field button in the [E-mail Settings dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010096541-E-mail-Settings-Dialog-Box).

![Select Field dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848413207/slctfld.gif)

You see the following options in the dialog box:

**Data fields box**

The box lists all the available data fields that you can select to insert into the URL, e-mail address, or e-mail subject/comments to compose dynamic information. Select the field you need. You should make sure the specified field contains the appropriate data to compose the information.

**![Marks content and feature updates for Logi Report v17.1. New feature new content.](https://devnet.logianalytics.com/hc/article_attachments/4404856832151/___newv17.1.jpg "New for Logi Report v17.1.")All Available Values**

Designer displays the option when you use the dialog box to insert a field into the URL that an object is linked to and the object is in a chart, table, banded object, or crosstab. Select it if you want to display in the URL all the runtime values of the selected field, instead of only one value.

**OK**

Applies the selected field and closes the dialog.

**Cancel**

Cancels the selection and exits the dialog.

**Help**

Select to view information about the dialog box.
