# Change Property - Web Behavior Dialog Box

You can use the Change Property-Web Behavior dialog box to [apply the Change Property web action to a web control](https://devnet.logianalytics.com/hc/en-us/articles/1500010057542-Using-Basic-Web-Controls-in-the-Configuration-Panel-of-a-Library-Component#Property) in a library component, so as to change the properties of a specified object in the library component at runtime. This topic describes the options in the dialog box.

Designer displays the Change Property-Web Behavior dialog box when you right-click a web control in the configuration panel of a library component and select Web Behaviors > Change Property from the shortcut menu.

![Change Property - Web Behavior dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848630167/chgptywbbhvr.gif)

You see the following options in the dialog box:

**Apply Action To**

The drop-down lists the objects in the report. Select the object to which you want to apply the Change Property action.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404848403095/btn_add.gif)**Add button**

Select to add a new property line.

![Trash Can button](https://devnet.logianalytics.com/hc/article_attachments/4404848403351/btn_trashcan.gif)**Remove button**

Select to delete the specified property line.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404848403735/btn_mvup.gif)**Move Up button**

Select to move the specified property line higher in the list.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404856822935/btn_mvdwn.gif)**Move Down button**

Select to move the specified property line lower in the list.

**Properties**

The column shows the properties the values of which you select to change. You can select from the properties supported by the object you select from the Apply Action To drop-down list.

**Value**

The column shows the values that you specify for the properties. You can type the value by yourself, or use a web control to obtain the value at runtime.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
