# Time Zone and Locale Options Dialog Box

The Time Zone and Locale Options dialog box helps you to specify the default time zone and locale for an XML instance. It appears when you select the Time Zone and Locale button in the [XML Connection Wizard dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010099221-XML-Connection-Wizard-Dialog-Box).

![Time Zone and Locale Options dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856888343/tmznlcl.gif)

The following are details about options in the dialog box:

**Default Time Zone**

Specifies the default time zone, in order to apply it to the Time data type, which is not specified for them in the XML instance. If the time zone has been specified in the XML instance, the time zone you set here will be ignored.

**Locale**

Specifies the locale for the XML instance.

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Does not retain changes and closes the dialog box.

**Help**

Displays the help document about this feature.
