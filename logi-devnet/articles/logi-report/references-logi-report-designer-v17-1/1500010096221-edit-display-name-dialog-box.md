# Edit Display Name Dialog Box

You can use the Edit Display Name dialog box to [customize the display names of data fields](https://devnet.logianalytics.com/hc/en-us/articles/1500010062542-Creating-and-Managing-Datasets#Customize) in datasets created on query resources in order to help the report users identify the data fields easier at runtime. This topic describes the options in the dialog box.

Designer displays the Edit Display Name dialog box when you select Report > Edit Display Name.

![Edit Display Name dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848439447/edtdsplynm.gif)

You see the following options in the dialog box:

**Report Dataset**

The drop-down list contains all the datasets in the current report. Select the dataset the data fields of which you want to customize.

**Auto Sort**

Select to sort the data fields automatically.

**Resource Name**

The column shows the names of the data fields in the selected dataset.

**Display Name**

The column shows the display names you specify for the data fields in the selected dataset.

**Advanced**

Select to open the [Edit Display Name for Component dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010096241-Edit-Display-Name-for-Component-Dialog-Box) to configure advanced display name settings.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
