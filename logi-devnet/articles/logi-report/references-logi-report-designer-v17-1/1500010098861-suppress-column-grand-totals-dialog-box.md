# Suppress Column Grand Totals Dialog Box

The Suppress Column Grand Totals dialog box helps you to specify whether or not to show the grand total column for a common crosstab, or to customize the grand totals of which column compound groups will be suppressed and which will be shown for a compound crosstab. It appears when you select the ellipsis button ![Ellipsis button](https://devnet.logianalytics.com/hc/article_attachments/4404856823191/btn_ellipsis1.gif) next to the Suppress Column Grand Totals option in the Layout screen of the crosstab wizard.

![Suppress Column Grand Totals dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856893975/sprsclmngrdtl.gif)

The following are details about options in the dialog box:

**Same for all**

It is meaningful to use this option for a compound crosstab. If it is selected, the setting for the grand total of the first column compound group in below box will be applied to all the grand totals of the column compound groups in the crosstab.

**Name**

Lists all the column compound groups in the crosstab. A common crosstab has only one column compound group.

**Suppress**

Specifies whether the grand total for a column compound group will be suppressed. To specify the value, select in the value cell and select true or false from the drop-down list.

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
