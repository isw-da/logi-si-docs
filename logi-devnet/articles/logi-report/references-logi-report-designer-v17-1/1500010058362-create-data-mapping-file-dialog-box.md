# Create Data Mapping File Dialog Box

You can use the Create Data Mapping File dialog box to [create a data mapping file](https://devnet.logianalytics.com/hc/en-us/articles/1500010061222-Mapping-Data-Values-to-Different-Languages) that saves information of a certain language. This topic describes the options in the dialog box.

Designer displays the Create Data Mapping File dialog box when you select the New button in the [Data Mapping Editor dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010058582-Data-Mapping-Editor-Dialog-Box).

![Create Data Mapping File dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856863767/crtdtmpfile.gif)

You see the following options in the dialog box:

**File Name**

Specify the name of the data mapping file, which will be saved as filename\_language\_locale.properties.

**Language**

Select the language of the data mapping file. One language corresponds to one data mapping file.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
