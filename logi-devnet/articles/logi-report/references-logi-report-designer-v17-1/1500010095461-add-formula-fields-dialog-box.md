# Add Formula Fields Dialog Box

You can use the Add Formula Fields dialog box to [add formulas into a query to be columns](https://devnet.logianalytics.com/hc/en-us/articles/1500010062582-Creating-Queries-in-a-Catalog#FormulaField) of the query. This topic describes the options in the dialog box.

Designer displays the Add Formula Fields dialog box when you select Menu > Column > Add Formula Fields in the [Query Editor dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010098321-Query-Editor-Dialog-Box).

![Add Formula Fields dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848436631/addfmlfld.gif)

You see the following options in the dialog box:

**Formulas**

The box lists all the available formulas that you can add to the query.

**Expression**

The box shows the expression of the selected formula.

**Add**

Select to add the specified formula to the query.

**Close**

Select to exit the dialog box.

**Help**

Select to view information about the dialog box.
