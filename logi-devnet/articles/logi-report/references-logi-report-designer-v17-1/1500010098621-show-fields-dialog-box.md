# Show Fields Dialog Box

The Show Fields dialog box helps you to select the fields the values of which to show as the hint in a heat map, which will be displayed when you hover the mouse pointer over the innermost rectangles of the heat map in Web Report Studio and JDashboard at runtime on Server. It appears when you select Customized from the drop-down list of the [Show Fields](https://devnet.logianalytics.com/hc/en-us/articles/1500010100301-Chart-Paper-Properties#ShowFields) property for the chart paper object of a heat map in the Report Inspector.

![Show Fields dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848498071/shwfld.gif)

The following are details about options in the dialog box:

**Resources**

Lists the available fields from which hint can be got. The available fields are the same as those that have been used in the heat map.

Select a field and select ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404848391191/btn_addarrow.gif) to add it into the right box to make its value shown as the hint. Repeat this to add more fields. The names of the selected fields will be displayed in the value cell of the Show Fields property, separated by ",".

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
