# Go Up - Web Action Builder Dialog Box

You can use the Go Up-Web Action Builder dialog box to [build a Go Up web action](https://devnet.logianalytics.com/hc/en-us/articles/1500010057562-Using-Basic-Web-Controls-in-a-Report#GoUp) on an object, so that the report users can drill data of the specified data components in a report to upper-level groups according to the predefined hierarchies, when they trigger the event bound with the action on the object at runtime. This topic describes the options in the dialog box.

Designer displays the Go Up-Web Action Builder dialog box when you select \*Go Up and then select OK in the [Web Action List dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010060822-Web-Action-List-Dialog-Box)

![Go Up - Web Action Builder dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848528279/goupwbactnbld.gif)

You see the following options in the dialog box:

**Apply Action To**

The box lists the data components in the report. Select a data component to define the go-up condition for it.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404848403095/btn_add.gif)**Add button**

Select to add a new condition line. You can add one condition at most.

![Trash Can button](https://devnet.logianalytics.com/hc/article_attachments/4404848403351/btn_trashcan.gif)**Remove button**

Select to delete the condition.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404848403735/btn_mvup.gif)**Move Up button**

Designer does not enable this button for building the Go Up web action.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404856822935/btn_mvdwn.gif)**Move Down button**

Designer does not enable this button for building the Go Up web action.

**Go Up On**

Specify the group on which to perform the go-up action for the selected data component.

**Use Hierarchy**

Specify the hierarchy for the go-up action to follow for the selected data component.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
