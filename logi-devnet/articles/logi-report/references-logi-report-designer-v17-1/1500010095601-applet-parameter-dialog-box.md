# Applet Parameter Dialog Box

You can use the Applet Parameter dialog box to [insert an applet](https://devnet.logianalytics.com/hc/en-us/articles/1500010094821-Working-with-Multimedia-Objects#Applet) into your report. This topic describes the options in the dialog box.

Designer displays the Applet Parameter dialog box when you select Insert > Multimedia Objects > Applet, or drag Applet from the Components panel to a query-based page report.

![Applet Parameter dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404857011991/apltprm.gif)

You see the following options in the dialog box:

**Code**

Specify the name for the class file which is contained in this archive file.

**Plug-in Page**

Specify the URL from which you can download JDK to run this applet.

**Archive**

Specify the position, where locates the resources that can make the applet run in this report properly. Select the Browse button to locate the position.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
