# Suppress Column Subtotals Dialog Box

The Suppress Column Subtotals dialog box helps you to customize which subtotals of the column fields in a crosstab will be suppressed and which will be shown. It appears when you select the ellipsis button ![Ellipsis button](https://devnet.logianalytics.com/hc/article_attachments/4404856823191/btn_ellipsis1.gif) next to the Suppress Column Subtotals option in the Layout screen of the crosstab wizard.

![Suppress Column Subtotal dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856893719/sprsclmnsbtl.gif)

The following are details about options in the dialog box:

**Same for all**

If the option is selected, the setting for the subtotal of the first column field in below box will be applied to all the subtotals of the column fields in the crosstab.

**Name**

Lists the column fields excluding the one on the topmost column group level; for a compound crosstab the column fields excluding the one on the topmost column group level in each column compound group.

**Suppress**

Specifies whether the subtotal for a column field will be suppressed. To specify the value, select in the value cell and select true or false from the drop-down list. The value can also be [controlled by a constant level formula](https://devnet.logianalytics.com/hc/en-us/articles/1500010061002-Using-Formulas-to-Control-Object-Properties).

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
