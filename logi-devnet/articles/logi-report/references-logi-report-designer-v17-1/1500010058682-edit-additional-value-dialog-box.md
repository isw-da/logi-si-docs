# Edit Additional Value Dialog Box

You can use the Edit Additional Value dialog box to add [additional values](https://devnet.logianalytics.com/hc/en-us/articles/1500010057042-Inserting-Charts-in-a-Report#Additional) to display in a chart or modify the existing additional values. This topic describes the options in the dialog box.

Designer displays the Edit Additional Value dialog box when you select Constant Value/Average Value in the Resources box in the Display screen of the chart wizard and select ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404848391191/btn_addarrow.gif) to add it to the value box, or select an additional value in the value box and select ![Edit button](https://devnet.logianalytics.com/hc/article_attachments/4404856829719/btn_edit.gif) when you create or modify a chart, and provides you with different options in the dialog box according to different value types: [constant value](#Constant) or [average value](#Average).

---

For a constant value, Designer displays the following options in the dialog box:

![Edit Additional Value dialog box - Constant Value](https://devnet.logianalytics.com/hc/article_attachments/4404848588567/edtaddtnvlu_cnst.gif)

**Name**

Specify the display name of the constant value.

**Value**

Specify the constant value. Its data type must be numerical.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.

---

For an average value, Designer displays the following options in the dialog box:

![Edit Additional Value dialog box- Average Value](https://devnet.logianalytics.com/hc/article_attachments/4404848588823/edtaddtnvlu_avg.gif)

**Name**

Specify the display name of the average value.

**Based On**

Select the summary or formula field based on which to calculate the average value.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
