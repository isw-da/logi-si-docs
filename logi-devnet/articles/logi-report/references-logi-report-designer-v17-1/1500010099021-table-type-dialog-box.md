# Table Type Dialog Box

The Table Type dialog box helps you to choose a type for the [table](https://devnet.logianalytics.com/hc/en-us/articles/1500010057422-Inserting-Tables-in-a-Report) you want to create. It appears when you select Insert > Table.

![Table Type dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856891287/tbltype.gif)

The following are details about options in the dialog box:

**Table (Group Above)**

Creates a table with group information above the detail row.

**Table (Group Left)**

Creates a table with group information left to the detail row.

**Table (Group Left Above)**

Creates a table with group information left above the detail row.

**Summary Table**

Creates a table with only group and summary information.

**OK**

Applies the changes and opens the [Create Table](https://devnet.logianalytics.com/hc/en-us/articles/1500010095941-Create-Table-Dialog-Box) wizard to create the table.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
