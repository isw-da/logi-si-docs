# Change Property - Web Action Builder Dialog Box

You can use the Change Property-Web Action Builder dialog box to [build a Change Property web action](https://devnet.logianalytics.com/hc/en-us/articles/1500010057562-Using-Basic-Web-Controls-in-a-Report#Property) on an object in a report, so as to change the properties of the objects in the report when the report users trigger the event bound with the action at runtime. This topic describes the options in the dialog box.

Designer displays the Change Property-Web Action Builder dialog box when you select \*Property and then select OK in the [Web Action List dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010060822-Web-Action-List-Dialog-Box).

![Change Property - Web Action Builder dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848630423/chgptywbactnbld.gif)

You see the following options in the dialog box:

**Apply Action To**

The box lists all the objects in the report. Select an object to edit its property values.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404848403095/btn_add.gif)**Add button**

Select to add a new property line.

![Trash Can button](https://devnet.logianalytics.com/hc/article_attachments/4404848403351/btn_trashcan.gif)**Remove button**

Select to delete the specified property line.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404848403735/btn_mvup.gif)**Move Up button**

Select to move the specified property line higher in the list.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404856822935/btn_mvdwn.gif)**Move Down button**

Select to move the specified property line lower in the list.

**Properties**

The column shows the properties the values of which you specify to change.

**Value**

The column shows the values that you specify for the properties.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
