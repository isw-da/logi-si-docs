# Download Customized Control Dialog Box

You can use the Download Customized Control dialog box to [import customized control files](https://devnet.logianalytics.com/hc/en-us/articles/1500010062722-Publishing-and-Downloading-Resources#DownCtrl) from a started Server. This topic describes the options in the dialog box.

Designer displays the Download Customized Control dialog box when you do either of the following:

* Select File > Download > Download Customized Control from Server.
* Select File > Download > Download Customized Control from Server, specify the connection and user information in the [Connect to Logi Report Server dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010095741-Connect-to-Logi-Report-Server-Dialog-Box) and select Connect.

![Download Customized Control dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848423575/dwldcstmzdctrl.gif)

You see the following options in the dialog box:

Left box

The box lists all the customized control files that you can download.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404848391191/btn_addarrow.gif)**Add button**

Select to add the specified customized control files from the left box to the right box for downloading.

![Add All button](https://devnet.logianalytics.com/hc/article_attachments/4404848391575/btn_addall.gif)**Add All button**

Select to add all the customized control files in the left box to the right box for downloading.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404848391831/btn_rmvarrow.gif)**Remove button**

Select to remove the specified customized control files from the right box.

![Remove All button](https://devnet.logianalytics.com/hc/article_attachments/4404848392215/btn_rmvall.gif)**Remove All button**

Select to remove all the customized control files form the right box.

Right box

The box lists the customized control files that you select to download.

**OK**

Select to exit the dialog box and start downloading. If a downloaded customized control file uses the same name as a customized control file on your local, the former replaces the latter directly without prompting. Make sure that you have properly backup the customized control files at the Designer side before downloading from Server.

**Cancel**

Select to cancel downloading the customized control files from Server and close the dialog box.

**Help**

Select to view information about the dialog box.
