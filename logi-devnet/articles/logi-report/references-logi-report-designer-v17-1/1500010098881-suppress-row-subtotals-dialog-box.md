# Suppress Row Subtotals Dialog Box

The Suppress Row Subtotals dialog box helps you to customize which subtotals of the row fields in a crosstab will be suppressed, and which will be shown. It appears when you select the ellipsis button ![Ellipsis button](https://devnet.logianalytics.com/hc/article_attachments/4404856823191/btn_ellipsis1.gif) next to the Suppress Row Subtotals option in the Layout screen of the crosstab wizard.

![Suppress Row Subtotal dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848492951/sprsrowsbtl.gif)

The following are details about options in the dialog box:

**Same for all**

If the option is selected, the setting for the subtotal of the first row field in below box will be applied to all the subtotals of the row fields in the crosstab.

**Name**

Lists the row fields excluding the one on the topmost row group level; for a compound crosstab the row fields excluding the one on the topmost row group level in each row compound group.

**Suppress**

Specifies whether the subtotal for a row field will be suppressed. To specify the value, select in the value cell and select true or false from the drop-down list. The value can also be [controlled by a constant level formula](https://devnet.logianalytics.com/hc/en-us/articles/1500010061002-Using-Formulas-to-Control-Object-Properties).

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
