# Format Cell Dialog Box

You can use the Format Cell dialog box to specify [the border properties for a selected table cell](https://devnet.logianalytics.com/hc/en-us/articles/1500010057442-Modifying-Tables#Cell). This topic describes the options in the dialog box.

Designer displays the Format Cell dialog box when you right-click a table cell and select Format Border from the shortcut menu.

![Format Cell dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856959767/fmtcell.gif)

You see the following options in the dialog box:

**None**

Select if you do not want to display border for the table cell.

**Box**

Select to insert a box border around the table cell. Designer formats the border with the color and line that you specify by the Color and Style options.

**Custom**

Select to customize the border style by yourself. To customize the border style, select a border icon and set the Color and Style options respectively.

**Color**

Select the color of the border. You can select Custom from the drop-down list to customize a color in the [Pick a Color dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010060102-Pick-a-Color-Dialog-Box#Color), or select the button ![Formula button](https://devnet.logianalytics.com/hc/article_attachments/4404848413975/btn_fmla.gif) to [use a formula to control the color](https://devnet.logianalytics.com/hc/en-us/articles/1500010061002-Using-Formulas-to-Control-Object-Properties).

**Style**

Specify the line style of the border.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Apply**

Select to apply all changes and leave the dialog box open.

**Help**

Select to view information about the dialog box.
