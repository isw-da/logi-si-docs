# Save Report As Dialog Box

The Save Report As dialog box helps you to [save the current report](https://devnet.logianalytics.com/hc/en-us/articles/1500010101381-Saving-Reports) with another name in the current location. It appears when you select File > Save As.

![Save Report As dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848491415/svrptas.gif)

The following are details about options in the dialog box:

**Report Name**

Specifies the name of the report.

**Report of Type**

Specifies the report type:

* **Page Report (\*.cls)**  
   Saves the page report as a binary formatting file.
* **Report Template (\*.rpt)**  
   Saves the page report as a text formatting file. A \*.rpt file can be opened and edited as a text file.
* **Self Contained Page Report (\*.clx)**  
   Saves the page report as a binary formatting file which self-contains its own catalog.
* **Page Report XML Format (\*.cls.xml)**  
   Saves the page report as an XML (Extensible Markup Language) formatting file.
* **Web Report (\*.wls)**  
   The only format the web report can be saved in.

**Save**

Saves the report with the name you specified.

**Cancel**

Closes the dialog box, leaving any changes unsaved.

**Help**

Displays the help document about this feature.
