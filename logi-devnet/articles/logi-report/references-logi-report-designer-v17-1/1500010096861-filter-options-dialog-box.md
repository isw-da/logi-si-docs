# Filter Options Dialog Box

You can use the Filter Options dialog box to define the [filter-related shortcut menu items](https://devnet.logianalytics.com/hc/en-us/articles/1500010100541-Label-Properties#FilterOptions) you want to display for a field when end users right-click the field in Page Report Studio. This topic describes the options in the dialog box.

Designer displays Filter Options dialog box when you select the ellipsis button ![Ellipsis button](https://devnet.logianalytics.com/hc/article_attachments/4404856823191/btn_ellipsis1.gif) in the value cell to specify the Filter Options property for a field in the Report Inspector.

![Filter Options dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856851607/fltroptn.gif)

You see the following options in the dialog box:

**Remove Filter**

Select to display the Remove Filter item on the shortcut menu when end users right-click the field in Page Report Studio.

**Top N**

Select to display the Top N item on the shortcut menu when end users right-click the field in Page Report Studio.

**Bottom N**

Select to display the Bottom N item on the shortcut menu when end users right-click the field in Page Report Studio.

**More**

Select to display the More item on the shortcut menu when end users right-click the field in Page Report Studio.

**Default**

Select to apply the default settings.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
