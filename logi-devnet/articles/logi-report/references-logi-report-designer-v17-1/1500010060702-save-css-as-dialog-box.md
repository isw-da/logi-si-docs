# Save CSS As Dialog Box

The Save CSS As dialog box appears when you select OK in the [New CSS Style dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010097961-New-CSS-Style-Dialog-Box) if you select New Style Sheet File from the Add Selector to Style drop-down list. After building a report, if you are satisfied with the format of certain object, you can [save it as a CSS style](https://devnet.logianalytics.com/hc/en-us/articles/1500010062822-Working-with-CSS-Styles#Create) in order to use the same format again in other reports, without having to set all the properties again.

![Save CSS As dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848395415/svcss.gif)

The following are details about options in the dialog box:

**Save the CSS File to**

Shows the directory that the CSS File will be saved to.

**Existing CSS Files**

Lists all the existing CSS files.

**File Name**

Specifies the name for the new CSS file. By default, it is New CSS File.css.

**Files of Type**

Specifies the type for the new file. Here it should be CSS File(\*.css).

**Save**

Saves the CSS style and goes into the [CSS Style Definition dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010095961-CSS-Style-Definition-Dialog-Box).

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
