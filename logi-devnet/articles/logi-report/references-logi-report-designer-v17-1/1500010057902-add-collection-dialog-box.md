# Add Collection Dialog Box

You can use the Add Collection dialog box to [add more schemas](https://devnet.logianalytics.com/hc/en-us/articles/1500010057802-Working-with-MongoDB-Connections-in-a-Catalog#AddSchema) into a catalog via the MongoDB connection you set up. This topic describes the options in the dialog box.

Designer displays the Add Collection dialog box when you right-click the Schemas node or a database node in the Schemas node of a MongoDB connection and select Add Collection on the shortcut menu in the Catalog Manager.

![Add Collection dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404857015063/addclctn.gif)

You see the following options in the dialog box:

**Collections**

* If you open the dialog box by right-clicking the Schemas node, the box lists the collections in all databases that you can access.
* If you open the dialog box by right-clicking a database node, the box lists the collections in this database.

**Added Schemas**

The box lists the schemas to be added to the catalog, which Designer transforms from the collections you select.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404848391191/btn_addarrow.gif)**Add button**

Select to add the specified collections you want to transform to relational schemas.

![Remove button](https://devnet.logianalytics.com/hc/article_attachments/4404848391831/btn_rmvarrow.gif)**Remove button**

Select to remove the specified schemas from the Added Schema box.

**OK**

Select to add the specified schemas to the catalog.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
