# New Catalog Dialog Box

You can use the New Catalog dialog box to [create a catalog](https://devnet.logianalytics.com/hc/en-us/articles/1500010056622-Creating-Opening-and-Saving-Catalogs#Create) in Designer. This topic describes the options in the dialog box.

Designer displays the New Catalog dialog box when you select File > New Catalog.

![New Catalog dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856907671/nwctlg.gif)

You see the following options in the dialog box:

**Name**

Specify the name of the catalog file. The name should contain the extension ".cat" or ".cat.xml".

**Data Source Name**

Specify the name for the first data source in the catalog. The name cannot contain special characters other than underscore.

**Directory**

Specify the location where you want to save the new catalog. Type the directory in the text box or select the ellipsis button ![Ellipsis button](https://devnet.logianalytics.com/hc/article_attachments/4404848397591/btn_ellipsis.gif) to find a directory path. You do not want to select a directory that already contains a catalog.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
