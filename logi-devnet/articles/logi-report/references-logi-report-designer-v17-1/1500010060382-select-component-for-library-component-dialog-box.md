# Select Component for Library Component Dialog Box

The Select Component for Library Component dialog box helps you to [create a library component](https://devnet.logianalytics.com/hc/en-us/articles/1500010101181-Creating-Reports#LC). It appears when you select New > Library Component on the Home or File menu bar.

![Select Component for Library Component dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848419095/slctcmpntlc.gif)

The following are details about options in the dialog box:

**Library Component Title**

Specifies the title of the library component.

**Component layout**

Specifies the layout with which to create the library component.

* **Table (Group Above)**  
   Creates a library component containing a table with group information above the detail row.
* **Table (Group Left)**  
   Creates a library component containing a table with group information left to the detail row.
* **Table (Group Left Above)**  
   Creates a library component containing a table with group information left above the detail row.
* **Summary Table**  
   Creates a library component containing a table with only group and summary information.
* **Chart**  
   Creates a library component containing a chart.
* **Crosstab**  
   Creates a library component containing a crosstab.
* **Blank**  
   Creates a library component with nothing in it.

**OK**

Creates a library component with the selected layout.

**Cancel**

Closes the dialog box and deletes any unsaved changes.

**Help**

Displays the help document about this feature.
