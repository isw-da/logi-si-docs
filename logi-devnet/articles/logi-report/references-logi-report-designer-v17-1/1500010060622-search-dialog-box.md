# Search Dialog Box

The Search dialog box helps you to search resources that you need in the [Catalog Manager](https://devnet.logianalytics.com/hc/en-us/articles/1500010056642-Managing-the-Data-Resources-in-a-Catalog#Search) or [Data panel](https://devnet.logianalytics.com/hc/en-us/articles/1500010101501-Data-Panel#Search). It appears when you select the Search button the the toolbar of the Catalog Manager or ![Search button](https://devnet.logianalytics.com/hc/article_attachments/4404848376087/btn_fnd.gif) on the toolbar of the Data panel.

![Search dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848377367/srch.gif)

The following are details about options in the dialog box:

**Search Expression**

Specifies the text to search for.

**Case Sensitive**

Specifies whether to match whole word only or match case.

**Scope**

Specifies the search range.

* **Selected Resources**  
  Specifies to let Designer search from the resources you have selected in the Catalog Manager/Data panel before you open the dialog box.
* **Search In**  
  Specifies to let Designer search from the selected data source in the drop-down list.

**Search**

Begins the search process and displays the results in the Search Result panel.

**Cancel**

Cancels the search process and closes the dialog box.

**Help**

Displays the help document about this feature.
