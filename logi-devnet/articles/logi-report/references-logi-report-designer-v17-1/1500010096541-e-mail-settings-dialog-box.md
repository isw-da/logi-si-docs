# E-mail Settings Dialog Box

You can use the E-mail Settings dialog box to edit the additional settings of the e-mail when you specify to [distribute a report to e-mail](https://devnet.logianalytics.com/hc/en-us/articles/1500010101161-Creating-Bursting-Reports#Create). This topic describes the options in the dialog box.

Designer displays the E-mail Settings dialog box when you select E-mail or Logi Report Server User/Group/Role > User E-mail in the Recipient section and then select the E-mail Settings button in the [Bursting dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010058102-Bursting-Dialog-Box).

![E-mail Settings dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848420375/emlst.gif)

You see the following options in the dialog box:

**Cc**

Select the field from the recipient query which retrieves e-mail addresses that you want to send additional copies of the e-mail to.

**Bcc**

Select the field from the recipient query which retrieves e-mail addresses that you want to secretly send additional copies of the e-mail to.

**Subject**

Specify the subject of the e-mail.

**Comments**

Specify the contents of the e-mail or comments to the contents.

**Add Dynamic Field**

Designer enables the button when you select in the Subject or Comments text box. Select it to open the [Select Field dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010098661-Select-Field-Dialog-Box) to select a field and insert it in the text box so as to compose dynamic information.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
