# Go To - Web Action Builder

You can use the Go To-Web Action Builder dialog box to [build a Go To web action](https://devnet.logianalytics.com/hc/en-us/articles/1500010057562-Using-Basic-Web-Controls-in-a-Report#GoTo) on an object, so that the report users can change the data fields in the specified data components of a report, when they trigger the event bound with the action on the object at runtime. This topic describes the options in the dialog box.

Designer displays the Go To-Web Action Builder dialog box when you select \*Go To and select OK in the [Web Action List dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010060822-Web-Action-List-Dialog-Box).

![Go To - Web Action Builder dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856923031/gotowbactnbld.gif)

You see the following options in the dialog box:

**Apply Action To**

The box lists the data components in the report. Select a data component to define the go-to condition for it.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404848403095/btn_add.gif)**Add button**

Select to add a new condition line. You can add one condition at most.

![Trash Can button](https://devnet.logianalytics.com/hc/article_attachments/4404848403351/btn_trashcan.gif)**Remove button**

Select to delete the condition.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404848403735/btn_mvup.gif)**Move Up button**

Designer does not enable this button for building the Go To web action.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404856822935/btn_mvdwn.gif)**Move Down button**

Designer does not enable this button for building the Go To web action.

**Go To On**

Specify the field on which to perform the go-to action for the selected data component.

**To Column**

Specify to which field the Go To On field will be changed for the selected data component.

**By Value**

Select to perform the go-to action by filtering the specified value of the Go To On field for the selected data component. Designer disables the option if the action is to be performed on a crosstab aggregate field.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
