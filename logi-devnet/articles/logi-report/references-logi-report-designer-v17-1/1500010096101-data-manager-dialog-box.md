# Data Manager Dialog Box

You can use the Data Manager dialog box to [control the data retrieval and keep track of the run information](https://devnet.logianalytics.com/hc/en-us/articles/1500010062602-Managing-the-Data-Retrieval-of-Queries) of the data resource. This topic describes the options in the dialog box.

Designer displays the Data Manager dialog box when you right-click a query, stored procedure, UDS, or an imported SQL in the Catalog Manager or in the Data panel and select Data Manager from the shortcut menu.

![Data Manager dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848429719/dtmngr.gif)

You see the following options in the dialog box:

**Maximum Duration (second)**

The option shows the value of the Maximum Duration property that you have specified for the data resource.

**Maximum Rows**

The option shows the value of the Maximum Rows property that you have specified for the data resource.

**Number**

The column shows the numbers of the data resource running records.

**Date**

The column shows the days on which the reports ran.

**Time**

The column shows the time when the reports ran.

**Number of Rows**

The column shows the number of rows in the reports.

**Duration**

The column shows the running duration of the reports.

**Report**

The column shows the names of the reports that use the data resource.

**Refresh**

Select to refresh the records in the dialog box.

**Close**

Select to exit the dialog box.

**Help**

Select to view information about the dialog box.
