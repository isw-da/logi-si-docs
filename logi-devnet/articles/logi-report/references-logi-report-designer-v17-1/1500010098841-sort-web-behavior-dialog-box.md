# Sort - Web Behavior Dialog Box

The Sort-Web Behavior dialog box helps you to [apply the Sort web action to a web control](https://devnet.logianalytics.com/hc/en-us/articles/1500010057542-Using-Basic-Web-Controls-in-the-Configuration-Panel-of-a-Library-Component#Sort) in a library component so as to sort data of the library component at runtime. It appears when you right-click a web control in the configuration panel of a library component and select Web Behaviors > Sort from the shortcut menu.

![Sort - Web Behavior dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404856894231/sortwbbhvr.gif)

The following are details about options in the dialog box:

**Apply Action To**

Specifies to which component to apply the sort action. Select the desired component from the drop-down list.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404848403095/btn_add.gif)

Adds a new sort criterion.

![Trash Can button](https://devnet.logianalytics.com/hc/article_attachments/4404848403351/btn_trashcan.gif)

Removes the selected sort criterion.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404848403735/btn_mvup.gif)

Moves the selected sort criterion higher in the list.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404856822935/btn_mvdwn.gif)

Moves the selected sort criterion lower in the list.

**Sort On**

Specifies which field to be sorted on. Which fields are available here depends on which object you choose from the Apply Action To drop-down list. Select the field you want to sort on from the drop-down list.

**Sort Value**

Specifies the value of how to sort on the field. It could be Ascending or Descending, or be specified by the value of a web control.

**OK**

Accepts the changes and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
