# Format Series (Z) Gridline Dialog Box

You can use the Format Series (Z) Gridline dialog box to [format the gridlines on the series (Z) axis](https://devnet.logianalytics.com/hc/en-us/articles/1500010094481-Formatting-the-Gridlines) of a chart. This topic describes the options in the dialog box.

Designer displays the Format Series (Z) Gridline dialog box when you right-click a chart and select Format Gridlines > Series (Z) Gridline from the shortcut menu, or double-click any gridline on the series axis of a chart.

![Format Series (Z) Gridline dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848539799/fmtseriesgrdln.gif)

You see the following options in the dialog box:

**Line Color**

Specify the color of the gridlines. To edit the color, select the color indicator and select a color from the [color palette](https://devnet.logianalytics.com/hc/en-us/articles/1500010062922-Menu-Tabs#Palette) or input the hexadecimal value (for example, 0xff0000) of a color in the text box.

**Line Style**

Specify the line style of the gridlines.

**Transparency**

Specify the transparency for the color of the gridlines.

**Thickness**

Specify the thickness of the gridlines, in pixels.

**OK**

Select to apply all changes and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Apply**

Select to apply all changes and leave the dialog box open.

**Help**

Select to view information about the dialog box.
