# Style List Dialog Box

The Style List dialog box helps you to customize the style for pointers in the same data series respectively in a gauge chart. It appears when you select the Style List button in the Pointer tab of the [Format Bar Gauge dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010059102-Format-Bar-Gauge-Dialog-Box#Pointer), [Format Dial Gauge dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010059182-Format-Dial-Gauge-Dialog-Box#Pointer), or [Format Solid Gauge dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010097261-Format-Solid-Gauge-Dialog-Box#Pointer).

The dialog box varies according to the gauge type: [dial/solid gauge](#Dial) and [bar gauge](#Bar).

---

For a dial or solid gauge, the dialog box contains the following options:

![Style List dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848492439/stylst.gif)

**Styles**

Lists all the specified styles.

* ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404848403095/btn_add.gif)  
   Adds a new style.
* ![Trash Can button](https://devnet.logianalytics.com/hc/article_attachments/4404848403351/btn_trashcan.gif)  
   Removes the selected style.
* ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404848403735/btn_mvup.gif)  
   Moves the selected style higher in the list.
* ![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404856822935/btn_mvdwn.gif)  
   Moves the selected style lower in the list.

**Arrow**

Specifies to use arrow as the pointer style.

* **Value Pointer**  
   Specifies the style of the value pointer. Select a style from the drop-down list or select Customized and specify another image as the value pointer.
* **Width**   
   Specifies the width of the arrow.
* **Height**   
   Specifies the height of the arrow.

**Mark**

Specifies to use mark as the pointer style.

* **Value Pointer**Specifies the style of the value pointer. Select a style from the drop-down list or select Customized and specify another image as the value pointer.
* **Width**   
   Specifies the width of the mark.
* **Height**   
   Specifies the height of the mark.
* **Position**   
   Specifies the position relationship of the mark and the arc.
* **Gap**   
   Specifies the distance between the pointer and the arc, in pixels.

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.

---

For a bar gauge, the dialog box contains the following options:

![Style List dialog box for bar gauge](https://devnet.logianalytics.com/hc/article_attachments/4404856892823/stylst_bar.gif)

**Styles**

Lists all the specified styles.

* ![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404848403095/btn_add.gif)  
   Adds a new style.
* ![Trash Can button](https://devnet.logianalytics.com/hc/article_attachments/4404848403351/btn_trashcan.gif)  
   Removes the selected style.
* ![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404848403735/btn_mvup.gif)  
   Moves the selected style one step up.
* ![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404856822935/btn_mvdwn.gif)  
   Moves the selected style one step down.

**Use Pointer**

Specifies the style of the pointer.

* **Value Pointer**  
   Specifies the style of the value pointer. Select a style from the drop-down list or select Customized and specify another image as the value pointer.
* **Width**  
   Specifies the width of the pointer.
* **Height**  
   Specifies the height of the pointer.
* **Position**  
   Specifies the position of the pointer relative to the bar.
* **Gap**  
   Specifies the distance between the pointer and the bar, in pixels.

**Use Color Bar**

This option is disabled.

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
