# Insert Detail Column Dialog Box

You can use the Insert Detail Column dialog box to [insert a new detail column](https://devnet.logianalytics.com/hc/en-us/articles/1500010057442-Modifying-Tables#DetailColumn) into a table. This topic describes the options in the dialog box.

Designer displays the Insert Detail Column dialog box when you right-click a table or a table column and select Insert > Detail Column from the shortcut menu.

![Insert Detail Column dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848523927/instdtlclmn.gif)

You see the following options in the dialog box:

**Resources**

The box contains the data fields in and related to the data resource the table uses which you can use for a detail column. Select the field you want from the box.

**OK**

Select to insert the detail column which contains the specified data field and close the dialog box.

**Cancel**

Select to close the dialog box without saving any changes.

**Help**

Select to view information about the dialog box.
