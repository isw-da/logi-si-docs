# Select Report Tab Dialog Box

The Select Report Tab dialog box helps you to select the [report tabs you want to import](https://devnet.logianalytics.com/hc/en-us/articles/1500010101281-Managing-Page-Reports#Import) from the specified report. It appears when you select File > Import> Logi Report Page Report, specify a report in the Open Report dialog box and then select Open.

![Select Report Tab dialog box](https://devnet.logianalytics.com/hc/article_attachments/4404848410007/slctrpt.gif)

The following are details about options in this dialog box:

**Name**

Displays the names of the report tabs.

**Description**

Displays the description of the report tabs.

**Customize Dataset for Components**

Specifies to customize datasets used by components within the selected report tabs. If this option is selected, when you select OK in the dialog box, the [Customize Dataset dialog box](https://devnet.logianalytics.com/hc/en-us/articles/1500010096041-Customize-Dataset-Dialog-Box) will be displayed.

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
