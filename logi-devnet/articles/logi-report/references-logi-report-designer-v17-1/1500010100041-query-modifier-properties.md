# Query Modifier Properties

This topic lists the properties of a Query Modifier object in a catalog.

| Property Name | Description |
| --- | --- |
| Data Source Name | Specifies the data source that includes this query modifier. Data type: String |
| Description | Specifies the description of the query modifier. Data type: String |
| Name | Specifies the name of the query modifier. Data type: String |
| Query | Specifies the query which the query modifier refers to. Data type: String |
| Query Modifier | Specifies the WHERE string of the query modifier. Data type: String |
| Structured | Specifies the type for defining the WHERE string. If true, it is Structured; if false, it is String. Data type: Boolean |
