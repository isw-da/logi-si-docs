# Turn to Page Dialog Box Properties

You can use the Turn to Page dialog box to input the page number so as to turn to the specified page. This topic describes the options in the dialog box.

![Turn to Page dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880471063/tn2pg.gif)

**Input Page Number**

Specifies the page number you would like to turn to.

**Go**

Goes to the specified page.

**Cancel**

Closes this dialog box without turning the report page.

**Help**

Displays the help document about this feature.
