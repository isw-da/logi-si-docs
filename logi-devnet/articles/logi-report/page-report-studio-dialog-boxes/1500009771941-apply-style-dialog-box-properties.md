# Apply Style Dialog Box Properties

You can use the Apply Style dialog box to set a style for the data component you selected. This topic describes the options in the dialog box.

![Apply Style dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880558231/appstyl.gif)

**Style**

Lists the styles applicable to the component.

* **Custom**  
   There is no style information in this style. Logi Report only uses it to support reports that were created in previous versions when they did not bind any style or their bound styles are not in the style list.

**Preview**

Displays a diagram to illustrate the effect of the selected style on the component.

**Inherit Style**

Specifies to take the style of the parent component. The option is available only when the banded object, table, crosstab, or chart is in a banded object (for a chart, in a table also).

**OK**

Accepts all changes and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
