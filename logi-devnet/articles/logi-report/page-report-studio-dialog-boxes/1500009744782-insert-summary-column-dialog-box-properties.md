# Insert Summary Column Dialog Box Properties

You can use the Insert Summary Column dialog box to [insert a new summary column](https://devnet.logianalytics.com/hc/en-us/articles/1500009749022-Manipulating-Data-Components-in-Page-Report#InsertColumn) into a table. This topic describes the options in the dialog box.

![Insert Summary Column dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880257815/insrtsumclmn.gif)

**Resources**

Lists all the available data resources for the new summary column. Select the resource you want from the list.

**OK**

Inserts the column with the specified resource and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
