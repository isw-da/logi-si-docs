# Bottom N Dialog Box Properties

You can use the Bottom N dialog box to filter data to display records that meet the [Bottom N condition](https://devnet.logianalytics.com/hc/en-us/articles/1500009777561-Applying-Filters-to-Page-Report#Component). This topic describes the options in the dialog box.

![Bottom N dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885600279/bottomn.gif)

**Value**

Type a positive integer number N here to display records with the field value equal to one of the bottom N field values.

**OK**

Applies the settings and closes the dialog box.

**Cancel**

Closes the dialog box and discards any changes.

**Help**

Displays the help document about this feature.
