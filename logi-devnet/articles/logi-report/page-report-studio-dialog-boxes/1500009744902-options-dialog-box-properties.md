# Options Dialog Box Properties

You can use the Options dialog box to [set the user interface of Page Report Studio](https://devnet.logianalytics.com/hc/en-us/articles/1500009749002-General-Operations-in-Page-Report#Option). This topic describes the options in the dialog box.

This topic contains the following sections:

* [Option Tab Properties](#Option)
* [Customize Tab Properties](#Customize)

**Restore Defaults**

Restores the default settings.

**OK**

Applies the settings and closes the Options dialog box.

**Cancel**

Cancels the settings and closes the Options dialog box.

**Help**

Displays the help document about this feature.

## Option Tab Properties

This tab enables you to specify the skin of Page Report Studio user interface.

![Options dialog - Option tab](https://devnet.logianalytics.com/hc/article_attachments/4404885393175/optn_optn.gif)

**Skin**

Specifies the skin of the Page Report Studio user interface. The skin can be the Standard, Classical, or Windows XP style.

## Customize Tab Properties

This tab enables you to customize the toolbars in Page Report Studio.

![Options dialog - Customize tab](https://devnet.logianalytics.com/hc/article_attachments/4404880261527/optn_cstmz.gif)

**Current Toolbar**

Lists all the available toolbars. If you want to customize a toolbar, select it to select.

![New Toolbar Name](https://devnet.logianalytics.com/hc/article_attachments/4404880163351/btn_add1.gif)

Displays the New Toolbar Name dialog box for you to type the name for the new toolbar. The new toolbar name will then be displayed in the Current Toolbar list box.

![Delete button](https://devnet.logianalytics.com/hc/article_attachments/4404885309335/btn_delete.gif)

Removes the selected toolbar.

**Available Tools**

Displays the tools that you can add to the toolbars. The tools available in the box are determined by [which Page Report Studio feature profile is applied](https://devnet.logianalytics.com/hc/en-us/articles/1500009777681-Customizing-Page-Report-Studio-Profile).

**Selected Tools**

Displays the tools of the toolbar selected in the Current Toolbar list box in the order that they will appear (from left to the right) on the toolbar.

![Add Item button](https://devnet.logianalytics.com/hc/article_attachments/4404880170519/btn_additem.gif)

Adds the selected tool from the Available Tools list box to the Selected Tools list box.

![Remove Item button](https://devnet.logianalytics.com/hc/article_attachments/4404885336599/btn_rmvitem.gif)

Removes the selected tool.

![Remove All button](https://devnet.logianalytics.com/hc/article_attachments/4404880491159/btn_pgrpt_rmvall.gif)

Empties the contents of the Selected Tools list box.

![Add All button](https://devnet.logianalytics.com/hc/article_attachments/4404880491543/btn_pgrpt_addall.gif)

Adds all the available tools from the Available Tools list box to the Selected Tools list box.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404885326231/btn_mvup.gif)

Adjusts the location of the selected tool by moving it to the left on the toolbar.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404880154263/btn_mvdown.gif)

Adjusts the location of the selected tool by moving it to the right on the toolbar.
