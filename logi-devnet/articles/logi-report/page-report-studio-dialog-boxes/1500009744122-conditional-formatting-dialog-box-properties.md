# Conditional Formatting Dialog Box Properties

You can use the Conditional Formatting dialog box to [add conditional formats to the values](https://devnet.logianalytics.com/hc/en-us/articles/1500009777581-Adding-Conditional-Formats-to-Fields-in-Page-Report) of the selected field. This topic describes the options in the dialog box.

![Conditional Formatting dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880263319/cndtnlfmt.gif)

**Condition**

Displays all the conditions on the field.

* ![Create Condition](https://devnet.logianalytics.com/hc/article_attachments/4404880163351/btn_add1.gif)  
  Displays the [Edit Conditions](https://devnet.logianalytics.com/hc/en-us/articles/1500009772121-Edit-Conditions-Dialog-Box-Properties) dialog box for you to create a new condition.
* ![Edit Condition](https://devnet.logianalytics.com/hc/article_attachments/4404880196759/btn_pgrpt_edit.gif)  
  Displays the [Edit Conditions](https://devnet.logianalytics.com/hc/en-us/articles/1500009772121-Edit-Conditions-Dialog-Box-Properties) dialog box for you to edit the selected condition.
* ![Delete button](https://devnet.logianalytics.com/hc/article_attachments/4404885309335/btn_delete.gif)  
   Removes the selected condition.
* **Priority**  
   Specifies the priority of each condition.
  + **![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404880197143/btn_mvup1.gif)**  
     Moves a condition up for a higher priority.
  + **![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404885328919/btn_mvdown1.gif)**  
     Moves a condition down for a lower priority.

**Format**

You can specify a format for each condition. Studio will apply the format to the field values when the specified condition is fulfilled.

* **Font**  
   Specifies the font type for the field values.
* **Border**  
   Specifies the border line style for the field values.
* **Size**  
   Specifies the font size for the field values.
* **Bold**  
   Specifies whether to bold the field values.
* **Italic**  
   Specifies whether to make the field values italic.
* **Underline**  
   Specifies whether to underline the field values.
* **Foreground Color**   
   Specifies the foreground color for the field values.
* **Background Color**   
   Specifies the background color for the field values.
* **Sample Text**  
   Displays a preview sample of your settings.
* **Blinking Text**  
   Specifies whether to make the field values blink.
  + **Duration**  
     Specifies how long it takes the field values to complete the transition from the foreground color to the transparent color, in seconds.

**OK**

Applies the settings and exits the dialog box.

**Cancel**

Cancels the settings and closes the dialog box.

**Help**

Displays the help document about this feature.
