# Aggregate On Dialog Box Properties

You can use the Aggregate On dialog box to [calculate data in the selected table detail column](https://devnet.logianalytics.com/hc/en-us/articles/1500009749022-Manipulating-Data-Components-in-Page-Report#AggregateOn). This topic describes the options in the dialog box.

![Aggregate On dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885390743/agrgton.gif)

**Aggregate On**

Displays the field that is bound with the detail column. This field is also the one on which the summary is based.

**Function**

Specifies the function to compute the field in the detail column.

**Distinct On**

Studio displays this option and you should set it when you have selected **DistinctSum** as the aggregate function. It specifies the fields according to whose unique values to calculate DistinctSum. Select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404880145687/btn_chsr2.gif) to select the required fields in the [Select Fields](https://devnet.logianalytics.com/hc/en-us/articles/1500009745042-Select-Fields-Dialog-Box-Properties) dialog box.

**OK**

Summarizes the detail column data with the specified function and closes the dialog box.

**Cancel**

Does not retain any changes and closes the dialog box.

**Help**

Displays the help document about this feature.
