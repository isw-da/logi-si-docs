# Sort Dialog Box Properties

You can use the Sort dialog box to specify the settings to [sort the records or groups](https://devnet.logianalytics.com/hc/en-us/articles/1500009777621-Sorting-Page-Report-Data) in a banded object. This topic describes the options in the dialog box.

![Sort dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885384599/sort.gif)

**Sort in Scope**

Specifies on which level the sorting is based.

![Delete button](https://devnet.logianalytics.com/hc/article_attachments/4404880189847/btn_pgrpt_no.gif)

Deletes the specified sort expression line.

**Field**

Specifies the field by which to sort the records or groups.

**Ascend**

Specifies to sort the data ascending.

**Descend**

Specifies to sort the data descending.

![Move Up button](https://devnet.logianalytics.com/hc/article_attachments/4404885326231/btn_mvup.gif)

Moves up a sort expression line.

![Move Down button](https://devnet.logianalytics.com/hc/article_attachments/4404880154263/btn_mvdown.gif)

Moves down a sort expression line.

![Add button](https://devnet.logianalytics.com/hc/article_attachments/4404880163351/btn_add1.gif)

Adds a new sort expression line after the current one. This button appears after the first sort condition has been composed.

**OK**

Accepts the changes and re-loads the report result using the sort settings with this dialog box closed.

**Cancel**

Cancels the operation and closes this dialog box.

**Apply**

Accepts the changes and re-loads the report result using the sort settings with this dialog box displayed.

**Reset**

Restores this dialog box to its opening status.

**Help**

Displays the help document about this feature.
