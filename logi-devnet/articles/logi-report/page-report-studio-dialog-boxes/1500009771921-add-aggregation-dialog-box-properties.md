# Add Aggregation Dialog Box Properties

You can use the Add Aggregation dialog box to [create a dynamic aggregation object](https://devnet.logianalytics.com/hc/en-us/articles/1500009748982-Using-Dynamic-Resources-in-Page-Report#Agg) in a report. This topic describes the options in the dialog box.

![Add Aggregation dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885397783/addaggrgtn.gif)

**Aggregation Name**

Specifies the display name of the aggregation object.

**Mapping Name**

After you select ![Select Resource](https://devnet.logianalytics.com/hc/article_attachments/4404880145687/btn_chsr2.gif), Studio displays the [Select Resource](https://devnet.logianalytics.com/hc/en-us/articles/1500009773021-Select-Resource-Dialog-Box-Properties) dialog box. You can then select a field or a formula which the aggregation object is based on. Studio will display the field with its parent node path in the text box.

**Aggregate Function**

Specifies the aggregate function of the aggregation object. Studio will apply the function to the group where the aggregation is or to the entire report if the aggregation is in the report header or footer.

**Distinct On**

Studio displays this option and you should set it when you have selected **DistinctSum** as the aggregate function. It specifies the fields according to whose unique values to calculate DistinctSum. Select ![Choose button](https://devnet.logianalytics.com/hc/article_attachments/4404880145687/btn_chsr2.gif) to select the required fields in the [Select Fields](https://devnet.logianalytics.com/hc/en-us/articles/1500009745042-Select-Fields-Dialog-Box-Properties) dialog box.

**OK**

Creates the aggregation object and closes the dialog box.

**Cancel**

Cancels the creation and closes the dialog box.

**Help**

Displays the help document about this feature.
