# Customized Page Dialog Box Properties

The Customized Page dialog box shows the parameters that the current report and its subreports use. You can change the parameter values to re-run the report. This topic describes the options in the dialog box.

Server displays the dialog box when you trigger the web action user\_popCustomizedPage() that is bound with a web control. You can also customize the JSP page of this dialog box to make it display what you need using the Page Report Studio API.

For more information, see [Enter Parameter Values](https://devnet.logianalytics.com/hc/en-us/articles/1500009772161-Enter-Parameter-Values-Dialog-Box-Properties).

![Customized Page dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885597335/cstmzdpg.gif)

**OK**

Applies the changes and closes the dialog box.

**Cancel**

Cancels the changes and exits the dialog box.

**Help**

Displays the help document about this feature.
