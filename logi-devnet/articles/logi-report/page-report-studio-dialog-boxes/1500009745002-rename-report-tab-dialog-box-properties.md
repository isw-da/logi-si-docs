# Rename Report Tab Dialog Box Properties

You can use the Rename Report Tab dialog box to rename the current report tab. This topic describes the options in the dialog box.

![Rename Report Tab dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885552407/renmrpt.gif)

**Input New Name**

Specifies the new name for the report tab.

**OK**

Applies the new name to the report tab and closes this dialog box.

**Cancel**

Discards the setting and closes this dialog box.

**Help**

Displays the help document about this feature.
