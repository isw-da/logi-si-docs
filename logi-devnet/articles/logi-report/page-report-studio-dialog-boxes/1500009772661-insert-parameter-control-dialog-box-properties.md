# Insert Parameter Control Dialog Box Properties

You can use the Insert Parameter Control dialog box to [insert a parameter control](https://devnet.logianalytics.com/hc/en-us/articles/1500009749062-Applying-Web-Controls-in-Page-Report#Parameter) into a report for delivering parameter values to the report. This topic describes the options in the dialog box.

![Insert Parameter Control dialog](https://devnet.logianalytics.com/hc/article_attachments/4404885383319/insrtprmctrl.gif)

**Select a Parameter**

Specifies a parameter from the list where all the parameters except cascading parameters used by the current report are displayed.

**OK**

Inserts a parameter control into the report and closes the dialog box.

**Cancel**

Cancels the operation and closes the dialog box.

**Help**

Displays the help document about this feature.
