# Enter Parameter Values Dialog Box Properties

Use the Enter Parameter Values dialog box to specify parameter values for a report. This topic describes how to specify parameter values.

![Enter Parameter Values dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880540951/enterprmvlu.gif)

The way to specify a parameter value varies with the type and properties of the parameter. You can use the following ways to specify parameter values:

* In the parameter value combo box, select the down arrow ![Drop Down button](https://devnet.logianalytics.com/hc/article_attachments/4404885319191/btn_dshbrd_more1.gif) and then select a value from the drop-down list. To ensure performance, Logi Report lists at most 500 values by default.

  When it encrypts the values of a parameter, Logi Report disables the value drop-down list. In this case, if you see the **Hide Parameter Value** check box, you can clear it to enable the drop-down list to view and select a value.
* Type the value manually in the parameter value text box if the parameter enables type-in values but not multiple values.
* Select in the parameter value combo box to [specify multiple values](https://devnet.logianalytics.com/hc/en-us/articles/1500009743882-Specifying-Parameter-Values-for-a-Dashboard#MultiValue) in the [Enter Values](https://devnet.logianalytics.com/hc/en-us/articles/1500009744282-Enter-Values-Dialog-Box-Properties) dialog box if the parameter enables multiple values.
* Select or clear the check box to specify a Yes/No value.
* Select the **Calendar** icon ![Calendar icon](https://devnet.logianalytics.com/hc/article_attachments/4404885317143/btn_clndr.gif) to [specify a date and time value](https://devnet.logianalytics.com/hc/en-us/articles/1500009778361-Specifying-Parameter-Values#DateTime).

**OK**

Select **OK** to run the report using the parameter values you specified here.

**Help**

Select **Help** to view information about the Enter Parameter Values dialog box.
