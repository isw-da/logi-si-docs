# Save Report Template Dialog Box Properties

You can use the Save Report Template dialog box to confirm whether to save the report template when [saving a report](https://devnet.logianalytics.com/hc/en-us/articles/1500009777601-Saving-a-Page-Report). This topic describes the options in the dialog box.

![Save Report Template dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880478871/svtmplt.gif)

**Save Sort Criteria**

Specifies whether to save the sorting criteria with the report.

**Save Filter Criteria**

Specifies whether to save the filtering criteria with the report.

**Yes**

Applies the settings and closes this report.

**Cancel**

Does not save the changes to the report and returns to the report.

**Help**

Displays the help document about this feature.
