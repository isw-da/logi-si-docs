# Group Properties

You can use the Group Properties dialog box to edit the properties of a group object. This topic describes the options in the dialog box.

This topic contains the following sections:

* [General Tab Properties](#General)
* [Others Tab Properties](#Others)

**OK**

Applies the settings and closes this dialog box.

**Cancel**

Cancels the settings and closes this dialog box.

**Help**

Displays the help document about this feature.

## General Tab Properties

This tab shows some general information of the group object.

![Group Properties dialog - General tab](https://devnet.logianalytics.com/hc/article_attachments/4404885560855/grpprpty_gnrl.gif)

**Expand Detail Data**

Specifies whether to expand or collapse the detailed records in the group level.

**Keep Group Together**

Specifies whether to try to keep groups wholly in a report page (try not to make a group across two pages).

**Repeat While Group Footer**

If the group header is set to be repeated, you can specify whether to still repeat the group header when a page break occurs on the group footer.

**Shrink Footer**

Specifies whether to shrink the group footer when you shrink detailed records in the group level.

**Special Function**

Specifies a special function for the group level to group. This option is available only when the group field is of the Date/Time type.

**Select Type**

Specifies the type for Select N condition:

* **ALL**: The Select N condition will not take effect, and all groups will be retrieved.
* **TOP\_N**: The first N groups of the group level will be displayed. Herein N is what you type in the Select N text box.
* **BOTTOM\_N**: The last N groups of the group level will be displayed. N is the integer you type in the Select N text box.

**Select N**

If you want only the first or last several groups of the group level to be displayed, you can type an integer which will be the number of groups to be displayed.

**Sort**

Specifies a sort direction for the groups of the group level.

## Others Tab Properties

You can use this tab to view and configure some miscellaneous settings.

![Group Properties dialog - Others tab](https://devnet.logianalytics.com/hc/article_attachments/4404885561495/grpprpty_other.gif)

**TOC Anchor**

Specifies whether to add the node that represents the object to the TOC tree in the TOC Browser.
