# Open Report Tabs Dialog Box Properties

You can use the Open Report Tabs dialog box to open or hide report tabs in the current report, and to see the display name and description of each report tab. This topic describes the options in the dialog box.

![Open Report Tabs dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880260631/opnrpt.gif)

In this dialog box, Studio marks the open report tabs in the current report with a check symbol. To change the display state of the report tabs in the report, select those report tabs you want to open, clear those you want to close, and then select OK.

**Display Name**

Shows the display names of the report tabs in the report.

**Instance Name**

Shows the instance names of the report tabs in the report.

**Description**

Shows the descriptions of the report tabs if any.

**OK**

Closes this dialog box, opens the selected report tabs, and closes those you do not select.

**Cancel**

Closes this dialog box without changing the report display state.

**Help**

Displays the help document about this feature.
