![Note icon](https://devnet.logianalytics.com/hc/article_attachments/4404885305111/noteicon.jpg)The object properties dialog boxes may be different for an [advanced user](https://devnet.logianalytics.com/hc/en-us/articles/1500009777681-Customizing-Page-Report-Studio-Profile#AdvUser) and for a normal user. An advanced user can modify some properties that a common user cannot, and can use formulas to control properties. The documents for properties dialog boxes in this topic are based on an advanced user.

The server administrator can enable the **Advanced User** option for all users in the server profile:

1. In the Server Console, point to **Administration** on the system toolbar, and then select **Server Profile** > **Customize Profile** from the drop-down menu.
2. Go to the **Page Report Studio** > **Properties** > **Advanced** tab and select the **Advanced User** option.
3. Select **OK** to accept the change.
