# New Page Report Dialog Box Properties

You can use the New Page Report dialog box to [create a new page report](https://devnet.logianalytics.com/hc/en-us/articles/1500009748862-Creating-Page-Reports) that contains one report tab. This topic describes the options in the dialog box.

![New Page Report dialog](https://devnet.logianalytics.com/hc/article_attachments/4404880275863/newrpt.gif)

**Report Title**

Specifies the title of the report tab.

**Choose Report Layout**

Lists the report layouts on which you can create your report tab.

* **Banded**  
   Creates a report containing a banded object.
* **Table (Group Above)**  
   Creates a report containing a table with group information above the detail row.
* **Table (Group Left)**  
   Creates a report containing a table with group information left to the detail row.
* **Table (Group Left Above)**  
   Creates a report containing a table with group information left above the detail row.
* **Summary Table**  
   Creates a report containing a table with only group and summary information.
* **Chart**  
   Creates a report containing a chart.
* **Crosstab**  
   Creates a report containing a crosstab.
* **Blank**  
   Creates a report with nothing in it.

**OK**

Closes this dialog box, and in accord with the selected report layout, the corresponding report wizard will be displayed, or a new page report will be created with a blank report tab if Blank is selected.

* [Banded Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009744062-Banded-Wizard-Properties)
* [Table Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009773181-Table-Wizard-Properties)
* [Chart Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009772001-Chart-Wizard-Properties)
* [Crosstab Wizard](https://devnet.logianalytics.com/hc/en-us/articles/1500009744202-Crosstab-Wizard-Properties)

**Cancel**

Closes this dialog box without creating a page report.

**Help**

Displays the help document about this feature.
