# Setting Up Business View Security

You can use business view security to limit user access to [elements](https://devnet.logianalytics.com/hc/en-us/articles/4405661913239-Business-View-Elements) of the business views in a catalog. This topic introduces business view security and how you can configure business view security in a catalog. It also provides an example to demonstrate how Server applies business view security at runtime to generate different reports to different users.

You can export business view security, together with the [security entries](https://devnet.logianalytics.com/hc/en-us/articles/4405661342359-Working-with-RLS-and-CLS-at-Data-Source-Scope) defined in a catalog to a security file using the Export Security command on the File menu. Designer saves the security file in the same location as the current catalog by default. The Server administrators can then use the security file to create dynamic security for catalogs on Server (for more information, see Dynamic Security in the *Logi Report Server Guide*).
