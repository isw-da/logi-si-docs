# Design/View Area

The Design/View area, or report-designing window, comprises of the following parts:

**Title bar**

The title bar shows the report file name, and three buttons for you to minimize, maximize and close the active report.

**Design tab**

The Design tab is a WYSIWYG area for laying out your report. You can make use of rulers, grids and guidelines to position objects.

**View tab**

The View tab is for previewing the report layout and result. The report may not display exactly as the user would see it in Logi JReport Server. Interactive objects such as parameter and filter objects are not active and report links are not active. In this tab there is a toolbar which consists of the following buttons:

* **![First Page button](https://devnet.logianalytics.com/hc/article_attachments/4404893801623/btn_fstpg.gif) First Page**  
   Displays the first page of the report.
* **![Previous Page button](https://devnet.logianalytics.com/hc/article_attachments/4404893801879/btn_prvspg.gif) Previous Page**  
   Displays the previous page of the report.
* **![Next Page button](https://devnet.logianalytics.com/hc/article_attachments/4404889284503/btn_nxtpg.gif) Next Page**  
   Displays the next page of the report.
* **![Last Page button](https://devnet.logianalytics.com/hc/article_attachments/4404889284759/btn_lastpg.gif) Last Page**  
   Displays the last page of the report.
* **![Toc button](https://devnet.logianalytics.com/hc/article_attachments/4404893802135/btn_toc.gif) Toc**  
   Shows the table of contents.
* **![Refresh Data button](https://devnet.logianalytics.com/hc/article_attachments/4404893802519/btn_refresh.gif) Refresh Data**  
   Refreshes the data with the same parameters.
* **![Refetch Data button](https://devnet.logianalytics.com/hc/article_attachments/4404893802775/btn_refetchdt.gif) Refetch Data**  
   Re-fetches data to re-run the report with different parameters.
* **![Stop button](https://devnet.logianalytics.com/hc/article_attachments/4404893803031/btn_stop.gif) Stop**  
   Select this button if you want Logi JReport to stop generating the report.
* **![Back button](https://devnet.logianalytics.com/hc/article_attachments/4404893803287/btn_bck.gif) Back**  
   Navigates to the last view.
* **![Forward button](https://devnet.logianalytics.com/hc/article_attachments/4404889285015/btn_fwd.gif) Forward**  
   Navigates forward.
* **![Drag Tool button](https://devnet.logianalytics.com/hc/article_attachments/4404893803799/btn_drgtl.gif) Drag Tool**  
   Drags the page up and down.  
    
   Select this button, and place your mouse pointer on any blank area of the report. When the mouse pointer becomes a small hand, you can drag the page.
* **![Zoom In Tool button](https://devnet.logianalytics.com/hc/article_attachments/4404893804055/btn_zmin.gif) Zoom In Tool**  
   Zooms in to see more detail. Select the Zoom In tool then select in the border of the report repeatedly to zoom in.
* **![Zoom Out Tool button](https://devnet.logianalytics.com/hc/article_attachments/4404889285271/btn_zmout.gif) Zoom Out Tool**  
   Zooms out to see the overall view of the page. Select the Zoom Out tool then select on the border of the report repeatedly to zoom out.

**Report tab bar**

The report tab bar is for switching among report tabs in a page report. It also provides a shortcut menu that allows you to rename, duplicate, or remove a report tab. For details, see [Managing Report Tabs in a Page Report](https://devnet.logianalytics.com/hc/en-us/articles/1500009592941-Managing-Report-Tabs-in-a-Page-Report).
