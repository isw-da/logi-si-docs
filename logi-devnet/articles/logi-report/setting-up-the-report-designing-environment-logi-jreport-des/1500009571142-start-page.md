# Start Page

When Logi JReport Designer is started, by default the Start Page is displayed which provides some quick entries to the key functions of Logi JReport Designer. If you do not want to display the page, you can check the **Do not show this page at start up** option at the bottom left of the page. The Start Page can be opened at any time by selecting Quick Start at the top right corner of the Logi JReport Designer main window.

![Start Page](https://devnet.logianalytics.com/hc/article_attachments/4404889279255/rptenv_start.gif)

**Connect**

The most popular connection types are listed in this category, You can select any one to [set up a corresponding connection](https://devnet.logianalytics.com/hc/en-us/articles/1500009563942-Connecting-to-Your-Data-Sources) to connect your database with a Logi JReport catalog. The More Connections item directs your to a full list of the connection types Logi JReport supports.

**Create**

You can create a page report, web report or library component by selecting the link in the category. The report will be created into the catalog that was in use when Logi JReport Designer is closed the last time.

**Open**

You can browse to open a report in Logi JReport Designer. The recently opened reports are also listed. You can select any one to open it directly.

**Tutorial**

Lists the topics in the Logi JReport Tutorial manual. Select any to learn about Logi JReport.
