# Search Result Panel

The Search Result panel lists the result of searching for a particular resource using the search function in the Catalog Manager or the Data panel. You can select any item to locate it in the Catalog Manager or Data panel to edit it as needed.

![Search Result panel](https://devnet.logianalytics.com/hc/article_attachments/4404893794327/rptenv_srchrst.gif)

**Resource Name**

Lists the resource names.

**Resource Type**

Shows the resource types.

**Resource Path**

Displays the resource paths.
