# Design/View Area

This topic introduces the Design/View area of Logi JReport Designer.

![Design/View Area](https://devnet.logianalytics.com/hc/article_attachments/4404909115415/rptenv_area.gif)

The Design/View area, or report designing window, comprises of the following parts:

**Title bar**

The title bar shows the report file name, and three buttons for you to minimize, maximize and close the active report.

**Design tab**

The Design tab, which is also called the report editor, is for laying out your report. You can make use of rulers, grids and guidelines to position objects. The appearance of the tab can be customized via the [Options dialog](https://devnet.logianalytics.com/hc/en-us/articles/1500010031962-Options-Dialog#Editor).

**View tab**

The View tab is for previewing the report layout and result. The report may not display exactly as the user would see it in Logi JReport Server. Interactive objects such as parameter and filter objects are not active and report links are not active. In this tab there is a toolbar which consists of the following buttons:

* **![First Page button](https://devnet.logianalytics.com/hc/article_attachments/4404909115671/btn_fstpg.gif) First Page**  
   Displays the first page of the report.
* **![Previous Topic button](https://devnet.logianalytics.com/hc/article_attachments/4404901117975/btn_prvspg.gif) Previous Topic**  
   Displays the Previous Topic of the report.
* **![Next Topic button](https://devnet.logianalytics.com/hc/article_attachments/4404901118231/btn_nxtpg.gif) Next Topic**  
   Displays the Next Topic of the report.
* **![Last Page button](https://devnet.logianalytics.com/hc/article_attachments/4404909115927/btn_lastpg.gif) Last Page**  
   Displays the last page of the report.
* **![Toc button](https://devnet.logianalytics.com/hc/article_attachments/4404901118487/btn_toc.gif) Toc**  
   Shows the table of contents.
* **![Refresh Data button](https://devnet.logianalytics.com/hc/article_attachments/4404909116183/btn_refresh.gif) Refresh Data**  
   Refreshes the data with the same parameters.
* **![Refetch Data button](https://devnet.logianalytics.com/hc/article_attachments/4404909116439/btn_refetchdt.gif) Refetch Data**  
   Re-fetches data to re-run the report with different parameters.
* **![Stop button](https://devnet.logianalytics.com/hc/article_attachments/4404909116823/btn_stop.gif) Stop**  
   select this button if you want Logi JReport to stop generating the report.
* **![Back button](https://devnet.logianalytics.com/hc/article_attachments/4404901118743/btn_bck.gif) Back**  
   Navigates to the last view.
* **![Forward button](https://devnet.logianalytics.com/hc/article_attachments/4404909117079/btn_fwd.gif) Forward**  
   Navigates forward.
* **![Drag Tool button](https://devnet.logianalytics.com/hc/article_attachments/4404909117335/btn_drgtl.gif) Drag Tool**  
   Drags the page up and down.  
    
   select this button, and place your mouse pointer on any blank area of the report. When the mouse pointer becomes a small hand, you can drag the page.
* **![Zoom In Tool button](https://devnet.logianalytics.com/hc/article_attachments/4404901118999/btn_zmin.gif) Zoom In Tool**  
   Zooms in to see more detail. Select the Zoom In tool then select in the border of the report repeatedly to zoom in.
* **![Zoom Out Tool button](https://devnet.logianalytics.com/hc/article_attachments/4404901119255/btn_zmout.gif) Zoom Out Tool**  
   Zooms out to see the overall view of the page. Select the Zoom Out tool then select on the border of the report repeatedly to zoom out.

**Report tab bar**

The report tab bar is for switching among report tabs in a page report. It also provides a shortcut menu that allows you to rename, duplicate, or remove a report tab. For details, see [Managing the report tabs in a page report](https://devnet.logianalytics.com/hc/en-us/articles/1500010070881-Managing-Page-Reports#Tab).
