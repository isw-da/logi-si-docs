# Components Panel

The Components panel lists all the components that can be inserted into a report (to display the panel, check **Components** on the Home or View menu tab).

![Components panel](https://devnet.logianalytics.com/hc/article_attachments/4404893801111/rptenv_cmpnt.gif)

Components in the panel are classified as follows and they vary according to different report types.

* **Visual**: chart, map, KPI, rank
* **Grid**: table, crosstab and tabular
* **Basic**: label, text box, image, subreport, banded object
* **Web Controls**
* **Others**: UDO, barcode, and multimedia objects

You can view the components in either icon view or list view. Select ![List View button](https://devnet.logianalytics.com/hc/article_attachments/4404893801367/btn_listvw.gif) to switch to list view, or ![Icon View button](https://devnet.logianalytics.com/hc/article_attachments/4404889284247/btn_iconvw.gif) to icon view.

The method to create an component using the Components panel is to drag a button representing that component from the panel to the desired location in the report design area. For different report type, the components available in panel vary.
