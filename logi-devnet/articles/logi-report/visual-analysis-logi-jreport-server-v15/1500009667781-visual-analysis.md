# Visual Analysis Logi JReport Server Guide v15

Visual Analysis is a Logi JReport Server WYSIWYG product you use to visualize the result of every step of your work. Simply by dragging and dropping data fields onto a layout module, you are able to visually create crosstabs and charts step by step. Visual Analysis has the following main features:

* Using dimensions & measures to quickly highlight colors, sizes & shapes
* Instantly change data visualizations with over 50 options
* Powerful in-memory cube technology

[Business views](https://devnet.logianalytics.com/hc/en-us/articles/1500009675121-An-Introduction-to-Business-Views) are the data sources used in Visual Analysis. Each time you can perform data analysis based on one business view, and later you can save the result as an analysis template into the server resource tree. However, Visual Analysis does not support the customization of parameter values, so if a business view contains parameters, the default parameter values will be applied when you perform visual analysis on the business view.

Visual Analysis is a separately licensed feature of Logi JReport Server. It is installed together with Logi JReport Server so only the license key needs to be updated to enable it to run. To find out how to license Visual Analysis please contact Jinfonet sales at [sales@jinfonet.com](mailto:sales@jinfonet.com) or contact your Enterprise Account Manager.

You can also use URLs to start Visual Analysis and work on it. For details, refer to [Accessing Visual Analysis via URL](https://devnet.logianalytics.com/hc/en-us/articles/1500009650562-Accessing-Visual-Analysis).

**Notes:**

* Visual Analysis cannot run on Internet Explorer 8.
* If Visual Analysis cannot work well on your Google Chrome, you can try disabling GPU accelerated 2D canvas as follows: access the URL `chrome://flags`, disable accelerated 2D canvas, and then relaunch Chrome.
