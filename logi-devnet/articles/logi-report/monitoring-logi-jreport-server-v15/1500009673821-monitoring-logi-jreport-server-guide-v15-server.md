# Monitoring Logi JReport Server Logi JReport Server Guide v15

Logi JReport Server Monitor is a standalone web-based application used to monitor the overall performance of Logi JReport Server. It allows you to inspect the status of Logi JReport Server, including the status of the servers in the cluster, the status of different reports, the status of on-line users and so on. Logi JReport Server Monitor is able to generate and display the performance chart of Logi JReport Server according to its statistics. Thus you can view the performance of Logi JReport Server in the form of Line Chart Graph and Text. You can also use Logi JReport Server Monitor to maintain Logi JReport Server, such as shutting down servers, stopping waiting/running tasks, logging out a valid user session and so on. By creating profiling reports using Logi JReport Server Monitor, you can inspect server performance in a certain period of time.

**To access Logi JReport Server Monitor:**

1. Launch MonitorServer.bat in `<monitor_install_root>\bin`.
2. Select **Monitor** on the system toolbar of the Logi JReport Administration page.

**Note:** The Monitor tab will not be displayed on the Logi JReport Administration page when the web.monitor.link.enable property in the server.properties file in <server\_install\_root>\bin is set to false.
