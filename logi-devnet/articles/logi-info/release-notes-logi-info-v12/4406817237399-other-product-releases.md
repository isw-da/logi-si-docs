# Other Product Releases

Logi Ad Hoc customers with a Logi Maintenance Plan contract who want to upgrade to a new product version can do so by logging into DevNet and using the **Support**![](https://devnet.logianalytics.com/hc/article_attachments/4417583616279/menuarrowrt.png)**Product Download** menu item.

All others should contact [Logi Customer
Service](mailto:CustomerService@LogiAnalytics.com?Subject=Upgrade%20Request) for
more information.

For more information about release content, see [Logi Info v12 Release Notes](https://devnet.logianalytics.com/hc/en-us/articles/4409573704471-Logi-Info-v12-Release-Notes).
