# Other Product Releases

Logi Ad Hoc customers with a Logi Maintenance Plan contract who want to upgrade to a new product version can do so by logging into DevNet and using the **Support**![](https://devnet.logianalytics.com/hc/article_attachments/7522803222807/menuarrowrt.png)**Product Download** menu item.

All others should contact [Logi Customer
Service](mailto:CustomerService@LogiAnalytics.com?Subject=Upgrade%20Request) for
more information.

For more information about release content, see [Release Notes](https://clm.logianalytics.com/rdPage.aspx?rdReport=RelNotesINF).
