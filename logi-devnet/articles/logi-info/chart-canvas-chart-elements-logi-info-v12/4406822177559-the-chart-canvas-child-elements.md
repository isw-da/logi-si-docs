# The Chart Canvas Child Elements

The Chart Canvas element also uses the following child elements to generate, control, and format the canvas and charts. Select the element link for more information.

| Element | Description |
| --- | --- |

| [Caption Style and SubCaption Style](https://devnet.logianalytics.com/hc/en-us/articles/4406822057111-Caption-Style-and-SubCaption-Style) | Specifies Caption and SubCaption font-related and positioning attributes. |

| [Input Selection](https://devnet.logianalytics.com/hc/en-us/articles/4406822104983-Input-Selection) | Provides the ability to select chart areas or points with the mouse. |

![](https://devnet.logianalytics.com/hc/article_attachments/4417583587863/noteicon.png) There is no "Series.Polar" element. To produce a Polar chart, set the Chart Canvas element's **Polar** attribute to *True*, which converts Cartesian series (Area, Line, Bar, etc.) to the Polar/Radial coordinate system.
