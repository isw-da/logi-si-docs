# Series.Funnel - Using Input Selection

This series can also be used with the **Input Selection** family of elements, which turn the chart into an input control. This allows users, at runtime, to select points or values on the chart with their mouse and your report can then take action using the selected data. This is very useful, for example, for drilling into the data. For more information about this functionality, see [Input Selection](https://devnet.logianalytics.com/hc/en-us/articles/4406822104983-Input-Selection).
