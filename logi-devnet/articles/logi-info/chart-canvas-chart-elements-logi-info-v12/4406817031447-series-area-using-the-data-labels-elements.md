# Series.Area - Using the Data Labels Elements

A "data label" is text shown next to each data point that shows its value. When the **Data Labels** element is used as a child of Series.Area, text representing the data values will appear on the chart:

  

![](https://devnet.logianalytics.com/hc/article_attachments/4417584163735/series_area_07.png)

The Data Labels element has attributes that allow you to control the font family, color, size, and weight, the data format, border color, and positioning of the text.

The Data Labels element's color-related attribute values can be set using @Chart tokens.
