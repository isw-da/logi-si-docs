# Create .NET Plug-in

"Plug-ins" give Logi developers the ability to *programmatically extend* the functionality of their Logi applications.

The following topics discuss the development of Logi plug-ins using the .NET framework:

* [Writing Your Plug-in](https://devnet.logianalytics.com/hc/en-us/articles/4406822207255--NET-Plug-in-Writing-Your-Plug-in)
* [Example: Change the Application Caption](https://devnet.logianalytics.com/hc/en-us/articles/4406817212439--NET-Plug-in-Example-Change-the-Application-Caption#exAppCaption)
* [Example: Modify an SQL Query with a Request Variable](https://devnet.logianalytics.com/hc/en-us/articles/4406822206359--NET-Plug-in-Example-Modify-an-SQL-Query-with-a-Request-Variable#exModSQL)
* [Example: Changing Data in a Datalayer](https://devnet.logianalytics.com/hc/en-us/articles/4406817213463--NET-Plug-in-Example-Changing-Data-in-a-Datalayer#Datalayer)
* [Implementing Your Plug-in](https://devnet.logianalytics.com/hc/en-us/articles/4406817214615--NET-Plug-in-Implementing-Your-Plug-in#Implementing)
* [Debugging in Visual Studio 2012](https://devnet.logianalytics.com/hc/en-us/articles/4406822205463--NET-Plug-in-Debugging-in-Visual-Studio-2012#VS2012)
* [Debugging in Visual Studio 2008/2005](https://devnet.logianalytics.com/hc/en-us/articles/4406822204567--NET-Plug-in-Debugging-in-Visual-Studio-2008-5#VS2005)
* [Plug-in Class Properties and Methods](https://devnet.logianalytics.com/hc/en-us/articles/4406817215895--NET-Plug-in-Plug-in-Class-Properties-and-Methods#PluginParams)
