# Embedded Reports API

The **Embedded Reports API** provides developers with easy and agile methods for embedding Logi report components into other, non-Logi web pages.

The following topics provide information about the API:

* [Including the API](https://devnet.logianalytics.com/hc/en-us/articles/4406817407255-Including-the-API)
* [Embedding Static Reports using Markup](https://devnet.logianalytics.com/hc/en-us/articles/4406822310679-Embedding-Static-Reports-using-Markup#Markup)
* [Embedding Dynamic Reports using JavaScript](https://devnet.logianalytics.com/hc/en-us/articles/4406817405975-Embedding-Dynamic-Reports-using-JavaScript#EmbeddingJs)
* [Accessing Embedded Reports on Different Domains](https://devnet.logianalytics.com/hc/en-us/articles/4406822309783-Accessing-Embedded-Reports-on-Different-Domains#EmbedDD)
* [Preventing Session Timeout](https://devnet.logianalytics.com/hc/en-us/articles/4406822312471-Preventing-Session-Timeout#Timeout)

![](https://devnet.logianalytics.com/hc/article_attachments/4417562936855/criticalicon.png) The Windows version of the **Safari** browser does not currently work with the Embed API.
