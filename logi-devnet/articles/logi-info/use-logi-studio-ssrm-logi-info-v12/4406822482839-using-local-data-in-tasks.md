# Using Local Data in Tasks

The **Local Data** element is available for use in Process definitions.

![](https://devnet.logianalytics.com/hc/article_attachments/4417575560727/workproctask_12.png)

It operates in them in the same manner that it does when used in Report definitions and the data its datalayer retrieves can be accessed from any task in the Process definition using an @Local Data token. For a full explanation of this element, see our [Datalayers](https://devnet.logianalytics.com/hc/en-us/articles/4406817655319-Datalayers) document.
