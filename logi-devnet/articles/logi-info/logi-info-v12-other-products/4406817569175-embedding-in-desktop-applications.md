# Embedding in Desktop Applications

If developers want to **embed** Logi application output in their **desktop applications**, this can be easily accomplished.
Their desktop application just needs the ability to make an **HTTP request** and to **render** the **HTTP response**. Many development languages include browser objects that can do this; for example Microsoft Visual Studio includes a Web Browser component. Once again, the formal interaction with a Logi application is a simple URL call rather than actual code integration.
