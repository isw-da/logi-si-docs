# Working with Logi Security

**Logi Security** can be used to secure reports and data from being viewed by unauthorized users. This topic is for developers who already understand the underlying concepts of Logi Security and it discusses the elements and mechanisms involved in its implementation.

The following topics discuss the use of Logi Security:

* [The Security Element](https://devnet.logianalytics.com/hc/en-us/articles/4406818122519-The-Security-Element#The)
* [Authenticating the User](https://devnet.logianalytics.com/hc/en-us/articles/4406822644375-Authenticating-the-User#Authenticating)
* [Gathering Role Information](https://devnet.logianalytics.com/hc/en-us/articles/4406818119447-Gathering-Role-Information#RoleInfo)
* [Working with Rights](https://devnet.logianalytics.com/hc/en-us/articles/4406818123543-Working-with-Rights#DefRights)
* [Securing Reports and Elements](https://devnet.logianalytics.com/hc/en-us/articles/4406818121495-Securing-Reports-and-Elements#SecReports)
* [Securing Data](https://devnet.logianalytics.com/hc/en-us/articles/4406818120471-Securing-Data#SecData)

If you're unfamiliar with Logi Security, you should begin by reading [Logi Security](https://devnet.logianalytics.com/hc/en-us/articles/4406822422295-Logi-Security).
