# Starting the Default Web Site

To start the IIS "Default Web Site", open the IIS Manager
tool:

![](https://devnet.logianalytics.com/hc/article_attachments/4417583639063/iisreg_03.png)

As shown in the **IIS 8 Manager** example above, locate and select the
Default Web Site. To start the site, if it's stopped, click the
**Start** button in the toolbar, which is circled above. Or, you can
right-click the web site entry and start or stop it from the pop-up
context menu that appears.
