# Mobile Reporting Sample Application

## **Application Description**

This sample demonstrates Logi Mobile Reporting features, for Logi applications on mobile devices. These features are available in Logi Info only. Logi Report developers interested in mobile reports should see the Mobile Web Sample App.

Mobile Reporting apps are "mobile device microsites" and are browsed using the device's browser. Features presented include:

* HTML table
* Animated Bar chart
* Gauges
* Mobile Dashboard
* Data table, with demand paging
* User input examples

You can view this sample **on your mobile device** at: [sampleapps.logianalytics.com/samplemobilereporting](http://sampleapps.logianalytics.com/samplemobilereporting)

![Mobile_reporting.PNG](https://devnet.logianalytics.com/hc/article_attachments/360074100934/Mobile_reporting.PNG)

## **Installation Instructions**

1. Download and extract the sample application zip file that is attached to this knowledge base article
2. Create a new Logi Application using the Logi Studio IDE - follow the installation wizard instructions
3. Navigate to the \_Settings file, update the path element to point to the installation directory
4. In your file system, navigate to Logi application installation directory and replace the \_Definitions, \_SupportFiles, and \_Themes directory with the content from the sample application zip file
5. Run the application
