# Logi Info - Clarity Sample Theme

![themeClarityLg.jpg](https://devnet.logianalytics.com/hc/article_attachments/360068993274/themeClarityLg.jpg)

## **Theme Description**

This theme offers a clean look that's "easy on the eyes". Updated for v12.1.

## **Installation Instructions**

Download the ZIP file containing the sample theme. Extract the content of the ZIP file into the \_Themes directory. Under the \_Settings file, modify the themes element to reflect the new theme.
