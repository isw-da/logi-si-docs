# Definition Modifier Files Sample Application

## **Application Description**

This application demonstrates the effects of the Definition Modifier File (DMF). DMFs allow Logi Info developers to modify any of the elements or attributes in a report definition, based on conditional criteria, at runtime.

In this sample, when a DMF is included, the report's data table caption and column header text is translated to English, French, or German, and export links are added to the report footer.

![DMF.PNG](https://devnet.logianalytics.com/hc/article_attachments/360075809333/DMF.PNG)

## **Installation Instructions**

1. Download and extract the content of the sample application zip file that is attached to this knowledge base article
2. Create a new Logi Application using the Logi Studio IDE - follow the installation wizard instructions
3. Navigate to the \_Settings file, update the path element to point to the installation directory
4. In your file system, navigate to Logi application installation directory and replace the \_Definitions, \_SupportFiles, and \_Themes directory with the content from the sample application zip file
5. Run the sample application
