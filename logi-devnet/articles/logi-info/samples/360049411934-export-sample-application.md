# Export Sample Application

## **Application Description**

This sample application demonstrates the use of Action elements to export reports to various formats:

* Export report data to a CSV file
* Export report as an Excel file
* Export report as an Adobe PDF file
* Export report, using different styling, as an Adobe PDF file
* Export report as a Word document
* Export report data to an XML data file
* Use of Excel Column Format elements to control formatting
* Use of built-in Show Modes values to control which elements are exported

For Logi Info developers, there's also a sample application, Process Task: Export, available that demonstrates how to export data from a process definition task.

![Export.PNG](https://devnet.logianalytics.com/hc/article_attachments/360074048634)

## **Installation Instructions**

1. Download and extract the sample application zip file that is attached to this knowledge base article
2. Create a new Logi Application using the Logi Studio IDE - follow the installation wizard instructions
3. Navigate to the \_Settings file, update the path element to point to the installation directory
4. In your file system, navigate to Logi application installation directory and replace the \_Definitions, \_SupportFiles, and \_Themes directory with the content from the sample application zip file
5. Run the application
