# Logi Info - Silver Sample Theme

![themeSilverLg.jpg](https://devnet.logianalytics.com/hc/article_attachments/360070086233/themeSilverLg.jpg)

## **Theme Description**

Enjoy the slick look of this theme, which produces a metallic and futuristic appearance. Updated for v12.1.

## **Installation Instructions**

Download the ZIP file containing the sample theme. Extract the content of the ZIP file into the \_Themes directory. Under the \_Settings file, modify the themes element to reflect the new theme.
