# Crime Stats Sample Application

## **Application Description**

This sample application demonstrates a variety of Chart Canvas Charts and advanced development techniques, including:

* Bar, Bubble, Pie, and Scatter chart series
* Bar and Indicator gauges
* Tabs and AJAX refresh of report sections
* Use of Subreports
* Use of a custom theme
* Use of custom Quicktips with charts

The report definitions use included Excel data files as a data source and no connection to a database server is required.

![Crime_Stats.PNG](https://devnet.logianalytics.com/hc/article_attachments/360075807833/Crime_Stats.PNG)

## **Installation Instructions**

1. Download and extract the content of the sample application zip file that is attached to this knowledge base article
2. Create a new Logi Application using the Logi Studio IDE - follow the installation wizard instructions
3. Navigate to the \_Settings file, update the path element to point to the installation directory
4. In your file system, navigate to Logi application installation directory and replace the \_Definitions, \_SupportFiles, and \_Themes directory with the content from the sample application zip file
5. Run the sample application
