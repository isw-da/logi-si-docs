# Arizona Sample Theme

![themeArizonaLg.jpg](https://devnet.logianalytics.com/hc/article_attachments/360068990994)

## **Theme Description**

A theme with earthy tones in the grand tradition of the American Southwest. This is a custom theme only available on DevNet.

## **Installation Instructions**

Download the ZIP file containing the sample theme. Extract the content of the ZIP file into the \_Themes directory. Under the \_Settings file, modify the themes element to reflect the new theme.
