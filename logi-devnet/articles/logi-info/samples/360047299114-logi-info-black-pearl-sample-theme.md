# Logi Info - Black Pearl Sample Theme

![themeBlackPearlLg.jpg](https://devnet.logianalytics.com/hc/article_attachments/360070080533/themeBlackPearlLg.jpg)

## **Theme Description**

Paint it black, baby! This black theme includes shades of black and gray, lots of highlights, and other slick stuff for your reporting interface pleasure. Updated for v12.1.

## **Installation Instructions**

Download the ZIP file containing the sample theme. Extract the content of the ZIP file into the \_Themes directory. Under the \_Settings file, modify the themes element to reflect the new theme.
