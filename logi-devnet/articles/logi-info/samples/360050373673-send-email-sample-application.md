# Send Email Sample Application

## **Application Description**

This sample application demonstrates the use of process tasks to send reports via email:

* Evaluate request variable for conditional processing
* If necessary, create a new temporary folder for attachment file
* Build filename using session variables
* Set session variables
* Send email with report embedded within it
* Send email with report as a HTML attachment
* Send email with report as a PDF attachment

![sendemail.PNG](https://devnet.logianalytics.com/hc/article_attachments/360074592254)

## **Installation Instructions**

1. Download and extract the content of the sample application zip file that is attached to this knowledge base article
2. Create a new Logi Application using the Logi Studio IDE - follow the installation wizard instructions
3. Navigate to the \_Settings file, update the path element to point to the installation directory
4. In your file system, navigate to Logi application installation directory and replace the \_Definitions, \_SupportFiles, and \_Themes directory with the content from the sample application zip file
5. Run the sample application.

   Click **[here](https://devnet.logianalytics.com/hc/article_attachments/360074592334)** to download the SampleSchedulerConsole.lgxzip.
