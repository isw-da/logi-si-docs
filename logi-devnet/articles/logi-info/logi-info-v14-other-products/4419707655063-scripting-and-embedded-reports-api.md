# Scripting and Embedded Reports API

Logi applications support the use of JavaScript within them and can reference external scripting libraries, such as JQuery. Logi Info also includes our own [Embedded Reports API](https://devnet.logianalytics.com/hc/en-us/articles/4419715274135-Embedded-Reports-API), with makes it a breeze to embed active Logi reports into other HTML pages.
