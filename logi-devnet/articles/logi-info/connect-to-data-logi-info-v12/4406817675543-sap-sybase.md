# SAP Sybase

As mentioned earlier, the data provider for Sybase is *not* distributed with Logi Info. You must have installed the ADO.NET driver from Sybase, which is typically included with your Sybase installation software, or can be [downloaded from the Sybase web site](https://www.progress.com/net/sybase).
