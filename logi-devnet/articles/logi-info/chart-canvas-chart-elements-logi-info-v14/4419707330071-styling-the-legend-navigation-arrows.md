# Styling the Legend Navigation Arrows

It's possible for your legend to include so many items that it exceeds the height of the legend (as set by either the height of the canvas or the legend's Maximum Height attribute).

![](https://devnet.logianalytics.com/hc/article_attachments/4419706655895/introcccharts_112a.png)

In this case, as shown above, the legend items will automatically be placed in a scrolling list within the legend, and up-down navigation arrows and a "page counter" will appear. The **Legend Navigation Style** element can be added beneath the Legend element to control the color and size of the navigation arrows.
