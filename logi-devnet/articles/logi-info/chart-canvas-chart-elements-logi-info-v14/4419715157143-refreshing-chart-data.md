# Refreshing Chart Data

Chart visualizations can be updated by refreshing their data using two methods:

* The **Refresh Series Timer** element is discussed in the documentation for individual Series elements.
* The [Logi JavaScript API for Chart Canvas Charts](https://devnet.logianalytics.com/hc/en-us/articles/4419707331351-Logi-JavaScript-API-for-Chart-Canvas-Charts) is a JavaScript library that developers can use to load data into Chart Canvas Charts, updating the visualizations whenever desired.
