# Series.Bubble - Using the Marker Points Element

A "marker point" is a symbol that appears on the chart at each
data point. These behave differently in a Bubble chart than in other types
of charts: the marker point *is* the bubble itself. When the
**Marker Points** element is used as a child of Series.Bubble, several
properties can be configured.

![](https://devnet.logianalytics.com/hc/article_attachments/4419699564183/series_bubble_07.png)
The default marker points are shown above. The Marker Points element allows
you to control the bubble color, border color, and transparency. When the
cursor hovers over it, a bubble's color can change - the
**Marker Points Hover Style** child element lets you configure that
behavior and other properties.
