# Styling the Legend Caption

The Legend's **Caption** attribute can be set to place a caption inside the Legend:

![](https://devnet.logianalytics.com/hc/article_attachments/4419699553303/introcccharts_110.png)

As shown above, the **Caption Style** element can be added to configure the caption's font family, color, size, and weight.
