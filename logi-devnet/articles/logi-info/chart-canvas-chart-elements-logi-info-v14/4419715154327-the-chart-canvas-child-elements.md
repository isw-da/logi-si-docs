# The Chart Canvas Child Elements

The Chart Canvas element also uses the following child elements to generate, control, and format the canvas and charts. Select the element link for more information.

![](https://devnet.logianalytics.com/hc/article_attachments/7522725179159/noteicon.png) There is no "Series.Polar" element. To produce a Polar chart, set the Chart Canvas element's **Polar** attribute to *True*, which converts Cartesian series (Area, Line, Bar, etc.) to the Polar/Radial coordinate system.
