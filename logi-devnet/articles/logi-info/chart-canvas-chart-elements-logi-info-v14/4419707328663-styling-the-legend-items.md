# Styling the Legend Items

The text of the legend items can be styled using the **Label Style** element:

![](https://devnet.logianalytics.com/hc/article_attachments/4419706655639/introcccharts_111.png)
As shown in the example above, this element can be added to configure the legend items' font family, color, size, and weight.  

![](https://devnet.logianalytics.com/hc/article_attachments/4419699553559/introcccharts_112.png)

A related element, the **Hidden Label Style** element, is used to configure of the font family, color, size, and weight that's used when a legend item is clicked to engage Legend Filtering, as shown above. The color affects both the item symbol and the item text and, without the element, defaults to light gray, as shown above.
