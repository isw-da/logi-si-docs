# Series.Area Spline - Using the Trend Line Element

The **Trend Line** element creates a line on the chart that indicates the "trend" of the data. The line connects a number of data points generated using a regression algorithm.

![](https://devnet.logianalytics.com/hc/article_attachments/4419699557655/series_areaspline_09.png)

The Trend Line element is a child of the Series.Area Spline element and can be styled for color, line width, etc. When configured with a legend label, it will be represented by an item in the legend, as shown above.
