# Create a Java Plug-in

"Plug-ins" give Logi Info developers the ability to *programmatically extend* the functionality of their Logi applications.

The following topics discuss the development of Logi plug-ins using Java:

* [Writing Your Plug-in](https://devnet.logianalytics.com/hc/en-us/articles/4419707435927-Java-Plug-in-Writing-Your-Plug-in#Writing)
* [Example: Change the Application Caption](https://devnet.logianalytics.com/hc/en-us/articles/4419715185047-Java-Plug-in-Example-Change-the-Application-Caption#exAppCaption)
* [Example: Modify an SQL Query with a Request Variable](https://devnet.logianalytics.com/hc/en-us/articles/4419715186071-Java-Plug-in-Example-Modify-a-SQL-Query-with-a-Request-Variable#exModSQL)
* [Implementing Your Plug-in](https://devnet.logianalytics.com/hc/en-us/articles/4419707434775-Java-Plug-in-Implementing-Your-Plug-in#Implementing)
* [Debugging Example: Eclipse and Tomcat](https://devnet.logianalytics.com/hc/en-us/articles/4419707433751-Java-Plug-in-Debugging-Example-Eclipse-and-Tomcat#Debugging)
* [Plug-in Class Methods and Properties](https://devnet.logianalytics.com/hc/en-us/articles/4419715186967-Java-Plug-in-Plug-in-Class-Methods-and-Properties#Properties)
