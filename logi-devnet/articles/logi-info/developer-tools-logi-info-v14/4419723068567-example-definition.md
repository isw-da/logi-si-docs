# Example Definition

Here's a complete report definition that includes all of the previous code
examples and some embellishments.
[Download it](https://clm.logianalytics.com/downloads/info/GM_API_Sample.txt) from DevNet and when it opens in your browser, save it, with an .lgx file
extension, to the \_Definitions\\_Reports folder in one of your Logi
applications. The application version must be Logi Info v12.0.036 SP2 or
later for it to work.
After you save the file, in Studio you may need to right-click the Reports
folder in the Application panel and select **Refresh Application** in
order to see the new definition.
