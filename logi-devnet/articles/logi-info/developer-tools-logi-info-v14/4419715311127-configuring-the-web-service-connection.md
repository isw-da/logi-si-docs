# Configuring the Web Service Connection

The web service connection requires the
**Connection.Google Maps** element:

![](https://devnet.logianalytics.com/hc/article_attachments/4419706849943/gmtutorial_01.png)

For detailed information about getting an API Key and configuring this element, see
[Google Connections](https://devnet.logianalytics.com/hc/en-us/articles/4419707607703-Google-Connections).

Logi Info now includes the **Connection.Nominatim** element, for
connections to a free, public, OpenStreetMap (OSM)
[Nominatim](https://wiki.openstreetmap.org/wiki/Nominatim)
server for geographic data. More information is available in [Datasource Connections](https://devnet.logianalytics.com/hc/en-us/articles/4419707715735-Datasource-Connections).

We don't need any other connections for this tutorial: you're going to
create sample Google Maps using data from two different XML data files,
which do not require connections.
