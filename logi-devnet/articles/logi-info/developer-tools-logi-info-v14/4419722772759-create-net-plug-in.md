# Create .NET Plug-in

"Plug-ins" give Logi developers the ability to *programmatically extend* the functionality of their Logi applications.

The following topics discuss the development of Logi plug-ins using the .NET framework:

* [Writing Your Plug-in](https://devnet.logianalytics.com/hc/en-us/articles/4419707431703--NET-Plug-in-Writing-Your-Plug-in)
* [Example: Change the Application Caption](https://devnet.logianalytics.com/hc/en-us/articles/4419722773911--NET-Plug-in-Example-Change-the-Application-Caption#exAppCaption)
* [Example: Modify an SQL Query with a Request Variable](https://devnet.logianalytics.com/hc/en-us/articles/4419722774935--NET-Plug-in-Example-Modify-an-SQL-Query-with-a-Request-Variable#exModSQL)
* [Example: Changing Data in a Datalayer](https://devnet.logianalytics.com/hc/en-us/articles/4419707429399-Java-Plug-in-Example-Changing-Data-in-a-Datalayer#Datalayer)
* [Implementing Your Plug-in](https://devnet.logianalytics.com/hc/en-us/articles/4419715181591--NET-Plug-in-Implementing-Your-Plug-in#Implementing)
* [Debugging in Visual Studio 2012](https://devnet.logianalytics.com/hc/en-us/articles/4419715179927--NET-Plug-in-Debugging-in-Visual-Studio-2012#VS2012)
* [Debugging in Visual Studio 2008/2005](https://devnet.logianalytics.com/hc/en-us/articles/4419715178007--NET-Plug-in-Debugging-in-Visual-Studio-2008-5#VS2005)
* [Plug-in Class Properties and Methods](https://devnet.logianalytics.com/hc/en-us/articles/4419707430551--NET-Plug-in-Plug-in-Class-Properties-and-Methods#PluginParams)
