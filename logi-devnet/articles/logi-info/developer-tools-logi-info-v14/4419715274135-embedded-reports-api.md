# Embedded Reports API

The **Embedded Reports API** provides developers with easy and agile methods for embedding Logi report components into other, non-Logi web pages.

The following topics provide information about the API:

* [Including the API](https://devnet.logianalytics.com/hc/en-us/articles/4419715275287-Including-the-API)
* [Embedding Static Reports using Markup](https://devnet.logianalytics.com/hc/en-us/articles/4419707556375-Embedding-Static-Reports-using-Markup#Markup)
* [Embedding Dynamic Reports using JavaScript](https://devnet.logianalytics.com/hc/en-us/articles/4419715272983-Embedding-Dynamic-Reports-using-JavaScript#EmbeddingJs)
* [Accessing Embedded Reports on Different Domains](https://devnet.logianalytics.com/hc/en-us/articles/4419707555351-Accessing-Embedded-Reports-on-Different-Domains#EmbedDD)
* [Preventing Session Timeout](https://devnet.logianalytics.com/hc/en-us/articles/4419715276183-Preventing-Session-Timeout#Timeout)

![](https://devnet.logianalytics.com/hc/article_attachments/7522725232151/criticalicon.png) The Windows version of the **Safari** browser does not currently work with the Embed API.
