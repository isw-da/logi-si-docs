# Conditions Example: Setting Session Variables

The ability to conditionally set
Session variables
is very useful. These variables with global scope provide a great way to
share small amounts of data across report definitions. For more information, see [Session Variables](https://devnet.logianalytics.com/hc/en-us/articles/4419723341847-Session-Variables).

![](https://devnet.logianalytics.com/hc/article_attachments/7522731719191/usingcond_01.png)

As shown in the example above, the **Set Session Variables** element
will run and create Session variables *only* if the formula in its
**Condition** attribute is true.
