# Creating the Basic Application

Let's get started by building your Google Map application and placing the
sample data appropriately:

1. In Logi Studio, **create** and **register** a new application.
   Test it to ensure that it works.
2. Extract all four files from the sample data .zip file downloaded in the
   previous section and save them in the \_SupportFiles folder under your
   Logi application folder.
