# DataLayer.XML - Using DataLayer.XML with Input Elements

One convenient use of XML data files is as a source of options for menus and list-type Input elements. An XML data file can be created quickly and is easy to edit and, unlike static data hard-coded into the application, an XML data file can be edited externally. Here's an example of a common use of XML data and DataLayer.XML for this purpose:

![](https://devnet.logianalytics.com/hc/article_attachments/4419714724503/dlxmlfile_03.png)

A **DataLayer.XML** has been added beneath an **Input Select List** element and its attributes set to use the sample XML file, shown in [DataLayer.XML - Working with DataLayer.XML](https://devnet.logianalytics.com/hc/en-us/articles/4419707510807-DataLayer-XML-Working-with-DataLayer-XML). In the Input Select List's attributes, "Brand" is identified as the **Caption Column** and "Code" as the **Value Column**.
