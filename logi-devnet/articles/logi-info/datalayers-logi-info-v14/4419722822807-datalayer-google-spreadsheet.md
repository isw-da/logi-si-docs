# DataLayer.Google Spreadsheet

**Google Docs** is a free, web-based word processor, spreadsheet, and presentation application offered as a web service by
Google. The **DataLayer.Google Spreadsheet** element allows developers to connect to Google Docs and retrieve data from spreadsheets stored there; the data can then be used in Logi applications.
