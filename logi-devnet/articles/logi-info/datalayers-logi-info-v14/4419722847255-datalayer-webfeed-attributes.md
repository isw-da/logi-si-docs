# DataLayer.WebFeed - Attributes

# DataLayer.WebFeed - Attributes

The DataLayer.WebFeed element has the following attributes:

| Attribute | Description |
| --- | --- |
| ID | Specifies an optional element ID. Recommended for easier identification when debugging. |
| Web Feed URL | Specifies the address (URL) of a RSS or Atom web feed, e.g.  https://news.google.com/rss |
| Include Headers | Specifies whether or not feed-level elements (management information) is included with the usual news item information. Default: *False* |
