# Thinkspace - Using Legend Filtering

Once you've created a chart with multiple data series, you might want to temporary hide one or more of them to see the other series in isolation.

![](https://devnet.logianalytics.com/hc/article_attachments/4417562962071/dm30_workcharts_27.png)

Click one or more of the legend labels, as shown above, and that data will be temporarily filtered out of the chart. Click again to restore the data.
