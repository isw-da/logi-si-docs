# Thinkspace - Resizing Columns

# Thinkspace - Resizing Columns

Columns can be resized by dragging them to the desired width:

![](https://devnet.logianalytics.com/hc/article_attachments/4417563079959/dm30_workcols_05.png)

Place your cursor in the space between columns, to the right of the column you want to resize. A gray "edge" line and a special cursor will appear, as shown above. Drag this line to the desired width and release it.
