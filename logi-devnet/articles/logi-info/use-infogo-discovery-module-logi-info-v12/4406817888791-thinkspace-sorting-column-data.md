# Thinkspace - Sorting Column Data

# Thinkspace - Sorting Column Data

You can sort the data by a column in the table:

![](https://devnet.logianalytics.com/hc/article_attachments/4417575537815/dm30_workcols_06.png)

Click one of the two gray arrows in the column pill, as shown above, to sort the column in ascending or descending order. The arrow you clicked will darken, becoming an indicator of the sort order selected. Click it again to toggle the sorting *on* or *off*.
