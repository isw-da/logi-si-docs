# Managing Your Profile

# Managing Your Profile

The details of your profile are stored in the system database. If you click the main menu option with your user name, you'll see the *Profile Information* page:

![](https://devnet.logianalytics.com/hc/article_attachments/4417584003095/ls31_dvauth_04.png)

In the related page, you can manage the data associated with your user account and change your password. Be sure to provide an email address here if you're going to be using DataHub 3.0 to cache data and you want to be notified when data loading completes.

![](https://devnet.logianalytics.com/hc/article_attachments/4417584003863/ls31_dvauth_05.png)

You can also choose a specific avatar, as shown above.
