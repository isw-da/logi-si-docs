# Thinkspace Crosstabs

A Cross Tabulation (often abbreviated as "crosstab") is a data visualization that displays the *joint distribution*
of two or more variables simultaneously. Crosstabs can be charts or they can be tables, the latter sometimes called "pivot tables". Both make it easy to sort, count, and total the data.

The following topics discuss the use of Thinkspace Crosstabs:
