# Configure a Connection

Launch Logi Studio and open, or create, an application for this exercise. Ensure the application version is 12.5 SP1+.

![](https://devnet.logianalytics.com/hc/article_attachments/4417583935767/ls31_getstart_01.png)

Add a **Connection.Logi Application Service** element to your \_Settings definition to connect to the Logi Application Service, as shown above. This element will only be present in Studio *after* Logi Platform Services is installed. Set the attributes as shown above, using the "connLAS", "http://localhost:3000", and "admin" values. Enter your own admin password and remember it for later use.
