# Managing Data Sources

# Managing Data Sources

As you create Sources, they're shown on the *Sources* page as graphic "pills":

![](https://devnet.logianalytics.com/hc/article_attachments/4417575754519/ls31_dvauth_07.png)

The collection of Sources can be searched and filtered using the provided controls.

Each pill displays the Source name and database type, as shown above,
and includes a *gear* icon. Hovering your mouse cursor over the icon
displays a menu of management actions.

![](https://devnet.logianalytics.com/hc/article_attachments/4417584004503/ls31_dvauth_08.png)

The *Info* gear menu option allows you to see, among other details, who
created the Source, as shown above. This can be helpful if the Source
has been shared with you by another user.

The *Delete* gear menu option will only be included if you're the user who created the Source.

![](https://devnet.logianalytics.com/hc/article_attachments/4417575754775/ls31_dvauth_09.png)

Source pills are also visible in the *Create a New Dataview* page, under
the From Source tab, as shown above. The gear icon, however, only
displays an information option from this page.
