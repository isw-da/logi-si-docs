# Thinkspace - Creating a Combo Chart

A Combo chart allows you combine several chart types, overlaying them, for easier comparison of the data.

![](https://devnet.logianalytics.com/hc/article_attachments/4417562964759/dm30_workcharts_42.png)

In order to create a Combo chart in the Thinkspace, scroll down in the Visualization menu and select Combo Charts![](https://devnet.logianalytics.com/hc/article_attachments/4417583616279/menuarrowrt.png)(base chart type of your choice).

![](https://devnet.logianalytics.com/hc/article_attachments/4417575440023/dm30_workcharts_43.png)

Drag-and-drop the data pills necessary to create the first chart series (in the example above *CategoryName* and *UnitsInStock*). Then drag the pill for your second series (*UnitPrice*) into the Secondary axis drop zone, as shown above.

![](https://devnet.logianalytics.com/hc/article_attachments/4417575440279/dm30_workcharts_44.png)

Your chart will be drawn, with the two series stacked, as shown above.

![](https://devnet.logianalytics.com/hc/article_attachments/4417562965015/dm30_workcharts_45.png)

Next, click the Secondary pill's gear icon and select the Visualization Settings in its gear menu, as shown above. You can now choose a different chart type for the secondary data. We'll choose a Line chart for this example.

![](https://devnet.logianalytics.com/hc/article_attachments/4417562965271/dm30_workcharts_46.png)

And the resulting chart is shown above.

![](https://devnet.logianalytics.com/hc/article_attachments/4417583616791/dm30_workcharts_47.png)

You can add more series by dropping more pills in the Secondary drop zone and the chart will add them, as shown above. Their chart types will default to that of the last pill added.
