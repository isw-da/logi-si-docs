# Install the Self-Service Reporting Module - Removing or Repairing the Installation

Should you ever need to, you can repair damaged SSRM files or remove the
SSRM completely by re-running the installer:

![](https://devnet.logianalytics.com/hc/article_attachments/4419714815255/modify_repair_remove.png)

Don't forget to right-click it and use "Run as administrator" to
start it. You can also use Control Panel![](https://devnet.logianalytics.com/hc/article_attachments/4419714617623/menuarrowrt.png)Programs
to do this.

![](https://devnet.logianalytics.com/hc/article_attachments/4419706599319/criticalicon.png)
Removing the SSRM will remove the special elements *and* the
InfoGo application (any InfoGo definition or support files you've
modified, however, will be left in place).
