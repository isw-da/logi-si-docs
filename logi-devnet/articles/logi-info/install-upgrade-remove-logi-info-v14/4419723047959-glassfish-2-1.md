# Glassfish 2.1

# Glassfish 2.1

This topic demonstrates how to configure Glassfish 2.1.

1. Manually add a .war extension to the Logi application folder. Deploy your application manually using the Glassfish Admin Console.
2. Click this link [sun-web.xml](https://clm.logianalytics.com/downloads/info/java/glassfish2.1/sun-web.xml) to open it in your browser. Right-click and select "View page source" (or similar) to see a plain text version of the code. Save this code as *<LogiAppFolder>/*WEB-INF/sun-web.xml.
