# Install the Discovery Module - Windows - Removing the Discovery Module Installation

You can remove the DM completely by selecting it in Windows' Control
Panel applet, Programs and Features, and clicking **Uninstall/Change**:

![](https://devnet.logianalytics.com/hc/article_attachments/4419706894871/installdm31_09.png)

Select "Yes", as shown above, when prompted to uninstall the
product, and "Yes" again if prompted to remove logs and database
files. If JRE 8 was installed as part of the DM installation, it will be
uninstalled, but if you had JRE 8 installed prior to installing the DM, it
will not be uninstalled as part of this process.
