# Including the Font Awesome Library

First, you need to include the Font Awesome (FA) library. You can either download it from the [FA web site](https://fortawesome.github.io/Font-Awesome/) and store it in a sub-folder in your Logi app's \_SupportFiles folder, or link to it directly.

![](https://devnet.logianalytics.com/hc/article_attachments/4419699989911/fa_01_590x246.png)

Examples of both approaches are shown above. For a local copy of the library, use this path to the FA style sheet:

\_SupportFiles/font-awesome-4.5.0/css/font-awesome.min.css

To use the online version instead, use this URL:

https://maxcdn.bootstrapcdn.com/font-awesome/4.5.0/css/font-awesome.min.css

![](https://devnet.logianalytics.com/hc/article_attachments/4419706599575/noteicon.png) You can also use any of the standard Logi built-in Themes with the FA library.
