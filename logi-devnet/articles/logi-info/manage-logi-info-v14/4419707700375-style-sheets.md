# Style Sheets

Cascading Style Sheets (CSS) provide a class-based mechanism for controlling the appearance of Logi application pages and can eliminate the need for inline formatting and some organizational structures.

The following topics assume that you've already created a basic application whose appearance you want to affect by using a style sheet:

* [Cascading Style Sheets for Newbies](https://devnet.logianalytics.com/hc/en-us/articles/4419723030423-Cascading-Style-Sheets-for-Newbies)
* [Including a Style Sheet in Your Application](https://devnet.logianalytics.com/hc/en-us/articles/4419707699479-Including-a-Style-Sheet-in-Your-Application#addStyleSheet)
* [Editing a Style Sheet in the Studio Workspace](https://devnet.logianalytics.com/hc/en-us/articles/4419715381783-Editing-a-Style-Sheet-in-your-Studio-Workspace#Workspace)
* [Editing a Style Sheet in the Style Sheet Class Selector](https://devnet.logianalytics.com/hc/en-us/articles/4419707698199-Editing-a-Style-Sheet-in-the-Style-Sheet-Class-Selector#Selector)
* [Editing a Style Sheet Using an External Style Sheet Editor](https://devnet.logianalytics.com/hc/en-us/articles/4419715380887-Editing-a-Style-Sheet-Using-the-External-Style-Sheet-Editor)
* [Assigning a Style Sheet to Your Definition](https://devnet.logianalytics.com/hc/en-us/articles/4419707696791-Assigning-a-Style-Sheet-to-Your-Definition#Definition)
* [Assigning Classes to Elements](https://devnet.logianalytics.com/hc/en-us/articles/4419707695895-Assigning-Classes-to-Elements#assignClasses)
