# The DsCondition Filter

# The DsCondition Filter

The **DsCondition Filter** element is used with DataLayer.Data Services to filter out data rows. It's analogous to using a *WHERE* clause in a SQL query and adds a filtering operation to a Dataview.

The following topics discuss the DsCondition Filter:

* [Applying the DsCondition Filter](https://devnet.logianalytics.com/hc/en-us/articles/4419722882199-Applying-the-DsCondition-Filter)
* [Using Multiple and Dynamic DsCondition Filters](https://devnet.logianalytics.com/hc/en-us/articles/4419707544855-Using-Multiple-and-Dynamic-DsCondition-Filters#Dynamic)
* [Use with DsCompare Filters](https://devnet.logianalytics.com/hc/en-us/articles/4419707545879-Use-with-DsCompare-Filters#DsCompare)

For more information about Dataviews, see [Dataviews](https://devnet.logianalytics.com/hc/en-us/articles/4419707255831-Dataviews).
