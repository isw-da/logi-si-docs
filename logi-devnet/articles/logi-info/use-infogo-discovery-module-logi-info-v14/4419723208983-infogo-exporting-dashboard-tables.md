# InfoGo - Exporting Dashboard Tables

If your Dashboard panel contains a table or Crosstab Table, you can export its data to an Excel or CSV file.

To export the data, click the Dashboard panel's **gear** icon to display a drop-down menu with an Export option, as shown below. Select the desired **export type**, Excel or CSV, and specify the file location and name when prompted.

![](https://devnet.logianalytics.com/hc/article_attachments/7522752106903/usinginfogo125_90a.png)

The exported data will be affected by any global or panel filters that have been applied.
