# InfoGo - Grouping Rows

The **Group** feature in the Table configuration area, which can be
displayed by clicking the gear icon or by clicking any table column
header, lets you group table rows:

![](https://devnet.logianalytics.com/hc/article_attachments/4419714918807/usinginfogo125_25.png)

Here's how to use this feature:

1. In the configuration area, select the **Grouping Column** for the first level of
   grouping from the list of columns. Depending in the column's data type,
   additional input controls may be displayed. Click **Add** to group the data and refresh the table. Repeat as desired to create sub-groups.
2. As groups and sub-groups are created, they're added to the Groups list. Use the adjacent **Remove** and **Replace** buttons to manage the list.
3. The **Exclude Detail Rows** check box can be used
   to hide the rows that have been grouped, "collapsing" each group into a
   single row in the table.
4. Grouping and un-grouping can be also be accomplished by clicking
   a column header in the table and then selecting the desired options
   from the context menu.

Click the gear icon to hide the configuration area.
