# Thinkspace - Accessing Gear Menu Options

Each column pill has a "gear" icon and selecting it causes a menu to appear below the pill:

![](https://devnet.logianalytics.com/hc/article_attachments/4419707073175/dm30_workcols_08.png)

The options available in the menu vary depending on the data type of the column and other factors, as shown above.

Some options can be enabled and disabled, using the sliding ![](https://devnet.logianalytics.com/hc/article_attachments/4419707074199/dm30_workcols_09.png) Enable/Disable switch.

Each of the options is described in detail in the following sections.
