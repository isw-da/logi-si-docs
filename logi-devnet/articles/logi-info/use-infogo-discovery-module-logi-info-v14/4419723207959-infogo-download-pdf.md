# InfoGo - Download PDF

If you choose to include a "PDF Link" component, you should be aware of what the PDF version of the report will look like.

![](https://devnet.logianalytics.com/hc/article_attachments/4419699922711/usinginfogo125_65.png)

When the report is converted into a PDF document, the internal InfoGo report name, the Design and View links, and the PDF Link component itself are removed. The example above shows the differences. This means you'll most likely want to include a Label at the top of the report, as its *title*, if you plan to use the PDF Link component.
