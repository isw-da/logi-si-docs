# Session Timeout Element

The **Session Timeout** element allows you to ensure that a user's session will not end prematurely, either automatically or by prompting the user, and is discussed in detail in [Managing Session Timeout](https://devnet.logianalytics.com/hc/en-us/articles/4406817840407-Managing-Session-Timeout-).
