# Security Element

# Security Element

The **Security** element configures the application's security features and enables Logi Security.
Flexible authentication and authorization elements provide access control at
many levels: application, report, process, element and data
row. This is discussed in detail in several documents, beginning with [Managing Session Timeout](https://devnet.logianalytics.com/hc/en-us/articles/4406817840407-Managing-Session-Timeout-).
