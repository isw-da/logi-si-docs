# Global Style Element

# Global Style Element

The **Global Style** element allows you to specify a Theme (see [Themes](https://devnet.logianalytics.com/hc/en-us/articles/4406818168855-Themes)) and/or a separate style sheet (see [Style Sheets](https://devnet.logianalytics.com/hc/en-us/articles/4406817644311-Style-Sheets)) that will apply to the entire application. This provides a single point for appearance management across the application. Values entered here can be overridden in individual reports, if desired, by using the
**Style** element in their definitions.
