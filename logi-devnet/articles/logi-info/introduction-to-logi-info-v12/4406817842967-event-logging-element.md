# Event Logging Element

# Event Logging Element

The **Event Logging** element enables logging of various events such as session starts, authentication, and
database queries. Logging works in conjunction with a Process
definition file that usually handles inserts into a logging database
table.

  

![](https://devnet.logianalytics.com/hc/article_attachments/4417563094423/settingsdef_04.png)

The Event Logging element is a parent for one or more Log Event elements, each of which logs a specific event. More information about event logging is available in [Event Logging](https://devnet.logianalytics.com/hc/en-us/articles/4406817415447-Event-Logging).
