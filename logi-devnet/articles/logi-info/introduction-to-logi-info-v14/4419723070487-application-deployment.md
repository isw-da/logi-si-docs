# Application Deployment

All of the files that typically make up a Logi application reside under one folder, the *application root folder*. This folder is registered with the web server as a "virtual directory" or web application, and is almost always stored on the web server. However, it is possible to distribute the application in clustered server environments.

Publishing Logi applications, such as deploying them from a development web server to a production web server, is a simple matter of copying the application folder to the production server. Studio includes a convenient Deployment Tool that deploys applications using a variety of techniques. Updating definitions in an published application is also just a simple matter of copying them. For more information, see [Deploy a Logi Application](https://devnet.logianalytics.com/hc/en-us/articles/4419707523735-Deploy-a-Logi-Application).
