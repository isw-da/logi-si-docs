# Duplicating Bookmarks

Bookmarks can be duplicated to store back ups and enable you to edit bookmarks that were shared with you, without making changes to the original.

Access the desired folder and select the **duplicate** icon next to the bookmark you want to copy:

![](https://devnet.logianalytics.com/hc/article_attachments/7522823188503/select_duplicate.jpg)

Duplicated bookmarks display in the same folder as the original, with the prefix "Copy of" to differentiate the two.

## New for 14.1 Batch Duplicating

If your application has been configured for it, you can also batch duplicate SSRM bookmarks to their original folder. To enable this feature, set goAllowBatchBookmarkDuplication to "True". By default, this constant is "False" and your screen will look like the following:

![](https://devnet.logianalytics.com/hc/article_attachments/7522776948759/image-20210617-161339_1108x103.png)

![](https://devnet.logianalytics.com/hc/article_attachments/7522755629847/noteicon.jpg) You are still able to duplicate *individual* bookmarks and batch *delete* bookmarks when goAllowBatchBookmarkDuplication is set to "False".

Once this feature has been enabled, choose which bookmarks to copy by selecting the corresponding **check box**, or select all bookmarks in a folder by checking the check box at the top of the list:

![](https://devnet.logianalytics.com/hc/article_attachments/7522776981655/select_individual_bookmarks_1108x339.jpg)

Then, select the **duplicate** icon:

![](https://devnet.logianalytics.com/hc/article_attachments/7522790101655/duplicate_selected_bmarks.jpg)

Duplicated bookmarks display in the same folder, as shown below:

![](https://devnet.logianalytics.com/hc/article_attachments/7522798051863/copy_of_duplicate_dash.jpg)

![](https://devnet.logianalytics.com/hc/article_attachments/7522725179159/noteicon.png) Batch duplicating shared bookmarks stores them in the "My Items" folder.
