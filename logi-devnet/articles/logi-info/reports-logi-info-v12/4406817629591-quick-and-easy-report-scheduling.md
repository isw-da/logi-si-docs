# Quick and Easy Report Scheduling

Developers or administrators who just want to run Logi applications on a
scheduled basis can do so by:

1. Installing the Logi Scheduler
2. Downloading the
   [Scheduler Console Sample Application](https://devnet.logianalytics.com/hc/en-us/articles/360050242853-Scheduler-Console-Sample-Application)
3. Registering the sample application with your web server via Logi Studio
4. And running it to create and manage Scheduler tasks.
