## About Logi Mobile Reports

If you haven't done so, we encourage you to read
[Logi Mobile Reports](https://devnet.logianalytics.com/hc/en-us/articles/4406822416407-Logi-Mobile-Reports)
before proceeding with this topic.

Logi Mobile Reports allow developers to create reports that are to be
viewed on a mobile device. This is achieved by generating web pages that
have been optimized for the mobile device environment and that run on the
"mobile web". Developers design and code Mobile Report
definitions in Logi Studio, using many of the same techniques used to
develop standard Logi reports.

The menu in the following examples is built based on a data table. In
a separate topic,
[Data Menu](https://devnet.logianalytics.com/hc/en-us/articles/4406822230423-Data-Menu), you can learn how to use the Data Menu element to create more
complicated mobile device-specific menus.
