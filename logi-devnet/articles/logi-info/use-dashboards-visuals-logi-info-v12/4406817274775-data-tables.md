# Data Tables

This topic introduces the Logi Data Table, which is one of the primarystructures for displaying report data. It consists of a grid of rows and columns, which displays data (usually) in a row and column arrangement. These "tabular" displays in a report look good and are easy to work with.

The following topics provide more details about using data tables in Logi Studio:
