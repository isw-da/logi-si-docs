# Working with Dashboard Tabs

A dashboard can consist of a single collection of panels (a base page) or a series of tabbed pages, each with its own collection of panels. Dashboards can be configured by your developer to be *static* or *adjustable*. If a dashboard has been configured to be adjustable*and*to have tabs, then you can add, remove, and rearrange the tabs.

![](https://devnet.logianalytics.com/hc/article_attachments/4417575794071/usingdash_03.png)

A dashboard with tabs will have one tab created by default. In the example shown above, an additional dashboard tab has been added. Tabs can be dragged with the mouse to rearrange them, as shown above.

![](https://devnet.logianalytics.com/hc/article_attachments/4417575794327/usingdash_04.png)

Adjustable dashboards display this "gear" icon![](https://devnet.logianalytics.com/hc/article_attachments/4417583795991/gear_icon121.png) on the selected tab; click the icon to configure their settings. To add a new tab to the dashboard, click the Plus Sign icon, as indicated above.

![](https://devnet.logianalytics.com/hc/article_attachments/4417575794583/usingdash_05.png)

A tab's Settings pop-up menu, shown above, appears when its gear icon is clicked and allows you to add panels to the tab, change the number of columns being used in the tab, to rename the tab, and to remove the tab.
