# Google Maps - Getting a Google Maps API Key

In order to use the Google Map elements in your application, you'll need a Google API key. Logi Info uses Google's *Maps JavaScript API* version 3.

![](https://devnet.logianalytics.com/hc/article_attachments/7522725232151/criticalicon.png) In July 2018, Google changed its licensing scheme, ending the free API access previously offered at some usage levels. This specifically affected *geocoding* in Logi applications.

Logi Info now includes the **Connection.Nominatim** element, for connections to a free, public, OpenStreetMap (OSM) [Nominatim](https://wiki.openstreetmap.org/wiki/Nominatim) server for geographic data. Using a simple REST API, geocoding results are returned as XML or JSON data. For more information, see [Datasource Connections](https://devnet.logianalytics.com/hc/en-us/articles/4419707715735-Datasource-Connections).

Information about how to get a Google Maps JavaScript API key is available in [Google Connections](https://devnet.logianalytics.com/hc/en-us/articles/4419707607703-Google-Connections). Please read that topic and take the necessary steps before proceeding to implement Google Map in your Logi application.
