# Using the Data List with a Theme

When you assign a Theme to a Logi application, special standard classes for use with the Data List and List Item elements are made available. For more information, see [Themes](https://devnet.logianalytics.com/hc/en-us/articles/4419723357847-Themes).

![](https://devnet.logianalytics.com/hc/article_attachments/4419707197847/datalist_03.png)![](https://devnet.logianalytics.com/hc/article_attachments/4419714994455/datalist_03a.png)

As shown above, you can assign these classes to either the Data List or List Item elements' Class attribute (either one, not both).
When the Data List is created as an *unordered* list, the **ThemeUL**... classes, shown above, can be used to set the shape of the bullet.
When the Data List is created as an *ordered* list, the **ThemeOL**... classes can be used to set the style of the numbering.
