# Logi Product Upgrades

# Logi Product Upgrades

Each new release of Logi Analytics products provides feature enhancements and improvements. Customers are encouraged to take advantage of these improvements by *upgrading* to the latest releases when they become available.

The following topics provide guidance for the process of upgrading, modifying, and uninstalling your Logi product installations, and managing the related versions of your Logi applications:

* [General Requirements](https://devnet.logianalytics.com/hc/en-us/articles/4406817919767-General-Requirements)
* [Upgrade Impacts](https://devnet.logianalytics.com/hc/en-us/articles/4406817920791-Upgrade-Impacts#Impacts)
* [Upgrading the Products](https://devnet.logianalytics.com/hc/en-us/articles/4406817921815-Upgrading-the-Products#Upgrading)
* [Changing .NET Application Versions](https://devnet.logianalytics.com/hc/en-us/articles/4406822545815-Changing-NET-Application-Versions#NETStudio)
* [Changing Java Application Versions](https://devnet.logianalytics.com/hc/en-us/articles/4406817917591-Changing-Java-Application-Versions#Java)
* [Creating Application Pools in IIS 7+](https://devnet.logianalytics.com/hc/en-us/articles/4406817918743-Creating-Application-Pools-in-IIS-7-#IIS7)
* [Creating Application Pools in IIS 6](https://devnet.logianalytics.com/hc/en-us/articles/4406822546711-Creating-Application-Pools-in-IIS-6#IIS6)

![](https://devnet.logianalytics.com/hc/article_attachments/4417583587863/noteicon.png) Information about which Logi Info and add-on module
versions we recommend using together is available in [Release Pairings](https://devnet.logianalytics.com/hc/en-us/articles/4417584258327-Release-Pairings). For more information about Logi Info releases, see [Release Notes](https://clm.logianalytics.com/rdPage.aspx?rdReport=RelNotesINF).

![](https://devnet.logianalytics.com/hc/article_attachments/4417562936855/criticalicon.png) The installation tool, InstallShield, *does not remove* any files or folders created or modified after the initial installation, including Sample Applications provided with Logi products that you may have modified.
