# Modifying or Repairing an Installation

Suppose you suspect a .DLL file is missing or is corrupted and you want to fix it, or you need to otherwise change your Studio installation.

![](https://devnet.logianalytics.com/hc/article_attachments/4417575648535/installinfo12w7_19.png)

These kinds of situations can be addressed by either modifying or repairing the installation, which you should do by re-running the installation .exe file. *Do not* use Control Panel ![](https://devnet.logianalytics.com/hc/article_attachments/4417575446295/menuarrow.gif)Add-Remove Programs
to do this; it
will request an .msi file, which is not retained after the original installation.
