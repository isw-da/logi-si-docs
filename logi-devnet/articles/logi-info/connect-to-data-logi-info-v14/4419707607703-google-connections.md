# Google Connections

Logi Info's **Connection.Google Docs** and **Connection.Google Maps** elements require specialized connection configurations.

Using Geocode Columns elements? Logi Info now includes the **Connection.Nominatim** element, for connections to a free, public, OpenStreetMap (OSM) [Nominatim](https://wiki.openstreetmap.org/wiki/Nominatim) server for the geographic data used in geocoding. This provides a free alternative to engaging with Google for the data. More information is available in [Datasource Connections](https://devnet.logianalytics.com/hc/en-us/articles/4419707715735-Datasource-Connections).
