# Displaying Literal Text and Data

In the same way that tokens can be combined, as discussed in [Displaying Tokens Adjacent to One Another](https://devnet.logianalytics.com/hc/en-us/articles/4419715166359-Displaying-Tokens-Adjacent-to-One-Another), literal text and tokens can be shown together:

![](https://devnet.logianalytics.com/hc/article_attachments/4419706695959/combinetext_05.png)

As shown above, the literal text "Lat Name:", and a separating space, is entered, followed by a Data token. When the page is rendered, the token will be replaced by the data value and will be preceded by the literal text in each table row.
