# Exporting Visuals

# Exporting Visuals

As mentioned in [Report Components](https://devnet.logianalytics.com/hc/en-us/articles/4419707800983-Report-Components), you can drag the PDF and Excel links into your Author Report to allow your users to export the report for further analysis.

## Export to PDF

1. In Design mode, begin by dragging the New PDF Link to the report:

![](https://devnet.logianalytics.com/hc/article_attachments/7522768390679/drag_new_pdf_link_1016x360.png)

Info displays a Settings dialog, shown below:

![](https://devnet.logianalytics.com/hc/article_attachments/7522789818391/image-20220601-190722_516x504.png)

Here, you have the ability to customize the font size, font color, page size, and orientation of the report, as well as additional border settings. ![](https://devnet.logianalytics.com/hc/article_attachments/7522745474327/_newplus.jpg) New for 14.2 Un-check the "Include in Schedule" check box to remove an object from the final scheduled output.

1. Make any necessary changes, then, select **Set**.
2. Next, select **Download as PDF**:

![](https://devnet.logianalytics.com/hc/article_attachments/7522768412695/select_download_as_pdf.png)

The report downloads automatically in the browser. Your Author Report is now available for further analysis in PDF format.

## Export to Excel

1. In Design mode, begin by dragging the New Excel Link to the report:

![](https://devnet.logianalytics.com/hc/article_attachments/7522804493847/new_excel_link_669x526.png)

Info displays a Settings dialog, shown below:

![](https://devnet.logianalytics.com/hc/article_attachments/7522804521111/image-20220601-191002_540x441.png)

Here, you have the ability to customize the font size, font color, page size, and orientation of the report, as well as additional border settings. ![](https://devnet.logianalytics.com/hc/article_attachments/7522745474327/_newplus.jpg) New for 14.2 Un-check the "Include in Schedule" check box to remove an object from the final scheduled output.

1. Make any necessary changes, then, select **Set**.
2. Next, select **Download as Excel**:

![](https://devnet.logianalytics.com/hc/article_attachments/7522758358551/new_excel_link1.png)

The report downloads automatically, or will open within the browser, depending on your Excel application settings. Your Author Report is now available for analysis in Excel.

![](https://devnet.logianalytics.com/hc/article_attachments/7522725179159/noteicon.png) Excel results will be HTML-based output.
