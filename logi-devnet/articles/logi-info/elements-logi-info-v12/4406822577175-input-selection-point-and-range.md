# Input Selection Point and Range

The **Input Selection Point** and **Input Selection Range** families
of elements are available as children of some Chart Canvas chart Series
elements. They provide the ability to interactively select portions of
charts using the mouse and to initiate actions using the selected data.
These input elements are discussed in
[Input Selection](https://devnet.logianalytics.com/hc/en-us/articles/4406822104983-Input-Selection).

More advanced information relating to many of input elements can be found
in
[Working with User Input Elements](https://devnet.logianalytics.com/hc/en-us/articles/4406818159767-Working-with-User-Input-Elements).
