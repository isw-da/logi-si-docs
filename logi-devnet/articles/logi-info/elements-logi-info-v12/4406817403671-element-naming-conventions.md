# Element Naming Conventions

Almost all elements have an ID (element identifier) attribute. This is an important attirubte and should always be set to a unique value; if left blank, Logi Info will automatically provide an ID for some elements using a **standard****naming convention** when assigning IDs is good practice, as it results in a more easily understood definition.

The following topics discuss the naming conventions recommended by Logi Analytics:

* [Element ID Notation](https://devnet.logianalytics.com/hc/en-us/articles/4406817402391-Element-ID-Notation)
* [Formulating Mnemonics](https://devnet.logianalytics.com/hc/en-us/articles/4406822308887-Formulating-Mnemonics#Formulating)
