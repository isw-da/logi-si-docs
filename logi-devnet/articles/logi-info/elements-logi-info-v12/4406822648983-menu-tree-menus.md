# Menu Tree Menus

The **Menu Tree** element, and its child elements, are used to define a menu, in a horizontal or vertical orientation, with optional sub-menu branches and menu items.

The following topics discuss the Menu Tree element:

* [Menu Tree Attributes](https://devnet.logianalytics.com/hc/en-us/articles/4406818131223-Menu-Tree-Attributes#Tree)
* [Menu Branch and Leaf Elements](https://devnet.logianalytics.com/hc/en-us/articles/4406822648087-Menu-Branch-and-Leaf-Elements#Branch)
* [Creating a Menu](https://devnet.logianalytics.com/hc/en-us/articles/4406818129047-Creating-a-Menu-Tree-Menu#Creating)
* [Customizing Menu Appearance](https://devnet.logianalytics.com/hc/en-us/articles/4406818130071-Customizing-Menu-Appearance#Customizing)

## About the Menu Tree

Menus are typically used to launch actions such as running another report or linking to another web page. The **Menu Tree** element and its child elements, introduced in v12.1, allow you to easily to create a multi-level menu.

![](https://devnet.logianalytics.com/hc/article_attachments/4417583601559/menutree_01.png)

The Menu Tree can be configured for three different modes; the default mode is *HorizontalPopup*, shown above. The example uses two menu *branches* (MainOpt 1, MainOpt 2), each of which has three menu *leaves* (SubOpt 1, SubOpt 2, SubOpt 3). You can also nest branches and leaves to get additional suboptions, as shown at the far right.

![](https://devnet.logianalytics.com/hc/article_attachments/4417583601815/menutree_02.png)

The *Vertical* mode, shown above, lists the branches and leaves in a vertical hierarchy that can be expanded and collapsed.

![](https://devnet.logianalytics.com/hc/article_attachments/4417583601943/menutree_03.png)

And the *VerticalPopup* mode uses a vertical list of branches and pop-up panels to display the leaves.
The arrangement of menus on the page is dependent on their containers (Divisions, Rows, etc.). The styling of the examples shown is controlled by a theme. You can supply additional styling to change their appearance.

### Other Menu Variants

Logi Info also includes other menu elements you may wish to investigate; they may be more useful depending on the nature of your application. Pop-up menus (see [Popup Menus](https://devnet.logianalytics.com/hc/en-us/articles/4406822654487-Popup-Menus)) display a panel with menu options and the [ReportCenter Menu](https://devnet.logianalytics.com/hc/en-us/articles/4406822493207-ReportCenter-Menu) is useful for displaying a menu based on the available reports.
