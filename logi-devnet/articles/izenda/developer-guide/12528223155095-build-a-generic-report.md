# Build a Generic Report

# Build a Generic Report

This article describes how to build a generic report with the Izenda BI.

1. Save as draft the report with name, category and selected data sources
   1. Prepare an empty [ReportDefinition](https://devnet.logianalytics.com/hc/en-us/articles/12528297929111-ReportDefinition) object.
   2. Populate report name and categories.
   3. Populate selected data sources.
   4. Prepare a [ReportSavingParameter](https://devnet.logianalytics.com/hc/en-us/articles/12528282939799-ReportSavingParameter) object as API input.
   5. Call [Save report as draft](https://devnet.logianalytics.com/hc/en-us/articles/12528279020567-Report-Designer-Details-APIs#_bm) API, then receive back the [ReportSavingResult](https://devnet.logianalytics.com/hc/en-us/articles/12528282949143-ReportSavingResult) object.
2. Populate the draft with report parts
   1. Extract the `report` field from the [ReportSavingResult](https://devnet.logianalytics.com/hc/en-us/articles/12528282949143-ReportSavingResult) object. This is a [ReportDefinition](https://devnet.logianalytics.com/hc/en-us/articles/12528297929111-ReportDefinition) object which has been assigned a report id.
   2. Populate the report parts:
      * [Build a simple Grid](https://devnet.logianalytics.com/hc/en-us/articles/12528207410839-Build-a-Simple-Grid)
      * [Build a simple Chart](https://devnet.logianalytics.com/hc/en-us/articles/12528223176343-Build-a-Simple-Chart)
      * [Build a simple Gauge](https://devnet.logianalytics.com/hc/en-us/articles/12528207404311-Build-a-Simple-Gauge)
      * [Build a simple Map](https://devnet.logianalytics.com/hc/en-us/articles/12528207474839-Build-a-Simple-Map)
      * [Build a simple Form](https://devnet.logianalytics.com/hc/en-us/articles/12528207381271-Build-a-Simple-Form)
3. Save the report
   1. Prepare a [ReportSavingParameter](https://devnet.logianalytics.com/hc/en-us/articles/12528282939799-ReportSavingParameter) object as API input.
   2. Call [Save report](https://devnet.logianalytics.com/hc/en-us/articles/12528279020567-Report-Designer-Details-APIs#_bm2) API.
