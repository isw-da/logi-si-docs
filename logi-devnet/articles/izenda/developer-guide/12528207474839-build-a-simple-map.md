# Build a Simple Map

# Build a Simple Map

Steps:

1. Follow [Build a Generic Report](https://devnet.logianalytics.com/hc/en-us/articles/12528223155095-Build-a-Generic-Report) until the step to add report parts.
2. Prepare an empty [ReportPartMap](https://devnet.logianalytics.com/hc/en-us/articles/12528298227095-ReportPartMap) object with default properties.
3. For each selected data source fields, populate a [ReportPartElement](https://devnet.logianalytics.com/hc/en-us/articles/12528298149527-ReportPartElement) object with default properties.
4. Add the [ReportPartElement](https://devnet.logianalytics.com/hc/en-us/articles/12528298149527-ReportPartElement) objects into either `pointOptions.elements`, `shadingMetric.elements` or `bubbleMetrics.elements` in [ReportPartMap](https://devnet.logianalytics.com/hc/en-us/articles/12528298227095-ReportPartMap) object.
5. Update the properties of the map in `properties` field per user selection (See [ReportPartMapProperties](https://devnet.logianalytics.com/hc/en-us/articles/12528298234903-ReportPartMapProperties)).
6. Back to the steps in [Build a Generic Report](https://devnet.logianalytics.com/hc/en-us/articles/12528223155095-Build-a-Generic-Report).

## Populate selected data sources fields

1. Refer to the [similar step in Building a Grid guide](https://devnet.logianalytics.com/hc/en-us/articles/12528207410839-Build-a-Simple-Grid#Populate_selected_data_sources_fields) to:

   1. Get the list of available data sources fields from [POST report/availableQuerySourceFields](https://devnet.logianalytics.com/hc/en-us/articles/12528278171415-Report-Designer-Data-Sources-APIs#POST_report/availableQuerySourceFields)
   2. Build a corresponding ReportPartElement object for each selected data source field
   3. Populate a default ReportPartElementProperties for `properties` field in each ReportPartElement object

   See [Sample Properties for a ReportPartElement](https://devnet.logianalytics.com/hc/en-us/articles/12528223341335-Sample-Properties-for-a-ReportPartElement) for some samples.
2. Add the [ReportPartElement](https://devnet.logianalytics.com/hc/en-us/articles/12528298149527-ReportPartElement) objects into `pointOptions.elements`, `shadingMetric.elements` and `bubbleMetrics.elements` in [ReportPartMap](https://devnet.logianalytics.com/hc/en-us/articles/12528298227095-ReportPartMap) object.
3. Update `properties.reportPartState.pointOptionsList` and `properties.reportPartState.activePointOption` with the selected data source fields in Point Options boxes.

Sample full ReportPartMap object

* Highlighted in `pointOptions.elements` are the ReportPartElement objects for `GROUP(ShipCountry)` and `GROUP(ShipCity)`
* Highlighted in `bubbleMetrics.elements` is the ReportPartElement for `SUM(Freight)`
* Highlighted `properties.reportPartState.pointOptionsList` and `properties.reportPartState.activePointOption` have been updated with the select data source fields in Point Options boxes
* Highlighted in `properties.chartType` is the chart type (World)
