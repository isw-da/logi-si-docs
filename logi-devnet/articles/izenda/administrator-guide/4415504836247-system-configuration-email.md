# System Configuration/Email

# System Configuration/Email

The **System Configuration/Email** page allows user to set up email
sending options.

1. In browser, log in to Izenda as a user with System Configuration
   permission.
2. Click Settings, then System Configuration then Email in the left
   menu.
3. Select the Setting Level: either System or a specific tenant.

   > [![../_images/System_Configuration_Email_Tenant_Configuration.png](https://devnet.logianalytics.com/hc/article_attachments/4415511728791/system_configuration_email_tenant_configuration_323x60.png)](https://devnet.logianalytics.com/hc/article_attachments/4415504362519/system_configuration_email_tenant_configuration.png)
   >
   > For tenant level, select to re-use configuration from system or to define a separate one.
4. Enter the email server details into SMTP Configuration section.
5. Enter the display name and the from address into Email Info section.

   [![../_images/System_Configuration_Email.png](https://devnet.logianalytics.com/hc/article_attachments/4415504362903/system_configuration_email_652x298.png)](https://devnet.logianalytics.com/hc/article_attachments/4415492459927/system_configuration_email.png)

   e.g. ACME and [no-reply@acme.com](mailto:no-reply%40acme.com) will result in this email.

   [![../_images/Email_from_no-reply@acme.png](https://devnet.logianalytics.com/hc/article_attachments/4415492460183/email_from_no-reply_acme_304x104.png)](https://devnet.logianalytics.com/hc/article_attachments/4415504363159/email_from_no-reply_acme.png)
