# Tenant APIs

# Tenant APIs

The **Tenant Setup** page allows users to

* manage tenants
* manage tenants’ modules
* manage tenants’ users and roles

## List of APIs

| API | Purpose |
| --- | --- |
| [GET tenant/{tenant\_id}](#get-tenant-tenant-id) | Returns the tenant specified by tenant\_id. |
| [GET tenant/activeTenants](#id1) | Returns an array of active tenants. |
| [GET tenant/allTenantGroups](#get-tenant-alltenantgroups)  New in version 3.11.0. | Returns an array of all tenant groups. |
| [GET tenant/allTenants](#get-tenant-alltenants) | Returns an array of all tenants. |
| [GET tenant/availableCategory/{type}](#get-tenant-availablecategory-type) | Deprecated since version 2.0.0: superseded by [POST role/availableCategory](https://devnet.logianalytics.com/hc/en-us/articles/12528280786967-Role-APIs#post-role-availablecategory)  Returns an array of available categories for report/template (type=0) or dashboard (type=1). |
| [GET tenant/basicInfos](#get-tenant-basicinfos) | Returns all active tenants with basic info. |
| [GET tenant/namesOnly](#get-tenant-namesonly) | Returns all active tenants with names only. |
| [GET tenant/tenantGroupsHierarchy](#get-tenant-tenantgroupshierarchy)  New in version 3.11.0. | Returns an array of all tenant groups and tenants. |
| [POST tenant](#id2) | Saves a tenant. |
| [POST tenant/accessLimits](#post-tenant-accesslimits) | Lazy load access limit for each tenant. |
| [POST tenant/active/{tenant\_id}](#post-tenant-active-tenant-id) | Sets active for the tenant specified by tenant\_id. |
| [POST tenant/deactive/{tenant\_id}](#post-tenant-deactive-tenant-id) | Sets inactive for the tenant specified by tenant\_id. |
| [POST tenant/intergration/saveTenant](#post-tenant-intergration-savetenant) | Adds or updates external tenant.  Note  To be renamed to “integration” |
| [POST tenant/scheduleLimits](#post-tenant-schedulelimits) | Lazy load schedule limit for each tenant. |
| [DELETE tenant/{tenant\_id}](#delete-tenant-tenant-id) | Deletes the tenant specified by tenant\_id. |

## GET tenant/basicInfos

Returns all active tenants with basic info.

Returns only the current tenant with basic info if logged in user is a tenant user.

**Request**

> No payload

**Response**

> An array of the following objects:
>
> | Field | Description | Note |
> | --- | --- | --- |
> | **id** string (GUID) | The id of the tenant |  |
> | **tenantId** string | The user-selected id of the tenant |  |
> | **name** string | The name of the tenant |  |
> | **active** boolean | Whether the tenant is active |  |
> | **description** string | The description of the tenant |  |
> | **tenantModules** array of strings | An array of selected module names for the tenant |  |

> ```
> GET/api/tenant/basicInfosHTTP/1.1
> ```
>
> Sample response:
