# TokenRequest

| Field | NULL | Description | Note |
| --- | --- | --- | --- |
| **licenseKey**  string |  | The license key |  |
| **tokenKey**  string | Y | The token, in case [offline mode](https://devnet.logianalytics.com/hc/en-us/articles/12528224699927-Glossary#term-offline-mode) is used |  |

```
{"LicenseKey":"1aBcD+==","TokenKey":"1aBcD+="}
```
