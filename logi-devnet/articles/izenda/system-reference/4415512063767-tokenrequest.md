# TokenRequest

| Field | NULL | Description | Note |
| --- | --- | --- | --- |
| **licenseKey**  string |  | The license key |  |
| **tokenKey**  string | Y | The token, in case [offline mode](https://devnet.logianalytics.com/hc/en-us/articles/4415492696343-Glossary#term-offline-mode) is used |  |

**Sample**:

```
{"LicenseKey":"1aBcD+==","TokenKey":"1aBcD+="}
```
