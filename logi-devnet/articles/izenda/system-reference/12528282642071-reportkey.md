# ReportKey

| Field | NULL | Description | Note |
| --- | --- | --- | --- |
| **key** string (GUID) | Y | The id of the report |  |
| **tenantId** string (GUID) | Y | The id of the tenant |  |
| **modified** datetime |  | The modification date time |  |

```
{"reportKey":{"key":"552d20ce-1269-4cb4-a679-0cd51d3e2058"}}
```
