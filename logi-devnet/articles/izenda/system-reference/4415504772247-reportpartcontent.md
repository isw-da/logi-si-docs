# ReportPartContent

# ReportPartContent

| Field | NULL | Description | Note |
| --- | --- | --- | --- |
| **type**  integer |  | The type of the report part   * 0 = Chart * 1 = Form * 2 = Gauge * 3 = Grid * 4 = Map |  |
| **properties**  object |  | A dynamic object to keep the properties in Report Part Properties box, can be user-defined |  |
| **settings**  object |  | A dynamic object to keep the properties in Report Part Configuration, can be user-defined |  |
| **title**  object |  | A [ContentContainer](https://devnet.logianalytics.com/hc/en-us/articles/4415504701719-ContentContainer) object |  |
| **description**  object |  | A [ContentContainer](https://devnet.logianalytics.com/hc/en-us/articles/4415504701719-ContentContainer) object |  |
| **expandedLevel**  integer |  | The expand level of the report part |  |
| **isCrossFiltering**  boolean |  | Whether this is selected to apply [cross filtering](https://devnet.logianalytics.com/hc/en-us/articles/4415492696343-Glossary#term-cross-filtering) |  |
| **isFormatted**  boolean |  | Whether this report part is formatted |  |
| **selectedMetric**  string |  | The selected metric value on Chart/Gauge/Map |  |
| **vizState**  object  New in version 3.5.0. |  | A [VizState](https://devnet.logianalytics.com/hc/en-us/articles/4415504812311-VizState) object |  |
