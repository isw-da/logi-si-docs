# DatabaseType

# DatabaseType

| Field | NULL | Description | Note |
| --- | --- | --- | --- |
| **name** string |  | Database type name |  |
| **shortName** string |  | The short name |  |
| **id** string (GUID) |  | The unique id of the database type |  |

**Sample**:

```
{"name":"[MSSQL] SQLServer","shortName":"MSSQL","id":"572bd576-8c92-4901-ab2a-b16e38144813"}
```
