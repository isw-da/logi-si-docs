# User APIs

# User APIs

The User APIs are used in the log in, log out, user setup and migration features.

## Summary

| API | Purpose | Usage in Izenda Front-end |
| --- | --- | --- |
| [GET user/all/(tenant\_id)](#get-user-all-tenant-id) | Returns an array of users, filtered by tenant\_id if available.  Deprecated, please use [POST user/all](#post-user-all) to avoid timeouts loading large data. | Settings menu > User Setup |
| [POST user/all](#post-user-all) | Returns an array of users by tenant. | Settings menu > User Setup |
| [POST user/load](#post-user-load) | Returns an array of users for Report Access or Schedule, with paging. | Report Designer > Schedule > Add Schedule > select Recipient(s)  Report Designer > Access > Add Sharing > Share With > User > select |
| [POST user/login](#id1) | Performs login. | Login page |
| [POST user/logout](#post-user-logout) | Performs logout. | User section in page header > Sign Out |
| [POST user](#id2) | Saves a user.  Note  To set up a user with password, this API is only the first step. See [Add Internal User](https://devnet.logianalytics.com/hc/en-us/articles/12528223106839-Add-Internal-User) for the guide. | Settings menu > User Setup > Add User > Save |
| [POST user/userProfile](#post-user-userprofile) | Saves a user profile. | not used |
| [POST user/passwordProfile](#post-user-passwordprofile) | Saves a password profile. | not used |
| [POST user/securityQuesitions](#post-user-securityquesitions) | Saves security questions for a user.  Note  Will be renamed to securityQuestions | User section in page header > My Profile > Security Options > Change Security Options > Save |
| [POST user/password](#post-user-password) | Saves password for a user.  Note  Obsolete, use [POST user/passwordAndSecurityQuestion](#id3) instead | Not used |
| [POST user/active](#post-user-active) | Activates a user. | Settings menu > User Setup > Activate link |
| [POST user/deactive](#post-user-deactive) | Deactivates a user. | Settings menu > User Setup > Deactivate link |
| [DELETE user/{user\_id}](#delete-user-user-id) | Deletes a user. | Settings menu > User Setup > Delete link |
| [POST user/passwordAndSecurityQuestion](#id3) | Saves a user password and security question. | Password Link page > Save |
| [GET user/securityQuestion/{user\_name}/(tenant\_display\_id)](#get-user-securityquestion-user-name-tenant-display-id) | Returns security question for a user and tenant. | User section in page header > My Profile > Security Options |
| [POST user/generatePasswordLink](#id4) | Generates [password link](https://devnet.logianalytics.com/hc/en-us/articles/12528224699927-Glossary#term-password-link). | Settings menu > User Setup > Configure Password Option > Generate Password Link |
| [POST user/validatePasswordLink](#post-user-validatepasswordlink) | Validates [password link](https://devnet.logianalytics.com/hc/en-us/articles/12528224699927-Glossary#term-password-link). | Not used |
| [POST user/validateSecurityQuestion](#post-user-validatesecurityquestion) | Validates security questions. | Not used |
| [POST user/validateUserInfo](#post-user-validateuserinfo) | Validates user information. | Not used |
| [POST user/validateExpirationPasswordLink](#post-user-validateexpirationpasswordlink) | Validates password expiration link. | When user opens password link |
| [POST user/sendPasswordLink](#post-user-sendpasswordlink) | Sends [password link](https://devnet.logianalytics.com/hc/en-us/articles/12528224699927-Glossary#term-password-link) via email to user. | Settings menu > User Setup > Configure Password Option > Generate Password Link > Send password link in email |
| [POST user/integration/saveUser](#post-user-integration-saveuser) | Adds or updates external user.  Changed in version 1.25: Used to be named “intergration” | Not used |
| [POST user/validateUserRoleAssociation](#post-user-validateuserroleassociation) | Validates user and role association after some roles are removed. | Settings menu > User Setup > remove a role > Save |
| [POST user/allowedSharingUsers/(tenant\_id)](#post-user-allowedsharingusers-tenant-id) | Returns a list of users allowed to be selected in report/dashboard access. | Report Designer > Access > Add Sharing |
| [GET user/isLastSystemAdmin](#get-user-islastsystemadmin) | Checks if the number of not deleted system admins equals 1. | Settings menu > User Setup > delete a system admin |
| [POST external/user](#post-external-user) | Add or update external user.  New in version 2.6.16. | To be updated |

## POST user/all

Returns an array of users by tenant.

**Request**

> A [PagedRequest](https://devnet.logianalytics.com/hc/en-us/articles/12528297501463-PagedRequest) object
>
> Note:
>
> The keys for [SearchCriteria](https://devnet.logianalytics.com/hc/en-us/articles/12528283106455-SearchCriteria) that this API support:   
> - FullName   
> - All will be returned when no search criteria is provided

> A [PagedResult](https://devnet.logianalytics.com/hc/en-us/articles/12528282006423-PagedResult) object with **result** field containing an array of [UserDetail](https://devnet.logianalytics.com/hc/en-us/articles/12528298705815-UserDetail) objects

> - UserName   
> - EmailAddress   
> - Role   
> - FullName   
> - All

> A [PagedResult](https://devnet.logianalytics.com/hc/en-us/articles/12528282006423-PagedResult) object with **result** field containing an array of [UserDetail](https://devnet.logianalytics.com/hc/en-us/articles/12528298705815-UserDetail) objects
