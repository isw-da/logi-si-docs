## Installing Language Packs to Izenda

The Izenda 7 Series language pack allows you to easily support additional languages in the application UI and customize some of the default application messages. The resources and instructions to install additional language packs in Izenda can be found on our GitHub [page](https://github.com/Izenda7Series/Languagepacks).
