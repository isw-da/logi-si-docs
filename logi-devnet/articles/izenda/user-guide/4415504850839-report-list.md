# Report List

# Report List

The **Report List** page allows users to

|  |  |
| --- | --- |
| * browse reports by type, categories and sub-categories * view report history * copy and move reports between categories * print and export reports | * set up subscriptions * delete reports * rename and delete report categories |

## Browse Reports

1. In browser, log in to Izenda as a user with Report permission.
2. Click Reports.
3. The reports are listed in their respective categories and
   sub-categories.
4. Select REPORTS or TEMPLATES to view reports of this type.

   [![../_images/Report_List.png](https://devnet.logianalytics.com/hc/article_attachments/4415504499735/report_list_600x268.png)](https://devnet.logianalytics.com/hc/article_attachments/4415504499351/report_list.png)
5. Click the plus icon on
   the right to view report properties.

   [![../_images/Report_Properties_in_Report_List.png](https://devnet.logianalytics.com/hc/article_attachments/4415492588695/report_properties_in_report_list_803x246.png)](https://devnet.logianalytics.com/hc/article_attachments/4415492587543/report_properties_in_report_list.png)
6. The buttons at the bottom allow user to Open (in Report Viewer),
   Design, Copy or Move to another name or category, Print or Export to
   another format, add Subscriptions, or Detele the report.

## View Report History

1. Click the history icon (similar to ↺) next to version number to open
   Report History pop-up.
2. The report versions together with modified date and user are
   displayed with the most current one on top.
3. There are shortcuts for the current version to be opened in Report
   Designer, copied and moved, or deleted.

   [![../_images/Report_History_Pop-up.png](https://devnet.logianalytics.com/hc/article_attachments/4415492589847/report_history_pop-up_679x408.png)](https://devnet.logianalytics.com/hc/article_attachments/4415492588951/report_history_pop-up.png)
4. The previous versions can
   be removed if no longer needed, or copied to another name or category
   then used in addition to the current version.

   [![../_images/Report_History_Copy_Version.png](https://devnet.logianalytics.com/hc/article_attachments/4415504503063/report_history_copy_version_679x410.png)](https://devnet.logianalytics.com/hc/article_attachments/4415504502679/report_history_copy_version.png)

## Search for Reports

The Search box at the top allows user to search for specific reports.

1. Select a specific element to search for in the drop-down on the left of the Search box. Default is All.
2. Type a partial name and click the search icon (🔍).
3. The matching reports will be displayed.

   ![../_images/Report_List_-_Search.png](https://devnet.logianalytics.com/hc/article_attachments/4415504503319/report_list_-_search_296x228.png)

## Rename Report Categories

1. Click the pencil icon to rename a report category.

   ![../_images/Report_List_-_Category_Rename_Icon.png](https://devnet.logianalytics.com/hc/article_attachments/4415492485143/report_list_-_category_rename_icon_200x100.png)
2. Enter the name in place and click the check icon to save.

   ![../_images/Report_List_-_Category_Rename_Confirmation.png](https://devnet.logianalytics.com/hc/article_attachments/4415492485399/report_list_-_category_rename_confirmation_213x107.png)
3. The new name will be checked for duplication with existing categories.

   ![../_images/Report_List_-_Category_Rename_Duplication_Error.png](https://devnet.logianalytics.com/hc/article_attachments/4415504391063/report_list_-_category_rename_duplication_error_196x125.png)

## Copy a Report

The Copy function allows user to copy a report to a new name and/or a new category and sub-category. leaving the original report intact.

1. Click the Copy button to open Copy Report pop-up.
2. Enter a new name for the report if changing the name.
3. Enter or select a new category and sub-category if needed.
4. The report name in will be checked for duplication in the destination
   category.
5. Click OK to close the pop-up and perform the copy.

![../_images/Report_List_Copy_Duplication.png](https://devnet.logianalytics.com/hc/article_attachments/4415504504087/report_list_copy_duplication_412x224.png)

## Move a Report

The Move function allows user to move a
report to a new category and sub-category, optionally changing the name
in the process.

1. Click the Move button to open Move Report pop-up.
2. Enter a new name for the report if changing the name.
3. Enter or select a new category and sub-category.
4. The report name in will be checked for duplication in the destination
   category.
5. Click OK to close the pop-up and perform the move.

![../_images/Report_List_Move_Duplication.png](https://devnet.logianalytics.com/hc/article_attachments/4415511837975/report_list_move_duplication_411x229.png)

## Add Report Subscriptions for Current User

1. Click the Subscribe button to open Report Subscription pop-up.
2. The report’s current subscriptions are displayed with a summary of
   the schedule.

   ![../_images/Report_Add_Subscription_Pop-up.png](https://devnet.logianalytics.com/hc/article_attachments/4415511838231/report_add_subscription_pop-up_600x414.png)
3. Click Add Subscription button to open Add Subscription pop-up.
4. Enter a name for this subscription.
5. In Schedule tab:

   1. Select Subscription Type:
      * Subscription Alert: deliver to current user only if it has data
        at scheduled time.
      * Subscription Report: deliver to current user regardless whether
        it has data or not at scheduled time.
   2. Select a more suitable time zone than the default if needed.
   3. Select a start date in the future.
   4. Enter the start time (some common values can be quickly selected
      from the pre-defined list).
   5. Select a common recurrence period from the list, or select Custom Recurrence to define a different one.

   [![../_images/Report_Subscription_-_Recurrence_Second_Friday_of_June.png](https://devnet.logianalytics.com/hc/article_attachments/4415504504471/report_subscription_-_recurrence_second_friday_of_june_635x237.png)](https://devnet.logianalytics.com/hc/article_attachments/4415511838487/report_subscription_-_recurrence_second_friday_of_june.png)
6. In Delivery tab, select:

   * “Email” Delivery Type: send the subscription to the email address
     of current user.
     1. Select Delivery Method: email a **Link**, email an
        **Attachment**, or email with **Embedded HTML** body.
     2. Customize the default template if necessary.
   * “File Location” Delivery Type: save the subscription as a file.
     1. Delivery Method is Send to disk.
     2. Select the Export File Type: PDF, Word Doc, Excel or CSV.

   [![../_images/Report_Subscription_-_Delivery_Type_and_Template.png](https://devnet.logianalytics.com/hc/article_attachments/4415504504727/report_subscription_-_delivery_type_and_template_600x415.png)](https://devnet.logianalytics.com/hc/article_attachments/4415492591383/report_subscription_-_delivery_type_and_template.png)
7. Click OK to close Add Subscription pop-up.
8. Click Save to close Report Subscription pop-up.

## Remove Report Subscriptions for Current User

1. Click the Subscribe button to open Report Subscription pop-up.
2. The report’s current subscriptions are displayed with a summary of
   the schedule.
3. Click the Delete icon (x) to delete each subscription.
4. Click Save to close Report Subscription pop-up.

To delete multiple subscriptions:

1. Tick the checkboxes beside each one.
2. Click the now enabled Delete icon at the bottom (that looks like a
   waste basket)
3. Click Save to close Report Subscription pop-up.

[![../_images/Report_Subscription_-_Delete_Multiple.png](https://devnet.logianalytics.com/hc/article_attachments/4415492486039/report_subscription_-_delete_multiple_600x409.png)](https://devnet.logianalytics.com/hc/article_attachments/4415492485783/report_subscription_-_delete_multiple.png)
