# Dashboard List

# Dashboard List

The **Dashboard List and Viewer** page allows users to:

|  |  |
| --- | --- |
| * browse and search for dashboards * copy and move dashboards between categories * switch to presentation mode * configure sharing access * set up subscriptions and scheduled delivery | * rename and delete dashboards and dashboard categories * print dashboard * email dashboard * design dashboard * configure child report parts:   + view, quick edit and design report parts in new window   + print and export report parts   + re-select report parts |

## Browse Dashboards

1. In browser, log in to Izenda as a user with Dashboard permission.
2. Click Dashboards.
3. The dashboards are listed in their respective categories and sub-categories.

   [![../_images/Dashboard_List.png](https://devnet.logianalytics.com/hc/article_attachments/12528166463895/dashboard_list_600x194.png)](https://devnet.logianalytics.com/hc/article_attachments/12528150365463/dashboard_list.png)
4. Click the plus icon on the right to view dashboard properties .

   [![../_images/Dashboard_Properties_in_Dashboard_List.png](https://devnet.logianalytics.com/hc/article_attachments/12528150377111/dashboard_properties_in_dashboard_list_803x157.png)](https://devnet.logianalytics.com/hc/article_attachments/12528150371223/dashboard_properties_in_dashboard_list.png)
5. The buttons at the bottom allow user to Open (in Dashboard Designer), Copy or Move to another name or category, Print, Email, add Subscriptions, or Delete the dashboard.

## Search for Dashboards

The Search box at the top allows user to search for specific dashboards.

1. Select a specific element to search for in the dropdown on the left of the Search box. Default is All.
2. Type a partial name and click the search icon (🔍).
3. The matching dashboards will be displayed.

> [![../_images/Dashboard_List_-_Search.png](https://devnet.logianalytics.com/hc/article_attachments/12528166495511/dashboard_list_-_search_600x302.png)](https://devnet.logianalytics.com/hc/article_attachments/12528150393495/dashboard_list_-_search.png)

## Rename Dashboard Categories

1. Click the pencil icon to rename a dashboard category.

   ![../_images/Report_List_-_Category_Rename_Icon.png](https://devnet.logianalytics.com/hc/article_attachments/12528150420247/report_list_-_category_rename_icon_200x100.png)
2. Enter the name in place and click the check icon to save.

   ![../_images/Report_List_-_Category_Rename_Confirmation.png](https://devnet.logianalytics.com/hc/article_attachments/12528166509975/report_list_-_category_rename_confirmation_213x107.png)
3. The new name will be checked for duplication with existing categories.

   ![../_images/Report_List_-_Category_Rename_Duplication_Error.png](https://devnet.logianalytics.com/hc/article_attachments/12528166514071/report_list_-_category_rename_duplication_error_196x125.png)

## Copy or Move Dashboard

1. Click the Copy or Move button to open the pop-up.
2. The pop-up is similar to Report ([Copy a Report](https://devnet.logianalytics.com/hc/en-us/articles/12528299853335-Report-List#copy-a-report) and [Move a Report](https://devnet.logianalytics.com/hc/en-us/articles/12528299853335-Report-List#move-a-report))
3. Click OK to close the pop-up and perform the operation.

## Manage Subscriptions for Current User

1. Click the Subscribe button to open Subscription box.
2. Add and Edit Dashboard Subscription are similar to [Report Subscription](https://devnet.logianalytics.com/hc/en-us/articles/12528299853335-Report-List#add-report-subscriptions-for-current-user).
3. To delete multiple subscriptions:
   1. Tick the checkboxes beside each one.
   2. Click the now enabled Delete icon at the bottom (that looks like a waste basket)
4. Click Close to collapse Subscription box.

[![../_images/Report_Subscription_-_Delete_Multiple.png](https://devnet.logianalytics.com/hc/article_attachments/12528166525207/report_subscription_-_delete_multiple_600x409.png)](https://devnet.logianalytics.com/hc/article_attachments/12528166519831/report_subscription_-_delete_multiple.png)

## Manage Schedule Deliveries

1. Click the Schedule button to open Schedule list.
2. The Schedule List, Search, Add and Delete functions are similar to [Report Designer](https://devnet.logianalytics.com/hc/en-us/articles/12528299836311-Report-Designer-Schedule).

## Print Dashboard

1. Click the Print button and select either HTML or PDF format to open Print pop-up.
2. The Print pop-up is similar to [Dashboard Designer](https://devnet.logianalytics.com/hc/en-us/articles/12528299472663-Dashboard-Designer#print-dashboard).

## Email

1. Click the Email button and select either Link or Embedded HTML to open Compose Email pop-up.
2. The Compose Email pop-up is similar to [Dashboard Designer](https://devnet.logianalytics.com/hc/en-us/articles/12528299472663-Dashboard-Designer#email-dashboard).

## View Dashboard

Click on a dashboard’s name to go to Dashboard Designer screen.

## Configure Child Report Parts

Click the Configuration mode (page flip) icon on report part’s title bar to configure it and show the Report Part Menu.

* The title and description can be edited and formatted.
* The Report Part Menu allows user to:
  + View, quick edit and design report parts in new window
  + Print and export report parts
  + Re-select report parts by clicking Select Report Part button, then select another report part in Report Part Selection pop-up similarly to the steps in [Dashboard Designer](https://devnet.logianalytics.com/hc/en-us/articles/12528299472663-Dashboard-Designer#create-a-new-dashboard).
