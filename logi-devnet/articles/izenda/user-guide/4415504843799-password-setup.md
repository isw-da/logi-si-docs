# Password Setup

## Password Setup

The **Password Setup** page allows user to create password for the first
time. The page can only be opened from a password link, which is a
short-expiry web address uniquely created from the user information.

1. Click the password link to open Password Setup page in browser.

   ![../_images/Password_Setup.png](https://devnet.logianalytics.com/hc/article_attachments/4415492507287/password_setup_404x236.png)
2. Verify the link by entering the exact information used to create the
   link and click Next.
3. In the next screen, enter the password and answers to security
   questions if available.

   > The password must meet the [Password Complexity setting in
   > Security
   > Policies](https://devnet.logianalytics.com/hc/en-us/articles/4415512092183-System-Configuration-Security-Policies#configure-password-complexity-sample).
4. Click OK to save.

## Password Change for Current User

1. Log in as the user having the password to change.
2. Click My Profile under the user name at the top right of the screen.
3. Click Security Options in the left menu, then click Change Password.

   [![../_images/Security_Options_Change_Password.png](https://devnet.logianalytics.com/hc/article_attachments/4415492507799/security_options_change_password_750x152.png)](https://devnet.logianalytics.com/hc/article_attachments/4415504422551/security_options_change_password.png)
4. Enter the current password, then the new password twice to verify in
   the Change Password pop-up.

   ![../_images/Change_Password_Pop-up.png](https://devnet.logianalytics.com/hc/article_attachments/4415504423319/change_password_pop-up_458x216.png)

## Password Reset for Another User

1. In browser, log in to Izenda as a user with User Setup permission.
2. Click Settings, then User Setup in the left menu.
3. Click Configure Password Option.
4. Click Generate Password Link.
5. Click the copy icon (that looks a bit like this ❐) to copy the
   password link to clipboard.
6. Paste and open the password link in browser to reset the password
   (similarly to [Password Setup](#password-setup-section) steps).
