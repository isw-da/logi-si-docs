# Report Viewer

# Report Viewer

The **Report Viewer** page allows users to:

|  |  |
| --- | --- |
| * view report in browser * use quick edit mode to configure report part properties * set up subscriptions * print and export reports | * email report as a link or embedded HTML * copy and move reports between categories * view report history * show and hide report header and footer |

## View Report

1. Report is initially displayed in View mode with
   all header, footer and report parts.

   [![../_images/Report_Viewer.png](https://devnet.logianalytics.com/hc/article_attachments/4415492591767/report_viewer_773x485.png)](https://devnet.logianalytics.com/hc/article_attachments/4415511838999/report_viewer.png)
2. The buttons at the top allow user to:

   * Show and Hide Header & Footer in any report which contains these options
   * Add Subscriptions (See in [Report
     List](https://devnet.logianalytics.com/hc/en-us/articles/4415504850839-Report-List#add-report-subscriptions-for-current-user))
   * Print
   * Email
   * Export to another format
   * Quick Edit
   * Design
   * Copy or Move to another name or category (See in Report List
     [Copy](https://devnet.logianalytics.com/hc/en-us/articles/4415504850839-Report-List#copy-a-report) and
     [Move](https://devnet.logianalytics.com/hc/en-us/articles/4415504850839-Report-List#move-a-report))
   * View History (See in [Report
     List](https://devnet.logianalytics.com/hc/en-us/articles/4415504850839-Report-List#view-report-history))

[![../_images/Report_Viewer_Menu.png](https://devnet.logianalytics.com/hc/article_attachments/4415492592023/report_viewer_menu.png)](https://devnet.logianalytics.com/hc/article_attachments/4415492592023/report_viewer_menu.png)

## Quick Edit Report

1. The Quick Edit button at the top switches
   to Quick Edit Mode.

   In this mode, user can configure report part properties while keeping the selected data sources.

   [![../_images/Report_Viewer_in_Quick_Edit_mode.png](https://devnet.logianalytics.com/hc/article_attachments/4415492592407/report_viewer_in_quick_edit_mode_782x477.png)](https://devnet.logianalytics.com/hc/article_attachments/4415504504983/report_viewer_in_quick_edit_mode.png)
2. After editing, clik View Mode button to switch back to View mode to
   see the effects.
3. Click Save or Save As to apply the changes to report.

Note: In the review mode, the report part backside is impacted by the “Show Preview section in Configuration Mode” option in Advanced Setting > Others tab like a report part in Report Designer. Please see [Report Part Menu has only Configuration section](https://devnet.logianalytics.com/hc/en-us/articles/4415512102551-Report-Designer-Report-Part-Menu#hide-preview-section) for more details.
