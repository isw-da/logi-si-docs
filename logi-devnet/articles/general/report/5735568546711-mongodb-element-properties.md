# MongoDB Element Properties

This topic describes the properties of an Element object in a MongoDB collection.

| Property Name | Description |
| --- | --- |
| BSON Data Type | Specifies the [data type](https://devnet.logianalytics.com/hc/en-us/articles/5735499559063-How-Designer-Gets-Data-from-MongoDB-Databases#ConvertRule) of the element. Choose an option from the drop-down list. Data type: Enumeration |
| Data Type Name | Shows the data type name of the element, which Designer changes automatically according to the BSON Data Type property. Read only. |
| Name | Specifies the name of the element. Data type: String |
| Sub Data Type | Specifies the [sub data type](https://devnet.logianalytics.com/hc/en-us/articles/5735499559063-How-Designer-Gets-Data-from-MongoDB-Databases#Sub) of the element. Choose an option from the drop-down list. Data type: Enumeration  Note icon This property takes effect when you set the BSON Data Type property to 5. |
| Sub Type Name | Shows the sub data type name of the element, which Designer changes automatically according to the Sub Data Type property. Read only. |
