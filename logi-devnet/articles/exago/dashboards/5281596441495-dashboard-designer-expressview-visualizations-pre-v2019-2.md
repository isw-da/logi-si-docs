# Dashboard Designer: ExpressView Visualizations (*pre-v2019.2*)

> ![red exclamation point](https://devnet.logianalytics.com/hc/article_attachments/5432173635607/criticalicon.gif "Important")
>
> This topic applies to *pre-v2019.2* of the application. For current Dashboard documentation, see [Dashboard Designer (v2019.2+)](https://devnet.logianalytics.com/hc/en-us/articles/5281580652951-Dashboard-Designer-v2019-2-).

Create tabular reports and visualizations directly on the Dashboard with **ExpressView** visualizations. To do so, either:

* Drag the ![Add2.png](https://devnet.logianalytics.com/hc/article_attachments/5432238606103/add2.png)**New Tile** icon onto the **Dashboard** and select **Visualization.**
* Drag a field from the Data Pane directly onto the **Dashboard**.

A KPI chart is the default visualization for a single data field. To add data fields, drag fields from the Data Pane onto the visualization. To swap to a tabular report, click the **Show Table**![TileModeTabularSelected.png](https://devnet.logianalytics.com/hc/article_attachments/5432355766807/tilemodetabularselected.png) icon. To swap back to a chart, click the **Show Chart**![TileModeVizSelected.png](https://devnet.logianalytics.com/hc/article_attachments/5432350467479/tilemodevizselected.png) icon.

For details on building a tabular **ExpressView** visualization, see [ExpressView: Introduction (pre-v2021.1)](https://devnet.logianalytics.com/hc/en-us/articles/5281612161815-ExpressView-Introduction-pre-v2021-1-)

For details on building a chart **ExpressView** visualization, see [ExpressView: Visualizations (pre-v2021.1)](https://devnet.logianalytics.com/hc/en-us/articles/5281597688983-ExpressView-Visualizations-pre-v2021-1-).

To save the visualization as a new **ExpressView,** click the menu ![screen.dashboard_tilemenu.png](https://devnet.logianalytics.com/hc/article_attachments/5432329696535/screen.dashboard_tilemenu.png) icon and select **Save as ExpressView**. The visualization will open in the **[ExpressView Designer](https://devnet.logianalytics.com/hc/en-us/articles/5281612161815-ExpressView-Introduction-pre-v2021-1-)**.

## Suppress Detail Rows v2017.3+

To remove all data rows from the visualization and only show summary data, from the **Settings** pane, select **Suppress Tabular Detail Rows**. This may improve the performance for visualizations that do not depend on the detail values.
