## Transcript

Welcome to the Exago Technical Training Series!

These videos are short informational segments aimed at getting administrators and developers more comfortable with the process of implementing Exago BI into their application. Some of the content will be separated to demonstrate both Windows and Linux operating systems. So, all of the videos will not be relevant to every technology.

The series begins with installation of the various components of Exago BI! We explain necessary configurations of the web server, the REST Web Service API and the Scheduler Service. Next, we explain the Admin Console and how to create the base configuration. By the end of this series, system administrators will understand how to add a new Data Source, configure Data Objects and setup Exago’s Storage Management system to control the folders and reports users are authorized to access. To bring the video series to a conclusion, we show a complete integration example using a minimum viable implementation with a sample host application.

Let’s get started! The very first step to deploying Exago BI is to download the application installer. Exago BI downloads can be found on our Support Center, [support.exagoinc.com](mailto:exagosupport@insightsoftware.com). Click Downloads in the top-right corner. Documentation for the current supported three major versions is available to all users but downloading application installers requires a Support Center account. Contact an Exago Customer Success Manager for more information. The major version is listed on the left and the release number on the right. The most recent installer is always displayed at the top. Once a version is selected, links are available for either the Windows installer, the Linux installer or the ZIP archive, for scripting the Exago install into another application’s installer. When updating Exago, always be sure to read the Updating Guide. Updates made to our APIs, configuration files or other potentially breaking changes are listed there. The Release Notes are also published below the download links.

This concludes the Introduction video. Keep watching to learn to install Exago on either Windows or Linux!

## About the Technical Training Series

The **Technical Training Series** is a new video series produced by Exago’s Education Group designed to introduce the concepts of installing, configuring and deploying Exago to developers and system administrators. There are two paths to follow: one for Linux environments and the other for Windows. The two paths converge when it comes time to setup Exago’s base configuration with the Admin Console. The series concludes with a basic implementation example with the REST Web Service API.
