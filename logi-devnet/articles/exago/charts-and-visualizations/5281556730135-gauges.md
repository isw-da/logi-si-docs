# Gauges

# Gauges

A **Gauge** gives a visual representation of the scale of a value. Refer to the [Examples](#Examples) at the end of this topic.

## Gauge Wizard

To add a gauge to an Advanced Report:

1. Select a cell on the design grid where the gauge will reside. Gauges can be placed in any section of the report except for the **Page Header** and **Page Footer** sections.
2. Open the Gauge Wizard:
   * *(v2021.1+)* Click the **Insert**![insert icon](https://devnet.logianalytics.com/hc/article_attachments/5432182571543/insert.png)![DropDownArrow2.png](https://devnet.logianalytics.com/hc/article_attachments/5432176827159/dropdownarrow2.png) icon on the toolbar, then click ![gauge icon](https://devnet.logianalytics.com/hc/article_attachments/5432233271959/gauge.png)**Gauge**
   * *(pre-v2021.1)* Click the **Gauge** ![gauge icon](https://devnet.logianalytics.com/hc/article_attachments/5432233271959/gauge.png) icon in the toolbar.

The Gauge Wizard will appear. The Gauge Wizard has two tabs: Appearance and Data, detailed below.

### Appearance

In the Appearance tab, choose the style and size of the gauge.

#### Type

Select the **Type** of gauge that will appear:

* ![Semi-circular angular guage with a needle pointing to 75% of the scale](https://devnet.logianalytics.com/hc/article_attachments/5432362933911/gaugetypeangular.png)*Angular* — the indicating needle sweeps from left (minimum) to right (maximum) along a curved dial
* ![Rectangular guage with pointer at 90% of the scale](https://devnet.logianalytics.com/hc/article_attachments/5432336364567/gaugetypelinear.png)*Linear* — the indicating needle sweeps from left (minimum) to right (maximum) along a horizontal line
* ![Tri-colored circles with arrows pointing left and right](https://devnet.logianalytics.com/hc/article_attachments/5432391499927/gaugetypebulb.png)*Bulb* — value is indicated by the presence of a single color, similar to a flag or semaphore. The value is shown below the bulb.
* ![A thermmometer filled with red liquid and small arrow pointing at 90% up the scale](https://devnet.logianalytics.com/hc/article_attachments/5432391519127/gaugetypethermometer.png)*Thermometer* — value is indicated by the thermometer tube filling from the bottom (minimum) to the top (maximum)

#### Dimensions

Select the size (**Dimensions**) of the gauge by either:

* **Drag-and-drop** the resizing handle ![chart_size_grippy.png](https://devnet.logianalytics.com/hc/article_attachments/5432379644055/chart_size_grippy.png) at the bottom-right corner.
* Check the **Fit to Cell** checkbox and resize the chart cell on the Design Grid.
* Enter a custom **Height** and **Width** in pixels directly into the corresponding inputs

> ![light bulb](https://devnet.logianalytics.com/hc/article_attachments/5432173099031/noteicon.jpg "Note")
>
> Angular gauges have a minimum width of 120 pixels.

![The Advanced Report Designer's Gauge Wizard building a linear guage](https://devnet.logianalytics.com/hc/article_attachments/5432391560343/gzjafog3ft.png)

The Appearance tab of the Gauge Wizard. A linear gauge 106 pixels tall and 277 pixels tall is selected.

Click **Next** to advance to the **Data** tab.

### Data

Use the Data tab to choose the value presented in the gauge, the range of values and the coloring of those values.

#### Value and Range

![The Value dropdown is selected with an aggregate from the report. The Static Value radio button is clicked and the Maximum value field is 100.](https://devnet.logianalytics.com/hc/article_attachments/5432363048471/b9pgivonnz.png)

1. From the **top dropdown**, choose a cell with a numeric value that determines the gauge’s indicator position. In the example above, a cell with the formula `=aggsum({OrderDetails.Revenue})` that computes the sum of all order revenues is selected.
2. Specify the range (minimum and maximum) values by choosing one of the **Provide range as****options**. The Min and Max values must be numeric.
   * *Static value* — specify explicit numeric **Min** and **Max** values  
     ![cysqhkiQ6Z.png](https://devnet.logianalytics.com/hc/article_attachments/5432391596183/cysqhkiq6z.png)
   * *Cell value* — choose cells on the report that represent the **Min** and **Max** values  
     ![7QCKRmbLfS.png](https://devnet.logianalytics.com/hc/article_attachments/5432363095191/7qckrmblfs.png)
     > ![light bulb](https://devnet.logianalytics.com/hc/article_attachments/5432173099031/noteicon.jpg "Tip")
     >
     > To create a “semi-static” gauge, include a cell on the report with the desired static value preceded with an equals sign. In the example above, the gauge’s minimum value will be 0 and the maximum value will be whatever the value of the `@SalesGoal@` parameter is at the time of report execution.

#### Color Ranges

1. Choose a **Color by** method to determine how the color breakpoints will be applied to the gauge’s dial:
   * *Percentage* — breakpoints are calculated as a percentage of the data values with the smallest value at 0% and the largest value at 100%
   * *Static Value* — breakpoints in color happen at explicitly defined values of the data
   * *Cell Value* — breakpoints in color happen at value changes that occur in cells on the report
2. **Add** or **Remove** color breakpoints, and select the locations of the breakpoints. Breakpoints values must be in ascending numeric order. There are three breakpoints by default.  
   > ![light bulb](https://devnet.logianalytics.com/hc/article_attachments/5432173099031/noteicon.jpg "Note")
   >
   > Thermometers only have one color.

   1. Click **Add ![Add2.png](https://devnet.logianalytics.com/hc/article_attachments/5432238606103/add2.png)** to add additional breakpoints to the end of the list. Click ![Delete.png](https://devnet.logianalytics.com/hc/article_attachments/5432311840791/delete.png) Remove to remove the break point at the end of the list.
   2. Enter a hexadecimal color code into the text field, or click the **Color Chooser** icon to select a color for each break point.  
      ![G3VEGEniaG.png](https://devnet.logianalytics.com/hc/article_attachments/5432363133335/g3vegeniag.png)
3. Click **Finish** to close the Gauge Wizard and add the visualization to the report.
