# Report Wizard

# Report Wizard

The **New Report Wizard** opens at the start of each new Express Report, Advanced Report, or CrossTab Report. The wizard lets you add and configure data and quickly set a layout for the report before launching the full Report Designer.

![screen.reportwizard.png](https://devnet.logianalytics.com/hc/article_attachments/5432393999639/screen.reportwizard.png)

The Layout tab of the Express Report Wizard

## Using the Report Wizard

The Report Wizard has several tabs, which progress through from left to right. The following topics detail how to use each page in the wizard:

[Name](https://devnet.logianalytics.com/hc/en-us/articles/5281542311703-Report-Wizard-Name) • [Categories](https://devnet.logianalytics.com/hc/en-us/articles/5281549844631-Report-Wizard-Categories) • [Sorts](https://devnet.logianalytics.com/hc/en-us/articles/5281573193111-Report-Wizard-Sorts) • [Layout](https://devnet.logianalytics.com/hc/en-us/articles/5281564615959-Report-Wizard-Layout)

> ![light bulb](https://devnet.logianalytics.com/hc/article_attachments/5432173099031/noteicon.jpg "Note")
>
> Only **Express Reports** have an Options tab. **CrossTab Reports** do not have a Sorts tab. The Layout tab differs slightly for each report type. The Name and Categories tabs are mandatory. If a tab is not available, the system administrator may have disabled it.
