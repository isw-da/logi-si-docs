# *v2019.1* Configuration Changes

Travis McGrath explains new configuration options coming in version *v2019.1*!

**There are two minor corrections to the content in the lab:**

* Exago *v2019.1*‘s release date is going to be later in May and will not May 3rd. We are using that extra time to conduct additional testing before the release.
* The Configuration Options shown in the Admin Console will not contain an Expiration Date setting as mentioned.
* Server and Action Events should be stored in the *Dynamic* (child) configuration file.
