# Custom Functions

Nick explains how to build custom functions using Exago BI, touching on how to access and leverage the SessionInfo object in the process. The lab walks through building three custom functions total. The first retrieves a report's export type, the second loads company logos dynamically from a file system, and the third pings an external API for tax rate information.
