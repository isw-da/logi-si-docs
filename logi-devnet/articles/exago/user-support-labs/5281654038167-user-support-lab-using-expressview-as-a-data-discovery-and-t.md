# User Support Lab — Using ExpressView as a Data Discovery and Troubleshooting Tool

On this Support Lab Nick explains how to use ExpressView as a data discovery tool that can help troubleshoot Advanced Reports. He walks through examples of breaking down formulas into pieces to check for syntax errors, as well as understanding if joins are the cause of incorrect report outputs.
