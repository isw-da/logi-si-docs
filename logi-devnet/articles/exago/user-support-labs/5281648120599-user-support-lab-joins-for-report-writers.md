# User Support Lab — Joins for Report Writers

* What is a join?
* What types of joins exist?
* As an end user building reports, what benefit can understanding joins provide me?
* Finally, how can I use this understanding to configure joins on the report level?

Travis will provide clarity to the above questions and explain how even a basic understanding of joins can help with proper report design.
