# What’s the Difference between the Composer REST API and the ComposerJavaScript SDK?

Composer's JavaScript SDK and REST API are related but distinct products.

* The Composer REST API provides methods for retrieving, updating, and deleting Composer metadata pertaining to accounts, connections, users, and so forth.

  API documentation is provided with your Composer installation at this link: `https://<composer-URL>/composer/swagger-ui.html`.
* The Composer Software Development Kit (SDK) is a set of tools to help software developers extend Composer's capabilities and embed them into custom and third-party applications. It is intended for use by software developers.
  See [*Get Started with the SDK*](https://devnet.logianalytics.com/hc/en-us/articles/4402955434263-Get-Started-with-the-SDK).
