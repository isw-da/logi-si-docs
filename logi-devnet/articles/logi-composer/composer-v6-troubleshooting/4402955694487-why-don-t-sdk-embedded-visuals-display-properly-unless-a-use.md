# Why Don't SDK-Embedded Visuals Display Properly Unless a User is Logged In?

Please make sure that the API key that was generated for your SDK implementation has appropriate access to ALL the data sources being used by your embedded visuals in your custom HTML page.

Please refer to our [*Generate a Security Key*](https://devnet.logianalytics.com/hc/en-us/articles/4402963047831-Generate-a-Security-Key) and [*Embed a Visual in Your Web Page*](https://devnet.logianalytics.com/hc/en-us/articles/4402955438231-Embed-a-Visual-in-Your-Web-Page) support topics for more information.
