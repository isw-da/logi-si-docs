# Change Passwords

![](https://devnet.logianalytics.com/hc/article_attachments/4404959473559/noteicon.jpg) Only Composer administrators or groups that have been assigned the **Can Administer Users** privilege can change or reset user passwords. They can also require a user to change their password when they first log in.

If you are logged into Composer as an administrator of an account, you can change or reset the login passwords of any user in the account. If you are logged in as a supervisor, you only change your own password. No other passwords can be changed in the supervisor UI. See [*Change a Supervisor Password*](https://devnet.logianalytics.com/hc/en-us/articles/4402955414807-Change-a-Supervisor-Password).

To change or reset a password:

1. Log into Composer as a Composer administrator or a user who has been assigned to a group with [user management privileges](https://devnet.logianalytics.com/hc/en-us/articles/4402955416087-Group-Privilege-Reference).
2. Select **Users & Groups** on the [UI menu](https://devnet.logianalytics.com/hc/en-us/articles/4402963040407-The-Composer-UI-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4404952783767/hamburger.png)). The Users and Groups page appears. It contains two options: **Users** and **Groups**. By default, **Users** is selected.
3. In the list on the left of the page, select the user whose password you want to reset. The account details for that user appear on the right side of the page.
4. On the **Info** tab, select **Change Password**.
5. Type the new password in the **Password** and **Confirm Password** boxes.
6. Optionally, change the **Require password change** switch. When this switch is set to **Yes**, the user is prompted (and forced) to change their password when they next attempt to log in. When this switch is set to **No**, the user is not prompted to change their password. The default setting for this switch is **No**.
7. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404959575831/save-blue-supervisor-btn_63x22.png) to save the new password. As soon as you do this, the user can log into Composer using the new password.
