# Remove Account Definitions

To remove a Composer account definition:

1. Log into Composer as a supervisor of the account.
2. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743720343/hamburger.png) to access the [supervisor menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859441687-The-Composer-Supervisor-Menu) and then select **Accounts**.

   The Manage Accounts page appears. It lists the accounts that have been defined.
3. In the list of accounts, locate the account you want to delete and select the ![](https://devnet.logianalytics.com/hc/article_attachments/4406747463575/delete-blue-x.png) in the **Delete** column.

   A warning dialog appears that prompts you to confirm that you want to delete the account.
4. Select **![](https://devnet.logianalytics.com/hc/article_attachments/4406743948439/delete-blue-btn.png)** on the warning dialog to remove the account definition. All user definitions, data source configurations, data source connections, and custom dashboards that are not associated with other Composer accounts are removed.
