# Add Composer Account Definitions

To create a new Composer account definition:

1. Log into Composer as a supervisor of the account.
2. Select **Accounts** on the [supervisor menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955649943-The-Composer-Supervisor-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4404952783767/hamburger.png)). The Manage Accounts page appears. It lists the accounts that have been defined.
3. To add a new account, select ![](https://devnet.logianalytics.com/hc/article_attachments/4404952932503/add-acct-btn.png) on the Manage Accounts page.

   The Create New Account page appears.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404952932887/acct-new_288x254.png)
4. Enter a name for the new account in the **Account name** box. The name must be at least four characters long.
5. Every account must have at least one administrator. Select one of the following options:

   * Select **Assign Existing User As Admin** to select an existing user as the administrator for the account. After selecting this option, select ![](https://devnet.logianalytics.com/hc/article_attachments/4404959581079/select-users-btn.png) and then select a user on the Select User(s) dialog that appears. After you have selected one or more users on the dialog, select ![](https://devnet.logianalytics.com/hc/article_attachments/4404959581335/apply-white.png).
   * Select **Create A New Admin User** to define a new user to use as the administrator for the account. Supply the user name and password for the new user in the **User name**, **Password**, and **Confirm Password** boxes. The new user is assigned to the new account as an administrator.
6. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404952933143/create-acct-btn.png).

   The new Composer account is created and is automatically enabled. The administrator you assigned to the account is included as an account user (see [*List and Review User Definitions*](https://devnet.logianalytics.com/hc/en-us/articles/4402955421207-List-and-Review-User-Definitions)). No groups are yet assigned to the account. To assign groups to the account, see [*Manage Group Definitions*](https://devnet.logianalytics.com/hc/en-us/articles/4402962859799-Manage-Group-Definitions). To disable the account, see [*Enable or Disable Composer Accounts*](https://devnet.logianalytics.com/hc/en-us/articles/4402955408663-Enable-or-Disable-Composer-Accounts).
