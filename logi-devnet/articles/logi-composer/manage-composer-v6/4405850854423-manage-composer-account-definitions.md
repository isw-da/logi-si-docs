# Manage Composer Account Definitions

As the Composer supervisor, you can manage Composer accounts.

Composer accounts can be used to separate Composer resources, as necessary. Users can be assigned to multiple accounts. However, Composer resources, such as group definitions, data source configuration definitions, data store connection definitions, dashboards and visuals are only available in the Composer accounts in which they are defined.
