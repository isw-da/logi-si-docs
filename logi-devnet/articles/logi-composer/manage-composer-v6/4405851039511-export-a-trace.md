# Export a Trace

To export a trace:

1. While viewing a trace, select the JSON link in the upper right corner to generate an exported JSON trace:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406747419799/trace-export-how_714x137.png)

   A dialog appears containing the trace.
2. Select the download icon on the upper right corner to save the JSON file locally.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743805719/trace-export_576x310.png)

   The resulting trace JSON file can be provided to Composer Support, along with the accompanying microservice logs for diagnosis. See [*Diagnose Problems*](https://devnet.logianalytics.com/hc/en-us/articles/4405859335575-Diagnose-Problems).

To view and exported trace, see [*View an Exported Trace*](https://devnet.logianalytics.com/hc/en-us/articles/4405851041303-View-an-Exported-Trace)
