# Specify User Regional Settings

You can set regional settings for a user definition if you are logged in as a Composer account administrator, a Composer supervisor, or a user who is assigned to a group with [user management privileges](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference). Regional settings determine the language of the user and the format of numeric fields in a data source. For example, a user based in the United States would see number fields formatted for Italian currency differently than a user based in Italy. For more information about data formatting for fields, see [Number Formatting for Data Sources](https://devnet.logianalytics.com/hc/en-us/articles/4405851006231-Configure-Number-Formatting).

The user locale can be set using the **Regional Settings** tab.

![](https://devnet.logianalytics.com/hc/article_attachments/4406743939223/user-reg-set_480x306.png)

To access the **Regional Settings** tab, see [*Add User Definitions*](https://devnet.logianalytics.com/hc/en-us/articles/4405850861463-Add-User-Definitions) or [*Modify User Definitions*](https://devnet.logianalytics.com/hc/en-us/articles/4405859121047-Modify-User-Definitions). For information on the user definition settings that can be changed on other tabs of the user definition editor, see:

* [*Assign and Remove Users in Composer Accounts*](https://devnet.logianalytics.com/hc/en-us/articles/4405859115671-Assign-and-Remove-Users-in-Composer-Accounts)
* [*Specify General User Definition Information*](https://devnet.logianalytics.com/hc/en-us/articles/4405850865047-Specify-General-User-Definition-Information)
* [*Specify Custom User Attributes*](https://devnet.logianalytics.com/hc/en-us/articles/4405850862359-Specify-Custom-User-Attributes)
* [*Enable and Disable User Definitions*](https://devnet.logianalytics.com/hc/en-us/articles/4405859118359-Enable-and-Disable-User-Definitions)

To set the user locale for a user definition:

1. Access the **Regional Settings** tab for a user definition. See [*Add User Definitions*](https://devnet.logianalytics.com/hc/en-us/articles/4405850861463-Add-User-Definitions) or [*Modify User Definitions*](https://devnet.logianalytics.com/hc/en-us/articles/4405859121047-Modify-User-Definitions).
2. Select the location for the user definition in the drop-down list in the **Regional Presets** box.
3. When all values have been specified, select ![](https://devnet.logianalytics.com/hc/article_attachments/4406747458071/save-blue-supervisor-btn_63x22.png) to save the user definition.
