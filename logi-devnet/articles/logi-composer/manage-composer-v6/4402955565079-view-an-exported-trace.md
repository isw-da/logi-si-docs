# View an Exported Trace

You can load an exported JSON trace file into a local instance to view graphically.

To view an exported trace:

1. Log into any tracing facility, including the Composer tracing facility.
2. Select **View Saved Trace** from the menu.
3. Specify the JSON trace file that should be uploaded.

   The trace file can now be viewed in as if it had been created locally.
