# Add and Remove Members of a Group

You can add or remove users in a group definition when you are logged in as a Composer system administrator or as a user who has been assigned to a group with [group management privileges](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference). You can add or delete users in a group at any time.

![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) Management of the [supplied **Administrators** group](https://devnet.logianalytics.com/hc/en-us/articles/4405859110935-About-Supplied-Group-Definitions) can only be performed by a member of that group or by a user in a group with *all* the following [privileges](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference): **Can Administer Users**, **Can Administer Groups**, and **Can Administer Dashboards**.

To add or remove members in a group:

1. Log into Composer as a Composer administrator or a user who has been assigned to a group with [group management privileges](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference).

   If the user name you log in with is also associated with other Composer accounts, verify that the correct account is selected. See [*Switch Accounts*](https://devnet.logianalytics.com/hc/en-us/articles/4405850869015-Switch-Accounts).
2. Select **Users & Groups** on the [UI menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859444119-The-Composer-UI-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4406743720343/hamburger.png)). The Users and Groups page appears. It consists of two sections: **Users** and **Groups**.
3. List the group definitions defined for the account. See [*List and Review Group Definitions*](https://devnet.logianalytics.com/hc/en-us/articles/4405850857495-List-and-Review-Group-Definitions).
4. Select the group definition to which you want to add or remove members. The group definition editor appears on the right side of the page.
5. Select the **Members** tab.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406747462423/group-members_768x383.png)
6. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406747462679/add-members.png). An Add Member(s) pop-up dialog appears.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743947927/group-members1_192x223.png)
7. Select (check) the names of the users in the dialog that you want to add to the group. To remove members, clear (uncheck) the checkboxes corresponding to the user names in the dialog that you want to remove from the group.

   If all users should be added or removed in the group, select the **Select All** option.

   You can sort the user list by name in ascending or descending order to help you locate the user names you need.

   Use the search bar to easily locate a specific user when the user list is very long.
8. After the users are selected appropriately, select ![](https://devnet.logianalytics.com/hc/article_attachments/4406747463191/apply1-btn.png). The selected user(s) are added or removed in the editor, but the group definition has not yet been saved.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743948183/group-members2_768x382.png)

   Note that you can remove users from the group on this screen by selecting the ![](https://devnet.logianalytics.com/hc/article_attachments/4406743726743/delete-open.png) icon associated with their user name. Then select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743948439/delete-blue-btn.png) on the resulting confirmation dialog.
9. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406747400087/save-blue-btn.png) to save the group definition. The selected user(s) are added or removed in the group definition. A save confirmation message displays.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743948951/group-save-msg.png)
