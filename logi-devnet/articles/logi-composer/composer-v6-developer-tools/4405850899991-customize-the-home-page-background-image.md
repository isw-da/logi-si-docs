# Customize the Home Page Background Image

The background image appears on the [Home page](https://devnet.logianalytics.com/hc/en-us/articles/4405851121559-Home-Page).

To customize the Login screen background image:

1. Log into Composer as a supervisor.
2. Select **Customize UI** on the [supervisor menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859441687-The-Composer-Supervisor-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4406743720343/hamburger.png)). The Customize UI page appears.
3. Under Login and About , locate the **Home Background Image** box. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743925783/browse_58x18.png) to browse for and select a new background image. The image must be in `.jpg` or `.png` format with a transparent background.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743926167/custom-login_576x405.png)

   The selected image is rendered on the Customize UI page after it is selected.
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743816343/save-blue-btn_43x23.png) to save and apply your changes.
