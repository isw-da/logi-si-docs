# Customize the Application Favicon

You can customize the favicon used by the Composer application. The favicon is displayed on the tab of your browser window:

![](https://devnet.logianalytics.com/hc/article_attachments/4404952918167/application-title.png)

To customize the Composer application favicon:

1. Log into Composer as a supervisor.
2. Select **Customize UI** on the [supervisor menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955649943-The-Composer-Supervisor-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4404952783767/hamburger.png)). The Customize UI page appears.
3. Under Application, locate the **Favicon** box. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404959569943/browse_58x18.png) to browse for and select a new icon. The icon must be in `.ico` format with a recommended size of 32 x 32 pixels.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404952917911/custom-application_576x329.png)

   The selected icon is rendered on the Customize UI page after it is selected.
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404959543447/save-blue-btn_43x23.png) to save and apply your changes.
