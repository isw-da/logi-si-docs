# Manage Custom Charts in the UI

This section describes how you can manage custom charts in the Composer UI.

![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) You must be an administrator to manage custom charts.

* [*List Custom Charts*](https://devnet.logianalytics.com/hc/en-us/articles/4405850979223-List-Custom-Charts)
* [*Enable the Custom Chart in UI Menus*](https://devnet.logianalytics.com/hc/en-us/articles/4405850978199-Enable-the-Custom-Chart-in-UI-Menus)
* [*Delete a Custom Chart*](https://devnet.logianalytics.com/hc/en-us/articles/4405859252119-Delete-a-Custom-Chart)
* [*Download a Custom Chart*](https://devnet.logianalytics.com/hc/en-us/articles/4405859253271-Download-a-Custom-Chart)
* [*Import a Custom Chart Using the UI*](https://devnet.logianalytics.com/hc/en-us/articles/4405859254679-Import-a-Custom-Chart-Using-the-UI)

You can also maintain custom charts using the CLI. See [*Maintain Custom Charts Using the Custom Chart CLI*](https://devnet.logianalytics.com/hc/en-us/articles/4405859247127-Maintain-Custom-Charts-Using-the-Custom-Chart-CLI).
