# Customize the Home Page Background Image

The background image appears on the [Home page](https://devnet.logianalytics.com/hc/en-us/articles/4402963020951-Home-Page).

To customize the Login screen background image:

1. Log into Composer as a supervisor.
2. Select **Customize UI** on the [supervisor menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955649943-The-Composer-Supervisor-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4404952783767/hamburger.png)). The Customize UI page appears.
3. Under Login and About , locate the **Home Background Image** box. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404959569943/browse_58x18.png) to browse for and select a new background image. The image must be in `.jpg` or `.png` format with a transparent background.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404952916503/custom-login_576x405.png)

   The selected image is rendered on the Customize UI page after it is selected.
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404959543447/save-blue-btn_43x23.png) to save and apply your changes.
