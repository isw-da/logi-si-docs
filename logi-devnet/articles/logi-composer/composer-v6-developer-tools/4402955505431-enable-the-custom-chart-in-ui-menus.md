# Enable the Custom Chart in UI Menus

To enable a custom chart to appear in UI menus:

1. Log into the Composer UI as an administrator.
2. Access the Manage Custom Charts page. See [*List Custom Charts*](https://devnet.logianalytics.com/hc/en-us/articles/4402955506071-List-Custom-Charts).
3. Locate the visual in the table of visuals.
4. Select the **Enable Menu** option for the visual.
