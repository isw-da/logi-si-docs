# Customize the Documentation Link

To customize the documentation link:

1. Log into Composer as a supervisor.
2. Select **Customize UI** on the [supervisor menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859441687-The-Composer-Supervisor-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4406743720343/hamburger.png)). The Customize UI page appears.
3. Under Banner, locate the **Documentation Link** slider. To enable your custom documentation link, slide the **Documentation Link** slider to the right (on). To disable your custom documentation link, slide the **Documentation Link** slider to the left (off). By default the **Documentation Link** slider is on.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743927063/custom-banner_576x251.png)
4. Specify the documentation link in the box below the **Documentation Link** slider. By default, the link is set to `https://documentation.logianalytics.com/composerltsactive/default.htm`.
5. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743816343/save-blue-btn_43x23.png) to save and apply your changes.
