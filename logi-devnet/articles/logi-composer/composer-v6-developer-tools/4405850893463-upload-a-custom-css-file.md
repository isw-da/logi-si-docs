# Upload a Custom CSS File

A custom CSS file can be uploaded to modify the default Composer skin.

![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) Composer recommends you to check the HTML structure of the page to provide the rules for the corresponding selectors in your CSS file. To override existing CSS rules, use `!important`.

To upload a custom CSS file:

1. Log into Composer as a supervisor.
2. Select **Customize UI** on the [supervisor menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859441687-The-Composer-Supervisor-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4406743720343/hamburger.png)). The Customize UI page appears.
3. Under Application, locate the **Custom CSS** box. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743925783/browse_58x18.png) to browse for and select a CSS file.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743927575/custom-application_576x329.png)
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743816343/save-blue-btn_43x23.png) to save and apply your changes.
