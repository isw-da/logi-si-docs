# Customize the Header Logo

You can change the header logo. It appears on the banner of the Composer working area:

![](https://devnet.logianalytics.com/hc/article_attachments/4406747453975/banner.jpg)

To customize the header logo:

1. Log into Composer as a supervisor.
2. Select **Customize UI** on the [supervisor menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859441687-The-Composer-Supervisor-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4406743720343/hamburger.png)). The Customize UI page appears.
3. Under Banner, locate the **Header Logo** box. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743925783/browse_58x18.png) to browse for and select a new logo. The logo must be in `.svg` or `.png` format with a transparent background. It can have a maximum height of 72 pixels.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743927063/custom-banner_576x251.png)

   The selected logo is rendered on the Customize UI page after it is selected.
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743816343/save-blue-btn_43x23.png) to save and apply your changes.
