# Supplied Themes

# Supplied Themes

When Composer is installed, three themes are provided: **composer**, **modern** (light) and **dark**. The **composer** theme is used by default. You can switch to (activate) a different theme when needed. See [*Activate a Theme*](https://devnet.logianalytics.com/hc/en-us/articles/4405859476887-Activate-a-Theme).
