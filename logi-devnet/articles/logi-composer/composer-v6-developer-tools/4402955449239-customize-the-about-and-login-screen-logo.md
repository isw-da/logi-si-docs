# Customize the About and Login Screen Logo

The logo appears on both the [About box](https://devnet.logianalytics.com/hc/en-us/articles/4402963017367-About-Dialog) and the [Login screen](https://devnet.logianalytics.com/hc/en-us/articles/4402955424535-Log-Into-the-Composer-UI).

To customize the About and Login screen logos:

1. Log into Composer as a supervisor.
2. Select **Customize UI** on the [supervisor menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955649943-The-Composer-Supervisor-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4404952783767/hamburger.png)). The Customize UI page appears.
3. Under Login and About , locate the **Login Page Logo** box. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404959569943/browse_58x18.png) to browse for and select a new logo. The logo must be in `.svg` or `.png` format with a transparent background. In addition, it must be 500 pixels wide and 184 pixels high.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404952916503/custom-login_576x405.png)

   The selected logo is rendered on the Customize UI page after it is selected.
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404959543447/save-blue-btn_43x23.png) to save and apply your changes.
