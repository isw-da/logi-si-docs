# Customize the Application Title

You can customize the Composer application title. The application title is displayed on the tab of your browser window:

![](https://devnet.logianalytics.com/hc/article_attachments/4406743927319/application-title_192x27.png)

To customize the Composer application title:

1. Log into Composer as a supervisor.
2. Select **Customize UI** on the [supervisor menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859441687-The-Composer-Supervisor-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4406743720343/hamburger.png)). The Customize UI page appears.
3. Under Application, locate the **Page Title** box. Specify a new title in the box. By default, the title is set to Composer.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743927575/custom-application_576x329.png)
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743816343/save-blue-btn_43x23.png) to save and apply your changes.
