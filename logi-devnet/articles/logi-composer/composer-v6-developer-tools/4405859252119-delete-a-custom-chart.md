# Delete a Custom Chart

To delete a custom chart:

1. Log into the Composer UI as an administrator.
2. Access the Manage Custom Charts page. See [*List Custom Charts*](https://devnet.logianalytics.com/hc/en-us/articles/4405850979223-List-Custom-Charts).
3. Locate the visual in the table of visuals.
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743722391/trashcan.png) for the visual in the Delete column of the table.
