# Download a Custom Chart

To download a custom chart:

1. Log into the Composer UI as an administrator.
2. Access the Manage Custom Charts page. See [*List Custom Charts*](https://devnet.logianalytics.com/hc/en-us/articles/4405850979223-List-Custom-Charts).
3. Locate the visual in the table of visuals.
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743889815/download_13x14.png) for the visual. The `.zip` file for the visual is downloaded.

![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) When you download the visual from the Manage Custom Charts page, the zip file includes a file called `version` that contains the version of the Composer server. If you later try to import this zip file to the Composer server using a different version an error will occur indicating that the versions do not match. An easy workaround is to modify the version in the `version` file before trying the import.
