# Composer 6.9.17 Enhancements

This topic describes the significant enhancements in Composer 6.9.17.

* [Time Bar](#time-bar)
* [Loading Times](#loading-times)

## Time Bar

The time bar and Data Details work area in Composer now display time at the same level of granularity you define for your visual at the source level.

## Loading Times

Composer now returns lists of users more efficiently when called in the user interface or using the API. User Name and User ID information are provided: select an individual user to return that user’s full user information.
