# Logi Composer (Zoomdata) Release Vehicles

Logi Composer (Zoomdata) offers multiple release vehicles to suit the varying needs of its customer base. Some customers want faster, more incremental access to new features while others value invariability and feature predictability more than a new feature set. Read below to understand the differences between our long-term supported (LTS) release and rapid release models, to help you choose the one which is right for you. In addition, see [*Request and Apply a New License Key*](https://devnet.logianalytics.com/hc/en-us/articles/4405851108247-Request-and-Apply-a-New-License-Key) to obtain a license appropriate for your release of choice from our support team. For information about retired releases, see [*Retired Releases*](https://devnet.logianalytics.com/hc/en-us/articles/4405859450391-Retired-Releases).

### LTS Support Releases (Currently Composer Versions 5 and 6)

Our LTS Support releases receive stability updates and maintenance patches. This allows you to use a predictable and stable version of the product, while offering fixes and security patches as needed. If you don’t want to frequently introduce new features to your end-users or if your product configuration has many customizations, we recommend selecting this version. The documentation for these releases is always up to date with the latest release. Changes to existing functionality, bug fixes, and workarounds are described in the Release Notes.

![](https://devnet.logianalytics.com/hc/article_attachments/4406743739159/criticalicon.gif) Support for Zoomdata 4.9 will end on September 25, 2021. With the release of Composer 6.9 LTS, Zoomdata 4.9 entered Passive Support. Our LTS Passive Support release receives only critical patches and security updates until its end-of-life date. Security updates are not applicable to the third-party dependencies. We encourage you to migrate to our most current LTS release. The documentation for the LTS Passive Support release is updated only when necessary. If there are changes to existing functionality or workarounds that you need to be aware of, you can find the information you need in the Release Notes.

![](https://devnet.logianalytics.com/hc/article_attachments/4406743739159/criticalicon.gif) Zoomdata versions 2.6 and 3.7 are no longer supported. For information about these retired release, see [Retired Releases](https://documentation.logianalytics.com/zoomdataltsactive/topics/release-notes/retired-release.html).

### Rapid Release (Currently, Composer 7)

The newest version of Composer follows a rapid release cadence and is still in development. This release model offers new functionality and updates every four weeks. These releases are intended for you to take into your development environments to test new features and plan ahead for upgrades. Each rapid release includes new features, updates to existing functionality, and maintenance patches all rolled into one release. The documentation for the rapid release is always up to date. Changes to existing functionality, bug fixes, and workarounds are described in the Release Notes.
