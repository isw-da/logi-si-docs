# Composer 6.9.19 Enhancements

This topic describes the significant enhancements in Composer 6.9.19.

* Expanded OpenStreetMap Support

## Expanded OpenStreetMap Support

You can now create map visuals using a custom URL that points to a local or other specific instance of OpenStreetMap. Enter the URL in the **Provider URL** field, available after selecting OpenStreetMap as the Tile Provider for a visual in Composer.
