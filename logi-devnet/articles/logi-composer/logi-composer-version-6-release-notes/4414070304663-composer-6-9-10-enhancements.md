# Composer 6.9.10 Enhancements

This topic describes the significant enhancements in Composer 6.9.10.

* [*Null/Zero Conversion for KPI Charts*](https://devnet.logianalytics.com/knowledge/articles/4414070304663#Null-Zero)

## Null/Zero Conversion for KPI Charts

A new toggle "Display Null as Zero" is now available for KPI Charts, enabling you to change the display from 'Null' to '0' in cases where there is no associated data for the KPI.

![display_null_as_zero.png](https://devnet.logianalytics.com/hc/article_attachments/4414062855831/display_null_as_zero.png)

This toggle is disabled by default; enable to use. After you've enabled it, a '0' value displays instead of 'Null' for the selected metric(s), including the comparison metric.

![noteicon.jpg](https://devnet.logianalytics.com/hc/article_attachments/4414070261015/noteicon.jpg) The "Volume" metric is never 'Null' and is not affected by this logic.
