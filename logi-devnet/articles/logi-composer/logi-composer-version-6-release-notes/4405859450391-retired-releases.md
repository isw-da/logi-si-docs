# Retired Releases

# Retired Releases

Zoomdata versions 2.6 and 3.7 have been retired.

![](https://devnet.logianalytics.com/hc/article_attachments/4406743739159/criticalicon.gif) Support for Zoomdata 4.9 will end on September 25, 2021. With the release of Composer 6.9 LTS, Zoomdata 4.9 entered Passive Support. Our LTS Passive Support release receives only critical patches and security updates until its end-of-life date. Security updates are not applicable to the third-party dependencies. We encourage you to migrate to our most current LTS release. The documentation for the LTS Passive Support release is updated only when necessary. If there are changes to existing functionality or workarounds that you need to be aware of, you can find the information you need in the Release Notes.

We encourage you to upgrade to the most recent Composer LTS version and to read the documentation at the following links:

* [*Logi Composer (Zoomdata) Release Vehicles*](https://devnet.logianalytics.com/hc/en-us/articles/4405851109143-Logi-Composer-Zoomdata-Release-Vehicles)
* [*Upgrade Composer*](https://devnet.logianalytics.com/hc/en-us/articles/4405859509271-Upgrade-Composer)

To understand the changes between each major release, we recommend that you read the Release Summaries provided for each release.

Contact your Logi Analytics Customer Account Manager for information about upgrading your Composer software.
