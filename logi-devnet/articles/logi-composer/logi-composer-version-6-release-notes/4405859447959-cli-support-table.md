# CLI Support Table

The following table lists the version of the CLI supported by different versions of Zoomdata and Composer.

| CLI Version | Supported Zoomdata/Composer Versions |
| --- | --- |
| 4 | Zoomdata 4 and Composer 5.9 and earlier |
| 5 | Composer 5.9.1 and later |

For information about migrating your existing custom charts to version 4.0 format, see [*Migrate Custom Charts*](https://devnet.logianalytics.com/hc/en-us/articles/4405859243927-Migrate-Custom-Charts). You can get additional information about the CLI at <https://www.npmjs.com/package/zoomdata-chart-cli> and in [*A Custom Chart Tutorial*](https://devnet.logianalytics.com/hc/en-us/articles/4405859250711-A-Custom-Chart-Tutorial).
