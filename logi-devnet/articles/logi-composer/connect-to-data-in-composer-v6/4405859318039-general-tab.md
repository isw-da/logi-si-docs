# General Tab

# General Tab

![](https://devnet.logianalytics.com/hc/article_attachments/4406743844247/general-tab_480x212.png) or ![](https://devnet.logianalytics.com/hc/article_attachments/4406743844503/general-tab-with-connector_480x211.png)

Use the General tab in the data source configuration wizard to specify a name and description for the data source configuration. The General tab may or may not have a connection selection box on it, depending on your user permissions.

![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) You must be logged in as an administrator or as a user with the [group privilege](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference)**Can Create new Data Sources** to see the General tab of the data source configuration wizard.

1. Specify a unique name in the **Source Name** box.
2. Optionally, specify a description in the **Description** box.
3. If you are logged in as an administrator or as a user with both the [group privileges](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference)**Can Manage Connections** and **Can Create new Data Sources**, select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743833879/next-btn.png) to select (or add) the data store connection for the data source on the [*Connection Tab*](https://devnet.logianalytics.com/hc/en-us/articles/4405851023767-Connection-Tab) in the configuration wizard.

   If you are logged in as a user with the [group privilege](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference)**Can Create new Data Sources**, but **not** with the privilege **Can Manage Connections**, select the data store connection for the data source in the **Select an existing connection box** on the General tab. (The first available connection definition for the selected connector type is selected by default.) Then select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743844887/next-btn_36x21.png) to proceed to the [*Tables/Indices Tab*](https://devnet.logianalytics.com/hc/en-us/articles/4405859323287-Tables-Indices-Tab) in the configuration wizard. The **Can Manage Connections** privilege is required to create or maintain data store connection definitions.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) A user without the **Can Manage Connection** privilege cannot add a data source configuration to Composer if data store connections have not previously been defined.
