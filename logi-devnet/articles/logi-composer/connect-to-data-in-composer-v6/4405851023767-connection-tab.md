# Connection Tab

# Connection Tab

![](https://devnet.logianalytics.com/hc/article_attachments/4406747431319/connection-tab_576x254.png)

Use the Connection tab in the data source configuration wizard to specify the connection credentials required to access the data store used for the data source. You can specify new connection credentials or use a previously validated data store connection. After a connection is validated, it is saved. See [*Manage Data Store Connections*](https://devnet.logianalytics.com/hc/en-us/articles/4405850909079-Manage-Data-Store-Connections) for information on managing saved validated connections.

If no connections for the selected connector type have been defined, the options **Input New Credentials** and **Use Validated Credentials** on the left of this tab are not shown and you are required to create a new connection.

![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) You must be logged in as an administrator or as a user with the [group privileges](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference)**Can Create new Data Sources** and **Can Manage Connections** to see the Connection tab of the data source configuration wizard.

![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) For flat file data connectors, the Connection tab is replaced with a Data tab and there is no Tables or Indices tab.

To specify a new connection definition on the Connection tab:

1. Select **Input New Credentials** on the left side of the Connection tab.
2. Specify a name for the connection in the **Connection Name** box.
3. The connection details required to connect to a data store vary by data store. Use the boxes on the tab to specify the URL and other connection details and, if applicable, authentication credentials (**User Name** and **Password** boxes) required to connect to the data store. Any connection requirements for a specific data store are described in the connector documentation for that data store. See the [*Composer Data Connector Reference*](https://devnet.logianalytics.com/hc/en-us/articles/4405859436055-Composer-Data-Connector-Reference).
4. If the **Do As User** box is available and you want to enable user delegation, optionally specify the custom user attribute set up for user delegation. See [*Apply User Delegation to a Connection*](https://devnet.logianalytics.com/hc/en-us/articles/4405850928919-Apply-User-Delegation-to-a-Connection).
5. Each data source configuration specifies refresh settings for the data from the data store. If a data store connection requires special credentials in order for Composer to refresh the data source data, select the ![](https://devnet.logianalytics.com/hc/article_attachments/4406747431959/add-override-btn_90x28.png) button under **Scheduler Overrides** and select an override setting to use. The override settings you can specify mirror the regular data store connection settings (except for the connection definition name) and vary based on the type of data store connector used for the connection. See [*Composer Data Connector Reference*](https://devnet.logianalytics.com/hc/en-us/articles/4405859436055-Composer-Data-Connector-Reference).

   More than one override setting can be specified. Simply select the ![](https://devnet.logianalytics.com/hc/article_attachments/4406743850775/add-override-btn_89x27.png) button again and select a different setting and provide its input value. Repeat this process until all override settings required by the data store have been specified.
6. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743851287/validate-btn_55x23.png) to validate the connection. If the connection is valid, you can continue the data source configuration process.
7. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743833879/next-btn.png) to proceed to the [*Tables/Indices Tab*](https://devnet.logianalytics.com/hc/en-us/articles/4405859323287-Tables-Indices-Tab) in the data source configuration wizard. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743834135/back-btn.png) to return to the [*General Tab*](https://devnet.logianalytics.com/hc/en-us/articles/4405859318039-General-Tab).

See also [Manage Data Store Connections](https://devnet.logianalytics.com/hc/en-us/articles/4405850909079-Manage-Data-Store-Connections).

To use a previously-validated connection definition:

1. Select **Use Validated Credentials** on the left side of the Connection tab.
2. From the drop-down list, select the data store connection definition you want to use.
3. Optionally, make changes to the data store connection definition if needed and select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743851287/validate-btn_55x23.png) to validate them. If you change the connection definition here, it affects all the data sources that use the connection definition.
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743833879/next-btn.png) to proceed to the [*Tables/Indices Tab*](https://devnet.logianalytics.com/hc/en-us/articles/4405859323287-Tables-Indices-Tab) in the data source configuration wizard. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743834135/back-btn.png) to return to the [*General Tab*](https://devnet.logianalytics.com/hc/en-us/articles/4405859318039-General-Tab).
