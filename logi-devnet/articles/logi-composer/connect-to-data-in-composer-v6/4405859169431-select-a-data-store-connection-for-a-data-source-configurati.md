# Select a Data Store Connection for a Data Source Configuration

Data store connections are assigned in data source configurations. If you can modify and save data sources, you can select or change the data store connections they use. However, where you make this selection varies, depending on your user permissions.

![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) You must be logged in as an administrator or as a user with the [group privilege](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference)**Can Manage Connections** to select or change the data store connection for a data source configuration.

* If you are logged in as an administrator or as a user with both the [group privileges](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference)**Can Manage Connections** and **Can Create new Data Sources**, you select the data store connection for a data source on the Connection tab of the data source configuration wizard. See [*Connection Tab*](https://devnet.logianalytics.com/hc/en-us/articles/4405851023767-Connection-Tab) and [*Manage Data Store Connections*](https://devnet.logianalytics.com/hc/en-us/articles/4405850909079-Manage-Data-Store-Connections).
* If you are logged in as a user with the [group privilege](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference)**Can Create new Data Sources**, but **not** with the privilege **Can Manage Connections**, you select the data store connection for a data source on the General tab of the data source configuration wizard. You cannot maintain data store connection definitions without the **Can Manage Connections** privilege. See [*General Tab*](https://devnet.logianalytics.com/hc/en-us/articles/4405859318039-General-Tab).
