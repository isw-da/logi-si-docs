# Manage Cloudera Connectors

Composer provides connectors to the
following Cloudera data stores:

* Cloudera Impala is a query engine that accesses data stored in clusters running Apache Hadoop.
* Cloudera Search enables searches of data stored in Hadoop and provides a simple full-text interface to conduct those searches. Cloudera Search supports Cloudera's open source Hadoop platform - Cloudera Distributed Hadoop (CDH). Composer also connects to CDH via the
  [HDFS connector](https://devnet.logianalytics.com/hc/en-us/articles/4402962915863-Manage-the-HDFS-Connector) .

Cloudera also supports a secure CDH cluster using Kerberos authentication so that access requests can be validated and confirmed. Composer can connect
[*Connect to a Kerberized CDH Cluster*](https://devnet.logianalytics.com/hc/en-us/articles/4402962907415-Connect-to-a-Kerberized-CDH-Cluster).
