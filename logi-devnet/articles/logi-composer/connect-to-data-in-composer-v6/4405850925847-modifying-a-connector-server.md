# Modifying a Connector Server

The only way to modify a connector server on the Manage Connector Services page is to delete and re-register the connector server. See [*Delete a Connector Server*](https://devnet.logianalytics.com/hc/en-us/articles/4405859188119-Delete-a-Connector-Server), [*Obtain Additional Connector Servers*](https://devnet.logianalytics.com/hc/en-us/articles/4405850923799-Obtain-Additional-Connector-Servers), and [*Register a New Connector Server*](https://devnet.logianalytics.com/hc/en-us/articles/4405850926999-Register-a-New-Connector-Server).
