# Schemas

# Schemas

When a Composer connector supports schemas, it supports namespace, schema, or catalog notation for organizing collections. When schemas are supported, the [*Tables/Indices Tab*](https://devnet.logianalytics.com/hc/en-us/articles/4405859323287-Tables-Indices-Tab) of the [data source configuration](https://devnet.logianalytics.com/hc/en-us/articles/4405851027479-Manage-Data-Source-Configurations) displays with the Schema drop-down and the Tables list and you can select a schema for the data source configuration. If schemas are not supported, then the list of collections displays on the [*Tables/Indices Tab*](https://devnet.logianalytics.com/hc/en-us/articles/4405859323287-Tables-Indices-Tab). Elasticsearch has a custom UI for displaying multiple indices.
