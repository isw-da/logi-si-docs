# Partitions

# Partitions

Some Composer connectors support partitions and pruning. This feature enables the Partition column on the Fields tab of the [data source configuration](https://devnet.logianalytics.com/hc/en-us/articles/4405851037719-Sources-Page), when the data source uses a supporting connector. Partitioning allows you to link a partitioned field to another field to help improve the performance of filtering operations for a data source.

Although many data stores support partitions in some form, this feature specifically tells Composer that the partitions may be used for manual pruning of result sets to increase speed.
