# Placing Existing Visuals on a Dashboard

When you create a new dashboard, you also create the first visual in the dashboard or place existing visuals from the [Visual Gallery](https://devnet.logianalytics.com/hc/en-us/articles/4402537943575-Using-the-Visual-Gallery) on the dashboard. A dashboard requires at least one visual. See [*Creating Dashboards*](https://devnet.logianalytics.com/hc/en-us/articles/4402537741463-Creating-Dashboards). After an initial visual is created, you can create or place additional visuals on your dashboard. For information on adding a new visual to a dashboard, see [*Adding New Visuals to a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402552936087-Adding-New-Visuals-to-a-Dashboard).

To place an existing visual on a dashboard from the Visual Gallery:

1. [Edit](https://devnet.logianalytics.com/hc/en-us/articles/4402537743383-Editing-a-Dashboard) the dashboard.
2. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406753440151/add.png) on the [dashboard icon bar](https://devnet.logianalytics.com/hc/en-us/articles/4402537752343-Using-the-Dashboard-Icon-Bar). A drop-down menu appears.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406753440663/add-vis-menu_192x97.png)
3. Select **Place Existing Visual** to add a new visual to the dashboard. The Select a Visual dialog appears.
4. Select a visual on the **Select a Visual** dialog. The visual is placed on the dashboard.
5. [Modify](https://devnet.logianalytics.com/hc/en-us/articles/4402537942295-Editing-Visuals) the visual as needed.
6. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406753438615/dash-save.png) on the [dashboard icon bar](https://devnet.logianalytics.com/hc/en-us/articles/4402537752343-Using-the-Dashboard-Icon-Bar) to save the dashboard.
