# Deleting a Dashboard

You can delete a dashboard from the dashboard library or from the dashboard itself.

![](https://devnet.logianalytics.com/hc/article_attachments/4406753437463/noteicon.jpg) The delete option is only available to the dashboard creator and the Composer administrator.

If you delete a dashboard that has been shared with another user, it is deleted for all users.

You cannot delete a linked dashboard. If any other visual is linked to the dashboard, you must first remove the link from the visual or delete the visual's source dashboard.

To delete a dashboard from the Composer Home page or the dashboard library:

1. Log into Composer as an administrator or a user who has been assigned to a group with the [**Can Administer Dashboards** privilege](https://devnet.logianalytics.com/hc/en-us/articles/4402537668503-Group-Privilege-Reference).
2. Select **Library** on the [top-level navigation banner](https://devnet.logianalytics.com/hc/en-us/articles/4402552863383-The-Top-Level-Navigation-Banner) or the [UI menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537848215-The-Composer-UI-Menu), or select the **Dashboards** box on the [Home page](https://devnet.logianalytics.com/hc/en-us/articles/4402552851095-Home-Page). The dashboard library displays.
3. Locate the dashboard you want to delete.
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406763906839/trashcan.png) in the Actions column
5. Confirm by selecting the
   **Delete** button on the warning dialog.

To delete a dashboard from the dashboard itself:

1. [Edit](https://devnet.logianalytics.com/hc/en-us/articles/4402537743383-Editing-a-Dashboard) the dashboard.
2. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406764035223/dash-delete_14x15.png) from the [dashboard icon bar](https://devnet.logianalytics.com/hc/en-us/articles/4402537752343-Using-the-Dashboard-Icon-Bar).
3. Confirm by selecting the **Delete** button.
