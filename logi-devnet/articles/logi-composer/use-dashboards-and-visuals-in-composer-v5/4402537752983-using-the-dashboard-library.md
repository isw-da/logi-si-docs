# Using the Dashboard Library

# Using the Dashboard Library

The working area of the dashboard library contains all dashboards in the Composer environment to which you have access. You can make a dashboard a favorite, delete it (if your [privileges](https://devnet.logianalytics.com/hc/en-us/articles/4402537668503-Group-Privilege-Reference) allow), and open the dashboard.

If your user definition has not been given access to dashboards, you will see no dashboards in the dashboard library and you will not be able to import any dashboards. You will be able to create dashboards, but you will not be able to save them. Contact your system administrator to increase your [privileges](https://devnet.logianalytics.com/hc/en-us/articles/4402537668503-Group-Privilege-Reference).

## Accessing the Dashboard Library

To access the dashboard library, select **Library** on the [UI menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537848215-The-Composer-UI-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4406763904151/hamburger.png)) or the [top-level navigation menu](https://devnet.logianalytics.com/hc/en-us/articles/4402552863383-The-Top-Level-Navigation-Banner), or select **Dashboards** on the [Home page](https://devnet.logianalytics.com/hc/en-us/articles/4402552851095-Home-Page). The dashboard library appears. The dashboards display in a table (or list) format.

![](https://devnet.logianalytics.com/hc/article_attachments/4406764018839/dashboard-list_576x109.png)

## Search Box

You can use the search box to filter the dashboards on the screen by dashboard name. For example, if you type a **C** in the search box, only the dashboards with names that start with the letter *C* are shown in the working area.

![](https://devnet.logianalytics.com/hc/article_attachments/4406764019095/search-box.png)

## Sorting the Dashboards

You can sort the dashboard list by dashboard name, the date the dashboard was last modified, the data source for the dashboard, or the dashboard owner name. By default, dashboards are sorted by the date the dashboard was last modified. Select a column title in the list to sort the dashboards in alphabetical order by the data in the column. To sort the list in reverse alphabetical order, select the column title a second time.

![](https://devnet.logianalytics.com/hc/article_attachments/4406753491863/dashboard-list_768x145.png)

The list view table columns are described below.

| Column | Description |
| --- | --- |
| Fav | This column identifies favorite dashboards using a star icon. If the star is colored (), the dashboard is a favorite. If the star is empty (), the dashboard is not a favorite. You can also click on the star in this dashboard to make the dashboard a favorite or remove it from favorites. |
| Name | The name of the dashboard. |
| Data Source | The name of the data sources used by the dashboard. |
| Author | The user who created the dashboard. |
| Date Modified | The date the dashboard was last modified. |
| Permissions | Select the  icon in this column to assign permissions to a dashboard. The Dashboard Permissions dialog appears. See [*About Dashboard Authorization*](https://devnet.logianalytics.com/hc/en-us/articles/4402537739927-About-Dashboard-Authorization). |
| Schedule | Select the clock () in this column to create a scheduled dashboard report for the dashboard. The Scheduled Reports dialog appears. See [*About Scheduled Dashboard Reports*](https://devnet.logianalytics.com/hc/en-us/articles/4402552776855-About-Scheduled-Dashboard-Reports). |
| Actions | Shows icons you can select to perform actions for the dashboard.   * Select the trash can () in this column to delete a dashboard. See [Deleting a Dashboard](https://devnet.logianalytics.com/hc/en-us/articles/4402537742743-Deleting-a-Dashboard). * Select the  icon to generate an embeddable code snippet for the dashboard. The Embed Code dialog appears. See [*Embed Dashboards*](https://devnet.logianalytics.com/hc/en-us/articles/4402552772375-Embed-Dashboards). |

## Top-Level Navigation Banner

The top-level navigation banner identifies the user that is logged in and provides access to other areas of the UI. The available menu options vary depending on the group [privileges](https://devnet.logianalytics.com/hc/en-us/articles/4402537668503-Group-Privilege-Reference) assigned to your user definition. See [*The Top-Level Navigation Banner*](https://devnet.logianalytics.com/hc/en-us/articles/4402552863383-The-Top-Level-Navigation-Banner)

## Side Menu

The Side Menu bar allows easy access to saved visuals, as well as other visuals created by other users in your Composer environment. From the Side Menu, you can also create a new dashboard, filter the dashboards that are shown, or import a dashboard.

![](https://devnet.logianalytics.com/hc/article_attachments/4406764021015/side-menu_144x214.png)

The following options are available:

1. **View All** - This option removes any filters for the dashboard library and displays all dashboards within your Composer environment.
2. **Favorites** - Displays only the dashboards that you have marked as favorites.
3. **My Items** - Displays only dashboards that you created and saved.
4. **Shared With Me** - Displays only the dashboards that other users shared with you.
   See [*Share a Dashboard with Users Inside Your Account*](https://devnet.logianalytics.com/hc/en-us/articles/4402552777623-Share-a-Dashboard-with-Users-Inside-Your-Account).
5. **Import Dashboard** -- Allows you to [import a dashboard](https://devnet.logianalytics.com/hc/en-us/articles/4402537746583-Importing-a-Dashboard).

You create new dashboards directly from the Home page by selecting ![](https://devnet.logianalytics.com/hc/article_attachments/4406764021271/new-btn.png).
