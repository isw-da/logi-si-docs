# Using the Visual Gallery

The Visual Gallery can be used to manage the visuals defined in a Composer instance.

![](https://devnet.logianalytics.com/hc/article_attachments/4406753437463/noteicon.jpg) To see the Visual Gallery, you must be logged in as an administrator or as a user in a group with the **Can Manage Visuals** privilege.

![](https://devnet.logianalytics.com/hc/article_attachments/4406763903511/visual-gallery_576x240.png)

The Visual Gallery page includes the following features:

1. The ![](https://devnet.logianalytics.com/hc/article_attachments/4406763903767/add-visual-btn.png) button allows you to add a new visual to the Visual Gallery. See [*Adding Visuals to the Visual Gallery*](https://devnet.logianalytics.com/hc/en-us/articles/4402537939607-Adding-Visuals-to-the-Visual-Gallery).
2. The table lists and allows you to maintain your previously defined visuals. You can add edit and delete visuals in this table.
3. A search bar at the top of the page that you can use to search for a visual in the table.

To access the Visual Gallery:

1. Log into Composer as an administrator or as a user with the **Can Manage Visuals**[privilege](https://devnet.logianalytics.com/hc/en-us/articles/4402537668503-Group-Privilege-Reference).
2. Select **Visual Gallery** on the [UI menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537848215-The-Composer-UI-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4406763904151/hamburger.png)) or the [top-level navigation menu](https://devnet.logianalytics.com/hc/en-us/articles/4402552863383-The-Top-Level-Navigation-Banner), or select the **Visuals** box on the [Home page](https://devnet.logianalytics.com/hc/en-us/articles/4402552851095-Home-Page). The Visuals page appears.

You can edit and delete visuals listed in the table on the Visuals page. If there are many visuals listed, you may need to search for the visual you need. Use the search bar to search for a visual in the list. See [Searching for a Definition in a List](https://devnet.logianalytics.com/hc/en-us/articles/4402552738967-Searching-for-a-Definition-in-a-List).

Each column in the table is described below. The Type, Name, Author, and Modified Date columns are sortable.

| Column | Description |
| --- | --- |
| Type | An icon identifying the visual type (style) of the visual in the visual. See [*Visual Metrics and Attributes Reference*](https://devnet.logianalytics.com/hc/en-us/articles/4402537848855-Visual-Metrics-and-Attributes-Reference) to see all the possibilities. |
| Name | The name assigned the visual. The visual name is not necessarily the same as the visual name. See [*Understanding Visual Names and Titles*](https://devnet.logianalytics.com/hc/en-us/articles/4402537945495-Understanding-Visual-Names-and-Titles) for more information about how visual names are set. |
| Author | The name of the user who defined the visual. |
| Usage | The number of times the visual is used in dashboards. If this count is not zero, you cannot delete a visual from the Visual Gallery. |
| Modified Date | The timestamp when the visual definition was last modified. |
| Actions | The  button in this column allows you to delete a visual. Before you delete a visual, you must delete it from all the dashboards that use it. If the Usage column for a visual is not zero, the  button is disabled. See [*Deleting and Removing Visuals*](https://devnet.logianalytics.com/hc/en-us/articles/4402537941015-Deleting-and-Removing-Visuals). |
