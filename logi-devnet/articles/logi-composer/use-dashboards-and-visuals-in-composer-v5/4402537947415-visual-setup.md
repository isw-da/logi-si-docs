# Visual Setup

When you configure a data source, you can adjust the default settings for visuals that use the data source on the data source configuration [Visuals tab](https://devnet.logianalytics.com/hc/en-us/articles/4402537788695-Visuals-Tab). The default settings vary by visual style. For information about the default settings for a visual style, select a visual style in the [visual reference](https://devnet.logianalytics.com/hc/en-us/articles/4402537848855-Visual-Metrics-and-Attributes-Reference).

Settings that are common for all visual styles (time bar and search settings) can be specified in Global Default Settings on the [Visuals tab](https://devnet.logianalytics.com/hc/en-us/articles/4402537788695-Visuals-Tab) of a data source configuration. See [*Configuring Time Bar Defaults*](https://devnet.logianalytics.com/hc/en-us/articles/4402537788055-Configuring-Time-Bar-Defaults).

To add a visual using the default visual settings defined in a [data source configuration](https://devnet.logianalytics.com/hc/en-us/articles/4402552802199-Manage-Data-Source-Configurations), see:

* [*Creating Dashboards*](https://devnet.logianalytics.com/hc/en-us/articles/4402537741463-Creating-Dashboards)
* [*Adding Visuals to the Visual Gallery*](https://devnet.logianalytics.com/hc/en-us/articles/4402537939607-Adding-Visuals-to-the-Visual-Gallery)
* [*Adding New Visuals to a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402552936087-Adding-New-Visuals-to-a-Dashboard)
* [*Placing Existing Visuals on a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402537946135-Placing-Existing-Visuals-on-a-Dashboard)

After the visual has been created, you can alter its settings. See [*The Visual Interface*](https://devnet.logianalytics.com/hc/en-us/articles/4402537944855-The-Visual-Interface).
