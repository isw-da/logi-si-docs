# Deleting and Removing Visuals

You can remove visuals from dashboards and delete them from the Composer instance in the Visual Gallery. You cannot delete a visual if it is used by any dashboard.

## Removing a Visual From a Dashboard

To remove a visual from your dashboard:

1. Edit the dashboard. See [*Editing a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402537743383-Editing-a-Dashboard).
2. Select the visual to be removed.
3. Select **Remove Visual** on the [visual drop-down (![](https://devnet.logianalytics.com/hc/article_attachments/4406763902743/three-dots-menu_19x11.png)) menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537941655-Using-the-Visual-Menu). A pop-up dialog verifying the deletion.

   To control whether a visual can be removed, use the interactivity sidebar. See [*Controlling How Users Interact With a Visual*](https://devnet.logianalytics.com/hc/en-us/articles/4402537944215-Controlling-How-Users-Interact-With-a-Visual).
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406763906583/delete-btn_44x23.png) on the warning dialog to confirm the removal.

## Deleting a Visual from the Visual Gallery

A visual can only be deleted from the [Visual Gallery](https://devnet.logianalytics.com/hc/en-us/articles/4402537943575-Using-the-Visual-Gallery) when it is not used in any dashboard. If the **Usage** column for a visual in the Visual Gallery shows any number except zero, the option to delete the visual is disabled.

To delete a visual from the Visual Gallery:

1. Access the [Visual Gallery](https://devnet.logianalytics.com/hc/en-us/articles/4402537943575-Using-the-Visual-Gallery). See [*Using the Visual Gallery*](https://devnet.logianalytics.com/hc/en-us/articles/4402537943575-Using-the-Visual-Gallery).
2. Locate the visual in the Visual Gallery.
3. Verify that the visual is removed from all dashboards. The **Usage** column value for the visual in the [Visual Gallery](https://devnet.logianalytics.com/hc/en-us/articles/4402537943575-Using-the-Visual-Gallery) should be zero.
4. Select the delete (trashcan) icon (![](https://devnet.logianalytics.com/hc/article_attachments/4406763906839/trashcan.png)) in the **Actions** column associated with the visual. A pop-up dialog verifying the deletion.
5. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406763907095/delete-btn_48x25.png) on the warning dialog to confirm the deletion.
