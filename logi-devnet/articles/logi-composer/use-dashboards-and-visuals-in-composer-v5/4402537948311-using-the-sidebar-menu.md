# Using the Sidebar Menu

Composer's visual canvas provides a comprehensive suite of tools to help you tailor visuals quickly so they provide the information you need in the best way for you to analyze your data. These tools are available from the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537941655-Using-the-Visual-Menu) and from the visual sidebar menu. This topic discusses the sidebar menu.

The sidebar menu appears to the right of every visual edited from the Visual Gallery. It is also available when you select **Edit Visual** on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537941655-Using-the-Visual-Menu) when editing a visual in a dashboard.

![](https://devnet.logianalytics.com/hc/article_attachments/4406763899415/visual-sidebar-menu.png)

Selecting options on the sidebar menu opens and closes sidebars that allow you to control:

* The style of a selected dashboard visual
* The settings used for a selected visual
* The configuration of pivot tables and raw data tables
* The filters used for a selected visual
* Time bar settings for a visual
* Sort and limit options for a selected visual
* Ruler and reference line settings for a selected visual
* Colors for a selected visual
* Interactivity settings for a selected visual
* General information for a selected visual

When you are editing a visual on a dashboard, you must first select a visual (a blue border appears around the selected visual) and then select **Edit Visual** on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537941655-Using-the-Visual-Menu).

If an option cannot be selected, the settings controlled by that sidebar are not available for the visual style you have selected.

The following sidebar menu options are available:

|  | Opens or closes the Filter sidebar. Use this sidebar to filter the data for the selected visual. See [*Applying Row-Level Filters*](https://devnet.logianalytics.com/hc/en-us/articles/4402537794711-Applying-Row-Level-Filters), [*Applying Wild Card Filters to a Visual or Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402537802519-Applying-Wild-Card-Filters-to-a-Visual-or-Dashboard), and [*Applying Group Filters*](https://devnet.logianalytics.com/hc/en-us/articles/4402537800599-Applying-Group-Filters). |
|  | Opens or closes the Time Bar sidebar. Use this sidebar to control the time bar for the selected visual. See [*Using the Time Bar Sidebar*](https://devnet.logianalytics.com/hc/en-us/articles/4402537902231-Using-the-Time-Bar-Sidebar). |
|  | Opens or closes an appropriate Visual Settings sidebar. Use this sidebar to change settings for the selected visual. Different settings can be altered for different visual styles. In addition, some visual styles do not have any settings you can control in this way; this icon is disabled for these visual styles. See [*Changing Visual Settings*](https://devnet.logianalytics.com/hc/en-us/articles/4402537946775-Changing-Visual-Settings). |
|  | Opens or closes the Ruler sidebar. Use this sidebar to control the rulers on the selected visual. See . |
|  | Opens or closes the Color sidebar. Use this sidebar to change the color settings for the selected visual. Different color settings are available for different chart styles. See [Changing Color Schemes](https://devnet.logianalytics.com/hc/en-us/articles/4402537921943-Changing-Color-Schemes). |
|  | Opens or closes the visual interactivity sidebar. Use this sidebar to control how users can interact with the selected visual. See [*Controlling How Users Interact With a Visual*](https://devnet.logianalytics.com/hc/en-us/articles/4402537944215-Controlling-How-Users-Interact-With-a-Visual). |
|  | Opens or closes the visual information sidebar. Use this sidebar to change basic information for the selected visual. See [*Modifying the Visual Title and Description*](https://devnet.logianalytics.com/hc/en-us/articles/4402552922263-Modifying-the-Visual-Title-and-Description). |
