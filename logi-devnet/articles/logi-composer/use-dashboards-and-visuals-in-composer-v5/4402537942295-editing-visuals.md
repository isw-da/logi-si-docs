# Editing Visuals

# Editing Visuals

You can edit visuals directly on a Composer dashboard or in the Visual Gallery. When you modify a visual from the Visual Gallery and save your changes, the changes are made to the visual in the Visual Gallery and everywhere the visual is used on dashboards.

To edit a visual:

1. Select the visual on a dashboard or in the Visual Gallery. When selected on a dashboard, a blue border appears around the visual. If you select a visual with a streaming data source, the [time bar](https://devnet.logianalytics.com/hc/en-us/articles/4402537901591-Use-the-Time-Bar) (Data DVR) appears.
2. Depending on the edits you need to make, select an item on the visual drop-down menu (![](https://devnet.logianalytics.com/hc/article_attachments/4406763905943/three-dots-menu.png)) or one of the [sidebars](https://devnet.logianalytics.com/hc/en-us/articles/4402537938967-Using-the-Visual-Style-Sidebar) for the visual. You can also use the [axes labels](https://devnet.logianalytics.com/hc/en-us/articles/4402537918615-Changing-the-Axes) on the visual to change what is displayed and the [radial menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537934359-Using-the-Radial-Menu) to visualize the data on your visual, adjust the settings, and more.

   | Edit Type | Procedure |
   | --- | --- |
   | Change the data shown | Depending on the style of the visual, select new data fields in the axes labels or select **Configure** on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537941655-Using-the-Visual-Menu). See [*Changing the Axes*](https://devnet.logianalytics.com/hc/en-us/articles/4402537918615-Changing-the-Axes) |
   | Modify the visual style | Select on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537948311-Using-the-Sidebar-Menu). A corresponding sidebar appears. See [*Changing the Visual Style*](https://devnet.logianalytics.com/hc/en-us/articles/4402537920663-Changing-the-Visual-Style). |
   | Modify the colors | Select on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537948311-Using-the-Sidebar-Menu). A corresponding sidebar appears. See [*Changing Color Schemes*](https://devnet.logianalytics.com/hc/en-us/articles/4402537921943-Changing-Color-Schemes). |
   | Modify the rulers and reference lines | Select on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537948311-Using-the-Sidebar-Menu). A corresponding sidebar appears. See [*Using Rulers and Reference Lines*](https://devnet.logianalytics.com/hc/en-us/articles/4402552932375-Using-Rulers-and-Reference-Lines). |
   | Modify visual information | Select on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537948311-Using-the-Sidebar-Menu). A corresponding sidebar appears. See [Modifying the Visual Title and Description](https://devnet.logianalytics.com/hc/en-us/articles/4402552922263-Modifying-the-Visual-Title-and-Description). |
   | Manage the visual time bar | Select  and **Time Bar** from the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537941655-Using-the-Visual-Menu). See [*Use the Time Bar*](https://devnet.logianalytics.com/hc/en-us/articles/4402537901591-Use-the-Time-Bar). |
   | Sort or otherwise limit the visual | Select on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537948311-Using-the-Sidebar-Menu). See [*Sorting and Limiting Visual Data*](https://devnet.logianalytics.com/hc/en-us/articles/4402537936919-Sorting-and-Limiting-Visual-Data) |
   | Apply new or saved filters to your visual. | Select  to the left of the visual name to display the Filters sidebar. Use the Filters sidebar to add and apply a new filter to your data or to select a saved filter to apply to your data. See [*Filter Data*](https://devnet.logianalytics.com/hc/en-us/articles/4402552815383-Filter-Data). |
   | Export the visual | Select  and **Export** from the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537941655-Using-the-Visual-Menu). See [*Exporting Visuals*](https://devnet.logianalytics.com/hc/en-us/articles/4402537942935-Exporting-Visuals). |
   | Copy the visual | Select  and **Copy Visual** from the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537941655-Using-the-Visual-Menu). See [*Copying Visuals*](https://devnet.logianalytics.com/hc/en-us/articles/4402537940247-Copying-Visuals). |
   | Save the visual | Select  and **Save** from the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537941655-Using-the-Visual-Menu). Changes are saved in the dashboard and in the Visual Gallery. |
   | Remove a visual from the dashboard. | Select  and **Remove Visual** from the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537941655-Using-the-Visual-Menu). See [Deleting and Removing Visuals](https://devnet.logianalytics.com/hc/en-us/articles/4402537941015-Deleting-and-Removing-Visuals). |
3. Make your changes.
4. When you have finished making changes to your visual, be sure to save it.

   * If you are editing the visual from a dashboard, click **Save** on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537941655-Using-the-Visual-Menu) or [save the dashboard](https://devnet.logianalytics.com/hc/en-us/articles/4402537747863-Saving-a-Dashboard).
   * If you are editing the visual from the Visual Gallery, click ![](https://devnet.logianalytics.com/hc/article_attachments/4406763906199/save-blue-btn.png).
