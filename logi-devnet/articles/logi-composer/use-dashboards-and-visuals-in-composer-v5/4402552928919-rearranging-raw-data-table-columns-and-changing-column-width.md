# Rearranging Raw Data Table Columns and Changing Column Widths

You can rearrange the columns in a raw data table. After you save the dashboard or visual, the column rearrangement is retained when close the dashboard. It is also retained when you share or export the dashboard or visual.

By default, the columns of a raw data table are 100 pixels wide. You cannot change this default. However, you can enlarge or decrease the size of the table columns while the table is open for viewing. After you save the dashboard, your customized column widths are saved when you close the table and when you share or share it.

To adjust the table columns in a raw data table:

1. Edit the raw data table you want to modify. See [*Editing Visuals*](https://devnet.logianalytics.com/hc/en-us/articles/4402537942295-Editing-Visuals).
2. If you are editing the visual in a dashboard, select **Edit Visual** from the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537941655-Using-the-Visual-Menu). The [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537948311-Using-the-Sidebar-Menu) for the visual appears.

   If you are editing the visual from the Visual Gallery, the sidebar appears to the right of the visual.
3. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406753439511/sidebar-chtsettings.png) on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537948311-Using-the-Sidebar-Menu). The Raw Data Table Settings sidebar for the visual appears.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406763912855/rdt-settings_192x376.png)
4. To move a column, select its name and then drag it up or down in the list, as appropriate. The first column in the list will appear on the left side of the raw data table.
5. Select **Apply** to apply your changes to the raw data table.
6. To change the widths of the columns in the table, drag the separator between two columns in the appropriate direction.
7. [Save](https://devnet.logianalytics.com/hc/en-us/articles/4402537747863-Saving-a-Dashboard) the dashboard.
