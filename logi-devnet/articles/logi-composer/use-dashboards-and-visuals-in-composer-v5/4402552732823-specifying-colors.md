# Specifying Colors

# Specifying Colors

Colors can be specified in Composer using:

* The color name (for example "cornflowerblue" or "red"). For valid color names, see <https://www.w3schools.com/cssref/css_colors.asp>.
* RGB color encoding (for example, `rgb(255,0,0)`). For RGB color values, see <https://www.rapidtables.com/web/color/RGB_Color.html>.
* Hexadecimal color encoding (for example, `#0096b6`). For hexadecimal color values, see <https://www.rapidtables.com/web/color/RGB_Color.html>.
