# Using the Dashboard Icon Bar

# Using the Dashboard Icon Bar

Each dashboard provides a series of selectable dashboard icons on an icon bar in the upper right corner of the dashboard.

![](https://devnet.logianalytics.com/hc/article_attachments/4406764021655/dashboard-icons.png)

Select an icon to perform a dashboard function, as described in the following table.

| Icon | Description |
| --- | --- |
|  | Provides options that allow you to:   * Link fields between disparate data sources to create cross-source links. See [*Using Cross-Source Links*](https://devnet.logianalytics.com/hc/en-us/articles/4402537728023-Using-Cross-Source-Links). * Link dashboard visuals to other dashboards. See [*Linking a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402537754263-Linking-a-Dashboard). |
|  | Exports a dashboard. The Export dialog appears. See [*Exporting a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402537745943-Exporting-a-Dashboard). |
|  | Refreshes the data on the dashboard. See [*Refreshing Data on a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402537742103-Refreshing-Data-on-a-Dashboard). |
|  | Marks the dashboard as a favorite. |
|  | Deletes a dashboard. See [*Deleting a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402537742743-Deleting-a-Dashboard). |
|  | Provides options that allow you to create a new visual on the dashboard or place an existing visual on the dashboard. See [*Adding New Visuals to a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402552936087-Adding-New-Visuals-to-a-Dashboard) and [*Placing Existing Visuals on a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402537946135-Placing-Existing-Visuals-on-a-Dashboard). |
|  | Provides options that allow you to save a dashboard or save a dashboard with a new name (which copies it). See [*Saving a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402537747863-Saving-a-Dashboard) and [*Copying a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402537740823-Copying-a-Dashboard). |
