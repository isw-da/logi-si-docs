# Adjusting the Time Bar Range Using the Time Range Dialog

To adjust the time bar time range for a visual using the time range dialog:

1. Select the visual.
2. Select the range setting icon (![](https://devnet.logianalytics.com/hc/article_attachments/4406753454103/time-bar-range_29x24.png)) on the time bar. The Time Range dialog appears.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406753454359/time-range-dialog_244x213.png)
3. Use the **From** and **To** boxes to specify the default time bar range.

   You can set the range in static time or dynamic time, or use preset ranges provided with Composer.

   * Select **Static Time** or **Dynamic Time** in the **fx** drop-down menu.

     ![](https://devnet.logianalytics.com/hc/article_attachments/4406753454615/time-bar-range-types_288x167.png)

     If you select **Static Time**, the **From** and **To** boxes are filled with default dates and times. Use the boxes to select specific from and to times:

     ![](https://devnet.logianalytics.com/hc/article_attachments/4406763935383/time-bar-static-range_143x201.png)

     If you select **Dynamic Time**, the **From** and **To** boxes are filled with **Start of data** and **End of data** automatically. Use the boxes to select different dynamic from and to times:

     ![](https://devnet.logianalytics.com/hc/article_attachments/4406753454871/time-bar-dynamic-range_218x136.png)
   * Alternatively, select ![](https://devnet.logianalytics.com/hc/article_attachments/4406763935895/presets-btn_57x20.png) to fill the **From** and **To** boxes with predefined time ranges provided by Composer:

     ![](https://devnet.logianalytics.com/hc/article_attachments/4406763936151/time-bar-presets_125x217.png)

     Use the filter box at the top of the presets list to locate the preset setting you want. Descriptions of each of the preset options are provided in [*Preset Time Ranges*](https://devnet.logianalytics.com/hc/en-us/articles/4402552860055-Preset-Time-Ranges).
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406753455127/apply1-btn_41x23.png) to apply the new time range setting. It is applied to the visual. If the visual is using a unified time bar, the new time range setting is applied to all of the visuals using the unified time bar. See [*Working with Unified Time Bars*](https://devnet.logianalytics.com/hc/en-us/articles/4402552909207-Working-with-Unified-Time-Bars).
5. Save the dashboard.
