# Scatter (Bubble) Charts

Scatter charts include floating bubbles, packed bubbles, and scatter plot styles. You can configure the defaults for all scatter chart styles in a data source configuration. For more information, select one of the following scatter chart styles:

* [*Floating Bubble Charts*](https://devnet.logianalytics.com/hc/en-us/articles/4402537923223-Floating-Bubble-Charts)
* [*Packed Bubble Charts*](https://devnet.logianalytics.com/hc/en-us/articles/4402537929239-Packed-Bubble-Charts)
* [*Scatter Plots*](https://devnet.logianalytics.com/hc/en-us/articles/4402537936279-Scatter-Plots)
