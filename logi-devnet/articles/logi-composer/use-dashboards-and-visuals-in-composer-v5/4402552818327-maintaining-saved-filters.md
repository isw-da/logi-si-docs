# Maintaining Saved Filters

After a filter is created and applied to a visual, it can be saved and applied to other visuals that use the same data source. A saved filter can include the settings of one or more filters.

The following topics describe how to maintain a saved filter.

* [*Creating a Saved Filter*](https://devnet.logianalytics.com/hc/en-us/articles/4402537801239-Creating-a-Saved-Filter)
* [*Editing a Saved Filter*](https://devnet.logianalytics.com/hc/en-us/articles/4402537801879-Editing-a-Saved-Filter)
* [*Deleting a Saved Filter*](https://devnet.logianalytics.com/hc/en-us/articles/4402552817431-Deleting-a-Saved-Filter)
