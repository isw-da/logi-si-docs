# Adding Visuals to the Visual Gallery

When you create a [new visual on a dashboard](https://devnet.logianalytics.com/hc/en-us/articles/4402552936087-Adding-New-Visuals-to-a-Dashboard), the new visual is automatically added to the Visual Gallery. You can also create new visuals independently of dashboards by adding them to the Visual Gallery. The new visual in the Visual Gallery can later be placed on dashboards.

To add a visual to the Visual Gallery:

1. Log into Composer as an administrator or a user who has been assigned to a group with the [**Can Manage Visuals** privilege](https://devnet.logianalytics.com/hc/en-us/articles/4402537668503-Group-Privilege-Reference).
2. Select **Visual Gallery** on the [UI menu](https://devnet.logianalytics.com/hc/en-us/articles/4402537848215-The-Composer-UI-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4406763904151/hamburger.png)) or the [top-level navigation menu](https://devnet.logianalytics.com/hc/en-us/articles/4402552863383-The-Top-Level-Navigation-Banner), or select the **Visuals** box on the [Home page](https://devnet.logianalytics.com/hc/en-us/articles/4402552851095-Home-Page). The Visuals page appears.
3. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406763903767/add-visual-btn.png). A dialog prompts you to select a data source for the new visual.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406763907351/vg-visualds_288x291.png)
4. Select a data source from the dialog list. The dialog then prompts you to select a visual style.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406763907607/vg-visualstyle_288x289.png)
5. Select a visual style from the dialog list. For example, select **Bars**. The visual is created.
6. Optionally, click on the visual name and change it. See [*Understanding Visual Names and Titles*](https://devnet.logianalytics.com/hc/en-us/articles/4402537945495-Understanding-Visual-Names-and-Titles).

   The minimum length of visual names and titles is one character; the maximum length is 255 characters. They can start with and contain numbers, special characters, and uppercase and lowercase characters. They can contain spaces, but cannot start with a space. They cannot be empty or contain only spaces and they cannot have leading or trailing spaces.
7. Optionally, click on the visual title and change it. See [*Understanding Visual Names and Titles*](https://devnet.logianalytics.com/hc/en-us/articles/4402537945495-Understanding-Visual-Names-and-Titles).

   The minimum length of visual names and titles is one character; the maximum length is 255 characters. They can start with and contain numbers, special characters, and uppercase and lowercase characters. They can contain spaces, but cannot start with a space. They cannot be empty or contain only spaces and they cannot have leading or trailing spaces.
8. Make any other changes to the visual that you need.
9. Click ![](https://devnet.logianalytics.com/hc/article_attachments/4406763906199/save-blue-btn.png) to save the visual.
