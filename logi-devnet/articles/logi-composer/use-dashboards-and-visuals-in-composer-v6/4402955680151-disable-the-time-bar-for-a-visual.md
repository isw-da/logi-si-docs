# Disable the Time Bar for a Visual

To disable the time bar for a visual:

1. Select the visual in the dashboard or in the Visual Gallery.
2. If you selected the visual on a dashboard, select **Edit Visual** on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955730967-Use-the-Visual-Menu) to access the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955735319-Use-the-Visual-Sidebar-Menu) for the visual.
3. Select the visual time bar icon (![](https://devnet.logianalytics.com/hc/article_attachments/4404959475223/dashboard-schedule.png)) on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955735319-Use-the-Visual-Sidebar-Menu) for the visual. The [Time Bar sidebar](https://devnet.logianalytics.com/hc/en-us/articles/4402955726359-Use-the-Visual-Style-Sidebar) opens.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404959501079/sidebar-timebar_288x496.png)
4. Select the **None (Time Bar is Off)** option. This option is not available for the
   **Line** charts.
5. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404952811159/apply-btn_122x22.png).
6. [Save](https://devnet.logianalytics.com/hc/en-us/articles/4402962944151-Save-a-Dashboard) the dashboard.

The time bar is disabled for the visual.
