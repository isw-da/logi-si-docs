# Change Aggregation Using the Table Context Menu

To change the aggregation metrics in a table using the table context menu:

1. [Edit](https://devnet.logianalytics.com/hc/en-us/articles/4402955510551-Edit-a-Dashboard) a dashboard with a table that uses a data source containing metric fields.
2. Group the table. See [*Group and Ungroup Table Data*](https://devnet.logianalytics.com/hc/en-us/articles/4402955718551-Group-and-Ungroup-Table-Data).
3. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404952786071/three-dots-menu.png) next to the column heading name of any metric field in the table to access the table context menu.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404959494935/rdt-contextmenu-num.png)
4. In any of the metric fields in the **Columns** section of the sidebar, select the aggregation method you want. Possible aggregation methods include **None**, **Avg**, **Min**, **Max**, **Sum**, **Last Value**, and **Distinct Count** (for attributes with [distinct counts enabled](https://devnet.logianalytics.com/hc/en-us/articles/4402955539351-Enable-Distinct-Counts)). For information about aggregation methods, see [*Metric Aggregation Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4402955545751-Metric-Aggregation-Functions).

   The selected aggregation method is applied to the metric.
5. Save the dashboard and visual.
