# Zoom Into a Time Bar Range

To zoom into a time bar range:

1. Select a visual on the dashboard.
2. Select a time range for the time bar. See [*Adjust the Time Bar Range*](https://devnet.logianalytics.com/hc/en-us/articles/4405851195543-Adjust-the-Time-Bar-Range).
3. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743741847/time-bar-zoom.png) on the right side of the time bar to zoom into the selected range. Only the times for the time range you selected are shown on the time bar.

   To view all the available times in the data, select ![](https://devnet.logianalytics.com/hc/article_attachments/4406747396759/time-bar-range-avail_25x25.png) on the time bar. To select the maximum range of values, select ![](https://devnet.logianalytics.com/hc/article_attachments/4406747397015/time-bar-select-max_21x23.png) on the time bar.
4. Save the dashboard.
