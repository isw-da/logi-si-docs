# Sort Data in a Pivot Table

The data in the table can be sorted . After you save the dashboard or visual, the sort settings are retained when you close it. The sort settings are also retained when you export and share the table.

To sort the rows in a table by the data in a single column:

* To sort the data in ascending order, select the up arrow in the heading.
* To sort the data in descending order, select the down arrow in the heading.

Remember to [save](https://devnet.logianalytics.com/hc/en-us/articles/4405850986647-Save-a-Dashboard) the dashboard if you want to retain the sort order.
