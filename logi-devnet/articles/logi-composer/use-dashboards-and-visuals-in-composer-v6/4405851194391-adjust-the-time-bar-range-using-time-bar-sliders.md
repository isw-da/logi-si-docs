# Adjust the Time Bar Range Using Time Bar Sliders

To adjust the time bar time range for a visual using the time bar sliders:

1. Select the visual.
2. Slide the From slider (![](https://devnet.logianalytics.com/hc/article_attachments/4406743742999/time-bar-from-slider_24x27.png)) on the time bar to the earliest time for the time range you want.
3. Slide the To slider (![](https://devnet.logianalytics.com/hc/article_attachments/4406743743255/time-bar-to-slider_22x18.png)) on the time bar to the latest time for the time range you want.
4. Save the dashboard.

To select the maximum range of values, select ![](https://devnet.logianalytics.com/hc/article_attachments/4406747397015/time-bar-select-max_21x23.png) on the time bar.
