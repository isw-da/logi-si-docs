# Change the Visual Color Metric

You can change the metric used to determine the colors used on a visual while you are viewing it. (If you want to alter the defaults used by a visual, make your changes on the [Visuals tab](https://devnet.logianalytics.com/hc/en-us/articles/4405859325591-Visuals-Tab) of your data source configurations.

To change the color metric while you are viewing a visual:

1. Select the color metric directly on your visual. A Color dialog appears.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743737367/color-metric_150x159.png)
2. Select a new metric to be used to determine the colors on your visual.
3. Select the aggregation function that should be used with the color metric: SUM, AVG, MAX, MIN, or (for some data sources) LAST VALUE. See [*Metric Aggregation Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405851021975-Metric-Aggregation-Functions).

   The visual renders using colors determined by the select metric and its aggregated values.
