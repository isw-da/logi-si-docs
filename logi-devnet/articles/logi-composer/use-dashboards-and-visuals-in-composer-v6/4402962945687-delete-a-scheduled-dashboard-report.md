# Delete a Scheduled Dashboard Report

To delete a scheduled dashboard report:

1. Log into Composer as an administrator or a user with the **Can Create Scheduled Reports** [privilege](https://devnet.logianalytics.com/hc/en-us/articles/4402955416087-Group-Privilege-Reference). The [Home page](https://devnet.logianalytics.com/hc/en-us/articles/4402963020951-Home-Page) appears.
2. Select **Library** on the [top-level navigation banner](https://devnet.logianalytics.com/hc/en-us/articles/4402963039639-The-Top-Level-Navigation-Banner) or select **Dashboard** on the [Home page](https://devnet.logianalytics.com/hc/en-us/articles/4402963020951-Home-Page). The [dashboard library](https://devnet.logianalytics.com/hc/en-us/articles/4402955515671-Use-the-Dashboard-Library) appears.
3. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404952894999/list-view.png) to view the dashboard library in list view.
4. Locate the dashboard in the dashboard library list containing the scheduled you want to remove.
5. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404959475223/dashboard-schedule.png) in the associated **Schedule** column. The Scheduled Reports dialog appears.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404952895255/dash-sched_576x413.png)

   Scheduled reports for this dashboard that have already been defined appear on the left side of the dialog. A message appears on the dialog if none have been defined.
6. Locate and select the scheduled report that you want to remove on the left side of the Scheduled Reports dialog. The settings for the report appear on the right side of the dialog.
7. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404959483287/trashcan-btn.png) next to the scheduled report name on the left side of the dialog. A warning dialog appears.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404952895767/dash-sched-warn_288x98.png)
8. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404959556887/delete-blue1-btn_43x22.png) to delete the scheduled dashboard report. The name of the scheduled report is removed from the list on the left of the Scheduled Reports dialog.
