# Use the Time Bar Sidebar

The Time Bar sidebar lets you control some of the time bar settings for a visual. Controls for changing the time bar settings are provided using the interactivity sidebar. See [*Control How Users Interact With a Visual*](https://devnet.logianalytics.com/hc/en-us/articles/4402955732247-Control-How-Users-Interact-With-a-Visual).

To access the Time Bar sidebar for a visual:

1. Select the visual in the Visual Gallery or on a dashboard.
2. If you selected the visual on a dashboard, select **Edit Visual** on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955730967-Use-the-Visual-Menu) to access the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955735319-Use-the-Visual-Sidebar-Menu) for the visual.
3. Select the visual style option (![](https://devnet.logianalytics.com/hc/article_attachments/4404959475223/dashboard-schedule.png)) on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955735319-Use-the-Visual-Sidebar-Menu) for the visual. The Time Bar sidebar opens.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404952806807/sidebar-timebar_192x330.png)
4. Using the Time Bar sidebar, you can:

   * Enable the time bar. See [*Enable the Time Bar for a Visual*](https://devnet.logianalytics.com/hc/en-us/articles/4402955680791-Enable-the-Time-Bar-for-a-Visual).
   * Disable the time bar. See [*Disable the Time Bar for a Visual*](https://devnet.logianalytics.com/hc/en-us/articles/4402955680151-Disable-the-Time-Bar-for-a-Visual).
   * Select or change the time bar field. See [*Change the Time Bar Field*](https://devnet.logianalytics.com/hc/en-us/articles/4402963062295-Change-the-Time-Bar-Field).
   * Enable playback, if the selected time bar field supports it. See [*Live Mode and Historical Playback*](https://devnet.logianalytics.com/hc/en-us/articles/4402955681431-Live-Mode-and-Historical-Playback).
5. After making changes, select ![](https://devnet.logianalytics.com/hc/article_attachments/4404952794519/apply-btn.png) to apply the time bar changes you have specified.
6. Save the visual or dashboard to save your changes.
