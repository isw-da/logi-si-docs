# Select the Maximum Time Range

To select the maximum time range on the time bar:

1. Select the visual.
2. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743743511/time-bar-select-max_22x24.png) on the right side of the time bar to select the maximum time range on the time bar.
3. Save the dashboard.
