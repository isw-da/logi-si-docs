# Delete a Reference Line

To delete a reference line on a visual:

1. Access the Rulers sidebar for the visual. See [*The Rulers Sidebar*](https://devnet.logianalytics.com/hc/en-us/articles/4402963101591-The-Rulers-Sidebar).
2. In the **Reference Lines** section of the sidebar, locate the reference line you want to delete.
3. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404959489559/delete-open.png) to the right of the reference line definition on the Rulers sidebar.
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404952794519/apply-btn.png) to delete the reference line onthe visual.
