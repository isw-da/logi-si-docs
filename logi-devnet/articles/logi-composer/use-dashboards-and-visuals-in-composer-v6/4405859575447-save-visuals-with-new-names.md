# Save Visuals With New Names

When you have finished making changes to a visual on a dashboard, you can save it with a new name. This essentially makes a copy of the visual in the system. The visual on the dashboard is renamed and is saved with the new name when you save the dashboard. But the original visual still exists in the system using the original name.

![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) You can also copy a visual using the **Copy** option on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu) when you are editing a dashboard. See [*Copy Visuals*](https://devnet.logianalytics.com/hc/en-us/articles/4405859567895-Copy-Visuals).

![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) To save visuals with a new name, you must be logged in as an administrator or as a user with the **Can Create Visuals** or **Can Administer Visuals**[group privilege](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference).

To save a visual with a new name:

1. Select the visual on a dashboard or in the Visual Gallery. When selected on a dashboard, a blue border appears around the visual. If you select a visual with a streaming data source, the [time bar](https://devnet.logianalytics.com/hc/en-us/articles/4405851196695-Use-the-Time-Bar) (Data DVR) appears.
2. Save the visual.

   * If you are editing the visual in a dashboard, select **Save As** on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu). The following Save As Options dialog appears.

     ![](https://devnet.logianalytics.com/hc/article_attachments/4406747383831/vis-save-as-options-dialog_288x202.png)

     Specify a new visual name in the **Visual Name** box and select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743715607/save-blue2-btn.png). The visual is renamed on the dashboard and saved as a new visual with the new name.
   * If you are editing the visual in the Visual Gallery, select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743716631/save-as_25x27.png) in upper right corner of the visual. The following Save As Options dialog appears.

     ![](https://devnet.logianalytics.com/hc/article_attachments/4406743716887/save-as-options-dialog2_288x201.png)

     Specify a default title in the **Default Title** box and a new visual name in the **Visual Name** box and select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743715607/save-blue2-btn.png). The visual is saved using the new name.

   The original visual still exists with its original name in the system.
