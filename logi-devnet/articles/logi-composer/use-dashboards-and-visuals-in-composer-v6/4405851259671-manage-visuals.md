# Manage Visuals

Composer provides a wide array of different visual styles and tools to help you visualize and explore your data. The Visual Gallery provides a list of all the visuals you have saved and allows you to select and manage them.

This section guides you in setting up and exploring the data using Composer's robust capabilities.
