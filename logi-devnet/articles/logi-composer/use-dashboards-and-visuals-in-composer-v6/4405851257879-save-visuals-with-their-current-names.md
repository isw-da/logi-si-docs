# Save Visuals With Their Current Names

When you have finished making changes to a visual on a dashboard, you can save it with its original name. Changes you've made to the visual will appear on all dashboards that use the visual.

![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) If you are editing the visual in a dashboard, you can save the visual with its current name when you save the dashboard. However, the default is not to save them when you save the dashboard. You must explicitly elect to save them on the dashboard Save Options dialog. See [*Save a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4405850986647-Save-a-Dashboard).

![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) To save visuals, you must be logged in a user with [write permissions](https://devnet.logianalytics.com/hc/en-us/articles/4405851253271-About-Visual-Permissions) for the visual or with the **Can Administer Visuals**[group privilege](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference).

To save a visual with its current name:

1. Select the visual on a dashboard or in the Visual Gallery. When selected on a dashboard, a blue border appears around the visual. If you select a visual with a streaming data source, the [time bar](https://devnet.logianalytics.com/hc/en-us/articles/4405851196695-Use-the-Time-Bar) (Data DVR) appears.
2. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743710231/dash-save.png) in the upper right corner of the visual.

   * If you are editing the visual in the Visual Gallery, the Save Options dialog appears.

     ![](https://devnet.logianalytics.com/hc/article_attachments/4406743715223/vis-save-options-dialog_288x202.png)

     Optionally, specify a default title in the **Default Title** box and a new visual name in the **Visual Name** box and select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743715607/save-blue2-btn.png). The visual is saved.
   * If you are editing the visual in a dashboard, a warning appears indicating that the visual changes will occur for all dashboards that use the visual.

     ![](https://devnet.logianalytics.com/hc/article_attachments/4406743715991/save-warn_384x140.png)

     Select OK on the warning dialog. The visual is saved.
