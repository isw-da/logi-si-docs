# Configure Settings for a Specific Table

You can configure the settings for a specific table while you are viewing it using the Table Settings sidebar or the [table context menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859541015-Table-Context-Menu). See the following sections:

* [*Add or Remove Table Columns*](https://devnet.logianalytics.com/hc/en-us/articles/4405851235863-Add-or-Remove-Table-Columns)
* [*Move Table Columns*](https://devnet.logianalytics.com/hc/en-us/articles/4405859539735-Move-Table-Columns)
* [*Change Column Widths*](https://devnet.logianalytics.com/hc/en-us/articles/4405859538455-Change-Column-Widths)
* [*Group and Ungroup Table Data*](https://devnet.logianalytics.com/hc/en-us/articles/4405851238551-Group-and-Ungroup-Table-Data)
* [*Apply Even Time Intervals on Tables*](https://devnet.logianalytics.com/hc/en-us/articles/4405850992023-Apply-Even-Time-Intervals-on-Tables)
* [*Change Time Field Granularity in Tables*](https://devnet.logianalytics.com/hc/en-us/articles/4405851240599-Change-Time-Field-Granularity-in-Tables)
* [*Change Metric Aggregation in Tables*](https://devnet.logianalytics.com/hc/en-us/articles/4405851233431-Change-Metric-Aggregation-in-Tables)
* [*Change Table Pagination*](https://devnet.logianalytics.com/hc/en-us/articles/4405851239447-Change-Table-Pagination)
* [*Sort Data in a Table*](https://devnet.logianalytics.com/hc/en-us/articles/4405859549591-Sort-Data-in-a-Table)

In addition, default settings can be specified for tables in data source configurations. See [*Configure Default Table Settings*](https://devnet.logianalytics.com/hc/en-us/articles/4405859542295-Configure-Default-Table-Settings).
