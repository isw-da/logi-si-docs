# The Visual Interface

# The Visual Interface

Composer's canvas provides a comprehensive suite of tools to help you conduct in-depth analysis, filter data points, and discover needed insights quickly and efficiently. This topic describes the capabilities and tools for visuals that are available.

The following picture and accompanying legend describes the features of a visual:

![](https://devnet.logianalytics.com/hc/article_attachments/4406743718295/visual-layout-new_657x313.png)

1. **Visual title:** By default, the visual title is the name of the data source used for the visual. Select the title to edit it.
2. **Dashboard Name or Title**or **Visual Name**:

   * If you are creating a visual on a new dashboard, this is the dashboard name. Select the name to edit it or select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743710231/dash-save.png) to save the dashboard and be prompted for a dashboard name. The default name of the dashboard is "Untitled dashboard;" the default name of the visual is the name of the data source used for the visual.
   * If you are creating a visual from the [Visual Gallery](https://devnet.logianalytics.com/hc/en-us/articles/4405859572119-Use-the-Visual-Gallery), this is the visual name. Select the name to edit it. The default name of the visual is "New Visual."
3. Indicates whether the dashboard has been saved. This information is not available if you are editing a visual from the Visual Gallery.
4. **Time Bar** and **Time Controls**:

   * Filter your data by a time attribute
   * Zoom into specific period within the selected date range
   * Explore the dynamics of your data using the **Play/Pause** button

   See [*Use the Time Bar*](https://devnet.logianalytics.com/hc/en-us/articles/4405851196695-Use-the-Time-Bar).
5. **X-** and **Y- Axis Labels** and the **Color Metric** (availability varies, depending on the visual style): Select an axis label to change an attribute or metric. See [*Change the Axes*](https://devnet.logianalytics.com/hc/en-us/articles/4405851212567-Change-the-Axes) and [*Change the Visual Color Metric*](https://devnet.logianalytics.com/hc/en-us/articles/4405851216151-Change-the-Visual-Color-Metric).

   If you disable the Volume property in a data source configuration, it does not appear in visual tooltips. The only exception to this is in histograms which plot the volume or number of records.
6. **Color Legend:** By default, a color legend is enabled for all visual styles, but can be switched off.
7. The dashboard icon bar. Use the options here to manage the dashboard. See [*Use the Dashboard Icon Bar*](https://devnet.logianalytics.com/hc/en-us/articles/4405850991127-Use-the-Dashboard-Icon-Bar). This is only available on a dashboard; not on a visual edited from the [Visual Gallery](https://devnet.logianalytics.com/hc/en-us/articles/4405859572119-Use-the-Visual-Gallery).
8. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406747384471/expand.png) to expand a visual on a dashboard. This allows you to get a closer look at it. This is only available on a dashboard; not on a visual edited from the [Visual Gallery](https://devnet.logianalytics.com/hc/en-us/articles/4405859572119-Use-the-Visual-Gallery).
9. The ![](https://devnet.logianalytics.com/hc/article_attachments/4406743718551/three-dots-menu_19x11.png) at the top right of a visual displays the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu). This menu includes an option to edit the visual which allows you to view the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859578391-Use-the-Visual-Sidebar-Menu).
10. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743718807/filter-dash.png) to the left of the dashboard or visual title to filter the data on a dashboard or visual. The dashboard-level filter icon is available only when all visuals on a dashboard are from the same data source.
11. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743719191/undo-new.png) to undo the most recent change you made to a visual.
12. This bar is the visual [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859578391-Use-the-Visual-Sidebar-Menu). Select a sidebar menu option to open an appropriate sidebar for a visual. To see the sidebar menu when you are working with a visual in a dashboard, select the visual first and then select **Edit Visual** from the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu). See [*Use the Visual Sidebar Menu*](https://devnet.logianalytics.com/hc/en-us/articles/4405859578391-Use-the-Visual-Sidebar-Menu).
