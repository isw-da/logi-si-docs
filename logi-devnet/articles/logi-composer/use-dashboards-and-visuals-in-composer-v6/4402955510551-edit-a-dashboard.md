# Edit a Dashboard

To edit a dashboard:

1. Log into Composer as an administrator or a user who has been assigned to a group with the [**Can Administer Dashboards** privilege](https://devnet.logianalytics.com/hc/en-us/articles/4402955416087-Group-Privilege-Reference).
2. Select **Library** on the [top-level navigation banner](https://devnet.logianalytics.com/hc/en-us/articles/4402963039639-The-Top-Level-Navigation-Banner) or the [UI menu](https://devnet.logianalytics.com/hc/en-us/articles/4402963040407-The-Composer-UI-Menu), or select the **Dashboards** box on the [Home page](https://devnet.logianalytics.com/hc/en-us/articles/4402963020951-Home-Page). The dashboard library displays.
3. Select the dashboard in the list. The dashboard opens.
4. Adjust the dashboard or the visuals on the dashboard as needed.
5. [Save](https://devnet.logianalytics.com/hc/en-us/articles/4402962944151-Save-a-Dashboard) your dashboard.
