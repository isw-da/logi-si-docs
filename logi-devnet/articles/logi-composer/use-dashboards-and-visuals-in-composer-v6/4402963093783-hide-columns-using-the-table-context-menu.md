# Hide Columns Using the Table Context Menu

To hide table columns using the table context menu:

1. View the table visual in a dashboard or from the Visual Gallery.
2. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404952786071/three-dots-menu.png) next to the column heading name to access the table context menu.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404952797847/rdt-contextmenu.png)
3. Select the ![](https://devnet.logianalytics.com/hc/article_attachments/4404952799383/rdt-fieldlist.png) tab on the context menu. A list of all the fields in the data source appears.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404952799639/rdt-fieldselect_192x253.png)
4. Clear (uncheck) the fields you want to hide in the table.

   Use the search bar to search for a field in the table.

   Fields you hide disappear from the table.
5. Save the dashboard and visual.
