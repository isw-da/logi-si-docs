# Maps

# Maps

Maps include map marker maps, US region maps, and world country maps. These visual styles are supported by most data sources. You can configure the defaults for all map styles in your data source configurations. To configure your visuals, see the following:

* [*Marker Maps*](https://devnet.logianalytics.com/hc/en-us/articles/4405851224727-Marker-Maps)
* [*US Region Maps*](https://devnet.logianalytics.com/hc/en-us/articles/4405859527831-US-Region-Maps)
* [*World Maps*](https://devnet.logianalytics.com/hc/en-us/articles/4405859528855-World-Maps)
