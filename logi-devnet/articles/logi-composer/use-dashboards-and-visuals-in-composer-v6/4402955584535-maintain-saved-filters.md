# Maintain Saved Filters

After a filter is created and applied to a visual, it can be saved and applied to other visuals that use the same data source. A saved filter can include the settings of one or more filters.

The following topics describe how to maintain a saved filter.

* [*Create a Saved Filter*](https://devnet.logianalytics.com/hc/en-us/articles/4402955583127-Create-a-Saved-Filter)
* [*Edit a Saved Filter*](https://devnet.logianalytics.com/hc/en-us/articles/4402955583767-Edit-a-Saved-Filter)
* [*Delete a Saved Filter*](https://devnet.logianalytics.com/hc/en-us/articles/4402962992791-Delete-a-Saved-Filter)
