# Scatter (Bubble) Charts

Scatter charts include floating bubbles, packed bubbles, and scatter plot styles. You can configure the defaults for all scatter chart styles in a data source configuration. For more information, select one of the following scatter chart styles:

* [*Floating Bubble Charts*](https://devnet.logianalytics.com/hc/en-us/articles/4405851218967-Floating-Bubble-Charts)
* [*Packed Bubble Charts*](https://devnet.logianalytics.com/hc/en-us/articles/4405851225623-Packed-Bubble-Charts)
* [*Scatter Plots*](https://devnet.logianalytics.com/hc/en-us/articles/4405851248279-Scatter-Plots)
