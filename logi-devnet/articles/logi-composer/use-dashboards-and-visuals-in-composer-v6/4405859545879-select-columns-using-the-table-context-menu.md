# Select Columns Using the Table Context Menu

To select fields for table columns using the table context menu:

1. View the table visual in a dashboard or from the Visual Gallery.
2. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743722135/three-dots-menu.png) next to the column heading name to access the table context menu.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743729303/rdt-contextmenu.png)
3. Select the ![](https://devnet.logianalytics.com/hc/article_attachments/4406743730583/rdt-fieldlist.png) tab on the context menu. A list of all the fields in the data source appears.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743730839/rdt-fieldselect_192x253.png)
4. Select the fields you want to view as columns in the table. To select all the fields, select the checkbox to the left of the search bar.

   Use the search bar to search for a field in the table.

   Fields you select appear in the table.
5. Save the dashboard and visual.
