# Ungroup Table Data Using the Table Context Menu

You can ungroup table data using the table context menu.

To ungroup table data using the table context menu:

1. View the table visual in a dashboard or from the Visual Gallery. Determine which field you want to remove from grouping.
2. Locate the field in the table and select ![](https://devnet.logianalytics.com/hc/article_attachments/4404952786071/three-dots-menu.png) next to its column heading to access the table context menu.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404959491863/rdt-contextmenu-ungroup.png)
3. Select **Ungroup by <field>** on the context menu.

   The table is ungrouped by that field.
4. Save the dashboard and visual.
