# Keyboard Controls

If keyboard controls are supported for a UI widget, you can use the following keyboard controls instead of a mouse:

| Keyboard Control | Description |
| --- | --- |
| Tab key | Use the tab key to move between areas of the widget. |
| Space bar | Use the space to select or clear a value or item in a widget. |
| Space bar + Up or Down Arrow key | Drag and drop items in the widget. Hold the space bar down while you use the up and down arrow keys. When you have placed reached the location where you want the item, push the space bar key again to place it. |
