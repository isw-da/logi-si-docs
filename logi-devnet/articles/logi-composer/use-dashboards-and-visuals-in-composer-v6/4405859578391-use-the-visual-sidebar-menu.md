# Use the Visual Sidebar Menu

# Use the Visual Sidebar Menu

Composer's visual canvas provides a comprehensive suite of tools to help you tailor visuals quickly so they provide the information you need in the best way for you to analyze your data. These tools are available from the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu) and from the visual sidebar menu. This topic discusses the visual sidebar menu.

The visual sidebar menu appears to the right of every visual edited from the Visual Gallery. With the exception of the visual interactivity option, it is also available when you select **Edit Visual** on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu) when editing a visual in a dashboard.

Selecting options on the visual sidebar menu opens and closes sidebars that allow you to control:

* The style of a selected dashboard visual
* The settings used for a selected visual
* The configuration of pivot tables and tables of raw data
* The filters used for a selected visual
* Time bar settings for a visual
* Sort and limit options for a selected visual
* Ruler and reference line settings for a selected visual
* Colors for a selected visual
* Interactivity settings for a selected visual (only available in the Visual Gallery)
* General information for a selected visual

When you are editing a visual on a dashboard, you must first select a visual (a blue border appears around the selected visual) and then select **Edit Visual** on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu). This makes the visual sidebar menu available.

If an option on the visual sidebar menu cannot be selected, the settings controlled by that sidebar are not available for the visual style you have selected.

The following visual sidebar menu options are available:

| Menu Option | Opens Sidebar |
| --- | --- |
|  | Opens or closes the Visual Style sidebar. Use this sidebar to change the visual style used for the selected visual (for example, to change a pie chart to a donut chart). See [*Use the Visual Style Sidebar*](https://devnet.logianalytics.com/hc/en-us/articles/4405859562775-Use-the-Visual-Style-Sidebar). This option is not available for [list filter visuals](https://devnet.logianalytics.com/hc/en-us/articles/4405859526295-List-Filter-Visuals). |
|  | Opens or closes the Sort & Limit sidebar. Use this sidebar to sort and limit the data for the selected visual. See [*Sort and Limit Visual Data*](https://devnet.logianalytics.com/hc/en-us/articles/4405859559063-Sort-and-Limit-Visual-Data). This option is not available for [list filter visuals](https://devnet.logianalytics.com/hc/en-us/articles/4405859526295-List-Filter-Visuals). |
|  | Opens or closes the Filter sidebar. Use this sidebar to filter the data for the selected visual. See [*Apply Row-Level Filters*](https://devnet.logianalytics.com/hc/en-us/articles/4405851055639-Apply-Row-Level-Filters), [*Apply Wild Card Filters to a Visual or Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4405851071383-Apply-Wild-Card-Filters-to-a-Visual-or-Dashboard), [*Apply Group Filters*](https://devnet.logianalytics.com/hc/en-us/articles/4405851068567-Apply-Group-Filters), [*Create a Keyset From a CSV File*](https://devnet.logianalytics.com/hc/en-us/articles/4405859388567-Create-a-Keyset-From-a-CSV-File), and [*Review and Apply a Keyset as a Visual Filter*](https://devnet.logianalytics.com/hc/en-us/articles/4405859387543-Review-and-Apply-a-Keyset-as-a-Visual-Filter). |
|  | Opens or closes the Time Bar sidebar. Use this sidebar to control the time bar for the selected visual. See [*Use the Time Bar Sidebar*](https://devnet.logianalytics.com/hc/en-us/articles/4405851197719-Use-the-Time-Bar-Sidebar). This option is not available for [list filter visuals](https://devnet.logianalytics.com/hc/en-us/articles/4405859526295-List-Filter-Visuals). |
|  | Opens or closes an appropriate Visual Settings sidebar. Use this sidebar to change settings for the selected visual. Different settings can be altered for different visual styles. In addition, some visual styles do not have any settings you can control in this way; this icon is disabled for these visual styles. See [*Change Visual Settings*](https://devnet.logianalytics.com/hc/en-us/articles/4405859576727-Change-Visual-Settings). |
|  | Opens or closes the Ruler sidebar. Use this sidebar to control the rulers and reference lines on the selected visual. See [*Use Rulers*](https://devnet.logianalytics.com/hc/en-us/articles/4405859556247-Use-Rulers) and [*Use Reference Lines*](https://devnet.logianalytics.com/hc/en-us/articles/4405859552919-Use-Reference-Lines). This option is not available for [list filter visuals](https://devnet.logianalytics.com/hc/en-us/articles/4405859526295-List-Filter-Visuals). |
|  | Opens or closes the Color sidebar. Use this sidebar to change the color settings for the selected visual. Different color settings are available for different chart styles. See [Change Color Schemes](https://devnet.logianalytics.com/hc/en-us/articles/4405859519383-Change-Color-Schemes). This option is not available for [list filter visuals](https://devnet.logianalytics.com/hc/en-us/articles/4405859526295-List-Filter-Visuals). |
|  | Opens or closes the visual interactivity sidebar. Use this sidebar to control how users can interact with the selected visual. See [*Control How Users Interact With a Visual*](https://devnet.logianalytics.com/hc/en-us/articles/4405859573143-Control-How-Users-Interact-With-a-Visual). This option is only available for a visual when you are viewing it in the Visual Gallery. |
|  | Opens or closes the visual information sidebar. Use this sidebar to change basic information for the selected visual. See [*Modify the Visual Title and Description*](https://devnet.logianalytics.com/hc/en-us/articles/4405859521047-Modify-the-Visual-Title-and-Description). |
