# Change the Visual Style

After creating a visual on a dashboard, you can change its visual style. Based on the data source you are using, certain visual styles may not be available.

To change the visual style:

1. Select the visual in the dashboard or in the Visual Gallery.
2. Select the visual style icon (![](https://devnet.logianalytics.com/hc/article_attachments/4406743710743/sidebar-chttype.png)) on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859578391-Use-the-Visual-Sidebar-Menu) for the visual. The [Visual Style sidebar](https://devnet.logianalytics.com/hc/en-us/articles/4405859562775-Use-the-Visual-Style-Sidebar) opens.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743724951/sidebar-selectvis_192x356.png)
3. In the Select Visual Type sidebar, select the visual style you want to use. Use the search bar at the top of the sidebar to quickly locate a style in the list. Type any letter in the search bar and sidebar is filtered to show only visual style names that include that letter.
