# Edit Visuals

# Edit Visuals

You can edit visuals directly on a Composer dashboard or in the Visual Gallery. You can also edit a visual when it has been maximized in a dashboard. When you modify a visual from the Visual Gallery and save your changes, the changes are made to the visual in the Visual Gallery and everywhere the visual is used on dashboards.

To edit a visual, you must be logged in as an administrator or as a user belonging to a group with the [privilege **Can Administer Visuals**](https://devnet.logianalytics.com/hc/en-us/articles/4402955416087-Group-Privilege-Reference) enabled.

To edit a visual:

1. Select the visual on a dashboard or in the Visual Gallery. When selected on a dashboard, a blue border appears around the visual. If you select a visual with a streaming data source, the [time bar](https://devnet.logianalytics.com/hc/en-us/articles/4402963065239-Use-the-Time-Bar) (Data DVR) appears.
2. Depending on the edits you need to make, select an item on the visual drop-down menu (![](https://devnet.logianalytics.com/hc/article_attachments/4404952786071/three-dots-menu.png)) or one of the [sidebars](https://devnet.logianalytics.com/hc/en-us/articles/4402955726359-Use-the-Visual-Style-Sidebar) for the visual. You can also use the [axes labels](https://devnet.logianalytics.com/hc/en-us/articles/4402955699607-Change-the-Axes) on the visual to change what is displayed and the [radial menu](https://devnet.logianalytics.com/hc/en-us/articles/4402963090199-Use-the-Radial-Menu) to visualize the data on your visual, adjust the settings, and more.

   Some of the options described here can be performed in the Visual Gallery, some only when editing the visual in a dashboard, and some in both.
