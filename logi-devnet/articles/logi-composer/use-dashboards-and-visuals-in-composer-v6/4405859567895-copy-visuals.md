# Copy Visuals

You can copy visuals on your dashboard using the **Copy Visual** on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu). The visual is copied and displays on the dashboard.

![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) You can also copy a visual using the **Save As** option on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu) to save the visual with a new name. See [*Save Visuals With New Names*](https://devnet.logianalytics.com/hc/en-us/articles/4405859575447-Save-Visuals-With-New-Names).

![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) To copy visuals, you must be logged in as an administrator or as a user with the **Can Create Visuals** or **Can Administer Visuals**[group privilege](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference).

To control whether a visual can be copied, use the interactivity sidebar. See [*Control How Users Interact With a Visual*](https://devnet.logianalytics.com/hc/en-us/articles/4405859573143-Control-How-Users-Interact-With-a-Visual).

To copy a visual:

1. Select the visual on a dashboard. A blue border appears around the visual. If you select a visual with a streaming data source, the [time bar](https://devnet.logianalytics.com/hc/en-us/articles/4405851196695-Use-the-Time-Bar) (Data DVR) appears.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) You cannot copy a visual from the Visual Gallery or by editing it from the Visual Gallery.
2. Select **Copy Visual** on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu).

   A copy of the visual appears on the dashboard. Its initial name is the same as the original visual, with an incremented number added to the end of the name. For example, if the original name was **Orders**, the initial name of the first copied visual is **Orders (1)**.

   Every stored setting for the visual is copied, including sort and limit options, filters, ruler properties and reference lines, colors, time bar options. and other visual information. However, zoom levels for maps are not copied.
