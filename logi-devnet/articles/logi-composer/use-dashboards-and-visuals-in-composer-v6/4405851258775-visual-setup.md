# Visual Setup

# Visual Setup

When you configure a data source, you can adjust the default settings for visuals that use the data source on the data source configuration [Visuals tab](https://devnet.logianalytics.com/hc/en-us/articles/4405859325591-Visuals-Tab). The default settings vary by visual style. For information about the default settings for a visual style, select a visual style in the [visual reference](https://devnet.logianalytics.com/hc/en-us/articles/4405851153303-Composer-Visual-Metrics-and-Attributes-Reference).

Settings that are common for all visual styles (time bar and search settings) can be specified in Global Default Settings on the [Visuals tab](https://devnet.logianalytics.com/hc/en-us/articles/4405859325591-Visuals-Tab) of a data source configuration. See [*Configure Time Bar Defaults*](https://devnet.logianalytics.com/hc/en-us/articles/4405859324311-Configure-Time-Bar-Defaults).

To add a visual using the default visual settings defined in a [data source configuration](https://devnet.logianalytics.com/hc/en-us/articles/4405851027479-Manage-Data-Source-Configurations), see:

* [*Create Dashboards*](https://devnet.logianalytics.com/hc/en-us/articles/4405859261207-Create-Dashboards)
* [*Add Visuals to the Visual Gallery*](https://devnet.logianalytics.com/hc/en-us/articles/4405859564183-Add-Visuals-to-the-Visual-Gallery)
* [*Add New Visuals to a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4405851250199-Add-New-Visuals-to-a-Dashboard)
* [*Place Existing Visuals on a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4405851256983-Place-Existing-Visuals-on-a-Dashboard)

After the visual has been created, you can alter its settings. See [*The Visual Interface*](https://devnet.logianalytics.com/hc/en-us/articles/4405859574167-The-Visual-Interface).
