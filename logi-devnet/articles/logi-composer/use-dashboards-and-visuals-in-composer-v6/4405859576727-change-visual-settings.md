# Change Visual Settings

You can change the settings for a visual. Visual settings can be changed using the Visual Settings sidebar. Settings can only be changed for some visual styles.

To access the Visual Settings sidebar:

1. Select the visual in the dashboard for which the settings will be changed.
2. Select the visual settings icon (![](https://devnet.logianalytics.com/hc/article_attachments/4406747383319/sidebar-chtsettings.png)) on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859578391-Use-the-Visual-Sidebar-Menu) for the visual. If there are no settings to modify for a specific visual style, the visual settings icon is disabled.

   An appropriate visual settings sidebar appears. The one below is for a bar chart.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743714967/sidebar-vissettingsbar_192x353.png)
3. Adjust the settings as needed. See the description of the specific [visual style](https://devnet.logianalytics.com/hc/en-us/articles/4405851153303-Composer-Visual-Metrics-and-Attributes-Reference) for more information.
