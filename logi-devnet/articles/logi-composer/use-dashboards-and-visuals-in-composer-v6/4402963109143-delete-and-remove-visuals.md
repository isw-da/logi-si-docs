# Delete and Remove Visuals

You can remove visuals from dashboards and delete them from the Composer instance in the Visual Gallery. You cannot delete a visual if it is used by any dashboard.

## Remove a Visual From a Dashboard

To remove a visual from your dashboard:

1. Edit the dashboard. See [*Edit a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402955510551-Edit-a-Dashboard).
2. Select the visual to be removed.
3. Select **Remove Visual** on the [visual drop-down (![](https://devnet.logianalytics.com/hc/article_attachments/4404952780951/three-dots-menu_19x11.png)) menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955730967-Use-the-Visual-Menu). A pop-up dialog verifying the deletion.

   To control whether a visual can be removed, use the interactivity sidebar. See [*Control How Users Interact With a Visual*](https://devnet.logianalytics.com/hc/en-us/articles/4402955732247-Control-How-Users-Interact-With-a-Visual).
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404952786327/delete-btn_44x23.png) on the warning dialog to confirm the removal.

## Delete a Visual from the Visual Gallery

A visual can only be deleted from the [Visual Gallery](https://devnet.logianalytics.com/hc/en-us/articles/4402963110935-Use-the-Visual-Gallery) when it is not used in any dashboard. If the **Usage** column for a visual in the Visual Gallery shows any number except zero, the option to delete the visual is disabled.

To delete visuals, you must be logged in as an administrator or as a user belonging to a group with the [privilege **Can Administer Visuals**](https://devnet.logianalytics.com/hc/en-us/articles/4402955416087-Group-Privilege-Reference) enabled.

To delete a visual from the Visual Gallery:

1. Access the [Visual Gallery](https://devnet.logianalytics.com/hc/en-us/articles/4402963110935-Use-the-Visual-Gallery). See [*Use the Visual Gallery*](https://devnet.logianalytics.com/hc/en-us/articles/4402963110935-Use-the-Visual-Gallery).
2. Locate the visual in the Visual Gallery.
3. Verify that the visual is removed from all dashboards. The **Usage** column value for the visual in the [Visual Gallery](https://devnet.logianalytics.com/hc/en-us/articles/4402963110935-Use-the-Visual-Gallery) should be zero.
4. Select the delete (trashcan) icon (![](https://devnet.logianalytics.com/hc/article_attachments/4404952790039/trashcan.png)) in the **Actions** column associated with the visual. A pop-up dialog verifying the deletion.
5. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404952790295/delete-btn_48x25.png) on the warning dialog to confirm the deletion.
