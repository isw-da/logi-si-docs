# Combo Charts

# Combo Charts

Combo charts are based on a single independent variable (usually the x-axis) and up to four dependent variables (usually the y-axes). The dependent variables can be plotted using either bar or line visual styles. The independent variable (x-axis) can be any attribute or metric in the data; the dependent variables (y-axes) must be a metric (numeric or integer field). A single attribute (independent variable) and two metrics (dependent variables) are required.

The legend on a combo chart is interactive. It allows you to show or hide different plots on the chart. By default, the plots for all of the metrics on the combo chart are shown. If you select one of the metrics in the legend, its name is grayed out and its plot is removed from the chart. To see its plot again, select the metric in the legend again. In the following example, the plot for the `Usersentiment` metric is not shown on the chart and its name is grayed out in the legend.

![](https://devnet.logianalytics.com/hc/article_attachments/4406747392151/combo-chart_768x287.png)

Combo charts are supported by all Composer [data connectors](https://devnet.logianalytics.com/hc/en-us/articles/4405859436055-Composer-Data-Connector-Reference).

## Configure Default Combo Chart Settings for a Data Source

To configure default standard combo chart settings for a data source configuration:

1. Edit the appropriate data source configuration. See [*Edit a Data Source Configuration*](https://devnet.logianalytics.com/hc/en-us/articles/4405851024663-Edit-a-Data-Source-Configuration).
2. Select the [Visuals](https://devnet.logianalytics.com/hc/en-us/articles/4405859325591-Visuals-Tab) tab in the data source configuration wizard.
3. On the **Standard** subtab of the [Visuals](https://devnet.logianalytics.com/hc/en-us/articles/4405859325591-Visuals-Tab) tab, select **Combo Chart**.

   If you want to be able to display the data collected by this data source configuration as a combo chart, make sure the **Combo Chart** checkbox is selected.

   Default combo chart settings appear on the right side of the page.
4. Configure the default settings as follows:

   | Setting | Description |
   | --- | --- |
   | Group By | Select the default field by which the data should be grouped. This field appears on the x-axis of the combo chart. It can be changed on the combo chart after it is created.  If you select a date or time attribute, a second box appears to the right for this setting. Use this second box to specify the date and time granularity used to group the data.  Valid granularity options are **Millisecond**, **Second**, **Minute**, **Hour**, **Day**, **Week**, **Month**, **Quarter**, or **Year**. If some of these granularity options do not appear, verify that the granularity specified for this field on the **Fields** tab of the data source definition is set at the correct granularity level.  The refinement level of the field data in a data store defines the minimum level of granularity that should be set for the field. Specifying granularity for a field that is lower than the refinement level of the field data will not produce a visual with data grouped at the requested lower level. For example, if a field's data is stored in hours, requesting the granularity of that data lower than hours will produce the values up to the hour level, and the more detailed level information will be zeros (i.e., 0 minutes, 0 seconds, and 0 milliseconds). |
   | Group By Limit | Specify the number of attribute items to display on the visual. |
   | Group By Sort | Select a metric or attribute by which the Group attribute items should be sorted. You can also select the sorting order (**Ascending** or **Descending**). For metrics, you can define the default aggregation function: **SUM**, **AVG**, **MAX**, or **MIN**. See [*Metric Aggregation Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405851021975-Metric-Aggregation-Functions). |
   | Y1 Axis | Select the numeric or integer field in the data source for the first dependent variable of the combo chart. This is plotted on the first y-axis (Y1) of the combo chart and appears on the left. (The other Y axes appear on the right of the combo chart.)  The values of the selected field are depicted by the Y1 axis color on the chart. The default color for the Y1 axis is based on the color palette that is selected for the visual. This can be changed in combo chart settings in the visual, but it cannot be changed in the data source default settings for combo charts. |
   | Y2 Axis | Select the numeric or integer field in the data source for the second dependent variable of the combo chart. This is plotted on the second y-axis (Y2) of the combo chart and appears on the right.  The values of the selected field are depicted by the Y2 axis color on the chart. The default color for the Y2 axis is set in the **Y2 Color** field in these data source combo chart defaults. This can be changed in the combo chart color settings in the visual. |
   | Y3 Axis | Select the numeric or integer field in the data source for the third dependent variable of the combo chart. This is plotted on the third y-axis (Y3) of the combo chart and appears on the right.  The values of the selected field are depicted by the Y3 axis color on the chart. The default color for the Y3 axis is set in the **Y3 Color** field in these data source combo chart defaults. This can be changed in the combo chart color settings in the visual. |
   | Y4 Axis | Select the numeric or integer field in the data source for the fourth dependent variable of the combo chart. This is plotted on the fourth y-axis (Y4) of the combo chart and appears on the right.  The values of the selected field are depicted by the Y4 axis color on the chart. The default color for the Y4 axis is set in the **Y4 Color** field in these data source combo chart defaults. This can be changed in the combo chart color settings in the visual. |
   | Y2 Color | Select the default color used to plot the Y2 axis values. This can be changed in the combo chart color settings in the visual. |
   | Y3 Color | Select the default color used to plot the Y3 axis values. This can be changed in the combo chart color settings in the visual. |
   | Y4 Color | Select the default color used to plot the Y4 axis values. This can be changed in the combo chart color settings in the visual. |
5. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406747382679/save-white-btn_40x23.png).

## Configure Settings for a Specific Combo Chart

To change the settings for a specific combo chart:

1. Edit the combo chart you want to modify. See [*Edit Visuals*](https://devnet.logianalytics.com/hc/en-us/articles/4405859570839-Edit-Visuals).
2. If you are editing the visual in a dashboard, select **Edit Visual** from the [chart drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu). The [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859578391-Use-the-Visual-Sidebar-Menu) for the visual appears.

   If you are editing the visual from the Visual Gallery, the sidebar appears to the right of the visual.
3. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406747383319/sidebar-chtsettings.png) on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859578391-Use-the-Visual-Sidebar-Menu). The Combo Chart Settings sidebar for the visual appears.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743736215/combo-standard-settings_288x428.png)
4. Alter the settings as needed:

   | Setting | Description |
   | --- | --- |
   | Enable Y3 | Use this switch to enable or disable the third dependent variable (on the Y3 axis) on the combo chart. When disabled, the Y3 variable is not plotted. |
   | Enable Y4 | Use this switch to enable or disable the fourth dependent variable (on the Y4 axis) on the combo chart. When disabled, the Y4 variable is not plotted. |
   | Y1 Axis Visual Style | Use the drop-down list to select a visual style for the Y1 axis variable. Only **Bar** or **Line** can be selected. |
   | Y2 Axis Visual Style | Use the drop-down list to select a visual style for the Y2 axis variable. Only **Bar** or **Line** can be selected. |
   | Y3 Axis Visual Style | Use the drop-down list to select a visual style for the Y3 axis variable. Only **Bar** or **Line** can be selected. |
   | Y4 Axis Visual Style | Use the drop-down list to select a visual style for the Y4 axis variable. Only **Bar** or **Line** can be selected. |
5. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743710231/dash-save.png) to save the dashboard and the visual with its updated settings.

## Configure Colors for a Specific Combo Chart

To specify the color settings for a specific combo chart using the Color sidebar:

1. Edit the visual you want to modify. See [*Edit Visuals*](https://devnet.logianalytics.com/hc/en-us/articles/4405859570839-Edit-Visuals).
2. If you are editing the visual in a dashboard, select **Edit Visual** from the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu). The [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859578391-Use-the-Visual-Sidebar-Menu) for the visual appears. If you are editing the visual from the Visual Gallery, the sidebar appears to the right of the visual.

   Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743709975/sidebar-color.png) on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859578391-Use-the-Visual-Sidebar-Menu). The Color sidebar for the visual appears.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406747392663/combo-color_192x414.png)
3. Configure the color settings as described below. As you change the color settings, the legend at the top of the Color sidebar shows how the legend will appear on the visual. Supported color specifications are described in [*Specify Colors*](https://devnet.logianalytics.com/hc/en-us/articles/4405859162903-Specify-Colors).

   | Setting | Description |
   | --- | --- |
   | Legend | Slide the bar to the right if you want the legend to be displayed on the visual or to the left to hide the legend. |
   | Color Settings | Select colors for the Y2, Y3, and Y4 dependent variables plotted on the combo chart. |
   | Y1 Axis Color Palette | Select the color palette for the Y1 axis for this specific combo chart. |
   | Inherit from theme | Select this checkbox to use the color palette specified by the [theme](https://devnet.logianalytics.com/hc/en-us/articles/4405859480983-Manage-UI-Themes). |
4. Close the Color sidebar and the color settings are dynamically applied to the visual.
5. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743710231/dash-save.png) to save the dashboard and the visual with its updated settings.
