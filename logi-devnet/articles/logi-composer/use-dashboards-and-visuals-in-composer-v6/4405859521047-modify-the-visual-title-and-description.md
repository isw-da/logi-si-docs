# Modify the Visual Title and Description

The Info tool lets you edit the visual title that is displayed on the canvas.

To edit the visual title or description:

1. Select the visual in the Visual Gallery or on a dashboard.
2. If you selected the visual on a dashboard, select **Edit Visual** on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu) to access the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859578391-Use-the-Visual-Sidebar-Menu) for the visual. If you selected the visual from the Visual Gallery, skip this step.
3. Select the information option (![](https://devnet.logianalytics.com/hc/article_attachments/4406743714711/sidebar-info.png)) on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859578391-Use-the-Visual-Sidebar-Menu) for the visual. The Visual Information sidebar opens.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743735959/sidebar-visinfo_192x314.png)

   The data source and visual type are shown on the sidebar, although you cannot change them. If you are editing the visual from the dashboard, the visual name also cannot be changed.

   If you are editing the visual from the Visual Gallery, you cannot change the description of the visual. However, you can change the description if you edit the visual from a dashboard. The image above was captured while editing the visual from a dashboard.
4. Optionally change the visual title in the **Default Title** box. You can also change the title of a visual by clicking on the title. See [*Understand Visual Names and Titles*](https://devnet.logianalytics.com/hc/en-us/articles/4405851256087-Understand-Visual-Names-and-Titles).
5. Optionally change the visual name in the **Visual Name** box. If you are editing the visual in the Visual Gallery, you can change the name of a visual by clicking on the name in the visual itself. If you are editing the visual on a dashboard, you cannot change the visual name after it has been initially created and saved, you must use the Visual Gallery to change the visual name. See [*Understand Visual Names and Titles*](https://devnet.logianalytics.com/hc/en-us/articles/4405851256087-Understand-Visual-Names-and-Titles).
6. Optionally change the visual description in the **Description** box. By default, there is no description. A maximum of 750 characters can be specified.
7. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743710231/dash-save.png) to save the dashboard and the visual with its updated settings.
