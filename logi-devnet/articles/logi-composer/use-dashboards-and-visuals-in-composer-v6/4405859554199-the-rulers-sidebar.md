# The Rulers Sidebar

The Rulers sidebar for a visual allows you to customize the rulers and reference lines used by the visual. When you save or export your visual, the settings configured on the Rulers sidebar are saved or exported respectively.

To access the Rulers sidebar:

1. Select the visual in the [Visual Gallery](https://devnet.logianalytics.com/hc/en-us/articles/4405859572119-Use-the-Visual-Gallery) or on a dashboard.
2. Select **Edit Visual** on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu) to access the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859578391-Use-the-Visual-Sidebar-Menu) for the visual. Then select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743713815/sidebar-ruler.png) on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859578391-Use-the-Visual-Sidebar-Menu).

   The Rulers sidebar opens. The sidebar consists of two sections: the (metric) ruler settings and, possibly, reference lines. If the visual does not support reference lines, this section is not available. See [*Enable and Disable Ruler Grid Lines*](https://devnet.logianalytics.com/hc/en-us/articles/4405851246487-Enable-and-Disable-Ruler-Grid-Lines) and [*Use Reference Lines*](https://devnet.logianalytics.com/hc/en-us/articles/4405859552919-Use-Reference-Lines).

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406747382295/noteicon.jpg) Arc gauges use the Rulers sidebar to control some different settings than are used by most visuals. See [*Arc Gauges*](https://devnet.logianalytics.com/hc/en-us/articles/4405851211287-Arc-Gauges).
