# Use the Visual Style Sidebar

The Visual Style sidebar lets you quickly change the style of visual displayed in a dashboard visual or when you are editing a visual in the Visual Gallery. Controls for changing the style of a visual are provided using the interactivity sidebar. See [*Control How Users Interact With a Visual*](https://devnet.logianalytics.com/hc/en-us/articles/4402955732247-Control-How-Users-Interact-With-a-Visual).

![](https://devnet.logianalytics.com/hc/article_attachments/4404959473559/noteicon.jpg) When you change the visual style of a visual, its [published and subscribed links](https://devnet.logianalytics.com/hc/en-us/articles/4402963028631-Control-How-Cross-Visual-Filters-Interact-in-a-Dashboard) are removed. See [*Control How Cross-Visual Filters Interact in a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402963028631-Control-How-Cross-Visual-Filters-Interact-in-a-Dashboard).

![](https://devnet.logianalytics.com/hc/article_attachments/4404959473559/noteicon.jpg) A list filter visual cannot be converted to a different style. Likewise, other visual styles cannot be converted to the list filter visual style.

To access the Visual Style sidebar for a visual:

1. Select the visual in the Visual Gallery or on a dashboard.
2. If you selected the visual on a dashboard, select **Edit Visual** on the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955730967-Use-the-Visual-Menu) to access the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955735319-Use-the-Visual-Sidebar-Menu) for the visual.
3. Select the visual style option (![](https://devnet.logianalytics.com/hc/article_attachments/4404952769303/sidebar-chttype.png)) on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955735319-Use-the-Visual-Sidebar-Menu) for the visual. The Visual Style sidebar opens.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404959487639/sidebar-selectvis_192x356.png)
4. Select a visual style from the list.
