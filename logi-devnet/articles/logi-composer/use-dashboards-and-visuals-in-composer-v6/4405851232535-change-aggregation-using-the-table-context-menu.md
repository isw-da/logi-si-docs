# Change Aggregation Using the Table Context Menu

To change the aggregation metrics in a table using the table context menu:

1. [Edit](https://devnet.logianalytics.com/hc/en-us/articles/4405850982039-Edit-a-Dashboard) a dashboard with a table that uses a data source containing metric fields.
2. Group the table. See [*Group and Ungroup Table Data*](https://devnet.logianalytics.com/hc/en-us/articles/4405851238551-Group-and-Ungroup-Table-Data).
3. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743722135/three-dots-menu.png) next to the column heading name of any metric field in the table to access the table context menu.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743732119/rdt-contextmenu-num.png)
4. In any of the metric fields in the **Columns** section of the sidebar, select the aggregation method you want. Possible aggregation methods include **None**, **Avg**, **Min**, **Max**, **Sum**, **Last Value**, and **Distinct Count** (for attributes with [distinct counts enabled](https://devnet.logianalytics.com/hc/en-us/articles/4405859300503-Enable-Distinct-Counts)). For information about aggregation methods, see [*Metric Aggregation Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405851021975-Metric-Aggregation-Functions).

   The selected aggregation method is applied to the metric.
5. Save the dashboard and visual.
