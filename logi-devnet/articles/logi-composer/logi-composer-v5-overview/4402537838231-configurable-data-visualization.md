# Configurable Data Visualization

# Configurable Data Visualization

You can visualize data from your data stores in visuals and dashboards in the Composer UI. A dashboard is a container for one or more visuals. The visuals in a dashboard do not have to be related, but for easier data analysis, probably should be. See [*Multisource Analysis*](https://devnet.logianalytics.com/hc/en-us/articles/4402537836951-Multisource-Analysis).

Data can be visualized using a wide variety of visual styles: arc gauges, bar charts, box plots, donuts, heat maps, KPI charts, line charts, maps, pie charts, scatter charts, tables, tree maps, and word clouds are all supported. Variations of these visual styles can also be produced. For example, you can produce standard bar charts, stacked bar charts, clustered bar charts, histograms, and multiple metric bar charts. See the [*Visual Metrics and Attributes Reference*](https://devnet.logianalytics.com/hc/en-us/articles/4402537848855-Visual-Metrics-and-Attributes-Reference).

You can customize visuals in a variety of ways. When you define a data source configuration, you can specify default customizations for visuals created using the data source. Read about the [*Visuals Tab*](https://devnet.logianalytics.com/hc/en-us/articles/4402537788695-Visuals-Tab) of a data source configuration.

After a visual is created in a dashboard, you can customize it in the following ways. Some of these customizations depend on the selected visual style or the data source configuration used for the visual.

| Customization | Description |
| --- | --- |
| Visual Colors | You can change the colors used in the visual. See [*Changing Color Schemes*](https://devnet.logianalytics.com/hc/en-us/articles/4402537921943-Changing-Color-Schemes) and [*Changing the Visual Color Metric*](https://devnet.logianalytics.com/hc/en-us/articles/4402537921303-Changing-the-Visual-Color-Metric). |
| Visual Style | You can change the visual style. See [*Changing the Visual Style*](https://devnet.logianalytics.com/hc/en-us/articles/4402537920663-Changing-the-Visual-Style). |
| Titles and Descriptions | You can change the title and description of dashboards and visuals. See [*Renaming a Dashboard*](https://devnet.logianalytics.com/hc/en-us/articles/4402537747223-Renaming-a-Dashboard) and [*Modifying the Visual Title and Description*](https://devnet.logianalytics.com/hc/en-us/articles/4402552922263-Modifying-the-Visual-Title-and-Description). |
| Data Sorting | You can sort visual data. See [*Sorting and Limiting Visual Data*](https://devnet.logianalytics.com/hc/en-us/articles/4402537936919-Sorting-and-Limiting-Visual-Data). |
| Data Filtering | You can filter visual data by specific field values or ranges. See [*Filter Data*](https://devnet.logianalytics.com/hc/en-us/articles/4402552815383-Filter-Data). |
| You can filter visual and dashboard data by time ranges using the time bar. See [*Use the Time Bar*](https://devnet.logianalytics.com/hc/en-us/articles/4402537901591-Use-the-Time-Bar) |
| You can filter visual data by unique data values stored in a keyset. See [*Using Keysets*](https://devnet.logianalytics.com/hc/en-us/articles/4402537825175-Using-Keysets). |
| You can filter dashboard data by cross-source linked fields from different data sources. See [*Using Cross-Source Links*](https://devnet.logianalytics.com/hc/en-us/articles/4402537728023-Using-Cross-Source-Links). |

In addition to these dashboard and visual configuration features, you can link dashboards and you can share dashboards with other users. You can also export visuals and dashboards in a variety of formats: PNG, PDF, CSV (visuals only), and JSON (dashboards only).  Exported visuals and dashboards can be imported into other Composer environments running the same version of Composer. See [*Manage Dashboards*](https://devnet.logianalytics.com/hc/en-us/articles/4402552774423-Manage-Dashboards) and [*Manage Visuals*](https://devnet.logianalytics.com/hc/en-us/articles/4402537948951-Manage-Visuals).

Finally, you can change the look of the UI altogether using themes. See [*Manage UI Themes*](https://devnet.logianalytics.com/hc/en-us/articles/4402537896343-Manage-UI-Themes).
