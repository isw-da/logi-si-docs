# Access the Custom Metrics Editor from a Data Source Configuration

To access the Custom Metrics Editor from a data source configuration:

1. Edit the data source configuration in the UI. See [*Edit a Data Source Configuration*](https://devnet.logianalytics.com/hc/en-us/articles/4405851024663-Edit-a-Data-Source-Configuration).
2. In the data source configuration wizard, select the **Fields** tab.
3. Scroll to the Custom Metric table at the bottom of the Fields tab.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743856663/custmetriclist_576x54.png)
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406747435159/add-cust-metric-btn_125x23.png) to add a custom metric. To modify a custom metric, select the custom metric name on the Fields tab.

   The [*Custom Metrics Editor*](https://devnet.logianalytics.com/hc/en-us/articles/4405859284119-Custom-Metrics-Editor) appears.
