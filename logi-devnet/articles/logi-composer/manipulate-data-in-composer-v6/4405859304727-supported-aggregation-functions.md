# Supported Aggregation Functions

# Supported Aggregation Functions

Use aggregate functions in [custom metrics](https://devnet.logianalytics.com/hc/en-us/articles/4405859285399-Maintain-Custom-Metrics). Custom metrics can also include the use of [row-level functions](https://devnet.logianalytics.com/hc/en-us/articles/4405851021079-Supported-Row-Level-Functions). In addition, they can be [filtered](https://devnet.logianalytics.com/hc/en-us/articles/4405851003543-Apply-Filters-to-Custom-Metrics) (using [date and time filter functions](https://devnet.logianalytics.com/hc/en-us/articles/4405851015447-Date-and-Time-Filter-Aggregation-Functions) and [SQL-like expressions](https://devnet.logianalytics.com/hc/en-us/articles/4405859312535-Supported-SQL-Like-Expressions)).

Data can be aggregated using column, table, or window aggregation functions. Each is explained in the links below.

* [*Column Aggregation Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405859301911-Column-Aggregation-Functions)
* [*Table Aggregation Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405859303319-Table-Aggregation-Functions)
* [*Window Aggregation Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405851016343-Window-Aggregation-Functions)
* [*Date and Time Filter Aggregation Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405851015447-Date-and-Time-Filter-Aggregation-Functions)
