# Access the Custom Metrics Editor from the Filters Sidebar

To access the Custom Metrics Editor from the Filters sidebar of a visual or dashboard:

1. Access the visual filter sidebar or the dashboard filter sidebar.

   * To access the visual filter sidebar, select the visual filter icon (![](https://devnet.logianalytics.com/hc/article_attachments/4406743754263/filter-vis.png)) on the visual or select **Edit Visual** from the [visual menu](https://devnet.logianalytics.com/hc/en-us/articles/4405851254295-Use-the-Visual-Menu) (![](https://devnet.logianalytics.com/hc/article_attachments/4406743718551/three-dots-menu_19x11.png)) and then select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743782935/sidebar-filter_20x21.png) on the [sidebar menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859578391-Use-the-Visual-Sidebar-Menu).
   * To access the dashboard filter sidebar, select the dashboard filter icon (![](https://devnet.logianalytics.com/hc/article_attachments/4406743718807/filter-dash.png)) next to the dashboard title. The dashboard-level filter icon is available only when all the visuals are from the same data source.

   The Filters sidebar appears.
2. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743717271/add.png) at the top of the Attribute, Number, or Time sections on the Row or Group tab. The following menu appears:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743853207/add-derived-field-custom-metric.png)
3. Select **Add Custom Metric** to access the [*Custom Metrics Editor*](https://devnet.logianalytics.com/hc/en-us/articles/4405859284119-Custom-Metrics-Editor) and create a custom metric.
4. Be sure to [save the dashboard](https://devnet.logianalytics.com/hc/en-us/articles/4405850986647-Save-a-Dashboard) after creating or modifying your custom metric. If you close the dashboard without saving, your custom metric will not be saved.

You can also access the [*Custom Metrics Editor*](https://devnet.logianalytics.com/hc/en-us/articles/4405859284119-Custom-Metrics-Editor) when you edit any custom metric listed on the Filters sidebar. See the following steps.

1. Select the filter icon to access the Filters sidebar for a [visual](https://devnet.logianalytics.com/hc/en-us/articles/4405851057431-Apply-a-Row-Level-Filter-to-a-Visual) or [dashboard](https://devnet.logianalytics.com/hc/en-us/articles/4405851056535-Apply-a-Row-Level-Filter-to-a-Dashboard).
2. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743856919/add-filter-btn_92x22.png).
3. Locate a custom metric listed on the Filters sidebar and select the ellipsis (![](https://devnet.logianalytics.com/hc/article_attachments/4406743718551/three-dots-menu_19x11.png)) next to it. The following menu appears:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743853463/derived-field-metric-edit-menu.png)
4. Select **Edit** on the menu to edit the custom metric. The [*Custom Metrics Editor*](https://devnet.logianalytics.com/hc/en-us/articles/4405859284119-Custom-Metrics-Editor) appears.
5. Be sure to [save the dashboard](https://devnet.logianalytics.com/hc/en-us/articles/4405850986647-Save-a-Dashboard) after creating or modifying your custom metric. If you close the dashboard without saving, your custom metric will not be saved.
