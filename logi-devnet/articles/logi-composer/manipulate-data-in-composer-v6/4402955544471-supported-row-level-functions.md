# Supported Row-Level Functions

# Supported Row-Level Functions

Use row-level functions in the row-level expressions you use to create the following calculation types:

* [derived fields](https://devnet.logianalytics.com/hc/en-us/articles/4402955538711-Maintain-Derived-Fields)
* [custom metrics](https://devnet.logianalytics.com/hc/en-us/articles/4402955525655-Maintain-Custom-Metrics)
* [admin-defined functions](https://devnet.logianalytics.com/hc/en-us/articles/4402955519511-Admin-Defined-Functions).

The row-level functions described in this section are fully supported by all three types. For information about the aggregate functions and SQL-like expressions you can use in custom metrics, see [*Supported Aggregation Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4402962963095-Supported-Aggregation-Functions) and [*Supported SQL-Like Expressions*](https://devnet.logianalytics.com/hc/en-us/articles/4402962968599-Supported-SQL-Like-Expressions).

Row-level functions are divided into the following categories:

* [*Arithmetic Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4402955541911-Arithmetic-Functions)
* [*Conditional Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4402955542551-Conditional-Functions)
* [*Logical Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4402955543191-Logical-Functions)
* [*Numerical Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4402962964247-Numerical-Functions)
* [*Relational Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4402962965015-Relational-Functions)
* [*Text Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4402955543831-Text-Functions)
* [*Time Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4402962965911-Time-Functions)
