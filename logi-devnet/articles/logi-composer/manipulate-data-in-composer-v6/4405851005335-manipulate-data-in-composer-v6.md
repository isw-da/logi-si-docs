# Manipulate Data in Composer v6

Composer allows you to manipulate the data read from your data store. You can enable distinct counts and Data Sharpening, change the formatting of numeric and time fields, convert appropriate existing data store fields into time fields, and use existing data store fields to create derived fields and custom metrics. See the following sections:

* [*Use Data Sharpening*](https://devnet.logianalytics.com/hc/en-us/articles/4405851008023-Use-Data-Sharpening)
* [*Enable Distinct Counts*](https://devnet.logianalytics.com/hc/en-us/articles/4405859300503-Enable-Distinct-Counts)
* [*Configure Number Formatting*](https://devnet.logianalytics.com/hc/en-us/articles/4405851006231-Configure-Number-Formatting)
* [*Configure Date and Time Fields*](https://devnet.logianalytics.com/hc/en-us/articles/4405859289879-Configure-Date-and-Time-Fields)
