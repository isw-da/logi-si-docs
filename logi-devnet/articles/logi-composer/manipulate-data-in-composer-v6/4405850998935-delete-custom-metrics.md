# Delete Custom Metrics

You can delete custom metrics from a data source using the data source configuration wizard or using the Metric or Filters sidebar on a dashboard.

To delete custom metrics from a data source while editing the data source configuration:

1. Edit the data source configuration in the UI. See [*Edit a Data Source Configuration*](https://devnet.logianalytics.com/hc/en-us/articles/4405851024663-Edit-a-Data-Source-Configuration).
2. In the data source configuration wizard, select the **Fields** tab.
3. Scroll to the Custom Metric table at the bottom of the Fields tab.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743856663/custmetriclist_576x54.png)
4. Locate the custom metric in the Custom Metrics table that you would like to remove.
5. Select the delete icon (![](https://devnet.logianalytics.com/hc/article_attachments/4406743776663/delete-grey.png)) in the Delete column.
6. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743860375/delete-blue1-btn_48x25.png) in the pop-up confirmation dialog. The custom metric is deleted.

To delete custom metrics from a data source using the Metric selection dialog:

1. On a visual, select the metric label (y-axis label) to view the Metric selection dialog.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743860759/metric-selection-panel_150x300.png)
2. Locate a custom metric listed on the Metric selection dialog and select the ellipsis (![](https://devnet.logianalytics.com/hc/article_attachments/4406743718551/three-dots-menu_19x11.png)) next to it. The following menu appears:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743853463/derived-field-metric-edit-menu.png)
3. Select **Remove** on the menu to delete the custom metric.
4. Be sure to save the dashboard after deleting the custom metric. If you close the dashboard without saving, your custom metric will not be deleted.

To delete custom metrics from a data source using the Filters sidebar:

1. Select the filter icon to access the Filters sidebar for a [visual](https://devnet.logianalytics.com/hc/en-us/articles/4405851057431-Apply-a-Row-Level-Filter-to-a-Visual) or [dashboard](https://devnet.logianalytics.com/hc/en-us/articles/4405851056535-Apply-a-Row-Level-Filter-to-a-Dashboard).
2. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406747415959/add-filter-btn_76x21.png).
3. Locate a custom metric listed on the Group tab of the Filters sidebar and select the ellipsis (![](https://devnet.logianalytics.com/hc/article_attachments/4406743718551/three-dots-menu_19x11.png)) next to it. The following menu appears:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743853463/derived-field-metric-edit-menu.png)
4. Select **Remove** on the menu to delete the custom metric.
5. Be sure to save the dashboard after deleting the custom metric. If you close the dashboard without saving, your custom metric will not be deleted.
