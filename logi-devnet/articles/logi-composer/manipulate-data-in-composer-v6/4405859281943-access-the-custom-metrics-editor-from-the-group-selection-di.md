# Access the Custom Metrics Editor from the Group Selection Dialog

To access the Custom Metrics Editor from the Group selection dialog of a visual:

1. On a visual, select the group label (x-axis label) to view the Group selection dialog.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743853719/group-selection-panel_150x278.png)
2. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743717271/add.png) in the Attributes, Number or Time section of the dialog. The following menu appears:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743853207/add-derived-field-custom-metric.png)
3. Select **Add Custom Metric** to access the [*Custom Metrics Editor*](https://devnet.logianalytics.com/hc/en-us/articles/4405859284119-Custom-Metrics-Editor) and create a custom metric.
4. Be sure to [save the dashboard](https://devnet.logianalytics.com/hc/en-us/articles/4405850986647-Save-a-Dashboard) after creating or modifying your custom metric. If you close the dashboard without saving, your custom metric will not be saved.
