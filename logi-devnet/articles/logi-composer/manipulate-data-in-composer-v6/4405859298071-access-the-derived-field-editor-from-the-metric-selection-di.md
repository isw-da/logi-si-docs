# Access the Derived Field Editor from the Metric Selection Dialog

To access the Derived Field Editor from the metric selection dialog of a visual:

1. On a visual, select the metric label (y-axis label) to view the Metric selection dialog.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743852695/metric-selection-panel_150x301.png)
2. Select the ![](https://devnet.logianalytics.com/hc/article_attachments/4406743717271/add.png) icon in the Number or Custom Metric section of the dialog. The following menu appears:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743853207/add-derived-field-custom-metric.png)
3. Select **Add Derived Field** to access the [*Derived Field Editor*](https://devnet.logianalytics.com/hc/en-us/articles/4405851014551-Derived-Field-Editor) and create a derived field.
4. Be sure to [save the dashboard](https://devnet.logianalytics.com/hc/en-us/articles/4405850986647-Save-a-Dashboard) after creating or modifying your derived field. If you close the dashboard without saving, your derived field will not be saved.

You can also access the [*Derived Field Editor*](https://devnet.logianalytics.com/hc/en-us/articles/4405851014551-Derived-Field-Editor) when you edit any derived field listed on the Metric selection dialog. See the following steps.

1. On a visual, select the metric label (y-axis label) to view the Metric selection dialog.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743852695/metric-selection-panel_150x301.png)
2. Locate a derived field listed on the Metric selection dialog and select the ellipsis (![](https://devnet.logianalytics.com/hc/article_attachments/4406743718551/three-dots-menu_19x11.png)) next to it. The following menu appears:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743853463/derived-field-metric-edit-menu.png)
3. Select **Edit** on the menu to edit the derived field. The [*Derived Field Editor*](https://devnet.logianalytics.com/hc/en-us/articles/4405851014551-Derived-Field-Editor) appears.
4. Be sure to [save the dashboard](https://devnet.logianalytics.com/hc/en-us/articles/4405850986647-Save-a-Dashboard) after creating or modifying your derived field. If you close the dashboard without saving, your derived field will not be saved.
