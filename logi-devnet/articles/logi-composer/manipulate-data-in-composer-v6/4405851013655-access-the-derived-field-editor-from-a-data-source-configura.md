# Access the Derived Field Editor from a Data Source Configuration

To access the Derived Field Editor from a data source configuration:

1. Edit the data source configuration in the UI. See [*Edit a Data Source Configuration*](https://devnet.logianalytics.com/hc/en-us/articles/4405851024663-Edit-a-Data-Source-Configuration).
2. In the data source configuration wizard, select the **Fields** tab.
3. Scroll to the Derived Fields table at the bottom of the Fields tab.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4406743851927/derivedfieldslist_576x54.png)
4. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4406743852311/add-derived-field-btn_123x23.png) to add a derived field. To modify a derived field, select the Type or Expression of the derived field on the Fields tab.

   The [*Derived Field Editor*](https://devnet.logianalytics.com/hc/en-us/articles/4405851014551-Derived-Field-Editor) appears.
