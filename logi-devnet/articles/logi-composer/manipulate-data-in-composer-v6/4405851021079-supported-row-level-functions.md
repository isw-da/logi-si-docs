# Supported Row-Level Functions

# Supported Row-Level Functions

Use row-level functions in the row-level expressions you use to create the following calculation types:

* [derived fields](https://devnet.logianalytics.com/hc/en-us/articles/4405859299479-Maintain-Derived-Fields)
* [custom metrics](https://devnet.logianalytics.com/hc/en-us/articles/4405859285399-Maintain-Custom-Metrics)
* [admin-defined functions](https://devnet.logianalytics.com/hc/en-us/articles/4405850995991-Admin-Defined-Functions).

The row-level functions described in this section are fully supported by all three types. For information about the aggregate functions and SQL-like expressions you can use in custom metrics, see [*Supported Aggregation Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405859304727-Supported-Aggregation-Functions) and [*Supported SQL-Like Expressions*](https://devnet.logianalytics.com/hc/en-us/articles/4405859312535-Supported-SQL-Like-Expressions).

Row-level functions are divided into the following categories:

* [*Arithmetic Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405859305751-Arithmetic-Functions)
* [*Conditional Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405851017239-Conditional-Functions)
* [*Logical Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405859306903-Logical-Functions)
* [*Numerical Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405851018263-Numerical-Functions)
* [*Relational Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405851019159-Relational-Functions)
* [*Text Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405851020183-Text-Functions)
* [*Time Functions*](https://devnet.logianalytics.com/hc/en-us/articles/4405859308823-Time-Functions)
