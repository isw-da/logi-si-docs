# Access the Custom Metrics Editor from the Color Sidebar

To access the Custom Metrics Editor from the Color sidebar of a visual:

1. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404952780951/three-dots-menu_19x11.png) and then **Color** from the [visual drop-down menu](https://devnet.logianalytics.com/hc/en-us/articles/4402955730967-Use-the-Visual-Menu) to access the Color sidebar.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404952884631/color-simple1-new_149x293.png)
2. Select the **Color Attribute** box on the Color sidebar. A list of attribute, number, and time fields you can select for the color attribute appears in the Color sidebar.
3. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404959479319/add.png) in the Attribute, Number, or Time fields section of the sidebar. The following menu appears:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404959551767/add-derived-field-custom-metric.png)
4. Select **Add Custom Metric** to access the [*Custom Metrics Editor*](https://devnet.logianalytics.com/hc/en-us/articles/4402955523607-Custom-Metrics-Editor) and create a custom metric.
5. Be sure to save the dashboard after creating or modifying your custom metric. If you close the dashboard without saving, your custom metric will not be saved.

You can also access the Custom Metrics Editor when you edit any custom metric listed on the Color sidebar. See the following steps.

1. Select ![](https://devnet.logianalytics.com/hc/article_attachments/4404952780951/three-dots-menu_19x11.png) and then Color from the menu to access the Color sidebar.
2. Select the **Color Metric** box on the Color sidebar. A list of fields you can select for the color metric appears in the Color sidebar.
3. Locate a custom metric listed on the Color sidebar and select the ellipsis (![](https://devnet.logianalytics.com/hc/article_attachments/4404952780951/three-dots-menu_19x11.png)) next to it. The following menu appears:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404959552023/derived-field-metric-edit-menu.png)
4. Select **Edit** on the menu to edit the custom metric. The [*Custom Metrics Editor*](https://devnet.logianalytics.com/hc/en-us/articles/4402955523607-Custom-Metrics-Editor) appears.
5. Be sure to save the dashboard after creating or modifying your custom metric. If you close the dashboard without saving, your custom metric will not be saved.
