# Access the Custom Metrics Editor from the Metric Selection Dialog

To access the Custom Metrics Editor from the metric selection dialog of a visual:

1. On a visual, select the metric label (y-axis label) to view the Metric selection dialog.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404959551511/metric-selection-panel_150x301.png)
2. Select the ![](https://devnet.logianalytics.com/hc/article_attachments/4404959479319/add.png) icon in the Number or Custom Metric section of the dialog. The following menu appears:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404959551767/add-derived-field-custom-metric.png)
3. Select **Add Custom Metric** to access the [*Custom Metrics Editor*](https://devnet.logianalytics.com/hc/en-us/articles/4402955523607-Custom-Metrics-Editor) and create a custom metric.
4. Be sure to [save the dashboard](https://devnet.logianalytics.com/hc/en-us/articles/4402962944151-Save-a-Dashboard) after creating or modifying your custom metric. If you close the dashboard without saving, your custom metric will not be saved.

You can also access the [*Custom Metrics Editor*](https://devnet.logianalytics.com/hc/en-us/articles/4402955523607-Custom-Metrics-Editor) when you edit any custom metric listed on the Metric selection dialog. See the following steps.

1. On a visual, select the metric label (y-axis label) to view the Metric selection dialog.

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404959551511/metric-selection-panel_150x301.png)
2. Locate a custom metric listed on the Metric selection dialog and select the ellipsis (![](https://devnet.logianalytics.com/hc/article_attachments/4404952780951/three-dots-menu_19x11.png)) next to it. The following menu appears:

   ![](https://devnet.logianalytics.com/hc/article_attachments/4404959552023/derived-field-metric-edit-menu.png)
3. Select **Edit** on the menu to edit the custom metric. The [*Custom Metrics Editor*](https://devnet.logianalytics.com/hc/en-us/articles/4402955523607-Custom-Metrics-Editor) appears.
4. Be sure to [save the dashboard](https://devnet.logianalytics.com/hc/en-us/articles/4402962944151-Save-a-Dashboard) after creating or modifying your custom metric. If you close the dashboard without saving, your custom metric will not be saved.
