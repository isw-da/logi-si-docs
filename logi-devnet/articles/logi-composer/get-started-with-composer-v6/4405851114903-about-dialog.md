# About Dialog

The About dialog identifies the version of Composer you are using and provides a link to the terms of service. It can be accessed by selecting **About** on the [UI](https://devnet.logianalytics.com/hc/en-us/articles/4405859444119-The-Composer-UI-Menu) or [supervisor](https://devnet.logianalytics.com/hc/en-us/articles/4405859441687-The-Composer-Supervisor-Menu) menus.
