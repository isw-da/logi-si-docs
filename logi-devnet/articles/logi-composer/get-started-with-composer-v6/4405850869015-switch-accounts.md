# Switch Accounts

If a user is assigned to multiple accounts, they can switch between accounts while they are logged into Composer. In addition,
a user definition can be assigned to the different accounts with different group authorizations.

You can specify the account used by a user the next time they log in. See [*Set the Current Account for a User*](https://devnet.logianalytics.com/hc/en-us/articles/4405850859927-Set-the-Current-Account-for-a-User). You can also assign users to multiple Composer accounts. See [*Assign and Remove Users in Composer Accounts*](https://devnet.logianalytics.com/hc/en-us/articles/4405859115671-Assign-and-Remove-Users-in-Composer-Accounts).

To switch accounts:

1. Log into Composer as the user that is assigned to multiple accounts.
2. From the menu (![](https://devnet.logianalytics.com/hc/article_attachments/4406743720343/hamburger.png)), select **My Accounts**. A submenu listing the accounts to which your user name has access appears.
3. Select the account to which you want to switch from the submenu.

Your user definition may provide you with different levels of access to Composer features and functions in different accounts. After switching accounts, the groups to which you are assigned in the new account determine which account data source configurations, data store connections, and dashboards (that you created or that were shared by other users of the account) are available for you to use.
