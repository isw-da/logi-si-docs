# The Top-Level Navigation Banner

# The Top-Level Navigation Banner

The top-level navigation banner appears in the Composer UI when you are logged in as an administrator or a non-supervisor user. This menu provides easy access to connection definitions, data source configuration definitions, and the dashboards in the UI. It looks like this:

![](https://devnet.logianalytics.com/hc/article_attachments/4406747409303/toplevel-navigation-banner.png)

Each option is described in the following table.

| Option | Description |
| --- | --- |
| Composer logo | Select the Composer logo to return to the Home page from anywhere in the product. See [*Home Page*](https://devnet.logianalytics.com/hc/en-us/articles/4405851121559-Home-Page). |
| Home | Select this option to return to the Home page from anywhere in the product. See [*Home Page*](https://devnet.logianalytics.com/hc/en-us/articles/4405851121559-Home-Page). |
| Connections | Select this option to review and manage the connection definitions used by Composer connectors. See [*Manage Data Store Connections*](https://devnet.logianalytics.com/hc/en-us/articles/4405850909079-Manage-Data-Store-Connections). |
| Sources | Select this option to review and manage the data source configuration definitions used for Composer queries. See [*Manage Data Source Configurations*](https://devnet.logianalytics.com/hc/en-us/articles/4405851027479-Manage-Data-Source-Configurations) |
| Visual Gallery | Select this option to review and manage stored visuals. See [*Manage Visuals*](https://devnet.logianalytics.com/hc/en-us/articles/4405851259671-Manage-Visuals). |
| Library | Select this option to review and manage dashboards on the dashboard list from anywhere in the product. See [*Manage Dashboards*](https://devnet.logianalytics.com/hc/en-us/articles/4405850985623-Manage-Dashboards). |
