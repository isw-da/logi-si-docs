# Composer Metadata Stores

# Composer Metadata Stores

Composer uses the following metadata stores:

| Store Name | Description |
| --- | --- |
| zoomdata | Stores Composer metadata. This is the primary metadata store for Composer. |
| zoomdata-config | Stores metadata associated with the Composer [configuration microservice](https://devnet.logianalytics.com/hc/en-us/articles/4405859473687-Use-the-Composer-Configuration-Microservice-to-Maintain-Application-Properties). This store name is also configurable. |
| zoomdata-keyset | Stores [keyset](https://devnet.logianalytics.com/hc/en-us/articles/4405859392151-Use-Keysets) data. |
| zoomdata-upload | Stores flat files that have been uploaded using the [flat file connector](https://devnet.logianalytics.com/hc/en-us/articles/4405850954647-Upload-a-Flat-File-into-Composer). |
