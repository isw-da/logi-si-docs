# Cautionary Note About Internal APIs

# Cautionary Note About Internal APIs

Composer offers a number of REST APIs so you can extend and embed Composer in your own custom applications. These APIs are denoted as publicly available using the
`/api/` path (for example, `www.yourcompany.com/composer/api/accounts`).
