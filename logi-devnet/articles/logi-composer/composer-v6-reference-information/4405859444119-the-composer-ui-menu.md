# The Composer UI Menu

# The Composer UI Menu

A user interface (UI) menu is available when you select the hamburger icon (![](https://devnet.logianalytics.com/hc/article_attachments/4406743720343/hamburger.png)) in the upper right corner of the page. The available menu options vary depending on whether you are logged in as a supervisor and on the group [privileges](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference) assigned to your user definition. This topic describes the menu options available on the regular Composer UI menu, when you are logged in as any non-supervisor user or as an administrator. For information about supervisor menu options, see [The Composer Supervisor Menu](https://devnet.logianalytics.com/hc/en-us/articles/4405859441687-The-Composer-Supervisor-Menu).

The Composer UI menu looks like this. Some options may not be available if your user definition does not have appropriate group [privileges](https://devnet.logianalytics.com/hc/en-us/articles/4405859114647-Group-Privilege-Reference) assigned.

![](https://devnet.logianalytics.com/hc/article_attachments/4406743774103/admin-menu_144x350.png)

Each menu option is described in the following table.

| Menu Option | Select this option to... |
| --- | --- |
| Connections | Review and manage the connection definitions used by Composer connectors. See [Manage Data Store Connections](https://devnet.logianalytics.com/hc/en-us/articles/4405850909079-Manage-Data-Store-Connections). |
| Sources | Review and manage Composer data source configurations. See [Manage Data Source Configurations](https://devnet.logianalytics.com/hc/en-us/articles/4405851027479-Manage-Data-Source-Configurations). |
| Visual Gallery | Review and manage Composer visuals. See [*Use the Visual Gallery*](https://devnet.logianalytics.com/hc/en-us/articles/4405859572119-Use-the-Visual-Gallery). |
| Library | Review and manage Composer dashboards in the dashboard library. See [*Use the Dashboard Library*](https://devnet.logianalytics.com/hc/en-us/articles/4405859273751-Use-the-Dashboard-Library). |
| Actions | Review and manage Composer action templates. See [Integrate Visual Data Into Your Applications](https://devnet.logianalytics.com/hc/en-us/articles/4405850870807-Integrate-Visual-Data-Into-Your-Applications). |
| Console | Review the Console of Refreshing Jobs. See [Review Refresh Jobs](https://devnet.logianalytics.com/hc/en-us/articles/4405851155095-Review-Refresh-Jobs). |
| Custom Charts | Manage custom Composer visuals from the UI. See [Manage Custom Charts](https://devnet.logianalytics.com/hc/en-us/articles/4405859248279-Manage-Custom-Charts). |
| Users & Groups | Manager Composer users and groups and to assign users to groups. See [Manage User Definitions](https://devnet.logianalytics.com/hc/en-us/articles/4405859120023-Manage-User-Definitions) and [Manage Group Definitions](https://devnet.logianalytics.com/hc/en-us/articles/4405850856599-Manage-Group-Definitions). |
| About | View the About dialog that identifies the version of Composer you are using and provides a link to the terms of service. See [About Dialog](https://devnet.logianalytics.com/hc/en-us/articles/4405851114903-About-Dialog). |
| Help | Access the Composer documentation. |
| My Accounts | Switch to a different Composer account. |
| Support | Access [Technical Support](https://devnet.logianalytics.com/hc/en-us/articles/4405859363351-Contact-Technical-Support). |
| Logout | Log out of Composer. |
