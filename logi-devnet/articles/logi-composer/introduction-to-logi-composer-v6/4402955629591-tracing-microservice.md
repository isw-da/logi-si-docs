# Tracing Microservice

# Tracing Microservice

The Composer tracing microservice provides some limited distributed tracing for Composer microservices. It is enabled by default and is automatically discovered by the Consul and Composer microservices.

For complete information, see [*Trace Composer Problems*](https://devnet.logianalytics.com/hc/en-us/articles/4402955568279-Trace-Composer-Problems).
