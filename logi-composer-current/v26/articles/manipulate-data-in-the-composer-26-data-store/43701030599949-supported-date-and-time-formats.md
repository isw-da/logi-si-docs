# Supported Date and Time Formats

Composer allows you to use number and attribute fields as time fields by creating a derived field you can use as time data. See [Convert Attributes to Time Fields in Data Source Field Specifications](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701078376461-Convert-Attributes-to-Time-Fields-in-Data-Source-Field-Specifications).
