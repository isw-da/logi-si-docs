# Arithmetic Functions

| Operator | Description |
| --- | --- |
| - (SUBTRACT) | Subtracts one numeric value from another |
| \* (MULTIPLY) | Multiply one number by another |
| / (DIVIDE) | Divide one number by another |
| + (ADD) | Addition: adds two numbers |
