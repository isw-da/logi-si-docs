# Home Page

When you [log in](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43700988311437-Log-Into-the-Logi-Composer-UI), the Logi Composer home page opens. Explore and interact with your Logi Composer environment, including its dashboards, visuals, and visualizations in a variety of ways.

This topic covers the different options and features available to you.

## View the Home Page

Easily access your home page from anywhere in the UI by selecting the Composer logo in the upper left corner or by selecting **Home** on the [top-level navigation banner](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701115577869-The-Top-Level-Navigation-Banner).

![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242472553997)

Depending on your application privileges, this home page displays the numbers of connection definitions, data source configuration definitions, visuals, and dashboards in the environment. It also provides easy access to lists of connection and data source configuration definitions, the Visual Gallery, and the dashboard library. The home page does not show information or provide links for resources that your application privileges restricts you from viewing. The following additional selections are possible.

| Category | Description |
| --- | --- |
| Connections | Shows the number of connections defined in your environment. Select this card to manage connections to data stores. The Connections page appears.  See [Manage Data Store Connections](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701038107149-Manage-Data-Store-Connections). |
| Sources | Shows the number of data source configurations defined in your environment. Select this card to manage data source configurations. The Sources page appears.  See [Manage Data Sources](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701068934669-Manage-Data-Sources). |
| Visual Gallery | Shows the number of visuals defined in your environment. Select this card to manage visuals in the Visual Gallery. The Visuals page appears.  See [Manage Visuals](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701171638925-Manage-Visuals) and [Use the Visual Gallery](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701180261773-Use-the-Visual-Gallery). |
| Library | Shows the number of dashboards defined in your library. Select the **Dashboards** card to manage dashboards. The dashboard library appears.  See [Manage Dashboards](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701029082893-Manage-Dashboards) and [Use the Library for Dashboards](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701047640717-Use-the-Library-for-Dashboards). |

In addition to these selections, the user name you logged in with is shown in this work area. You can select the included hamburger icon (![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243217062157)) to access the [UI menu](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701160499853-The-Composer-UI-Menu).
