# Manage User Groups

Use groups to assign [privileges](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701005611789-Group-Privilege-Reference) to groups of users. Groups are most useful when a number of users require the same access restrictions. Users can be assigned to multiple groups.

**Note:** SAML and LDAP groups that are automatically created in Composer must be manually assigned privileges.

A Composer system administrator, tenant administrator, or a user who has been assigned to a group with [group management privileges](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701005611789-Group-Privilege-Reference) can manage groups. They can:

* Add, edit, or remove groups
* Assign and remove users in a group
* Authorize users in the group to perform specific functions

If your user account is not assigned the **Administer Groups** privilege or is not an administrator, you cannot assign groups to a user. In addition, only administrators can assign users to the **Administrators** group.

For more info about managing users, see [Manage Users](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701051820941-Manage-Users).
