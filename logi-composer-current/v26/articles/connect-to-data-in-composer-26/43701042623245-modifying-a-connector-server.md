# Modifying a Connector Server

The only way to modify a connector server on the Manage Connector Services page is to delete and re-register the connector server. See [Delete a Connector Server](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701039168653-Delete-a-Connector-Server), [Obtain Additional Connector Servers](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701024898061-Obtain-Additional-Connector-Servers), and [Register a New Connector Server](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701042690957-Register-a-New-Connector-Server).
