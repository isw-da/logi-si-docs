# Change the Axes

Many visual styles show data on a standard coordinate grid, with axes labels identifying the values depicted horizontally and vertically on the visual.
Examples include [bar charts](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701165163405-Bar-Chart-Styles), [line charts](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701166096269-Line-Charts), and [scatter charts](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701213904781-Scatter-Bubble-Charts).

Use the axes labels on these visuals to change the metric (y-axis) and attribute (x-axis) fields graphed in the visual. The number of labels displayed on a visual depend on the [number of metrics and attributes](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701115864205-Composer-Visual-Metrics-and-Attributes-Reference) on which the visual is based.

![select an axis label to change the field used in this visual](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242355466637 "Axes Labels on a bar chart")

You can specify the default labels for visuals on the [Fields tab](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701116424973-Manage-Fields) of your data source configurations. Controls for these options are provided using the interactivity sidebar. See [Control How Users Interact With a Visual](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701185104653-Control-How-Users-Interact-With-a-Visual).

See the following topics:

* [Change a Group Attribute](#Changing)
* [Change a Metric Field](#Changing2)

You can also change the metric used to determine the colors on the visual. See [Change the Visual Color Metric](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701165465741-Change-the-Visual-Color-Metric).

## Change a Group Attribute

You can change the x-axis field, or group attribute, on a visual that plots data on a coordinated grid.

**Change the visual group (x-axis) attribute**

1. Select the group attribute (x-axis) label on your visual. A Group dialog appears.

   ![select a group attribute for the visual](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242324197645 "Group Attribute work area")
2. Select a new field to be viewed on the x-axis of your visual.

   The visual renders the newly selected attribute.

## Change a Metric Field

You can change the y-axis field, or metric, on a visual that plots data on a coordinated grid.

**Change the visual metric (y-axis field)**

1. Select the metric (y-axis) label on your visual. A Metric dialog appears.

   ![Select a metric for this visual](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242324343053 "Metric Selection")
2. Select a new metric to use as the y-axis of your visual and a metric function to use to aggregate the data on the visual. See [Metric Aggregation Functions](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701068532493-Metric-Aggregation-Functions).

   The visual is updated and renders the newly selected metric. If you need to perform more complex analysis of your data set, you can
   [add custom metrics](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701030223373--Maintain-Custom-Metrics).
