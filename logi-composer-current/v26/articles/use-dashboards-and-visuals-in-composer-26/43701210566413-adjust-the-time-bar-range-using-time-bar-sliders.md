# Adjust the Time Bar Range Using Time Bar Sliders

**Adjust the time bar time range for a visual using the time bar sliders**

1. Select the visual.
2. Slide the From slider (![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242394616717)) on the time bar to the earliest time for the time range you want.
3. Slide the To slider (![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242363051149)) on the time bar to the latest time for the time range you want.
4. Save the dashboard.

To select the maximum range of values, select ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243138309901) on the time bar.
