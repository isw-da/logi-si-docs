# Add Rich Text Snippets to a Dashboard

When you create a dashboard or edit an existing dashboard, you can add rich text snippets to provide context to the visuals for your users.

You can fully author your own rich text snippet, or have Logi Composer generate visual description and add it to your dashboard in a rich text snippet.

**Note:** 
When you create a rich text snippet, it is linked to the dashboard you've added it to. If you delete a dashboard, the snippet is deleted as well.

**Create a** **blank rich text snippet**

1. [Create](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701076467981-Create-Dashboards) or [edit](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701014144397-Edit-a-Dashboard) a dashboard.

   * To add a snippet to an existing dashboard, log in as a user with `READ` and `WRITE` permissions for the dashboard.
   * If you are creating a new dashboard, log in as a user with the **Create Dashboards** or **Administer Dashboards** [group privilege](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701005611789-Group-Privilege-Reference).
2. Select ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242365598221) on the [dashboard icon bar](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701047597837-Use-the-Dashboard-Icon-Bars). A blank, new rich text snippet is added in a widget on the dashboard, ready to edit.
3. Add text or other information in the snippet, using the [snippet format tools](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701176415117-Format-Rich-Text-Snippets) to adjust the look and layout of your text, or add images and links.

   As you add and update your text, use the undo and redo icons or keyboard shortcuts to undo and redo text changes. If you employ custom attributes in your Logi Composer environment, incorporate them into your snippets as needed.
4. When you've added the information you need, select the edit icon (![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243171052685)) to hide the format tools and put the snippet in view mode.
5. [Save](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701214973197-Save-Visuals-With-Their-Current-Names) the dashboard.

**Generate and insert****a rich text snippet with a visual description**

1. [Create](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701076467981-Create-Dashboards) or [edit](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701014144397-Edit-a-Dashboard) a dashboard with one or more visuals.

   * To add a snippet to an existing dashboard, log in as a user with `READ` and `WRITE` permissions for the dashboard.
   * If you are creating a new dashboard, log in as a user with the **Create Dashboards** or **Administer Dashboards** [group privilege](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701005611789-Group-Privilege-Reference).
2. Select the [visual drop-down menu](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701184743565-Use-the-Visual-Menu#DescVisual), and choose **Describe Visual > Create Snippet and Insert**. A new rich text snippet is added in a widget on the dashboard with the text of the visual description, ready to edit. Other options include:

   * **Describe Visual > Copy Visual Summary**: This copies the summary to your clipboard. You can add it to an existing rich text snippet, or use it outside of the Logi Composer environment as needed.
   * **Describe Visual > Insert into [Existing Snippet Name]**: This inserts the description into the selected snippet.

     **Note:** If more than one rich text snippet widget exists, you can select the appropriate snippet from a list.
3. Add more text or other information in the snippet, using the [snippet format tools](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701176415117-Format-Rich-Text-Snippets) to adjust the look and layout of your text, or add images and links.

   As you add and update your text, use the undo and redo icons or keyboard shortcuts to undo and redo text changes. If you employ custom attributes in your Logi Composer environment, incorporate them into your snippets as needed.
4. When you've added the information you need, select the edit icon (![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243171052685)) to hide the format tools and put the snippet in view mode.
5. [Save](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701214973197-Save-Visuals-With-Their-Current-Names) the dashboard.

**Note:** 
Visual Description is [a menu option](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701184743565-Use-the-Visual-Menu#DescVisual) available for most, but not all visuals.

**Note:** Any changes you make to a visual will be reflected in the generated description after you save your work with the updates.
