# Select Columns Using the Table Context Menu

**Select fields for table columns using the table context menu**

1. View the table visual in a dashboard or from the Visual Gallery.
2. Select ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243146377997) next to the column heading name to access the table context menu.

   ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243146379661)
3. Select the ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242348235789) tab on the context menu. A list of all the fields in the data source appears.

   ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242330505997)
4. Select the fields you want to view as columns in the table. To select all the fields, select the checkbox to the left of the search bar.

   Use the search bar to search for a field in the table.

   Fields you select appear in the table.
5. Save the dashboard and visual.
