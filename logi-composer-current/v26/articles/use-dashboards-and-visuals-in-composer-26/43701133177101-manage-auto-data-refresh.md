# Manage Auto Data Refresh

You can use the **Manage Auto Data Refresh** toggle for filters to pause the default behavior of some dashboards to automatically refresh the data when you make filter changes.

If you are working with a large data set from a single source, you can disable auto data refresh in the filter snippets to make changes. With auto data refresh disabled, make your filter and filter snippet changes quickly and easily, then enable it again when you are ready to see the results of filters on refreshed data.

The **Publish Filters Settings** icon is available in dashboards that:

* Include visuals linked through at least one filter snippet
* Use a single source for data

## Enable and Disable Auto Data Refresh

1. Open a dashboard that includes several filter snippets, from one data source, for editing or viewing. Select the **Publish Filters Settings** icon in the group of available dashboard icons.
2. A **Published Filters Setting** work area opens, listing the **Published** and **Unpublished Filters** applied to the dashboard, visuals, and filter snippets in your dashboard.
3. Disable the **Auto Apply Filters** setting. This prevents Composer from refreshing data in your filter snippets from your source when you make filter changes.
4. Make your filter changes as desired.
5. Select **Apply** to refresh your filter snippet data and view your results. Alternatively, enable **Auto Apply Filters** again to view your results and apply filter changes (if applicable).
