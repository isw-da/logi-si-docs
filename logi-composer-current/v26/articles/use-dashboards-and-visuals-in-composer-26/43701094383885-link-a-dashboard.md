# Link a Dashboard

While working with your dashboard, you may need to analyze data from other dashboards.
You can link a visual from your dashboard to another dashboard. This allows you to quickly access other dashboards with related information. To link dashboards, you must have permissions to share and link dashboards. When linking one dashboard to another, you can elect to have the target dashboard inherit the filters from the source dashboard.

**Link a dashboard visual to another dashboard**

1. Open the dashboard you want to link.
2. Select dashboard links ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243273154061) on the [dashboard icon bar](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701047597837-Use-the-Dashboard-Icon-Bars). The Dashboard Links dialog appears.

   ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242627876365)
3. Select **Add Link**. The Select Link dialog opens.

   ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242688939789)
4. Select a visual and dashboard to link, then select **Apply**. The Dashboard Link dialog opens, showing the visual and dashboard links.

   ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242689069581)
5. By default, **Inherit Visual Filter** is enabled, and the dashboard passes the filters currently used on the dashboard (or the filters applied based on what is selected when the dashboard link occurs) to the linked dashboard. If you do not want the filters from the dashboard carried over, disable **Inherit Visual Filter**. The filters are only carried over if the dashboards are using the same data source configuration.
6. Close the dialog.
7. [Save](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701047225613-Save-a-Dashboard) the dashboard to save the dashboard link settings.
8. On the [visual drop-down menu](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701184743565-Use-the-Visual-Menu) for the linked visual, a new option in the following format appears:

   ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242689169677)**Go to "<dashboard>"**

   Select this option to jump to the linked dashboard.

**Navigate to a linked dashboard**

* Select the ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242689169677)**Go to "<dashboard>"** link on the [visual drop-down menu](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701184743565-Use-the-Visual-Menu).
* You can also navigate to the linked dashboard by selecting **LINK** on the dashboard [context menu](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701212059021-Use-the-Context-Menu).

**Delete a dashboard link**

1. Open the dashboard containing the visual you want to unlink.
2. Select ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243273154061) on the [dashboard icon bar](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701047597837-Use-the-Dashboard-Icon-Bars). The Dashboard Links dialog appears showing the links to other dashboards.

   ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242689069581)
3. On the Dashboard Links dialog, select the visual and select the remove (![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242659750541)) icon. The link is removed.

   You must remove all links to a dashboard before you can delete the dashboard.
4. [Save](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701047225613-Save-a-Dashboard) the dashboard to save the dashboard link settings.
