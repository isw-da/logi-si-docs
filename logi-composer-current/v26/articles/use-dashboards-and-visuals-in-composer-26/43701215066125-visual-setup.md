# Visual Setup

# Visual Setup

You can control visuals available for each data source configuration you create using the Available Visual Types work area. See [Available Visual Types](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701177930509-Available-Visual-Types).

After saving a data source definition, adjust settings applicable to new visuals on the [Global Settings Tab](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701085233549-Global-Settings-Tab).

To add a visual, see:

* [Create Dashboards](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701076467981-Create-Dashboards)
* [Create and Add Visuals to the Visual Gallery](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701179894541-Create-and-Add-Visuals-to-the-Visual-Gallery)
* [Add Visuals to a Dashboard](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701214171661-Add-Visuals-to-a-Dashboard)
* [Add Local Visuals to a Dashboard](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701214222477-Add-Local-Visuals-to-a-Dashboard)
* [Add Existing Visuals to a Dashboard](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701184486797-Add-Existing-Visuals-to-a-Dashboard)

After you create a visual, you can alter its settings. See [The Visual Interface](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701180347661-The-Visual-Interface).
