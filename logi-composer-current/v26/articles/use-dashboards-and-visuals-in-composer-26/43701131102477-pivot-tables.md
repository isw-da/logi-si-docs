# Pivot Tables

# Pivot Tables

Pivot tables
are tables of summarized statistics. They are useful for finding unique values for a field. If needed, you can [add custom metrics](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701030223373--Maintain-Custom-Metrics) to your pivot table metrics.

Pivot tables are supported by all Composer [data connectors](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701113495693-Data-Connector-Reference) except Cloudera Search. They are supported by Composer Apache Solr connectors for version 5.2 or later.

When you select a table metric, the context menu appears. See [Use the Context Menu](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701212059021-Use-the-Context-Menu).

When you select the table context menu (![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243123086605)) in a column header, you can [sort your data](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701212800013-Sort-Data-in-a-Pivot-Table), autosize your columns, show or hide metric labels, and collapse or expand all rows for hierarchies.

After you create a pivot table, you can alter the rows, columns, metrics and metric layouts, and horizontal scroll bar used in a pivot table. In addition, you can sort the data in the table and rearrange the table fields. See the following topics:

* [Configure Settings for a Specific Pivot Table](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701178651149-Configure-Settings-for-a-Specific-Pivot-Table)
* [Modify Pivot Table Rows](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701212750349-Modify-Pivot-Table-Rows)
* [Modify Pivot Table Columns and Column Widths](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701178611981-Modify-Pivot-Table-Columns-and-Column-Widths)
* [Modify Pivot Table Metrics and Metric Layout](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701166265357-Modify-Pivot-Table-Metrics-and-Metric-Layout)
* [Rearrange Pivot Table Fields](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701178558093-Rearrange-Pivot-Table-Fields)
* [Sort Data in a Pivot Table](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701212800013-Sort-Data-in-a-Pivot-Table)

**Note:** 
You can use [Keyboard Controls](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701212342157-Keyboard-Controls) on the Pivot Table sidebar instead of a mouse.
