# Table Context Menu

# Table Context Menu

The column headings of a table and pivot table include a context menu you can use to perform various functions on the table, as described below. Select ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243134960781) next to the column heading name to access the table context menu. The context menu has two tabs:

* The ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243134960781) tab lists the functions you can perform.
* The ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243134963085) tab lists all the available fields in the data source.

![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243134963853)![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242302812813)![select to format numbers in a table column](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243134964109 "table context menu with number formatting") or ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243151369869)![select show and hide metric or timezone options](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242330817037 "select show and hide metric or timezone options")![select show and hide metric or timezone options](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242348816781 "select show and hide metric or timezone options")

Possible menu options on the table context menu tab are described below.

| Menu Option | Description |
| --- | --- |
| Aggregation | Shown only for numeric fields or metrics, this option allows you to change the aggregation method for the field.  Valid options are AVG, MIN, MAX, SUM, NONE, or LAST VALUE. See [Change Metric Aggregation in Tables](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701183230989-Change-Metric-Aggregation-in-Tables). |
| Autosize all columns | This option autosizes all the column widths in the table.  See [Change Column Widths](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701178773005-Change-Column-Widths). |
| Format | Format data or aggregated data in this column. See [Format Numeric Table Data Using the Table Context Menu](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701213266061-Format-Numeric-Table-Data-Using-the-Table-Context-Menu). |
| Format <field> | Use this option to modify the formatting of numeric and time fields for Table visuals.  See [Format Numeric Table Data Using the Table Context Menu](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701213266061-Format-Numeric-Table-Data-Using-the-Table-Context-Menu) or [Format Time Table Data Using the Table Context Menu](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701213292429-Format-Time-Table-Data-Using-the-Table-Context-Menu). |
| Format <field> as URL | Format the data in this column as a URL or image. See [Format Images or a URL Using the Table Context Menu](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701183720973-Format-Images-or-a-URL-Using-the-Table-Context-Menu). |
| Group by <field> | This option allows you to group the table by the field represented by the table column.  See [Group and Ungroup Table Data](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701183802381-Group-and-Ungroup-Table-Data). |
| Granularity | Shown only for time fields used to group the table, this option allows you to change the granularity of the grouped time field. Options vary based on the [granularity you define](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701095259917-Configure-Date-and-Time-Formatting-Data-Sources) at the source level.  See [Change Time Field Granularity in Tables](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701183936013-Change-Time-Field-Granularity-in-Tables). |
| Hide/Show | * Shown in columns using aggregation, this allows you to hide or show the label of the metric used to aggregate your data. * Shown in columns that include time data, this allows you to hide or show the time zone label. |
| Include Blanks | Shown only for time fields used to group the table, this option allows you apply even time intervals for the grouped time field.  Select this option once to apply even time intervals.  Select it a second time to remove even time intervals.  See [Even Time Intervals](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701077393293-Even-Time-Intervals). |
| Reset <field> sorting | Shown after you have sorted a column. Select to reset the sort you performed on the unsaved table. See[Sort Data in a Table](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701213550221-Sort-Data-in-a-Table) or [Sort Data in a Pivot Table](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701212800013-Sort-Data-in-a-Pivot-Table). |
| Reset all sorting | Shown after you have sorted one or more columns. Select to reset all sorting you performed on the unsaved table. See[Sort Data in a Table](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701213550221-Sort-Data-in-a-Table) or [Sort Data in a Pivot Table](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701212800013-Sort-Data-in-a-Pivot-Table). |
| Resort other sorting | Shown after you have sorted one or more columns, but not this column. Select to reset all sorting you performed on the unsaved table. See[Sort Data in a Table](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701213550221-Sort-Data-in-a-Table) or [Sort Data in a Pivot Table](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701212800013-Sort-Data-in-a-Pivot-Table). |
| Ungroup <field> | This option removes grouping by the table column. See [Group and Ungroup Table Data](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701183802381-Group-and-Ungroup-Table-Data). |
| Wrap Column | Select to wrap the text or data in a column to improve readability. |

You can also select and hide a column from a table using the ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243134963085) tab on the table context menu. See [Select Columns Using the Table Context Menu](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701213162253-Select-Columns-Using-the-Table-Context-Menu) and [Hide Columns Using the Table Context Menu](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701178928653-Hide-Columns-Using-the-Table-Context-Menu).
