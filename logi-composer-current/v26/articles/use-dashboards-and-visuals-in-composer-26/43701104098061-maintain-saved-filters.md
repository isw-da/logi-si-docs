# Maintain Saved Filters

After a filter is created and applied to a visual or filter snippet, it can be saved and applied to other visuals or filter snippets that use the same data source. A saved filter can include the settings of one or more filters.

The following topics describe how to maintain a saved filter.

* [Create a Saved Filter](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701133760653-Create-a-Saved-Filter)
* [Edit a Saved Filter](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701071735693-Edit-a-Saved-Filter)
* [Delete a Saved Filter](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701119232141-Delete-a-Saved-Filter)
