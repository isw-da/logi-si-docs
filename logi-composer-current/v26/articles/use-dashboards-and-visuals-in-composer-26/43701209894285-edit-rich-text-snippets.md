# Edit Rich Text Snippets

Edit rich text snippets quickly and easily in a dashboard. Update links, reformat text, or add images as needed.

**Edit a rich text snippet**

1. Select the snippet on the dashboard. A blue border appears around the snippet widget.
2. Select the edit pencil (![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242365113485)) in the widget, or ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242376687117)**Edit** from the drop-down menu (![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243177601293)). The rich text snippet opens for editing.
3. Make your content and [format](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701176415117-Format-Rich-Text-Snippets) changes.
4. When you have finished making changes to the snippet, select the edit pencil (![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242365113485)) to hide the format tools and put the snippet in view mode.
5. [Save](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701214973197-Save-Visuals-With-Their-Current-Names) the dashboard.
