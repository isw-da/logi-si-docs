# Hide Columns Using the Table Context Menu

**Hide table columns using the table context menu**

1. View the table visual in a dashboard or from the Visual Gallery.
2. Select ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243146543373) next to the column heading name to access the table context menu.

   ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243131994637)
3. Select the ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46243146544141) tab on the context menu. A list of all the fields in the data source appears.

   ![](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242348444557)
4. Clear (uncheck) the fields you want to hide in the table.

   Use the search bar to search for a field in the table.

   Fields you hide disappear from the table.
5. Save the dashboard and visual.
