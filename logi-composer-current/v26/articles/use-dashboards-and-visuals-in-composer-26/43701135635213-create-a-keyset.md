# Create a Keyset

You can create a keyset in several ways. Keysets are shared by all users of your Composer instance.

* [Create a Keyset From the Visual Result Set](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701135587597-Create-a-Keyset-From-the-Visual-Result-Set)
* [Create a Keyset From a Data Point](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701073474701-Create-a-Keyset-From-a-Data-Point)
* [Create a Keyset From a CSV File](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701105830669-Create-a-Keyset-From-a-CSV-File)
* [Update Keyset Values From a CSV File Using the UI](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701121159181-Update-Keyset-Values-From-a-CSV-File-Using-the-UI)

To control whether keysets can be created or managed from a visual, use the interactivity sidebar. See [Control How Users Interact With a Visual](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701185104653-Control-How-Users-Interact-With-a-Visual).
