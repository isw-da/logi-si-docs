# Use the Dashboard Icon Bars

When you create or edit a dashboard, a series of icons are available you can use to perform specific dashboard functions.

![manage your dashboard layout and enable view mode](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242674159373 "dashboard icons")![options available for dashboards for sharing, exporting, adding widgets and more](https://logi-composer-v26.insightsoftware.com/hc/article_attachments/46242674278541 "dashboard icons bar")

Select an icon to perform a dashboard function, as described in the following table.

**Note:** 
The visibility of these icons on a dashboard are affected by dashboard interactivity settings. See [Control How Users Interact With a Dashboard](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701076696717-Control-How-Users-Interact-With-a-Dashboard).

| Icon | Description |
| --- | --- |
|  | Filter the data on a dashboard. Available only when all visuals on a dashboard are from the same data source. |
|  | Open dashboard and visual interactivity settings. See [Control How Users Interact With a Dashboard](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701076696717-Control-How-Users-Interact-With-a-Dashboard). |
|  | Select to open the Dashboard Layout work area. |
|  | Enable and disable the toggle to control widget locking in this dashboard. See [Lock and Unlock Widget Positions](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701029061261-Lock-and-Unlock-Widget-Positions). |
|  | Enable and disable the toggle to control responsive layout for this dashboard. See [Using the Responsive Dashboard Layout](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43700998041357-Using-the-Responsive-Dashboard-Layout). |
|  | Set the minimum widget width and height. See [Using the Responsive Dashboard Layout](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43700998041357-Using-the-Responsive-Dashboard-Layout). |
|  | Toggle dashboard view mode between Viewer and Editor modes. See [Use Dashboard View Mode](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701014828173-Use-Dashboard-View-Mode). |
|  | Search visuals content in the dashboard, if supported by the source. |
|  | Manage alerts. See [Manage Alerts](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701052718605-Manage-Alerts) |
|  | Link fields between disparate data sources to create cross-source links. See [Use Cross-Source Links](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701012571789-Use-Cross-Source-Links). |
|  | Link dashboard visuals to other dashboards. See [Link a Dashboard](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701094383885-Link-a-Dashboard). |
|  | Export a dashboard. The Export drop-down menu appears. See [Export Dashboards](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701076601229-Export-Dashboards). |
|  | Share a dashboard. The Share dashboard work area opens. See [Share a Dashboard with Users](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701077162637-Share-a-Dashboard-with-Users). |
|  | Schedule dashboard reports. The Scheduled Reports work area opens. See [Schedule a Dashboard Report](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701093791245-Schedule-a-Dashboard-Report). |
|  | Refresh the data on the dashboard. See [Refresh Data on a Dashboard](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701093307533-Refresh-Data-on-a-Dashboard). |
|  | Mark the dashboard as a favorite. |
|  | Select to toggle the searchable information about this dashboard as visible in the dashboard header. When visible, you can select the description and edit it. |
|  | Select to add a new local visual to the dashboard or add a Visual Gallery visual to the dashboard. See [Add Visuals to a Dashboard](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701214171661-Add-Visuals-to-a-Dashboard) and [Add Existing Visuals to a Dashboard](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701184486797-Add-Existing-Visuals-to-a-Dashboard). |
|  | Add a rich text snippet to the dashboard. See [Add Rich Text Snippets to a Dashboard](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701209714957-Add-Rich-Text-Snippets-to-a-Dashboard). |
|  | Add a filter snippet to the dashboard. See [Add Filter Snippets to a Dashboard](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701071841933-Add-Filter-Snippets-to-a-Dashboard). |
|  | Delete the dashboard. See [Delete a Dashboard](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701076525453-Delete-a-Dashboard). |
|  | Save a dashboard with a new name (which copies it). See [Copy a Dashboard](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701076444557-Copy-a-Dashboard). |
|  | Save a dashboard. See [Save a Dashboard](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701047225613-Save-a-Dashboard). |
