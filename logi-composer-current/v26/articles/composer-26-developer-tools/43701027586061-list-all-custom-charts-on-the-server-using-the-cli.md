# List All Custom Charts on the Server Using the CLI

**List all custom charts stored on the** Composer **server using the Custom Chart CLI**

Run the following command:

$ cmp-chart ls

**Note:** Composer v24 Custom Chart CLI version 1 only supports the use of `vnd.composer.v3+json` as the `Content-Type` for API routes.
