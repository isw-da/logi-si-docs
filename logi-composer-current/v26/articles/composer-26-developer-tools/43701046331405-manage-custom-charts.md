# Manage Custom Charts

Administrators can create custom charts using the custom chart CLI. Download, import, and delete custom charts in the user interface (UI). Custom charts can also be made visible on various menus in the Composer user interface (UI). By default, custom charts (visual types) are not visible until enabled for a source. See [Available Visual Types](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701177930509-Available-Visual-Types).

**Note:** 
You must be an administrator to manage custom visual types.

* [Maintain Custom Charts Using the Custom Chart CLI](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701075906189-Maintain-Custom-Charts-Using-the-Custom-Chart-CLI)
* [Manage Custom Charts in the UI](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701028167053-Manage-Custom-Charts-in-the-UI)
* [Visual Type Configuration Properties](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701046187277-Visual-Type-Configuration-Properties)
* [Custom Chart API](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43700991619085-Custom-Chart-API)

A step-by-step tutorial for creating a custom chart is also included in [A Custom Chart Tutorial](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701028307597-A-Custom-Chart-Tutorial).
