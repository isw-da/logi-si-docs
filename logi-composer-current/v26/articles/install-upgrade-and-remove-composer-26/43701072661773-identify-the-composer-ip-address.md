# Identify the ComposerIP Address

After you have installed Composer and configured the [firewall](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701134548877-Configure-the-Firewall), you need to identify the Composer IP address so you can access Composer on your web browser. Record the IP address of the host where Composer is installed.

To
obtain the Composer IP address, enter the following command in a terminal window on the Composer server:

[zoomdata@localhost ~]$ hostname -I

Make note of this IP address. You will need it when you [access Composer](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701052048397-Access-Logi-Composer) from a web browser.
