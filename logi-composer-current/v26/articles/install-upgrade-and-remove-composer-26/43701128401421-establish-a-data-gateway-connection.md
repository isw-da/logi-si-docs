# Establish a Data Gateway Connection

Once you or your system administrator has [enabled the data gateway](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701209528205-Set-Up-and-Use-the-Data-Gateway-Service#Enable) and [added a connector](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43700993403661-Define-a-New-Connector), you can make a [connection](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43700992503437-Add-Data-Store-Connections) your users can select when creating a new data source.

Users will see the new connector as an option during source creation, and can simply select it as they would any other source. See [Define a Source](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701080450061-Define-a-Source).

**Note:** You are not prompted for credentials when you create a connection: they were provided as part of defining the connector.
