# Running Composer in Kubernetes

Use Kubernetes to mange your Composer configuration management tasks. This scalable solution allows you deploy your Composer environment securely and scale all microservices as needed.

**Important:** 
Kubernetes is supported for new installations only for Composer version 23.1 and later.
