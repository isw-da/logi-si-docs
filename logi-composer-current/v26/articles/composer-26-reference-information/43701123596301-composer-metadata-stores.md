# Composer Metadata Stores

# Composer Metadata Stores

Composer uses the following metadata stores:

| Store Name | Description |
| --- | --- |
| zoomdata | Stores Composer metadata. This is the primary metadata store for Composer. |
| zoomdata-config | Stores metadata associated with the Composer [configuration microservice](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701148945549-Use-the-Composer-Configuration-Microservice-to-Maintain-Application-Properties). This store name is also configurable. |
| zoomdata-keyset | Stores [keyset](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701105959565-Use-Keysets) data. |
| zoomdata-upload | Stores flat files that have been uploaded using the [flat file connector](https://logi-composer-v26.insightsoftware.com/hc/en-us/articles/43701044052621-Manage-File-Uploads). |
