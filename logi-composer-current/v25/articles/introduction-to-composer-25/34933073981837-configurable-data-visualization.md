# Configurable Data Visualization

# Configurable Data Visualization

You can visualize data from your data stores in visuals and dashboards in the Composer UI. A dashboard is a container for one or more visuals. The visuals in a dashboard do not have to be related, but for easier data analysis, probably should be. See [Multisource Analysis](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933104442765-Multisource-Analysis).

Data can be visualized using a wide variety of visual styles: arc gauges, bar charts, box plots, donuts, heat maps, KPI charts, line charts, maps, pie charts, scatter charts, tables, tree maps, and word clouds are all supported. Variations of these visual styles can also be produced. For example, you can produce standard bar charts, stacked bar charts, clustered bar charts, histograms, and multiple metric bar charts. See the [Composer  Visual Metrics and Attributes Reference](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933120619149-Composer-Visual-Metrics-and-Attributes-Reference).

You can customize visuals in a variety of ways. When you define a data source configuration, you can specify some default global settings for visuals created using the data source. Read about the [Global Settings Tab](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932890843277-Global-Settings-Tab) of a data source configuration.

After a visual is created, you and add it to a dashboard, and customize it in the following ways. Some of these customizations depend on the selected visual style or the data source used for the visual.

| Customization | Description |
| --- | --- |
| Visual Colors | You can change the colors used in the visual. See [Change Color Schemes](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933242767245-Change-Color-Schemes) and [Change the Visual Color Metric](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933197872013-Change-the-Visual-Color-Metric). |
| Visual Style | You can change the visual style. See [Change the Visual Style](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933220133645-Change-the-Visual-Style). |
| Names and Descriptions | You can change the title and description of dashboards and visuals. See [Rename a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932792285965-Rename-a-Dashboard) ,[Descriptions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933196391437-Descriptions) , [Modify Visual Names, Display Names, and Descriptions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933236861197-Modify-Visual-Names-Display-Names-and-Descriptions). |
| Data Sorting | You can sort visual data. See [Sort and Limit Visual Data](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933301977613-Sort-and-Limit-Visual-Data). |
| Data Filtering | You can filter visual data by specific field values or ranges. See [Filter Data](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933000389517-Filter-Data). |
| You can filter visual and dashboard data by time ranges using the time bar. See [Use the Time Bar](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933204115725-Use-the-Time-Bar) |
| You can filter visual data by unique data values stored in a keyset. See [Use Keysets](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933047406605-Use-Keysets). |
| You can filter dashboard data by cross-source linked fields from different data sources. See [Use Cross-Source Links](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932758491405-Use-Cross-Source-Links). |

In addition to these dashboard and visual configuration features, you can link dashboards and you can share dashboards with other users. You can also export visuals and dashboards in a variety of formats: PNG, PDF, CSV (visuals), and JSON (dashboards and visuals). When you export visuals and dashboards in JSON format, you can impor them into other Composer environments running the same version of Composer. See [Manage Dashboards](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932808466829-Manage-Dashboards) and [Manage Visuals](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933295194637-Manage-Visuals).

Finally, you can change the look of the UI altogether using themes. See[Manage UI Themes](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933172226573-Manage-UI-Themes).
