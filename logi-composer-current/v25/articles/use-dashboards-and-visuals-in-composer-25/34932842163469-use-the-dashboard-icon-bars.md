# Use the Dashboard Icon Bars

When you create or edit a dashboard, a series of icons are available you can use to perform specific dashboard functions.

![manage your dashboard layout and enable view mode](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167324526349 "dashboard icons")![options available for dashboards for sharing, exporting, adding widgets and more](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167346506253 "dashboard icons bar")

Select an icon to perform a dashboard function, as described in the following table.

**Note:** 
The visibility of these icons on a dashboard are affected by dashboard interactivity settings. See [Control How Users Interact With a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932814650893-Control-How-Users-Interact-With-a-Dashboard).

| Icon | Description |
| --- | --- |
|  | Filter the data on a dashboard. Available only when all visuals on a dashboard are from the same data source. |
|  | Open dashboard and visual interactivity settings. See [Control How Users Interact With a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932814650893-Control-How-Users-Interact-With-a-Dashboard). |
|  | Select to open the Dashboard Layout work area. |
|  | Enable and disable the toggle to control widget locking in this dashboard. See [Lock and Unlock Widget Positions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932815023885-Lock-and-Unlock-Widget-Positions). |
|  | Enable and disable the toggle to control responsive layout for this dashboard. See [Using the Responsive Dashboard Layout](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932815319437-Using-the-Responsive-Dashboard-Layout). |
|  | Set the minimum widget width and height. See [Using the Responsive Dashboard Layout](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932815319437-Using-the-Responsive-Dashboard-Layout). |
|  | Toggle dashboard view mode between Viewer and Editor modes. See [Use Dashboard View Mode](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932832240909-Use-Dashboard-View-Mode). |
|  | Search visuals content in the dashboard, if supported by the source. |
|  | Manage alerts. See [Manage Alerts](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932655921293-Manage-Alerts) |
|  | Link fields between disparate data sources to create cross-source links. See [Use Cross-Source Links](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932758491405-Use-Cross-Source-Links). |
|  | Link dashboard visuals to other dashboards. See [Link a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932833747469-Link-a-Dashboard). |
|  | Export a dashboard. The Export drop-down menu appears. See [Export Dashboards](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932788789645-Export-Dashboards). |
|  | Share a dashboard. The Share dashboard work area opens. See [Share a Dashboard with Users](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932824740749-Share-a-Dashboard-with-Users). |
|  | Schedule dashboard reports. The Scheduled Reports work area opens. See [Schedule a Dashboard Report](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932823729293-Schedule-a-Dashboard-Report). |
|  | Refresh the data on the dashboard. See [Refresh Data on a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932805012877-Refresh-Data-on-a-Dashboard). |
|  | Mark the dashboard as a favorite. |
|  | Select to toggle the searchable information about this dashboard as visible in the dashboard header. When visible, you can select the description and edit it. |
|  | Select to add a new local visual to the dashboard or add a Visual Gallery visual to the dashboard. See [Add Visuals to a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933303418509-Add-Visuals-to-a-Dashboard) and [Add Existing Visuals to a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933272990989-Add-Existing-Visuals-to-a-Dashboard). |
|  | Add a rich text snippet to the dashboard. See [Add Rich Text Snippets to a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933155337229-Add-Rich-Text-Snippets-to-a-Dashboard). |
|  | Add a filter snippet to the dashboard. See [Add Filter Snippets to a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933003207309-Add-Filter-Snippets-to-a-Dashboard). |
|  | Delete the dashboard. See [Delete a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932790859021-Delete-a-Dashboard). |
|  | Save a dashboard with a new name (which copies it). See [Copy a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932804731533-Copy-a-Dashboard). |
|  | Save a dashboard. See [Save a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932815496845-Save-a-Dashboard). |
