# Zoom Into a Time Bar Range

**Zoom into a time bar range**

1. Select a visual on the dashboard.
2. Select a time range for the time bar. See [Adjust the Time Bar Range](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933203901965-Adjust-the-Time-Bar-Range).
3. Select zoom icon ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167827131405) on the right side of the time bar to zoom into the selected range. Only the times for the time range you selected are shown on the time bar.

   To view all the available times in the data, select all icon ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167036511117) on the time bar. To select the maximum range of values, select maximize ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167026752909) on the time bar.
4. Save the dashboard.
