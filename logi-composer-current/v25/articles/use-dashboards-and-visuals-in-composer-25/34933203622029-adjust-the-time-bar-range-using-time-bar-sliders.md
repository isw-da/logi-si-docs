# Adjust the Time Bar Range Using Time Bar Sliders

**Adjust the time bar time range for a visual using the time bar sliders**

1. Select the visual.
2. Slide the From slider (![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167026998157)) on the time bar to the earliest time for the time range you want.
3. Slide the To slider (![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167037132045)) on the time bar to the latest time for the time range you want.
4. Save the dashboard.

To select the maximum range of values, select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167833701005) on the time bar.
