# Even Time Intervals

Even time intervals allow you to include all time groupings from your data source on a dashboard that have date or time data. You can select whether you want to display all time data, or just the data that returns a value. If time groups include a NULL value, these values are included. Filter by row as needed:

* Apply `Is not NULL` to hide null values
* Apply `Is NULL` to include only null values

Even time interval grouping is only available on dashboards that use a data source with date and time fields.
It is also not available for KPI charts or maps.

Even time intervals can be displayed in live mode and historical playback using the time bar. See [Live Mode and Historical Playback](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933163012621-Live-Mode-and-Historical-Playback).
