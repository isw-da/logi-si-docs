# Use the Time Bar Sidebar

The Time Bar sidebar lets you control some of the time bar settings for a visual. Controls for changing the time bar settings are provided using the interactivity sidebar. See [Control How Users Interact With a Visual](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933282294029-Control-How-Users-Interact-With-a-Visual).

**Access the Time Bar sidebar for a visual**

1. Select the visual in the Visual Gallery or on a dashboard.
2. If you selected the visual on a dashboard, select **Settings** on the [visual drop-down menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933290868621-Use-the-Visual-Menu) to access the [sidebar menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933295031181-Use-the-Visual-Sidebar-Menu) for the visual.
3. Select the visual style option (![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167826972173)) on the [sidebar menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933295031181-Use-the-Visual-Sidebar-Menu) for the visual. The Time Bar sidebar opens.

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167024454669)
4. Using the Time Bar sidebar, you can:

   * Enable the time bar. See [Enable the Time Bar for a Visual](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933178405645-Enable-the-Time-Bar-for-a-Visual).
   * Disable the time bar. See [Disable the Time Bar for a Visual](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933178292237-Disable-the-Time-Bar-for-a-Visual).
   * Select or change the time bar field. See [Change the Time Bar Field](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933178568461-Change-the-Time-Bar-Field).
   * Enable playback, if the selected time bar field supports it. See [Live Mode and Historical Playback](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933163012621-Live-Mode-and-Historical-Playback).

   **Note:** If you are using a field that has time zone information disabled (select **[Not Specified](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932879958669-Configure-Time-Bar-Defaults#Default_Time_Attribute)**), only the time-related information is shown in the user interface and exported with your data. Time zone labels are not included.
5. After making changes, select **Apply** to apply the time bar changes you have specified.
6. Save the visual or dashboard to save your changes.
