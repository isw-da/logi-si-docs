# Select the Maximum Time Range

**Select the maximum time range on the time bar**

1. Select the visual.
2. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167027398669) on the right side of the time bar to select the maximum time range on the time bar.
3. Save the dashboard.
