# Use the Filter Snippet Menu

The filter snippet menu includes options that help you modify and use these snippets in your dashboard. Access it by selecting![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167890628493) in the upper right corner of a snippet widget.

The menu options are described in the following table.

| Option | Description |
| --- | --- |
| Edit | Select to edit the [custom values](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932990744589-Custom-Values-for-Filter-Snippets) of a filter snippet by removing them, or if it is open for editing, put it in view mode. Select  to remove a custom value.  You can additionally access this option by selecting the edit icon () in the snippet header. |
| Settings | Open the sidebar menu. Use to   * Define data settings for the filter snippet * Define filter snippet settings * Apply filters and hierarchical filters (if available) to the filter snippet * Adjust widget settings for the filter snippet * Manage comments for the filter snippet |
| Connect Widgets | Select to connect visuals or filters to the filter snippet. See [Connect Visuals to a Filter Snippet](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932972339213-Connect-Visuals-to-a-Filter-Snippet) and [Link Filter Snippets](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933003435149-Link-Filter-Snippets). |
| Remove Widget | Remove the selected filter snippet from the dashboard. See [Edit or Delete Filter Snippets](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933004082701-Edit-or-Delete-Filter-Snippets). |
