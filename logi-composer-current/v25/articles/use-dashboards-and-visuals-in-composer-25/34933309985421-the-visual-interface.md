# The Visual Interface

# The Visual Interface

Composer's canvas provides a comprehensive suite of tools to help you conduct in-depth analysis, filter data points, and discover needed insights quickly and efficiently.

## Tools in the Visual Interface

![Use this work area to adjust your visuals](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166930658061 "Visual Interface in Dashboard")

1. **Display Name**: By default, a visual's display name is the name of the data source used for the visual for local visuals, or the display name for a visual from the visual gallery. Select the title to edit.
2. **Dashboard Name or Title**: The dashboard name if editing a visual in a dashboard. Select the name to edit it or save the dashboard to be prompted for a dashboard name. The default name of the dashboard is **Untitled dashboard**.
3. Indicates whether the dashboard has been saved. This information is not available if you are editing a visual from the Visual Gallery.
4. Indicates whether a visual gallery visual has been saved.

   * If the visual is a local visual, this area is blank. All local visuals are saved when you save the dashboard.
5. **Time Bar** and **Time Controls**:

   * Filter your data by a time attribute.
   * Zoom into specific period within the selected date range.
   * Explore the dynamics of your data using the **Play/Pause** button.

   See [Use the Time Bar](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933204115725-Use-the-Time-Bar).
6. **X-** and **Y- Axis Labels** and the **Color Metric** (availability varies, depending on the visual style): Select an axis label to change an attribute or metric. See [Change the Axes](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933196810125-Change-the-Axes) and [Change the Visual Color Metric](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933197872013-Change-the-Visual-Color-Metric).

   If you disable the Volume property in a data source configuration, it does not appear in visual tooltips. The only exception to this is in histograms which plot the volume or number of records.
7. **Legend:** By default, a legend is available for supported visual styles, and can be switched on or off in the Colors sidebar menu.
8. The dashboard icon bar. Use the options here to manage the dashboard. See [Use the Dashboard Icon Bars](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932842163469-Use-the-Dashboard-Icon-Bars). This is only available on a dashboard; not on a visual edited from the [Visual Gallery](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933265735565-Use-the-Visual-Gallery).
9. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166915300237) to expand a visual on a dashboard. This allows you to get a closer look at it. This is only available on a dashboard; not on a visual edited from the [Visual Gallery](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933265735565-Use-the-Visual-Gallery).
10. The ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166871399693) at the top right of a visual displays the [visual drop-down menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933290868621-Use-the-Visual-Menu). This menu includes an option to open **Settings** for the visual, and its [sidebar menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933295031181-Use-the-Visual-Sidebar-Menu).
11. The visual [sidebar menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933295031181-Use-the-Visual-Sidebar-Menu). Select a sidebar menu option to open an appropriate sidebar for a visual. See [Use the Visual Sidebar Menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933295031181-Use-the-Visual-Sidebar-Menu).
12. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166910226957) to undo the most recent change you made to a visual.
13. Visual-specific options:

    1. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166910345357) to filter the data on a visual.
    2. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166915799053) to view published filters.
    3. The visual icon indicates if a visual is local (![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166910591245)) or a visual gallery visual (![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166910696205)).
14. Dashboard-specific options:

    1. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166910345357) to filter the data on a dashboard. The dashboard-level filter icon is available only when all visuals on a dashboard are from the same data source.
    2. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166916193933) to open dashboard and visual interactivity settings.
    3. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166886943245) to toggle dashboard view mode between [Viewer and Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932832240909-Use-Dashboard-View-Mode) modes.
    4. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166922716941) to enable and disable locking the dashboard layout.
    5. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166922826893) to enable and disable responsive layout for this dashboard. See [Using the Responsive Dashboard Layout](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932815319437-Using-the-Responsive-Dashboard-Layout).
15. The context menu. Use menu options to view or adjust data. See [Use the Context Menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933236349453-Use-the-Context-Menu).
