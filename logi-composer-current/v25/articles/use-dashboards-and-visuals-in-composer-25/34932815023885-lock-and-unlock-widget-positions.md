# Lock and Unlock Widget Positions

The dashboard layout provides a flexible grid format, grouping widgets in rows and columns you can move, swap, or resize to meet your users' needs. Additionally, you can lock some widgets in place by layout row or layout column to preserve their position and placement in your dashboard.

**Note:** 
You must convert older dashboard layouts to use this feature. See [Convert Dashboards](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932779934605-Dashboard-Layouts#convert-dash).

**Lock and unlock widgets**

1. Open a dashboard as an Owner or Editor.
2. Select the Dashboard Layout icon ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167394835597). The Dashboard Layout work area opens. Select the toggle and enable Locking Mode.

   The dashboard layout changes, displaying locked or unlocked icons on widgets and locked icons on locked rows.

   ![lock and unlock layout columns and rows](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167410453133 "A dashboard in locking mode")
3. To lock a widget, select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167349513357) to lock the column position and row height of the widget. The icon changes to ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167333014029) to show it is locked.

   To unlock a widget, select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167333014029) to unlock the column position and row height of the widget. The icon changes to ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167349513357) show it is unlocked.

   ![lock and unlock layout columns and rows](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167384872205 "A dashboard in locking mode")
4. To lock or unlock a row of widgets, select the far left border of a row to highlight the row. The dashboard layout changes, displaying an unlocked icon on the selected unlocked row.

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167366845581)
5. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167349513357) to lock the row's height and position, or ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167333014029) to unlock the row.
6. Select the Dashboard Layout icon ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167394835597) and disable the Locking Mode toggle.
7. Save your changes.

After you've locked the position of some of the widgets in your dashboard, you can't:

* Change the width of the locked layout column.
* Move the locked widgets.
* Change the height of a locked layout row or any layout row that includes locked widgets.
* Insert widgets next to or among some sides of locked widgets. Instead of a solid blue guidance line, Composer displays a gray dashed line.

You can swap an unlocked widget with a locked widget: select and drag an unlocked widget to swap it with a locked widget. See [Move, Swap, and Resize Visuals and Widgets in a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932819109389-Move-Swap-and-Resize-Visuals-and-Widgets-in-a-Dashboard).
