# Remove Cross-Source Links

Cross-source links can be removed on the Cross-Source Links tab of the Dashboard Interactions dialog. This will delete the cross-source link definition from the Composer environment.

**Remove a cross-source link from a dashboard**

1. Open the dashboard containing the cross-source link you want to edit.
2. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167956167181) on the [dashboard icon bar](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932842163469-Use-the-Dashboard-Icon-Bars). The Dashboard Interactions dialog appears. Cross-source links are listed on the **Cross-Source Links** tab.

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167416015117)
3. Select the link you want to remove in the **Links** list on the left of the tab. Its definition appears on the right.
4. Select the ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167956168077) icon associated with the link you want to remove. A warning dialog appears prompting you to confirm the deletion.

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167390545933)
5. Select **Delete**. The cross-source link is removed from the dialog.
6. [Save](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932815496845-Save-a-Dashboard) the dashboard to delete the cross-source link definition.
