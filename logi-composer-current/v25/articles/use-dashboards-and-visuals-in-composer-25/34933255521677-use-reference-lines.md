# Use Reference Lines

Reference lines on a visual can be used to mark limits, thresholds, and critical values. They can mark both positive and negative values on an axis. Multiple reference lines can be added in a visual.

Reference lines are available only for bar and line charts. Controls are provided using the Rulers interactivity sidebar. See [Control How Users Interact With a Visual](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933282294029-Control-How-Users-Interact-With-a-Visual).

Reference lines can be customized for a visual using the Rulers sidebar.
