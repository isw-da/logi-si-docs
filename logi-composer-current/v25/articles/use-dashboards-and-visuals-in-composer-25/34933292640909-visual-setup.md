# Visual Setup

# Visual Setup

You can control visuals available for each data source configuration you create using the Available Visual Types work area. See [Available Visual Types](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933207922445-Available-Visual-Types).

After saving a data source definition, adjust settings applicable to new visuals on the [Global Settings Tab](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932890843277-Global-Settings-Tab).

To add a visual, see:

* [Create Dashboards](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932763355021-Create-Dashboards)
* [Create and Add Visuals to the Visual Gallery](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933273224589-Create-and-Add-Visuals-to-the-Visual-Gallery)
* [Add Visuals to a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933303418509-Add-Visuals-to-a-Dashboard)
* [Add Local Visuals to a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933273131405-Add-Local-Visuals-to-a-Dashboard)
* [Add Existing Visuals to a Dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933272990989-Add-Existing-Visuals-to-a-Dashboard)

After you create a visual, you can alter its settings. See [The Visual Interface](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933309985421-The-Visual-Interface).
