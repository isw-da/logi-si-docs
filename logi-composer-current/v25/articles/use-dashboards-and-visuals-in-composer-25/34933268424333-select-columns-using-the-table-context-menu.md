# Select Columns Using the Table Context Menu

**Select fields for table columns using the table context menu**

1. View the table visual in a dashboard or from the Visual Gallery.
2. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167839792781) next to the column heading name to access the table context menu.

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167839794189)
3. Select the ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166992012557) tab on the context menu. A list of all the fields in the data source appears.

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166992133773)
4. Select the fields you want to view as columns in the table. To select all the fields, select the checkbox to the left of the search bar.

   Use the search bar to search for a field in the table.

   Fields you select appear in the table.
5. Save the dashboard and visual.
