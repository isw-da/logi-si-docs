# About Scheduled Dashboard Reports

You can generate scheduled dashboard reports and send them to your users. Select users from the list of users available to you in Composer, or add an email address for a non-Composer user to a scheduled report for delivery. Send reports in PDF, PNG, Data (XLSX) > Raw Data or Data (XLSX) > Visual Data formats.

Log messages related to dashboard report scheduling are stored in the `zoomdata.log`[log file](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933097277069-Composer-Log-Files-Reference).

Specific prerequisites must be met to schedule dashboard reports. For more information, see:

* [Configure Recipient Rules](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932843478029-Configure-Recipient-Rules)
* [Update a Scheduled Dashboard Report](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932809627021-Update-a-Scheduled-Dashboard-Report)

* [Scheduled Dashboard Report Permissions and Behavior](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932823947149-Scheduled-Dashboard-Report-Permissions-and-Behavior)

**Note:** 
By default, you can send scheduled reports to both users with Logi Composer accounts, and email addresses outside of Logi Composer. Work with Technical Support if you need to disable this functionality in your environment. See [Disable Sending Scheduled Reports to External Users](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932791412749-Disable-Sending-Scheduled-Reports-to-External-Users).
