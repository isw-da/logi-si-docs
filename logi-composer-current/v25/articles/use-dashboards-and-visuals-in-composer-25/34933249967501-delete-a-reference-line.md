# Delete a Reference Line

**Delete a reference line on a visual**

1. Access the Rulers sidebar for the visual. See [The Rulers Sidebar](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933277411085-The-Rulers-Sidebar).
2. In the **Reference Lines** section of the sidebar, locate the reference line you want to delete.
3. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166937188621) to the right of the reference line definition on the Rulers sidebar.
4. Select **Apply** to delete the reference line onthe visual.
