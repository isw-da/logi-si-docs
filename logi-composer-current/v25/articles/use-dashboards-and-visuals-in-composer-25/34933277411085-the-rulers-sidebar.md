# The Rulers Sidebar

# The Rulers Sidebar

The Rulers sidebar for a visual allows you to customize the rulers and reference lines used by the visual. When you save or export your visual, the settings configured on the Rulers sidebar are saved or exported respectively.

**Access the Rulers sidebar**

1. Select the visual in the [Visual Gallery](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933265735565-Use-the-Visual-Gallery) or on a dashboard.
2. Select **Settings** on the [visual drop-down menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933290868621-Use-the-Visual-Menu) to access the [sidebar menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933295031181-Use-the-Visual-Sidebar-Menu) for the visual. Then select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167822628877) on the [sidebar menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933295031181-Use-the-Visual-Sidebar-Menu). The Rulers sidebar opens.

   The sidebar consists of two sections: the (metric) ruler settings and, possibly, reference lines. If the visual does not support reference lines, this section is not available. See [Enable and Disable Ruler Grid Lines](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933286653197-Enable-and-Disable-Ruler-Grid-Lines) and [Use Reference Lines](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933255521677-Use-Reference-Lines).

   **Note:** 
   Arc gauges use the Rulers sidebar to control some different settings than are used by most visuals. See [Arc Gauges](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933196538125-Arc-Gauges).

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46166956215693)
3. Select **Apply** to apply any changes you make.
