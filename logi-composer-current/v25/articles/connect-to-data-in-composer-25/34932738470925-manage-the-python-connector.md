# Manage the Python Connector

Connect your data in Composer using a Python script. Access information exposed by APIs, or other data generated by calculation or prediction models.

**Important:** 
The Python connector is available as a Docker image; you must install Docker on your server running the Python connector. See [Install the Python Connector](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932781059213-Install-the-Python-Connector).

Python code is executed using [JEP](https://github.com/ninia/jep). To circumvent some Global Interpreter Lock issues in Python, some queries use [processed based parallelism](https://docs.python.org/3/library/multiprocessing.html). Based on the request type, the connector functions in one of two ways:d

* Interpret the script in the same process for validation and describe requests.
* Interpret the script in the same process and invoke the function in a sub process for fetch data requests.

For more information, see [Use the Python Connector](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932798246413-Use-the-Python-Connector).
