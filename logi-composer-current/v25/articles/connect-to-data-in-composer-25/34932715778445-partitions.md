# Partitions

# Partitions

Some connectors support partitions and pruning. This feature enables the Partition column on the Fields tab of the [data source configuration](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932895238669-Data-Sources-Page), when the data source uses a supporting connector. Partitioning allows you to link a partitioned field to another field to help improve the performance of filtering operations for a data source.

Although many data stores support partitions in some form, this feature specifically tells Composer that the partitions may be used for manual pruning of result sets to increase speed.
