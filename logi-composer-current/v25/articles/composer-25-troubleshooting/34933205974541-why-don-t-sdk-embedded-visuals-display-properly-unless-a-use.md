# Why Don't SDK-Embedded Visuals Display Properly Unless a User is Logged In?

Please make sure that the API key that was generated for your SDK implementation has appropriate access to ALL the data sources being used by your embedded visuals in your custom HTML page.

Please refer to our  [Security Configuration Object](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933128778637--Security-Configuration-Object) support topic for more information.
