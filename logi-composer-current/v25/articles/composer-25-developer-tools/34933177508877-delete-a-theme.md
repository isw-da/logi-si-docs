# Delete a Theme

You can delete the JSON definition of a theme for the Composer [UI](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933073981837-Configurable-Data-Visualization). You must know the theme ID before you can delete it.

**Delete a theme**

1. Obtain the theme ID. You can do this by listing the themes in your environment. See [List Themes](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933202083341-List-Themes).
2. Use the `/api/customization/themes/<id>` API endpoint in a DELETE request to delete the theme. The following request deletes the theme named **mytheme**.

   curl -X DELETE "http://<ip-address>:<port>/composer/api/customization/themes/mytheme" -H "accept: application/vnd.composer.v3+json"

   where `<ip-address>` is the IP address or host name of your Composer instance and `<port>` is its port.

   The theme is deleted.
