# Access the Custom Metrics Editor from the Color Sidebar

**Access the Custom Metrics Editor from the Color sidebar of a visual**

1. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167931385101) and then **Color** from the [visual drop-down menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933290868621-Use-the-Visual-Menu) to access the Color sidebar.

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167941486989)
2. Select the **Color Attribute** box on the Color sidebar. A list of attribute, number, and time fields you can select for the color attribute appears in the Color sidebar.
3. Select ![add icon](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167941487629 "add icon") in the Attribute, Number, or Time fields section of the sidebar. A menu opens with two options: **Add Derived Field** and **Add Custom Metric**.
4. Select **Add Custom Metric** to access the [Custom Metrics Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932820475405-Custom-Metrics-Editor) and create a custom metric.

You can also access the Custom Metrics Editor when you edit any custom metric listed on the Color sidebar.

1. Select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167931385101) and then Color from the menu to access the Color sidebar.
2. Select the **Color Metric** box on the Color sidebar. A list of fields you can select for the color metric appears in the Color sidebar.
3. Locate a custom metric listed on the Color sidebar and select the ellipsis (![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167931385101)) next to it. The following menu appears:

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167931389837)
4. Select **Edit** on the menu to edit the custom metric. The [Custom Metrics Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932820475405-Custom-Metrics-Editor) appears.
