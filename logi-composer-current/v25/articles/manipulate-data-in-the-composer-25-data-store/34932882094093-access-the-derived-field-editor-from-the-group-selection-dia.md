# Access the Derived Field Editor from the Group Selection Dialog

**Access the Derived Field Editor from the Group selection dialog of a visual**

1. On a visual, select the group label (x-axis label) to view the Group selection dialog.

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167290287245)
2. Select ![add icon](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167918070669 "add icon") in the Attributes, Number or Time section of the dialog. A menu opens with two options: **Add Derived Field** and **Add Custom Metric**.
3. Select **Add Derived Field** to access the [Derived Field Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932882620813-Derived-Field-Editor) and create a derived field.

You can also access the Derived Field Editor when you edit any derived field listed on the Group selection dialog. See the following steps.

**Edit a derived field**

1. On a visual, select the group label (x-axis label) to view the Group selection dialog.

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167290398605)
2. Locate a derived field listed on the Group selection dialog and select the ellipsis (![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167906719629)) next to it. The following menu appears:

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167918071821)
3. Select **Edit** on the menu to edit the derived field. The [Derived Field Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932882620813-Derived-Field-Editor) appears.
