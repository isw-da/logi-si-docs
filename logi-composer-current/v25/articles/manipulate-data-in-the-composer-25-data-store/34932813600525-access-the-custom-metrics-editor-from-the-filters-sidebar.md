# Access the Custom Metrics Editor from the Filters Sidebar

**Access the Custom Metrics Editor from the Filters sidebar of a visual or dashboard**

1. Access the visual filter sidebar or the dashboard filter sidebar.

   * To access the visual filter sidebar, select the filter icon (![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167907472653)) on the visual or select ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167907476237) on the [sidebar menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933295031181-Use-the-Visual-Sidebar-Menu). If the visual is in a dashboard, select **Settings** from the [visual menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933290868621-Use-the-Visual-Menu) (![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167918859149)), then select the filter icon ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167907476237) on the [sidebar menu](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34933295031181-Use-the-Visual-Sidebar-Menu).
   * To access the dashboard filter sidebar, select the dashboard filter icon (![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167918859533)) next to the dashboard title. The dashboard-level filter icon is available only when all the visuals are from the same data source.

   The Filters sidebar appears.
2. Select the add icon ![add icon](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167918864525 "add icon") at the top of the Attribute, Number, or Time sections on the Row or Group tab. A menu opens with two options: **Add Derived Field** and **Add Custom Metric**.
3. Select **Add Custom Metric** to access the [Custom Metrics Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932820475405-Custom-Metrics-Editor) and create a custom metric.

You can also access the [Custom Metrics Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932820475405-Custom-Metrics-Editor) when you edit any custom metric listed on the Filters sidebar. See the following steps.

**Edit a custom metric**

1. Select the filter icon to access the Filters sidebar for a [visual](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932931145741-Apply-a-Row-Level-Filter-to-a-Visual-or-Filter-Snippet) or [dashboard](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932922987533-Apply-a-Row-Level-Filter-to-a-Dashboard).
2. Select **Add Filter**.
3. Locate a custom metric listed on the Filters sidebar and select the ellipsis (![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167918859149)) next to it. The following menu appears:

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167907481869)
4. Select **Edit** on the menu to edit the custom metric. The [Custom Metrics Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932820475405-Custom-Metrics-Editor) appears.
