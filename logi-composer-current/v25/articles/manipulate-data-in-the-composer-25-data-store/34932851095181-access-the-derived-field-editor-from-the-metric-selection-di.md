# Access the Derived Field Editor from the Metric Selection Dialog

**Access the Derived Field Editor from the metric selection dialog of a visual**

1. On a visual, select the metric label (y-axis label) to view the Metric selection dialog.

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167306410893)
2. Select the ![add icon](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167940518669 "add icon") icon in the Number or Custom Metric section of the dialog. A menu opens with two options: **Add Derived Field** and **Add Custom Metric**.
3. Select **Add Derived Field** to access the [Derived Field Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932882620813-Derived-Field-Editor) and create a derived field.

You can also access the [Derived Field Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932882620813-Derived-Field-Editor) when you edit any derived field listed on the Metric selection dialog. See the following steps.

**Edit a derived field**

1. On a visual, select the metric label (y-axis label) to view the Metric selection dialog.

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167306410893)
2. Locate a derived field listed on the Metric selection dialog and select the ellipsis (![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167930378765)) next to it. The following menu appears:

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167350590477)
3. Select **Edit** on the menu to edit the derived field. The [Derived Field Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932882620813-Derived-Field-Editor) appears.
