# Access the Custom Metrics Editor from the Group Selection Dialog

**Access the Custom Metrics Editor from the Group selection dialog of a visual**

1. On a visual, select the group label (x-axis label) to view the Group selection dialog.

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167931315085)
2. Select ![add icon](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167931315341 "add icon") in the Attributes, Number or Time section of the dialog. A menu opens with two options: **Add Derived Field** and **Add Custom Metric**.
3. Select **Add Custom Metric** to access the [Custom Metrics Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932820475405-Custom-Metrics-Editor) and create a custom metric.
