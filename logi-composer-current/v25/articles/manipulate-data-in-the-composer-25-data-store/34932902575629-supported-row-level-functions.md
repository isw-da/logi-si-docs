# Supported Row-Level Functions

# Supported Row-Level Functions

Use row-level functions in the row-level expressions you use to create the following calculation types:

* [derived fields](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932839148813-Maintain-Derived-Fields)
* [custom metrics](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932836147341--Maintain-Custom-Metrics)
* [admin-defined functions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932796673933-Admin-Defined-Functions)

The row-level functions described in this section are fully supported by all three types. For information about the aggregate functions and SQL-like expressions you can use in custom metrics, see [Supported Aggregation Functions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932900526221-Supported-Aggregation-Functions) and [Supported SQL-Like Expressions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932861652621-Supported-SQL-Like-Expressions).

Row-level functions are divided into the following categories:

* [Arithmetic Functions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932852858253-Arithmetic-Functions)
* [Conditional Functions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932874388621-Conditional-Functions)
* [Logical Functions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932884487693-Logical-Functions)
* [Numerical Functions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932884622733-Numerical-Functions)
* [Relational Functions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932853343373-Relational-Functions)
* [Text Functions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932874867597-Text-Functions)
* [Time Functions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932853603341-Time-Functions)
