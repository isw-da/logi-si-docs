# Supported Aggregation Functions

Use aggregate functions in [custom metrics](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932836147341--Maintain-Custom-Metrics). Custom metrics can also include the use of [row-level functions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932902575629-Supported-Row-Level-Functions), conditional CASE expressions, statistical functions, an expansive list of arithmetic functions, as well as FIRST and LAST values. In addition, they can be [filtered](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932814261773-Apply-Filters-to-Custom-Metrics) (using [date and time filter functions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932852083725-Date-and-Time-Filter-Aggregation-Functions) and [SQL-like expressions](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932861652621-Supported-SQL-Like-Expressions)).

Data can be aggregated using column, table, or window aggregation functions. Each is explained in the links below. Aggregation functionality is explained in the links below.
