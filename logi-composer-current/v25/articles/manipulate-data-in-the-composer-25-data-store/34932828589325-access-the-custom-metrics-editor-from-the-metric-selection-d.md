# Access the Custom Metrics Editor from the Metric Selection Dialog

**Access the Custom Metrics Editor from the metric selection dialog of a visual**

1. On a visual, select the metric label (y-axis label) to view the Metric selection dialog.

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167907389581)
2. Select the ![add icon](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167918785421 "add icon") icon in the Number or Custom Metric section of the dialog. A menu opens with two options: **Add Derived Field** and **Add Custom Metric**.
3. Select **Add Custom Metric** to access the [Custom Metrics Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932820475405-Custom-Metrics-Editor) and create a custom metric.

You can also access the [Custom Metrics Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932820475405-Custom-Metrics-Editor) when you edit any custom metric listed on the Metric selection dialog. See the following steps.

**Edit a custom metric**

1. On a visual, select the metric label (y-axis label) to view the Metric selection dialog.

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167907389581)
2. Locate a custom metric listed on the Metric selection dialog and select the ellipsis (![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167918785677)) next to it. The following menu appears:

   ![](https://logi-composer-v25.insightsoftware.com/hc/article_attachments/46167907393037)
3. Select **Edit** on the menu to edit the custom metric. The [Custom Metrics Editor](https://logi-composer-v25.insightsoftware.com/hc/en-us/articles/34932820475405-Custom-Metrics-Editor) appears.
